## Community projects

# [Alibaba](/projects/spring-cloud-alibaba)

With Spring Cloud Alibaba, you can easily connect your Spring applications to Alibaba services.

# [Amazon Web Services](/projects/spring-cloud-aws)

Spring Cloud for Amazon Web Services eases the integration of Spring applications with hosted Amazon Web Services.

# [Google Cloud Platform](/projects/spring-cloud-gcp)

Using Google Cloud from Spring needn’t be taxing. The Spring Cloud GCP project makes the Spring Framework a first-class citizen of Google Cloud Platform (GCP)

# [Microsoft](/projects/spring-cloud-azure)

The Spring Cloud for Microsoft Azure project provides seamless integration to Azure managed services from your Spring applications. And there’s also Azure Spring Cloud, a platform for running Spring Boot microservice applications on Azure.
