# Why Tanzu Spring

## The Stewards of Spring

All of the committers for Spring Boot, Spring Framework, and the rest of the OSS Spring portfolio are part of the Tanzu team. That team not only develops the frameworks developers know and love, but are in a unique position to provide trustworthy security patches only available via Tanzu Spring and craft highly-relevant commercial features.

## Reduced upgrade costs

Upgrading libraries takes time away from developing features that provide business value. Tanzu’s Spring Application Advisor provides automated upgrades for your Spring applications so that you can spend more time focusing on delivering features than upgrading libraries.

## Premium support for the full application stack

Global, 24x7 support for over 50 Spring projects, plus OpenJDK™ and Tomcat® with fast response times and access to Broadcom docs, resources, and knowledge base.

## Security and compliance

Tanzu Spring includes Spring Boot extensions that simplify adding enterprise-grade security and compliance features to your applications. FIPS, PCI-DSS, and other compliance audit tasks are simplified by just adding a Spring Boot starter to your application.

## Easy setup

Get started easily with Tanzu Spring's Maven artifact repository. Being a Maven repository means it seamlessly integrates into your Maven and Gradle builds. Learn more about configuration Spring Artifact Repository [here](https://techdocs.broadcom.com/us/en/vmware-tanzu/spring/tanzu-spring/commercial/spring-tanzu/spring-enterprise-subscription.html).
