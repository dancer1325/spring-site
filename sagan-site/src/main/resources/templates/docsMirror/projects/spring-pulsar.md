# ![Spring for Apache Pulsar](/img/projects/spring.svg)Spring for Apache Pulsar2.0.3[](http://github.com/spring-projects/spring-pulsar "Github")[](https://stackoverflow.com/questions/tagged/spring-pulsar "Stack Overflow")

The Spring for Apache Pulsar (spring-pulsar) project applies core Spring concepts to the development of Pulsar-based messaging solutions. It provides a "template" as a high-level abstraction for sending messages. It also provides support for Message-driven POJOs with `@PulsarListener` annotations and a "listener container". You will see similarities to the Apache Kafka support provided in the Spring for Apache Kafka project.

## [](#features)[](#features)Features

-   `PulsarTemplate`
    
-   `@PulsarListener`
    
-   `@PulsarReader`
    

## [](#pulsar-client-and-spring-boot-compatibility)[](#pulsar-client-compatibility)Pulsar Client and Spring Boot Compatibility

Spring for Apache Pulsar is based on the pure java Pulsar client. The compatibility matrix can be found [here](https://docs.spring.io/spring-pulsar/reference/appendix/version-compatibility.html).
