# This Week in Spring - July 3rd, 2012

_Engineering | Josh Long |  July 04, 2012 | 0 Comments_

Welcome back to another installment of *This Week in Spring*. Today, [we bid farewell to the father of Spring, Rod Johnson](http://blog.springsource.org/2012/07/03/oh-the-places-youll-go/). We wish Rod well on his next endeavors.

1.  Spring contributor and all-around-build-ninja Chris Beams put together a nice blog talking [about some of the infrastructure changes in the Spring projects of late](http://blog.springsource.org/2012/06/27/spring-project-infrastructure-updates/), including their moves to GitHub and to the build-tool Gradle.
2.  Martin Lippert has announced the [availability of the SpringSource Tool Suites 3.0.0.M2, one for Spring and one for Groovy and Grails](http://www.springsource.org/node/3582).
3.  [Spring Data Redis 1.0.1 has been released](http://www.springsource.org/node/3581). Among other things the new release supports support for a new (4th) Redis driver SRP, Redis native execution (`RedisConnection#execute`), and improved pipeline execution tracking potential errors and bulk results consistently across all drivers

7.  Our friend Willie Wheeler has a great post [on pageable custom queries with Spring Data JPA](http://springinpractice.com/2012/06/30/pageable-custom-queries-with-spring-data-jpa/).

It's nice when the aggregators pick up good content, but readers should also be sure to check out the source for all your SpringSource videos and tutorials, the [SpringSource Dev YouTube channel](http://www.youtube.com/SpringSourceDev).
