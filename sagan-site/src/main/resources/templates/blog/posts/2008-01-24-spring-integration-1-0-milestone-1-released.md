# Spring Integration 1.0 Milestone 1 Released

_Releases | Mark Fisher |  January 24, 2008 | 0 Comments_

Dear Spring community,  
  
I am pleased to announce that Spring Integration 1.0.0.m1 has been released.  
[Download](http://static.springframework.org/downloads/nightly/milestone-download.php?project=INT)| [Reference Documentation](http://static.springframework.org/spring-integration/reference/htmlsingle/spring-integration-reference.html)| [JavaDoc](http://static.springframework.org/spring-integration/apidocs/)  
  
This is the first milestone release of this new addition to the Spring Portfolio. To learn more about the project and what lies ahead, visit the [Spring Integration Home](/spring-integration). To ask questions, provide feedback, or report issues during this important phase of design and development, please visit the Spring Integration [Forum](http://forum.springframework.org/forumdisplay.php?f=42) and [Issue Tracker](http://jira.springframework.org/browse/INT).  
  
Mark Fisher  
Spring Integration Lead
