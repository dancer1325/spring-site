# Spring Security 3.1.0.RC1 Released

_Releases | Luke Taylor |  February 10, 2011 | 0 Comments_

We're pleased to an announce the first release candidate for Spring Security 3.1

Please see the [changelog](http://static.springsource.org/spring-security/site/changelog.html) for a full list of updates since the M2 release.
