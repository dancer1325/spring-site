# Spring Cloud 2022.0.11 (aka Kilburn) Has Been Released

_Releases | Ryan Baxter |  June 02, 2025 | 0 Comments_

On behalf of the Spring Cloud team, I am pleased to announce that the [Spring Cloud 2022.0.11](https://enterprise.spring.io/projects/spring-cloud) Release Train is available for [support customers](https://www.vmware.com/products/app-platform/tanzu-spring). The release can be found in [https://packages.broadcom.com](https://packages.broadcom.com).

## [](#notable-changes-in-the-202201-release-train)Notable Changes in the 2022.0.1 Release Train

-   This release is compatible with Spring Boot 3.0.20 and 3.1.17.
-   Spring Cloud Commons 4.0.7 has dependency upgrades.
-   Spring Cloud Task 3.0.6 has dependency upgrades.
-   Spring Cloud Config 4.0.11 dependency upgrades and contains a fix for [CVE-2025-22232](https://spring.io/security/cve-2025-22232).
-   Spring Cloud Netflix 4.0.8 has dependency upgrades.
-   Spring Cloud OpenFeign 4.0.9 has dependency upgrades.
-   Spring Cloud Gateway 4.0.12 has dependency upgrades and contains a fix for [CVE-2025-41235](https://spring.io/security/cve-2025-41235).
-   Spring Cloud Contract 4.0.11 has dependency upgrades.
-   Spring Cloud Kubernetes 3.0.7 has dependency upgrades.
-   Spring Cloud Vault 4.0.6 has dependency upgrades.
