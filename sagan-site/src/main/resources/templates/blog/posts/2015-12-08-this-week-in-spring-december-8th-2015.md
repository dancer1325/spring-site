# This Week in Spring - December 8th, 2015

_Engineering | Josh Long |  December 08, 2015 | 0 Comments_

Welcome to another installation of *This Week in Spring*! This week I'm in Chicago speaking to customers and then it's off to [Washington D.C.](http://www.meetup.com/DC-Cloud-Native-Meetup/events/226775603/), and [New York City](http://www.meetup.com/Cloud-Native-New-York/events/226870607/) for Cloud Native meetups! I hope you'll consider joining us at either event!

Time sure is flying! We're darned close to 2016!

Anyway, as usual, *lots* to cover so let's get to it!

-   I really like the looks of this Caffeine local cache. It uses a pretty [interesting algorithm to ensure near optimal cache hits](https://www.voxxed.com/blog/2015/12/add-a-boost-of-caffeine-to-your-java/) It already features a JSR 170 (JCache) integration which means using it could be as simple as configuring the JCache adapter along with Spring's Cache abstraction.
-   I really enjoyed Pivotal Labs' Brian Kelly's presentation, [*Spring Boot for Rails Developers*](http://presentations.spilth.org/spring-boot-for-rails-developers/#1). Check it out!
-   3Pillar has an interesting [post on using Spring Boot with Docker](http://www.3pillarglobal.com/insights/building-microservice-architecture-spring-boot-docker-part-iv)
-   Spring Framework lead Juergen Hoeller [has revisited the Spring 5.0 roadmap in light of the recent 6 month delay for JDK 9](https://spring.io/blog/2015/12/03/spring-framework-5-0-roadmap-update)
-   Spring RestDocs lead, Spring IO Platform lead, and Spring Boot ninja Andy Wilkinson [has just announced that the Gradle dependency management plugin 0.5.4.RELEASE is now available](https://spring.io/blog/2015/12/03/gradle-dependency-management-plugin-0-5-4-released)
-   Spring Cloud and Cloud Foundry ninja Scott Frederick has just [announced that the Spring Cloud Connectors 1.2.1 are now available](https://spring.io/blog/2015/12/02/spring-cloud-connectors-1-2-1-released)
-   Simon Baslé and Laurent Doguin's presentation [on using RxJava with Spring Boot is now available on InfoQ](http://www.infoq.com/presentations/rxjava-spring-boot)
-   our pal Petri Kainulainen is at it again, this time with a [post on adding custom methods into *all* Spring Data repositories](http://www.petrikainulainen.net/programming/spring-framework/spring-data-jpa-tutorial-adding-custom-methods-into-all-repositories)
-   I really enjoyed Biju Kunjummen's blog on [using the Spring Cloud auto-configured `RestTemplate` with Netflix Ribbon](http://www.java-allandsundry.com/2015/12/spring-cloud-rest-client-with-netflix.html)
-   ..as well as on [using Spring Cloud's support for Hystrix](http://www.java-allandsundry.com/2015/11/spring-cloud-support-for-hystrix.html)
