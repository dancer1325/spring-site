# RabbitMQ 2.0 is Now Available

_Releases | Adam Fitzgerald |  August 30, 2010 | 0 Comments_

[![](/files/rabbitmq_logo_258px.png)](http://www.rabbitmq.com)

Great news for everyone interested in advanced, scalable, enterprise and cloud messaging! The RabbitMQ team has just released a brand new version, [RabbitMQ 2.0](http://www.rabbitmq.com/blog/2010/08/27/growing-up/), of their popular messaging server. The release includes an all new scalable storage engine, native support for multi-protocol messaging, improvements to plugin distribution and much more.

[Download](http://www.rabbitmq.com/download.html) | [Documentation](http://www.rabbitmq.com/documentation.html) | [Release Notes](http://lists.rabbitmq.com/pipermail/rabbitmq-announce/attachments/20100825/2c672695/attachment.txt) | [Blog](http://www.rabbitmq.com/blog/2010/08/27/growing-up/)

Spring Community members can use the new [Spring AMQP](/spring-amqp) project for both Java and .NET connectivity to RabbitMQ. RabbitMQ and Spring AMQP will be among the incredible technical content featured at [SpringOne 2GX 2010](http://www.springone2gx.com) in Chicago, October 19th-21st. Be sure to [register now](http://www.springone2gx.com/conference/chicago/2010/10/register) to lock in an early registration discount.
