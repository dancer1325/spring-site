# Spring for GraphQL 1.1.4 released

_Releases | Brian Clozel |  May 16, 2023 | 0 Comments_

I'm pleased to announce that the Spring for GraphQL 1.1.4 maintenance version is now available on Maven Central.

Spring for GraphQL 1.1.4 includes [11 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.1.4), including important fixes for Observability support. This version will be shipped with Spring Boot 3.0.7, [to be released this Thursday](https://calendar.spring.io/).
