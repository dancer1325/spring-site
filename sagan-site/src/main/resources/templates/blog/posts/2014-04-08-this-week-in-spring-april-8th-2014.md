# This Week in Spring - April 8th, 2014

_Engineering | Josh Long |  April 08, 2014 | 2 Comments_

Welcome to another installment of *This Week in Spring*!

Tomorrow, I'll be presenting on the [vJUG, a *virtual* JUG](http://www.meetup.com/virtualJUG/), about [Spring Boot](http://spring.io/projects/spring-boot). Don't tell anyone, but the presentation itself is going to be based on a talk that Spring Boot co-founder [Phil Webb](http://twitter.com/phillip_webb) and I have been planning. The code is already [online](https://github.com/joshlong/spring-doge), so check it out and I look forward to seeing you tomorrow! If you miss the talk, and you happen to be in the bay area, then [join us on Thursday evening at the Pivotal offices](http://www.meetup.com/Pivotal-Open-Source-Hub/events/170704582/) here in San Francisco for a more caffeinated version of the same talk with *both* Phil and myself!

Then, this Sunday, I'm off to spread the Spring love! I'll touchdown first in Paris for [Devoxx FR](http://www.devoxx.fr/). From there, it's off to Bangalore, India, for the [Great Indian Developer Summit](http://www.developermarch.com/developersummit/). I haven't been there since 2012 and am very much looking forward to it and to seeing the amazing Spring community there. Then, on my way home I'll pop in to Belgium for a [talk at the Belgium JUG](http://www.bejug.org/confluenceBeJUG/display/BeJUG/Home). Most of the talks I'll be doing are Boot-centric, so don't miss a chance to see why Spring Boot is turning heads.

1.  Spring co-lead [Dr. Syer announced that Spring Boot 1.0.1 is now available](http://spring.io/blog/2014/04/07/spring-boot-1-0-1-release-available-now). The new release includes a security fix *and* new features. Check it out, upgrade now!

-   Spring ninja Stéphane Nicoll reminds us that JSR-107 annotations support will be available in Spring Framework 4.1.0.BUILD-SNAPSHOT. Nice!
-   [http://spring.io](http://spring.io) architect Chris Beams [has posted another installment](http://spring.io/blog/2014/04/04/project-sagan-zero-downtime-deployments) in his series looking at project Sagan (the [spring.io website](http://spring.io)). This post details how the website acheives zero-downtime deployments. Check it out!
-   [Spring Data GemFire 1.3.4](http://projects.spring.io/spring-data-gemfire/) has been released. I wish I had release notes or details on whats included, but I don't.
-   Check out this case study of how [EMC uses Spring and tcServer](http://blog.gopivotal.com/pivotal/case-studies-2/emc-qa-part-2-how-spring-reduces-costs-and-improves-productivity)
-   Dr. Mark Pollack has [announced that the Spring Shell 1.1 RC1](http://spring.io/blog/2014/04/03/spring-shell-1-1-rc1-released) has been released. The new release features bug fixes, an update to JLine 2, and a rewrite of the command parser.
-   Thomas Risberg has announced that [Spring for Apache Hadoop 2.0 RC2](http://spring.io/blog/2014/04/02/spring-for-apache-hadoop-2-0-rc2-released) has been released
-   Spring web-dude Rossen Stoyanchev has announced that Spring WebFlow 2.4 RC1 has been released. You know what *my* favorite new feature in that release is? Java configuration support! Last week, the first cuts of the Spring Integration Java configuration API were announced and, I'm happy to report, work is underway for the Spring WS Java configuration API. With Spring Web Flow, Spring Integration and Spring WS, there will be Java configuration APIs for all the major Spring projects. (wooo!)
-   Our friend Tobias Flohre's at it again, this time with a post [detailing how the Spring Batch `BatchStatus` works](https://blog.codecentric.de/en/2014/04/spring-batch-batchstatus-state-transitions/). Nice!
-   Hortonworks have put up a nice post detailing how to use [Spring XD and Hortonworks Hadoop to stream Tweets to Hadoop for sentiment analysis](http://hortonworks.com/hadoop-tutorial/using-spring-xd-to-stream-tweets-to-hadoop-for-sentiment-analysis/). Of course, you could use other Hadoops as well.
