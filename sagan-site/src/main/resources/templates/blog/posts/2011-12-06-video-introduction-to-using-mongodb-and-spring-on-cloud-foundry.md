# Video: Introduction to using MongoDB and Spring on Cloud Foundry

_News | Adam Fitzgerald |  December 06, 2011 | 0 Comments_

In this video [Jared Rosoff](http://twitter.com/#!/forjared) from 10gen and [Thomas Risberg](http://blog.springsource.org/author/trisberg/) from SpringSource show what the combination of [MongoDB](http://www.mongodb.org/) and [Cloud Foundry](http://cloudfoundry.com/signup?utm_source=springsource_web&utm_medium=banner&utm_campaign=cloudfoundry-signup) has to offer to Spring developers. They develop a sample application combining MongoDB and the Spring Data project, and show how to build and deploy it to the cloud. After 60 minutes, you too will be ready to develop and deploy to the cloud your own Spring applications that use MongoDB.
