# Spring Tools 4.28.0 released

_Releases | Martin Lippert |  February 05, 2025 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.28.0 release of the Spring Tools for Visual Studio Code, Eclipse and Theia.

#### [](#highlights)highlights

-   *(Spring Boot)* Spring index cache optimizations leading to performance and memory improvements
-   *(Spring Boot)* additional performance improvements while running Spring specific validations
-   *(Spring Boot)* Boot LS JAR size decreased to 207Mb from 271Mb

#### [](#updates-to-the-spring-tools-for-eclipse-distribution)updates to the Spring Tools for Eclipse distribution

-   early access builds available for Eclipse 2025-03 milestone builds

Detailed changes can be found in the release notes: [https://github.com/spring-projects/sts4/releases/tag/4.28.0.RELEASE](https://github.com/spring-projects/sts4/releases/tag/4.28.0.RELEASE)

#### [](#downloads)downloads

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools: [https://spring.io/tools/](https://spring.io/tools/)

Spring Tools 4.29.0 is scheduled to be released in mid March 2025.

Enjoy!
