# Spring for Apache Pulsar 1.2.9 and 2.0.0-M2 are now available

_Releases | Chris Bono |  August 18, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.2.9` and `2.0.0-M2` have been released and are now available from Maven Central.

The `1.2.9` release will be included in the upcoming Spring Boot [3.4.9](https://github.com/spring-projects/spring-boot/milestones/3.4.9) and [3.5.5](https://github.com/spring-projects/spring-boot/milestones/3.5.5) releases. The `2.0.0-M2` release will be included in the upcoming Spring Boot [4.0.0-M2](https://github.com/spring-projects/spring-boot/milestones/4.0.0-M2) release.

Please see the release notes ([1.2.9](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.9) and [2.0.0-M2](https://github.com/spring-projects/spring-pulsar/releases/tag/v2.0.0-M2)) for more details.
