# Spring Security Kerberos 1.0.1 Released

_Releases | Rob Winch |  July 24, 2015 | 0 Comments_

On behalf of the Spring Security Kerberos team, I'm pleased to announce the release of Spring Security Kerberos 1.0.1. The highlights of the release are:

-   Support Spring Security 4
-   Support Spring LDAP 2.0
-   Fix SPNEGO auth fails if client proposes MS krb5 OID [#34](https://github.com/spring-projects/spring-security-kerberos/issues/34)
