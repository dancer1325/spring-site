# Spring Session 1.2.0 Released

_Releases | Rob Winch |  May 12, 2016 | 2 Comments_

On behalf of the community, I'm pleased to announce the release of Spring Session 1.2.0.RELEASE. This release evolved through [1.2.0 RC1](https://github.com/spring-projects/spring-session/issues?q=milestone%3A1.2.0.RC1), [1.2.0.RC2](https://github.com/spring-projects/spring-session/issues?utf8=%E2%9C%93&q=milestone%3A1.2.0.RC2), [1.2.0.RC3](https://github.com/spring-projects/spring-session/issues?utf8=%E2%9C%93&q=milestone%3A1.2.0.RC3), and [1.2.0.RELEASE](https://github.com/spring-projects/spring-session/issues?utf8=%E2%9C%93&q=milestone%3A1.2.0) closing over 60 issues.

# [](#whats-new-in-spring-session-120)What’s New in Spring Session 1.2.0

You can find highlights of what's new in the [What’s New in Spring Session 1.2.0](http://docs.spring.io/spring-session/docs/current/reference/html5/#what-s-new-in-1-2) section of the reference. For details refer to the changlog links above.

# [](#contributions)Contributions

Without the community we couldn't be the successful project we are today. I'd like to thank everyone that created issues & provided feedback.
