# Spring Data JPA 1.0 GA released

_Releases | Oliver Drotbohm |  July 21, 2011 | 0 Comments_

Dear Spring Community,

we are pleased to announce that the first final release of the Spring Data JPA project is now available! Thank you to all of the early adopters who have helped shaping and strengthening the codebase. To easily get started feel free to dig through the [sample code](https://github.com/SpringSource/spring-data-jpa-examples/tree/1.0.0.RELEASE) or read up the [reference documentation](http://static.springsource.org/spring-data/data-jpa/docs/1.0.0.RELEASE/reference/pdf/spring-data-jpa-reference.pdf).

</dependency>
