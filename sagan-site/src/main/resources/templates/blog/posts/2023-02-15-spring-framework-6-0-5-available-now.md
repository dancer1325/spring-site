# Spring Framework 6.0.5 available now

_Releases | Brian Clozel |  February 15, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.0.5` is available now.

Spring Framework `6.0.5` includes [109 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.5). This version will be shipped with Spring Boot 3.0.3, [to be released next week](https://calendar.spring.io/).

[Project Page](https://projects.spring.io/spring-framework/) | [GitHub](https://github.com/spring-projects/spring-framework) | [Issues](https://github.com/spring-projects/spring-framework/issues) | [Documentation](https://docs.spring.io/spring-framework/docs/6.0.5/reference/html/)
