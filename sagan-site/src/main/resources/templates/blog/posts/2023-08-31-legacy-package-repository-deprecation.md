description: On August 15, 2023, the Kubernetes project announced the general availability of the community-owned package repositories for Debian and RPM packages available at pkgs.k8s.io. The new package repositories are replacement for the legacy Google-hosted package repositories: apt.kubernetes.io and yum.kubernetes.io. The announcement blog post for pkgs.k8s.io highlighted that we will stop publishing packages to the legacy repositories in the future.
Today, we\'re formally deprecating the legacy package repositories (apt.kubernetes.io and yum.kubernetes.io), and we\'re announcing our plans to freeze the contents of the repositories as of September 13, 2023.
