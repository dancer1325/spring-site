# This Week in Spring - October 6th, 2015

_Engineering | Josh Long |  October 07, 2015 | 0 Comments_

Welcome to another installation of *This Week in Spring*! As usual we've got a lot to cover this week so let's get to it!

-   I love the [new Spring Initializr](http://start.spring.io)!
-   Spring Tool Suite [lead Martin Lippert just announced STS 3.7.1](http://spring.io/blog/2015/10/06/spring-tool-suite-3-7-1-released)
-   Spring framework lead Juergen Hoeller has put together a lovely post that speaks [to how to read claimed Java EE 7 proliferation](http://spring.io/blog/2015/10/06/a-brief-update-on-java-ee-7-adoption)
-   Jaime Lopez - who was in some of my talks at Spring I/O in Barcelona, Spain, earlier this year - [wrote an *amazing* introduction to Spring Boot](http://blog.mimacom.com/introduction-to-spring-boot/) - check it out!
-   This blog nicely introduces using Spring's `AbstractFeedView`, powered by the ROME RSS and Atom library, [to build feeds for a web application](http://kielczewski.eu/2015/10/adding-rss-to-spring-boot-application-with-rome/)
-   Want to run Spring Boot [on the Glassfish-powered Payara Micro](https://www.voxxed.com/blog/2015/10/creating-rest-web-services-with-spring-boot-hosted-on-payara-micro/)?
-   Hey, the *ProgrammingFree* YouTube video channel has a nice post [on using Spring Security and Spring to build a simple application](https://www.youtube.com/watch?v=XO-OdIXNU_o&feature=youtu.be)
-   the [4cats.io YouTube channel](https://www.youtube.com/watch?v=iJjXQJ4IXpI) has all manner of cool Chinese-language videos on Spring Boot!
-   David Parish put together a nice [post on running Spring Boot applications on Websphere](https://bp3.zendesk.com/hc/en-us/articles/210443048)
-   Alberto Souza put together a nice Portguese-language [post on using Spring Integration](https://domineospring.wordpress.com/2015/10/02/primeiros-passos-com-o-spring-integration/)
