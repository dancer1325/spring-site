# Spring Integration 2.2.0 GA has been released

_Releases | Mark Fisher |  December 03, 2012 | 0 Comments_

We are pleased to announce that [Spring Integration](http://www.springsource.org/spring-integration) 2.2 is now GA.

The artifacts are available in the [SpringSource Repository](http://repo.springsource.org) as well as Maven Central, or you can download the zip [here](http://www.springsource.org/download/community?project=Spring%20Integration).

For a big picture view, please read the ["what's new"](http://static.springsource.org/spring-integration/docs/2.2.0.RELEASE/reference/htmlsingle/#whats-new) section of the reference manual.

For more detail on some of the main additions, refer to the following blog series:

Part 1: [MongoDB Adapters](http://blog.springsource.org/2012/09/24/whats-new-in-spring-integration-2-2-rc1-part-1-mongodb/) Part 2: [Transaction Synchronization](http://blog.springsource.org/2012/09/26/whats-new-in-spring-integration-2-2-part-2-transaction-synchronization/) Part 3: [JPA Support](http://blog.springsource.org/2012/10/05/whats-new-in-spring-integration-2-2-part-3-jpa-support/) Part 4: [Retry and More](https://spring.io/blog/2012/10/09/what-s-new-in-spring-integration-2-2-part-4-retry-and-more)
