# This Week in Spring - April 4th, 2017

_Engineering | Josh Long |  April 04, 2017 | 2 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in London, UK; Paris, France (for the [epic Devoxx France](http://cfp.devoxx.fr/2017/speaker/josh_long) event where I'll be speaking with [my buddy Matt Raible](http://twitter.com/mraible)) and Stuttgart, Germany, where I'll be speaking at conferences and to customers! If you're around, don't hesitate to say ["hi"!](http://twitter.com/starbuxman)

-   the new release of Spring Tool Suite is jam-packed with new features supporting [Spring Boot and Cloud Foundry](https://spring.io/blog/2017/04/03/spring-tool-suite-3-8-4-released). [Check the New and Noteworthy](http://docs.spring.io/sts/nan/v384/NewAndNoteworthy.html) and get it while it's hot! (I've just gotten a new laptop, so I'm doing that right now, too!)
-   last week in the latest installment of *Spring Tips* I looked at some the various [utility classes in the Spring Framework](https://spring.io/blog/2017/03/29/spring-tips-the-spring-framework-utils-classes)
-   I just presented at the premier Voxxed Days CERN event and [the organizers interviewed the amazing Spring Framework lead Juergen Hoeller, and me (his biggest fan)](https://www.youtube.com/watch?v=L61VkzdpiF4)
-   don't miss the upcoming webinar on April 13th, [Spring Boot Under the Hood](https://spring.io/blog/2017/03/28/webinar-april-13th-spring-boot-under-the-hood)
-   Check out the Spring Team and Pivotal at [OSCON Austin 2017](https://spring.io/blog/2017/04/03/spring-team-and-pivotal-at-oscon-austin-2017)!
-   I really enjoyed Matt Raible's new posts on the Okta blog introducing how to use Spring Boot and Okta [with SAML](http://developer.okta.com/blog/2017/03/16/spring-boot-saml) and [with OAuth 2.0](http://developer.okta.com/blog/2017/03/21/spring-boot-oauth)
-   My friend from Alibaba, Jacky Chan, put together a [fantastic sample application demonstrating Spring Boot and Angular4](https://github.com/linux-china/spring-boot-thymeleaf3-bootstrap4-angular4)
-   version 1.6.0 of the project [Find Security Bugs](http://find-sec-bugs.github.io), the FindBugs plugin for security audits of Java web applications, was recently released. This new version adds some Spring specific checks, like [disabling of CSRF protection](http://find-sec-bugs.github.io/bugs.htm#SPRING_CSRF_PROTECTION_DISABLED), unrestricted use of [RequestMapping causing CSRF vulnerabilities](http://find-sec-bugs.github.io/bugs.htm#SPRING_CSRF_UNRESTRICTED_REQUEST_MAPPING) and [dangerous server-side redirect using user input](http://find-sec-bugs.github.io/bugs.htm#SPRING_FILE_DISCLOSURE). Thanks to Pablo Tamarit for bringing this to my attention - very cool!
-   our friend Nicolas Frankel has an interesting post on where [to put common code in a Spring MVC application](https://blog.frankel.ch/common-code-spring-mvc/#gsc.tab=0)
-   Yes, April Fools is past, but I just wanted to share [this epic creation from 2016 Spring Test lead Sam Brannen, Spring Polyglot, demonstrating composed applications for different (human) languages](https://github.com/sbrannen/spring-polyglot)
-   This isn't specifically to do with Spring, though it is something you could implement with Spring Cloud Zuul or the new Spring Cloud Gateway technology, but this post [by Stripe on the nuances of load-balancers and shedders was quite informative](https://stripe.com/blog/rate-limiters)
-   I quite liked this [post introducing some Kotlin-language best practices](https://blog.philipphauer.de/idiomatic-kotlin-best-practices/)
-   Spring Cloud ninja Dave Turanski put together a great post introducing [Spring Cloud Stream for event driven architectures](https://dturanski.wordpress.com/2017/03/26/spring-cloud-stream-for-event-driven-architectures/)
