# Spring Boot 1.1.10 released

_Releases | Phil Webb |  December 11, 2014 | 0 Comments_

Spring Boot 1.1.10 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a small [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.1.9) and enhancements and is recommended for anyone that doesn't yet want to upgrade to the [new 1.2.0 release](http://spring.io/blog/2014/12/11/spring-boot-1-2-0-released).
