# Spring Security 5.1.1, 5.0.9, and 4.2.9 Released

_Releases | Rob Winch |  October 16, 2018 | 0 Comments_

On behalf of the community, I am pleased to announce that Spring Security 5.1.1 ([changelog](https://github.com/spring-projects/spring-security/milestone/124?closed=1)) Spring Security 5.0.9 ([changelog](https://github.com/spring-projects/spring-security/milestone/122?closed=1)) and 4.2.9 ([changelog](https://github.com/spring-projects/spring-security/milestone/123?closed=1)) have been released. The releases primarily deliver bug fixes and dependency version updates along with some minor improvements. The releases will be found in the upcoming Spring Boot maintenance releases coming this week.
