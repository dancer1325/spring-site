# Spring Framework 6.1.17 and 6.2.3 Available Now

_Releases | Stéphane Nicoll |  February 13, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.17` and `6.2.3` are available now.

Spring Framework `6.1.17` ships with [17 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.17). This version will be shipped next week with Spring Boot `3.3.9`.

Spring Framework `6.2.3` ships with [60 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.3). This version will be shipped next week with Spring Boot `3.4.3` and `3.5.0-M2`.
