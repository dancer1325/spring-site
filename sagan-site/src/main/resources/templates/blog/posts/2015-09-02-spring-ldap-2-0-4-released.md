# Spring LDAP 2.0.4 Released

_Releases | Rob Winch |  September 02, 2015 | 0 Comments_

I’m pleased to announce the release of Spring LDAP 2.0.4.RELEASE. The highlights of this release include:

[LDAP-333](https://jira.spring.io/browse/LDAP-333) - Support for Spring Data Commons 1.11 (Spring Data Gosling) [LDAP-334](https://jira.spring.io/browse/LDAP-334) \- Compatibility with Spring IO Platform 2.0 [LDAP-335](https://jira.spring.io/browse/LDAP-335) - Fix NoClassDefFoundError: RepositoryConfigurationExtension

For additional information on the release, refer to the [changelog](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10071&version=15001).
