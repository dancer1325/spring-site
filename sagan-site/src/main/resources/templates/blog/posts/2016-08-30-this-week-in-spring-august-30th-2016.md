# This Week in Spring - August 30th, 2016

_Engineering | Josh Long |  August 30, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I've been in San Francisco, (where I live and) where I addressed the Silicon Valley Spring User Group. Now it's off to beautiful China to bring some Spring and Pivotal (and, maybe, take a little vacation!)

As usual, we have a *lot* to get to so let's!

-   Spring ninja Greg Turnquist has just announced [that Spring Web Services 2.3.1 and 2.4.0 are now available](https://spring.io/blog/2016/08/29/spring-web-services-2-3-1-2-4-0-are-released)
-   Spring Cloud ninja Marcin Grzejszczak just announced [that Spring Cloud Camden M1 is now available](https://spring.io/blog/2016/08/29/spring-cloud-camden-m1-is-available)
-   Spring Cloud Data Flow ninja Thomas Risberg just announced [that Spring Cloud Data Flow for Mesos 1.0.RC2 is now available](https://spring.io/blog/2016/08/26/spring-cloud-data-flow-for-mesos-1-0-rc2-released)
-   Marius Bogoevici just announced Spring Cloud Stream Brooklyn M1 is now available. This release is [packed full of new features including support for reactive programming](https://spring.io/blog/2016/08/26/spring-cloud-stream-brooklyn-m1-is-available)
-   Spring Cloud Data Flow ninja Eric Bottard just announced [that Spring Cloud Data Flow for Cloud Foundry 1.0. has gone GA](https://spring.io/blog/2016/08/25/spring-cloud-data-flow-for-cloud-foundry-goes-1-0-ga)
-   [if you're using Spring Security SAML, you should check out this security update](https://spring.io/blog/2016/08/24/check-your-spring-security-saml-config-xxe-security-issue)
-   Spring Security lead Rob Winch [just announced Spring Security 4.1.3, packed full of fixes](https://spring.io/blog/2016/08/24/spring-security-4-1-3-released)
-   Recent Spring Cloud team addition Ryan Baxter [just announced Spring Cloud Brixton SR5](https://spring.io/blog/2016/08/23/spring-cloud-brixton-sr5-is-available)
-   Spring Integration lead Gary Russell just announced that [Spring for Apache Kafka 1.1.0 M1 is now available.](https://spring.io/blog/2016/08/23/spring-for-apache-kafka-1-1-0-milestone-1-available)
-   I put on a 4 hour long workshop in Singapore introducing [Spring Boot, Spring Cloud, Spring Coud Data Flow, Spring Cloud Security and Spring Security OAuth. The whole thing was recorded and is available online here](https://www.youtube.com/watch?v=JUqZxUlixSw)
-   on August 14th, Reno Chou came to one of my *Cloud Native Java* talks in China and started writing up some of the information discussed in a series of blogs. Here are his [first](http://blog.itmuch.com/article/13) and [second](http://blog.itmuch.com/article/15) Chinese-language installments. Very cool Reno!
-   I Antoine's French language post detailing how he migrated [the Spring Petclinic application from Spring 4 to Spring Boot](http://javaetmoi.com/2016/08/migrer-vers-spring-boot/)
-   Cool! Vojtech Ruzicka put together a nice review of his [experience with the Pivotal Certified Spring Web Application Developer exam](http://vojtechruzicka.com/pivotal-certified-spring-web-application-developer-notes/)
-   I liked Tim Spann's post on [using Spring Boot to access data from Apache Hive](https://community.hortonworks.com/articles/53629/writing-a-spring-boot-microservices-to-access-hive.html)
