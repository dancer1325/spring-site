# Spring 3.0.1 is Now Available

_Releases | Adam Fitzgerald |  February 18, 2010 | 0 Comments_

[Arjen Poutsma](http://blog.springsource.com/author/arjenp/) has just [announced](http://twitter.com/poutsma) that Spring 3.0.1 is now available and [Juergen Hoeller](http://blog.springsource.com/author/juergenh/) has [blogged](http://blog.springsource.com/2010/02/18/spring-framework-3-0-1-released/) about the changes in this minor release.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.1.RELEASE) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.x/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.1.RELEASE/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

Don't forget that Spring users can ask questions in the [community forum](http://forum.springsource.org/forumdisplay.php?f=21) and identify issues in [JIRA](http://jira.springframework.org/browse/SPR) as well. Please keep up the good work so that we can continue to make all the Spring projects better and better. European developers can learn more about Spring 3.0 at the one day [S2G Forum](http://www.springsource.com/events/s2gforum-3-18-2010) in Munich on March 18th, or at the other S2G Forums in London and Amsterdam (dates to be announced soon).
