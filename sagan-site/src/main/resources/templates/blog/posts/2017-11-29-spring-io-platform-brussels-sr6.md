# Spring IO Platform Brussels-SR6

_Releases | Andy Wilkinson |  November 29, 2017 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR6 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   AspectJ 1.8.13
-   Spring Boot 1.5.9
-   Spring Cloud Connectors 1.2.5
-   Spring Data Ingalls-SR9
-   Spring Framework 4.3.13
-   Spring Kafka 1.1.7
-   Spring LDAP 2.3.2
-   Spring Web Services 2.4.2

The versions of a number of third-party dependencies have also been updated.
