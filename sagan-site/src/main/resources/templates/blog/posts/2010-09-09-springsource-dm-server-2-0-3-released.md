# SpringSource dm Server 2.0.3 Released

_Engineering | Glyn Normington |  September 09, 2010 | 0 Comments_

SpringSource dm Server 2.0.3 is available for [download](http://www.springsource.org/dmserver). This purpose of this release is to fix a severe problem on Windows when the root logging level is configured to "debug". This [bug](https://bugs.eclipse.org/bugs/show_bug.cgi?id=323609) was reported and fixed in Virgo and then backported for existing customers since Virgo has not yet shipped its baseline release (planned for 4Q10).
