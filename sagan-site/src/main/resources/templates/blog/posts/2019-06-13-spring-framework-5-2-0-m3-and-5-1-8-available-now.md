# Spring Framework 5.2.0.M3 and 5.1.8 available now

_Releases | Stéphane Nicoll |  June 13, 2019 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.2.0.M3` and `5.1.8` are available now.

The third milestone of Spring Framework 5.2 includes [50 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.0.M3) while Spring Framework `5.1.8` includes [31 fixes and selected improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.1.8.RELEASE).

If you want to give the milestone a try, you can boostrap a new application on [start.spring.io](https://start.spring.io) once Spring Boot `2.2.0.M4` is released early next week.
