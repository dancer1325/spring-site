# Spring Framework 5.3.4 and 5.2.13 available now

_Releases | Stéphane Nicoll |  February 16, 2021 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.4` and `5.2.13` are available now.

Spring Framework `5.3.4` includes [62 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.4). Spring Framework `5.2.13` includes [18 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.13.RELEASE).

Stay tuned for follow-up Spring Boot releases later this week.
