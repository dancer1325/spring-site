# Spring Session 1.3.0 M2 Released

_Releases | Rob Winch |  September 14, 2016 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Session 1.3.0.M2. This release release closes lots of community submitted Pull Requests. For a complete list of changes see the [changelog](https://github.com/spring-projects/spring-session/milestone/6?closed=1).

Note

Spring Session 1.3.0.M1 release had some problems, so we followed up with an immediate release of Spring Session 1.3.0.M2

# [](#whats-new-in-spring-session-130-m2)[](#what-s-new-in-spring-session-1-3-0-m2)What’s New in Spring Session 1.3.0 M2

Highlights include:

-   First Class Support for [Hazelcast](http://docs.spring.io/spring-session/docs/1.3.0.M2/reference/html5/#httpsession-hazelcast)
    
-   First class [Spring Security’s concurrent session management](http://docs.spring.io/spring-session/docs/1.3.0.M3/reference/html5/#spring-security-concurrent-sessions-how)
    
-   [Redis Sample with GenericJackson2JsonRedisSerializer](https://github.com/spring-projects/spring-session/tree/1.3.0.M2/samples/httpsession-redis-json)
    
-   `spring.session.cleanup.cron.expression` can be used to override the cleanup task’s cron expression
    
-   Lots of performance improvements and bug fixes
