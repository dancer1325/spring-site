# This week in Spring: January 18th, 2011

_Engineering | Josh Long |  January 18, 2011 | 0 Comments_

Another fantastic week for the Spring community at large. In addition to the exciting announcement of the stable release of Tomcat 7, there were numerous great posts from the community.

1.  Continuing the back-to-basics "Green Beans" series, we've just published a post on using [Spring and Maven together.](http://blog.springsource.com/2011/01/17/green-beans-getting-started-with-maven-and-spring/)
2.  The Tomcat Expert community has announced [the first stable build of Tomcat 7.](http://www.tomcatexpert.com/blog/2011/01/14/apache-tomcat-706-released-first-stable-build) We know Tomcat is the favorite place for Spring developers to run their applications, so be sure to check out the latest features.
3.  Nicholas Sankel has written up a blog on using [Spring Roo with the Vaadin UI framework.](http://java.dzone.com/articles/playing-spring-roo-and-vaadin)
4.  The Tomcat Expert community has provided [a list of the top 10 articles from 2010.](http://www.tomcatexpert.com/blog/2010/12/30/tomcat-experts-top-10-2010)
5.  Tomasz Nurkiewicz has written a fantastic [blog exploring how you can completely strip out XML](http://nurkiewicz.blogspot.com/2011/01/spring-framework-without-xml-at-all.html) using the Spring framework.
6.  Sivaprasadreddy Katamreddy [has written up an introduction to Spring AOP](http://java.dzone.com/articles/aspect-oriented-programming-0?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed:+javalobby/frontpage+\(Javalobby+/+Java+Zone\)) that progresses from problematic code all the way to the improved code using Spring AOP.
7.  A gentleman named (from what I can tell) Warlock has blogged [on Spring MVC's annotation-based session-handling support](http://vard-lokkur.blogspot.com/).
8.  Sal Peace has written up a fantastic introduction to using [Spring MVC 3.0's portlet support with WebSphere 7.](http://www.theserverside.com/tutorial/WebSphere-Portal-7-and-Spring-30-Portlet-MVC-Integration)
9.  Viral Patel has written up a tutorial on [saving and retrieving BLOB object data in Spring 3 MVC and Hibernate](http://viralpatel.net/blogs/2011/01/tutorial-save-get-blob-object-spring-3-mvc-hibernate.html)
10.  Jorram Barrez (from the [Activiti](http://www.activiti.org) project) has posted on using Spring to connect [an Activiti BPMN 2.0 business process to EJB business services.](http://www.jorambarrez.be/blog/2011/01/13/tutorial-call-an-ejb-in-a-bpmn-2-0-process-using-activiti-and-spring/)
