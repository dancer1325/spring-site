# Spring Data Kay SR4 released

_Releases | Mark Paluch |  February 20, 2018 | 0 Comments_

On behalf of the Spring Data team, I'm pleased to announce the fourth service release of the Kay release train in prospect of Spring Boot 2.0 RC2.

This service release ships on top of the just-released Spring Framework 5.0.4 and Project Reactor 3.1.4, and provides [the answer to persistence the code and everything](https://jira.spring.io/issues/?filter=15836). Kay SR4 will be picked up by Spring Boot 2.0 RC2 for your convenience and is a recommended upgrade to users of the Kay release train.
