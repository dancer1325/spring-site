# Spring Cloud Camden.SR4 Is Available

_Releases | Ryan Baxter |  January 12, 2017 | 0 Comments_

On behalf of the team, I am pleased to announce that Camden Service Release 4 of [Spring Cloud](http://cloud.spring.io) is available today. The release can be found in our [Spring Release repository](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Ca%3A%22spring-cloud-dependencies%22).

This release includes primarily bug fixes. Release notes can be found on our [wiki](https://github.com/spring-projects/spring-cloud/wiki).

#### [](#the-following-modules-were-updated-as-part-of-camdensr4)The following modules were updated as part of Camden.SR4:

-   Spring Cloud Commons [1.1.7.RELEASE](https://github.com/spring-cloud/spring-cloud-commons/milestone/21?closed=1)
-   Spring Cloud Netflix [1.2.4.RELEASE](https://github.com/spring-cloud/spring-cloud-netflix/milestone/35?closed=1)
-   Spring Cloud Sleuth [1.1.1.RELEASE](https://github.com/spring-cloud/spring-cloud-sleuth/milestone/20?closed=1)
-   Spring Cloud Contract [1.0.3.RELEASE](https://github.com/spring-cloud/spring-cloud-contract/milestone/7?closed=1)

And, as always, we welcome feedback: either on [GitHub](https://github.com/spring-cloud), on [Gitter](https://gitter.im/spring-cloud/spring-cloud), on [Stack Overflow](http://stackoverflow.com/questions/tagged/spring-cloud), or on [Twitter](https://twitter.com/SpringCloud).
