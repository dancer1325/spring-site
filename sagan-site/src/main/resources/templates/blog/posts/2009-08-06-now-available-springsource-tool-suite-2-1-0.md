# Now Available: SpringSource Tool Suite 2.1.0

_Releases | Adam Fitzgerald |  August 06, 2009 | 0 Comments_

[SpringSource Tool Suite](http://www.springsource.com/products/sts) 2.1.0 is now GA and ready for [download](http://www.springsource.com/products/springsource-tool-suite-download). [Christian Dupuis](http://blog.springsource.com/author/cdupuis/)' [blog post](http://blog.springsource.com/2009/08/06/springsource-tool-suite-210-now-available/) covers all the new features included in the best development tool for building Spring powered applications:

-   Eclipse 3.5 (Galileo) support and platform specific installers for Windows/Mac OS X/Linux
-   Support for milestones of Spring 3.0 including XML editing and validation, support for @Configuration and @Bean annotations
-   Support for milestones of [SpringSource dm Server](/dmserver) 2.0, RFC66 web modules and [SpringSource Bundlor](/bundlor) integration
-   Integration of [Spring Roo](/roo) for rapid application development
-   Optional integration of the new alpha [Groovy Eclipse Plugin](http://blog.springsource.com/2009/07/30/a-groovier-eclipse-experience/)
