# Spring Framework 5.2.8, 5.1.17, 5.0.18, and 4.3.28 available now

_Releases | Brian Clozel |  July 21, 2020 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce a full round of Spring Framework releases.

Spring Framework 5.2.8 includes [36 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.8.RELEASE). Spring Framework 5.1.17 includes [15 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.1.17.RELEASE).

The maintenance release for `5.0.x` (`5.0.18`) ships with [13 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.0.18.RELEASE). The `4.3.x` branch (`4.3.28`) also ships with [13 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v4.3.28.RELEASE), including a CORS configuration change (see [gh-25414](https://github.com/spring-projects/spring-framework/issues/25414) and the [CORS section of the reference documentation](https://docs.spring.io/spring/docs/4.3.x/spring-framework-reference/html/cors.html#_introduction) for more details).

As usual, we'll follow up shortly with corresponding Spring Boot releases (`2.3.2`, `2.2.9` and `2.1.16`).
