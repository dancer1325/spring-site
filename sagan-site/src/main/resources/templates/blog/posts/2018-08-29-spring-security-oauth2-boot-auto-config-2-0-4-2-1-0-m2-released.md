# Spring Security OAuth2 Boot Auto-config 2.0.4 & 2.1.0.M2 Released

_Releases | Josh Cummings |  August 29, 2018 | 0 Comments_

I’m pleased to announce on behalf of the community [Spring Security OAuth2 Boot Auto-config](https://github.com/spring-projects/spring-security-oauth2-boot) 2.0.4 and 2.1.0.M2.

Both releases primarily deliver bug fixes and dependency version updates along with some minor improvements. Of particular note is that these align with [recent](https://github.com/spring-projects/spring-boot/releases/tag/v2.1.0.M2) [releases](https://github.com/spring-projects/spring-boot/releases/tag/v2.0.4.RELEASE) of Spring Boot.

For a complete list of changes, please refer to the 2.0.4 [changelog](https://github.com/spring-projects/spring-security-oauth2-boot/milestone/7?closed=1) and 2.1.0.M2 [changelog](https://github.com/spring-projects/spring-security-oauth2-boot/milestone/9?closed=1).
