# Spring Security 4.2.14 Released

_Releases | Rob Winch |  February 06, 2020 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 4.2.14 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.14.RELEASE)). These releases deliver bug fixes, dependency updates, and improvements with our build. Users are encouraged to update to the latest patch release.
