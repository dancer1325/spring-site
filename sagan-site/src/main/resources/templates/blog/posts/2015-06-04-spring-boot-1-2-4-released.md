# Spring Boot 1.2.4 released

_Releases | Stéphane Nicoll |  June 04, 2015 | 3 Comments_

Spring Boot 1.2.4 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [small number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.4) and is a recommended upgrade for all users.

The first milestone of Spring Boot 1.3 will be released in the coming days and adds [many exciting features](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.0.M1). Stay tuned!
