# Spring Shell 1.0 M1 released

_Releases | Costin Leau |  July 18, 2012 | 0 Comments_

Dear Spring Community,

I am pleased to announce the first milestone release of Spring Shell project! Spring Shell is an interactive shell that can be easily extended with commands using a Spring based programming model, extracted from the Spring Roo project.

[Downloads](http://www.springsource.org/download/community?project=Spring Shell) | [JavaDocs](http://static.springsource.org/spring-shell/docs/1.0.0.M1/api/) | [Reference Documentation](http://static.springsource.org/spring-shell/docs/1.0.0.M1/reference/html) | [Changelog](http://static.springsource.org/spring-shell/docs/1.0.0.M1/changelog.txt)

For more information about Spring Shell please see the [home page](http://www.springsource.org/spring-shell).

We look forward to your feedback on the [forum](http://forum.springsource.org/forum.php) or in the [issue tracker](https://jira.springsource.org/browse/SHL).
