# This Week in Spring - May 3rd, 2016

_Engineering | Josh Long |  May 04, 2016 | 0 Comments_

I can't even believe it's May already! STOP THE WORLD I WANT OFF! Well, at least the times are exciting! There's so much cool stuff to look at this week so let's get to it!

-   New sessions are up on the shiny new [SpringOne Platform website](https://springoneplatform.io/sessions) check it out! In addition to an Adrian Cockcroft keynote, we're got speakers booked from Netflix, Google, Comcast, Express Scripts, Kroger, along with Spring leaders like Juergen Hoeller, Dr. Dave Syer, Rossen Stoyanchev, Oliver Gierke, and of course, yours truly!
-   and join Spring Cloud Data Flow leads Mark Fisher and Mark Pollack for a look at [Data Driven Microservices with Spring Cloud Data Flow](https://spring.io/blog/2016/04/27/webinar-data-microservices-with-spring-cloud-data-flow)
-   Spring Session and Spring Security lead Rob Winch [has just announced Spring Session 1.2.0.RC3](https://spring.io/blog/2016/04/28/spring-session-1-2-0-rc3-released).
-   Spring REST Docs lead [Andy Wilkinson just announced Spring REST Docs](https://spring.io/blog/2016/04/28/spring-rest-docs-1-1-0-rc1)
-   Spring Cloud Data Flow co-lead [Mark Fisher just announced Spring Cloud Data Flow 1.0.0.M3](https://spring.io/blog/2016/04/29/spring-cloud-data-flow-1-0-0-m3-released)
-   Janne Valkealahti [just announced Spring Cloud Data Flow for Apache YARN](https://spring.io/blog/2016/04/29/spring-cloud-data-flow-for-apache-yarn-1-0-0-m2-released)
-   Spring and Data wizard Thomas Risberg [just announced Spring Cloud Data Flow for Apache Mesos and Kubernetes 1.0.0.M2](https://spring.io/blog/2016/04/29/spring-cloud-data-flow-for-apache-mesos-and-kubernetes-1-0-0-m2-versions-released)
-   Spring Cloud Data Flow ninja Eric Bottard just announced [Spring Cloud Data Flow for Cloud Foundry 1.0.0.M2](https://spring.io/blog/2016/04/29/spring-cloud-data-flow-for-cloud-foundry-1-0-0-m2-released)
-   Spring Data lead Christoph Strobl put together a great writeup of [what's new in Spring Data Hopper](https://spring.io/blog/2016/05/03/what-s-new-in-spring-data-hopper)
-   Spring Data ninja John Blum just wrote up [a nice post on what's new in Spring Data Geode](https://spring.io/blog/2016/04/29/spring-data-geode-1-0-0-apache-geode-incubating-m2-released)
-   our pal Michael Simmons just put together a [great writeup of how Spring Boot's auto-configuration works](http://info.michael-simons.eu/2016/05/02/spring-boot-starter-comfortable-modularization-and-abstraction/)
-   Have you been [following the latest and greatest in Cloud Foundry](https://www.cloudfoundry.org/community/planet/)?
-   Dan Vega put together a nice video [introducing using Project Lombok with Spring Boot](http://therealdanvega.com/blog/2016/04/27/using-project-lombok-spring-boot)
-   Anirudh Bhatnagar put together a nice post [on using Spring Boot, Thymeleaf and Angular.js together](https://www.javacodegeeks.com/2016/04/build-new-web-application-scratch-using-spring-boot-thymeleaf-angularjs-part-1.html)
-   I loved this post on [using CXF to develop JAX-RS services on top of Spring Boot](https://www.javacodegeeks.com/2016/05/laziness-extreme-developing-jax-rs-services-spring-boot.html). (It would've been even easier if they'd just used Jersey!)
-   Want to get a local Cloud Foundry environment? [Meet PCF Dev: Your Ticket To Running Cloud Foundry Locally](https://blog.pivotal.io/pivotal-cloud-foundry/products/meet-pcf-dev-your-ticket-to-running-cloud-foundry-locally)
