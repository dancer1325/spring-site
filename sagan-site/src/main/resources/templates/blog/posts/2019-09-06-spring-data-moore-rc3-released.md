# Spring Data Moore RC3 released

_Releases | Mark Paluch |  September 06, 2019 | 0 Comments_

On behalf of the team I am pleased to announce Spring Data Moore RC3. The release ships with [84 tickets fixed](https://jira.spring.io/issues/?filter=16439) and it will be picked up by Spring Boot 2.2 M6 for your convenience.

This release candidate finalizes our effort around Kotlin Coroutines which went GA with their 1.3 release. Besides that, the release ships with dependency upgrades, API refinements, bugfixes, and improvements to our documentation.

This is the last release candidate release before shipping Moore GA in late September. As [SpringOne Platform](https://springoneplatform.io/) happens in about a month, we're excited to meet with the community in Austin, Texas October 7-10, 2019.
