# Spring Framework 5.0.8 available now

_Releases | Stéphane Nicoll |  July 26, 2018 | 1 Comment_

On behalf of the team, I am pleased to announce that Spring Framework 5.0.8 [is available now](http://projects.spring.io/spring-framework).

Spring Framework 5.0.8 [includes 40 fixes and selected improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16855) and is the base for the upcoming Spring Boot 2.0.4 release next week.
