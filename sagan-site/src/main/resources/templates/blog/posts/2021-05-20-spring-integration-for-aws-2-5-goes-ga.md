# Spring Integration for AWS 2.5 goes GA

_Releases | Artem Bilan |  May 20, 2021 | 0 Comments_

Dear Spring (and AWS) Community,

Today it’s my pleasure to announce a release of Spring Integration for Amazon Web Services extension version `2.5`.

It is available from Maven Central:

```
Copycompile "org.springframework.integration:spring-integration-aws:2.5.0"
```

Mostly this version is a major move from Spring Cloud for AWS project hosted on Spring Cloud side to its own [organization](https://awspring.io/). Plus it includes upgrades to the latest dependencies, including recently released Spring Integration [5.5](https://spring.io/blog/2021/05/19/spring-integration-5-5-goes-ga).

Many thanks to everyone from the Community for all the feedback and contribution to this project.
