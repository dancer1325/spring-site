# Spring Security 5.4.2, 5.3.6, and 5.2.8 released

_Releases | Josh Cummings |  December 03, 2020 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.4.2 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.4.2)), 5.3.6 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.6.RELEASE)), and 5.2.8 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.8.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
