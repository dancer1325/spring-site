_Engineering | Josh Long |  May 16, 2018 | 3 Comments_

speaker: [Josh Long](http://twitter.com/Starbuxman)

Hi Spring fans! In this installment of Spring Tips we look at Spring Cloud Function (which lets you write serverless functions that run on your favorite function-as-a-service provider) and [Project Riff](http://projectRiff.io), the new open-source function-as-a-service platform from Pivotal.
