# Spring Security 5.2.10, 5.3.9, and 5.4.6 released

_Releases | Josh Cummings |  April 12, 2021 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.2.10 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.10.RELEASE)), Spring Security 5.3.9 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.9.RELEASE)), and Spring Security 5.4.6 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.4.6)). These releases delivers bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
