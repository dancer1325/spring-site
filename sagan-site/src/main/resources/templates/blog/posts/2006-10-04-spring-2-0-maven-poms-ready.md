# Spring 2.0 Maven POMs ready

_Engineering | Ben Hale |  October 04, 2006 | 0 Comments_

One last Spring 2.0 announcement on a day full of them. The Maven POMs for Spring 2.0 are up in Spring's private repository. If you want to point to it directly check [](https://svn.sourceforge.net/svnroot/springframework/repos/repo/)[https://svn.sourceforge.net/svnroot/springframework/repos/repo/](https://svn.sourceforge.net/svnroot/springframework/repos/repo/). If you want to wait, they should be replicated into the Ibiblio Maven repository over the next couple of days.

For those of you who like to browse around with a bit more metadata, the [ViewVC interface](http://svn.sourceforge.net/viewvc/springframework/repos/repo/) from SourceForge is a good choice. Remember that you should use the earlier link as a URL for maven, but you can use the later link for browsing.
