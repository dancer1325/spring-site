# Video: SpringOne 2GX - Spring MVC 3.1 Update

_News | Adam Fitzgerald |  January 04, 2012 | 0 Comments_

[![SpringOne 2GX Video: Spring MVC 3.1 Update](http://www.springsource.org/files/SpringOne2GX2011SpringMVC3_1Update.png)](http://www.infoq.com/presentations/Spring-MVC-3-1-Update)

Here is more great content from SpringOne 2GX 2011, this time a complete [update to Spring MVC 3.1](http://www.infoq.com/presentations/Spring-MVC-3-1-Update) covering URI variables, Redirect & Flash attributes, UriComponentsBuilder, Multipart Request Support, and HDIV Integration. This video presentation is by [Rossen Stoyanchev](http://blog.springsource.org/author/rstoyanchev/), one of the Spring experts who commits to Spring MVC and Spring Web Flow.

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
