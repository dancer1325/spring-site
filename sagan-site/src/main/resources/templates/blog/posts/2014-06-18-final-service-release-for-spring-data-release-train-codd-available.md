# Final Service Release For Spring Data Release Train Codd Available

_Releases | Oliver Drotbohm |  June 18, 2014 | 0 Comments_

I'd like to announce the availability of the third and final service release for the Spring Data release train named Codd. The release contains a lot of bug fixes that made it into the RC and GA releases of the Dijkstra release train already.

-   Spring Data Commons 1.7.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-commons/1.7.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/commons/docs/1.7.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/commons/docs/1.7.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/commons/docs/1.7.3.RELEASE/changelog.txt)
-   Spring Data JPA 1.5.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-jpa/1.5.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/jpa/docs/1.5.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/jpa/docs/1.5.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/jpa/docs/1.5.3.RELEASE/changelog.txt)
-   Spring Data MongoDB 1.4.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-mongodb/1.4.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/mongodb/docs/1.4.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/mongodb/docs/1.4.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/mongodb/docs/1.4.3.RELEASE/changelog.txt)
-   Spring Data Neo4j 3.0.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-neo4j/3.0.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/neo4j/docs/3.0.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/neo4j/docs/3.0.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/neo4j/docs/3.0.3.RELEASE/changelog.txt)
-   Spring Data Solr 1.1.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-solr/1.1.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/solr/docs/1.1.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/solr/docs/1.1.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/solr/docs/1.1.3.RELEASE/changelog.txt)
-   Spring Data REST 2.0.3 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-rest-webmvc/2.0.3.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/rest/docs/2.0.3.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/rest/docs/2.0.3.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/rest/docs/2.0.3.RELEASE/changelog.txt)

A complete list of changes can be found [here](https://jira.spring.io/issues/?filter=14739). As this is a bugfix release we recommend to upgrade. If you can upgrade to the current [Spring Data Dijkstra GA release](https://spring.io/blog/2014/05/20/spring-data-release-train-dijkstra-goes-ga), even better. SR3 is the last release of the Codd release train. We will proceed with the first service release for Dijkstra followed by a first milestone for Evans.

If you want to learn more about Spring Data and all the new stuff we're going to ship with the Evans release train, make sure to check out the [schedule](https://2014.event.springone2gx.com/schedule/2014-09-09) and [register today](https://2014.event.springone2gx.com/register).
