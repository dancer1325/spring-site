# Spring Boot 1.3.1 and 1.2.8 available now

_Releases | Stéphane Nicoll |  December 18, 2015 | 0 Comments_

Spring Boot 1.2.8 and 1.3.1 have been released and are available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

If you haven't tried Spring Boot 1.3 yet, now may be the time to do so: 1.3.1 is a maintenance release that brings [126 fixes and improvements](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.1+is%3Aclosed), almost half of those being pull requests. Thanks again to our amazing contributors!

Spring Boot 1.2.8 is a maintenance release with [a selection of fixes and improvements](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.8+is%3Aclosed).

As we have fixed a [security vulnerability](https://github.com/spring-projects/spring-boot/issues/4763), we highly recommend an immediate upgrade for all Spring Boot users.
