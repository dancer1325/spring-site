# Spring Boot 1.5.6 available now

_Releases | Andy Wilkinson |  July 27, 2017 | 0 Comments_

On behalf of the team, I am pleased to announce that Spring Boot 1.5.6 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

Spring Boot 1.5.6 fixes a [problem with Spring Data's dependency management](https://github.com/spring-projects/spring-boot/issues/9871) that was introduced in 1.5.5. Sorry for the inconvenience.

### [](#whats-next)What's next?

After the [recent release of Spring Framework 5.0 RC3](https://spring.io/blog/2017/07/24/spring-framework-5-0-rc3-on-the-way-to-ga) and the flurry of other releases that followed, Spring Boot 2.0 M3 [has just been released](https://spring.io/blog/2017/07/26/spring-boot-2-0-0-m3-available-now). If you want to take an early look at Spring Boot 2, and we’d love to hear your feedback if you do, please go to [start.spring.io](https://start.spring.io) and select Spring Boot `2.0.0.M3` or `2.0.0.BUILD-SNAPSHOT`.
