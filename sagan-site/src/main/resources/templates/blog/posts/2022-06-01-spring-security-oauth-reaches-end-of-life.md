# Spring Security OAuth reaches End-of-Life

_Engineering | Joe Grandja |  June 01, 2022 | 11 Comments_

The [Spring Security OAuth](https://spring.io/projects/spring-security-oauth) and [Spring Security OAuth Boot 2 auto-configuration](https://github.com/spring-projects/spring-security-oauth2-boot) projects have reached end of life.

The Spring Security OAuth project has been replaced by the Client and Resource Server support provided by [Spring Security](https://docs.spring.io/spring-security/reference/servlet/oauth2/index.html) and the Authorization Server support provided by [Spring Authorization Server](https://spring.io/projects/spring-authorization-server).
