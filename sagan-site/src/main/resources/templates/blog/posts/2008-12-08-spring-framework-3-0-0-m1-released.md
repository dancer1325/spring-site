# Spring Framework 3.0.0.M1 Released

_Releases | Adam Fitzgerald |  December 08, 2008 | 0 Comments_

[Juergen Hoeller](http://www.springsource.com/people/jhoeller)'s latest [blog](http://blog.springsource.com/2008/12/05/spring-framework-30-m1-released/) announces the release of the first milestone for Spring 3.0. The main highlights are the addition of REST support, the Spring Expression Language (EL) and the move to Java 5 and above. The blog post contains the details.
