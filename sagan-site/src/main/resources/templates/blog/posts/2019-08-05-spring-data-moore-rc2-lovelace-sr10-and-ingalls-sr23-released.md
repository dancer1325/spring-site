# Spring Data Moore RC2, Lovelace SR10, and Ingalls SR23 released

_Releases | Mark Paluch |  August 05, 2019 | 0 Comments_

On behalf of the team, I'm pleased to announce Spring Data releases Moore RC2, Lovelace SR10, and Ingalls SR23. All releases pick up the most recent Spring Framework releases and will be picked up by Spring Boot 2.2 RC1, 2.1.7, and 1.5.22 respectively.

Amongst many dependency upgrades, we migrated almost all of our Spring Data core project to use AssertJ assertions. As per our [previous annoucement](https://spring.io/blog/2019/01/10/spring-data-lovelace-sr4-kay-sr13-and-ingalls-sr18-released), Ingalls SR23 is the last release of the Spring Data 1.x line to reach its **end of life**. We highly recommend migrating to Spring Data Lovelace SR10 for all users who still use Spring Data 1.x.

The releases ship over 320 fixed tickets in total. The most notable new features of Moore RC2 (among others) are:

-   Annotation-based configuration of Gateway Senders for Geode/GemFire.
-   Fluent reactive ChangeStream API for MongoDB.
-   Type-based field projection derivation for MongoDB aggregations.
-   Query derivation for Cassandra delete queries.
-   Upgrade to Apache Solr 8.2.
-   Upgrade to Elasticsearch 6.8.1.

We are working now towards Spring Data Moore RC3 (due in September) to upgrade dependencies before shipping a final GA release mid-September.
