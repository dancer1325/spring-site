# Spring Tools 4.15.1 released

_Engineering | Martin Lippert |  June 18, 2022 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.15.1 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

#### [](#fixes-and-improvements)fixes and improvements

-   *(Spring Boot)* fixed: VScode incorrectly suggests removing @Autowired annotation from methods ([#787](https://github.com/spring-projects/sts4/issues/787))
-   *(Spring Boot)* fixed: VScode quick fix should not suggest removing @Autowired annotation from JUnit tests ([#786](https://github.com/spring-projects/sts4/issues/786))
-   *(Eclipse)* fixed: Not able to extract the new version of spring-tool-suite-4-4.15.0.RELEASE-e4.24.0-win32.win32.x86\_64.self-extracting.jar on windows 11 ([#788](https://github.com/spring-projects/sts4/issues/788))
-   *(Eclipse)* fixed: Cannot open Spring Boot Language Server Java Editor settings menu ([#789](https://github.com/spring-projects/sts4/issues/789))

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Detailed changes can be found here: [https://github.com/spring-projects/sts4/wiki/Changelog#2022-06-17-4151-release-incl-language-servers-version-1360](https://github.com/spring-projects/sts4/wiki/Changelog#2022-06-17-4151-release-incl-language-servers-version-1360)

Spring Tools 4.15.2 is scheduled to be released in early August 2022.

Enjoy!
