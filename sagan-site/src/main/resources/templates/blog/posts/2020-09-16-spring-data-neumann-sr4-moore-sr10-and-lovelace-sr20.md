# Spring Data Neumann SR4, Moore SR10 and Lovelace SR20

_Releases | Mark Paluch |  September 16, 2020 | 0 Comments_

On behalf of the Spring Data team, I’m pleased to announce a round of service releases for the Spring Data Neumann, Moore, and Lovelace release trains. `Neumann SR4` ships with [43 tickets fixed](https://jira.spring.io/issues/?filter=17038), `Moore SR10` ships with [46 tickets fixed](https://jira.spring.io/issues/?filter=17035), and, finally, `Lovelace SR20` ships with [35 tickets fixed](https://jira.spring.io/issues/?filter=17034).

The releases ship with mostly bug fixes and dependency upgrades and will be picked up by Spring Boot `2.3.4`, `2.2.10`, `2.1.17` ( respectively) in the coming days.

Here are the detailed links to each individual artifact and the reference documentation:
