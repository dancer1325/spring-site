# This week in Spring: February 22nd, 2011

_Engineering | Josh Long |  February 23, 2011 | 0 Comments_

What a week! Lots of good, foundational content available this week from, and for, the community. Let's get right to it.

2.  Chris Beams blogged about the upcoming *FeatureSpecification* support in Spring 3.1 Feature Specifications provide the same features as the XML namespace infrastructure elements, like <tx:annotation-driven>, in a Java configuration-centric way. This is a very powerful feature, and [this blog post is the best way to get started.](http://blog.springsource.com/2011/02/17/spring-3-1-m1-featurespec/)
3.  [SpringSource Tool Suite](http://www.springsource.com/developer/sts) lead Christian Dupuis - [tweets](http://twitter.com/#!/cdupuis/status/40124813544660992) that users can now download STS without submitting to the registration form. Very cool!
