# This Week in Spring - July 13th, 2021

_Engineering | Josh Long |  July 13, 2021 | 0 Comments_

Hi, Spring fans! How're you doing? I'm heads-down, hurriedly making final preparations for tomorrow's SpringOne Tour episode installment. This is going to be *SO MUCH FUN*. Do NOT miss [this show tomorrow morning, 10AM California time](https://tanzu.vmware.com/developer/tv/springone-tour/0018/). I'll be there, along with my friends Madhura Bhave, Olga Maciaszek-Sharma, Bella Bai, Whitney Lee, Core Iberkleid, Nate Schutta, and special guest star Angie Jones.

We look forward to seeing you there and we'll explore the wide and wonderful world of continuous delivery, testing and more together.

-   JOOQ 3.15.0 Release [is out, and includes support for R2DBC (and much more)](https://blog.jooq.org/2021/07/06/3-15-0-release-with-support-for-r2dbc-nested-row-array-and-multiset-types-5-new-sql-dialects-create-procedure-function-and-trigger-support-and-much-more/)
-   Neat! The JRE will be UTF-8 by default instead of dangerously assuming the host operating system's default: [openjdk/jdk](https://github.com/openjdk/jdk/pull/4733)
-   [A Bootiful Podcast: Matt Raible](https://spring.io/blog/2021/07/08/a-bootiful-podcast-matt-raible) Last week, I talked to Matt Raible, and you can too! I love that guy. So happy he was able to join the show again.
-   [Spring Tips: Spring Cloud Square](https://spring.io/blog/2021/07/07/spring-tips-spring-cloud-square)
-   This is neat: you can run TestNG tests with [JUnit 5 now](https://twitter.com/junitteam/status/1413162607947681794?s=12)
-   [Piotr Mińkowski on Twitter: JPAstreamer is a library for expressing JPA/Hibernate queries as Java streams. It can be also integrated with Spring.](https://twitter.com/piotr_minkowski/status/1413833421747638277?s=12)
-   [Apress' *R2DBC Revealed* book is available for order](https://twitter.com/probablyrealrob/status/1412960224386949129?s=12)
-   [Scaling Spring Boot Microservices](https://medium.com/javarevisited/scaling-spring-boot-microservices-81b7e8c61b73)
-   Have you seen the lineup for our annual tentpole conference, SpringOne, this [Sep 1–2, 2021](https://springone.io/2021/speakers)? I'll be there and so will a ton of other amazing people.
-   [Spring Authorization Server 0.1.2 available now](https://spring.io/blog/2021/07/09/spring-authorization-server-0-1-2-available-now)
