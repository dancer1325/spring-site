# This Week in Spring - February 18, 2014

_Engineering | Josh Long |  February 19, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it!

1.  Spring Data lead Oliver Gierke has announced that [Spring Data Babbage SR3 has been released](https://spring.io/blog/2014/02/17/spring-data-release-train-babbage-sr3-released). The new release rolls up Spring Data Commons 1.6.4, Spring Data JPA 1.4.4, Spring Data MongoDB 1.3.4, and Spring Data Neo4J 2.3.4. Like Spring 4 before it, this release train ships with [a Maven dependency *BOM* (bill of materials)](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-releasetrain%7CBabbage-SR3%7Cpom) that can be used to level set the dependency versions.
2.  **March 18th Webinar -** Join [Mattias Arthursson](https://twitter.com/marthursson) as he [introduces Spring LDAP 2.0.0](https://spring.io/blog/2014/02/11/webinar-spring-ldap-2-0-0-on-march-18-2014).
3.  **March 25 Webinar -** Be sure to check out [Spring lead Juergen Hoeller's upcoming webinar: Spring Framework 4 on top of Java 8](https://spring.io/blog/2014/02/11/webinar-spring-framework-4-0-on-java-8-march-25).
4.  SpringOne2GX 2013 replay: [Spring and Sencha - A Match Made In Heaven (or at least the cloud)](https://spring.io/admin/blog/1403-springone2gx-2013-replay-spring-and-sencha-a-match-made-in-heaven-or-at-least-the-cloud) is now available
5.  Check out a founding father of Spring - Rob Harrop talk about [Running Spring in Amazon Web Services](https://spring.io/blog/2014/02/18/springone2gx-2013-replay-running-spring-in-amazon-web-services), a great talk from SpringOne2GX 2013.
6.  HCL Technologies AVP Srinivas Ajjarapu penned a [nice post on some of the ways that Pivotal's stack is helping IT lower spending](http://blog.gopivotal.com/case-studies-2/system-integrator-qa-saving-60-70-on-it-costs-with-pivotal-spring-and-more) and get more.
7.  Nicolas Frankel is back, and has put together a nice post on [how to chain together Spring MVC `ViewResolver`s](http://blog.frankel.ch/chaining-url-view-resolvers-in-spring-mvc).
8.  Daniel Mikusa and Stuart Williams's webinar replay introducing [Apache Tomcat 8 is now available online](https://spring.io/blog/2014/02/18/webinar-replay-introduction-to-apache-tomcat-8).
9.  Steve Perkins put together a nice post on his [first steps with Spring Boot](http://steveperkins.net/use-spring-boot-next-project/). Check it out!
10.  John Thoms has put together a nice example of configuring [Spring Boot's embedded Tomcat application server with SSL support](https://github.com/jthoms/spring-boot-embedded-tomcat-ssl). Check it out!
11.  The stellar JHipster project, which sits on top of Spring Boot, is now available [complete with new monitoring endpoints and a lot more](http://jhipster.github.io/2014/02/18/jhipster-release-0.10.0.html)!
12.  Speaking of Spring Boot, [Spring Boot RC 3 is now available](https://spring.io/blog/2014/02/12/spring-boot-1-0-0-rc2-released). Don't hesitate to check it out.
13.  Our pal Xavier Padró is back with a nice look at how to [use the view templating library Thymeleaf with Spring](http://xpadro.blogspot.com/2014/02/thymeleaf-integration-with-spring-part-1.html)
14.  My pal Andy Piper and Eclipse Paho (an MQTT client) lead Andy Piper has put together a nice post [on how to use Spring Integration's MQTT support](http://andypiper.co.uk/2014/02/14/spring-integration-mqtt/).
