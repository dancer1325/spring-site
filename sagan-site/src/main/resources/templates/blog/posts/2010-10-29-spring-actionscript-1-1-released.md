# Spring ActionScript 1.1 Released

_Releases | Adam Fitzgerald |  October 29, 2010 | 0 Comments_

Dear community,

We're pleased to announce that Spring ActionScript 1.1 is now available.

[Download](http://www.springactionscript.org/downloads/spring-actionscript-1.1-distribution-with-dependencies.zip) | [API Documentation](http://www.springactionscript.org/asdoc/index.html) | [HTML Docs](http://www.springactionscript.org/docs/reference/html/springactionscript.html) | [Changelog](http://www.springactionscript.org/changes-report.html)

Besides a series of bugfixes, this release adds the following new features and enhancements:

-   Component Scanning & Context XML Namespace
-   Test Framework
-   PostConstruct, PreDestroy and Inject Metadata

Details about the release are include in the [Spring ActionScript blog](http://www.herrodius.com/blog/category/actionscript). Community members can always ask questions in the community [forum](http://forum.springsource.org/forumdisplay.php?f=60) and identify issues in [JIRA](https://jira.springframework.org/browse/SESPRINGACTIONSCRIPTAS) as well.
