# Spring Cloud Data Flow 2.9.4 Released

_Releases | Chris Bono |  April 05, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring Cloud Dataflow `2.9.4` has been released and is now available from Maven Central.

This release contains an update of the Spring Boot version and addresses a couple of CVEs. See the [release notes](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.9.4) for more information.

## [](#notable-changes-in-294)Notable Changes in 2.9.4

-   Update to Spring Boot `2.5.12`
-   Resolves [CVE-2022-22965](https://tanzu.vmware.com/security/cve-2022-22965)
-   Resolves [CVE-2021-29425](https://nvd.nist.gov/vuln/detail/CVE-2021-29425)
