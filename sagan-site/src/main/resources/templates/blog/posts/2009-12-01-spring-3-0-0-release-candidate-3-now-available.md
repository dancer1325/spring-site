# Spring 3.0.0 Release Candidate 3 Now Available

_Releases | Adam Fitzgerald |  December 01, 2009 | 0 Comments_

It's getting closer. [Arjen Poutsma](http://blog.springsource.com/author/arjenp/) has just [announced](http://twitter.com/poutsma) that Spring 3.0.0.RC3 is now available. This release candidate contains lots of fixes to issues raised by community members.

  

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.0.RC3) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.0.RC3/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.0.RC3/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

  

Thank you to all of the dedicated community members that have given [feedback](http://forum.springsource.org/forumdisplay.php?f=21) and identified [issues](http://jira.springframework.org/browse/SPR). Please keep up the good work so that we can get a high quality 3.0 GA release.
