# A Bootiful Podcast: Spring Boot and Spring Cloud co-founder Dr. Dave Syer

_Engineering | Josh Long |  January 03, 2020 | 0 Comments_

Hi, Spring fans! In this episode, [Josh Long](http://twitter.com/starbuxman) talks to the one, the only, the inimitable, the amazing, his hero, the good [Dr. Dave Syer](http://twitter.com/david_syer), Spring Framework committer, former Spring Batch lead, former Spring Security OAuth lead, co-founder of both Spring Boot and Spring Cloud, etc. Statistically, if you're using Spring today, you're probably using code he wrote. This episode was recorded at SpringOne Platform 2019. Happy new year, and enjoy!
