# Spring Session 1.0.1 Released

_Releases | Rob Winch |  April 16, 2015 | 0 Comments_

I'm pleased to announce the release of [Spring Session](http://projects.spring.io/spring-session/) 1.0.1.RELEASE. You can find the release in [Maven Central](http://repo1.maven.org/maven2/org/springframework/session/).

This release fixes [30+ tickets](https://github.com/spring-projects/spring-session/issues?q=milestone%3A1.0.1). You can find the highlights below:

## [](#highlights)Highlights

The highlights of Spring Session 1.0.1 are available below:

-   Support for AWS ElastiCache [#124](https://github.com/spring-projects/spring-session/issues/124)
-   Servlet 3.1 fixes [#152](https://github.com/spring-projects/spring-session/issues/152). This resolves issues when running Spring Security and Spring Boot 1.2
-   Servlet 2.5 fixes [#111](https://github.com/spring-projects/spring-session/issues/111) [#182](https://github.com/spring-projects/spring-session/issues/182)
-   Added Servlet 2.5 & XML based configuration [sample and guide](http://docs.spring.io/spring-session/docs/1.0.1.RELEASE/reference/html5/guides/httpsession-xml.html)
-   Embedded Redis used in Samples now works on Windows 64 bit [#174](https://github.com/spring-projects/spring-session/issues/174)
-   New Spring Session Logo [#130](https://github.com/spring-projects/spring-session/issues/130)
