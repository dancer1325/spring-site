# Spring Boot 2.2.3 released

_Releases | Andy Wilkinson |  January 16, 2020 | 11 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot 2.2.3 has been released and is now available from [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This release includes [75 bug fixes, enhancements, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-boot/releases/tag/v2.2.3.RELEASE). Thanks to all those who have contributed with issue reports and pull requests.

### [](#important-security-advisory)Important Security Advisory:

This version of Spring Boot includes a dependency upgrade to Spring Framework 5.2.3 which contains fixes for [CVE-2020-5398](https://pivotal.io/security/cve-2020-5398) and [CVE-2020-5397](https://pivotal.io/security/cve-2020-5397).
