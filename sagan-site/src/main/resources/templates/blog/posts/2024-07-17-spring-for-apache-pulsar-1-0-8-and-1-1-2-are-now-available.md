# Spring for Apache Pulsar 1.0.8 and 1.1.2 are now available

_Releases | Chris Bono |  July 17, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.0.8` and `1.1.2` have been released and are now available from Maven Central.

The releases will be included in the upcoming Spring Boot [3.2.8](https://github.com/spring-projects/spring-boot/milestones/3.2.8) and [3.3.2](https://github.com/spring-projects/spring-boot/milestones/3.3.2) releases, respectively.

This release includes numerous dependency upgrades.

Please see the release notes ([1.0.8](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.0.8) and [1.1.2](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.2)) for more details.
