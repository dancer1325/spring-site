# Spring Integration AWS 2.3.1 & Spring Cloud Stream Kinesis Binder 2.0.1 Available

_Releases | Artem Bilan |  January 10, 2020 | 0 Comments_

Dear Spring Community,

Today it’s my pleasure to announce patch releases of Spring Integration for Amazon Web Services extension version `2.3.1` and Spring Cloud Stream Binder for AWS Kinesis version `2.0.1`.

These releases can be downloaded from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release):

```
Copycompile "org.springframework.integration:spring-integration-aws:2.3.1.RELEASE"
```

If you don’t use Kinesis Binder. Or via Binder dependency:

```
Copycompile "org.springframework.cloud:spring-cloud-stream-binder-kinesis:2.0.1.RELEASE"
```

Mostly these versions have bug fixes and some community feedback refinements. Plus they are upgraded to the latest dependencies. So, it is highly recommended to bump your projects to this latest versions.

Many thanks to everyone from the Community for all the feedback and contribution to these projects.

Any feedback is welcome via all the available communication channels!

With recently announced [Spring Cloud Ilford](https://spring.io/blog/2019/12/23/spring-cloud-roadmap-and-hoxton-and-greenwich-maintenance-and-eol-announcements), we also are going to do some major changes in Spring Integration for AWS and Spring Cloud Stream Binder for AWS Kinesis to support AWS SDK v2. Everybody who’s willing to help with this, you are welcome to contribute any possible way!
