# Spring Boot 2.0.0.M3 available now

_Releases | Madhura Bhave |  July 26, 2017 | 0 Comments_

The third milestone of Spring Boot 2.0 is available now from the [Spring milestone repository](http://repo.spring.io/milestone/). This milestone ensures compatibility with Spring Framework 5.0.0.RC3 and contains a number of other [dependency upgrades](https://github.com/spring-projects/spring-boot/issues?q=is%3Aissue+is%3Aclosed+milestone%3A2.0.0.M3+label%3A%22type%3A+dependency-upgrade%22). This release closes over [100 issues and pull requests](https://github.com/spring-projects/spring-boot/milestone/55?closed=1), thanks to all that contributed!

For a complete list of changes, and upgrade instructions, see the [Spring Boot 2.0.0.M3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.0.0-M3-Release-Notes) on the WIKI and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.0.0.M3/reference/htmlsingle/).

If you want to get started and discover the new features, you can easily bootstrap a new project on [https://start.spring.io](https://start.spring.io)
