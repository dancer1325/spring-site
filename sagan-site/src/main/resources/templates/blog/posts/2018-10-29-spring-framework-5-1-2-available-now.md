# Spring Framework 5.1.2 available now

_Releases | Brian Clozel |  October 29, 2018 | 4 Comments_

On behalf of the team, I am pleased to announce that Spring Framework 5.1.2 [is available now](https://spring.io/projects/spring-framework).

This maintenance release of the 5.1 line includes [30 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=17198).

After its Milestone and RC phase, Spring Boot 2.1.0 GA will be released this week, shipping with this new Spring Framework version!
