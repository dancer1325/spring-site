# Spring Integration 3.0 Milestone 2 is Now Available

_Releases | Gary Russell |  May 24, 2013 | 0 Comments_

We are pleased to announce that the [second milestone for Spring Integration 3.0.0](http://www.springsource.com/download/community?project=Spring%20Integration) is now available.

64 issues are resolved, including many improvements and new features; see the [Project Home Page](http://www.springsource.org/spring-integration/) and [Release Notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=14017) for more information.
