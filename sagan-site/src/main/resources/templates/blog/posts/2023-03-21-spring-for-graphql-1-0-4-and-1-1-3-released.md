# Spring for GraphQL 1.0.4 and 1.1.3 released

_Releases | Brian Clozel |  March 21, 2023 | 0 Comments_

I'm pleased to announce that the Spring for GraphQL 1.0.4 and 1.1.3 maintenance versions are now available on Maven Central.

Spring for GraphQL 1.0.4 includes [6 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.0.4). This version will be shipped with Spring Boot 2.7.10, [to be released this Thursday](https://calendar.spring.io/).

Spring for GraphQL 1.1.3 includes [12 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.1.3). This version will be shipped with Spring Boot 3.0.5, [to be released this Thursday](https://calendar.spring.io/).
