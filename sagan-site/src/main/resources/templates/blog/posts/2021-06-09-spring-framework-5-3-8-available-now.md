# Spring Framework 5.3.8 available now

_Releases | Stéphane Nicoll |  June 09, 2021 | 1 Comment_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.8` is available now.

Spring Framework `5.3.8` includes [35 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.8).

Stay tuned for follow-up Spring Boot releases later this week.
