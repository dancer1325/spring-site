# Spring for GraphQL 1.4.3 released

_Releases | Brian Clozel |  October 21, 2025 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.4.3 is now available on Maven Central. This release closes [5 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.4.3).

This version will ship with Spring Boot 3.5.7, [to be released this week](https://spring.io/projects#release-calendar).
