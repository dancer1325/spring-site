# Spring Cloud Camden SR7 Released

_Releases | Ryan Baxter |  May 08, 2017 | 0 Comments_

On behalf of the team, I am pleased to announce that Service Release 7 of the [Spring Cloud Camden](http://cloud.spring.io) Release Train is available today. The release can be found in our [Spring Release repository](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Ca%3A%22spring-cloud-dependencies%22). The documentation can be found [here](http://cloud.spring.io/spring-cloud-static/Camden.SR7/). The release notes can be found [here](https://github.com/spring-projects/spring-cloud/wiki/Spring-Cloud-Camden-Release-Notes#camdensr7).

#### [](#the-following-modules-were-updated-as-part-of-camdensr7)The following modules were updated as part of Camden.SR7:

-   Spring Cloud Commons [1.1.9.RELEASE](https://github.com/spring-cloud/spring-cloud-commons/milestone/24?closed=1)
-   Spring Cloud Netflix [1.2.7.RELEASE](https://github.com/spring-cloud/spring-cloud-netflix/milestone/39?closed=1)
-   Spring Cloud AWS [1.1.4.RELEASE](https://github.com/spring-cloud/spring-cloud-aws/milestone/15?closed=1)
-   Spring Cloud Consul 1.1.4.RELEASE Dependency Fixes
-   Spring Cloud Security 1.1.4.RELEASE Documentation Fixes
-   Spring Cloud Sleuth [1.1.4.RELEASE](https://github.com/spring-cloud/spring-cloud-sleuth/milestone/24?closed=1)
-   Spring Cloud Stream Brooklyn.SR3 fixes issues with Spring Cloud Bus, Hystrix, and Turbine. See this [issue](https://github.com/spring-cloud/spring-cloud-stream/issues/810) for more information.
-   Spring Cloud Contract [1.0.5.RELEASE](https://github.com/spring-cloud/spring-cloud-contract/milestone/11?closed=1)

And, as always, we welcome feedback: either on [GitHub](https://github.com/spring-cloud), on [gitter](https://gitter.im/spring-cloud/spring-cloud), on [Stack Overflow](http://stackoverflow.com/questions/tagged/spring-cloud), or on [Twitter](https://twitter.com/SpringCloud).
