# Spring REST Docs 1.0.0.RELEASE

_Releases | Andy Wilkinson |  October 07, 2015 | 4 Comments_

I'm delighted to announce that [Spring REST Docs](https://github.com/spring-projects/spring-restdocs) 1.0.0.RELEASE has been released. It's available from Maven Central and our [release repository](https://repo.spring.io/release).

## [](#what-is-spring-rest-docs)What is Spring REST Docs?

Spring REST Docs helps you to document RESTful services. It combines hand-written documentation written with Asciidoctor and auto-generated snippets produced with Spring MVC Test. This approach frees you from the limitations imposed by tools like [Swagger](http://swagger.io). It helps you to produce documentation that is accurate, concise, and well-structured. This documentation then allows your users to get the information they need with a minimum of fuss.

If you'd like to see some documentation produced using Spring REST Docs, please look at the sample [API guide](http://docs.spring.io/spring-restdocs/docs/1.0.0.RELEASE/samples/restful-notes/api-guide.html) and [Getting Started Guide](http://docs.spring.io/spring-restdocs/docs/1.0.0.RELEASE/samples/restful-notes/getting-started-guide.html) for a simple RESTful notes service.

## [](#getting-started)Getting started

To get started with Spring REST Docs, please refer to the [getting started section](http://docs.spring.io/spring-restdocs/docs/current/reference/html5/#getting-started) of the [reference documentation](http://docs.spring.io/spring-restdocs/docs/1.0.0.RELEASE/reference/html5/).
