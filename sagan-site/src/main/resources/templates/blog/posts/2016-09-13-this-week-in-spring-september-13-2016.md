# This Week in Spring - September 13, 2016

_Engineering | Josh Long |  September 13, 2016 | 2 Comments_

Welcome to another installment of *This Week in Spring*! It's been an amazing week here in China, hanging out with Alibaba and Huawei, bringing the Spring. This week it's off to Shanghai, Beijing, Guangzhou, Shenzhen and Hong Kong!

Our pal [Pieter Humphrey](http://twitter.com/PieterHumphrey) has been busy this week getting the SpringOne Platform 2016 videos up and online for us to enjoy so there's a lot of good stuff this week! Thanks Pieter!

-   Want see Mark Fisher and Mark Pollack present and live demo their way through Spring Cloud Data Flow on September 29th? [Reserve your seat now!](https://spring.io/blog/2016/09/13/webinar-data-microservices-in-the-cloud)
-   Want to learn what's new in Spring Framework 4.3? [Join Spring framework lead Juergen Hoeller for this look at an awesome framework](https://spring.io/blog/2016/09/12/springone-platform-2016-replay-modern-java-component-design-with-spring-framework-4-3)
-   I liked this talk from Home Depot [on what 12 Factor applications means to a Java Developer](https://spring.io/blog/2016/09/12/springone-platform-2016-replay-12-factor-or-cloud-native-apps-what-exactly-does-that-mean-for-spring-developers) and turns out it was one of the top attended at the conference!
-   I gave an almost *2 hour* long talk at SpringOne Platform 2016 called [*Cloud Native Java*](https://spring.io/blog/2016/09/12/springone-platform-2016-replay-cloud-native-java) covering Spring Boot (REST, HATEOAS, Actuator, JPA), Spring Cloud (centralized configuration, service registration and discovery, circuit breakers, microproxies, API gateways, eventual consistency, stream processing, observability and monitoring, distributed tracing), Spring Cloud Security (OAuth and SSO) and Spring Cloud Data Flow. Get it while it's hot!
-   Join Apache Tomcat ninja Mark Thomas for [a great look at the Apache Tomcat roadmap](https://spring.io/blog/2016/09/12/springone-platform-2016-replay-apache-tomcat-roadmap) in this talk from SpringOne Platform 2016
-   Spring Cloud Task and Spring Batch lead Michael Minella has [just announced Spring Cloud Task 1.0.3 and 1.1.0.M1 are now available](https://spring.io/blog/2016/09/09/spring-cloud-task-1-0-3-release-and-1-1-0-m1-are-now-available)
-   Spring Integration lead Gary Russell has just announced Spring for Apache Kafka 1.1.0 M2, which now supports [*batching* consumption of messages from a Kafka destination](https://spring.io/blog/2016/09/08/spring-for-apache-kafka-1-1-0-milestone-2-available) and seek operations! Get it while it's hot!
-   Spring Cloud Stream lead Marius Bogoevici has just [announced Spring Cloud Stream Brooklyn RC1](https://spring.io/blog/2016/09/08/spring-cloud-stream-brooklyn-rc1-is-available)
-   Spring Session lead Rob Winch has just announced [Spring Session 1.2.2 which contains numerous bugfixes and improvements](https://spring.io/blog/2016/09/07/spring-session-1-2-2-released)
-   Cool! [Instance-specific routing has just landed in Cloud Foundry's `gorouter`](https://github.com/cloudfoundry/gorouter/commit/1647c65cf65f712f7597c811cf7f98cfaab9365a)! This is *ideal* for instance-specific Spring Boot actuator endpoints!
-   I like this [Chinese-language introduction to Spring Retry](http://www.huangyunkun.com/2016/08/30/spring-retry/)
-   Want to hack together a poor-man's authorization server with Spring Session? [This is a pretty good introduction!](https://beku8.wordpress.com/2016/09/12/configuring-cross-domain-sso-and-slo-with-spring-security-and-spring-session/)
-   [I loved this blog introducing Spring Cloud Sleuth and Zipkin support on Aliyun's blog](https://yq.aliyun.com/articles/60165). Aliyun, for those not aware, is like AWS to Alibaba.com's Amazon. This is super cool!
-   Our former CTO Adrian Coyler has a great article on his Blog "The Morning Paper" on handling [data in microservice architecture](https://blog.acolyer.org/2016/09/13/data-on-the-outside-versus-data-on-the-inside/).
