# Spring Boot 2.0.1 available now

_Releases | Stéphane Nicoll |  April 05, 2018 | 3 Comments_

Hot on the heels of [Spring Boot 1.5.11](https://spring.io/blog/2018/04/05/spring-boot-1-5-11-available-now), Spring Boot 2.0.1 is now available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

The first maintenance release of Spring Boot 2 includes [over 160 fixes, improvements and dependency updates](https://github.com/spring-projects/spring-boot/milestone/98?closed=1). Thanks to all that have contributed with issue reports and pull requests.

Note that the Spring Framework versions included in both Spring Boot releases contain fixes that address multiple vulnerability reports. Please see this [blog post](https://spring.io/blog/2018/04/05/multiple-cve-reports-published-for-the-spring-framework) for more details.
