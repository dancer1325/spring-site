# Spring for Apache Hadoop 1.0.0 M2 Released

_Releases | Costin Leau |  June 13, 2012 | 0 Comments_

Dear Spring Community, I am pleased to announce the release of Spring for Apache Hadoop 1.0 M2:

[Downloads](http://www.springsource.com/download/community?project=Spring%20Data%20Hadoop&version=1.0.0.M2) | [JavaDocs](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.M2/api/) | [Reference Documentation](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.M2/reference/html/) | [Changelog](http://static.springframework.org/spring-data/hadoop/docs/1.0.0.M2/changelog.txt)

The Spring for Apache Hadoop features include support for:

-   DAO support (Template & Callbacks) for HBase
-   Cascading Taps for Spring & Spring Integration
-   Support for Hadoop Security
-   Enhanced Map/Reduce support
-   Two New Samples (hbase-crud and pig-scripting)

For more information on this release of Spring for Apache Hadoop, see this blog [entry](http://blog.springsource.org/2012/06/13/spring-for-hadoop-1-0-m2/) or the reference documentation.

Speaking of announcements, Project [Serengeti](http://serengeti.cloudfoundry.com/) was announced today. See Richard McDougall's [blog post](http://cto.vmware.com/project-serengeti-theres-a-virtual-elephant-in-my-datacenter) for more information.

We look forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?80-NoSQL) or in the [issue tracker](https://jira.springsource.org/browse/SHDP).
