# Spring for Apache Pulsar 1.2.8 and 2.0.0-M1 are now available

_Releases | Chris Bono |  July 22, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.2.8` and `2.0.0-M1` have been released and are now available from Maven Central.

The `1.2.8` release will be included in the upcoming Spring Boot [3.4.8](https://github.com/spring-projects/spring-boot/milestones/3.4.8) and [3.5.4](https://github.com/spring-projects/spring-boot/milestones/3.5.4) releases. The `2.0.0-M1` release will be included in the upcoming Spring Boot [4.0.0-M1](https://github.com/spring-projects/spring-boot/milestones/4.0.0-M1) release.

Please see the release notes ([1.2.8](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.8) and [2.0.0-M1](https://github.com/spring-projects/spring-pulsar/releases/tag/v2.0.0-M1)) for more details.
