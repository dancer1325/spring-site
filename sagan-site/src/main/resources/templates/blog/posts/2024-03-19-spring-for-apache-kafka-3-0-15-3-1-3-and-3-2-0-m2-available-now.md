# Spring for Apache Kafka 3.0.15, 3.1.3 and 3.2.0-M2 Available Now

_Releases | Soby Chacko |  March 19, 2024 | 0 Comments_

On behalf of the entire team and everyone in the community who contributed, we are pleased to announce the general availability of Spring for Apache Kafka `3.0.15` and `3.1.3`.

Spring for Apache Kafka `3.0.15` includes a [few improvements and bug fixes](https://github.com/spring-projects/spring-kafka/releases/tag/v3.0.15). In addition, this version now supports the ability to call the `enforceRebalance` on the Kafka consumer. This release will be included as part of the upcoming [Spring Boot 3.1.10 release](https://github.com/spring-projects/spring-boot/milestone/337).

Spring for Apache Kafka `3.1.3` ships with [some new features, enhancements, bug fixes and documentation improvements](https://github.com/spring-projects/spring-kafka/releases/tag/v3.1.3). This release will be part of the upcoming [Spring Boot 3.2.4 release](https://github.com/spring-projects/spring-boot/milestone/338).

We also released the second milestone for Spring for Apache Kafka 3.2.0 (`3.2.0-M2`). This milestone release includes a number of new additions as [features, enhancements, bug fixes and documentation changes](https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.0-M2). Some notable improvements are the support for non-blocking retries when using `KafkaListener` as a class-level annotation, a new seek API based on offsets computed from a user-provided function, more enhancements in the Kafka Stream interactive query support, etc. This milestone release will be included in the upcoming [Spring Boot 3.3.0-M3 release](https://github.com/spring-projects/spring-boot/milestone/330).

We want to thank everyone who contributed to these releases.
