# Spring Session 1.3.1 Released

_Releases | Rob Winch |  April 27, 2017 | 0 Comments_

On behalf of the community, I'm pleased to announce the release of Spring Session 1.3.1.RELEASE. [This release contains](https://github.com/spring-projects/spring-session/milestone/21?closed=1) numerous bug fixes. A special thanks to [Vedran Pavić](https://github.com/vpavic) and [John Blum](https://github.com/jxblum) for all their hard work!

Some of the highlights include:

-   [#756](https://github.com/spring-projects/spring-session/issues/756) - Usage of Spring Data Redis 1.7.1 which has critical bug
-   [#757](https://github.com/spring-projects/spring-session/issues/757) - Restore proper behavior of HttpSession created events in GemFire
-   [#755](https://github.com/spring-projects/spring-session/issues/755) - Improve GemFire SessionRepository, (HTTP) Session copy logic
