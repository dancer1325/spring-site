# Micro Cloud Foundry Gives Developers Their Own Personal Cloud

_News | Adam Fitzgerald |  August 24, 2011 | 0 Comments_

Today the VMware team released [Micro Cloud Foundry](http://micro.cloudfoundry.com), a complete, local version of the popular, open source Platform as a Service that lets developers run a full featured cloud on their Mac or PC. Using Micro Cloud Foundry developers can build end-to-end cloud applications locally, without the hassles of configuring middleware while preserving the choice of where to deploy and the ability to scale their applications without changing a line of code.

Micro Cloud Foundry is available as a free downloadable virtual machine image and is compatible with VMware Fusion for Mac OS X and VMware Workstation and VMware Player (available as a free download) for Linux and Windows computers. It provides easy installation, setup, and virtual machine management and all you need is a [Cloud Foundry account](http://cloudfoundry.com/signup) to get started.

[Josh Long](http://blog.springsource.com/author/jolong/) has created a [getting started guide for Spring developers](http://blog.springsource.com/2011/08/24/micro-cloud-foundry-for-spring-developers) and produced a simple [video tutorial](http://www.youtube.com/watch?v=cKkz_vRNG1Q).
