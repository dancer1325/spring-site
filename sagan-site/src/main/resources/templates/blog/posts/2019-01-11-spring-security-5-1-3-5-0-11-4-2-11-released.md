# Spring Security 5.1.3, 5.0.11, 4.2.11 Released

_Releases | Josh Cummings |  January 11, 2019 | 4 Comments_

On behalf of the community I am pleased to announce the release of Spring Security 5.1.3 ([changelog](https://github.com/spring-projects/spring-security/issues?q=is%3Aclosed+milestone%3A5.1.3+sort%3Aupdated-desc)), 5.0.11 ([changelog](https://github.com/spring-projects/spring-security/issues?q=is%3Aclosed+milestone%3A5.0.11+sort%3Aupdated-desc)), and 4.2.11 ([changelog](https://github.com/spring-projects/spring-security/issues?q=is%3Aclosed+milestone%3A4.2.11+sort%3Aupdated-desc)). This series of releases provides a round of bug fixes and users are encouraged to update to the latest patch release.
