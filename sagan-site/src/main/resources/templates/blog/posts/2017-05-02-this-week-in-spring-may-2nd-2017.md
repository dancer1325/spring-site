# This Week in Spring - May 2nd, 2017

_Engineering | Josh Long |  May 02, 2017 | 2 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in Perth, Australia, for the [amazing YOW! West conference](http://west.yowconference.com.au/speakers/josh-long/), talking to developers about building cloud native Java applications. If you're around, say [hi](http://twitter.com/starbuxman). As usual, we've got a lot to cover so let's get to it!

I'm super excited about our first [Spring Days](https://www.springdays.io/ehome/spring-days/chicago#melendez) event, on May 30-31, in Chicago. I'll be there along with other Spring gurus like Eddu Melendez, Anupama Pradhan, and Michael Minella. I hope you'll join us!

-   Spring IO lead [Any Wilkinson has announced Spring IO Platform Athens-SR5](https://spring.io/blog/2017/04/28/spring-io-platform-athens-sr5). He [also announced Spring IO Platform Brussels-SR2](https://spring.io/blog/2017/04/28/spring-io-platform-brussels-sr2)
-   Spring Session lead [Rob Winch just announced Spring Session 1.3.1](https://spring.io/blog/2017/04/27/spring-session-1-3-1-released)
-   Spring Integration lead and all around great human [Gary Russell just announced Spring for Apache Kafka 2.0M1](https://spring.io/blog/2017/04/27/spring-for-apache-kafka-2-0-milestone-1-available)
-   Spring Cloud ninja Glenn Renfro [just announced Spring Cloud Task 1.2.0](https://spring.io/blog/2017/04/25/spring-cloud-task-1-2-0-release-is-now-available)
-   I *loved* (in case you were unclear about the sentiment I have for this post!) this look at real time sentiment [analysis with TensorFlow and Spring Cloud Data Flow](https://www.youtube.com/watch?v=QzF2Iii4s5c)
-   the results of this months' ZeroTurnaround web framework report are not too surprising: [Spring MVC is in a distant lead, then JSF and GWT, then Spring Boot](https://zeroturnaround.com/wp-content/uploads/2017/03/RebelLabs-WFI-Index-table-April-v3.png).
-   this post, "[Bootiful Development with Spring Boot and Angular](http://developer.okta.com/blog/2017/04/26/bootiful-development-with-spring-boot-and-angular)," on the Okta blog, is another gem by our buddy Matt Raible.
-   Check out the ZT (short for ZeroTurnaround) Live replay for an epic look at [Reactive Spring by Spring WebFlux engineer Sebastien Deleuze](http://pages.zeroturnaround.com/Reactive-Spring-Webinar_On-Demand-Gated-Landing-Page.html?utm_campaign=reactivespringwebinar&utm_medium=social&utm_source=twitter).
-   Spring Cloud engineer Jon Schneider is cooking up some very cool stuff, and it might be an indication of future direction. Who knows!? But you should [definitely check out his project, Spring Metrics](https://github.com/jkschneider/spring-metrics)
-   My buddy Matt Raible and I looked at building progressive web applications with Spring Boot and Angular at our recent two-parter talks at Devoxx France 2017: [part 1](https://www.youtube.com/watch?v=0MBsfdQiS64&feature=youtu.be) and [part 2](https://www.youtube.com/watch?v=xo7djiUBMpU). This was *so* much fun! Merci mille fois, Devoxx France! Thank you Devoxx France for letting us do these sessions!
-   I did an interview with the amazing [Software Engineering Daily Podcast on Spring Boot](https://softwareengineeringdaily.com/2017/04/26/spring-boot-with-josh-long/). It was a lot of fun being on there. I listen to the podcast regularly and really appreciate the opportunity!
-   I liked this post by Alex Soto on [using Arquillian to test Spring Data applications](http://www.lordofthejars.com/2017/04/testing-spring-data-spring-boot.html).
-   this is an interesting post on discovering an microservices [exposed port in a Docker container](https://040code.github.io/2017/04/20/discovery-agent/). Seems.. painful. I'm glad it's so thoroughly documented and articulated here, though! Good stuff!
