# Spring IO Platform 2.0.0.RC1

_Releases | Andy Wilkinson |  October 29, 2015 | 2 Comments_

Spring IO Platform 2.0.0.RC1 is available now from the [Spring milestone repository](http://repo.springsource.org/milestone/). The 2.0 release upgrades the versions of a number of components including Spring Boot 1.3 and Spring Framework 4.2.

Please note that a number of dependencies that were previously part of the Platform have been removed or replaced in this release. Please refer to [the documentation](http://docs.spring.io/platform/docs/2.0.0.RC1/reference/htmlsingle/#upgrading-dependency-management) for further details.

All being well, 2.0.0.RELEASE will be released in a few weeks time once Spring Boot 1.3.0.RELEASE is available. Please take RC1 for a spin and let us know if you find any problems.
