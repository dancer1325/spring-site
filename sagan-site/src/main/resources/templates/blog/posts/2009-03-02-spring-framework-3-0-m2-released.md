# Spring Framework 3.0 M2 released (03/2009)

_Releases | Adam Fitzgerald |  March 02, 2009 | 0 Comments_

If you aren't following the SpringSource [blog](http://blog.springsource.com/) you may have missed it, but last week [Juergen Hoeller](http://blog.springsource.com/author/juergenh/) announced the availability of the second milestone for Spring 3.0. Juergen's [blog post](http://blog.springsource.com/2009/02/25/spring-framework-30-m2-released/) covers all of the details about the milestone including the new RestTemplate, early JPA 2.0 support, more Java 5 style API updates and other improvements.

You can always get the latest milestones, release candidates and full releases for Spring from the [download center](http://www.springsource.com/download/community?project=Spring%20Framework).
