# Spring Security Kerberos 2.0.0-RC2 available now

_Releases | Janne Valkealahti |  May 25, 2023 | 0 Comments_

I'm happy to announce that a Spring Security Kerberos `2.0.0-RC2` milestone is now available via [https://repo.spring.io/milestone](https://repo.spring.io/milestone). This release candidate were created because project upgraded to use recently released `spring-security` `6.1.0`.
