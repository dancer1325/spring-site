# Spring Framework 4.3.4 available now

_Releases | Stéphane Nicoll |  November 08, 2016 | 0 Comments_

It is my pleasure to announce that Spring Framework [4.3.4](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15722) [is available now](http://projects.spring.io/spring-framework).

This maintenance release includes over 50 fixes and selected improvements, in particular user-requested configuration and runtime refinements.
