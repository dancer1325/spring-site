# Spring for Apache Pulsar 0.1.0-M1 is now available

_Releases | Chris Bono |  September 20, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `0.1.0-M1` has been released and is now available from [https://repo.spring.io/milestone](https://repo.spring.io/milestone).

This is the first milestone for the newly introduced Spring for Apache Pulsar framework and is based on Java17, Spring Boot `3.0.0-M5` and Spring Framework `6.0.0-M5`.

Please see our [previous blog post](https://spring.io/blog/2022/08/16/introducing-experimental-spring-support-for-apache-pulsar) to learn more about the background/details of the project and the [release notes](https://github.com/spring-projects-experimental/spring-pulsar/releases/tag/0.1.0-M1) for details on this milestone.

If you’re interested in trying this new release, you can follow the [Quick Tour](https://docs.spring.io/spring-pulsar/docs/0.1.0-M1/reference/html/#quick-tour) section in the reference docs.

While the project is in the experimental phase, there is a high level of unit and integration test coverage and we expect you to have a solid and smooth experience. That being said, when/if you run into any problems please report them on our [Github issue tracker](https://github.com/spring-projects-experimental/spring-pulsar/issues). If you have general feedback or want to discuss something else related to the release please use the [Github release discussion](https://github.com/spring-projects-experimental/spring-pulsar/discussions/131) or for other general feedback feel free to open a [new discussion](https://github.com/spring-projects-experimental/spring-pulsar/discussions/new).
