# Spring IO Platform Cairo-SR7

_Releases | Andy Wilkinson |  January 24, 2019 | 0 Comments_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Cairo-SR7 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Reactor Bismuth-SR15
-   Spring AMQP 2.0.11
-   Spring Batch 4.0.2
-   Spring Data Kay-SR13
-   Spring Framework 5.0.12
-   Spring Integration 5.0.11
-   Spring Kafka 2.1.12
-   Spring REST Docs 2.0.3
-   Spring Retry 1.2.3
-   Spring Security 5.0.11
-   Spring Session Apple-SR8
-   Spring Web Services 3.0.6

The versions of a number of third-party dependencies have also been updated.
