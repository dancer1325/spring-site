# Spring Boot for Apache Geode & Pivotal GemFire 1.0.0.RELEASE Available!

_Releases | John Blum |  May 07, 2019 | 0 Comments_

At long last, it is my distinct pleasure to announce the release of *Spring Boot for Apache Geode & Pivotal GemFire* (SBDG) `1.0.0.RELEASE`!

The list of people to thank is long.

First, to **Dr. Mark Pollack** for your feedback real early on in the beginning of the project.

To **Mr. Josh Long** for sitting with me for 2 hours after midnight during S1P-2018 to give me feedback on using Apache Geode with Spring Boot.

To **Mr. Phil Webb**, and the entire Spring Boot ream, for providing guidance.

To the few [community members](https://github.com/spring-projects/spring-boot-data-geode/graphs/contributors) who contributed PRs.

And finally, to everyone else for your support and patience while this project came to be.

\\o/ .oO(Victory!)

SBDG 1.0 GA [bits](https://search.maven.org/search?q=spring-geode-starter) are available in *Maven Central*.

## [](#whats-new)[](#whats-new)What’s New

The SBDG 1.0 GA release includes:

-   Polishes the [reference documentation](https://github.com/spring-projects/spring-boot-data-geode#documentation).
    
-   New Sample with [Guide](https://docs.spring.io/autorepo/docs/spring-boot-data-geode-build/1.0.0.RELEASE/reference/html5/guides/caching-look-aside.html) and [Example Application](https://github.com/spring-projects/spring-boot-data-geode/tree/1.0.0.RELEASE/spring-geode-samples/caching/look-aside) Code for the *Look-Aside Cache* pattern.
    
-   Upgrades to *Spring Test for Apache Geode & Pivotal GemFire* (STDG) `0.0.1.RELEASE` . More details on this new and upcoming project can be found [here](https://github.com/spring-projects/spring-test-data-geode).
    

For a complete list of changes, see the [changelog](https://github.com/spring-projects/spring-boot-data-geode/blob/1.0.0.RELEASE/spring-geode/src/main/resources/changelog.txt#L7-L14).

## [](#whats-next)[](#whats-next)What’s Next

Now with the 1.0 GA out the door, all attention moves to 1.1.

SBDG 1.1 will be rebased on:

-   Spring Framework 5.1
    
-   Spring Boot 2.1
    
-   Spring Data Lovelace
    
-   Spring Session 2.1
    

The `1.1.0.M1` will be released and announced shortly.
