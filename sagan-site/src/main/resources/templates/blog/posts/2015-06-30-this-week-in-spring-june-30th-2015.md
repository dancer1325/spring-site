# This Week in Spring - June 30th, 2015

_Engineering | Josh Long |  June 30, 2015 | 1 Comment_

Welcome to another installment of *This Week in Spring*! This week I'm in warm 'n wonderful Austin, TX to meet with customers and then off to Mexico City, Mexico to present the keynote at a [the SG Conference and Expo](http://sg.com.mx/sgce). If you're in Mexico City y si le gustaria hablarme sobre todos las cosas Spring y la informatica, sere a Mexico City desde miercoles. No hesita a contactarme en [Twitter (@starbuxman)](http://twitter.com/starbuxman)!

It's June 30th! Whew this year is *flying*! We've got a lot to cover this week so let's get to it!

-   Spring framework lead Juergen Hoeller just [announced Spring framework 4.1.7 and 3.2.14, both of which are maintenance releases](https://spring.io/blog/2015/06/30/spring-framework-4-1-7-3-2-14-released)
-   Spring framework ninja [Stephane Nicoll just announced Spring framework 4.2 RC2, packed with lots of new features!](https://spring.io/blog/2015/06/30/spring-framework-4-2-rc2-available-now)
-   Spring Tool Suite lead Martin Lippert just [announced updates version 3.7.0](http://spring.io/blog/2015/06/30/spring-tool-suite-3-7-0-released). The updates include Eclipse Mars 4.5 GA, Spring Boot YML support, improved Cloud Foundry integration, Pivotal tcServer 3.1.1 support, and early support for Spring Roo 2.0.0.M1 (thanks to Disid).
-   the good Dr. Syer, co-founder of the Spring Boot and Spring Cloud projects, just announced [the first Spring Cloud release train](http://spring.io/blog/2015/06/26/spring-cloud-service-release-angel-sr3), *Angel.SR3*.
-   Spring RESTDocs lead Andy Wilkinson's [just announced Spring REST Docs 1.0.0.M1](http://spring.io/blog/2015/06/25/spring-rest-docs-1-0-0-m1).
-   the *Spring Framework Guru* blog has put together a nice piece [on using Spring Boot with Thymeleaf](https://springframework.guru/spring-boot-web-application-part-2-using-thymeleaf/).
-   I hope you caught Spring Data lead Oliver Gierke's [epic vJUG presentation on elegant data access with Spring Data](http://zeroturnaround.com/rebellabs/state-of-the-art-data-access-with-spring-data-by-oliver-gierke/)!
-   Spring Data artist Thomas Darimont put together a nice GitHub GIST on how to [use custom SpEL statements in the `@Value` annotation](https://gist.github.com/thomasdarimont/7df3f04d3be7acea5dfa). It's full of cool examples, check it out!
-   Alberto Souza put together a nice (Spanish-language) post on how to [*get started with the magic of Spring Boot*](https://domineospring.wordpress.com/2015/06/26/comecando-com-a-magia-do-spring-boot/)
-   Mohamed Sanaulla has a nice post on using Spring Boot to run [full integration tests against Spring Boot-based REST services](http://www.javabeat.net/spring-boot-testing/)
-   The Keyhole software blog has a nice post on [moving a Spring Batch job from XML configuration to the Java configuration DSL](https://keyholesoftware.com/2015/06/29/spring-batch-javaconfig/)
-   the Nixmash blog has a nice post on changing the default `.favico` file that gets shown in Spring MVC. The post speaks about Spring MVC, but in fact, the approach given only works [when using Spring MVC *inside of Spring Boot*](http://nixmash.com/java/using-for-favorite-favicon-in-spring-mvc). Otherwise, great post!
-   Justin Musgrove put together a cool video and walkthrough [on how to prevent double click prevention using Spring MVC](http://www.leveluplunch.com/java/tutorials/035-prevent-duplicate-form-submission-spring-mvc/)
-   Check out this CFSummit talk that looks at some of the [new features in the Cloud Foundry Service Broker API](http://blog.pivotal.io/pivotal-cloud-foundry/products/new-cloud-foundry-service-broker-updates)
-   Our pal [@Making](http://twitter.com/making) put together a nice (Japanese language) talk for [the JSUG whose slides are available here](http://www.slideshare.net/makingx/spring-boot-java-jsug)
