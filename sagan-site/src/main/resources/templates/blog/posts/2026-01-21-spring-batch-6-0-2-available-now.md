# Spring Batch 6.0.2 available now

_Releases | Mahmoud Ben Hassine |  January 21, 2026 | 0 Comments_

On behalf of the team and all contributors, I am pleased to announce that Spring Batch `6.0.2` is now available from Maven Central.

This release comes with a number of improvements, bug fixes and documentation updates. The release notes can be found on GitHub [here](https://github.com/spring-projects/spring-batch/releases/tag/v6.0.2). This version will be available through Spring Boot `4.0.2`.
