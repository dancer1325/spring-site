# Video: SpringOne 2GX - Native Android Development Practices

_News | Adam Fitzgerald |  February 16, 2012 | 0 Comments_

[![SpringOne 2GX Video: Native Android Development Practices](http://www.springsource.org/files/SpringOne2GX2012NativeAndroidDevelopment.png)](http://www.infoq.com/presentations/Native-Android-Development-Practices)

In this popular session from SpringOne 2GX 2011, Spring developer advocate, [Josh Long](http://blog.springsource.org/author/jolong/) and Spring engineer, [Roy Clarkson](http://blog.springsource.org/author/rclarkson/) discuss [native Android development practices](http://www.infoq.com/presentations/Native-Android-Development-Practices) with the Spring framework. The session covers Spring MVC support for device detection, REST implementation for handling device requests and Spring Android support for organizing calls from the mobile client to the server.

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
