# This Week in Spring - January 21st, 2014

_Engineering | Josh Long |  January 22, 2014 | 0 Comments_

Welcome back to another installment of *This Week in Spring!* As usual, we've got a lot to cover so let's dive into it!

Quick reminder: if you're in the San Francisco bay area, be sure to join me tomorrow evening at the Oakland JUG where [I'll be speaking about Spring and Spring Boot](http://www.meetup.com/eastbayjug/events/159344362/)!

1.  Join us on February 11th for a webinar [introducing what's coming in Apache Tomcat 8](http://spring.io/blog/2014/01/21/webinar-intro-to-apache-tomcat-8).
2.  Join Spring Data engineers Oliver Gierke and Thomas Darimont on February 18 for a webinar [introducing Spring Data repositories best-practices](http://spring.io/blog/2014/01/21/webinar-spring-data-repositories-best-practices)
3.  Check out the replay of [Spring MVC ninja Rossen Stoyanchev](https://twitter.com/rstoya05)'s talk from SpringOne2GX 2013 on [WebSocket Applications with Spring Framework 4.0](http://spring.io/blog/2014/01/21/springone2gx-2013-replay-intro-to-websocket-applications-with-spring-framework-4-0).
4.  Check out the replay of [Spring Security lead Rob Winch's webinar introducing Spring Security 3.2](http://spring.io/blog/2014/01/21/webinar-replay-spring-security-3-2)
5.  Check out Mattias Severson's [SpringOne 2013 session replay reviewing Spring MVC test APIs](http://spring.io/blog/2014/01/21/springone2gx-2013-replay-spring-testing).
6.  [Spring Security lead Rob Winch](http://twitter.com/rob_winch) has a nice post [on how cookies can be exploited if not properly secured](http://spring.io/blog/2014/01/20/exploiting-encrypted-cookies-for-fun-and-profit).
7.  Meanwhile, over on the Pivotal blog, [C24 Founder and CTO John Davies](http://springone2gx.com/conference/speaker/john_davies) gives us a look at how [C24 is delivering high scalability for large financial-services companies with Spring, RabbitMQ, Pivotal, and GemFire](http://blog.gopivotal.com/case-studies-2/breaking-the-benchmark-moving-from-20k-to-1-million-complex-events-processed-per-second).
8.  Les Cast Codeurs, a French language coding podcast (though, fairly well dominated by JVM technologies), [interviewed Grails and Reactor committer Stephane Maldini](http://lescastcodeurs.com/2014/01/20/interview-sur-grails-et-spring-reactor-avec-stephane-maldini/), check it out!
9.  The *Squadron* blog has a nice [post on how to use TLS](http://blog.gosquadron.com/use-tls) (and the instructions are basically the same for SSL). This post is not specific to Spring, but if you're looking at [Spring Security's X.509](http://docs.spring.io/spring-security/site/docs/3.2.0.RELEASE/reference/htmlsingle/#x509) support, then this article will be *very* useful.
10.  Mkyong is back with another post, this time on [using TestNG with Spring](http://www.mkyong.com/unittest/testng-spring-integration-example/).
11.  The *Blas from the Pas* blog has a nice [post on how to use Thymeleaf and Spring MVC](http://theblasfrompas.blogspot.com/2014/01/spring-mvc-with-thymeleaf-html-template.html). You might also [check out our guide on the subject](http://spring.io/guides/gs/serving-web-content/).
12.  Andriy Redko has a nice post on [how to use the Scala language `sbt` build tool with a simple Spring application](http://java.dzone.com/articles/your-build-tool-your-good). The thrust of this post is to introduce `sbt`, which is an alternative build tool to things like Maven or Gradle.
13.  The *Develop and Conquer* blog has a nice post [on using Spring 4's websocket support](http://raymondhlee.wordpress.com/2014/01/19/using-spring-4-websocket-sockjs-and-stomp-support-to-implement-two-way-server-client-communication/).
14.  Our pal Xavier Padró is back, this time with a look at how [to migrate XML-configured web applications to the recently released Spring 4.0](http://xpadro.blogspot.com/2014/01/migrating-spring-mvc-restful-web.html)
15.  [Spring Data lead Oliver Gierke's](https://twitter.com/olivergierke) talk from Geecon, [*Data Access 2.0? Please welcome Spring Data!*](http://vimeo.com/84540646), is now available online to watch. Enjoy!
16.  Jakub Kubrynski has a nice post [on using Spring's Java-centric web configuration options](http://www.kubrynski.com/2014/01/understanding-spring-web-initialization.html).
17.  The *Remove duplications and fix bad names* blog has a nice post on [unit-testing with Spring and Mockito](http://rdafbn.blogspot.com/2014/01/testing-spring-components-with-mockito.html).
18.  The Isos Tech blog has a post [on the subject of using two JPA `EntityManager`s in one Spring application](http://blogs.isostech.com/enterprise-application-development/enterprise-application-development-using-multiple-entity-managers/).
