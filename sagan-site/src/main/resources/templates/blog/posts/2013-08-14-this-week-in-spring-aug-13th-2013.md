# This Week in Spring - Aug 13th, 2013

_Engineering | Josh Long |  August 14, 2013 | 0 Comments_

Welcome back to another installment of *This Week in Spring*. As usual, we've got a lot to cover, so let's get to it!

1.  The *How to do in Java* blog [has a nice post on how to setup Siteminder pre-authentication using Spring Security 3](http://howtodoinjava.com/2013/08/11/spring-3-security-siteminder-pre-authentication-example/).
2.  Another great [SpringOne2GX 2013](http://springone2gx.com/conference/santa_clara/2013/09/home) session's just been added to the SpringOne2GX 2013 lineup, [Real Time Analytics with Spring](https://springone2gx.com/conference/santa_clara/2013/09/session?id=29428). This talk introduces one use case for [Project Reactor](http://blog.springsource.org/2013/05/13/reactor-a-foundation-for-asynchronous-applications-on-the-jvm/), a *foundation for asynchronous applications on the JVM*.
3.  Andy Clement has just cut a [new release of AspectJ, 1.8.0.M1](http://dev.eclipse.org/mhonarc/lists/aspectj-dev/msg03031.html), which will be used in Spring 4 and support Java 8. It is available through the SpringSource Maven repository as 1.8.0.M1. It is also in today's release of [AJDT for Eclipse 4.3](http://www.eclipse.org/ajdt/downloads/).
4.  The GoPivotal blog has an in-depth look [at Apache Tomcat 8](http://blog.gopivotal.com/products/apache-tomcat-8-what-it-is-what-you-need-to-know). Definitely worth a look!
5.  Eberhard Wolff has put together a very nice video on using the recently announced [Spring Boot](https://twitter.com/ewolff/status/367336936492785664). Nice job, Eberhard! (as usual)
6.  Our pal Petri Kainulainen has written a very cool post on [unit testing Spring MVC REST APIs](http://www.javacodegeeks.com/2013/08/unit-testing-of-spring-mvc-controllers-rest-api.html).
7.  The *Being Java Guys* blog has a code-heavy post on how to do [file uploads with Spring MVC](http://www.beingjavaguys.com/2013/08/spring-mvc-file-upload-example.html). Nice job!
8.  This post from the *Matthew's Thoughts!* blog explains a simple Spring REST starter project that demonstrates how to use regular [Spring Security to add a username and password-based authentication with a Spring MVC-powered REST service](http://blogs.justenougharchitecture.com/?p=736).  
    
9.  The *Code with Zen Mind* blog has a nice series on building and testing Spring MVC applications. The [first post](http://codewithzenmind.wordpress.com/2013/07/11/spring-mvc-based-application-test-driven-development-part-1-project-setup-and-first-test/) introduces how to setup a test-driven project. The [second post](http://codewithzenmind.wordpress.com/2013/07/29/spring-mvc-based-application-test-driven-development-part-2-product-search-code-re-factoring-and-new-test-cases/) demonstrates how to do refactoring and how to introduce new test cases. The [third post](http://codewithzenmind.wordpress.com/2013/08/11/spring-mvc-based-application-test-driven-development-part-3-product-service/) demonstrates how to use the tests established in the first two posts to survice a major refactoring (the implementation of the service under test changes). Really insightful!
10.  [This post](http://ics.upjs.sk/~novotnyr/blog/1238/slf4j-a-spring-mvc-a-jetty-a-maven) from the *public static void blog()* blog introduces how Spring's logging layering works. The post is in what Google Translate insists is Slovak, however, the translation was pretty good and - if we're honest - the diagrams are quite explanatory by themselves! Good stuff. Take a look, and - if possible - a read.  
     
11.  The 1.5 version of the Cloud Foundry integration for Eclipse, which supports pushing applications to Pivotal Cloud Foundry organizations and spaces, using new Cloud Foundry services, and incrementally updating applications from Spring Tool Suite. The new integration may be installed from the STS dashboard or using [the update site](http://dist.springsource.com/release/TOOLS/cloudfoundry) in the *Help* > *Install New Software* menu.
