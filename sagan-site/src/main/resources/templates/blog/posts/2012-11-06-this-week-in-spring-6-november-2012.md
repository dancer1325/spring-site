# This Week in Spring - 6 November, 2012

_Engineering | Josh Long |  November 06, 2012 | 0 Comments_

Welcome back to another installment of *This Week in Spring*! We've got a lot to cover, so let's get to it!

1.  Did you guys miss [SpringOne2GX 2012](http://springone2gx.com)? I'm not going to lie to you - you *really* missed out! However, don't despair! Two of my absolute favorite talks from the show [are now available online - the keynotes](http://www.springsource.org/node/3722)! Both were amazing, but if you're looking for an amazing introduction to the trends underlying next generation of web applications, and a great (*amazing!*) tour of RabbitMQ, check out the day 2 keynote!

2.  Guys, I'm so excited to tell you [that SpringOne is coming to India](http://springoneindia.cloudfoundry.com/)! Register now!
3.  Does your PaaS support your workloads? If you're [using Cloud Foundry, then chances are good that it does!](http://www.infoq.com/news/2012/11/aws-elastic-beanstalk-ruby;jsessionid=266AB64478C0CED2CBCA48E6A6834488)
4.  Michael Isvy has put together a nice post [on Spring MVC and the view layer](http://t.co/8Bif6kTj)
5.  The Spring Framework TestContext framework now supports [session/request scoped beans for integration testing](https://jira.springsource.org/browse/SPR-4588)
6.  The first milestone [of Spring Data SOLR has been released!](http://www.springsource.org/node/3719)
7.  [Jonathan Brisbin](http://twitter.com/j_brisbin) has announced the [1.2 release of the Spring REST Shell](https://github.com/SpringSource/rest-shell). The new release features SSL, basic auth, dotrc support, an updated Homebrew install formula, HATEOAS and hypermedia.
8.  Check out Costin Leau's [webinar introducing Spring Data Hadoop](http://www.springsource.org/node/3669) this Thursday!
9.  RabbitMQ developer advocate Alvaro Videla linked to a [great post on visualizing RabbitMQ topologies](http://www.lshift.net/blog/2012/11/04/visualising-rabbitmq-topologies).
10.  Spring Data Batch and Spring Security OAuth ninja and Cloud Foundry UAA committer Dr. David Syer has written a [really fantastic post](http://blog.cloudfoundry.org/2012/11/05/how-to-integrate-an-application-with-cloud-foundry-using-oauth2/) on using Cloud Foundry's UAA agents OAuth2 endpoint. This has scarce little to do with Spring, except that Dr. Syer's posts are very educational and helped me tremendously in understanding the problems that Spring Security OAuth solves.
