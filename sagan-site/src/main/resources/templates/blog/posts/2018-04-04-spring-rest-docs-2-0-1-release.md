# Spring REST Docs 2.0.1.RELEASE

_Releases | Andy Wilkinson |  April 04, 2018 | 0 Comments_

It's my pleasure to announce that [Spring REST Docs](https://github.com/spring-projects/spring-restdocs) 2.0.1.RELEASE is available from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release). My thanks to everyone who contributed to this release by reporting bugs and opening pull requests.

This maintenance release includes [12 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-restdocs/issues?q=milestone%3A2.0.1.RELEASE). It is a recommend upgrade for all Spring REST Docs 2.x users.
