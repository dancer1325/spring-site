# Spring Roo 1.3.2 RC1 maintenance release now available

_News | Pieter Humphrey |  August 11, 2015 | 0 Comments_

On behalf on the Spring Roo team at DISID Corporation, I am pleased to announce that Spring Roo 1.3.2 RC1 released candidate is out.

Spring Roo is a rapid application development tool for Java, allowing you to create full enterprise applications in just minutes.

Spring Roo jar files have been published to Maven Central. As always, the release is available in the Spring IO repository.

The candidate release fixes the following issues:

-   Error adding InnerType constructor on ITD Generation [ROO-3583](https://jira.spring.io/browse/ROO-3583)
-   JavaBean implementing Interface defining getters and setters [ROO-3584](https://jira.spring.io/browse/ROO-3584)
-   JavaBean implementing Interface and overriding getter causing null pointer [ROO-3585](https://jira.spring.io/browse/ROO-3585)
-   Request loose additional parameters when change language [ROO-3595](https://jira.spring.io/browse/ROO-3595)
-   Add support to generate Generic Methods [ROO-3648](https://jira.spring.io/browse/ROO-3648)
-   Update Selenium to support later versions of Firefox [ROO-3634](https://jira.spring.io/browse/ROO-3634)
-   Add renderLabel attribute in form fields tags [ROO-3639](https://jira.spring.io/browse/ROO-3639)
-   JpaActiveRecord entity implementing interface with getId(): double generation [ROO-3587](https://jira.spring.io/browse/ROO-3587)
-   Update Tomcat7 Maven plugin version [ROO-3652](https://jira.spring.io/browse/ROO-3652)
-   Generate command "selenium all" [ROO-3636](https://jira.spring.io/browse/ROO-3636)
-   Add support to generate Generic Methods [ROO-3648](https://jira.spring.io/browse/ROO-3648)

It is highly recommended that all Spring Roo users upgrade to this version.

For full details on the changes made in the release, please refer to the [Release Note in Jira](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10340&version=15266)

For more project specific information please see the [Site](http://projects.spring.io/spring-roo/) | [GitHub](https://github.com/spring-projects/spring-roo) | [Issues in Jira](https://jira.spring.io/browse/ROO-3655)

Thanks to the awesome users who came up with useful bug reports and suggestions. This is a candidate release, please try out the updates and provide feedback. As always, you'll also find Roo on Twitter - either follow @SpringRoo or just include #SpringRoo in your tweets.

Stay tuned!
