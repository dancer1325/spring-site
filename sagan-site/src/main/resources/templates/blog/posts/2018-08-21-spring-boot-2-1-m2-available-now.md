# Spring Boot 2.1 M2 available now

_Releases | Stéphane Nicoll |  August 21, 2018 | 3 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the second milestone of Spring Boot 2.1 has been released and is available from [our milestone repository](http://repo.spring.io/milestone/). This release closes [75 issues and pull requests](https://github.com/spring-projects/spring-boot/releases/tag/v2.1.0.M2)!

This second milestone aligns with [Spring Framework 5.1.RC2](https://spring.io/blog/2018/08/17/spring-framework-5-1-rc2-available-now) and is the first release with support of Java 11.

Other highlights of this second milestone include:

-   Auto-configuration for task execution (`@EnableAsync`) and task scheduling (`@EnableScheduling`)
-   Configurable bootstrap mode for JPA
-   OAuth2 resource server support
-   Improved logging output for web applications
-   Improved binding for data size properties.

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.1 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.1.0-M2-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.1.0.M2/reference/htmlsingle/).

If you want to get started with 2.1 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).
