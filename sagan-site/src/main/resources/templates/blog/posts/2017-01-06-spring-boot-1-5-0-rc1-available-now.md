# Spring Boot 1.5.0 RC1 Available Now

_Releases | Madhura Bhave |  January 06, 2017 | 9 Comments_

Spring Boot v1.5.0.RC1 is available now from the [Spring milestone repository](http://repo.spring.io/milestone/). This release closes over [220 issues and pull requests](https://github.com/spring-projects/spring-boot/milestone/69?closed=1)! Thanks to everyone that has contributed!

Highlights of the new release include:

-   Out of the box security for actuators
-   Auto-configured `@IntegrationComponentScan`
-   Auto-detection of `spring.jpa.database`
-   Support for Apache Kafka
-   Support for Cloud Foundry and actuator integration
-   Support for embedded in-memory LDAP server and auto-configuration for any compliant LDAP server
-   Upgrades to Spring Data Ingalls, Jetty 9.4, JooQ 3.9, AssertJ 2.6.0, Hikari 2.5 and Neo4J 2.1 and more
-   Upgrades to several Maven plugins

For a complete list of changes, and upgrade instructions, see the [Spring Boot 1.5 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-1.5-Release-Notes) on the WIKI and the updated [reference documentation](http://docs.spring.io/spring-boot/docs/1.5.0.RC1/reference/htmlsingle/).

Please do try `1.5.0.RC1` on your projects and let us know if you find any [problems](https://github.com/spring-projects/spring-boot/issues). Check [the QuickStart instruction](http://projects.spring.io/spring-boot/) to upgrade your project to `1.5.0.RC1` or generate an empty project on [https://start.spring.io](https://start.spring.io) for a concrete example.

Stay tuned for the upcoming `1.5.0.RELEASE` scheduled for the end of the month!
