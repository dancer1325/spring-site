# Spring Security 5.5.1, 5.4.7, 5.3.10 and 5.2.11 released

_Releases | Joe Grandja |  June 22, 2021 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.5.1 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.5.1)), 5.4.7 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.4.7)), 5.3.10 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.10.RELEASE)), and 5.2.11 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.11.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
