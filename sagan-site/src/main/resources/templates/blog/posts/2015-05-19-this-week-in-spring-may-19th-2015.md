# This Week in Spring - May 19th, 2015

_Engineering | Josh Long |  May 19, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover this week so let's get to it!

-   the good Dr. Dave Syer is back it again with this seventh installment on how to better modularize the Angular.js client that talks to a [backend Spring Security, Spring Boot, and Spring Security OAuth-powered service](http://spring.io/blog/2015/05/13/modularizing-the-client-angular-js-and-spring-security-part-vii) and with this eighth installment which looks [at how to test that Angular.js application](http://spring.io/blog/2015/05/19/testing-an-angular-application-angular-js-and-spring-security-part-viii)!
-   Spring Security lead and all around amazing, test-driven dude, Rob Winch just [announced Spring Test HtmlUnit 1.0.0.RC1](http://spring.io/blog/2015/05/14/spring-test-htmlunit-1-0-0-rc1-released) with a lot of new polish.
-   Spring Batch lead Michael Minella [just announced that Spring Batch 3.0.4, a maintenance release, is now available](https://spring.io/blog/2015/05/19/spring-batch-3-0-4-release-is-now-available)
-   Spring Social lead Craig Walls just announced that two maintenance releases, [Spring Social 1.1.2](http://spring.io/blog/2015/05/19/spring-social-1-1-2-released) and [Spring Social 1.1.1](http://spring.io/blog/2015/05/15/spring-social-1-1-1-released), have been released
-   On May 28th, join Pivotal's own Greg Turnquist: [How to Debug and Maintain your Spring Boot App](http://spring.io/blog/2015/05/05/webinar-debug-and-maintain-your-spring-boot-app)
-   Turkcell's Erdem Gunay, and our pal, is giving a webinar on June 9th explaining [how Spring is pretty comprehensively used in Turkcell's personal cloud service](http://spring.io/blog/2015/05/06/webinar-a-comprehensive-spring-showcase-personal-cloud-storage-application).
-   longtime Spring community member [David Welch had a question about setting up a highly available Netflix Eureka installation](http://stackoverflow.com/questions/30288959/eureka-peers-not-synchronized) and Spring Cloud co-founder Spencer Gibb helped him along on this super useful StackOverflow.com post.
-   Fancy using JUnit `@Rule` instances instead of being forced to specify Spring's JUnit `TestRunner` (which effectively removes the ability to use *other* sometimes required `TestRunner` implementations)? Check out the [upcoming Spring framework 4.2 support already in master](https://github.com/spring-projects/spring-framework/commit/d1b1c4f888b5ab5c60cf4beb87f577143bdbebe7)!
-   on May 27th the Seattle Continuous Delivery Meetup [is having a Spring Boot talk](http://www.meetup.com/seattle-continuous-delivery/events/222464656/)! Naturally. Check it out. Spring Boot fits well in a continuous delivery pipeline: it's easy to build isolated, testable, and production-ready applications.
-   Yet another (well done, by the way!) post on how to [setup Spring Boot applications and containerize them with Docker](https://www.jiwhiz.com/blogs/Spring_Boot_Docker_MySQL_Demo). Seems this is a popular topic! [Good thing we have a guide](http://spring.io/guides/gs/spring-boot-docker/)!
-   Spring and Groovy Tool Suite ninja [Kris De Volder announced the STS support for YAML](http://spring.io/blog/2015/05/11/new-in-sts-3-7-0-spring-boot-yaml-editor) last week - have you tried it yet?
-   the *IS Applications Development Services* blog has a nice post on [documenting Spring Boot applications with Swagger](http://www.appsdev.is.ed.ac.uk/blog/?p=501). I like Swagger, but I'd recommend also trying out [Spring REST Docs](https://github.com/spring-projects/spring-restdocs) which are less invasive - they live in the unit-tests - *and* they're Asciidoctor-friendly.
-   Wow, were are finally out of SpringOne2GX replays! This is the last batch, mostly Groovy and Grails. First, Baruch Sadogursky from JFrog talks about [Grails Plugin Testing Strategies](http://spring.io/blog/2015/05/19/springone2gx-2014-replay-grails-plugin-testing-strategies). Then David Dawson, from Simplicity Itself talks about [Philosophical Architecture in Grails](http://spring.io/blog/2015/05/19/springone2gx-2014-replay-philosophical-architecture-in-grails). Also, Ken Kousen [Groovy Vampires: Combining Groovy, REST, NoSQL, and more](http://spring.io/blog/2015/05/19/springone2gx-2014-replay-groovy-vampires-combining-groovy-rest-nosql-and-more)
