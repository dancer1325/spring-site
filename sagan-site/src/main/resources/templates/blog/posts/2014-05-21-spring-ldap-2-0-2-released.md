# Spring LDAP 2.0.2 Released

_Engineering | Rob Winch |  May 21, 2014 | 0 Comments_

I'm pleased to announce the release of Spring LDAP 2.0.2.RELEASE. The highlights of this release include:

-   [LDAP-300](https://jira.spring.io/browse/LDAP-300) - Fix potential NullPointerException when using Spring LDAP in a uber jar
-   [LDAP-299](https://jira.spring.io/browse/LDAP-299) - Deprecation of spring-ldap-ldif-batch module in favor of using classes within Spring Batch

For additional information on the release, refer to the [changelog](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10071&version=14505).
