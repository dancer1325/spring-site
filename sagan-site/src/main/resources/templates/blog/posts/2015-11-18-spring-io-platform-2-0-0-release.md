# Spring IO Platform 2.0.0.RELEASE

_Releases | Andy Wilkinson |  November 18, 2015 | 3 Comments_

Spring IO Platform 2.0.0.RELEASE is now available from the [Spring release repository](https://repo.spring.io/release) and Maven Central.

Additions:

-   Spring REST Docs 1.0

Major upgrades:

-   Reactor 2.0
-   Spring Security 4.0

Minor upgrades:

-   Spring AMQP 1.5
-   Spring Boot 1.3
-   Spring Cloud Connectors 1.2
-   Spring Data Gosling
-   Spring Framework 4.2
-   Spring HATEOAS 0.19
-   Spring Integration 4.2

A number of other Spring projects and their dependencies have also been updated to their latest maintenance releases.

Please note that a number of dependencies that were previously part of the Platform have been removed or replaced in this release. Please refer to [the documentation](http://docs.spring.io/platform/docs/2.0.0.RELEASE/reference/htmlsingle/#upgrading-dependency-management) for further details.
