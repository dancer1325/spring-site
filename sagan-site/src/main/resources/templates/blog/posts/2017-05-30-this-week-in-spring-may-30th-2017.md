# This Week in Spring  - May 30th, 2017

_Engineering | Josh Long |  May 30, 2017 | 1 Comment_

Hi Spring fans! This week I'm in Chicago for the epic [Spring Days Chicago](http://springdays.io/) and then I'm off to Singapore for [VOXXED Singapore](https://voxxeddays.com/singapore/). We've got a lot to cover so let's get to it!

-   Spring Security ninja Joe Grandja has just announced [the new Spring Security OAuth 2.1.1 and 2.0.4](https://spring.io/blog/2017/05/29/spring-security-oauth-2-1-1-and-2-0-14-released)
-   Spring Cloud Data Flow engineer Gunnar Hillert just [announced Spring Cloud Data Flow 1.2.1](https://spring.io/blog/2017/05/25/spring-cloud-data-flow-1-2-1-released)
-   Spring Cloud co-founder Spencer Gibb [just announced Spring Cloud Dalston SR1](https://spring.io/blog/2017/05/24/spring-cloud-dalston-sr1)
-   I liked Jonas Hecht's post looking at how [to run Spring Boot on Docker Windows](https://blog.codecentric.de/en/2017/05/ansible-docker-windows-containers-scaling-spring-cloud-netflix-docker-compose/)
-   I loved this post from RedHat engineer Sébastien Blanc on [using Keycloak, their OAuth-aware identity provider, with Spring Boot applications](https://developers.redhat.com/blog/2017/05/25/easily-secure-your-spring-boot-applications-with-keycloak/). Keycloak is also supported on [the Spring Initializr](http://start.spring.io).
-   The Docker blog also has a nice post by Sophia Parafina on [running a Spring Boot application in a Docker container](https://blog.docker.com/2017/05/spring-boot-development-docker/)
-   The Pivotal engineering blog has a nice post by Ian Fisher on [the must-know Spring Boot annotations](http://engineering.pivotal.io/post/must-know-spring-boot-annotations-controllers/)
-   the IBM DeveloperWorks blog has a [nice post on building Spring Boot applications](https://developer.ibm.com/dwblog/2017/spring-boot/)
-   I really liked this post by Aboullaite Mohammed on [using the Spring Boot Actuator](https://aboullaite.me/an-introduction-to-spring-boot-actuator/)
-   the *MemoryNotFound* blog has a nice post on using [the `@ConfigurationProperties` annotation](http://memorynotfound.com/spring-boot-configurationproperties-annotation-example/) in a Spring Boot application.
-   Grygoriy Gonchar looks at the implications for doing reactive JDBC (even if it's just a facade around fundamentally blocking APIs) [with Spring 5's Reactor support](https://dzone.com/articles/spring-5-webflux-and-jdbc-to-block-or-not-to-block).
-   The *All and Sundry* blog [has a nice look at writing a ratio-aware proxy using Spring Cloud Netflix Zuul to ease the migration to a new service](http://www.java-allandsundry.com/2017/05/ratio-based-routing-to-legacy-and.html)
