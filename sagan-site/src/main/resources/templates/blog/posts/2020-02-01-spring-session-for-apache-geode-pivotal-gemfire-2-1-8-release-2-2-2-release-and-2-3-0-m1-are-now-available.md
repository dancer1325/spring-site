# Spring Session for Apache Geode & Pivotal GemFire 2.1.8.RELEASE, 2.2.2.RELEASE and 2.3.0.M1 are now available

_Releases | John Blum |  February 01, 2020 | 0 Comments_

On behalf the *Spring*, *Apache Geode* and *Pivotal GemFire* communities, it is my pleasure to announce the release of *Spring Session for Apache Geode and Pivotal GemFire* (SSDG) `2.1.8.RELEASE`, `2.2.2.RELEASE` as well as the `2.3.0.M1` release.

It should also be noted that SSDG works with [Pivotal Cloud Cache](https://pivotal.io/pivotal-cloud-cache) as well. Pivotal Cloud Cache is powered by Apache Geode. [Spring Boot for Apache Geode](https://github.com/spring-projects/spring-boot-data-geode#spring-boot-for-apache-geode%E2%80%94%E2%80%8Bpivotal-gemfire) (SBDG) makes configuring and using Spring Session for Pivotal Cloud Cache a simple matter. See [here](https://docs.spring.io/spring-boot-data-geode-build/1.2.x/reference/html5/#geode-session) and [here](https://docs.spring.io/spring-boot-data-geode-build/1.2.x/reference/html5/#geode-session-pcc) for more details.

SSDG `2.1.8.RELEASE` builds on the latest Spring Framework 5.1.13.RELEASE, Spring Data Lovelace-SR15 and Spring Session Bean-SR8 (with Spring Session core 2.1.9.RELEASE) releases along with Apache Geode 1.6.0 and Pivotal GemFire 9.5.4.

SSDG `2.2.2.RELEASE` builds on the latest Spring Framework 5.2.3.RELEASE, Spring Data Moore-SR4 and Spring Session Corn-RELEASE (with Spring Session core 2.2.0.RELEASE) releases along with Apache Geode 1.9.2 and Pivotal GemFire 9.8.4.

SSDG `2.3.0.M1` builds on the latest Spring Framework 5.2.3.RELEASE, Spring Data Neumann-M2 and Spring Session Corn-RELEASE (with Spring Session core 2.2.0.RELEASE) along with Apache Geode 1.11.0 and Pivotal GemFire 9.9.1.

Both SSDG `2.1.8.RELEASE` and `2.2.2.RELEASE` are available in [Maven Central](https://search.maven.org/artifact/org.springframework.session/spring-session-data-geode) while SSDG `2.3.0.M1` is available in the Spring [libs-milestone](https://repo.spring.io/libs-milestone/org/springframework/session/spring-session-data-geode/2.3.0.M1/) Repository.
