# Spring Framework 4.1.3 released

_Releases | Stéphane Nicoll |  December 09, 2014 | 0 Comments_

On behalf of the team I am pleased to announce the immediate availability of Spring Framework 4.1.3.

Spring Framework 4.1.3 is our third maintenance release in the 4.1.x line and comes with [over 50 fixes and improvements](https://jira.spring.io/issues/?jql=fixVersion%3D4.1.3%20AND%20project%3DSPR). It was initially scheduled for late December but we decided to release it early to incorporate user-suggested and user-contributed improvements right in time for the scheduled Spring Boot 1.2 release this week.

We are still planning on another release later this month (4.1.4) at which point 4.1.x will turn into a maintenance branch, with active development on 4.2 happening on master.
