# Spring for Apache Kafka 2.1.3, Spring Integration Kafka 3.0.2 are Available

_Releases | Gary Russell |  February 22, 2018 | 0 Comments_

I am pleased to announce that Spring for Apache Kafka 2.1.3.RELEASE is available.

In addition, the Spring Integration extension `spring-integration-kafka` 3.0.2.RELEASE is also available.

While these are point releases in the 2.1.x (and 3.0.x) lines; they introduce some important new features:

### [](#spring-for-apache-kafka)[](#spring-for-apache-kafka)Spring for Apache Kafka

[Change Log](https://github.com/spring-projects/spring-kafka/releases/tag/v2.1.3.RELEASE)

-   Request/Reply messaging with the `ReplyingKafkaTemplate`
    
-   pause/resume on listener containers
    
-   default `@KafkaHandler` method for multi-method `@KafkaListener`
    
-   `ChainedKafkaTransactionManager` to improve transaction synchronization scenarios
    

### [](#spring-integration-kafka)[](#spring-integration-kafka)Spring Integration Kafka

[Change Log](https://github.com/spring-projects/spring-integration-kafka/releases/tag/v3.0.2.RELEASE)

-   Spring Integration gateways for request/reply scenarios
    
-   Support for transaction initiation in outbound endpoints
    

For more information see the What’s New section of the [reference manual](https://docs.spring.io/spring-kafka/reference/html/).
