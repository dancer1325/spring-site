# Spring Security 5.0.0.RELEASE Released

_Releases | Rob Winch |  November 28, 2017 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.0.0.RELEASE. This release resolves 400+ tickets. For highlights and details about the release, refer to the [What’s New in Spring Security 5.0](https://docs.spring.io/spring-security/site/docs/5.0.0.RELEASE/reference/htmlsingle/#new) section.

We hope to see you at [SpringOne Platform](https://springoneplatform.io) next week. It will be packed with many Spring talks, opportunities to learn about the latest and greatest features and of course some previews about what we’re planning to do next.
