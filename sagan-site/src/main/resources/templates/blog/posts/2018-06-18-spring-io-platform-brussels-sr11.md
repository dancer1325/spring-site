# Spring IO Platform Brussels SR11

_Releases | Andy Wilkinson |  June 18, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR11 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.8
-   Spring Boot 1.5.14
-   Spring Cloud Connectors 1.2.6
-   Spring Data Ingalls SR13
-   Spring Framework 4.3.18
-   Spring Integration 4.3.17
-   Spring Security 4.2.7
-   Spring Session 1.3.3

The versions of a number of third-party dependencies have also been updated.
