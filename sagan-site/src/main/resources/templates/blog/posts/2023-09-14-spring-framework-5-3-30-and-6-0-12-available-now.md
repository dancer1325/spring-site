# Spring Framework 5.3.30 and 6.0.12 available now

_Releases | Brian Clozel |  September 14, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.30` and `6.0.12` are available now.

Spring Framework `5.3.30` ships with [20 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.30), including [5 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A5.3.30+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 2.7.16, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.0.12` ships with [75 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.12), including [7 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.0.12+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.0.11 and 3.1.4, [to be released next week](https://calendar.spring.io/).

We expect the next set of Spring Framework maintenance releases to ship next November, along with the new and exciting 6.1.0 we have been working on the last few months.
