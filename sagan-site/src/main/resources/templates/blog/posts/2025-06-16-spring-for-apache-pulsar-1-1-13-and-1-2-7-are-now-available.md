# Spring for Apache Pulsar 1.1.13 and 1.2.7 are now available

_Releases | Chris Bono |  June 16, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.1.13` and `1.2.7` have been released and are now available from Maven Central.

With the `1.1.13` release we have reached its Open Source End of Life. Please consider upgrading to the latest `1.2.x` version at your earliest convenience.

The `1.1.13` release will be included in the upcoming Spring Boot [3.3.13](https://github.com/spring-projects/spring-boot/milestones/3.3.13) release. The `1.2.7` release will be included in the upcoming Spring Boot [3.4.7](https://github.com/spring-projects/spring-boot/milestones/3.4.7) and [3.5.1](https://github.com/spring-projects/spring-boot/milestones/3.5.1) releases.

Please see the release notes ([1.1.13](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.13) and [1.2.7](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.7)) for more details.
