# Announcing Spring Cloud Function - 2.0.0.RC2

_Engineering | Oleg Zhurakousky |  November 19, 2018 | 1 Comment_

We are pleased to announce the second Release Candidate of the Spring Cloud Function - 2.0.0.RC2.

Individual modules of Spring Cloud Function 2.0.0.RC2 are available for use in the [Spring Milestone](http://repo.spring.io/libs-milestone-local/org/springframework/cloud/) repository.

As a follow up to 2.0.0.RC1, this release primarily encompasses minor enhancements and bug fixes. As it stands now this should be the final Release Candidate before General Availability release 2.0.0.RELEASE in several weeks.

If you're interested in helping out, check out the ["ideal for contribution" tag](https://github.com/spring-cloud/spring-cloud-function/labels/ideal-for-contribution) in the issue repository.
