# Spring Boot 4.0.0 available now

_Releases | Phil Webb |  November 20, 2025 | 16 Comments_

On behalf of the team and everyone who has contributed, I'm extremely happy to announce that Spring Boot `4.0.0` has been released and is now available from Maven Central.

This release is the beginning of a new Spring Boot generation providing solid foundations for coming years. Building on top of [Spring Framework 7](https://spring.io/blog/2025/11/13/spring-framework-7-0-general-availability), there are a number of exciting new features in this release, including:

-   A complete [modularization of the Spring Boot codebase](https://spring.io/blog/2025/10/28/modularizing-spring-boot) providing smaller and more focused jars.
-   Portfolio-wide [improvements for null safety](https://spring.io/blog/2025/11/12/null-safe-applications-with-spring-boot-4) with JSpecify.
-   First class support for Java 25 (whilst retaining Java 17 compatibility).
-   Support for [API Versioning](https://spring.io/blog/2025/09/16/api-versioning-in-spring) and [HTTP Service Clients](https://spring.io/blog/2025/09/23/http-service-client-enhancements) for REST based applications.

For a complete list of new feature, please see [the release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Release-Notes).

Since this is a major release of Spring Boot, upgrading existing applications can be a little more involved that usual so you'll also want to check out the [dedicated migration guide](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Migration-Guide).

Thanks to all those who have contributed with issue reports and pull requests for this release. Especially those that have tried the milestones and release candidates. Your feedback has been invaluable!
