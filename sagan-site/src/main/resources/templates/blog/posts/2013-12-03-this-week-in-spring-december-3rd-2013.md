# This Week in Spring - December 3rd, 2013

_Engineering | Josh Long |  December 03, 2013 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in Sofia, Bulgaria, talking to the great folks at the [amazing Java2Days conference](http://java2days.com/). This is my *5th* year here and it's been an amazing show every time!

1.  [Spring Integration 3.0 RC1](https://spring.io/blog/2013/11/27/spring-integration-3-0-release-candidate-1-is-now-available) is now available! The new release features many nice, new features including a file-tailing inbound adapter, a JMX inbound adapter, redis queues, and much more.
2.  Don't miss Matt Stine as he discusses [Spring with Immutability](https://spring.io/blog/2013/11/13/webinar-spring-with-immutability) on Dec 10th!
3.  The replay of my webinar from last week, [*Have You Seen Spring Lately?*](https://spring.io/blog/2013/12/02/webinar-replay-have-you-seen-spring-lately), is now available online! The talk is a sort of introduction to the Spring IO platform, introducing many of the awesome technology released since the Spring team became part of Pivotal. It also introduces some new tech inside of Spring 4, as well as some oldies-but-goodies like Spring Social and Spring Batch, which continue to grow.
4.  Ben Hale's epic talk from SpringOne2GX 2013 which introduces [REST API Design](https://spring.io/blog/2013/12/02/springone2gx-2013-replay-rest-ful-api-design) is now available online.
5.  Looking to get started quickly with Spring and application development? Check out this webinar replay of the SpringOne2GX 2013 talk introducing [Spring Boot](https://spring.io/blog/2013/12/03/springone2gx-2013-replay-zero-effort-spring)!
6.  InfoQ has a great post on how to use [Spring Boot in conjunction with the Aerospike database](http://www.infoq.com/articles/rest-webservice-spring-boot-aerospike)
7.  Want to integrate a custom single-sign on authentication provider using Spring Security? This [Stack Overflow discussion on integrating a custom single sign-on provider](http://stackoverflow.com/questions/5365054/integrate-single-sign-on-using-spring-security) is very helpful! (I got this from the always-useful [@SpringTip Twitter account](http://twitter.com/SpringTip))
8.  Xavier Padró's put together a nice post on [how Spring Integration error handling works](http://xpadro.blogspot.com.es/2013/11/how-error-handling-works-in-spring.html).
9.  [Sergi Almar (@SergiAlmar)](http://twitter.com/sergialmar) has put together a really great presentation on [using Spring 4's websocket support](http://www.slideshare.net/sergialmar/websockets-with-spring-4). Definitely check it out!
10.  Also from @SpringTip, a great Stack Overflow post on how to filter database records using [Spring Data REST and Spring MVC](http://stackoverflow.com/questions/20280708/filtering-database-rows-with-spring-data-jpa-and-spring-mvc/20283256#20283256)
11.  Nicolas Frankel is back at it again, this time with [a nice post on using Spring Data SOLR](http://blog.frankel.ch/solr-as-a-spring-data-module)
