# Announcing Spring Cloud Stream Applications 2021.1.1 GA Release

_Releases | Chris Bono |  March 07, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring Cloud Stream Applications `2021.1.1` has been released.

This release is based on Spring Boot `2.6.3` and Spring Cloud `2021.0.1` and contains many improvements and bug fixes - check out the [release notes](https://github.com/spring-cloud/stream-applications/releases/tag/v2021.1.1) for more information.

## [](#stay-in-touch)Stay in touch...

Please continue to engage with us by creating bug reports and feature requests on [Github](https://github.com/spring-cloud/stream-applications) and asking questions on StackOverflow using the [spring-cloud-stream](https://stackoverflow.com/questions/tagged/spring-cloud-stream) or [spring-cloud-dataflow](https://stackoverflow.com/questions/tagged/spring-cloud-dataflow) tags.
