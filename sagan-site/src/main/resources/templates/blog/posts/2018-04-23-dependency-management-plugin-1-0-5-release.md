# Dependency Management Plugin 1.0.5.RELEASE

_Releases | Andy Wilkinson |  April 23, 2018 | 6 Comments_

I am pleased to announce that [Gradle dependency management plugin](https://github.com/spring-gradle-plugins/dependency-management-plugin) 1.0.5.RELEASE is now available from Maven Central, Bintray, and the [Gradle Plugin Portal](https://plugins.gradle.org/plugin/io.spring.dependency-management). This maintenance release fixes a [handful of issues](https://github.com/spring-gradle-plugins/dependency-management-plugin/milestone/25?closed=1). It was included in the recent Spring Boot 1.5.12 and 2.0.1 releases and is also a recommended upgrade for all other users of the plugin.
