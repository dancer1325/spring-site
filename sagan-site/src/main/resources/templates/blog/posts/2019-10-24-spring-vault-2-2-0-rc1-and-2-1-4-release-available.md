# Spring Vault 2.2.0.RC1 and 2.1.4.RELEASE available

_Releases | Mark Paluch |  October 24, 2019 | 0 Comments_

On behalf of the community, I’d like to announce the availability of Spring Vault releases `2.2.0.RC1` and `2.1.4.RELEASE`.

Both releases ship with mostly bugfixes and dependency upgrades.

For a complete list of changes see the [2.2.0.RC1 changelog](http://docs.spring.io/spring-vault/docs/2.2.0.RC1/changelog.txt) respective [2.1.4.RELEASE changelog](http://docs.spring.io/spring-vault/docs/2.1.4.RELEASE/changelog.txt).
