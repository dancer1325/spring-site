# Spring Security 5.8.0-M2 is released

_Releases | Marcus Hert Da Coregio |  August 16, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security `5.8.0-M2` is available now.

This release includes dependency upgrades, bug fixes, and enhancements. Here are a few noteworthy changes:

-   [Hash-based Content-Security-Policy for SAML post pages](https://github.com/spring-projects/spring-security/issues/11631)
    
-   [Customization of redirect strategy](https://github.com/spring-projects/spring-security/pull/11387)
    

See the [5.8.0-M2](https://github.com/spring-projects/spring-security/releases/tag/5.8.0-M2) release notes for more details.
