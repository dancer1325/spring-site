# This Week in Spring - September 27, 2016

_Engineering | Josh Long |  September 27, 2016 | 0 Comments_

Wow! It's already almost October! This year has sure gone by quickly!

Welcome to another installment of *This Week in Spring*! This week I'm in the studio with [Phil Webb](https://twitter.com/phillip_webb) recording the next installment of our [Livelessons series on building Microservices with Spring Boot](https://www.safaribooksonline.com/library/view/building-microservices-with/9780134192468/). It's been a lot of fun so far and we've only just begun!

Tonight, I hope you'll join me for [Cloud Native Java](https://virtualjug.com/vjug24-session-cloud-native-java-by-josh-long/) on the vJUG's worldwide and live event going on in timezones all around the world!

We've got *so much* to cover, so let's get to it!

-   Spring lead Juergen Hoeller just announced Spring Framework 5.0.M2, another major step on the journey to Spring 5! [This release includes the functional endpoint mechanism for Spring MVC, as well](https://spring.io/blog/2016/09/21/spring-framework-5-0-m2-released)!
-   Hurray! Reactor 3.0 is out! Get the latest bits and [see why Reactor underpins Spring Framework 5's Reactive Runtime](https://spring.io/blog/2016/09/27/ready-your-java-8-reactive-apps-now-reactor-3-0-ga-is-out)
-   Check out Spring reactive ninja Arjen Poutsma's new look at the [functional endpoint mechanism in Spring Framework 5](https://spring.io/blog/2016/09/22/new-in-spring-5-functional-web-framework)
-   Spring Cloud ninja Spencer Gibb [has just announced Spring Cloud Camden.RELEASE and Spring Cloud Brixton.SR6](https://spring.io/blog/2016/09/26/spring-cloud-camden-release-and-brixton-sr6-are-available)! Spring Cloud Camden is *packed* with good stuff, including the new Spring Cloud Contract effort! Get it while it's hot!
-   Spring Integration Java DSL lead Artem Bilan just [announced that the Spring Integration Java DSL 1.2 RC1 is now available](https://spring.io/blog/2016/09/27/java-dsl-for-spring-integration-1-2-release-candidate-1-is-available)
-   check out this SpringOne Platform replay of Rossen Stoyanchev's [excellent look at Spring MVC in 4.3](https://spring.io/blog/2016/09/26/springone-platform-2016-replay-spring-mvc-4-web-apps)
-   check out this SpringOne Platform replay of a look [at Spring Cloud Services from Craig Walls and Roy Clarkson](https://spring.io/blog/2016/09/26/springone-platform-2016-replay-cloud-native-java-with-spring-cloud-services)
-   Spring and data ninja Thomas Risberg's [just announced Spring Cloud Data Flow for Apache Mesos 1.0.GA](https://spring.io/blog/2016/09/26/spring-cloud-data-flow-for-apache-mesos-1-0-ga-released)
-   Spring Cloud Contract lead Marcin Grzejszczak announced the [1.0.0.RELEASE of Spring Cloud Contract in the runup to Spring Cloud Camden itself](https://spring.io/blog/2016/09/23/spring-cloud-contract-1-0-0-release-is-available). Spring Cloud Contract is a *must-see* addition to the Spring Cloud portfolio that makes it easy for service providers and service consumers to stay on the same page and to build (and test!) with confidence that the implied contract between nodes is unbroken.
-   Spring Security lead Rob Winch has [just announced Spring Security 4.2.M1](https://spring.io/blog/2016/09/23/spring-security-4-2-m1), which closes out 50+ issues and has a lot of other improvements
-   in the run up to Spring Cloud Camden, Marius Bogoevici just [announced Spring Cloud Stream Brooklyn, which makes it easy to build messaging-based microservcies](https://spring.io/blog/2016/09/23/spring-cloud-stream-brooklyn-release-is-available)
-   Spring Data ninja Janne Valkealahti just [announced Spring Cloud Data Flow for Apache YARN 1.0.2](https://spring.io/blog/2016/09/22/spring-cloud-data-flow-for-apache-yarn-1-0-2-released)
-   Spring Platform lead Andy Wilkinson's [just announced Spring IO Platform Athens](https://spring.io/blog/2016/09/22/spring-io-platform-athens-release) and [Spring IO Platform 2.0.8](https://spring.io/blog/2016/09/22/spring-io-platform-2-0-8-release)
-   Spring Boot ninja Stéphane Nicoll [just announced Spring Boot 1.4.1. and 1.3.8](https://spring.io/blog/2016/09/21/spring-boot-1-4-1-and-1-3-8-available-now)
-   Atomist founder and Spring founder Rod Johnson has an interesting post up on [how his company's technology can be used to automate code edits](\(https://medium.com/the-composition/software-that-writes-and-evolves-software-953578a6fc36#.1fb657dhh\)) to facilitate upgrading codebases. His example, of course, is in terms of Spring Cloud and - in particular - Hystrix.
