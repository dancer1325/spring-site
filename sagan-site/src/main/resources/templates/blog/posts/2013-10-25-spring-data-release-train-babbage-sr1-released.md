# Spring Data Release Train Babbage - SR1 released

_News | Oliver Drotbohm |  October 25, 2013 | 0 Comments_

The Spring Data team has just released the first service release for the Babbage release train. It includes the following modules:

-   Spring Data Commons 1.6.2 - [Changelog](http://docs.spring.io/spring-data/commons/docs/1.6.2.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/commons/docs/1.6.2.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-commons%7C1.6.2.RELEASE%7Cjar)
-   Spring Data JPA 1.4.2 - [Changelog](http://docs.spring.io/spring-data/jpa/docs/1.4.2.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/jpa/docs/1.4.2.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-jpa%7C1.4.2.RELEASE%7Cjar)
-   Spring Data MongoDB 1.3.2 - [Changelog](http://docs.spring.io/spring-data/mongodb/docs/1.3.2.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/mongodb/docs/1.3.2.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-mongodb%7C1.3.2.RELEASE%7Cjar)
-   Spring Data Neo4j 2.3.2 - [Changelog](http://docs.spring.io/spring-data/neo4j/docs/2.3.2.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/neo4j/docs/2.3.2.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-neo4j%7C2.3.2.RELEASE%7Cjar)

The release bundles a bunch of important enhancements and bug fixes and is a recommended upgrade. You can find all issues fixed in this release in our [JIRA](https://jira.springsource.org/issues/?filter=14390).

The next step will be a first milestone of the Codd release train with a few great new features, especially for the MongoDB module. More details about that [here](https://github.com/spring-projects/spring-data-commons/wiki/Release-Train-Codd).
