# Spring IO Platform 2.0.6.RELEASE

_Releases | Andy Wilkinson |  July 08, 2016 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 2.0.6.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Boot 1.3.6.RELEASE
-   Spring Framework 4.2.7.RELEASE
-   Spring Integration 4.2.8.RELEASE
-   Spring REST Docs 1.0.1.RELEASE
-   Spring Retry 1.1.3.RELEASE
-   Spring Security OAuth 2.0.10.RELEASE
