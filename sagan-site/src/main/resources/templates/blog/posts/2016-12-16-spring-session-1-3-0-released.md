# Spring Session 1.3.0 Released

_Releases | Rob Winch |  December 16, 2016 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Session 1.3.0.RELEASE. This release evolved through [1.3.0.M1](https://github.com/spring-projects/spring-session/milestone/6?closed=1), [1.3.0.M2](https://github.com/spring-projects/spring-session/milestone/18?closed=1), [1.3.0.RC1](https://github.com/spring-projects/spring-session/milestone/16?closed=1), and [1.3.0.RELEASE](https://github.com/spring-projects/spring-session/milestone/19?closed=1)

# [](#whats-new-in-spring-session-130release)[](#what-s-new-in-spring-session-1-3-0-release)What’s New in Spring Session 1.3.0.RELEASE

You can find highlights of what’s new in the [What’s New in Spring Session 1.3.0.RELEASE](http://docs.spring.io/spring-session/docs/1.3.0.RELEASE/reference/html5/#what-s-new-in-1-3) section of the reference. For details refer to the changelog links above.

# [](#contributions)[](#contributions)Contributions

Without the community we couldn’t be the successful project we are today. I’d like to thank everyone that created issues & provided feedback.
