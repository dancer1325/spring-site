# Spring IDE 1.3 Released

_Releases | Torsten Juergeleit |  April 25, 2006 | 0 Comments_

Dear Spring Community,  
  
We are pleased to announce that Spring IDE 1.3 has been released.  
  
This release of Spring IDE provides some [new features](http://springide.org/project/wiki/Springide13NewFeatures) and a bunch of bugfixes. For a complete list visit [](http://springide.org/project/milestone/Release%201.3)[http://springide.org/project/milestone/Release%201.3](http://springide.org/project/milestone/Release%201.3)

Changes:

-   **Now Eclipse 3.1 and [Web Tools](http://www.eclipse.org/webtools/) are required!!!**
-   Updated to Spring 1.2.7
-   Updated version of [BeansXmlEditor](http://springide.org/project/wiki/BeansXmlEditor)
-   Support for [searching](http://springide.org/project/wiki/BeansSearch) the [BeansCoreModel](http://springide.org/project/wiki/BeansCoreModel)
-   Open Spring Bean dialog (similar to JDT's "Open Type" dialog)
-   [Wizard](http://springide.org/project/wiki/SpringProjectWizard) for creating a new [SpringProject](http://springide.org/project/wiki/SpringProject)
-   [BeansView](http://springide.org/project/wiki/BeansView) can be linked to [BeansXmlEditor](http://springide.org/project/wiki/BeansXmlEditor)
-   Now any filename extensions for [BeansConfig](http://springide.org/project/wiki/BeansConfig) files are supported

More details are available from the project's site [http://springide.org/](http://springide.org/)  
  
Use Eclipse's update manager to [install](http://springide.org/project/wiki/SpringideInstall) this this release from the project's update site [http://springide.org/updatesite/](http://springide.org/updatesite/)  
  
For bug reports tickets can be opened via [http://springide.org/project/newticket](http://springide.org/project/newticket)
