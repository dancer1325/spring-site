# Spring Batch 2.2.6.RELEASE is now available

_Releases | Michael Minella |  April 11, 2014 | 0 Comments_

We are pleased to announce that Spring Batch 2.2.6.RELEASE is now available via Maven Central, Github and the SpringSource download repository. This is the 6th maintenance release for the 2.2.x branch of Spring Batch and addresses a number of bugs, a few minor enhancements, and documentation updates. Many thanks to all of those who submitted the many [pull requests](https://github.com/springsource/spring-batch/pulls) that went into this release.  
  
[Spring Batch Home](http://spring.io/projects/spring-batch) | [Source on GitHub](https://github.com/spring-projects/spring-batch) | [Reference Documentation](http://docs.spring.io/spring-batch/)  
  
We look forward to your feedback.
