# Spring IO Platform Brussels SR7

_Releases | Andy Wilkinson |  January 31, 2018 | 8 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR7 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.6
-   Spring Boot 1.5.10
-   Spring Data Ingalls-SR10
-   Spring Framework 4.3.14
-   Spring Integration 4.3.14
-   Spring Retry 1.2.2
-   Spring Security 4.2.4
-   Spring Web Flow 2.4.7

The versions of a number of third-party dependencies have also been updated.
