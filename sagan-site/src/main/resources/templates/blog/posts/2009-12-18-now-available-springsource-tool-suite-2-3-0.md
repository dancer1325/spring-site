# Now Available: SpringSource Tool Suite 2.3.0

_Releases | Adam Fitzgerald |  December 18, 2009 | 0 Comments_

To assist all developers with their upgrade to [Spring 3.0 GA](http://blog.springsource.com/2009/12/16/spring-framework-3-0-goes-ga/), [SpringSource](http://www.springsource.com) has released an update to [SpringSource Tool Suite](http://www.springsource.com/products/sts) (STS), the best Eclipse-powered development environment for building Spring, Groovy and Grails powered enterprise applications. The new version (2.3.0) is now available for [download](http://www.springsource.com/products/springsource-tool-suite-download) and includes:

-   Comprehensive updates to support all new Spring 3.0.0.RELEASE features.
-   New support for [Spring Integration](/spring-integration), [Spring Web Services](http://static.springsource.org/spring-ws/sites/1.5/) and [Spring BlazeDS/Flex](/spring-flex) namespaces.
-   Updates to support the recent [Spring Roo](/roo) 1.0.0.RC4.
-   Improved [Grails](http://www.grails.org) tooling, including support for the latest Grails 1.2 snapshot.
