# Spring Boot 1.3.3 and 1.4.0.M1 available now

_Releases | Stéphane Nicoll |  February 26, 2016 | 0 Comments_

On behalf of the team, it is my pleasure to announce that Spring Boot 1.3.3 and 1.4.0.M1 have been released today. Spring Boot 1.3.3 includes [60 fixes and a selection of improvements and dependency upgrades](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.3). This release is a recommended upgrade for all users.

Spring Boot 1.4.0.M1 is available in the [Spring milestone repository](http://repo.springsource.org/milestone) and provides a number of [improvements and new features](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.4.0.M1). For a complete list of changes, and for upgrade instructions, see the [Spring Boot 1.4 Release Notes](http://github.com/spring-projects/spring-boot/wiki/Spring-Boot-1.4-Release-Notes) on the WIKI. The [reference documentation](http://docs.spring.io/spring-boot/docs/1.4.0.M1/reference/htmlsingle/) has also been updated.

Thanks again to everyone that has contributed to the release! Please give it a go and report any problems using [the project Issue tracker](http://github.com/spring-projects/spring-boot/issues) page.
