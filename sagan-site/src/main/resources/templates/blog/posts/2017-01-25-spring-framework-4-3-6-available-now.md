# Spring Framework 4.3.6 available now

_Releases | Stéphane Nicoll |  January 25, 2017 | 0 Comments_

On behalf of the team, I am happy to announce that Spring Framework [4.3.6](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15811) [is available now](http://projects.spring.io/spring-framework).

This maintenance release is the base for the upcoming Spring Boot 1.4.4 and 1.5.0 releases and includes over 60 fixes and selected improvements.
