# Spring Native 0.12.1 available now

_Releases | Sébastien Deleuze |  July 13, 2022 | 3 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Native `0.12.1` has been released and is now available from [https://repo.spring.io/release](https://repo.spring.io/release).

This release includes [15 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects-experimental/spring-native/releases/tag/0.12.1) to Spring Boot 2.7.1 and Native Build Tools 0.9.13. Thanks to all those who have contributed with issue reports and pull requests.

[GitHub](https://github.com/spring-projects-experimental/spring-native) | [Issues](https://github.com/spring-projects-experimental/spring-native/issues) | [Documentation](https://docs.spring.io/spring-native/docs/0.12.1/reference/htmlsingle/)
