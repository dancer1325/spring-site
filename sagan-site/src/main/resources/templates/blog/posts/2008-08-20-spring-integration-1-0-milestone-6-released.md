# Spring Integration 1.0 Milestone 6 Released

_Releases | Mark Fisher |  August 20, 2008 | 0 Comments_

Dear Spring community,  
We are pleased to announce that Spring Integration 1.0.0.M6 is now available.  
[Download](http://static.springframework.org/downloads/nightly/milestone-download.php?project=INT) | [Reference Documentation](http://static.springframework.org/spring-integration/reference/htmlsingle/spring-integration-reference.html) | [JavaDoc](http://static.springframework.org/spring-integration/apidocs/)  
  
To see a list of features added and issues resolved since Milestone 5, view the [changelog](http://static.springframework.org/spring-integration/changelog.txt). For more information, visit the [Spring Integration Home](/spring-integration) or join the discussion at the [Spring Integration Forum](http://forum.springframework.org/forumdisplay.php?f=42). Community feedback is more important than ever as we approach the Release Candidate.

Spring Integration Lead
