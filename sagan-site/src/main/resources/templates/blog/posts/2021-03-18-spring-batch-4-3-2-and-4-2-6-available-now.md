# Spring Batch 4.3.2 and 4.2.6 available now

_Releases | Mahmoud Ben Hassine |  March 18, 2021 | 0 Comments_

I am pleased to announce that Spring Batch 4.3.2 and 4.2.6 have been released and are available now from Maven Central repository. These releases include a number of bug fixes and improvements. You can find the change logs here: [4.2.6](https://github.com/spring-projects/spring-batch/releases/tag/4.2.6.RELEASE) and [4.3.2](https://github.com/spring-projects/spring-batch/releases/tag/4.3.2).

I would like to thank all contributors who helped in making these releases possible!
