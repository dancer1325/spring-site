# Spring Cloud Data Flow 2.8.4 and 2.9.1 Released

_Releases | Janne Valkealahti |  November 05, 2021 | 0 Comments_

Spring Cloud Data Flow team is pleased to announce the release of 2.8.4 and 2.9.1

Releases [2.9.1](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.9.1) and [2.8.4](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.8.4) contains a quick bug fix and more changes were made in [2.9.0](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.9.0) and [2.8.3](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.8.3). For complete list of changes for `2.9.x` you can see its milestone and rc notes in [releases](https://github.com/spring-cloud/spring-cloud-dataflow/releases).
