# Spring 3.1 Release Train Complete, Spring More Popular Than Ever

_Releases | Adam Fitzgerald |  March 14, 2012 | 0 Comments_

[![Spring 3.1 Release Train Complete](http://www.springsource.org/files/imagefield_default_images/placeholder_video_spring_projects.png)](http://www.vmware.com/company/news/releases/vmw-spring-momentum-3-14-12.html)

In a press release today, SpringSource and VMware [announced](http://www.vmware.com/company/news/releases/vmw-spring-momentum-3-14-12.html) that the Spring 3.1 Release Train is complete. [Spring Integration](/spring-integration), [Spring Security](/spring-security), [Spring Batch](/spring-batch), [Spring Data](/spring-data), [Spring Mobile](/spring-mobile) and [Spring for Android](/spring-android) all now support Spring 3.1, making it easier and faster for enterprise developers to build the next generation of enterprise applications.

The press release also highlighted important findings from Evans Data Research regarding the growing popularity and benefits of using Spring. More than two thirds of Java developers are either using Spring today or plan to do so within the next two years and those that do use Spring are 50% more productive.

SpringSource would like to thank all of the amazing community members that made this release possible. Community contributions form an important part of many of the projects and all the users that experimented with milestone and release candidate versions help substantially in making sure that Spring technology is high quality and production ready. Work is already underway on many other exciting projects like the recently announced [Spring for Apache Hadoop](http://blog.springsource.org/2012/02/29/introducing-spring-hadoop/) project. Community members are invited to attend the upcoming [SpringOne on the Road](/springone-on-the-road) events to meet the Spring experts and learn about the latest features available with Spring.
