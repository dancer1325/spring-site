# Spring Framework 6.1.1 Available Now

_Releases | Stéphane Nicoll |  November 23, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.1` is available now.

Spring Framework `6.1.1` ships with [15 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.1), in time for inclusion in Spring Boot 3.2.0 later today.
