# Spring Session 2.0.0 M5 Released

_Engineering | Rob Winch |  October 10, 2017 | 0 Comments_

On behalf of the community I’m pleased to announce the release of [Spring Session 2.0.0.M5](http://docs.spring.io/spring-session/docs/2.0.0.M5/reference/html5/). This release brings a Redis `ReactiveSessionRepository` implementation. The [webflux sample](https://github.com/spring-projects/spring-session/tree/2.0.0.M5/samples/javaconfig/webflux) has been updated to demonstrate its use.

You can find the complete details of the release in the [changelog](https://github.com/spring-projects/spring-session/milestone/25?closed=1)
