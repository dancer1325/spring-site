# Spring Modulith 2.0 M3 released

_Releases | Oliver Drotbohm |  September 19, 2025 | 0 Comments_

I am pleased to announce the availability of Spring Modulith 2.0 M3. It release ships the following new features:

-   Updated event publication repository implementation for JPA [#1375](https://github.com/spring-projects/spring-modulith/issues/1375)
-   Support for serialized event publication externalization [#1370](https://github.com/spring-projects/spring-modulith/issues/1370)
-   Jackson 3 support for event publication serialization and externalization [#1369](https://github.com/spring-projects/spring-modulith/issues/1369), [#1364](https://github.com/spring-projects/spring-modulith/issues/1364)
-   More lenient out-of-the-box verification for Hexagonal Architecture [#1366](https://github.com/spring-projects/spring-modulith/issues/1366)
-   Upgrade to Spring Boot 4.0 M3 [#1376](https://github.com/spring-projects/spring-modulith/issues/1376)
-   Upgrade to jMolecules 2025 RC5 [#1377](https://github.com/spring-projects/spring-modulith/issues/1377)

Find more details about the releases in the full [changelog](https://github.com/spring-projects/spring-modulith/releases/tag/2.0.0-M3).
