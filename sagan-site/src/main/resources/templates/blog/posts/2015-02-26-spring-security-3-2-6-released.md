# Spring Security 3.2.6 Released

_Releases | Rob Winch |  February 26, 2015 | 6 Comments_

I'm pleased to announce the release of Spring Security 3.2.6.RELEASE. This release contains some important fixes (many addressing Java Configuration and Spring Boot issues). You can find a complete list of changes in the [changelog](https://jira.spring.io/issues/?jql=fixVersion%20%3D%203.2.6%20AND%20project%20%3D%20SEC%20ORDER%20BY%20priority%20DESC).
