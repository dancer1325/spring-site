# Spring XD 1.1.1 Released

_Releases | Mark Pollack |  March 26, 2015 | 0 Comments_

On behalf of the Spring XD team, I am very pleased to announce the  
first maintenance release of Spring XD 1.1.

Download Link: 1.1.1.RELEASE: [zip](http://repo.spring.io/release/org/springframework/xd/spring-xd/1.1.1.RELEASE/spring-xd-1.1.1.RELEASE-dist.zip) and [brew](http://docs.spring.io/spring-xd/docs/1.1.1.RELEASE/reference/html/#osx-homebrew-installation).

The 1.1.1 release includes bug fixes and some enhancements. The full list of tickets is available in [our JIRA](https://jira.spring.io/issues/?filter=14948).

Here are some highlights.

-   Improved performance and offset management when using Kafka message bus.
-   Support for Spark Streaming's reliable receiver.
-   Kryo serialization improvements.
-   Option to clean up Rabbit queue/topics when deleting the stream.
-   Add a Sqoop example

Feedback is very important, so please get in touch with questions and comments via
