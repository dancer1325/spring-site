# Spring AI 1.0.3 Available Now

_Releases | Mark Pollack |  October 01, 2025 | 0 Comments_

On behalf of the Spring AI engineering team and everyone who has contributed, I'm happy to announce that Spring AI `1.0.3` has been released and is now available from Maven Central.

This patch release delivers important stability improvements and bug fixes.

## [](#release-summary)Release Summary

This release includes [27 improvements, bug fixes, and documentation updates](https://github.com/spring-projects/spring-ai/releases/tag/v1.0.3). The focus of this patch release is on:

-   **Improvements**: 18 enhancements to expand capabilities and functionality
-   **Stability**: 6 bug fixes addressing community-reported issues
-   **Documentation**: 3 improvements to help developers

Thanks to all those who have contributed with issue reports and pull requests.

## [](#key-highlights)Key Highlights

This patch release includes several improvements and fixes:

-   **GemFire Vector Store** - metadata filtering queries for more precise vector search operations
-   **AWS Bedrock** - new BedrockChatOptions API and proxy chat model fixes
-   **GraalVM Native Image** - ahead-of-time compilation improvements
-   **Spring Bean Injection** - OpenAiApi now available as an injectable Spring bean
-   **Streaming and Tool Calling** - improved event handling and error messages
-   **Developer Experience** - better IDE auto-completion and documentation updates

These changes continue to improve the stability and usability of Spring AI.

## [](#whats-next)What's Next

The Spring AI team is making good progress on PR merging with a downward trend in our backlog. We're targeting a 1.1 GA release in 4-5 weeks with a primary focus on Model Context Protocol (MCP) functionality.

For the latest updates and to contribute to the project, visit our [GitHub repository](https://github.com/spring-projects/spring-ai) or join the discussion in our community channels. For general questions, please ask on [Stack Overflow](https://stackoverflow.com) using the [`spring-ai` tag](https://stackoverflow.com/tags/spring-ai).
