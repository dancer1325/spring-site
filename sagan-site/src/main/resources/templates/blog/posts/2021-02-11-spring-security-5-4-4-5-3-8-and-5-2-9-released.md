# Spring Security 5.4.4, 5.3.8, and 5.2.9 released

_Releases | Rob Winch |  February 12, 2021 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.4.4 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.4.4)), 5.3.8 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.8.RELEASE)), and 5.2.9 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.9.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.

[Project Site](http://projects.spring.io/spring-security/) | [Reference](http://docs.spring.io/spring-security/site/docs/current/reference/htmlsingle/) | [Help](http://stackoverflow.com/questions/tagged/spring-security)

**NOTE**: There was an issue releasing 5.4.3 and 5.3.7 which caused us to need to release 5.4.4 and 5.3.8.
