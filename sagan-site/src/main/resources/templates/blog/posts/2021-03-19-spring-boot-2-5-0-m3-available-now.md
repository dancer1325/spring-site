# Spring Boot 2.5.0-M3 available now

_Releases | Phil Webb |  March 19, 2021 | 0 Comments_

I am pleased to announce that the third milestone of Spring Boot 2.5 has been released and is available from our [milestone repository](https://repo.spring.io/milestone/). This release [adds a number of new features and bug fixes](https://github.com/spring-projects/spring-boot/releases/tag/v2.5.0-M3).

Highlights of this milestone include:

-   OpenMetrics support for Prometheus
-   Support for environment variable prefixes
-   More Buildpack updates
-   A fresh new look for the [docs](https://docs.spring.io/spring-boot/docs/2.5.0-M3/reference/html/) (including a "dark theme").

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.5-M3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.5.0-M3-Release-Notes) on the wiki and the [updated reference documentation](https://docs.spring.io/spring-boot/docs/2.5.0-M3/reference/html/).

If you want to get started with 2.5 and try out the new features, you can bootstrap a new project on [start.spring.io](https://start.spring.io/#!type=maven-project&language=java&platformVersion=2.5.0.M3&packaging=jar&jvmVersion=11&groupId=com.example&artifactId=demo&name=demo&description=Demo%20project%20for%20Spring%20Boot&packageName=com.example.demo).
