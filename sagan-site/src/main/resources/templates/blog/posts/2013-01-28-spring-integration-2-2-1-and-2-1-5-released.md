# Spring Integration 2.2.1 and 2.1.5 Released

_Releases | Gary Russell |  January 28, 2013 | 0 Comments_

We are pleased to announce that [Spring Integration](http://www.springsource.org/spring-integration) 2.2.1.RELEASE is now available. A list of changes can be found [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=13807).

In addition, the 2.1.5.RELEASE is now available, with the list of changes [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=13635). However, 2.1.x users are encouraged to move to the 2.2.1 release - the 2.2. GA announcement is [here](http://www.springsource.org/node/3756).

The artifacts are available in the [SpringSource Repository](http://repo.springsource.org) as well as Maven Central, or you can download the zip [here](http://www.springsource.org/download/community?project=Spring%20Integration).
