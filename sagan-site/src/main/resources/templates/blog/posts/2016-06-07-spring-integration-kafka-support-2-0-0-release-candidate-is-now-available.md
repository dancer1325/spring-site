# Spring Integration Kafka Support 2.0.0 Release Candidate is now available

_Releases | Artem Bilan |  June 07, 2016 | 1 Comment_

I am pleased to announce that the `spring-integration-kafka` (Spring Integration Kafka Support) Release Candidate for version `2.0` is now available.

The artifact `org.springframework.integration:spring-integration-kafka:2.0.0.RC1` is available in the [Milestone Repository](https://repo.spring.io/milestone/).

This version is based on the just released Spring for Apache Kafka release candidate [1.0.0.RC1](https://spring.io/blog/2016/06/07/spring-for-apache-kafka-1-0-release-candidate-1-available).

There are not many changes since the previous [Milestone 1](https://spring.io/blog/2016/04/11/spring-integration-kafka-support-2-0-0-m1-is-now-available). Just some general internal fixes and upgrades to accommodate recent Spring for Apache Kafka changes.

The Spring Integration Java DSL `1.2` and its `Kafka09` Factory has also been changed to catch up the latest Spring for Apache Kafka API. Milestone 1 for the DSL will be available soon.

We are going to release `2.0.0.RELEASE` in the next week or so, together with Spring for Apache Kafka 1.0.0.RELEASE. So, this is opportunity to give it a try and report any issues you find.
