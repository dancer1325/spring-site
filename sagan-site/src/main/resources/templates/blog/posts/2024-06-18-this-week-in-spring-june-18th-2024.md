# This Week in Spring - June 18th, 2024

_Engineering | Josh Long |  June 18, 2024 | 1 Comment_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*! I've just come from Paris, France, and now I'm in equally beautiful Krakow, Poland, for the amazing Devoxx PL event. We've got a ton of good stuff to dive into, so let's get going!

-   In last week's installment of *Spring Tips*, [I looked at Spring Boot 3.3's new Software Bill of Materials support](https://www.youtube.com/watch?v=6dvFDZs_gro).
-   [Spring Tools 4.23.0 released](https://spring.io/blog/2024/06/12/spring-tools-4-23-0-released).
-   In last week's installment of [*A Bootiful Podcast*](https://spring.io/blog/2024/06/13/a-bootiful-podcast-abdel-sghiouar-cloud-native-developer-advocate-at-google), I talked to Google's Cloud legend Abdel Sghiouar.
-   [Spring Framework 6.1.9, 6.0.22, and 5.3.37 available now](https://spring.io/blog/2024/06/13/spring-framework-6-1-9-6-0-22-and-5-3-37-available-now).
-   [Spring Framework 6.2.0 M4 available now](https://spring.io/blog/2024/06/13/spring-framework-6-2-0-m4-available-now).
-   [Spring Data 2024.0.1 and 2023.1.7 released](https://spring.io/blog/2024/06/14/spring-data-2024-0-1-and-2023-1-7-released).
-   [Spring Tools 4.23.1 released](https://spring.io/blog/2024/06/17/spring-tools-4-23-1-released).
-   [Spring for Apache Pulsar 1.0.7, 1.1.1 now available](https://spring.io/blog/2024/06/18/spring-for-apache-pulsar-1-0-7-and-1-1-1-available-now).
-   [Spring Security 5.8.13, 6.2.5, and 6.3.1 are now available](https://spring.io/blog/2024/06/18/spring-security-5-8-13-6-2-5-and-6-3-1-are-available-now).
-   [Spring for Apache Kafka 3.1.6 and 3.2.1 available now](https://spring.io/blog/2024/06/18/spring-for-apache-kafka-3-1-6-and-3-2-1-available-now).
-   A quick, but awesome, discussion of Spring Cloud AWS between [Oleg Šelajev and Maciej Walkowiak](https://www.youtube.com/embed/D7HBJqhQR24).
-   I love the nice [new time-related debugging facilities for Spring applications in this new IntelliJ IDEA release!](https://blog.jetbrains.com/idea/2024/06/intellij-idea-2024-2-eap-5/).
-   [Daniel Garnier-Moiroux has a nice blog on OAuth 2 and OpenID token expiration](https://garnier.wf/blog/2024/06/11/oauth2-token-expiry.html).
-   [Our friend Thomas Vitale has a nice blog on text classification with Java, Spring AI, and LLMs](https://www.thomasvitale.com/text-classification-with-spring-ai/).
-   [Another amazing blog on Spring AI (this time incorporating Llama3 and pgvector)](https://dev.to/mcadariu/springai-llama3-and-pgvector-bragging-rights-2n8o).
-   JetBrains Developer Advocate Anton Arhipov and Kerry Beetge have a great [blog on static code analysis for Spring with Qodana](https://blog.jetbrains.com/qodana/2024/06/static-code-analysis-for-spring-run-analysis-fix-critical-errors-hit-the-beach/).
-   [Nice! Apache Maven ships with a Maven Daemon (`mvnd`) that is shipped as a GraalVM native image](https://x.com/fniephaus/status/1802812752563749136?s=12&t=n-UflcIbnx1lage-TBk0Cg).
