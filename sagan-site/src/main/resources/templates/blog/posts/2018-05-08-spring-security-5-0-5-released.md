# Spring Security 5.0.5 Released

_Releases | Josh Cummings |  May 08, 2018 | 0 Comments_

**2018-05-09 Update:** The releases address a vulnerability. Please see [this blog post](https://spring.io/blog/2018/05/09/spring-project-vulnerability-reports-published) published after the associated Spring Boot [2.0.2](https://spring.io/blog/2018/05/09/spring-boot-2-0-2) releases.

I'm pleased to announce the release of Spring Security 5.0.5. The release primarily delivers bug fixes and dependency version updates along with some minor improvements.

For a complete list of changes, please refer to the 5.0.5 [changelog](https://github.com/spring-projects/spring-security/milestone/114?closed=1).
