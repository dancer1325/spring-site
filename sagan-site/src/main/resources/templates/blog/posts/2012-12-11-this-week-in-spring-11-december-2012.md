# This Week in Spring - 11 December, 2012

_Engineering | Josh Long |  December 11, 2012 | 0 Comments_

Welcome to another installment of *This Week in Spring*! We [finished SpringOne China](http://springonechina.cloudfoundry.com) over the weekend and are today at the first of two events for [SpringOne India in Bangalore and SpringOne in Hyderabad](http://springoneindia.cloudfoundry.com). The shows have been really amazing! In particular, we've enjoyed the food, the sites and the *amazing* community that have turned up in droves!

-   The folks over at Trifork have a great blog on [properly testing Spring MVC controllers](http://blog.trifork.nl/2012/12/11/properly-testing-spring-mvc-controllers/)
-   I'm going to be honest here -- I don't know, exactly, what this post is about, as it's written in Korean and the Google translation's not particularly helpful. It seems like the author's [discovered a caveat related to Spring MVC's `@SessionAttribute`](http://kwon37xi.egloos.com/4759848) and provided a work around. Hopefully, it's useful to you if you read Korean!
-   Are you using RabbitMQ from a .NET environment? Check out this post on [EasyNewQ which is a .NET client for RabbitMQ.](http://dotnet.dzone.com/articles/rabbitmq-windows-net-case)
-   The *GoSmarter* blog has an interesting post on how to setup a [fake FTP server to unit test Spring Integration FTP flows](http://blog.gosmarter.net/2012/12/11/spring-integration-fakeftpserver-example/). The blog also has a nice post on mocking [SFTP support in Spring Integration](http://blog.gosmarter.net/2012/12/11/spring-integration-mock-sftpserver-example/).
-   The *GoSmarter* blog also has a nice post on using [Spring Security and container-managed security](http://blog.gosmarter.net/2012/12/11/container-based-security-and-spring-security/)
-   *Also* from the *GoSmarter* blog, a nice post on doing [test-driven development with Spring Webflow](http://blog.gosmarter.net/2012/09/14/junit-testing-with-spring-webflow/).
-   The *MonoCaffe* blog has a nice post on using a Spring MVC JSP tag to allow [pre-pending and appending of icons to Bootstrap's input fields](http://monocaffe.blogspot.in/2012/12/spring-mvc-bootstrap-errors-extended.html).
-   The *Zen Java* blog has a nice post on using creating [a RESTful Spring MVC service that is consumed from Java FX](http://www.zenjava.com/2012/12/04/javafx-with-a-springmvc-rest-server-in-5-minutes/). This is *really* slick.
-   The *Cafe Techno* blog has a nice post [that compares Spring MVC and Struts](http://thecafetechno.com/tutorials/interview-questions/compare-spring-mvc-and-struts-mvc-2/). Spring MVC fares very well, I'd say!
-   The *a Quick Guide to Java* blog has a good post [on Spring MVC internationalization](http://javakart.blogspot.in/2012/12/spring-mvc-internationalization-i18n.html).
