# Spring for Apache Pulsar 1.0.12 and 1.1.6 are now available

_Releases | Chris Bono |  November 18, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.0.12` and `1.1.6` have been released and are now available from Maven Central.

The releases will be included in the upcoming Spring Boot [3.2.12](https://github.com/spring-projects/spring-boot/milestones/3.2.12) and [3.3.6](https://github.com/spring-projects/spring-boot/milestones/3.3.6) releases, respectively.

This release includes numerous dependency upgrades.

Please see the release notes ([1.0.12](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.0.12) and [1.1.6](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.6)) for more details.
