# This Week in Spring - March 1, 2016

_Engineering | Josh Long |  March 01, 2016 | 4 Comments_

Welcome to another installment of *This Week in Spring*! This week, I'm in Shenzhen, China and [Zurich, Switzerland for Voxxed Days Zurich](https://voxxeddays.com/zurich). It's going to be a busy week indeed though I sure hope to see you there! Anyway, we've got a lot to cover this week so let's get to it!

-   Spring Boot ninja [Stéphane Nicoll has just announced Spring Boot 1.4.0M1](http://spring.io/blog/2016/02/26/spring-boot-1-3-3-and-1-4-0-m1-available-now) with excellent couchbase support
-   Spring Session lead Rob Winch, never resting on his laurels, just announced [Spring Session 1.1.0](http://spring.io/blog/2016/02/25/spring-session-1-1-0-released)
-   Also not resting on his laurels, Stéphane just [announced Spring Framework 4.2.5, chock full 'o bug fixes and updates](https://spring.io/blog/2016/02/25/spring-framework-4-2-5-available-now)
-   Spring XD lead Mark Pollack has just announced [that Spring XD 1.3.1 is now GA, along with Spring Flo for XD 1.0.1 GA](http://spring.io/blog/2016/02/24/spring-xd-1-3-1-ga-and-flo-for-spring-xd-1-0-1-ga-released)
-   Martin Lippert, the lead developer on Spring Tool Suite, has added better support for both Spring Boot and Cloud Foundry, as well as picking up the latest Eclipse Mars distro, in [STS 3.7.3](http://spring.io/blog/2016/03/01/spring-tool-suite-3-7-3-released)
-   Webinar Replay: [Creating a PageRank Analytics Platform Using Spring Boot Microservices](http://spring.io/blog/2016/03/01/webinar-replay-creating-a-pagerank-analytics-platform-using-spring-boot-microservices)
-   Webinar Replay: [Spring Boot Authentication with Stormpath... and More!](http://spring.io/blog/2016/03/01/webinar-replay-spring-boot-authentication-and-more)
-   Artem Bilan takes us through his SpringOne2GX 2015 talk, [Spring Integration Java DSL](http://spring.io/blog/2016/03/01/springone2gx-2015-replay-spring-integration-java-dsl)
-   Mark Fisher and Patrick Peralta give us the first ever glimpse at Spring Cloud Data Flow in this SpringOne replay: [Cloud Native Dataflow Orchestration](http://spring.io/blog/2016/03/01/springone2gx-2015-replay-cloud-native-dataflow-orchestration)
-   XML? Java Configuration? Annotations? Watch them duke it out in this SpringOne replay -[Spring Framework - the ultimate configuration battle](http://spring.io/blog/2016/03/01/springone2gx-2015-replay-spring-framework-the-ultimate-configuration-battle)
-   A great overview of the state of the streaming open source ecosystem from Michael Minella and Gunnar Hillert, followed by a in - depth look at [Who needs batch these days?](http://spring.io/blog/2016/03/01/springone2gx-2015-replay-who-needs-batch-these-days)
-   Rossen takes us through the latest in [Spring MVC 4.2, New and Noteworthy](http://spring.io/blog/2016/03/01/springone2gx-2015-replay-spring-mvc-4-2-new-and-noteworthy)
-   Our pal Geertjan, as part of last week's epic Spring Boot and Netbeans webinar, has released [the Spring Boot tools for Netbeans extensions](http://plugins.netbeans.org/plugin/63124/spring-boot-tools-4-netbeans). This is *super* cool!
-   I loved Spring ninja and team member Greg Turnquist's post, [*Spring Boot is still a gem.. waiting to be discovered*](http://greglturnquist.com/2016/02/spring-boot-is-still-a-gem-waiting-to-be-discovered.html)
