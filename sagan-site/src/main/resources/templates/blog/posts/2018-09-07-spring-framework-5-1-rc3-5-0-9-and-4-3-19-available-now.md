# Spring Framework 5.1 RC3, 5.0.9 and 4.3.19 available now

_Releases | Stéphane Nicoll |  September 07, 2018 | 1 Comment_

On behalf of the team, I am pleased to announce Spring Framework 5.1 RC3, 5.0.9 and 4.3.19 [are available now](http://projects.spring.io/spring-framework).

Spring Framework 5.0.9 [includes 36 fixes and selected improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16891) while Spring Framework 4.3.19 [includes 23 fixes and selected improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16856). We're also approaching Spring Framework 5.1 with a third release candidate that includes [30 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16939).

As usual, those are the base for the upcoming Spring Boot 2.1 M3, 2.0.5 and 1.5.16 releases early next week.
