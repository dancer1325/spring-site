# Spring for Apache Pulsar 1.1.0-RC1 available now

_Releases | Chris Bono |  April 15, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.1.0-RC1` has been released and is available from [https://repo.spring.io/milestone](https://repo.spring.io/milestone) now!

This release will be included in the upcoming Spring Boot [3.3.0-RC1](https://github.com/spring-projects/spring-boot/milestones/3.3.0-RC1) release.

This release includes numerous enhancements, documentation improvements, bug fixes, and dependency upgrades. Notable new features include:

-   Transaction support for `@PulsarListener` and `PulsarTemplate`

Please see the [release notes](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.0-RC1) for more details.
