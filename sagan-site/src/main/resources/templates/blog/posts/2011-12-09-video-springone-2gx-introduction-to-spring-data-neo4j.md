# Video: SpringOne 2GX - Introduction to Spring Data Neo4J

_News | Adam Fitzgerald |  December 09, 2011 | 0 Comments_

[![SpringOne 2GX 2011 Introduction to Spring Data Neo4J: Michael Hunger](http://www.springsource.org/files/SpringOne2GX2011IntroToSpringDataNeo4j.png)](http://www.infoq.com/presentations/Introduction-to-Spring-Data-Neo4j)

Here is another great session from [SpringOne 2GX 2011](http://www.infoq.com/springone_2gx_2011/), this time covering one of the [Spring Data](/spring-data) projects. This video presentation is by Michael Hunger, Software Developer for Neo4J Technologies, and he provides an [Introduction to Spring Data Neo4J](http://www.infoq.com/presentations/Introduction-to-Spring-Data-Neo4j). Michael covers

-   A NoSQL overview and the basics of the Spring Data projects
-   Graph databases and an introduction to Neo4J
-   An introduction to [Spring Data Neo4J](/spring-data/neo4j) and code examples about how to use it in your applications

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
