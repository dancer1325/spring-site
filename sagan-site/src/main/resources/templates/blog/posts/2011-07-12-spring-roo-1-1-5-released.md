# Spring Roo 1.1.5 Released

_Releases | Stefan Schmidt |  July 12, 2011 | 0 Comments_

Dear Spring Community,

Were happy to announce the availability of [Spring Roo](http://www.springsource.org/roo/ "Spring Roo") 1.1.5 ([download here](http://www.springsource.com/download/community?project=Spring%20Roo "Spring Roo Download")). Spring Roo is a rapid application development tool for Java, allowing you to create full enterprise applications in just minutes.

The new release features over [95 improvements and bug fixes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10340&version=12201) since the 1.1.4 release. The next release of Roo will be 1.2.0.RC1 where new features such as application layering support and a new JSF UI option will be introduced.

Also, please check out the [Vaadin Plugin for Spring Roo tutorial](http://vaadin.com/blog/-/blogs/spring-roo-tutorial).

The [reference guide](http://www.springsource.org/roo/guide "Spring Roo Download") is also included in the download [ZIP](http://www.springsource.com/download/community?project=Spring%20Roo "Spring Roo Download").

[Spring Roo](http://www.springsource.org/roo/ "Spring Roo") has a very active community and responsive development team, so if you have any questions on this new release, please feel free to ask the [Spring Roo engineering team](http://twitter.com/#!/SpringRoo/roo-team/members "Spring Roo Engineering Team") on the [Community Forum](http://forum.springsource.org/forumdisplay.php?67-Roo "Spring Roo Forum"). You'll also find us on Twitter - either follow [@SpringRoo](http://twitter.com/SpringRoo "Spring Roo Twitter") or just include #SpringRoo in your tweets.

We hope you enjoy this new release.
