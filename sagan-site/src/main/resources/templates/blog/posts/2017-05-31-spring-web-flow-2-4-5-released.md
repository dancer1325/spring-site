# Spring Web Flow 2.4.5 released

_Engineering | Rossen Stoyanchev |  May 31, 2017 | 0 Comments_

A minor maintenance release version 2.4.5 for Spring Web Flow is now available for use.

The release includes an important security fix for vulnerability report [CVE-2017-4971](https://pivotal.io/security/cve-2017-4971). Please review the report details and upgrade.
