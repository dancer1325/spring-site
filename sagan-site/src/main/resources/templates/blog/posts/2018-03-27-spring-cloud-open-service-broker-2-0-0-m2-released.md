# Spring Cloud Open Service Broker 2.0.0.M2 Released

_Releases | Roy Clarkson |  March 27, 2018 | 0 Comments_

Spring Cloud Open Service Broker 2.0.0.M2 is now available. Spring Cloud Open Service Broker is a framework for building Spring Boot applications that implement the [Open Service Broker API](https://www.openservicebrokerapi.org). This second milestone adds more polish and improvements as we continue toward a release candidate. The following list includes some of the updates:

-   Update JavaDoc, reference docs, some more JavaDoc, and more reference docs
    
-   Rename the `spring-cloud-open-service-broker` artifact to `spring-cloud-open-service-broker-core`
    
-   Add support for externalized configuration of the Catalog in YAML or Java properties
    
-   Add support to disable API version checking via a configuration property
    
-   Improve exception handling and logging
    

Please visit the [project page](https://cloud.spring.io/spring-cloud-open-service-broker/) for more information. [Reference](https://docs.spring.io/spring-cloud-open-service-broker/docs/2.0.0.M2/reference/html5/) and [API](https://docs.spring.io/spring-cloud-open-service-broker/docs/2.0.0.M2/apidocs/) documentation are also available. See the [GitHub](https://github.com/spring-cloud/spring-cloud-open-service-broker) repository to clone or view the source. [Sample apps](https://github.com/spring-cloud-samples/bookstore-service-broker) are also available on GitHub.

Thanks for your interest in Spring Cloud Open Service Broker! We look forward to your feedback.
