# This Week in Spring - April 15, 2014

_Engineering | Josh Long |  April 15, 2014 | 2 Comments_

Welcome to another installment of *This Week in Spring*! This week, I'm in Paris, France talking to developers about Spring Framework 4.0, Spring Boot, and more at [Devoxx FR](http://fr.devoxx.com/). Next week it's off to Bangalore, India for the Great Indian Developer Summit. As always, [ping me online](http://twitter.com/starbuxman) if you're around either location and want to talk Spring.

1.  Tuesday April 30th Webinar - don't miss Ashley Puls (New Relic) and myself as they track and trace through modern web apps: [Web Application Diagnostics with Java and Javascript](http://spring.io/blog/2014/04/08/webinar-web-application-diagnostics-with-java-and-javascript)
2.  Tuesday April 22nd Webinar - see what stole the EclipseCon 2014 keynote: Martin Lippert and Mike Milinkovich demoing alpha versions of Project Flux: Connecting Eclipse to the Cloud-Based Era of Developer Tooling. [Register](http://spring.io/blog/2014/04/08/webinar-connecting-eclipse-to-the-cloud-based-era-of-developer-tooling-with-project-flux) today!
3.  Thomas Risberg and Janne Valkealahti show how they can use a single programming model / configuration model for Java MapReduce, Streaming, Hive, Pig, Cascading, or HBase in this SpringOne2GX 2013 Replay: [Getting started with Spring Data and Apache Hadoop](http://spring.io/blog/2014/04/15/springone2gx-2013-replay-getting-started-with-spring-data-and-apache-hadoop)
4.  90 minutes with Jags Ramnarayan and Anthony Baker at SpringOne2GX 2013: [In-memory data and compute on top of Hadoop](http://spring.io/blog/2014/04/15/springone2gx-2013-replay-in-memory-data-and-compute-on-top-of-hadoop)
5.  Check out Kevin Nilson (Google) and myself co-presenting at SpringOne2GX 2013 on Spring Profiles, and how it is an amazing tool for managing code through the normal dev/stage/production lifecycle: [Multi Environment Spring Applications](http://spring.io/blog/2014/03/18/springone2gx-2013-replay-multi-environment-spring-applications).
6.  I'm back with Roy Clarkson at SpringOne2GX 2013 talking about Spring Mobile/Android, REST/OAUTH and more in: [Building Smart Clients with Spring](http://spring.io/blog/2014/03/18/springone2gx-2013-replay-building-smart-clients-with-spring)
7.  Erdem Günay has put together a very nice post on [how he used the Spring Expression Language (SpEL) and Spring Boot](http://egunaytech.tumblr.com/post/82779846928/spring-expression-language-spel-and-spring-boot-in) to dynamically send remote installation instructions to a wide array of Android clients, dynamically.
8.  Our pal Sam Brannen is going to be [speaking at the Atlanta Spring User Group](http://www.meetup.com/AtlantaSpring/events/177087502/)! One of the smart
9.  Moritz Schulze put together a very, *very* detailed post [on how his company developed a time tracking, vacation-time managing application called Trackr with Java 8 and Spring Boot](http://blog.techdev.de/trackr-an-angularjs-app-with-a-java-8-backend-part-i)
10.  Mario Arias has put together a version of the Spring [*relational data* Getting Started Guide](http://spring.io/guides/gs/relational-data-access/) using [the Kotlin language](https://github.com/MarioAriasC/gs-relational-data-access/tree/master/kotlin). [Kotlin](http://kotlin.jetbrains.org/) is JetBrains' statically typed programming language to JVM byte codes and JavaScript. *Very* cool! I have never seen the kotlinprimavera module(s) before, but I dig it!
11.  [Patrick Chanezon](http://twitter.com/chanezon), director of enterprise evangelism at Microsoft, sat in on [**the vJUG meetup talk I gave on Spring Boot**](http://www.meetup.com/virtualJUG/events/164640872/) recently and set about hacking! The result is some guidance on how to deploy a Spring Boot application [to Microsoft's Azure PaaS](https://github.com/chanezon/azure-java-samples). Nice! Merci, Patrick!
12.  Hai Nguyen has put together a very nice post on [how to configure / consume an embedded Jetty's JNDI-bound `DataSource`](http://esofthead.com/set-jndi-datasource-spring-application-embedded-jetty-container/)
13.  Did you miss Spring framework lead Juergen Hoeller's epic talk about Spring 4 and Java 8 from GOTO Amsterdam? Have no fear, [you can watch it online](http://www.techtalkshub.com/spring-4-java-8/)!
14.  This is a particularly PaaSy *This Week in Spring*! Kim Saabye Pedersen has written a nice post [on using Spring Boot on RedHat's OpenShift PaaS](http://kim.saabye-pedersen.org/2014/04/spring-boot-on-openshift.html)
15.  Our friend Petri Kainulainen is back, this time with a very [nice post on using JOOQ with Spring](http://www.petrikainulainen.net/programming/jooq/using-jooq-with-spring-crud/). Petri's an amazing person to have in the community. I hope you don't need me to tell you that. Bookmark his site. It's almost always worth it.
16.  Our pal at Netflix, Tomás Lin, is back! This time he's got a post on [bundling web content with Spring Boot and Gradle](http://fbflex.wordpress.com/2014/03/14/building-web-content-jars-for-spring-boot-with-gradle/)
17.  Michael Simons posted a very detailed post on how [he's developing a Boot web application](http://info.michael-simons.eu/2014/02/25/boot-your-application-with-spring-boot/). Thanks for the detailed writeup, Michael. Very valuable feedback.
