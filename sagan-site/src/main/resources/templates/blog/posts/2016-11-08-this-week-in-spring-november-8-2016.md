# This Week in Spring - November 8, 2016

_Engineering | Josh Long |  November 08, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in London, England for [µCon](https://skillsmatter.com/conferences/7062-mucon-2015-the-microservices-conference), Antwerp, Belgium for [Devoxx](https://devoxx.be/), and Malmö, Sweden for [Oredev](http://oredev.org/). If you're around, [ping me and say hi](http://twitter.com/starbuxman).

-   Spring Framework and Spring Boot ninja Stephane Nicoll just announced [that Spring Framework 4.3.4, mostly bugfixes, is available now](https://spring.io/blog/2016/11/08/spring-framework-4-3-4-available-now).
-   Spring Security ninja [Joe Grandja just announced Spring Security OAuth 2.0.12](https://spring.io/blog/2016/11/07/spring-security-oauth-2-0-12-released) which is primarily bugfixes and paves the way for some nice new features.
-   Spring Cloud Data Flow and [Spring Cloud Task ninja Glenn Renfro just announced Spring Cloud Task 1.1.0.RC1](https://spring.io/blog/2016/11/07/spring-cloud-task-1-1-0-rc1-is-now-available). Spring Cloud Task continues the journey to 1.1 which adds a lot of new features including error handling, improved options for customizing the launch of partitioned job workers.
-   Spring Data lead [Oliver Gierke just announced Spring Data release train Hopper SR5](https://spring.io/blog/2016/11/03/spring-data-release-train-hopper-sr5-released). This release includes a *lot* of updates across all of the modules including the modules for JPA, KeyValue, Cassandra, SOLR, Gemfire, Neo4J, MongoDB, Envers, Redis, ElasticSearch, Couchbase, and updates to Spring Data REST and and Spring Data Commons.
-   Spring Cloud co-lead [Spencer Gibb just announced the Spring Cloud CLI launcher](https://spring.io/blog/2016/11/02/introducing-the-spring-cloud-cli-launcher). This is one of my favorite new utilities - it makes getting to the punch even easier with Spring Cloud! Install it and then run `spring cloud configserver dataflow eureka h2 hystrixdashboard kafka zipkin` and you'll have those services running on the appropriate ports! Simple.
-   Spring Cloud Data Flow ninja Thomas Risberg [just announced Spring Cloud Data Flow for Kubernetes 1.1 M2](https://spring.io/blog/2016/11/02/spring-cloud-data-flow-for-kubernetes-1-1-m2-released)
-   Spring Cloud ninja [Ryan Baxter just announced Spring Cloud Camden SR2](https://spring.io/blog/2016/11/02/spring-cloud-camden-sr2-is-available) which consists of bugfixes to the various Spring Cloud modules.
-   Last week, I looked at [how to explicitly lesser known aspects of the Spring Framework component model](https://spring.io/blog/2016/11/02/spring-tips-spring-aware-beans)
-   [William Witt](http://twitter.com/unamanic) took one of my talks and adapted the examples to work in the [PCFDev Cloud Foundry environment](https://pivotal.io/pcf-dev) and [documented it step-by-step along with code - this is an *awesome* resource!](https://unamanic.blogspot.be/2016/11/tutorial-bootiful-microservices-on-pcf.html) Good job William!
-   want to use JHipster and Stormpath together? Spring community [hero Matt Raible's got just the fix](https://stormpath.com/blog/stormpath-jhipster-application)
-   friend of the Spring community Michael Simons just put together a really interesting look at exposing a [JOOQ-based analytics API as a Spring Boot and Spring MVC-based REST API](http://info.michael-simons.eu/2016/11/02/an-http-api-for-analytic-queries/).
-   Pivotal's own Ben Wilcock put together a very nice [look at using Spring Boot and the Axon 2.0 framework together](https://github.com/joshlong/cf-cqrs-microservice-sampler).
-   If you want to learn more about the new version of Axon 3.0, be [sure to join Axon lead Allard Buijze and me for a webinar on November 16!](https://spring.io/blog/2016/10/25/webinar-bootiful-cqrs-with-axon-nov-16)
