# Spring Mobile 1.1.1 Released

_Releases | Roy Clarkson |  February 21, 2014 | 0 Comments_

We're happy to announce that Spring Mobile 1.1.1 is now available. This is a maintenance release that primarily addresses an issue where some Samsung tablets were being incorrectly resolved as mobile devices. Additional testing was also added for Windows 8 and newer versions of iOS. This release is built against Spring Framework 3.2.8, and has been tested against Spring Framework 4.0.2. See the [changelog](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10482&version=14284) for more information.
