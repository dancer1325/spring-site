# Spring Social Facebook 2.0.1 Released

_Releases | Craig Walls |  April 27, 2015 | 1 Comment_

I'm pleased to announce the release of Spring Social Facebook 2.0.1.RELEASE. This maintenance release addresses a handful of bugs that were discovered following the 2.0.0.RELEASE two weeks ago. For complete details regarding this release, see the [changelog](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10607&version=14996).

Note that if you're using Spring Social Facebook with Spring Boot, the Spring Boot starter for Spring Social Facebook still references 1.1.1.RELEASE. But you can override that by explicitly declaring the 2.0.1.RELEASE dependency in your Maven or Gradle build. See the [Spring Social Showcase/Spring Boot example](https://github.com/spring-projects/spring-social-samples/blob/master/spring-social-showcase-boot/build.gradle) to see how this is done.
