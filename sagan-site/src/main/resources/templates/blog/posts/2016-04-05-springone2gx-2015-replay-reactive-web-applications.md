# SpringOne2GX 2015 replay: Reactive Web Applications

_News | Pieter Humphrey |  April 05, 2016 | 0 Comments_

Recorded at SpringOne2GX 2015. Speaker: Stephane Maldini, Rossen Stoyanchev Web / Javascript track Slides: [http://www.slideshare.net/SpringCentral/reactive-web-applications-53170985](http://www.slideshare.net/SpringCentral/reactive-web-applications-53170985) In our previous talk "Intro to Reactive Programming" we defined reactive programming and provided details around key initiatives such as Reactive Streams and ReactiveX. In this talk we'll focus on where we are today with building reactive web applications. We'll take a look at the choice of runtimes, how Reactive Streams may be applied to network I/O, and what the programming model may look like. While this is a forward looking talk, we'll spend plenty of time demoing code built with with back-pressure ready libraries available today.
