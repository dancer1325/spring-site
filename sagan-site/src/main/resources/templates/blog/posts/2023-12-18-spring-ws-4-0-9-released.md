# Spring Web Services 4.0.9 is released

_Releases | Corneil du Plessis |  November 21, 2022 | 0 Comments_

Greetings Spring community,

The Spring Web Services team has released **4.0.9** in order to address CVEs by updating WSS4J to version 2.4.3.

We also welcome a contribution by [@hduelme](https://github.com/hduelme) to improve the readability and performance.

For more details, check out our official [GitHub release](https://github.com/spring-projects/spring-ws/releases/tag/v4.0.9). This includes links to all the related tickets as well as the source code.

We have also adopted [GitHub releases](https://github.com/spring-projects/spring-ws/releases) as our avenue for tracking changes in a given release. All current and prior releases of Spring WS are now captured in the list of releases on GitHub. Feel free to check them out.

The artifacts are hosted on Maven Central so grab 'em while they’re hot!
