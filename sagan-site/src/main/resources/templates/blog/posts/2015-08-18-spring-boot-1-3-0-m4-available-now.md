# Spring Boot 1.3.0.M4 Available Now

_Releases | Stéphane Nicoll |  August 18, 2015 | 2 Comments_

We've just released Spring Boot v1.3.0.M4 as we found an annoying regression with CloudFoundry deployment. This release includes [additional fixes and improvements](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.0.M4+is%3Aissue+is%3Aclosed).

For a complete list of changes, see the [Spring Boot 1.3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-1.3-Release-Notes) on the wiki and the updated [reference documentation](http://docs.spring.io/spring-boot/docs/1.3.0.M4/reference/htmlsingle/).
