# Spring IO Platform Brussels SR12

_Releases | Andy Wilkinson |  July 31, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR12 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.9
-   Spring Boot 1.5.15
-   Spring Data Ingalls SR14

The versions of a number of third-party dependencies have also been updated.
