# Spring Security 5.3.3, 5.2.5, 5.1.11, 5.0.17, 4.2.17 Released

_Engineering | Rob Winch |  June 03, 2020 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.3.3 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.3.RELEASE)), 5.2.5 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.5.RELEASE)), 5.1.11 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.11.RELEASE)) , 5.0.17 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.17.RELEASE)), 4.2.17 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.17.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
