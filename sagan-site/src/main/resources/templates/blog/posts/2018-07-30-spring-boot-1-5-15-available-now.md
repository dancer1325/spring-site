# Spring Boot 1.5.15 available now

_Releases | Stéphane Nicoll |  July 30, 2018 | 2 Comments_

On behalf of the team, I am pleased to announce that Spring Boot 1.5.15 has been released and is is now available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

Spring Boot 1.5.15 includes [35 fixes, improvements and dependency updates](https://github.com/spring-projects/spring-boot/milestone/114?closed=1). Thanks to all that have contributed with issue reports and pull requests.

If you haven’t already upgraded to Spring Boot 2, now is a great time to do so as the end of life announcement for 1.x’s [has just been made](https://spring.io/blog/2018/07/30/spring-boot-1-x-eol-aug-1st-2019).
