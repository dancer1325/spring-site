# This Week in Spring - February 21st, 2017

_Engineering | Josh Long |  February 21, 2017 | 1 Comment_

Welcome to another installment of *This Week in Spring*! This week I'm in Atlanta, GA for [the incredible DevNexus event](http://DevNexus.com) and then it's off to Geneva, Switzerland for the Voxxed CERN event. As usual, [don't hesitate to say "Hi"](http://twitter.com/starbuxman) if you're around!

-   [Thomas Risberg just announced Spring Cloud Data Flow 1.2.M1](https://spring.io/blog/2017/02/15/spring-cloud-data-flow-1-2-m1-released)
-   Spring IO Platform lead Andy Wilkinson has just [announced the Spring IO Platform Brussels release](https://spring.io/blog/2017/02/21/spring-io-platform-brussels-release) which introduces Spring Kafka to the release train and updates several others.
-   InfoQ's [Charles Humble](https://www.twitter.com/charleshumble) takes a look at [the future of Enterprise Java](https://www.infoq.com/articles/enterprise-java-opinion) via the lens of the Geoffrey Moore's Chasm metaphor in this article.
-   Last week, in another installment of *Spring Tips*, we looked at [how to use JOOQ with Spring Boot](https://spring.io/blog/2017/02/15/spring-tips-jooq).
-   Using Java FX? This nice post by Gregg Bolinger shows how to use [Spring with JavaFX](http://www.greggbolinger.com/let-spring-be-your-javafx-controller-factory/), and have it be your controller factory.
-   Check out community hero Michael Simons' [German-language JAX talk introducing Spring Boot starters](https://jaxenter.de/spring-boot-starter-2-52525)
-   Deepak Tyagi put together a nice post on [building applications that carry on in the face of a cache failure using the Spring Cache abstraction](https://dzone.com/articles/safeguard-spring-app-from-cache-failure)
-   Aboullaite Mohammed put together [an interesting look at using JWT with Spring Boot](https://aboullaite.me/spring-boot-token-authentication-using-jwt/). It's a good first cut!
-   Want to test against an in-memory MongoDB database? This Brazillian-language blog explaining how [to use Spring Boot and Fongo might be just for you](https://www.paradigmadigital.com/dev/tests-integrados-spring-boot-fongo/)
-   Matt McCandless does a nice [job introducing Spring Boot in this post](https://keyholesoftware.com/2017/01/30/spring-boot-the-right-boot-for-you/): "just put them on and lace them up!"
-   Interesting post on the OpenShift blog on [how to use Spring Boot and their Hawkular APM tool](https://blog.openshift.com/performance-metrics-apm-spring-boot-microservices-openshift/).
-   This is an interesting look at [using Spring Boot with Hazelcast](http://pappanyn.me/blog/2017/02/17/a-run-around-with-hazelcast/)
-   Dhiraj Ray explains how [to use Spring MVC (in a Spring Boot application) with Apache Tiles](http://www.devglan.com/faq/spring-boot-mvc-apache-tiles-example).
-   Christoph Burmeister looks at [how to load-balance Spring Boot applications with Nginx](http://christoph-burmeister.eu/?p=2951)
-   This DZone post introduces [basic dependency injection with Spring](https://dzone.com/articles/spring-core-dependency-injection)
