# This Week in Spring - December 22, 2015

_Engineering | Josh Long |  December 22, 2015 | 6 Comments_

Welcome to another installment of *This Week in Spring*! This week, many of us will be off for the Christmas holiday. If you celebrate, then let me wish you the Merriest of Christmases from our team. If you don't celebrate, we'll see you next week as we look at our annual *This Year in Spring!* and welcome 2016!

Holiday or not, there's a lot of great stuff to read this week so let's get to it!

1.  If you're using Spring Cloud Eureka and Jersey you may run into issues because Eureka itself uses Jersey 1.x where as Spring Boot's Jersey support is based on Jersey 1. Read how [Aleksandar Stoisavljevic solved it](https://staleks.github.io/blog/2015/12/20/index.html)

-   Spring framework lead Juergen Hoeller has announced that [Spring Framework 4.2.4 and 4.1.9 have been released](http://spring.io/blog/2015/12/17/spring-framework-4-2-4-4-1-9-released)
-   Spring framework contributor Stéphane Nicoll has [just announced that Spring Framework 3.2.16 is now available](http://spring.io/blog/2015/12/17/spring-framework-3-2-16-available-now)
-   Spring Boot contributor Stéphane Nicoll *also* announced [that Spring Boot 1.3.1 and 1.2.8 are now available, as well](http://spring.io/blog/2015/12/18/spring-boot-1-3-1-and-1-2-8-available-now)
-   Spring Data lead Oliver Gierke has just announced [that Spring Data relese train Gosling SR2 is now available, as well](http://spring.io/blog/2015/12/18/spring-data-release-train-gosling-sr2-released)
-   Spring IO Platform lead Andy Wilkinson has just announced [1.1.5](http://spring.io/blog/2015/12/18/spring-io-platform-1-1-5-release) and [2.0.1](http://spring.io/blog/2015/12/18/spring-io-platform-2-0-1-release) are now available
-   Do you want build multitenancy into your system? Consider using [the Stormpath Identity provider which features a very powerful Spring Boot auto-configuration](https://stormpath.com/blog/idsite-multi-tenancy/)
-   our pal Biju Kunjummen is back at it again, this time with a post on [using Spring Cloud's auto-configured `RestTemplate` with Ribbon](http://www.javacodegeeks.com/2015/12/spring-cloud-rest-client-netflix-ribbon-basics.html)
-   this week, Pivotal announced the acquisition [of longtime Cloud Foundry partner Cloud Credo](https://blog.pivotal.io/pivotal-cloud-foundry/news/pivotal-acquires-cloudcredo): welcome aboard!
-   this post by Nedim Šabić has a nice look at various approaches to [implementing microservices using Spring Cloud and Go](http://rabbitstack.github.io/cloud/microservices/spring/a-pragmatic-approach-to-migration-from-monolithic-to-microservices/)
-   Voxxed has an interesting post by Holger Steinhauer [that introduces a simple Spring Boot application](https://www.voxxed.com/blog/2015/12/10729/) that leverages Spring MVC, Spring Data, JPA, operationalize it using the Spring Boot actuator, add a template, and so much more.
