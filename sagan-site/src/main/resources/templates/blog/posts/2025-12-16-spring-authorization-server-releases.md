# Spring Authorization Server 2025-12 Releases

_Releases | Rob Winch |  December 16, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce the availability of Spring Authorization Server `1.5.5`, `1.5.4`, `1.4.8`, and `1.4.7`. This set of releases includes bug fixes and back to back releases due do an error in the release process.
