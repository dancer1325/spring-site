# Spring Data Lovelace SR4, Kay SR13, and Ingalls SR18 released

_Releases | Mark Paluch |  January 10, 2019 | 0 Comments_

On behalf of the Spring Data team, I'd like to announce the availability of the Lovelace SR4, Kay SR13, and Ingalls SR18 maintenance releases. Today's releases pick up the latest Spring Framework maintenance releases: 5.1.4, 5.0.12, and 4.3.22, respectively. The upcoming Spring Boot releases (2.1.2, 2.0.8, and 1.5.19) are going to pick up Lovelace SR4, Kay SR13, and Ingalls SR18, for your convenience.

The service releases ship with bug fixes and dependency upgrades and address over 170 tickets in total. You can find the complete list of tickets by following the links below:

-   [Lovelace SR4](https://jira.spring.io/issues/?filter=16353)
-   [Kay SR13](https://jira.spring.io/issues/?filter=16352)
-   [Ingalls SR18](https://jira.spring.io/issues/?filter=16351)

## [](#spring-data-1x-and-20x-end-of-life-eol)Spring Data 1.x and 2.0.x End of Life (EOL)

All good things must come to an end. For the Spring Data `1.x` (Ingalls) and `2.0.x` (Kay) lines, we will cease maintenance in alignment with Spring Framework and Spring Boot. The end of life dates are:

-   Spring Data 1.x (Ingalls): August 2019.
-   Spring Data 2.0.x (Kay): March 2019.

Spring Data users should plan accordingly to ensure that they have upgraded to the latest 2.1 version before that date. Please note that Spring Data 2.0 and above requires Java 8+.
