# This Week in Spring - January 10th, 2017

_Engineering | Josh Long |  January 10, 2017 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in San Francisco and [New York City](https://www.meetup.com/NYC-SUG/events/236521420/) talking to developers and toiling on the final edits to the O'Reilly book [*Cloud Native Java*](http://shop.oreilly.com/product/0636920038252.do) (almost there!). Also, marketing ninja [Pieter Humphrey](https://www.twitter.com/pieterhumphrey) is ramping up the S1P 2016 replays, so fasten your seat belt and check the [S1P 2016 YouTube playlist](https://www.youtube.com/playlist?list=PLgGXSWYM2FpOSZW9G9QpHbOv_PIWRO8sj) for the most current releases. We've got a lot to cover this week so let's get to it!

-   Spring Cloud Task ninja [Glen Renfro just announced Spring Cloud Task 1.1.1](https://spring.io/blog/2017/01/06/spring-cloud-task-1-1-1-release-is-now-available) with a fast following, bug-fix only, [1.1.2 release](https://spring.io/blog/2017/01/10/spring-cloud-task-1-1-2-release-is-now-available).
-   our latest and greatest [Spring Boot team member Madhura Bhave just announced Spring Boot 1.5.0.RC1](https://spring.io/blog/2017/01/06/spring-boot-1-5-0-rc1-available-now)
-   Spring Integration guru [Artem Bilan just announced Spring Integration 5.0.M2](https://spring.io/blog/2017/01/05/spring-integration-5-0-milestone-2-available)
-   Spring Batch and Spring Cloud Task lead [Michael Minella just announced Spring Batch 4.0](https://spring.io/blog/2017/01/04/spring-batch-4-0-0-m1-is-now-available) complete with a lot of niceties and - in particular a feature I've wanted *forever* - an easy way to build instances of the more sophisticated `ItemReader` and `ItemWriter` intances, including the `FlatFileItemReader`! This is been a long time in coming and I can't wait for this release. This rounds out the already epic Java configuration story for Spring Batch.
-   Spring IO Platform lead Andy Wilkinson just announced [Spring IO Platform Athens SR2](https://spring.io/blog/2017/01/04/spring-io-platform-athens-sr2), which brings with it new revisions of Spring Boot, Spring Retry, Spring AMQP and so much more
-   Spring framework ninja and Kotlin afficianado Sebastien Deleuze just blogged about some of the coming improvements to support [the Kotlin language in Spring framework 5](https://spring.io/blog/2017/01/04/introducing-kotlin-support-in-spring-framework-5-0)
-   Microservice developers working on security and service disovery won't want to miss Spencer Gibb's [S1P 2016 replay: Spring Cloud with Consul and Vault](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-spring-cloud-with-consul-and-vault). If you are new to Vault, [Mark Paluch](https://www.twitter.com/mp911de) has you covered with a [S1P replay](https://spring.io/blog/2017/01/03/springone-platform-2016-replay-managing-secrets-at-scale) that was published last week in Spring.
-   Catch Spring Data lead Oliver Gierke as he delivers a [Spring Data REST deep dive](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-advanced-spring-data-rest) at S1P 2016, now on youtube!
-   [Shin Tanimoto's S1P 2016 session](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-let-s-visualize-your-spring-boot-applications) offers techniques for visualizing microservices built by Spring Boot applications using Elasticsearch, Kibana and Spring Cloud Sleuth.
-   Apache Geode and Pivotal Gemfire enthusiasts won't want to miss this masterclass from [hoopy froods](http://www.urbandictionary.com/define.php?term=A%20really%20hoopy%20frood) [John Blum](https://twitter.com/john_blum) and [Luke Shannon](https://www.twitter.com/lukewshannon) about [Spring Data and in memory data management in action](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-spring-data-and-in-memory-data-management-in-action), recorded at S1P 2016.
-   Using Spring Boot on Cloud Foundry? Scott Frederick shows you how to wield autoconfiguration and the Java Buildpack like a boss to access platform and off-platform services [in his talk recorded at S1P 2016](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-consuming-data-services-with-spring-apps-on-cloud-foundry).
-   Are you a Apache Cassandra user? Spring Data experts Mark Paluch, John Blum talk about the community module turned 1st class project, [Spring Data Cassandra in this S1P 2016 replay](https://spring.io/blog/2017/01/10/springone-platform-2016-replay-sneek-peek-at-spring-data-cassandra).
-   Pivotal's very own Ben Wilcock put together a very nice example project that demonstrates [how to use Spring Cloud Services with Pivotal Cloud Foundry](https://github.com/benwilcock/pcf-spring-cloud-services-demo)
-   He *also* put together a handy [demo of Spring Cloud Contracts](https://github.com/benwilcock/spring-cloud-contracts)
-   Our very own Toshiaki Maki Making put together a nice post on some of the [niceties in Spring Boot 1.5 and how they integrate with the Pivotal Apps Manager for Cloud Foundry](https://blog.ik.am/entries/401)
-   Daniel Olszewski has put together a quick and easy look at [configuring time-to-live values for the Spring Cache abstraction](http://dolszewski.com/spring/multiple-ttl-caches-in-spring-boot/)
-   I liked this German-language [post on building REST services with Spring Boot](http://blog.sogeo.services/blog/2017/01/01/rest-service-mit-spring-boot.html)
-   Jose Praveen did a [nice job demonstrating how to integrate an H2 database with a Spring Boot application](https://dzone.com/articles/integrate-h2-database-in-your-spring-boot-applicat)
-   I liked Sam Chu's Chinese-language post on [setting up multiple MongoDB connections in a Spring application](http://samchu.logdown.com/posts/1272942-springboot-data-set-multiple-mongodb)
-   this is a well-thought out look at how to [do validation of JSON payloads in a REST API](http://www.bbenson.co/post/spring-validations-with-examples/)
