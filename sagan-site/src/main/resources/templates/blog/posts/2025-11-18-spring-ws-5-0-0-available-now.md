# Spring Web Services 5.0.0 available now

_Releases | Stéphane Nicoll |  November 18, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Web Services `5.0.0` has been released and is now available from Maven Central.

For full [upgrade instructions](https://github.com/spring-projects/spring-ws/wiki/Spring-Web-Services-5.0-Release-Notes#upgrading-from-spring-web-services-41) and [new and noteworthy](https://github.com/spring-projects/spring-ws/wiki/Spring-Web-Services-5.0-Release-Notes#new-and-noteworthy) features please see the [release notes](https://github.com/spring-projects/spring-ws/wiki/Spring-Web-Services-5.0-Release-Notes).

## [](#whats-new-in-50)What's new in 5.0

The highlights of the 5.0 release include:

-   Upgrade to Spring Framework 7 and Spring Security 7
-   Upgrade to Jakarta EE 11 (Servlet 6.1)
-   [Comprehensive null safety through JSpecify](https://jspecify.dev/)

## [](#thank-you)Thank you

We want to take this opportunity to again thank all our users and contributors.

If you're interested in helping out, check out the ["ideal for contribution" tag](https://github.com/spring-projects/spring-ws/labels/status%3A%20ideal-for-contribution) in the issue repository. If you have general questions, please ask on [stackoverflow.com](https://stackoverflow.com) using the [`spring-ws` tag](https://stackoverflow.com/tags/spring-ws).
