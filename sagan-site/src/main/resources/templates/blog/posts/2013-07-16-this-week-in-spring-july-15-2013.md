# This Week in Spring - July 15, 2013

_Engineering | Josh Long |  July 16, 2013 | 0 Comments_

Welcome back to another installation of *This Week in Spring.* We've got a lot to cover, as usual, so let's get right to it! This week I'm at SenchaCon, talking to developers about building RESTful applications and clients, and then I'm off to OSCON next week, where I'll be hosting the Spring BOF, giving a talk on the latest and greatest in Spring 4, and helping to man the Pivotal booth. If you're at SenchaCon or OSCON, [don't hesitate to ping me](http://twitter.com/starbuxman) and we can talk [Spring](http://www.springsource.org), [Cloud Foundry](http://cloudfoundry.com), [big-data](http://www.springsource.org/spring-xd), and more!

1.  SpringOne 2GX 2013 early bird expires soon, [register now](http://www.springone2gx.com/conference/santa_clara/2013/09/register) to secure the discounted rate!
2.  Spring Data ninja Thomas Risberg has announced that [Spring For Apache Hadoop 1.0.1.RC1 has been released](http://www.springsource.org/node/22651). The new release supports Hadoop 2.0 and Pivotal HD, among other things.
3.  Tool Suite ninja and lead Martin Lippert has announced that [Spring Tool Suite And Groovy/Grails Tool Suite 3.3.0 have been released](http://www.springsource.org/node/22650). Very nice!
4.  Gary Russell has announced that [Spring AMQP 1.2.0 has been released](http://www.springsource.org/node/22652). Check out the [What's New](http://static.springsource.org/spring-amqp/reference/html/whats-new.html) for details.
5.  Join Mattias Severson & Johan Haleby and learn about [Functional Programming without Lambdas](http://www.springsource.org/node/22643) on July 18, 2013
6.  Join Hemant Joshi as he introduces how to use Spring and the [Cucumber BDD testing framework in a webinar on July 30th, 2013](https://www.springsource.org/node/22644).
7.  Our friends at Skills Matter are throwing a Spring-centric conference ([the Spring Exchange](https://plus.google.com/u/0/events/cuqrmfcosup8ghpdqs9j450ulso)) in London on November 14 and November 15. There are some killer speakers, and I highly encourage you to make it, if you can.
8.  Are you using [Spring Social](http://www.springsource.org/Spring-Social) in the wild? [We want to hear about it](http://forum.springsource.org/showthread.php?139750-Spring-Social-in-the-Wild)!
9.  Spring Security lead and ninja Rob Winch has put together a very nice post on *readability* [when using Spring Security Java configuration](http://blog.springsource.org/2013/07/11/spring-security-java-config-preview-readability/).
10.  A new "Quick Search" is included in Spring ToolSuite (STS) 3.3.0 and Groovy Grails Tool Suite (GGTS) 3.3.0 which have just been released. Kris De Volder, a senior developer on the Spring and Groovy and Grails Tool Suites, has just put together [a nice post on this new feature](http://blog.springsource.org/2013/07/11/eclipse-quick-search/).
11.  Our friend Johnathan Mark Smith is at it again! This time, he's written a post, [*How to use Fongo and nosql unit to test Spring Data project with MongoDB, JUnit, Log4J*](http://johnathanmarksmith.com/spring/spring%20data/mongodb/fongo/java/javaconfig/programming/spring%20java%20configuration/2013/06/28/how-to-use-fongo-and-nosql-unit-to-test-spring-data-project-with-mongodb-junit-log4j/). Check it out!
12.  Wow! Amir Kibbar, at the *HP Software Developer's blog*, has put together a *really* comprehensive look at how to develop a service tier, build a web tier, and then test both. The first post [on setting up a service tier](http://h30499.www3.hp.com/t5/HP-Software-Developers-Blog/A-Comprehensive-Example-of-a-Spring-MVC-Application-Part-1/ba-p/6135439#.UeR36GRASIc), the second is an example [of refining the service tier and testing it](http://h30499.www3.hp.com/t5/HP-Software-Developers-Blog/A-Comprehensive-Example-of-a-Spring-MVC-Application-Part-2/ba-p/6135443#.UeR4GmRASIc), the third post introduces how to [setup a REST endpoint](http://h30499.www3.hp.com/t5/HP-Software-Developers-Blog/A-Comprehensive-Example-of-a-Spring-MVC-Application-Part-3/ba-p/6135449#.UeR4HGRASIc), and the fourth post [talks about testing the REST service](http://h30499.www3.hp.com/t5/HP-Software-Developers-Blog/A-Comprehensive-Example-of-a-Spring-MVC-Application-Part-4/ba-p/6135463#.UeR6TGRASIc). Definitely worth a read (and a bookmark!) It's possible to do everything demonstrated in these posts using straight Java configuration, also
13.  Igor Artamonov has a nice, abbreviated post on how [to build a RESTful endpoint with Spring](http://igorartamonov.com/2013/07/restful-using-spring-framework/).
14.  Our friend at the *Baeldung* blog has put together a very nice post on how to use [digest authentication with Spring Security](http://www.baeldung.com/spring-security-digest-authentication).
