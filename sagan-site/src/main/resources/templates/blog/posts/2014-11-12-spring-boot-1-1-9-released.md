# Spring Boot 1.1.9 released

_Releases | Phil Webb |  November 12, 2014 | 0 Comments_

Spring Boot 1.1.9 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.1.9) and third-party dependency updates.

This release also upgrades to the latest Spring Framework, which includes an [important security patch](https://spring.io/blog/2014/11/11/spring-framework-4-1-2-4-0-8-3-2-12-released); as such it is highly recommended that all Spring Boot users upgrade.
