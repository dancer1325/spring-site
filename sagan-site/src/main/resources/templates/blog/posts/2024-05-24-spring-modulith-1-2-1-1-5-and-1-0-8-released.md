# Spring Modulith 1.2, 1.1.5, and 1.0.8 released

_Releases | Oliver Drotbohm |  May 24, 2024 | 1 Comment_

I am excited to announce the availability of Spring Modulith 1.2 alongside the service releases 1.1.5 and 1.0.8. The latter two contain a lot of bug fixes, improvements and dependency upgrades. 1.2 most notably ships with new features:

-   Support for [open application modules](https://docs.spring.io/spring-modulith/reference/fundamentals.html#modules.advanced.open)
-   Support for a [`package-info.java` alternative](https://docs.spring.io/spring-modulith/reference/fundamentals.html#modules.named-interfaces) for Kotlin
-   Indexes for event publication registry database tables (JDBC)
-   Spring AOT-generated classes excluded from verification
-   Upgrade to Spring Boot 3.3 (while retaining compatibility with 3.2 and 3.1)

Find more information in the full changelogs for [1.2](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0) ([RC1](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0-RC1), [M3](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0-M3), [M2](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0-M2), [M1](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0-M1)), [1.1.5](https://github.com/spring-projects/spring-modulith/releases/tag/1.1.5) and [1.0.8](https://github.com/spring-projects/spring-modulith/releases/tag/1.0.8).
