# Spring Security 4.0.0.RC2 Released

_Engineering | Rob Winch |  February 26, 2015 | 7 Comments_

We are please to announce the release of Spring Security 4.0.0.RC2.

We were very keen on doing a GA release, but due to community feedback we decided that another RC was necessary. Ultimately, this release resolved nearly [50 tickets](https://jira.spring.io/issues/?jql=project%20%3D%20SEC%20AND%20fixVersion%20%3D%204.0.0.RC2%20ORDER%20BY%20priority%20DESC).

A summary of changes can be seen below:

-   Support for enforcing [Same Origin for WebSocket connections](http://docs.spring.io/spring-security/site/docs/4.0.0.RC2/reference/htmlsingle/#websocket-sameorigin)
-   Refinements in [WebSocket Configuration](http://docs.spring.io/spring-security/site/docs/4.0.0.RC2/reference/htmlsingle/#websocket) ([SEC-2827](https://jira.spring.io/browse/SEC-2827) [SEC-2833](https://jira.spring.io/browse/SEC-2833) [SEC-2853](https://jira.spring.io/browse/SEC-2853) )
-   More intuitive HTTP Response Headers Configuration ([SEC-2846](https://jira.spring.io/browse/SEC-2846))

Assuming everything goes smoothly, the plan is to do a GA release in approximately two weeks. In the meantime, make sure to try everything out and give us feedback!
