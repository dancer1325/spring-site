# This Week in Spring - December 5th, 2017

_Engineering | Josh Long |  December 06, 2017 | 1 Comment_

Hi Spring fans and welcome to *This Week in Spring* from the premier JVM-language event SpringOne Platform 2017! There is a *massive* amount of stuff to cover, especially in light of SpringOne Platform, so let's get to it!

-   Spring Cloud co-founder [Spencer Gibb just announced Spring Cloud Finchley M5](https://spring.io/blog/2017/12/05/spring-cloud-finchley-m5-released)
-   Spring Tools lead Martin Lippert [just announced the first previews of Spring Tool Suite 4](https://spring.io/blog/2017/12/04/spring-tools-4-public-beta-released) and the new IDE-agnostic language servers. Also, I did a special (and early!) [Spring Tip that looked at these topics](https://spring.io/blog/2017/12/04/spring-tips-spring-tool-suite-4).
-   Spring Batch lead Michael Minella [just announced Spring Batch 4.0.0](https://spring.io/blog/2017/12/04/spring-batch-4-0-0-release-is-now-available)
-   Dr. Dave Syer has updated the [Spring Security and Angular\_](https://spring.io/guides/tutorials/spring-security-and-angular-js/) and [Spring Boot and OAuth tutorials](https://spring.io/guides/tutorials/spring-boot-oauth2/)
-   Spring ninja Greg Turnquist just [announced Spring Session updates for Core, MongoDB, and Apache Geode](https://spring.io/blog/2017/12/03/spring-session-core-mongodb-geode-2-0-0-rc2-released).
-   Spring Cloud Data Flow lead and co-founder [Dr. Mark Pollack just announced Spring Cloud Data Flow 1.3.0.M3](https://spring.io/blog/2017/12/01/spring-cloud-data-flow-for-cloud-foundry-1-3-0-m3-released).
-   Spring Integration lead Gary Russell just [announced Spring for Apache Kafka 2.1.0 and 1.3.2, and 2.0.2](https://spring.io/blog/2017/12/01/spring-for-apache-kafka-2-1-0-release-and-1-3-2-2-0-2-available)
-   Spring Boot co-founder and lead [Phil Webb just announced Spring Boot 2.0.M7](https://spring.io/blog/2017/11/30/spring-boot-2-0-0-m7-available-now)
-   Spring IO Platform [lead Andy Wilkinson just announced Spring IO Platform Brussels SR6](https://spring.io/blog/2017/11/29/spring-io-platform-brussels-sr6)
-   Spring Integration ninja Artem Bilan [just announced Spring Integration 5.0.GA](https://spring.io/blog/2017/11/29/spring-integration-5-0-ga-available)
-   Last week, I looked at the [new Spring Cloud Gateway project in a Spring Tips installment](https://spring.io/blog/2017/11/29/spring-tips-spring-cloud-gateway)
-   Spring Integration ninja just released [Spring AMQP 1.7.5 and 2.0.1](https://spring.io/blog/2017/11/28/spring-amqp-1-7-5-and-2-0-1-available)
-   not one to rest on his laurels, [Spring REST Docs Andy Wilkinson just announced Spring REST Docs 2.0.0](https://spring.io/blog/2017/11/28/spring-rest-docs-2-0-0-release)
-   Spring Security lead [Rob Winch just announced Spring Security 5.0](https://spring.io/blog/2017/11/28/spring-security-5-0-0-release-released)
-   Spring REST Docs lead [Andy Wilkinson *also* announced Spring REST Docs 1.2.3](https://spring.io/blog/2017/11/28/spring-rest-docs-1-2-3-release)
-   Spring Boot ninja [Stéphane Nicoll just announced Spring Boot 1.5.9](https://spring.io/blog/2017/11/28/spring-boot-1-5-9-available-now)
-   Spring Data ninja Mark Paluch pulled together an amazing look at [binding applications to Hashicorp's Vault with Spring in Cloud Foundry](https://spring.io/blog/2017/11/28/binding-applications-to-hashicorp-s-vault-with-spring-in-cloud-foundry)
