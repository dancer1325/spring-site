# Second Service Release For Spring Data Release Train Codd Released

_Releases | Oliver Drotbohm |  April 16, 2014 | 0 Comments_

I am happy to announce the second service release of the Spring Data release train named [Codd](http://en.wikipedia.org/wiki/Edgar_F._Codd). The full list of participating modules looks as follows:

-   Spring Data Commons 1.7.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-commons/1.7.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/commons/docs/1.7.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/commons/docs/1.7.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/commons/docs/1.7.2.RELEASE/changelog.txt)
-   Spring Data JPA 1.5.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-jpa/1.5.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/jpa/docs/1.5.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/jpa/docs/1.5.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/jpa/docs/1.5.2.RELEASE/changelog.txt)
-   Spring Data MongoDB 1.4.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-mongodb/1.4.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/mongodb/docs/1.4.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/mongodb/docs/1.4.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/mongodb/docs/1.4.2.RELEASE/changelog.txt)
-   Spring Data Neo4j 3.0.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-neo4j/3.0.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/neo4j/docs/3.0.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/neo4j/docs/3.0.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/neo4j/docs/3.0.2.RELEASE/changelog.txt)
-   Spring Data Solr 1.1.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-solr/1.1.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/solr/docs/1.1.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/solr/docs/1.1.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/solr/docs/1.1.2.RELEASE/changelog.txt)
-   Spring Data REST 2.0.2 - [Artifacts](http://repo.spring.io/libs-release/org/springframework/data/spring-data-rest-webmvc/2.0.2.RELEASE) - [JavaDocs](http://docs.spring.io/spring-data/rest/docs/2.0.2.RELEASE/api) - [Documentation](http://docs.spring.io/spring-data/rest/docs/2.0.2.RELEASE/reference/htmlsingle) - [Changelog](http://docs.spring.io/spring-data/rest/docs/2.0.2.RELEASE/changelog.txt)

For this release we've squashed [37 tickets](https://jira.spring.io/issues/?filter=14646). As the release contains a few important bug fixes it is a recommended upgrade for all Codd users. Also, users of the previous release train Babbage are encouraged to upgrade.

The next release following in two week will be the first release candidate of our next release train called [Dijkstra](http://en.wikipedia.org/wiki/Edsger_W._Dijkstra). A preliminary schedule is available at the [wiki page](https://github.com/spring-projects/spring-data-commons/wiki/Release-Train-Dijkstra) for the release. We're currently in the planning phase and will happily consider the input you cast in JIRA tickets.
