# This Week in Spring (JavaOne 2015 Edition) - October 27, 2015

_Engineering | Josh Long |  October 28, 2015 | 0 Comments_

Welcome to another installation of *This Week in Spring*! This week I'm at JavaOne 2015 in San Francisco along with the rest of the Pivotal team. This week the Pivotal Spring team is out in full force, come stop by and say hi!

There are some great new SpringOne2GX 2015 recordings on line as well as some great community content this week so let's get to it!

-   Spring Cloud cofounder's [Spencer Gibb and Dr. David Syer's talk](http://spring.io/blog/2015/10/22/springone2gx-2015-replay-developer-experience-with-spring-cloud) on the Spring Cloud Developer experience from SpringOne2GX 2015!
-   Spring tools ninja Martin Lippert [continues his series with this third installment on using Spring Boot Dashboard in STS](http://spring.io/blog/2015/10/22/the-spring-boot-dashboard-in-sts-part-3-spring-boot-devtools-reloaded)
-   Gary Russell has just announced a few [new maintenance releases of Spring Integration](uhttp://spring.io/blog/2015/10/27/spring-integration-maintenance-releases-available)
-   Check out this upcoming IntelliJ IDEA webinar on the [new Spring framework and Spring Boot support in IntelliJ IDEA with Spring ninja Stéphane Nicoll](http://info.jetbrains.com/IDEA-Webinar-November2015-registration.html)
-   this post by Rajith Delantha walks through [using Spring Security OAuth with Spring Boot](http://www.javacodegeeks.com/2015/10/spring-boot-oauth2-security.html)
-   My article on the [The Power, Patterns, and Pains of Microservices of microservices is now up on DZone](https://dzone.com/articles/the-power-patterns-and-pains-of-microservices)
-   Giau Ngo has a good example mapping on using JPA many-to-many relationship mapping [with Spring Boot, MySQL and Maven](http://hellokoding.com/jpa-many-to-many-relationship-mapping-example-with-spring-boot-maven-and-mysql/)
-   Papo Reto has a nice in Portguese-language video - I'm pretty sure - [on using Spring Boot on Heroku](https://www.youtube.com/watch?v=ROw92pLEYmA)

-   Do you [want to get Stormpath working with Spring Boot](http://docs.stormpath.com/java/spring-boot-web/quickstart.html)? Check out this handy getting-started example!
-   I thought this was a cool [blog on using Cloud Watch with Cloud Foundry](http://www.haydonryan.com/category/cloudfoundry/)
-   Roberto Venanzi of *The Spring Side blog* has an interesting Italian-language post on [using Spring Boot with JSF](http://www.thespringside.com/?p=19)
