# This Week in Spring - November 21st, 2017

_Engineering | Josh Long |  November 21, 2017 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! Can you beleive we're less than six weeks away from 2018? Time sure flies when you're having fun!

This week I'm in Los Angeles talking to community members and then taking in the Thanksgiving holiday here in the US surrounded by friends and family. Thanksgiving is a chance to reflect on that for which we are thankful. I am sure that I speak for all of us here on the Spring and Pivotal teams when I say that we are *thankful* for you. Thank you for making this the most amazing, dynamic, exhilarating and interesting community in tech. If you're in the US, happy Thanksgiving! Either way, we've got a lot to cover this week so let's get to it.

-   Spring Cloud co-founder and co-lead Spencer Gibb has just announced [Spring Cloud Finchley M4](https://spring.io/blog/2017/11/18/spring-cloud-finchley-m4-released). This new release updates everything to support Spring Boot 2.0.0.M6. As always, this is a milestone release and represents an ideal opportunity for you to try the bits.
-   In this last week's installment of [*Spring Tips*](http://twitter.com/SpringTipsLive) I looked at some of the more exotic features of the [Spring Boot build plugin](https://spring.io/blog/2017/11/15/spring-tips-the-spring-boot-build-plugin) including *fat* `.jar`s and *thin* `.jar`s.
-   Check out Spring Boot community legend Nicolas Fränkel's talk introducing [the under-the-hood mechanisms of Spring Boot](https://www.youtube.com/watch?v=zSJFx0iyrq4)
-   Abhinav Mehla introduces the availability [of MongoDB Enterprise on Pivotal Cloud Foundry](https://www.mongodb.com/blog/post/on-demand-mongodb-enterprise-on-pivotal-cloud-foundry). This is awesome!
-   This is such good news! Rod Johnson's startup, Atomist, which aims to automate the end-to-end lifecycle around application development, [has launched and announced their automation platform](https://sdtimes.com/atomist-announces-new-automation-platform-targeted-developers/). Congratulations Rod! Here's another great post [on Atomist's emerging from stealth](https://venturebeat.com/2017/11/15/atomist-launches-to-help-devs-automate-repository-creation-and-code-deployment/) . If you want to hear more, join [him at SpringOne Platform](http://springoneplatform.io)
-   Are you in the St. Louis area? Join fellow Spring Developer Advocate Mario Gray and me [for Bootiful Testing](https://twitter.com/steve_womack/status/931653007221121025) December 13th, 2017.
-   Auth0 has an interesting post that introduces using Gradle to build a Spring 5 application [that manually embeds Apache Tomcat 5](https://auth0.com/blog/spring-5-embedded-tomcat-8-gradle-tutorial/). (I think you'd probably be better off using Spring Boot 2 which auto-configures Apache Tomcat if its on the CLASSPATH).
-   Our @SpringCentral Twitter account hilariously broke the [news before even I realized](https://twitter.com/springcentral/status/931570478187126785), but I recently updated my [DZone Microservices in Java refcard and it's available now](https://dzone.com/refcardz/learn-microservices-in-java).
-   Spring Data lead Oliver Gierke [shares this sample application](https://st.inf.tu-dresden.de/SalesPoint/), *Salespoint*, which is a Spring Boot-based POS framework use for teaching purposes at Technische Universität Dresden (which, I assume, means the technical university of Dresden, Germany), where Oliver teaches. Here's [the project home](http://bit.ly/2zbY0z5) and [the repository](http://bit.ly/2zaPTmo).
-   This post introduces how T-Mobile refactored their major monolithic application [and introduced Cloud Foundry to change their culture](https://www.cloudfoundry.org/t-mobile-integrates-cloud-foundry-create-new-culture/?_lrsc=2e064143-6b50-44ec-a22d-ece03ddfe607).
-   OAuth expert Will Tran discusses [enforcing microservices’ security policies with OAuth 2.0](https://www.infoq.com/presentations/oauth2-microservices-security) in this fantastic InfoQ video.
-   Neo4J legend Michael Hunger walks us through a [network analysis using Neo4J's Cypher language and a dataset based on Devoxx](http://gist.asciidoctor.org/?dropboxs-56rhbvey5cp8c53%2Fdevoxx-conference-graph.adoc%3Fdl%3D0) conference talks. If you're using Neo4j (with, perhaps, Spring Data Neo4J) then this is a masterclass of a read for you.
-   I love it when cloud platforms demonstrate Spring Boot applications.

In this blog, [Oracle demonstrates using Spring Cloud Stream](https://medium.com/oracledevs/spring-cloud-stream-and-kafka-based-microservices-on-oracle-cloud-9889732149a).
