# Spring Tools 4.19.0 released

_Releases | Martin Lippert |  June 16, 2023 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.19.0 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

#### [](#updates-to-the-spring-tools-4-for-eclipse-distribution)updates to the Spring Tools 4 for Eclipse distribution

-   updated to Eclipse 2023-06 release ([new and noteworthy](https://eclipseide.org/release/noteworthy/))

#### [](#fixes-and-improvements)fixes and improvements

-   *(Spring Boot)* new: Spring Data Repositories: Validate ID type ([#457](https://github.com/spring-projects/sts4/issues/457))
-   *(Spring Boot)* fixed: upgrading from Spring Boot 3.0.x to 3.1.0 doesn't work ([#1051](https://github.com/spring-projects/sts4/issues/1051))
-   *(Spring Boot)* fixed: replace internals of symbols machinery with more comprehensive model of spring components ([#1006](https://github.com/spring-projects/sts4/issues/1006))

Detailed changes can be found in the release notes: [https://github.com/spring-projects/sts4/releases/tag/4.19.0.RELEASE](https://github.com/spring-projects/sts4/releases/tag/4.19.0.RELEASE)

#### [](#downloads)downloads

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Spring Tools 4.19.1 is scheduled to be released in mid August 2023.

Enjoy!
