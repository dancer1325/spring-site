# Spring Session 2021.2 Available

_Releases | Rob Winch |  May 18, 2022 | 0 Comments_

On behalf of the team, I’m please to announce the release of Spring Session `2021.2`. This release upgrades to [Spring Data 2021.2](https://spring.io/blog/2022/05/13/spring-data-2021-2-and-2022-0-m4-released) and provides dependency upgrades, minor enhancements, bug fixes, and strategic changes to prepare Spring Session and it’s users for the next generation of Spring Session built on Spring Framework 6.
