# This Week in Spring - Tuesday September 23rd, 2014

_Engineering | Josh Long |  September 23, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This morning, I had the joy of presenting with my pal [Joram Barrez](http://twitter.com/jbarrez) on using the [Activiti BPMN workflow engine](http://activiti.org) with Spring (and Spring Boot). That talk should be online in the next few weeks or so. I'll also be co-presenting with Joram this week at the [Alfresco Summit](http://summit.alfresco.com/) (on the very same topic). If you missed SpringOne2GX 2014, check out the [Day 1](http://spring.io/blog/2014/09/10/springone2gx-2014-day-1) and [Day 2](http://spring.io/blog/2014/09/15/springone2gx-2014-day-2) wrap ups where can learn about NetFlix, Spring Boot and microservice architecture, among many other things.

1.  The replay of the webinar talk that I gave last week, [*Bootiful Microservices with Spring Boot (and Spring Cloud)*](http://www.youtube.com/watch?v=VPtlZLdm7Nc&list=PLgGXSWYM2FpNmY5a1MuomSvvovSWHWoAR), is now available online. This talk - because of Boot and the subject matter - got a *lot* of attention and so this was actually available online within a few *days* (not the usual week or two) of my having presented it! Enjoy!

-   I'm really digging some of the posts that've come out of the Codeleak.pl blog of late! First up, this post on [unit testing e-mail integration code written atop Spring Boot](http://blog.codeleak.pl/2014/09/testing-mail-code-in-spring-boot.html) - check it out!
-   Shortly after the microservice talk, I received a tweet pointing me to a recently [added talk on how a Polish company builds microservices with Spring](http://www.youtube.com/watch?v=f_fGwa2-rMk&list=UUwAKjrTAPEbBzJ-tSWwGPMQ). The talk's *mostly* high level, with a spattering of Spring here and there. This is a cool talk and, at 30 minutes, easy to consume quickly).
-   Rafał Borowiec over on the Codeleak.pl site has done a really nice job! [He followed up his previous post (on unit testing e-mail) with a look at Spring Boot's `@ConfigurationProperties` support](http://blog.codeleak.pl/2014/09/using-configurationproperties-in-spring.html)
-   Rafał *also* put together this nice post [on using *validation groups* to validate Spring MVC models](http://blog.codeleak.pl/2014/08/validation-groups-in-spring-mvc.html) and this other post on testing that [a given Spring MVC model has registered global errors](http://blog.codeleak.pl/2014/08/spring-mvc-test-assert-given-model-attribute-global-errors.html)
-   Biju Kunjummen has put together a nice post [on customizing `HttpMessageConverter`s in a Spring Boot application](http://www.javacodegeeks.com/2014/09/customizing-httpmessageconverters-with-spring-boot-and-spring-mvc.html)
-   Adrian Matei has put together a *very* nice post on [building Spring Batch applications with Java configuration and Spring Boot](http://www.javacodegeeks.com/2014/09/spring-batch-tutorial-with-spring-boot-and-java-configuration.html)
-   Netflix's (and our pal) Tomas Lin has just published a nice post on [how to expose static resources (like JavaScript assets) from *different environments* in a configurable way](http://fbflex.wordpress.com/2014/09/22/spring-boot-and-webjars-overwriting-configuration-during-runtime/)
-   My pal and Spring ninja Greg Turnquist has a nice post looking at his [very *imminent* book on Spring Boot](http://blog.greglturnquist.com/category/learning-spring-boot). I can't wait! Greg's a smart and passionate guy, and if you ever read his *Spring Python* book, you know he's a fun read, too!
-   The *Develop and Conquer* blog has a great post on consuming [Active Directory LDAP from Spring Security in a Spring Boot application](http://raymondhlee.wordpress.com/2014/09/20/setup-spring-security-with-active-directory-ldap-in-spring-boot-web-application/) - check it out! He has quite a few nice posts (like using ActiveMQ and STOMP support behind Spring 4's websocket support) on other topics, too, so do check it out!
