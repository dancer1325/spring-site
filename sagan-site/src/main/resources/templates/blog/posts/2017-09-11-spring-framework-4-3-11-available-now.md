# Spring Framework 4.3.11 available now

_Releases | Stéphane Nicoll |  September 11, 2017 | 0 Comments_

It is my pleasure to announce that Spring Framework [4.3.11](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=16252) [is available now](http://projects.spring.io/spring-framework).

This maintenance release is the base for the upcoming Spring Boot 1.5.7 maintenance release.
