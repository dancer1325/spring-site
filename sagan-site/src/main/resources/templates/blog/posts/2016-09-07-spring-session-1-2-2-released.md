# Spring Session 1.2.2 Released

_Releases | Rob Winch |  September 07, 2016 | 0 Comments_

I'm pleased to announce the release of Spring Session 1.2.2.RELEASE. [This release contains](https://github.com/spring-projects/spring-session/issues?q=milestone%3A1.2.2) numerous bug fixes and trivial enhancements. Some of the highlights include:

-   Fixes and improvements for supporting [multiple sessions](http://docs.spring.io/spring-session/docs/current/reference/html5/#httpsession-multi) in the same browser when working with Tomcat 8.5 [#605](https://github.com/spring-projects/spring-session/issues/605) [#611](https://github.com/spring-projects/spring-session/issues/611) [#615](https://github.com/spring-projects/spring-session/issues/615)
-   Improvements for JDBC support to work with DB2 [#600](https://github.com/spring-projects/spring-session/pull/600)
-   [Unregistered GemFire Instantiator causes deserialization issues on app restarts](https://github.com/spring-projects/spring-session/issues/594)
