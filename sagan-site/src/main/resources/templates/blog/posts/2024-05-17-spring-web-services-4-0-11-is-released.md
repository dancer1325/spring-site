# Spring Web Services 4.0.11 is released

_Releases | Corneil du Plessis |  May 17, 2024 | 0 Comments_

Greetings Spring community,

The Spring Web Services team has released **4.0.11** which adds updates to springframwork to 6.0.20 and includes community contributions to improve JMS tests and Wss4jHandler.

For more details, check out our official [GitHub release](https://github.com/spring-projects/spring-ws/releases/tag/v4.0.11). This includes links to all the related tickets as well as the source code.

We have also adopted [GitHub releases](https://github.com/spring-projects/spring-ws/releases) as our avenue for tracking changes in a given release. All current and prior releases of Spring WS are now captured in the list of releases on GitHub. Feel free to check them out.

The artifacts are hosted on Maven Central so grab 'em while they’re hot!
