# Spring Security 5.4.1, 5.3.5, 5.2.7, 5.1.13, 5.0.19, 4.2.19 Released

_Releases | Rob Winch |  October 08, 2020 | 3 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.3.5 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.5.RELEASE)), 5.2.7 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.7.RELEASE)), 5.1.13 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.13.RELEASE)) , 5.0.19 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.19.RELEASE)), 4.2.19 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.19.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
