# Spring 3.1.0 M1 Released

_Releases | Chris Beams |  February 11, 2011 | 0 Comments_

The first milestone of Spring 3.1 is now available from our http://maven.springframework.org/milestone Maven repository or for direct download from our [community download page](http://www.springsource.com/download/community). This release includes new features such as:

-   Bean definition profiles
-   Enhancements to Java-based container configuration
-   Caching abstraction
-   new c: XML namespace
-   Improvements to Spring MVC

Stay tuned to the [SpringSource Blog](http://blog.springsource.com) over the coming week for a series of posts covering what's new in 3.1.0.M1.
