# Spring JavaConfig 1.0.0.M4 Released

_Releases | Chris Beams |  November 07, 2008 | 0 Comments_

Dear Spring Community,  
We are pleased to announce that Spring JavaConfig 1.0.0.M4 is now available.  
[Download](http://www.springsource.com/download/community?project=Spring%20JavaConfig) | [Reference Docs](http://static.springframework.org/spring-javaconfig/docs/1.0.0.M4/reference/html) | [API Docs](http://static.springframework.org/spring-javaconfig/docs/1.0.0.M4/javadoc-api/)  
  
**Major Highlights**  
  

-   @AnnotationDrivenTx - support for declarative transaction management
-   @AnnotationDrivenConfig - support for @Autowired, @Resource, et al.
-   @ComponentScan - scan for @Component classes directly from JavaConfig
-   @AspectJAutoProxy - first-class support for @Aspect beans
-   @MBeanExport - first class support for exporting JMX MBeans
-   Complete PetClinic sample now available with distribution demonstrating use of JavaConfig
-   Improved support for externalized values with @ExternalValue and @PropertiesValueSource
-   @ImportXml - bootstrap Spring XML bean definitions from JavaConfig
-   Improved error handling
-   ... and dozens of other [resolved issues](http://jira.springframework.org/browse/SJC/fixforversion/10710)

  
  
Please give this milestone a test drive, and provide your feedback through the Spring JavaConfig [Forum](http://forum.springframework.org/forumdisplay.php?f=48) or [Issue Tracker](http://jira.springframework.org/browse/SJC). For more information, visit the [Spring JavaConfig Home](/javaconfig).

Chris Beams
