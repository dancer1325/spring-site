# Spring Framework 5.2.5 available now

_Releases | Stéphane Nicoll |  March 24, 2020 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework 5.2.5 is available now.

Spring Framework 5.2.5 includes [over 50 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.5.RELEASE). This release adds support for the recently released Java 14.

As usual, we'll follow up shortly with Spring Boot `2.2.6`.
