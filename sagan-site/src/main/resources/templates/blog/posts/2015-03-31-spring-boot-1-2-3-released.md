# Spring Boot 1.2.3 released

_Releases | Phil Webb |  March 31, 2015 | 2 Comments_

Spring Boot 1.2.3 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [small number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.3) and is a recommended upgrade for all users.
