# Spring for Apache Kafka 1.1.3 available now

_Releases | Artem Bilan |  February 06, 2017 | 0 Comments_

It is my pleasure to announce that the Spring for Apache Kafka [1.1.3](https://github.com/spring-projects/spring-kafka/milestone/14?closed=1) maintenance release [is available now](http://projects.spring.io/spring-kafka/).

As usual, thanks to the community for any feedback and contribution as always. Looking forward for more!

This release contains several bug fixes, including proper offset commit handling when using a `BatchListener`; therefore an upgrade is highly recommended.

Right now `master` has been switched to the version `2.0` for Java 8 and Spring Framework 5.0 code base. We have some plans for high-level API for [Kafka Streams](https://kafka.apache.org/documentation/streams) and [Reactor Kafka](https://github.com/reactor/reactor-kafka) support.
