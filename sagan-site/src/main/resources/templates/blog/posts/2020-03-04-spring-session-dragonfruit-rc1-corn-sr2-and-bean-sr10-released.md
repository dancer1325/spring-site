# Spring Session Dragonfruit-RC1, Corn-SR2 and Bean-SR10 Released

_Releases | Eleftheria Stein-Kousathana |  March 04, 2020 | 0 Comments_

On behalf of the community I’m pleased to announce the releases of Spring Session `Dragonfruit-RC1`, `Corn-SR2` and `Bean-SR10`.

## [](#spring-session-dragonfruit-rc1)[](#spring-session-dragonfruit-rc1)Spring Session Dragonfruit-RC1

The `Dragonfruit-RC1` release is based on:

-   Spring Session core modules `2.3.0.RC1`
    
-   Spring Session Data Geode `2.3.0.RC1`
    
-   Spring Session Data MongoDB `2.3.0.RC1`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/73?closed=1).

## [](#spring-session-corn-sr2)[](#spring-session-corn-sr2)Spring Session Corn-SR2

The `Corn-SR2` release is based on:

-   Spring Session core modules `2.2.2.RELEASE`
    
-   Spring Session Data Geode `2.2.3.RELEASE`
    
-   Spring Session Data MongoDB `2.2.3.RELEASE`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/75?closed=1).

## [](#spring-session-bean-sr10)[](#spring-session-bean-sr10)Spring Session Bean-SR10

The `Bean-SR10` release is based on:

-   Spring Session core modules `2.1.11.RELEASE`
    
-   Spring Session Data Geode `2.1.9.RELEASE`
    
-   Spring Session Data MongoDB `2.1.7.RELEASE`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/74?closed=1).
