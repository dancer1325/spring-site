# Spring GemFire 1.0.1 Released for Java and .NET

_Releases | Costin Leau |  April 26, 2011 | 0 Comments_

Dear Spring Community,

I am pleased to announce that 1.0.1 GA release of the Spring GemFire project is now available for both Java and .NET! The Spring GemFire project aims to make it easier to build Spring-powered highly scalable applications using GemFire as distributed data management platform.

Download it now: [Spring GemFire for Java](http://www.springsource.com/download/community?project=Spring%20GemFire&version=1.0.1.RELEASE) | [Spring GemFire for .NET](http://www.springsource.com/download/community?project=Spring%20GemFire%20for%20.NET&version=1.0.1)

**Java**: [JavaDocs](http://static.springsource.org/spring-gemfire/docs/1.0.1.RELEASE/api/) | [Reference Documentation](http://static.springsource.org/spring-gemfire/docs/1.0.1.RELEASE/reference/html/) | [Changelog](http://static.springframework.org/spring-gemfire/docs/1.0.1.RELEASE/changelog.txt) **.NET** :  [ApiDocs](http://static.springsource.org/spring-gemfire-net/docs/1.0.1.GA/api/html/)  | [Reference Documentation](http://static.springsource.org/spring-gemfire-net/docs/1.0.1.GA/reference/html/) | [Changelog](http://static.springsource.org/spring-gemfire-net/changelog.txt)

The Spring GemFire project aims to make it easier to build Spring-powered highly scalable applications using [GemFire](http://www.gemstone.com/products/gemfire) as distributed data management platform.

To learn more about the project, visit the Spring GemFire [Homepage](http://www.springsource.org/spring-gemfire).

1.0.1 is mainly a maintenance release providing several enhancements on the configuration front. See the changelog for more details.

Looking forward to your feedback on this forum or in the [issue tracker](https://jira.springsource.org/browse/SGF).
