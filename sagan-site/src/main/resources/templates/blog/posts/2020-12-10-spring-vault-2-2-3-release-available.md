# Spring Vault 2.2.3.RELEASE available

_Releases | Mark Paluch |  December 10, 2020 | 0 Comments_

On behalf of the community, I’d like to announce the availability of Spring Vault service release `2.2.3.RELEASE`.

This release ships with mostly bugfixes, dependency upgrades, and selected features.

For a complete list of changes, see the [2.2.3.RELEASE changelog](https://github.com/spring-projects/spring-vault/releases/tag/2.2.3.RELEASE).
