# Spring for GraphQL 1.3.5 released

_Releases | Brian Clozel |  April 17, 2025 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.3.5 is now available on Maven Central.

1.3.5 closes [12 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.3.5). This version will ship with Spring Boot 3.3.11 and 3.4.5, [to be released next week](https://calendar.spring.io/).

### [](#how-can-you-help)How can you help?

If you have general questions, please ask on [stackoverflow.com](https://stackoverflow.com) using the [`spring-graphql` tag](https://stackoverflow.com/tags/spring-graphql).
