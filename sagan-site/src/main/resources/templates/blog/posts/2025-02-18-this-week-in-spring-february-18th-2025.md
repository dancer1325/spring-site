# This Week in Spring - February 18th, 2025

_Engineering | Josh Long |  February 18, 2025 | 0 Comments_

Hi, Spring fans! It's the week of February 18th, 2025, and you know what that means? ConFoo and Devnexus are nearly here! Next week, I'll be traveling to amazing Montreal, Canada, to speak at the ConFoo show for an amazing community, poutine, and technology! Then, a few days later, it's off to awesome Atlanta, Georgia, where I'll be doing a workshop, a talk on Spring AI with Dr. Mark Pollack, and my usual Spring Boot latest-and-greatest talk. If you're going to either of those shows (why not both?), please don't hesitate to say "salut" or "hello"!

There's a lot of great stuff in this week's roundup, so let's dive right into it!

-   [Spring for GraphQL 1.3.4 released](https://spring.io/blog/2025/02/18/spring-for-graphql-1-3-4-released)
-   [Spring for Apache Pulsar 1.1.9 and 1.2.3 released](https://spring.io/blog/2025/02/18/spring-for-apache-pulsar-1-1-9-and-1-2-3-are-now-available)
-   TL;DR: Anthropic formalized a specification for exposing modules to LLMs called *Model Context Protocol* (MCP), in effect teaching them new tricks. In November and December, we released an experimental project called Spring AI MCP. That core abstraction has since been donated and is now the official Model Context Protocol Java SDK. Spring AI M6, in turn, depends on that donated Java SDK, extending it with Spring Boot autoconfigurations and integrations. [Want to learn more about MCP? Check out this amazing blog](https://spring.io/blog/2025/02/14/mcp-java-sdk-released-2), co-authored by the Spring AI and Anthropic teams.
-   And here's [the Spring AI 1.0.0 M6 announcement](https://spring.io/blog/2025/02/14/spring-ai-1-0-0-m6-released)
-   And the tweet from Anthropic's David Soria Parra, founder of the MCP initiative, affirming that [Spring AI's core MCP support is now the official Anthropic Java SDK](https://x.com/dsp_/status/1890469048783089731)!
-   Speaking of [Spring AI M6 and the new official MCP support, I did a video you might like](https://www.youtube.com/watch?v=cE1h-rC2o2U&t=1s)
-   Oh, *one more thing*! All this new and novel Spring AI support for MCP is already on [the Spring Initializr](https://start.spring.io)!
-   Unsurprisingly, I love this blog, called: [Accelerate Spring AI Development with Effortless Privacy from CodeGate](https://dev.to/stacklok/accelerate-spring-ai-development-with-effortless-privacy-from-codegate-13hn)
-   [Honored to appear (twice!) in the 100 most-watched software engineering talks of 2024](https://www.techtalksweekly.io/p/100-most-watched-software-engineering) ranking
-   IntelliJ has some cool [new features, including the ability to debug GraalVM native images](https://www.youtube.com/watch?v=St68kE-Vtj0)
-   Catherine Edelveis has a nice video [on creating dynamic SQL queries with Spring Boot](https://www.youtube.com/watch?v=9AlXoh_ghzE)
