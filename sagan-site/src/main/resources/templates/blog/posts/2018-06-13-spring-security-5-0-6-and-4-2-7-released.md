# Spring Security 5.0.6 and 4.2.7 Released

_Releases | Rob Winch |  June 13, 2018 | 0 Comments_

On behalf of the community, I am pleased to announce that the Spring Security 5.0.6 ([changelog](https://github.com/spring-projects/spring-security/milestone/116?closed=1)) and 4.2.7 ([changelog](https://github.com/spring-projects/spring-security/milestone/117?closed=1)) have been released. The releases primarily delivers bug fixes and dependency version updates along with some minor improvements. The releases will be found in the upcoming Spring Boot maintenance releases coming later this week.
