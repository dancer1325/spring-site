# Spring Data Neumann M4 released

_Releases | Jens Schauder |  March 11, 2020 | 0 Comments_

On behalf of the community, I'd like to announce the availability of the Spring Data `Neumann M4` milestone release. It is planned to be the last milestone before the first release candidate.

`Neumann M4` ships with [69 tickets fixed](https://jira.spring.io/issues/?filter=16739). It is built on top of Spring Framework `5.2.4` and will be picked up by Spring Boot `2.3.0 M3` for your convenience.
