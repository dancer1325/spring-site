# Spring Social 1.0.0 Milestone 1 is Now Available

_Releases | Craig Walls |  November 03, 2010 | 1 Comment_

Dear Spring Community,

We are pleased to announce that the first milestone release of [Spring Social](http://www.springframework.org/spring-social) 1.0 is now available!

Spring Social enables development of social-ready applications, linking your application's users to the social networks where they hang out. What's more the capabilities of Spring Social are illustrated by the [Greenhouse](http://greenhouse.springsource.org) reference application.

Spring Social 1.0.0-M1 contains the following core features:

-   Social templates for interaction with Twitter, Facebook, LinkedIn, TripIt, and Greenhouse
-   An OAuth-enabled connection factory for signing RestTemplate requests with OAuth authentication details
-   A web argument resolver for extracting Facebook user ID and access token information in a Spring MVC controller

To get you started, I've posted [Socializing Spring Applications](http://blog.springsource.com/2010/11/03/socializing-spring-applications/), an introduction to Spring Social.

This milestone release is just the beginning. We look forward to your feedback to help us shape Spring Social.
