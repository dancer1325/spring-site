# Spring for GraphQL 1.1.2 released

_Releases | Brian Clozel |  February 21, 2023 | 0 Comments_

I'm pleased to announce that Spring for GraphQL 1.1.2 is now available on Maven Central.

This maintenance release includes [12 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.1.2). This version will be shipped with Spring Boot 3.0.2, [to be released later this week](https://calendar.spring.io/).

Note, we've also worked with [the GraphQL Java team](https://www.graphql-java.com/) to publish a [brand new getting started guide for this project](https://spring.io/guides/gs/graphql-server/).
