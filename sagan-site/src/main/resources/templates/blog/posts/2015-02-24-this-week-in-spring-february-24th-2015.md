# This Week in Spring - February 24th, 2015

_Engineering | Josh Long |  February 24, 2015 | 1 Comment_

Welcome to another installment of *This Week in Spring*! We've got a *lot* to cover so let's get to it!

1.  Our pal Adam Koblentz (from ZeroTurnaround) put up this [great post introducing building a websocket application with Spring Boot and JRebel](https://www.voxxed.com/blog/2015/02/no-snail-herding-websocket-addition-jrebel-spring-boot/). Check it out!
2.  Check out this replay of Mark Fisher, Dr. Mark Pollack, and Sabby Anandan's webinar [introducing Spring XD - A Platform for data at scale and developer productivity](http://spring.io/blog/2015/02/20/webinar-replay-spring-xd-a-platform-for-data-at-scale-and-developer-productivity)
3.  A huge part of the Pivotal Data Suite, of course, is Spring XD. Last week I surfaced some of the amazing Spring XD wiki content on [the new stream processing supports in Spring XD 1.1](http://spring.io/blog/2015/02/20/stream-processing-in-spring-xd-1-1). Check out [the Wiki page](https://github.com/spring-projects/spring-xd/wiki/Creating-a-Data-Stream-Processor-Module#reactor-streams) for a more detailed look by Spring XD ninja Ilayaperumal Gopinathan.
4.  Spring Cloud co-lead Spencer Gibb has been improving the Spring Cloud Netflix integration. Check out this example [demonstrating using RxJava's `Observable<T>` return-values from Spring MVC](https://github.com/spencergibb/myfeed/blob/master/myfeed-ui/src/main/java/myfeed/ui/UiController.java#L38-L57). [Here are the changes](https://github.com/spencergibb/myfeed/commit/a301aba6139fce7a87a1aab1d1311e6ceb8adfb7).
5.  March Webinars are here, and it's packed with Javascript, Spring Boot and Spring Cloud.

This killer lineup starts with Dr. Dave Syer's epic VII part blog series on Angular - lightweight API gateway - Spring Security - made into a [webinar on March 10th](https://spring.io/blog/2015/02/24/webinar-a-single-page-application-with-spring-security-and-angular-js). Then Scott Deeg will examine the same thing, except with Google's Polymer on the frontend, [on a webinar March 24th](https://spring.io/blog/2015/02/24/webinar-building-a-secure-polymer-app-with-spring-backend). Lastly, Julien Dubois will introduce the AngularJS+Boot application generator, JHipster on [March 17th, 2015](https://spring.io/blog/2015/02/24/webinar-jhipster-for-spring-boot).  
6\. Alvin Reyes's posts introducing how to configure Apache Tomcat's [`server.xml`](http://examples.javacodegeeks.com/enterprise-java/tomcat/tomcat-server-xml-configuration-example/) and [`users.xml`](http://examples.javacodegeeks.com/enterprise-java/tomcat/tomcat-users-xml-configuration-example) are *wonderful* resources. 7. Last week, we at Pivotal announced some big changes; we've open-sourced our Pivotal Data Suite! Check out this [InfoQ post on the subject](http://www.infoq.com/news/2015/02/pivotal-big-data-suite) 10. Spring IO Platform lead Andy Wilkinson has just released improved support for [Maven Bill-of-Material dependency management using Gradle](http://spring.io/blog/2015/02/23/better-dependency-management-for-gradle) 11\. Eclipse foundation member John Arthorne and Spring Tool Suite lead Martin Lippert's talk, [Eclipse IDE to the Cloud-Based Era of Developer Tooling](https://old.parleys.com/share_channel.html#play/543f97d7e4b08dc7823e54cc) from JavaOne 2014 is available online and well worth a watch! 12. [The SD Times article on the open-sourcing of the Pivotal Data Suite](http://sdtimes.com/pivotal-pivots-open-source/) is worth a read, too. 13.
