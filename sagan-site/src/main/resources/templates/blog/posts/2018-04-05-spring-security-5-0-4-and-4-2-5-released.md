# Spring Security 5.0.4 and 4.2.5 Released

_Releases | Joe Grandja |  April 05, 2018 | 0 Comments_

I’m pleased to announce the releases of Spring Security 5.0.4 and 4.2.5. Both releases primarily deliver bug fixes and dependency version updates along with some minor improvements.

For a complete list of changes, please refer to the 5.0.4 [changelog](https://github.com/spring-projects/spring-security/milestone/112?closed=1) and 4.2.5 [changelog](https://github.com/spring-projects/spring-security/milestone/109?closed=1).
