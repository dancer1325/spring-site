# This Week in Spring - August 7th, 2018

_Engineering | Josh Long |  August 08, 2018 | 0 Comments_

HI Spring fans! This week I'm in [Miami for the Java User Group](https://www.meetup.com/Miami-Java-User-Group/?_cookie-check=ijh6mnmDVvhGmEyx) and at home working on new episodes of Spring Tips. Just trying to keep moving! It's been a whirlwind week in Spring, though, so let's get to it!

-   Reactor team ninja Simon Baslé has just announced the first milestone of [the Californium release train of project Reactor](https://spring.io/blog/2018/08/07/reactor-californium-m1-this-summer-s-milestone-release-train)
-   This is a nice example of [using Spring Data JDBC with JOOQ](https://github.com/spring-projects/spring-data-examples/tree/master/jdbc/jooq).. *in the Spring Data examples*!

-   Ryan Baxter has just announced the first [release of Spring Cloud Finchley, SR1](https://spring.io/blog/2018/08/01/spring-cloud-finchley-sr1-is-available)
-   Our very own Paul Czarkowski has just published a [fairly lengthy tutorial on deploying Spinnaker to Pivotal Container Service (or any Kubernetes)](https://content.pivotal.io/blog/deploying-software-on-pks-creating-a-continuous-delivery-pipeline-with-spinnaker) using Helm and then creating pipelines in Spinnaker to deploy applications to PKS/Kubernetes. This is *awesome*!
