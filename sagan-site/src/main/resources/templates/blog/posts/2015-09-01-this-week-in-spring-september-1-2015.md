# This Week in Spring - September 1, 2015

_Engineering | Josh Long |  September 01, 2015 | 2 Comments_

Welcome to another installment of *This Week in Spring*! The Spring team is hard at work on all the latest and greatest ahead of the SpringOne2GX event in Washington DC! Time is *really* flying! I can't *beleive* what we're staring down September already! This week I'm in Tokyo, Japan, participating in the Spring User Group's *huge* *Spring in Summer* event where [I gave a keynote](https://speakerdeck.com/joshlong/the-macro-of-microservices) and two talks, on Spring Boot and Spring Cloud. The one day event attracted some of the company's largest websites and was a *lot* of fun!

Anyway, without further ado, let's get to it!

-   [Spring Data Gosling goes GA!](https://spring.io/blog/2015/09/01/spring-data-release-train-gosling-goes-ga)
-   Spring ninja Greg Turnquist just announced the first installment of [his tutorial on using Spring with React.js](https://spring.io/blog/2015/09/01/react-js-and-spring-data-rest-part-1-basic-features)
-   Erik, of the *Programming with Erik* blog, has put together [an interesting post on using Spring Boot with Ember.js](http://www.programwitherik.com/trying-out-spring-boot-with-ember-and-a-site-update/)
-   I dig this blog that answers the question, [*Why JHipster?*](https://www.openprogrammer.info/2015/08/25/why-jhipster-angularjs-spring-boot-liquibase-mongodb-elasticsearch/) (JHipster is a `yeoman` code-generator for Angular.js frontends and Spring Boot backends)
-   The Kotan code blog has a [nice post on securing Spring Boot webservices](http://www.kotancode.com/2015/08/26/securing-a-spring-boot-microservice/)
-   This Russian-language post looks at [how to use Java FX and Spring Boot](http://habrahabr.ru/post/265511/)
-   This blog looks at [the Spring Cloud configuration server](http://samchu.logdown.com/posts/292630-use-springcloud-to-synchronize-all-nodes-configuration)
-   Biju Kunjummen has a nice post on using [Spring Boot static web resource handling with Yeoman for single page applications](https://dzone.com/articles/spring-boot-static-web-resource-handling-w-yeoman)
-   Springfox - not associated with any project from Pivotal - [is an evolution of the Spring MVC and Swagger integration](http://springfox.github.io/springfox/docs/snapshot/)
-   Endy Muhardin has published a video [on using Spring Boot support for JPA](https://www.youtube.com/watch?v=Ptf1JdD-9jg)
-   Krishna Srinivasan has put together [a nice guide on using the Spring Boot Actuator framework](http://www.javabeat.net/spring-boot-actuator/)
-   Spring Toolsuite Lead Martin Lippert's entry into the Eclipse newsletter [on the value of STS for Spring is a pretty good, succinct, read](http://www.eclipse.org/community/eclipse_newsletter/2015/august/article2.php)
-   Our pal, Shin Tanimoto, has put together [an *amazing* Japanese-language slidedeck on using Spring Boot](http://www.slideshare.net/shintanimoto/spring-boot10)
