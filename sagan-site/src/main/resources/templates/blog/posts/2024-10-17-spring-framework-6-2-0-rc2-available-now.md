# Spring Framework 6.2.0-RC2 available now

_Releases | Brian Clozel |  October 17, 2024 | 0 Comments_

We are happy to announce the availability of the second release candidate of Spring Framework 6.2. We have been working on fixing regressions from previous milestones and shipping performance improvements.

Spring Framework 6.2.0-RC2 is available from [repo.spring.io/milestone](https://repo.spring.io/milestone) now, check out the [detailed changelog for this version](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.0-RC2).

Spring [3.4.0-RC1](https://github.com/spring-projects/spring-boot/milestone/352) will be released next week, a good opportunity for testing the upcoming Spring generation in existing applications!
