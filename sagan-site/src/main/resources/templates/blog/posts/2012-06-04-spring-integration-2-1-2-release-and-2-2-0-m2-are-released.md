# Spring Integration 2.1.2.RELEASE  and 2.2.0.M2 are released!

_Releases | Oleg Zhurakousky |  June 04, 2012 | 0 Comments_

The Spring Integration team is pleased to announce the release of:

**Spring Integration 2.1.2.RELEASE** - a small maintenance release with some important bug fixes

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=12835)
-   Artifacts are available in [SpringSource repo](http://repo.springsource.org/simple/libs-release-local/org/springframework/integration/), [community download page](http://www.springsource.org/download/community?project=Spring%2520Integration) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Corg.springframework.integration)

**Spring Integration 2.2.0.M2** - Second milestone release of 2.2 stream whose main theme is JPA support as well as other important features and improvements

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=12601)
-   Artifacts are available in [SpringSource repo](http://repo.springsource.org/simple/libs-milestone-local/org/springframework/integration/) and [community download page](http://www.springsource.org/download/community?project=Spring%2520Integration)

More information is available on project's [home page](http://www.springsource.org/spring-integration)
