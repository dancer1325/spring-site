# Spring IO Platform Brussels SR13

_Releases | Andy Wilkinson |  September 13, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR13 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.10
-   Spring Boot 1.5.16
-   Spring Data Ingalls SR15
-   Spring Framework 4.3.19
-   Spring Security 4.2.8

The versions of a number of third-party dependencies have also been updated.
