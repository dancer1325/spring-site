# Spring Cloud 2024.0.0 (aka Moorgate) Is Now Available

_Releases | Olga Maciaszek-Sharma |  December 03, 2024 | 3 Comments_

On behalf of the community, I am pleased to announce that the General Availability (RELEASE) of the [Spring Cloud 2024.0.0](https://cloud.spring.io) Release Train is available today. The release can be found in [Maven Central](https://repo1.maven.org/maven2/org/springframework/cloud/spring-cloud-dependencies/2024.0.0/). You can check out the 2024.0 [release notes for more information](https://github.com/spring-cloud/spring-cloud-release/wiki/Spring-Cloud-2024.0-Release-Notes).

---

## [](#notable-changes-in-the-20240-release-train)Notable Changes in the 2024.0 Release Train

This release is based upon Spring Boot [3.4.0](https://github.com/spring-projects/spring-boot/releases/tag/v3.4.0) and Spring Framework [6.2.0](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.0).

### [](#spring-cloud-bus)Spring Cloud Bus

-   Add a shutdown event, endpoint, and listener ([#277](https://github.com/spring-cloud/spring-cloud-bus/pull/277))

### [](#spring-cloud-circuitbreaker)Spring Cloud CircuitBreaker

-   TimeLimiter disable by group or instance level ([#202](https://github.com/spring-cloud/spring-cloud-circuitbreaker/issues/202))

### [](#spring-cloud-commons)Spring Cloud Commons

-   Adds support for load-balanced `RestTemplateBuilder` ([#1402](https://github.com/spring-cloud/spring-cloud-commons/issues/1402))

### [](#spring-cloud-config)Spring Cloud Config

-   Adds MongoDB environment repository support [(#2390)](https://github.com/spring-cloud/spring-cloud-config/pull/2390)
-   S3 Backend Behaves Differently With Multiple Application Names ([#2642](https://github.com/spring-cloud/spring-cloud-config/issues/2642))

### [](#spring-cloud-gateway)Spring Cloud Gateway

-   Implement ModifyResponseBody in MVC Server ([#3189](https://github.com/spring-cloud/spring-cloud-gateway/issues/3189)
-   Use new Spring Boot http client autoconfiguration framework ([#3571](https://github.com/spring-cloud/spring-cloud-gateway/issues/3571)
-   Adds an enabled flag to a route [(#3026)](https://github.com/spring-cloud/spring-cloud-gateway/pull/3026)
-   Cache gateway filters to avoid sorting in every single request ([#2756](https://github.com/spring-cloud/spring-cloud-gateway/issues/2756))
-   Support for SslBundles in HttpClientProperties ([#2981](https://github.com/spring-cloud/spring-cloud-gateway/issues/2981))
-   Provides configuration for Netty's http client pool leasing strategy ([#3575](https://github.com/spring-cloud/spring-cloud-gateway/pull/3575))

### [](#spring-cloud-kubernetes)Spring Cloud Kubernetes

-   Adds support for the configuration watcher to shut down the application to refresh the application ([#1799](https://github.com/spring-cloud/spring-cloud-kubernetes/pull/1799))

### [](#spring-cloud-netflix)Spring Cloud Netflix

-   Adds `RestClient` HTTP client implementation for Eureka Client ([#4257](https://github.com/spring-cloud/spring-cloud-netflix/issues/4257), [#4363](https://github.com/spring-cloud/spring-cloud-netflix/pull/4363))

### [](#spring-cloud-stream)Spring Cloud Stream

-   Ensuring Thread-Safety for BindingServiceProperties.bindings [#3011](https://github.com/spring-cloud/spring-cloud-stream/pull/3011)
-   PostProcessing functionality does not work with Supplier [#3009](https://github.com/spring-cloud/spring-cloud-stream/issues/3009)
-   Change the use of synchronized in the getBinder method of DefaultBinderFactory class for virtual-threads compliant [#3005](https://github.com/spring-cloud/spring-cloud-stream/pull/3005)

## [](#getting-started)Getting Started

To get started with Maven with a BOM (dependency management only):

```xml
Copy<dependencyManagement>
   <dependencies>
       <dependency>
           <groupId>org.springframework.cloud</groupId>
           <artifactId>spring-cloud-dependencies</artifactId>
           <version>2024.0.0</version>
           <type>pom</type>
           <scope>import</scope>
       </dependency>
   </dependencies>
</dependencyManagement>
<dependencies>
   <dependency>
       <groupId>org.springframework.cloud</groupId>
       <artifactId>spring-cloud-starter-config</artifactId>
   </dependency>
   <dependency>
       <groupId>org.springframework.cloud</groupId>
       <artifactId>spring-cloud-starter-netflix-eureka-client</artifactId>
   </dependency>
   ...
</dependencies>
```

or with Gradle:

```groovy
Copyplugins {
    id 'java'
    id 'org.springframework.boot' version '3.4.0'
    id 'io.spring.dependency-management' version '1.1.6'
}

repositories {
    mavenCentral()
    maven { url 'https://repo.spring.io/milestone' }
}

ext {
    set('springCloudVersion', "2024.0.0")
}

dependencies {
    implementation 'org.springframework.cloud:spring-cloud-starter-config'
    implementation 'org.springframework.cloud:spring-cloud-starter-netflix-eureka-client'
}

dependencyManagement {
    imports {
        mavenBom "org.springframework.cloud:spring-cloud-dependencies:${springCloudVersion}"
    }
}
```
