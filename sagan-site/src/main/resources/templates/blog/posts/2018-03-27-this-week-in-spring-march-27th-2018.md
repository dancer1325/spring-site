# This Week in Spring - March 27th, 2018

_Engineering | Josh Long |  March 27, 2018 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! This week I'm hanging out in sunny San Francisco talking to local customers and working on more installments of the next season of *Spring Tips*.

As usual, we've got a lot to cover so let's get to it!

-   Check this Pivotal showcase on [object detection with Spring Cloud Stream.](https://content.pivotal.io/pivotal-blog/real-time-object-detection-with-spring-cloud-stream)
-   Spring Security SAML DSL lead Filip Hanik has [just announced the 1.0.5.RELEASE of our SAML DSL project](https://spring.io/blog/2018/03/26/spring-security-saml-dsl-1-0-5-release). This release works against Spring Security SAML 1.0.4.RELEASE. This release adds a simple Java configuration option to your existing Spring Boot applications.
-   Ryan Baxter has just [announced Spring Cloud Finchley.M9](https://spring.io/blog/2018/03/23/spring-cloud-finchley-m9-has-been-released). This new release includes some nice features in Spring Cloud Gateway including support for rolling deployments and route refresh from service discovery heartbeats.
-   Spring Tool Suite lead Martin Lippert has [just announced Spring Tool Suite 3.9.3](https://spring.io/blog/2018/03/22/spring-tool-suite-3-9-3-released)
-   Erdem Günay shares the journey of [re-platforming Turkcell TV service to Cloud Native within a few months using Spring Boot and Spring Cloud.](https://www.infoq.com/presentations/turkcell-spring-boot-cloud)
-   Ryan Dawson looks at application reconfiguration [with Spring Cloud Config Server](https://dzone.com/articles/spring-cloud-config-server-for-the-impatient)
-   The *Java Dev Journal* has a simple [walkthrough introducing Spring Boot](https://www.javadevjournal.com/spring-boot/spring-boot-standalone-application/)
-   Shubham Aggarwal has a [nice post demonstrating Spring Data JPA with Spring Boot](https://www.javacodegeeks.com/2018/03/spring-data-jpa-example-with-spring-boot.html)
-   The Baeldung has a nice post on [property configuration with Spring Boot](http://www.baeldung.com/properties-with-spring)
-   Devoxx founder Stephan Janssen [details the use of Spring Boot and Spring Cloud](https://www.linkedin.com/pulse/new-voxxed-websites-journey-stephan-janssen) (via JHipster) to revamp a few different Voxxed web properties. Good job Stephan!
-   The *Sourced Blog* has a nice post on implementing the "REST Bucks" example [from the canonical tome *REST in Practice* using Spring.](https://blog.sourced-bvba.be//article/2018/02/25/tackling-restbucks-clean-architecture/) Note, this is not *that* ["REST Bucks" example](https://github.com/olivergierke/spring-restbucks).
-   [Spring Auto REST Docs 2.0.0 is out](https://github.com/ScaCap/spring-auto-restdocs/releases/tag/v2.0.0). It is based on Spring REST Docs 2.0.0 and comes with support for Kotlin's KDoc.
-   The Java Code Geeks have a nice post [on continuously delivering Spring Boot applications with Jenkins](https://www.javacodegeeks.com/2017/02/configure-jenkins-continuous-delivery-spring-boot-application.html).
-   Check out [the 0.0.5 release of Pivotal's function-as-a-service offering, Project Riff](https://projectriff.io/blog/announcing-riff-0-0-5/). The new release includes a Go function invoker that provides support for functions built as Go plugins, support for improved Python support, HTTP header whitelisting, and so much more.
