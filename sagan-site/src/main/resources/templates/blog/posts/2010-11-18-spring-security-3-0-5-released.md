# Spring Security 3.0.5 Released

_Releases | Luke Taylor |  November 18, 2010 | 1 Comment_

We're pleased to an announce the release of Spring Security 3.0.5.

This release fixes a few bugs which were causing problems with 3.0.4. Please see the [changelog](http://static.springsource.org/spring-security/site/changelog.html) for a full list of issues which have been addressed.

The release is available from the [Spring Community Downloads area](http://www.springsource.com/download/community?project=Spring%20Security) and should be available from the Maven Central repository soon.
