# Spring Integration 4.2 Release is Available

_Releases | Gary Russell |  September 10, 2015 | 5 Comments_

We are pleased to announce that the 4.2.0.RELEASE of Spring Integration is now available.

See the [release notes](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=15213) for a few changes that have been applied since the release candidate was announced.

See the [release candidate announcement](https://spring.io/blog/2015/08/17/spring-integration-4-2-release-candidate-is-available) and the [reference document what's new](http://docs.spring.io/spring-integration/docs/4.2.0.RELEASE/reference/html/whats-new.html) for complete details of the release content.

See the [project page](http://projects.spring.io/spring-integration/) for links to downloads, documentation etc.
