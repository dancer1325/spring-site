# Spring Roo 1.3.0 introduces JDK 8 support

_Releases | Pieter Humphrey |  November 21, 2014 | 6 Comments_

On behalf of the Spring Roo team, I am pleased to announce the release of Spring Roo 1.3.0, the first version released by [DISID Corporation](http://www.disid.com/) as project lead.

Spring Roo is a rapid application development tool for Java, allowing you to create full web enterprise applications in just minutes.

This is the first time that Spring Roo jar files have been published to Maven Central! As always, the release is available in the Spring IO repository.

The 1.3.0 release includes bug fixes and enhancements as well as several new features:

-   Now Roo has complete support for JDK 8:
    -   run Spring Roo with JDK 8
        
    -   run generated applications with JDK 8
-   Update Felix to 4.4.1

For full details on the changes made in the release, please refer to the [Release Notes in Jira](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10340&version=14315)

For more project specific information please see the [Project Page](http://projects.spring.io/spring-roo/) | [GitHub](https://github.com/spring-projects/spring-roo)

Please try out the updates and provide feedback.

As always, you’ll also find Roo on Twitter - either follow [@SpringRoo](https://www.twitter.com/springroo) or just include #SpringRoo in your tweets. The next version 2.0 will be amazing, the best is coming up. Stay tuned!
