# Spring Data Arora Service Release 2 available

_Releases | Oliver Drotbohm |  July 22, 2013 | 0 Comments_

I am pleased to announce the availability of the second service release of the Spring Data Arora release train. It includes quite a few significant bug fixes that we have already released in the first milestone for the Babbage release train. To makethem available in an official release, we backported them into this one here. The modules included are:

-   Spring Data Commons 1.5.2 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-commons/1.5.2.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/commons/docs/1.5.2.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/commons/docs/1.5.2.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/commons/docs/1.5.2.RELEASE/changelog.txt)
-   Spring Data JPA 1.3.4 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-jpa/1.3.4.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/jpa/docs/1.3.4.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/jpa/docs/1.3.4.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/jpa/docs/1.3.4.RELEASE/changelog.txt)
-   Spring Data MongoDB 1.2.3 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-mongodb/1.2.3.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/mongodb/docs/1.2.3.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/mongodb/docs/1.2.3.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/mongodb/docs/1.2.3.RELEASE/changelog.txt)
-   Spring Data Neo4j 2.2.2 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-neo4j/2.2.2.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/neo4j/docs/2.2.2.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/neo4j/docs/2.2.2.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/neo4j/docs/2.2.2.RELEASE/changelog.txt)

The artifacts are also available in Maven central. You find a comprehensive list of the 46 tickets resolved in our [JIRA](https://jira.springsource.org/issues/?filter=14367). The next release is going to be a release candidate for the Babbage release train in early August. You can find details for this release in the Spring Data Commons [wiki](https://github.com/SpringSource/spring-data-commons/wiki/Release-Train-Babbage).

If you want to learn more about the Spring Data family of projects, the best event to do is [SpringOne 2013](http://www.springone2gx.com/) being held in Santa Clara early September. Hope to see you there!
