# This Week in Spring - 23 October, 2012

_Engineering | Josh Long |  October 23, 2012 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it! A lot of this news come from SpringSource itself, from various projects timed to coincide with [SpringOne2GX](http://www.springone2gx.com) last week. It's like receiving an early birthday gift! And speaking of gifts, if you missed SpringOne2GX, don't worry! We recorded the SpringOne sessions and they will be posted soon. Also, you can check out the show wrap ups for [Day 1](http://www.springsource.org/node/3704), [Day 2](http://www.springsource.org/node/3709), and [Day 3](http://www.springsource.org/node/3711) on SpringSource.org to get a flavor for the conference.

1.  Costin Leau announced the [release of Spring for Apache Hadoop 1.0.RC1](http://www.springsource.org/node/3707) (the first-*ever* RC of Spring for Apache Hadoop!) featuring a template, exception translation and declarative runner for Pig and Hive scripts, enhanced map/reduce `.jar` and tool execution, and refined JVM scripting, M/R, and Cascading runners. Don't miss the upcoming [webinar](http://www.springsource.org/node/3669) on Nov 8th!

3.  Yusuf has done a nice job [introducing Spring's XA support in terms of the open source Bitronix](http://java.dzone.com/articles/xa-transactions-2-phase-commit) standalone JTA-XA transaction manager. For an alternative guide, might I humbly [suggest this blog post](http://blog.springsource.org/2011/08/15/configuring-spring-and-jta-without-full-java-ee/)?
4.  Oleg Zhurakousky announced the release [of Spring Integration 2.1.4 and 2.2.0.RC2](http://www.springsource.org/node/3710) . The new release features pseudo transactions, JPA support, built-in *retry* support, and a lot more, so be sure to check it out!
5.  Costin Leau announced the release [of Spring Data Redis 1.0.2](http://www.springsource.org/node/3706), which features cache expiration and ZSet methods, among other things.
6.  Thomas Risberg announced the [availability of the Spring Data JDBC extensions with Oracle database support, version 1.0.0.RC4](http://www.springsource.org/node/3708 ).
7.  Satish Ab has a nice post on building [the web layer with Spring MVC for his sample invoice application](http://satishab.blogspot.com/2012/10/part-3-web-layer-with-spring-mvc-for.html).
8.  The Mkyong blog has a nice post on how to autowire dependiecnes [into custom JSF validators](http://www.mkyong.com/jsf2/spring-autowired-into-jsf-custom-validator/). Very nice post!
9.  Our friend Krishna Prasad has a nice post on hwo to implement the control [bus pattern with Spring Integration and JMS](http://krishnasblog.com/2012/10/21/control-bus-pattern-with-spring-integration-and-jms/)
