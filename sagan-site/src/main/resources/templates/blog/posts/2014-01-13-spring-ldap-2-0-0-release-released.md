# Spring LDAP 2.0.0.RELEASE Released

_Releases | Rob Winch |  January 13, 2014 | 0 Comments_

I'm pleased to announce that Spring LDAP 2.0.0.RELEASE is now available from Maven Central and Bintray. A special thanks to [Mattias Arthursson](http://twitter.com/marthursson) for all the work he put into this release!

Refer to the [What's new in Spring LDAP 2.0](http://docs.spring.io/spring-ldap/docs/2.0.x/reference/#what-s-new-in-2-0) to find the full details of this release. A list of [changes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10071&version=14281) can be found within JIRA's change logs. Highlights include:

-   Spring Data Repository and QueryDSL support is now included in Spring LDAP.
-   Fluent LDAP query support has been added.
-   A custom XML namespace is now provided to simplify configuration of Spring LDAP.
-   Spring LDAP core has been updated with Java 5 features such as generics and varargs.
-   The ODM (Object-Directory Mapping) functionality has been moved to core and there are new methods in LdapOperations/LdapTemplate that uses this automatic translation to/from ODM-annotated classes.

Please consider helping us [spread the word on Twitter](https://twitter.com/rob_winch/status/422766895410843648), ask questions on [Stackoverflow with the spring-ldap tag](http://stackoverflow.com/questions/tagged/spring-ldap) and log any issues to the [Spring LDAP JIRA](jira.springsource.org/browse/LDAP).
