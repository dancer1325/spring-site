# Spring Session 1.3.0 RC1 Released

_Releases | Rob Winch |  November 23, 2016 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Session 1.3.0.RC1. This release release closes lots of community submitted Pull Requests. For a complete list of changes see the [changelog](https://github.com/spring-projects/spring-session/milestone/16?closed=1).

# [](#whats-new-in-spring-session-130-rc1)[](#what-s-new-in-spring-session-1-3-0-rc1)What’s New in Spring Session 1.3.0 RC1

Highlights include:

-   First class [Spring Security’s RememberMe](http://docs.spring.io/spring-session/docs/1.3.0.RC1/reference/html5/#spring-security) Support
    
-   Updated guides to [use Lettuce](https://github.com/spring-projects/spring-session/pull/652)
    
-   [OrientDB Support](http://docs.spring.io/spring-session/docs/1.3.x-SNAPSHOT/reference/html5/#community-extensions)
    
-   Configuration Improvements
    
-   Performance Optimizations
    
-   Bug Fixes
    

# [](#contributions)[](#contributions)Contributions

Without the community we couldn’t be the successful project we are today. I’d like to thank everyone that created issues & provided feedback.
