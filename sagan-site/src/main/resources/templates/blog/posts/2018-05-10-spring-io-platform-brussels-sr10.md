# Spring IO Platform Brussels SR10

_Releases | Andy Wilkinson |  May 10, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR10 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Boot 1.5.13
-   Spring Data Ingalls SR12
-   Spring Framework 4.3.17
-   Spring Integration 4.3.16
-   Spring Security 4.2.6
-   Spring Security OAuth 2.0.15

The versions of a number of third-party dependencies have also been updated.
