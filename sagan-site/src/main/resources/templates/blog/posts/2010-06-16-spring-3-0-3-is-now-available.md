# Spring 3.0.3 is Now Available

_Releases | Adam Fitzgerald |  June 16, 2010 | 1 Comment_

[Juergen Hoeller](http://blog.springsource.com/author/juergenh/) has [announced](http://blog.springsource.com/2010/06/15/spring-framework-3-0-3-released/) that Spring 3.0.3 is now available. This minor release addresses over 100 minor issues and catches up with some recent third-party releases.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.3.RELEASE) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.x/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.3.RELEASE/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

Please note that we are not providing a dependencies download anymore. The recommended way of obtaining third-party libraries for use with Spring is Maven/Ivy; you could also download third-party distributions of your choice and take the jars from there. Note that there is no reason to upgrade third-party libraries unless you want to: The simplest solution is to keep using the versions that you know and trust.

Don't forget that Spring users can ask questions in the [community forum](http://forum.springsource.org/forumdisplay.php?f=21) and identify issues in [JIRA](http://jira.springframework.org/browse/SPR) as well.
