# This Week in Spring - April 3, 2018

_Engineering | Josh Long |  April 03, 2018 | 1 Comment_

Hi Spring fans! What a week! This week I'm in beautiful Dallas, TX, for the Spring One Tour Dallas event which is, like so many of the other cities already, utterly and completely sold out! What a crowd!

And, so, without further ado, let's get to it!

-   Spring Framework Juergen Hoeller has just [announced Spring Framework 5.0.5 and 4.3.15 are now available.](https://spring.io/blog/2018/04/03/spring-framework-5-0-5-and-4-3-15-available-now)
-   Check out the PR [supporting Pact v3 in Spring Cloud Contract.next](https://github.com/spring-cloud/spring-cloud-contract/pull/569). Now would be a good time to try it out and feedback!
-   [PCF 2.1 and the Quest for a One Pizza Ops Team](https://content.pivotal.io/blog/pcf-2-1-and-the-quest-for-a-one-pizza-ops-team)
-   Check out this shiny new [Kotlin DSL for Spring Data Cassandra!](https://github.com/spring-projects/spring-data-cassandra/pull/125)
-   Spring Boot lead Phil Webb has [published a nice look at property binding in Spring Boot 2.0](https://spring.io/blog/2018/03/28/property-binding-in-spring-boot-2-0)
-   Spring Batch 1.0.0. was released ten years and a week ago, [on March 28th, 2018](https://github.com/spring-projects/spring-batch/releases/tag/spring-batch-1.0.0.FINAL). Happy tenth anniversary, Spring Batch!
-   Roy Clarkson [has just announced that the Spring Cloud Open Service Broker 2.0.0.M2](https://spring.io/blog/2018/03/27/spring-cloud-open-service-broker-2-0-0-m2-released) is now available. Spring Cloud Open Service Broker is a framework for building Spring Boot applications that implement the Open Service Broker API.
-   Ryan Baxter [has just announced Spring Cloud Edware SR3](https://spring.io/blog/2018/03/27/spring-cloud-edgware-sr3-has-been-released) which includes a number of refinements which you should definitely check out!
-   [GitHub - dsyer/riff-http](https://github.com/dsyer/riff-http)
-   [Java 10 migration: Is it a breeze or a tornado? - JAXenter](https://jaxenter.com/java-10-interview-series-part-3-142957.html)
-   Check out this post [on using JUnit 5 with Gradle](https://www.petrikainulainen.net/programming/testing/junit-5-tutorial-running-unit-tests-with-gradle/)
-   This [post](http://www.bytesville.com/springboot-micrometer-prometheus-grafana/) looks at using [Spring Boot 2](http://start.spring.io), [Micrometer](http://micrometer.io), Prometheus and Grafana together - check it out!
-   It was [recently Greg Turnquist's birthday](https://twitter.com/springcentral/status/979814661058043904). Greg is the chap who toils tirelessly on all the amazing [http://Spring.io/guides](http://spring.io/guides). So, for his birthday, won't you take a moment [and say "thanks"](http://twitter.com/gregturn)?
-   This is an interesting template application that demonstrates how to [wire up Spring, JWT, React and Redux together.](https://github.com/pram/spring-react-boilerplate)
-   Hey Atlanta, I'll be there April 10 looking at building reactive microservices with [Project Reactor, Spring Boot 2.0, and Spring Cloud Finchley](https://twitter.com/springcentral/status/979421909443010560). Check it out!
-   This is an oldie-but-a-goodie: it looks at [the details of connection pools](https://beansroasted.wordpress.com/2017/07/29/connection-pool-analysis/amp/?__twitter_impression=true)
-   Check out this link for an epic 5 part webinar series on domain-driven design [with Spring Boot, event-sourcing, microservices and more.](https://twitter.com/PieterHumphrey/status/979392883198324737)!
-   The latest [Camunda BPMN Spring Boot Starter (3.0-alpha1) now supports Spring Boot 2.0!](https://blog.camunda.com/post/2018/03/camunda-bpm-790-alpha3-released/)
-   March 28th, 2018, [was the ten year anniversary of Spring Batch 1.0](https://twitter.com/_benas_/status/978901174039580672)! What a wicked, wonderful journey it's been since then! Happy birthday, Batch!
-   I'm so looking forward to this upcoming webinar on [why Spring Boot and Java are magic together on Microsoft Azure](https://twitter.com/springcentral/status/979126491307896833)
-   Lettuce, a Redis driver that you can use with Spring Data Redis, [4.4.4.Final and 5.0.3.RELEASE are out now](https://twitter.com/SpringData/status/978943126017593344). [5.0.3.RELEASE fixes a critical Redis Cluster issue that left connections open](https://twitter.com/SpringData/status/978943126017593344).
-   TL;DR: [the Spring Initializr's use is expanding massively](https://twitter.com/rseroter/status/978684516616224773?s=12)! Over the past 30 days, 38.5% more projects created at [http://start.spring.io](http://start.spring.io) than the previous *month* (not year!) before. Well over 850,000 projects, now!
-   Check out this SpringOne Platform 2017 [talk from Christoph Strobl and Mark Paluch introducing reactive Spring Data](https://twitter.com/SpringData/status/978883190533345280)
-   JAXenter have a nice German-language post [on using Micrometer and Spring Boot 2.0 to collect metrics](https://twitter.com/springcentral/status/978762188788719616)

-   Check out the [second part of my series on building Reactive applications with Spring Boot](https://twitter.com/java/status/978698228974538752?s=12) in Java Magazine this month! (It's a free download!)
