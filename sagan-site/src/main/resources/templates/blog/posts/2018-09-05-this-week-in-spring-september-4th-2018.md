# This Week in Spring - September 4th, 2018

_Engineering | Josh Long |  September 05, 2018 | 0 Comments_

Hi Spring fans and welcome to another installment of *This Week in Spring*! If you're in the US, I hope you enjoyed your Labor Day weekend. I certainly did, gorging on the San Francisco sun and preparing for my O'Reilly online training for tomorrow, [*Bootiful Kotlin*](https://www.safaribooksonline.com/live-training/courses/spring-boot-and-kotlin/0636920202899/). I'm also busily preparing for the epic [SpringOne Tour event in Istanbul, Turkey](https://springonetour.io/2018/istanbul), customer visits in Atlanta, GA, and of course the *epic* tentpole SpringOne Platform event coming up later this month in Washington DC. If you're around, I sure hope to see you!

Well, let's get to it!

-   In this fifth installment of my series looking at Spring Cloud GCP for Google Cloud, I introduce [Google Cloud's support for runtime configuration](https://spring.io/blog/2018/09/03/bootiful-gcp-runtime-configuration-with-spring-cloud-gcp-runtime-config-5-8) and how to use it for Spring Cloud applications.
-   Spring Data for Apache Geode John Blum has just announced that [Spring Boot for Apache Geode & Pivotal GemFire 1.0.0.M2](https://spring.io/blog/2018/08/31/spring-boot-for-apache-geode-pivotal-gemfire-1-0-0-m2-released)!
-   Spring Data for Apache Geode lead John Blum [also just announced Spring Session for Apache Geode 2.0.5 and 2.1.0.M1](https://spring.io/blog/2018/08/30/spring-session-for-apache-geode-pivotal-gemfire-2-0-5-release-and-2-1-0-m1-released)
-   Spring Security ninja Josh Cummings just announced [the auto-configuration for Spring Security OAuth2 Boot Auto-config, 2.0.4 & 2.1.0.M2](https://spring.io/blog/2018/08/29/spring-security-oauth2-boot-auto-config-2-0-4-2-1-0-m2-released).
-   Vick Kelkar has a nice post on [using Redis Enterprise's Kubernetes release on Pivotal Container Service](https://dzone.com/articles/redis-enterprise-kubernetes-release-on-pivotal-con)
-   Prasanth Mohan has a nice post [on using Spring Boot with RabbitMQ](https://dzone.com/articles/spring-boot-messaging-with-rabbitmq-pubsub-in-pcf)
-   This is pretty interesting: Emil Forslund looks at using the Speedment Free version and the `spring-generator` to [generate a complete Spring application to serve a simple REST API](https://dzone.com/articles/auto-generate-a-rest-api-from-a-database-with-spri)
-   [Why is Pivotal’s DOJO so good? – Noteworthy - The Journal Blog](https://blog.usejournal.com/why-is-pivotals-dojo-so-good-c96cdb497ddf)
-   [It's time to put a spotlight on ZGC and dim Nashorn's star: Java influencers weigh in - JAXenter](https://jaxenter.com/java-influencers-series-4-148837.html)
-   I'm speaking with the amazing Trisha Gee, fellow Java Champion and legend, at Oracle's CodeOne conference later this year. We'll be talking about reactive Spring, Kotloin, MongoDB and JavaFX. I did a quick promotional [video for Oracle which you can watch here](https://www.youtube.com/watch?v=KTuH551BgAc&feature=youtu.be)
-   In this fourth installment of my series looking at Spring Cloud GCP and Google Cloud, I looked at integration [with Google Cloud Pub/Sub and Spring Cloud](https://twitter.com/springcentral/status/1035625370995707904).
