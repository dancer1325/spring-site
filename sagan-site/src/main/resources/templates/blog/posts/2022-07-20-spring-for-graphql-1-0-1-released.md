# Spring for GraphQL 1.0.1 released

_Releases | Brian Clozel |  July 20, 2022 | 0 Comments_

On behalf of everyone involved, I'm pleased to announce the first maintenance release of Spring for GraphQL, it is now available from Maven Central. This version will be shipped with Spring Boot 2.7.2.

This release includes [10 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-graphql/releases/tag/v1.0.1). Thanks to all those who have contributed with issue reports and pull requests.

### [](#how-can-you-help)How can you help?

If you have general questions, please ask on [stackoverflow.com](https://stackoverflow.com) using the [`spring-graphql` tag](https://stackoverflow.com/tags/spring-graphql).
