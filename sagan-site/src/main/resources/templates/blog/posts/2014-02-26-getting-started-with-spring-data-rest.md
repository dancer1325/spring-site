# Getting Started with Spring Data REST

_Engineering | Greg L. Turnquist |  February 26, 2014 | 0 Comments_

Hot on the heels of [Spring Data's Codd GA release](http://spring.io/blog/2014/02/24/spring-data-release-train-codd-goes-ga), there is a new Getting Started guide for Spring Data REST, [Accessing Data with REST](http://spring.io/guides/gs/accessing-data-rest/). It shows you how to create up an application with a JPA backend and a RESTful, hypermedia-driven front end.

[Spring Data REST](http://projects.spring.io/spring-data-rest) does all the heavy lifting of building a powerful RESTful front end as well as writing the JPA queries for the back end. To top it off, the guide leverages Spring Boot so you are using the latest-and-greatest from the Spring team.

Cheers!
