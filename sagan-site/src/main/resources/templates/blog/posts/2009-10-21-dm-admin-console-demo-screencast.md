# dm Admin Console Demo Screencast

_Engineering | Christopher Frost |  October 21, 2009 | 0 Comments_

Here is a screencast showing the *dm Admin Console* for the *dm Server*. It gives a brief tour of the *dm Admin Console* and then shows some of the features by deploying a simple demo application. The application consists of a configuration file, a web bundle and a content bundle. When deployed the application simply shows the contents of the configuration file in a web browser. The application is installed in parts for the purpose of this demo, in reality you would group the application together with either a '*par*' or a '*plan*', to see how this is done with a '*plan*' file another screencast is available [here](http://blog.springsource.com/2009/09/11/configuration-properties-screencast/).
