# Spring IO Platform Cairo-SR8

_Releases | Andy Wilkinson |  April 04, 2019 | 0 Comments_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Cairo-SR8 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central. With the IO Platform reaching [end of life on 9 April 2019](https://spring.io/blog/2018/04/09/spring-io-platform-end-of-life-announcement), this is expected to be the final IO Platform release.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 2.0.12
-   Spring Batch 4.0.3
-   Spring Cloud Connectors 2.0.5
-   Spring Data Kay-SR14
-   Spring Framework 5.0.13
-   Spring HATEOAS 0.25.1
-   Spring Integration 5.0.13
-   Spring Retry 1.2.4
-   Spring Security JWT 1.0.10
-   Spring Security OAuth 2.2.4
-   Spring Security 5.0.12
-   Spring Session Apple-SR9
-   Spring Web Services 3.0.7

The versions of a number of third-party dependencies have also been updated.
