# Spring Security 5.2.1 and 5.1.7 Released

_Releases | Josh Cummings |  November 04, 2019 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.2.1 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.1.RELEASE)) and 5.1.7 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.7.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
