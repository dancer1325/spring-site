# Spring Data Arora SR3 released

_Releases | Oliver Drotbohm |  October 01, 2013 | 0 Comments_

The Spring Data team has just released the final service release for the Arora release train. SR3 includes the following modules:

-   Spring Data Commons 1.5.3 - [Changelog](http://docs.spring.io/spring-data/commons/docs/1.5.3.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/commons/docs/1.5.3.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-commons%7C1.5.3.RELEASE%7Cjar)
-   Spring Data JPA 1.3.5 - [Changelog](http://docs.spring.io/spring-data/jpa/docs/1.3.5.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/jpa/docs/1.3.5.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-jpa%7C1.3.5.RELEASE%7Cjar)
-   Spring Data MongoDB 1.2.4 - [Changelog](http://docs.spring.io/spring-data/mongodb/docs/1.2.4.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/mongodb/docs/1.2.4.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-mongodb%7C1.2.4.RELEASE%7Cjar)
-   Spring Data Neo4j 2.2.3 - [Changelog](http://docs.spring.io/spring-data/neo4j/docs/2.2.3.RELEASE/changelog.txt) - [JavaDoc](http://docs.spring.io/spring-data/neo4j/docs/2.2.3.RELEASE/api/) - [Artifacts](http://search.maven.org/#artifactdetails%7Corg.springframework.data%7Cspring-data-neo4j%7C2.2.3.RELEASE%7Cjar)

In it's core the release includes all bug fixes made between Babbage RC1 and GA that were candidates for back porting. It's a recommended update for all users on Arora that cannot upgrade to Babbage for whatever reason.

The third service release is the last release for Arora. The Spring Data team is now focussing on the development of the first milestone for the upcoming release train called [Codd](https://github.com/spring-projects/spring-data-commons/wiki/Release-Train-Codd) as well as service releases for Babbage. You can find an overview of the further [release planning](https://github.com/spring-projects/spring-data-commons/wiki/Release-planning) in the Spring Data Commons wiki.
