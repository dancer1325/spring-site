# Spring Framework 4.0.6 Released

_Releases | Stéphane Nicoll |  July 08, 2014 | 4 Comments_

We are pleased to announce that Spring Framework 4.0.6 has just been released and is now available from [maven central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22) and [repo.spring.io](http://repo.spring.io/release/org/springframework/). This is a bugfix and minor enhancement release in the 4.0 series. This release includes nearly 50 bug fixes and improvements and we recommend that all users upgrade.

For a complete list of changes, please refer to [JIRA](https://jira.spring.io/browse/SPR/fixforversion/14629)
