# Spring Integration AWS 1.0.0.M1 is now available

_Releases | Artem Bilan |  April 14, 2016 | 0 Comments_

Dear Spring Community,

On behalf of Spring Integration Team, I'm glad to announce that Spring Integration [AWS](https://aws.amazon.com/) `1.0.0.M1` is available now for download from the [Spring milestone repository](http://repo.spring.io/milestone/).

Since the [0.5.0](https://github.com/spring-projects/spring-integration-extensions/releases/tag/aws.v0.5.0.RELEASE) pre-release version we have fully reworked the core bits for simpler programming model and introduced new components. Currently the Spring Integration AWS is based on the [Spring Cloud AWS](https://github.com/spring-cloud/spring-cloud-aws) project at its foundation.

Thank you to all who have contributed to this milestone!

# [](#highlights-of-this-milestone-include)Highlights of this Milestone include:

-   Amazon Simple Storage Service (S3) Adapters
    
-   Amazon Simple Queue Service (SQS) Adapters
    
-   Amazon Simple Notification Service (SNS) Adapters
    

Please refer to Project [README](https://github.com/spring-projects/spring-integration-aws/blob/master/README.md) for more information.

# [](#next-steps)Next Steps

-   Add AWS integration testing environment;
-   Add [Kinesis](https://aws.amazon.com/kinesis/) Adapters;
-   Add [DynamoDB](https://aws.amazon.com/dynamodb/) adapters and `MessageStore` implementation;
-   Add [SimpleDb](https://aws.amazon.com/simpledb/) adapters and `MessageStore` implementation;
-   Add [Spring Cloud Stream](http://cloud.spring.io/spring-cloud-stream/) binders;
-   Add [Spring Cloud Stream](http://cloud.spring.io/spring-cloud-stream/) applications for AWS adapters;

In the meantime, we welcome any feedback, questions, or help. We would love to hear from you:
