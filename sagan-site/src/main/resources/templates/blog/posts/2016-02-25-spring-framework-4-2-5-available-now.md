# Spring Framework 4.2.5 available now

_Releases | Stéphane Nicoll |  February 25, 2016 | 0 Comments_

It is my pleasure to announce that Spring Framework 4.2.5 is available from [repo.spring.io](https://repo.spring.io/) and [Maven Central](http://search.maven.org/)! This fifth maintenance release in the 4.2 line contains [40 fixes and improvements](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15439) and is a recommended upgrade for all 4.x users, in particular also from 4.1 now.

Our next stop is the first release candidate of Spring Framework 4.3 next month, with [many key features](https://jira.spring.io/browse/SPR/fixforversion/15350) (including several core container refinements) available in the snapshots already. Please give a 4.3 snapshot a try and let us know if you encounter any regressions.
