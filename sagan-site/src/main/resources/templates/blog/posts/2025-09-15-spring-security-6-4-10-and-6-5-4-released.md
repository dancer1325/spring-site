# Spring Security 6.4.10 and 6.5.4 Released

_Releases | Josh Cummings |  September 15, 2025 | 0 Comments_

**\--- IMPORTANT UPDATE ---**

> An error occurred in our release process for `6.4.10` and `6.5.4` that **did not include** Spring Framework `6.2.11`.
> 
> Given this, we have released `6.4.11` and `6.5.5`, which now includes Spring Framework `6.2.11`.

On behalf of the team and everyone who has contributed, I am pleased to announce the availability of Spring Security `6.4.10` and `6.5.4`.

Spring Security `6.4.10` ships with [4 fixes and several dependency upgrades](https://github.com/spring-projects/spring-security/releases/tag/6.4.10). This version will be shipped this week with Spring Boot `3.4.10`.

Spring Security `6.5.4` ships with [4 fixes and several dependency upgrades](https://github.com/spring-projects/spring-security/releases/tag/6.5.4). This version will be shipped this week with Spring Boot `3.5.6`.

These two releases also address [CVE-2025-41248](https://spring.io/security/cve-2025-41248), which was announced in conjunction with [CVE-2025-41249](https://spring.io/security/cve-2025-41249).

See [Spring Security and Spring Framework Release Fixes for CVE-2025-41248 and CVE-2025-41249](https://spring.io/blog/2025/09/15/spring-framework-and-spring-security-fixes-for-CVE-2025-41249-and-CVE-2025-41248) for further details.
