# Spring for Apache Pulsar 1.1.7 and 1.2.1 are now available

_Releases | Chris Bono |  December 18, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.1.7` and `1.2.1` have been released and are now available from Maven Central.

The releases will be included in the upcoming Spring Boot [3.3.7](https://github.com/spring-projects/spring-boot/milestones/3.3.7) and [3.4.1](https://github.com/spring-projects/spring-boot/milestones/3.4.1) releases, respectively.

This release includes numerous dependency upgrades.

Please see the release notes ([1.1.7](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.7) and [1.2.1](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.1)) for more details.
