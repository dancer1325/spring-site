# Modern Enterprise Java Architectures with Spring 3.1

_News | Josh Long |  January 24, 2012 | 0 Comments_

In this video [Juergen Hoeller](http://blog.springsource.org/author/juergenh/), SpringSource Principal Engineer for the [Spring Framework](http://www.springsource.org/spring-framework) gives a guided tour of the newest features in the Spring 3.1 release. Juergen introduces Spring's new capabilities, including the environment abstraction, profiles, the updated Spring MVC framework, Hibernate 4 and Quartz 2 support, and much, much more.
