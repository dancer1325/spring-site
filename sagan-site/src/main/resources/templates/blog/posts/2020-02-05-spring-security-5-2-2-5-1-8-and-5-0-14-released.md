# Spring Security 5.2.2, 5.1.8 and 5.0.14 Released

_Releases | Joe Grandja |  February 05, 2020 | 1 Comment_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.2.2 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.2.RELEASE)), 5.1.8 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.8.RELEASE)) and 5.0.14 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.14.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
