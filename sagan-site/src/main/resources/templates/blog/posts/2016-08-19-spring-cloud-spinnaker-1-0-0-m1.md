# Spring Cloud Spinnaker 1.0.0.M1

_Engineering | Greg L. Turnquist |  August 19, 2016 | 2 Comments_

Greetings Spring community,

I am happy to release the first milestone for [Spring Cloud Spinnaker](https://github.com/spring-cloud/spring-cloud-spinnaker). Spring Cloud Spinnaker bundles up the [continuous delivery Spinnaker platform](http://www.spinnaker.io/), and provides a 1-click installer to let you install it to any [certified Cloud Foundry provider](https://www.cloudfoundry.org/use/cloud-foundry-certified/).

At this year’s SpringOne Platform 2016 conference, there were two talks about Spinnaker. If you have early release access and missed them, you can watch right now. Otherwise you can catch them on the [SpringDeveloper YouTube Channel](http://spring.io/video) once they are published.

If your team/meetup/JUG is interested in hearing more about Spinnaker, [check in with me](https://spring.io/team/gturnquist) and we can arrange a Google Hangout.

Cheers,
