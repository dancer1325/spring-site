# Spring Framework 5.3.25 available now

_Releases | Brian Clozel |  January 11, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.25` is available now.

Spring Framework `5.3.25` includes [18 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.25). This version will be shipped with Spring Boot 2.7.8, [to be released next week](https://calendar.spring.io/). Please ensure that your Spring Boot 3.0 upgrade strategy is in line with [the official support dates](https://spring.io/projects/spring-boot#support).
