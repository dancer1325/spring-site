# Spring Security 4.0.1 Released

_Releases | Rob Winch |  April 23, 2015 | 5 Comments_

I'm pleased to announce the release of Spring Security 4.0.1.RELEASE. This release is the first maintenance release of the 4.0 line and focusses on fixing any major issues that were found in the new release. For complete details on the release, refer to the [Change Log](https://jira.spring.io/issues/?jql=fixVersion%20%3D%204.0.1%20AND%20project%20%3D%20SEC).
