# Spring Security 5.0.3 Released

_Releases | Rob Winch |  February 28, 2018 | 0 Comments_

I’m pleased to announce the release of Spring Security 5.0.3.

The main purpose of this release is to provide a significant [performance improvement](https://github.com/spring-projects/spring-security/issues/5040) for Spring Security WebFlux. It also contains dependency updates to prepare for Spring Boot 2.0.0.RELEASE. For a complete list of changes, refer to the [changelog](https://github.com/spring-projects/spring-security/milestone/111?closed=1).
