# Spring Session 1.1.1 Released

_Releases | Rob Winch |  March 17, 2016 | 0 Comments_

On behalf of the community, I'm pleased to announce the release of Spring Session 1.1.1.RELEASE. The release can be found in Maven Central.

## [](#change-log)Change Log

This release includes a number of minor fixes and upgrade is recommended for anyone currently using Spring Session. You can find details of the fixes in the [changelog](https://github.com/spring-projects/spring-session/issues?q=milestone%3A1.1.1)
