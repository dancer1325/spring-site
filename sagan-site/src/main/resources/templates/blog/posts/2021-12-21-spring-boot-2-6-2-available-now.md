# Spring Boot 2.6.2 available now

_Releases | Scott Frederick |  December 21, 2021 | 19 Comments_

**Update Jan 5: 2022:** The Spring Framework version in this release includes fixes for [CVE-2021-22060](https://tanzu.vmware.com/security/cve-2021-22060) whose official publication was deferred until today since many people take time off at the end of the year.

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot `2.6.2` has been released and is now available from Maven Central.

This release includes [55 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-boot/releases/tag/v2.6.2). Thanks to all those who have contributed with issue reports and pull requests.
