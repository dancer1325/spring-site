# Spring Web Flow 2.4.0 and 2.3.4 Available

_Releases | Rossen Stoyanchev |  May 30, 2014 | 0 Comments_

I'm pleased to announce a double bill release of Spring Web Flow 2.4.0 and 2.3.4.

Please refer to the ["What's new"](http://docs.spring.io/spring-webflow/docs/2.4.0.RELEASE/reference/html/whatsnew.html#whatsnew-swf-240) section of the reference for an overview of the 2.4.0 release. The 2.3.4 release is minor maintenance release mainly with fixes for issues reported by JSF users.
