# Spring Security 3.0.3 Released

_Releases | Luke Taylor |  July 02, 2010 | 0 Comments_

We're pleased to an announce the release of Spring Security 3.0.3.

This is mainly a minor bugfix release. Please see the [changelog](http://static.springsource.org/spring-security/site/changelog.html) for a list of issues which have been addressed. You must be using Spring 3.0.3 or later to use this version.
