# Dependency Management Plugin 1.0.4.RELEASE

_Releases | Andy Wilkinson |  December 12, 2017 | 2 Comments_

I am pleased to announce that [Gradle dependency management plugin](https://github.com/spring-gradle-plugins/dependency-management-plugin) 1.0.4.RELEASE is now available from Maven Central, Bintray, and the [Gradle Plugin Portal](https://plugins.gradle.org/plugin/io.spring.dependency-management). This maintenance release fixes a [handful of issues](https://github.com/spring-gradle-plugins/dependency-management-plugin/milestone/24?closed=1) and is a recommended upgrade for all users of the plugin.
