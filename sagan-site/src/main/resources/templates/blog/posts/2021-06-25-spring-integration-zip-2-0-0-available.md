# Spring Integration Zip 2.0.0 Available

_Releases | Artem Bilan |  June 25, 2021 | 0 Comments_

Dear Spring community,

On behalf of the team and everyone who contributed, it is my pleasure to announce `2.0.0` version for Spring Integration Zip extension.

This version is fully based on recently released [Spring Integration `5.5.1`](https://spring.io/blog/2021/05/19/spring-integration-5-5-goes-ga) has upgraded other dependencies and all the tests have been migrated to JUnit 5. The previously released [released version `1.0.4`](https://spring.io/blog/2021/03/01/spring-integration-zip-1-0-4-cve-2021-22114) of `spring-integration-zip` is not compatible with Spring Integration 5th generation.

Thank you everyone who contributed!

Cheers,   
Artem
