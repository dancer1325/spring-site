# First Eclipse Gemini Web Milestone Ships

_Engineering | Glyn Normington |  June 30, 2010 | 0 Comments_

[![Gemini](http://www.eclipse.org/gemini/images/logo/logo-small.png)](http://www.eclipse.org/gemini) The first milestone (1.1.0.M01) of Eclipse [Gemini Web](http://www.eclipse.org/gemini/web/) is now [available for download](http://www.eclipse.org/gemini/web/download), dual-licensed under the [Eclipse Public License](http://www.eclipse.org/legal/epl-v10.html) and the [Apache License 2.0](http://www.apache.org/licenses/LICENSE-2.0.html).

SpringSource developed the Reference Implementation of the OSGi Web Container and then [contributed](http://blog.springsource.com/2009/11/24/gemini-project-proposal-at-eclipse-org/) the code to Eclipse as the Web sub-project of [Gemini](http://www.eclipse.org/gemini). The goal of Gemini is to provide modular implementations of Java EE technology. Gemini Web provides servlet and JSP technology.

Gemini Web runs on the Equinox OSGi framework and embeds Tomcat. It is also integrated into the Eclipse [Virgo](http://www.eclipse.org/virgo/) web server.

For more information see the [documentation](http://www.eclipse.org/gemini/web/documentation) and the [wiki](http://wiki.eclipse.org/Gemini/Web).
