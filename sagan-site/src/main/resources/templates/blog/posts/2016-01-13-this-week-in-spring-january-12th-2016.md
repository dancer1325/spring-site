# This Week in Spring - January 12th, 2016

_Engineering | Josh Long |  January 13, 2016 | 0 Comments_

Welcome to another installation of *This Week in Spring!* Today, I spoke at the [San Francisco JUG](http://www.meetup.com/sfjava/events/225481175/) to a packed room and now I'm off to St. Louis for a day of customer meetings. Lots of great stuff to look at as we kick off the new year so let's get to it!

-   Check out Spring Integration contributor [Artem Bilan's post on the latest-and-greatest on Spring Integration](https://spring.io/blog/2016/01/12/spring-integration-zip-1-0-0-m1-and-others) including the debut of a new Spring Integration extension supporting zipping and unzipping of messages in flight.
-   the EuregJUG Maas-Rhine published [a post on its site about.. the site! Which is all Spring Boot](http://www.euregjug.eu/2016/01/07/new-site-is-live). Check it out.
-   Check out the replay of Andy Wilkinson's SpringOne2GX 2015 [talk on documenting REST APIs with Spring REST Docs](https://spring.io/blog/2016/01/12/springone2gx-2015-replay-documenting-restful-apis)
-   Check out the replay of Fatima Casau's [SpringOne2GX 2015 talk on using Spring Boot and Groovy](https://spring.io/blog/2016/01/12/springone2gx-2015-replay-spring-boot-and-groovy)
-   Marius Bogoevici's SpringOne2GX 2015 talk on stream processing [at scale with Spring XD and Apache Kafka is also now available](https://spring.io/blog/2016/01/12/springone2gx-2015-replay-stream-processing-at-scale-with-spring-xd-and-kafka)
-   Spring Security lead Rob Winch's SpringOne2GX 2015 [talk on securing REST services with Spring is also available online](https://spring.io/blog/2016/01/12/springone2gx-2015-replay-the-state-of-securing-restful-apis-with-spring)
-   Spring framework legend and [lead Juergen Hoeller chimed in to talk about some great Java conferences of interest in the next few months](https://spring.io/blog/2016/01/11/european-conferences-with-strong-spring-content)
-   Check out Dr. David Syer's [definitive guide to using Spring Security OAuth and Spring Boot](https://spring.io/guides/tutorials/spring-boot-oauth2/)
-   and then follow it up with Malike's post on [using Spring Security OAuth and MongoDB to secure applications](http://malike.github.io/Spring-Security-OAuth2/)
-   I'm pretty sure I posted these before, but they're worth reiteration: [how Amazon uses Microservices](http://www.infoq.com/news/2015/12/microservices-amazon) and [the Pain, Power, and Patterns of Microservices](https://dzone.com/articles/the-power-patterns-and-pains-of-microservices)
-   Red Hat's Christian Posta just published a blog that talks about a potential 25x [performance increase for certain use cases with ActiveMQ](http://blog.christianposta.com/activemq/speeding-up-activemq-persistent-messaging-performance-by-25x/) - definitely worth a read!
