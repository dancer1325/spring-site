# Spring Vault 3.1.3 and 3.2.0-M1 available

_Releases | Mark Paluch |  April 25, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that the Spring Vault `3.1.3` release is available now.

Spring Vault `3.1.3` ships with [several bugfixes](https://github.com/spring-projects/spring-vault/releases/tag/3.1.3).

Spring Vault `3.2.0-M1` is the first milestone of the 3.2.x series. Notable enhancements are:

-   Add support for IMDSv2 on EC2
-   Support for Github Token Authentication

You can find the full changelog at [https://github.com/spring-projects/spring-vault/releases/tag/3.2.0-M1](https://github.com/spring-projects/spring-vault/releases/tag/3.2.0-M1). Going forward, we are going to ship Spring Vault 3.2 GA in May. This is going to be the last release of the 3.x development line. We're working towards a first milestone of Spring Vault for in the second half of the year upgrading to Spring Framework 7 and providing JSpecify annotations.
