# Java DSL for Spring Integration 1.1.2 is available now

_Releases | Artem Bilan |  January 28, 2016 | 1 Comment_

Dear Spring Community,

The Spring Integration Java DSL 1.1.2 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cspring-integration-java-dsl).

This maintenance release includes several [fixes](https://github.com/spring-projects/spring-integration-java-dsl/commits/v1.1.2.RELEASE) and 3rd party dependency updates.

Thank you everyone for contribution and any kind of feedback!

We don't have particular plans for version `1.2`, because we are looking into the absorption of Java DSL project by the Spring Integration 5.0, where Java 8 is minimal, but feel free to ask us for more features and share any concerns which can be considered to be included in the next maintenance release.
