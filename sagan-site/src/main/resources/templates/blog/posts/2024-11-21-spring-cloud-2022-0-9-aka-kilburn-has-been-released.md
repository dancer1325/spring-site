# Spring Cloud 2022.0.9 (aka Kilburn) Has Been Released

_Releases | Ryan Baxter |  November 21, 2024 | 0 Comments_

On behalf of the Spring Cloud team, I am pleased to announce that the [Spring Cloud 2022.0.9](https://enterprise.spring.io/projects/spring-cloud) Release Train is available for [support customers](https://www.vmware.com/products/app-platform/tanzu-spring). The release can be found in [https://packages.broadcom.com](https://packages.broadcom.com).

## [](#notable-changes-in-the-202209-release-train)Notable Changes in the 2022.0.9 Release Train

-   This release is compatible with Spring Boot 3.0.18 and 3.1.14.
-   Spring Cloud Contract 4.0.9 has dependency upgrades.
-   Spring Cloud Netflix 4.0.7 has dependency upgrades.
-   Spring Cloud OpenFeign 4.0.8 has dependency upgrades.
