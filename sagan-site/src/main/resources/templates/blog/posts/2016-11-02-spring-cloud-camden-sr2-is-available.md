# Spring Cloud Camden.SR2 is available

_Releases | Ryan Baxter |  November 02, 2016 | 0 Comments_

On behalf of the team, I am pleased to announce that Service Release 2 of the [Spring Cloud Camden](http://cloud.spring.io) Release Train is available today. The release can be found in our [Spring Release repository](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Ca%3A%22spring-cloud-dependencies%22).

This release includes primarily bug fixes.

#### [](#the-following-modules-were-updated-as-part-of-camdensr2)The following modules were updated as part of Camden.SR2:

-   Spring Cloud Netflix [1.2.2.RELEASE](https://github.com/spring-cloud/spring-cloud-netflix/milestone/32?closed=1)
-   Spring Cloud Consul [1.1.1.RELEASE](https://github.com/spring-cloud/spring-cloud-consul/milestone/12?closed=1)
-   Spring Cloud CLI [1.2.2.RELEASE](https://github.com/spring-cloud/spring-cloud-cli/milestone/12?closed=1)
-   Spring Cloud Streams [Brooklyn.SR1](https://github.com/spring-cloud/spring-cloud-stream-binder-rabbit/milestone/5?closed=1)

And, as always, we welcome feedback: either on [GitHub](https://github.com/spring-cloud), on [gitter](https://gitter.im/spring-cloud/spring-cloud), on [Stack Overflow](http://stackoverflow.com/questions/tagged/spring-cloud), or on [Twitter](https://twitter.com/SpringCloudOSS).
