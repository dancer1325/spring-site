# Spring Framework 6.1 RC2 released

_Releases | Brian Clozel |  October 26, 2023 | 0 Comments_

On behalf of the Spring Framework team, it is my pleasure to announce that the second Spring Framework 6.1 release candidate is available from [repo.spring.io/milestone](https://repo.spring.io/milestone) now.

Spring Framework 6.1.0-RC2 ships [51 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.0-RC2).

This is our last stop before the official 6.1.0 release mid-November!

## [](#whats-next)What's next

Check out our [What's New page](https://github.com/spring-projects/spring-framework/wiki/What's-New-in-Spring-Framework-6.x) for details about the specific features shipped so far.

The second Spring Boot 3.2 release candidate [will be released next week](https://calendar.spring.io/).
