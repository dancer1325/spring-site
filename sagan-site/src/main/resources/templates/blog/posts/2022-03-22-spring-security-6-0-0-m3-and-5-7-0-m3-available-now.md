# Spring Security 6.0.0-M3 and 5.7.0-M3 available now

_Releases | Steve Riesenberg |  March 22, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security `6.0.0-M3` and `5.7.0-M3` are available now.

This third 6.0 milestone covers build and release enhancements as well as the [deprecation of `WebSecurityConfigurerAdapter`](https://spring.io/blog/2022/02/21/spring-security-without-the-websecurityconfigureradapter) that was already released with `5.7.0-M2`. The third 5.7 milestone covers several enhancements to SAML 2.0 and OAuth 2.0 support as well as a change to [use UTF-8 by default for HTTP Basic credentials](https://github.com/spring-projects/spring-security/issues/10903) in Spring WebFlux.

For the release changes, please refer to the [releases page](https://github.com/spring-projects/spring-security/releases).
