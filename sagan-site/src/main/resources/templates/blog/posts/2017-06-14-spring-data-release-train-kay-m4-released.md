# Spring Data Release Train Kay M4 Released

_Releases | Mark Paluch |  June 14, 2017 | 1 Comment_

On behalf of the Spring Data team, I'd like to announce the availability of the fourth Spring Data Kay milestone. This release ships 98 tickets fixed along with some notable changes like composable repositories, the fluent MongoDB collection operation API, Kotlin extension for MongoDB, and much more.

This milestone is going to be picked up by Spring Boot 2.0 M2 and we're heading towards a release candidate within the next month. We'll publish a [follow-up blog post](https://spring.io/blog/2017/06/20/a-preview-on-spring-data-kay) with an in-depth discussion of the new features added to the milestone in the upcoming days.
