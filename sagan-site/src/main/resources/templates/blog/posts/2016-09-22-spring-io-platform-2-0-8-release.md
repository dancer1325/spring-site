# Spring IO Platform 2.0.8.RELEASE

_Releases | Andy Wilkinson |  September 22, 2016 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 2.0.8.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Boot 1.3.8.RELEASE
-   Spring Framework 4.2.8.RELEASE
-   Spring Data Gosling SR5
-   Spring Security JWT 1.0.5.RELEASE
-   Spring Security OAuth 2.0.11.RELEASE

The versions of a number of third-party dependencies have also been updated.
