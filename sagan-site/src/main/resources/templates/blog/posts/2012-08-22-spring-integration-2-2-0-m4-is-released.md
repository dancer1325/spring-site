# Spring Integration 2.2.0.M4 is Released!

_Releases | Gary Russell |  August 22, 2012 | 0 Comments_

**Spring Integration 2.2.0.M4** - the fourth and final milestone release of the 2.2 stream, including a number of important features and improvements, is now available.

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=13099).
-   Artifacts are available in the [SpringSource repo](http://repo.springsource.org/simple/libs-milestone-local/org/springframework/integration/) and [the community download page](http://www.springsource.org/download/community?project=Spring%2520Integration).

More information is available on the project's [home page](http://www.springsource.org/spring-integration).
