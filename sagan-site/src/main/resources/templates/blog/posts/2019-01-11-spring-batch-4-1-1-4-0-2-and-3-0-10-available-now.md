# Spring Batch 4.1.1, 4.0.2, and 3.0.10 available now

_Releases | Mahmoud Ben Hassine |  January 11, 2019 | 2 Comments_

We are pleased to announce that Spring Batch 4.1.1, 4.0.2, and 3.0.10 are available from [repo.spring.io](https://repo.spring.io) and [Maven Central](https://search.maven.org/search?q=g:org.springframework.batch).

These releases contain security and bug fixes as well as dependencies updates. We recommend users upgrade to one of these versions. Please find the change log for each version in the following links: [4.1.1](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10090&version=17249) | [4.0.2](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10090&version=16717) | [3.0.10](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10090&version=16718)

As we announced [earlier](https://spring.io/blog/2018/03/07/spring-batch-3-0-9-release-and-4-0-1-release-are-now-available), please note that v3.0.10 is the last planned release for the 3.x line, so we highly recommend users to upgrade to the latest possible version.

P.S.: We have started work on v4.2, which will focus on **performance++** ! Stay tuned!
