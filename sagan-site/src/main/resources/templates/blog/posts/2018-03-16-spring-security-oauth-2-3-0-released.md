# Spring Security OAuth 2.3.0 Released

_Releases | Joe Grandja |  March 16, 2018 | 1 Comment_

On behalf of the community, I’m pleased to announce the release of Spring Security OAuth 2.3.0.RELEASE.

The [2.3.0 release](https://github.com/spring-projects/spring-security-oauth/milestone/30?closed=1) adds new support for Elliptic Curve signature verification in `JwkTokenStore`. Thank you [Michael Duergner](https://github.com/duergner) for this contribution! This release also includes a few minor enhancements and bug fixes.
