# Spring Data Release Train Codd Goes GA

_Releases | Oliver Drotbohm |  February 24, 2014 | 0 Comments_

I am happy to announce the final release of the Spring Data release train named [Codd](http://en.wikipedia.org/wiki/Edgar_F._Codd). The full list of participating modules looks as follows:

-   Spring Data Build 1.3 GA - [Changelog](https://github.com/spring-projects/spring-data-build/issues?milestone=10&state=closed)
-   Spring Data Commons 1.7 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-commons/1.7.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/commons/docs/1.7.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/commons/docs/1.7.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/commons/docs/1.7.0.RELEASE/changelog.txt)
-   Spring Data JPA 1.5 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-jpa/1.5.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/jpa/docs/1.5.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/jpa/docs/1.5.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/jpa/docs/1.5.0.RELEASE/changelog.txt)
-   Spring Data MongoDB 1.4 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-mongodb/1.4.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/mongodb/docs/1.4.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/mongodb/docs/1.4.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/mongodb/docs/1.4.0.RELEASE/changelog.txt)
-   Spring Data Neo4j 3.0 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-neo4j/3.0.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/neo4j/docs/3.0.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/neo4j/docs/3.0.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/neo4j/docs/3.0.0.RELEASE/changelog.txt)
-   Spring Data Solr 1.1 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-solr/1.1.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/solr/docs/1.1.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/solr/docs/1.1.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/solr/docs/1.1.0.RELEASE/changelog.txt)
-   Spring Data REST 2.0 GA - [Artifacts](http://repo.spring.io/libs-milestone/org/springframework/data/spring-data-rest-webmvc/2.0.0.RELEASE/) - [JavaDoc](http://docs.spring.io/spring-data/rest/docs/2.0.0.RELEASE/api/) - [Documentation](http://docs.spring.io/spring-data/rest/docs/2.0.0.RELEASE/reference/htmlsingle/) - [Changelog](http://docs.spring.io/spring-data/rest/docs/2.0.0.RELEASE/changelog.txt)

For this release train we've squashed over 260 tickets. We collected a [curated change log](https://github.com/spring-projects/spring-data-commons/wiki/Release-Train-Codd) for the release in the Spring Data Commons wiki. We highly recommend to skim through it to get an impression of the new features for the modules you're using. For a complete list of changes see this [JIRA query](https://jira.springsource.org/issues/?filter=14430). Also, be sure to checkout the [blog](https://spring.io/blog/2013/12/04/what-s-new-in-spring-data-mongodb-1-4-m1) [posts](https://spring.io/blog/2014/02/07/what-s-new-in-spring-data-codd-rc1) covering some of the new features in detail.

The upcoming release train will be named [Dijkstra](http://en.wikipedia.org/wiki/Edsger_W._Dijkstra). We'll have quite a few new modules joining the release train. A preliminary schedule is available at the [wiki page](https://github.com/spring-projects/spring-data-commons/wiki/Release-Train-Dijkstra) for the release. We're currently in the planning phase and will happily consider the input you cast in JIRA tickets.
