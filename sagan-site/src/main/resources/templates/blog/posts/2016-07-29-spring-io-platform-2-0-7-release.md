# Spring IO Platform 2.0.7.RELEASE

_Releases | Andy Wilkinson |  July 29, 2016 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 2.0.7.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Boot 1.3.7.RELEASE
-   Spring Integration 4.2.9.RELEASE
-   Spring Web Flow 2.4.4.RELEASE

The versions of a number of third-party dependencies have also been updated.
