# Spring Integration 3.0.0 Milestone 1 is Released!

_Releases | Gary Russell |  February 15, 2013 | 0 Comments_

We are pleased to announce that **Spring Integration 3.0.0.M1** is now available. There are no major new features in this first milestone, it is mainly refactoring, removing deprecations, etc. Browse the documentation 'what's new' and the release notes for more information.

We are happy to see more community contributors and are continuing our efforts to promote that growing trend, both in the [core project](https://github.com/SpringSource/spring-integration) and the [extensions respository](https://github.com/SpringSource/spring-integration-extensions).

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=12604).
-   Reference Documentation is [here](http://static.springsource.org/spring-integration/docs/3.0.0.M1/reference/html/).
-   Artifacts are available in the [SpringSource Milestone repo](http://repo.springsource.org/simple/libs-milestone-local/org/springframework/integration/) and [the community download page](http://www.springsource.org/download/community?project=Spring%2520Integration).

More information is available on the project's [home page](http://www.springsource.org/spring-integration).
