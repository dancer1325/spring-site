# Spring Batch 2.1.0 is Now Available

_Releases | Adam Fitzgerald |  February 04, 2010 | 0 Comments_

We're pleased to an announce the release of Spring Batch 2.1.0. Thank you to all the people who have been tracking the milestones and providing feedback.

Check out the [project website](http://static.springsource.org/spring-batch/index.html) for a list of supported features, and also the [release announcement](http://forum.springsource.org/showthread.php?t=84222) in the community forum for more detailed information.
