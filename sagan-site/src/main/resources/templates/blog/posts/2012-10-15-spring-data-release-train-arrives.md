# Spring Data release train arrives

_Releases | Oliver Drotbohm |  October 15, 2012 | 0 Comments_

Dear Spring community, I am pleased to announce the synced GA release of the following modules:

-   Spring Data Commons 1.4 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-commons-core/1.4.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/data-commons/docs/1.4.0.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/data-commons/docs/1.4.0.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/data-commons/docs/1.4.0.RELEASE/changelog.txt)
-   Spring Data JPA 1.2 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-jpa/1.2.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/data-jpa/docs/1.2.0.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/data-jpa/docs/1.2.0.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/data-jpa/docs/1.2.0.RELEASE/changelog.txt)
-   Spring Data MongoDB 1.1 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-mongodb/1.1.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/data-mongodb/docs/1.1.0.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/data-mongodb/docs/1.1.0.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/data-mongodb/docs/1.1.0.RELEASE/changelog.txt)
-   Spring Data Neo4j 2.1 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-neo4j/2.1.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/data-neo4j/docs/2.1.0.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/data-neo4j/docs/2.1.0.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-data/data-neo4j/docs/2.1.0.RELEASE/changelog.txt)
-   Spring Data Gemfire 1.2 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-gemfire/1.2.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/gemfire/docs/1.2.0.RELEASE/api/) - [Documentation](http://static.springsource.org/spring-data/gemfire/docs/1.2.0.RELEASE/reference/html/) - [Changelog](http://static.springsource.org/spring-gemfire/docs/1.2.0.RELEASE/changelog.txt)
-   Spring Data REST Exporters 1.0 - [Artifacts](http://repo.springsource.org/libs-release/org/springframework/data/spring-data-rest-webmvc/1.0.0.RELEASE/) - [JavaDocs](http://static.springsource.org/spring-data/rest/docs/1.0.0.RELEASE/api/) - [Documentation](https://github.com/SpringSource/spring-data-rest/wiki)

There's a more detailled blog post over at the [the SpringSource team blog](http://blog.springsource.org/2012/10/15/spring-data-release-train-arrived) to cover the content of the overall release, the future roadmap etc. Looking forward to your feedback!
