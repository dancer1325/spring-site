# Spring Data Redis 1.1 Released

_Releases | Jennifer Hickey |  September 11, 2013 | 0 Comments_

Dear Spring Community,

I am pleased to announce the SpringOne 2013 release of [Spring Data Redis](http://projects.spring.io/spring-data-redis/) 1.1! This release provides support for Redis 2.6 commands and features, including an abstraction for [running Redis scripts](http://docs.spring.io/spring-data/data-redis/docs/1.1.0.RELEASE/reference/html/redis.html#scripting) from your Spring application.

Other highlights include:

-   Significant enhancements to [pipelining](http://docs.spring.io/spring-data/data-redis/docs/1.1.0.RELEASE/reference/html/redis.html#pipeline) functionality
-   Data type conversion and deserialization of Redis [transaction](http://docs.spring.io/spring-data/data-redis/docs/1.1.0.RELEASE/reference/html/redis.html#tx) results
-   Modified API for adding or removing multiple List, Set, and Hash elements in one call

Run this [Jira Query](https://jira.springsource.org/issues/?filter=14374) for a complete list of changes.

We look forward to your feedback on the [forum](http://forum.spring.io/forum/spring-projects/data/nosql) or in the [issue tracker](https://jira.springsource.org/browse/DATAREDIS).
