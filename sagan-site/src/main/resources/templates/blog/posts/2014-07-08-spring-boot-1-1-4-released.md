# Spring Boot 1.1.4 released

_Releases | Andy Wilkinson |  July 08, 2014 | 1 Comment_

We are pleased to announce that Spring Boot 1.1.4 has been released and is available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This release addresses a number of bugs, adds support for the first stable release of Tomcat 8 (8.0.9), and also provides [improved logging configuration](http://docs.spring.io/spring-boot/docs/1.1.4.RELEASE/reference/htmlsingle/#boot-features-custom-log-levels) that allows you to configure logging levels in `application.properties` or `application.yaml`. For a complete list of changes please refer to the [issue tracker](https://github.com/spring-projects/spring-boot/issues?milestone=19&state=closed).

Thanks again to everyone who has contributed to the project.
