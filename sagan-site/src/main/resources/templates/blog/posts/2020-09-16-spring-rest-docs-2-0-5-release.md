# Spring REST Docs 2.0.5.RELEASE

_Releases | Andy Wilkinson |  September 16, 2020 | 0 Comments_

On behalf of the community and everyone who has contributed, it's my pleasure to announce that [Spring REST Docs](https://spring.io/projects/spring-restdocs) 2.0.5.RELEASE is available from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release). Thank you to everyone who contributed to this release by reporting bugs and opening pull requests.

This maintenance release includes a [small number of fixes and documentation improvements](https://github.com/spring-projects/spring-restdocs/releases/tag/v2.0.5.RELEASE). It is a recommend upgrade for all Spring REST Docs 2.x users.
