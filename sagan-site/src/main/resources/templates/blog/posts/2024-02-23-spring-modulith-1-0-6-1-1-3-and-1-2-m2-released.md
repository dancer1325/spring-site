# Spring Modulith 1.0.6, 1.1.3 and 1.2 M2 released

_Releases | Oliver Drotbohm |  February 23, 2024 | 0 Comments_

I'm excited to announce the availability of Spring Modulith [1.0.6](https://github.com/spring-projects/spring-modulith/releases/tag/1.0.6), [1.1.3](https://github.com/spring-projects/spring-modulith/releases/tag/1.1.3) and [1.2 M2](https://github.com/spring-projects/spring-modulith/releases/tag/1.2.0-M2). The milestone release adds indexes to the event publication registry database tables, improves the execution of the database initialization and moves off the legacy Structurizr diagram generation. On top of that we ship bug fixes in the event externalization, observability infrastructure and event publication registry APIs that all have been back-ported into the service releases.

More details about what went into which release can be found in the change logs linked above.
