# Gemini Web Release Candidate

_Engineering | Glyn Normington |  October 04, 2010 | 0 Comments_

The first release candidate of the [Gemini Web](http://www.eclipse.org/gemini/web/) project is [available](http://www.eclipse.org/gemini/web/download/) for download. See the [bug list](https://bugs.eclipse.org/bugs/buglist.cgi?query_format=advanced;classification=RT;product=Gemini.Web;target_milestone=1.1.0.RC1-incubation) for what's changed since milestone 4. Thanks to Violeta Georgieva of SAP for her continued contributions.

The project is due for an Eclipse review on 20 October so that it can graduate from incubation and ship a 1.1.0 release by the end of October. The [Virgo](http://www.eclipse.org/virgo/) project, which depends on Gemini Web, should also graduate and ship its 2.1.0 release concurrently with Gemini Web.
