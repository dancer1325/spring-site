# This Week in Spring - March 20th, 2018

_Engineering | Josh Long |  March 21, 2018 | 2 Comments_

Hi Spring fans, and welcome to the first day of the Spring season! This week I'm in San Francisco visiting some customers and just hanging out, working on new Spring Tips installments, enjoying the amazing weather. As if today wasn't exciting enough, Java 10 *also* shipped today! I know that all seems like enough already, but read on as we have a lot of good stuff this week!

-   We want your feedback on this Spring Cloud Contract proposal to support [fetching stubs over different protocols](https://github.com/spring-cloud/spring-cloud-contract/issues/580)
-   Spring Cloud Data Flow ninja [Gunnar Hillert has just announced Spring Cloud Data Flow 1.4.0](https://spring.io/blog/2018/03/19/spring-cloud-data-flow-1-4-0-released). The new release includes improved dashboards, versioned streams, a new stream deployment builder, support for Docker compose, security improvements, proxy server support for the shell, LDAP Role Mapping support and improved documentation, among other things. This is a massive release with a lot of good stuff, so don't miss it!
-   Spring Security and OAuth-ninja Joe Grandja [has just released Spring Security OAuth 2.3.0](https://spring.io/blog/2018/03/16/spring-security-oauth-2-3-0-released) which now supports Elliptic Curve signature verification in `JwkTokenStore`, among other things.
-   Micrometer project lead [Jon Schneider gives an *amazing* introduction to all things Micrometer in this blog](https://spring.io/blog/2018/03/16/micrometer-spring-boot-2-s-new-application-metrics-collector). Seriously, go read it. Micrometer is the metrics collection facade that underpins the `/actuator/events` endpoint in Spring Boot 2+. It is not tied to Spring or Spring Boot, though, and this post is well worth a read no matter what your technical stack.
-   Spring Cloud co-founder Spencer Gibb has been toying with integrating Retrofit, as a sort of alternative to using something like Feign. This is a work-in-progress example, but [feedback is always valuable and it's exciting to see what might be coming](https://github.com/spring-cloud-incubator/spring-cloud-square/pull/2)
-   Spring Cloud Open Service Broker lead Scott Frederick [has just announced Spring Cloud Open Service Broker 2.0.0.M1](https://spring.io/blog/2018/03/13/spring-cloud-open-service-broker-m1-released). This new release has a Spring Boot 2 baseline, supports both Spring MVC and Spring WebFlux, and provides improved Spring Boot auto-configuration. If you're building service brokers for Cloud Foundry or Kubernetes, you're going [to want to see this post](https://spring.io/blog/2018/03/13/spring-cloud-open-service-broker-m1-released)
-   This is a [must-read article](https://www.infoq.com/articles/Servlet-and-Reactive-Stacks-Spring-Framework-5) by Spring Web team ninja Rossen Stoyanchev on the choice between Servlet APIs and reactive APIs for Spring users in Spring MVC and Spring WebFlux: "Spring MVC or Spring WebFlux, which should you use? A perfectly valid question, but one that sets up an unsound dichotomy"
-   [Java 10 is here](http://mail.openjdk.java.net/pipermail/announce/2018-March/000247.html)! Congrats to the Java team! Get it while its hot! This is the first release to support local type inference (`var`).
-   The *SivaLabs* blog continues their look at microservices in Part 6, [*Distributed Tracing with Spring Cloud Sleuth and Zipkin*](https://sivalabs.in/2018/03/microservices-part-6-distributed-tracing-with-spring-cloud-sleuth-and-zipkin/)
-   The Trampoline project has been [updated to support Spring Boot 2.0](https://twitter.com/trampolinesb/status/975822544749170689?s=12)
-   [Asim Aslam](https://twitter.com/chuhnk/status/975113977675026433) likes Project Riff, the function-as-a-service offering from Pivotal. [You might, too](https://projectriff.io/).
-   IBM executive Eric Andersen, tweeting from the IBM Think 2018 event, looks forward to some of the exciting opportunities [for Cloud Foundry and the Istio proxy](https://twitter.com/cloudfoundry/status/975873322209591297).
-   Dhiraj Ray has a nice post over on the Java Code Geeks blog on how to [externalize configuration in a Spring application](https://www.javacodegeeks.com/2018/03/spring-cloud-configuration-externalize-application-configuration.html)
-   Spring Cloud Contract lead Marcin Grzejszczak [tweets](https://twitter.com/MGrzejszczak/status/973919108298432515) that, thanks to Olga Maciaszek, you can now understand Spring Cloud Contract [in 3 seconds](https://cloud.spring.io/spring-cloud-contract/multi/multi__spring_cloud_contract_verifier_introduction.html#spring-cloud-contract-verifier-intro-three-second-tour) or [3 minutes](https://cloud.spring.io/spring-cloud-contract/multi/multi__spring_cloud_contract_verifier_introduction.html#spring-cloud-contract-verifier-intro-three-minute-tour). There's [also a ten minutes version](https://cloud.spring.io/spring-cloud-contract/multi/multi__spring_cloud_contract_verifier_introduction.html#_step_by_step_guide_to_consumer_driven_contracts_cdc) and a [roughly five day version](http://cloud-samples.spring.io/spring-cloud-contract-samples/workshops.html)
-   Dan Newton has a nice post looking at using [the new reactive web runtime Spring WebFlux](https://lankydanblog.com/2018/03/15/doing-stuff-with-spring-webflux/)
-   The SivaLabs series on [building microservices with Spring Cloud looks, in this fifth post, on at using Spring Cloud Zuul](https://sivalabs.in/2018/03/microservices-part-5-spring-cloud-zuul-proxy-as-api-gateway/)
-   Robert Winkler has just [released Resilience4j 0.12.0](https://twitter.com/rbrtwnklr/status/973552844577083393?s=12) which upgrades Vavr from Spring Boot from 1.4.3.RELEASE to 1.5.5.RELEASE, among many other dependencies. It also adds a reset method to Circuit Breaker, adds a disable and force\_open states to Circuit Breaker, adds [Project Reactor](https://projectreactor.io/) support for circuit breaker, bulkhead and rate limiter, [adds support for Micrometer](http://micrometer.io), and ensures that the Reactor operators can be used together on a `Flux`.
-   Check out our latest post about [Spring Boot 2.0 and Micrometer on Touk Team Blog](https://touk.pl/blog/2018/03/05/spring-boot-2-0-http-request-metrics-with-micrometer/)
-   Matt McCandless continues his look [at using JHipster](https://www.javacodegeeks.com/2016/10/getting-started-jhipster-part-2.html), which builds upon Spring Boot and Angular. Check it out!
