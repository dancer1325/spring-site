# This Week in Spring -  February  3rd, 2015

_Engineering | Josh Long |  February 03, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! We've got a lot to cover, so without further ado, let's get to it!

1.  [Spring XD co-lead Dr. Mark Pollack](http://spring.io/team/mpollack)'s just announced [that Spring XD 1.1 RC1 is now available](https://spring.io/blog/2015/01/29/spring-xd-1-1-rc1-released)! Now's the time to get the bits, try it out and see if there any gaps!
2.  If you've been reading the amazing Dr. Syer's blogs of late, you'll know that he's been introducing people to how to use expose and secure REST services for a UI client. The [fourth post](https://spring.io/blog/2015/01/28/the-api-gateway-pattern-angular-js-and-spring-security-part-iv) looks at how to insert an API gateway between the clients and the backend service. The fifth post then [introduces OAuth as a drop-in replacement](https://spring.io/blog/2015/02/03/sso-with-oauth2-angular-js-and-spring-security-part-v) for the bespoke authentication session tokens being used. If you're not following this series, do go back and reread them. This series treats a subject that I get asked about all the time.
3.  Sabby Anandan is joining us for the Spring XD 1.1 launch webinar on [Tuesday, Feb 17th - Spring XD: A Platform for data at scale and developer productivity](https://spring.io/blog/2015/01/20/webinar-spring-xd-a-platform-for-data-at-scale-and-developer-productivity)
4.  I hope you'll join me for a fresh look at the HTTP session with the Spring Session project on [Tuesday, February 24th, 2015](https://spring.io/blog/2015/01/20/webinar-introducing-spring-session)
5.  The *Simplicity Itself* blog has a nice treatment of the topic of service discovery and registration and includes a nice look at various service registries, [Consul, Eureka, and Zookeeper](http://www.simplicityitself.com/learning/getting-started-microservices/service-discovery-overview/). Spring Cloud includes support for Consul, Eureka, and ZooKeeper.
6.  This week, we're lucky to have the founder of the [Spring Social Slideshare](https://github.com/ttddyy/spring-social-slideshare/) project, [Tadaya Tsuyukubo (@ttddyy)](https://twitter.com/ttddyy), introduce the [project to us in this guest post](https://spring.io/blog/2015/02/03/introducing-spring-social-slideshare).
7.  The good [Dr. Syer (@david\_syer)](https://twitter.com/david_syer) has [just announced Spring Cloud 1.0.0 RC2](https://spring.io/blog/2015/02/02/spring-cloud-1-0-0-rc2-available-now) is now available! The new release improves the security integrations, and cleans up some of the starters. This ship is saling to 1.0 quickly! Check out the bits while you can!
8.  Speaking of [Dr. Syer (@david\_syer)](https://twitter.com/david_syer), he *also* just [announced Spring Security 2.0.6](https://spring.io/blog/2015/02/02/spring-security-oauth-2-0-6-release-available-now).
9.  Check out [Dr. Paul King's SpringOne2GX 2014 talk on manipulating databases with Groovy](https://spring.io/blog/2015/02/02/springone2gx-2014-replay-working-with-databases-and-groovy)
10.  Disid [have announced Spring Roo 1.3.1](https://spring.io/blog/2015/02/02/spring-roo-1-3-1-release-now-available), including numerous bug fixes and some tweaks.
11.  Check out Grails project lead Graeme Rocher's SpringOne2GX 2014 talk, [*Making Spring Boot Even Groovier*](https://spring.io/blog/2015/02/02/springone2gx-replay-making-spring-boot-even-groovier)
12.  Spring Data ninja Thomas Risberg's SpringOne2GX 2014 post. [*Introduction to Spring for Apache Hadoop*](https://spring.io/blog/2015/02/02/springone2gx-2014-replay-introduction-to-spring-for-apache-hadoop), is also now available online. This is a great place to get started with Spring and big-data.
13.  Groovy project lead Guillaume LaForge's talk, [*Groovy in the Light of Java 8*](https://spring.io/blog/2015/02/02/springone2gx-2014-replay-groovy-in-the-light-of-java-8), is now available online. Check this out to understand where the two languages are similar and where Groovy leaves Java 8 in the dust
14.  Spring Data lead Oliver Gierke just announced [that Spring Data Evans SR2 and Spring Data Dijkstra SR5](https://spring.io/blog/2015/01/28/spring-data-evans-sr-2-and-dijkstra-sr-5-released) are now available! This release includes a *lot* of improvements, so an upgrade is recommended!
