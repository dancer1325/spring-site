# Spring Tips: Learn Spring for GraphQL (parts 1 and 2 of an ongoing series)

_Engineering | Josh Long |  June 02, 2022 | 0 Comments_

Hi, Spring fans! In thi^^^ these installments, we begin a new series introducing the Spring for GraphQL project.

In this first installment, GraphQL Java lead [Andi Marek (@andimarek)](https://twitter.com/andimarek) and [I (@starbuxman)](https://twitter.com/starbuxman) look at the basics of using the GraphQL Java engine that underpins Spring for GraphQL.

In this second installment, Andi and I are joined by Spring for GraphQL lead [Rossen Stoyanchev (@rstoya05)](https://twitter.com/rstoya05) as we look at using the Spring for GraphQL component model by writing queries.

I know it's been more than a month since we last published a video, so I'm going to be releasing every episode of this series very quickly, over the next week or two. I've already recorded and mostly finished producing eight episodes. I'm working on producing a ninth, and that'll be that! A complete introduction to all things Spring for GraphQL. These are of course part of the usual [`Spring Tips`](https://www.youtube.com/watch?v=zltMcbnAxyA&list=PLgGXSWYM2FpPw8rV0tZoMiJYSCiLhPnOc) playlist on YouTube, but also part of the [`Learn Spring for GraphQL with Josh Long`](https://www.youtube.com/watch?v=gvIqFDNGgwU&list=PLgGXSWYM2FpNRPDQnAGfAHxMl3zUG2Run) playlist. Follow that one to be notified of any new episodes!
