# Security Reports for Spring Framework, Spring Data REST, Spring AMQP, and Spring Cloud OpenFeign

_Engineering | Rossen Stoyanchev |  October 26, 2021 | 0 Comments_

The recently released Spring Boot [2.5.6](https://spring.io/blog/2021/10/21/spring-boot-2-5-6-is-now-available) and [2.4.12](https://spring.io/blog/2021/10/21/spring-boot-2-4-12-is-now-available) releases contain fixes for the following security vulnerabilities:

-   [CVE-2021-22096](https://tanzu.vmware.com/security/cve-2021-22096) for the Spring Framework
-   [CVE-2021-22047](https://tanzu.vmware.com/security/cve-2021-22047) for Spring Data REST
-   [CVE-2021-22097](https://tanzu.vmware.com/security/cve-2021-22097) for Spring AMQP

In addition, Spring Cloud OpenFeign has [released](https://spring.io/blog/2021/10/26/spring-cloud-openfeign-versions-2-2-10-release-3-0-5-and-3-1-0-m4-are-now-available) versions 3.0.5 and 2.2.10, based on the same Spring Boot versions, and containing a fix for the following security vulnerability:

-   [CVE-2021-22044](https://tanzu.vmware.com/security/cve-2021-22044) for Spring Cloud OpenFeign

Please, review the reports and upgrade!
