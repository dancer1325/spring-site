# Spring Security 3.1.0 Released

_Releases | Luke Taylor |  December 07, 2011 | 0 Comments_

We're pleased to an announce the GA release of Spring Security 3.1

The release is available from the [Spring Community Downloads area](http://www.springsource.com/download/community?project=Spring%20Security) and jars are available from Maven central repository. See the [project website](http://www.springsource.org/spring-security) for more information.

There's a high level summary of changes in the [What's New?](http://static.springsource.org/spring-security/site/docs/3.1.x/reference/springsecurity-single.html#new-3.1) section of the reference manual and a full list of changes can be found in the [issue tracker](https://jira.springsource.org/secure/IssueNavigator!executeAdvanced.jspa?jqlQuery=project+%3D+SEC+AND+fixVersion+in+%2812315%2C+11892%2C+11634%2C+11633%2C+11632%2C+11174%29+order+by+priority%2C+type&runQuery=true&clear=true).
