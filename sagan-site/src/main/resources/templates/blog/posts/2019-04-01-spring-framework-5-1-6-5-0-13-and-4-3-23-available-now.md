# Spring Framework 5.1.6, 5.0.13 and 4.3.23 available now

_Releases | Stéphane Nicoll |  April 01, 2019 | 1 Comment_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.1.6.`, `5.0.13` and `4.3.23` are available from [repo.spring.io](https://repo.spring.io/release) as well as Maven Central now.

Spring Framework `5.1.6` includes [49 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.1.6.RELEASE). Spring Framework `5.0.13` is the last regular release of the `5.0.x` line and includes [25 fixes and selected improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.0.13.RELEASE). At this point, we expect all `5.x` users to upgrade to `5.1.6`. We've also released `4.3.23` that includes [20 fixes and selected improvements](https://github.com/spring-projects/spring-framework/releases/tag/v4.3.23.RELEASE).

As usual, those maintenance releases will be used for the upcoming Spring Boot `1.5.20`, `2.0.9`, `2.1.4` releases coming up later this week.
