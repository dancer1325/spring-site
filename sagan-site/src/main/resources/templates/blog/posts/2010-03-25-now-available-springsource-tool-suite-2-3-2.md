# Now Available: SpringSource Tool Suite 2.3.2

_Releases | Adam Fitzgerald |  March 25, 2010 | 0 Comments_

The latest version of [SpringSource Tool Suite](http://www.springsource.com/products/sts) (STS) is now available. STS is the best Eclipse-powered development environment for building Spring, Groovy and Grails powered enterprise applications. The new version (2.3.2) is now available for [download](http://www.springsource.com/products/springsource-tool-suite-download) and provides mostly fixes for some minor bugs and some performance improvements.

[Download](http://www.springsource.com/products/springsource-tool-suite-download) | [Install Instructions](http://download.springsource.com/release/STS/doc/STS-installation_instructions-2.3.2.RELEASE.pdf) | [ChangeLog](https://issuetracker.springsource.com/secure/ReleaseNote.jspa?version=10591&styleName=Html&projectId=10010&Create=Create) | [New & Noteworthy](http://download.springsource.com/release/STS/doc/STS-new_and_noteworthy-2.3.2.RELEASE.pdf) | [JIRA](http://issuetracker.springsource.com/browse/STS)

Now is a great time to start working with STS and please use the [community forum](http://forum.springsource.org/forumdisplay.php?f=32) to give your feedback and ask questions.
