# Spring Data Redis 1.0.0.RC1 Released

_Releases | Costin Leau |  October 19, 2011 | 0 Comments_

Dear Spring Community,

I am pleased to announce the first release candidate of the Spring Data Redis 1.0 project is now available!

[Downloads](http://www.springsource.com/download/community?project=Spring%20Data%20Redis&version=1.0.0.RC1) | [JavaDocs](http://static.springsource.org/spring-data/data-redis/docs/1.0.0.RC1/api/) | [Reference Documentation](http://static.springsource.org/spring-data/data-redis/docs/1.0.0.RC1/reference/) | [Changelog](http://static.springframework.org/spring-data/data-redis/docs/1.0.0.RC1/changelog.txt)

This release features several fixes and improvements such as:

-   Upgrade to Spring 3.1 RC1
-   Improved JDK 5 compatibility
-   Improved Spring 3.1 cache abstraction on top of Redis

We look forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?80-NoSQL) or in the [issue tracker](http://jira.springsource.org/browse/DATAREDIS).
