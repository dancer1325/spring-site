# Video: SpringOne2GX - Cloud Foundry Inside the Machine

_News | Adam Fitzgerald |  February 21, 2012 | 0 Comments_

[![SpringOne 2GX Video: Cloud Foundry - Inside the Machine](http://www.springsource.org/files/SpringOne2GX2012CloudFoundryInsideTheMachine.png)](http://www.infoq.com/presentations/Cloud-Foundry-Inside-the-Machine)

Of course the sessions at SpringOne 2GX 2011 covered Spring, [Groovy](http://groovy.codehaus.org) and [Grails](http://www.grails.org) in amazing detail, but cloud workloads were hot topics of discussion last year as well. In this presentation, pioneering technologist Derek Collison gives a detailed breakdown of [Cloud Foundry - Inside the Machine](http://www.infoq.com/presentations/Cloud-Foundry-Inside-the-Machine). Derek gives a deep analysis of Cloud Foundrys architecture with specific details on the Cloud Controller, the Service Provisioning Agent, the Droplet Execution Agent, and the Messaging System. If you want to learn about the technology behind the world's first open platform-as-a-service then this presentation is for you.

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
