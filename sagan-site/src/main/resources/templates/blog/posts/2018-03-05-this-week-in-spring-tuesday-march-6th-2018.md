# This Week in Spring - Tuesday March 6th, 2018

_Engineering | Josh Long |  March 05, 2018 | 1 Comment_

Hi Spring fans and welcome to another installment of *This Week in Spring*! As I write this it's early morning Tuesday in Sydney, Australia, where I've been visiting with some of Pivotal's amazing customers, and I'm now preparing for my flight to Dubai, in six short hours, where I'll visit some more of Pivotal's amazing customers. Later this week I'll be in Bangalore, India, for the [amazing Agile India conference](https://2018.agileindia.org/schedule/), and then - early next week on Tuesday - I'll be in Boston, MA for the [first SpringOne Tour event](https://springonetour.io/2018/boston). If [you're around don't hesitate to say hi](http://twitter.com/starbuxman), as usual!

This week we've got a lot of great content celebrating the wonderful release of Spring Boot 2.0, so without further ado let's get to it!

-   Huzzah! Spring Boot co-founder Phil Webb has [just announced Spring Boot 2.0 GA](https://spring.io/blog/2018/03/01/spring-boot-2-0-goes-ga)! The new release is *packed* with too many new features to count so what're you doing? Get over there, read the post, say hi to [the `@SpringBoot`](http://twitter.com/SpringBoot) team on Twitter, and [get started](http://start.spring.io)
-   Join [the Spring Boot 2.0 launch webinar on March 13th](https://content.pivotal.io/webinars/mar-13-introducing-spring-boot-2-0-webinar) to learn about it from the source!
-   Spring Boot 2.0 is finally, at long last, here! I wrote about the journey to Spring Boot 2.0 in this Twitter thread, reviewing the history of Spring Boot from 1.0, on the 1st April 2014, to this momentous release. Click and join me [for a #bootiful trip down memory lane](https://twitter.com/starbuxman/status/968960930150100992)
-   Spring Security lead Rob Winch previews the [Spring Security SAML Roadmap](https://spring.io/blog/2018/03/05/spring-security-saml-roadmap)
-   Sandra Gómez has put together a [very nice Spanish-language post on documenting APIs with Spring REST Docs](https://blog-es.mimacom.com/documentacion-apis-spring-rest-docs/)
-   Guillermo Tantachuco has put together a really nice overview of Project Riff, the [new function-as-a-service offering from Pivotal that runs on top of Kubernetes](https://content.pivotal.io/blog/building-functions-with-riff?_lrsc=13c8ab0f-8cde-49d7-a5a3-7dc3d8a562d1).
-   Spring Cloud co-founder Spencer Gibb has just [announced Spring Cloud Finchley M8](https://spring.io/blog/2018/03/02/spring-cloud-finchley-m8-is-available), the first release after the recent Spring Boot 2.0 release.
-   Spring Security ninja Joe Grandja has just [announced the Spring Security OAuth Boot 2.0 auto-configuration has been released](https://spring.io/blog/2018/03/01/spring-security-oauth-boot-2-auto-config-2-0-0-released). It's used to help users transition between the old Spring Security OAuth 2.x support and the next generation OAuth 2.0 Support in Spring Security 5. It provides users of Spring Security OAuth 2.x the same auto-configuration capabilities in a Spring Boot 2.0 based application that is currently available in Spring Boot 1.5.x. For more details please refer to the documentation.
-   Spring Security lead Rob Winch chimes in on the state of [Spring Security SAML and this week's SAML Vulnerability](https://spring.io/blog/2018/03/01/spring-security-saml-and-this-week-s-saml-vulnerability)
-   Spring Session ninja Vedran Pavić has just announced Spring Session Apple SR1. The new release includes [a BOM, and a number of other features](https://spring.io/blog/2018/03/01/spring-session-apple-sr1-released)
-   Project Reactor ninja Simon Baslé [announced Reactor BISMUTH-SR7](https://spring.io/blog/2018/02/28/reactor-bismuth-sr7-is-out). The release was the final release needed for Spring Boot 2.0
-   Spring Security lead Rob Winch has just [announced Spring Security 5.0.3](https://spring.io/blog/2018/02/28/spring-security-5-0-3-released). This release introduces a major performance improvement.
-   [Spring Data Kay SR5 released](https://spring.io/blog/2018/02/28/spring-data-kay-sr5-released)
-   The [new Gradle 4.6 is out](https://docs.gradle.org/4.6/release-notes.html) and it is packed with lots of cool features including initial JUnit 5 support, initial bill-of-material (BOM) dependencies and optional dependencies support, `--fail-fast` for tests, and custom command line flags for tasks.
-   I recently spoke at the first IBM Index conference where I performed an [updated *Cloud Native Java*](https://www.youtube.com/watch?v=CjWADhZBiXE&list=PLpryjkO3KF2yh5AFr3PDgOSIcWoZFGH1E&index=39). This looks at building reactive microservices with Spring Cloud.

-   Marcos Barbero's got a really detailed post on how to build an API gateway (using Spring Cloud Netflix Zuul) that [will resolve and route requests (with Netflix Eureka) to other services in the same zone](http://blog.marcosbarbero.com/ha-and-zone-affinity-spring-cloud-eureka/).
