# This Week in Spring - July 22nd, 2014

_Engineering | Josh Long |  July 23, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week we're at OSCON in beautiful Portland, OR! It's been a *crazy* busy week, and only figures to be busier! It's been nice visiting with colleagues and community members alike. The booth on the expo floor has been *flooded* with users. I love this show. :)

1.  First, the big news: [the Spring framework 4.1 release candidate is now available](http://spring.io/blog/2014/07/21/spring-framework-4-1-release-candidate-available)! The new release features updated JSR-107 support, JMS 2.0 and annotation-centric JMS configuration, a SpEL compiler, and *so* much more.

-   [Spring XD co-lead Dr. Mark Pollack has just announced that Spring XD RC1 is now available](http://spring.io/blog/2014/07/18/spring-xd-1-0-0-rc1-released). [Installation is dead-simple](http://github.com/spring-projects/spring-xd/wiki/Getting-Started#osx-homebrew-installation). There are numerous new features, and this train's moving fast so check it out, download the bits, kick the tires, etc.
-   [Spring Data ninja Christoph Strobl](http://twitter.com/stroblchristoph) put together a nice blog post [introducing Spring Data MongoDB's full-text search](http://spring.io/blog/2014/07/17/text-search-your-documents-with-spring-data-mongodb)
-   The [A Piece of My Code blog](http://apieceofmycode.blogspot.com/2014/07/spring-integration-testing-under-spring.html) looks at how to use Spring's integration testing with mock services.
-   Tousif Khan has put together [a nice post on Spring's *view controllers*](http://java.dzone.com/articles/spring-mvc-3-view-controller)
-   The *Tech Zoo* blog wrote up how to [use Spring's Apache Tiles support](http://www.techzoo.org/spring-framework/spring-mvc-tiles-3-integration-tutorial.html)
-   The *softwarecave* blog introduces how to handle [charts with jqPlot, Spring's REST support, AJAX and JQuery](http://softwarecave.org/2014/07/19/charts-with-jqplot-spring-rest-ajax-and-jquery/)
-   Theodora Fragkouli has written up a [nice post on using Spring MVC's `Errors` tag](http://examples.javacodegeeks.com/enterprise-java/spring/mvc/spring-mvc-form-errors-tag-example/)
-   I've just stumbled upon this JavaCodeGeeks [academy trail on using Spring Data Redis](http://academy.javacodegeeks.com/news/spring-data-and-redis/) - looks interesting.
-   Want to use Spring, Hibernate 4, and Ehache together? [Check out this tutorial](http://www.hevi.info/2014/07/maven-3-hibernate-4-spring-3-ehcache-spring-cache/).
