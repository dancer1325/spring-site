# Spring AMQP 2.0 Release Candidate 2 Available

_Releases | Gary Russell |  September 27, 2017 | 0 Comments_

I am pleased to announce that the [2.0.0.RC2](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=16352) release candidate of Spring AMQP is now available in the [Spring milestone repository](https://repo.spring.io/milestone/).

Since the [first release candidate](https://spring.io/blog/2017/09/12/spring-amqp-2-0-release-candidate-1-7-4-and-1-6-11-are-available) the primary changes are:

-   update to the new 5.0 version of the `amqp-client` library
    
-   increase the default prefetch count in listener containers
    

A complete list of changes since RC1 can be found in the [release notes](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=16352).

Thanks to all the community members for their feedback and contributions!

The GA release will follow shortly after the Spring Framework 5.0 GA release.

For a complete list of changes in 2.0, see [What’s New in the reference manual](http://docs.spring.io/spring-amqp/docs/2.0.0.RC2/reference/html/_introduction.html#_changes_in_2_0_since_1_7).
