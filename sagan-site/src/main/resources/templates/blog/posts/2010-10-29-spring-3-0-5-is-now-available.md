# Spring 3.0.5 is Now Available

_Releases | Adam Fitzgerald |  October 29, 2010 | 1 Comment_

![](/files/Logo_Spring_258x151.png)

Lost somewhere in all the news and excitement about last week's [SpringOne 2GX](http://www.springone2gx.com) conference was the news that Spring 3.0.5 is now available. This release addresses over 80 minor issues and includes some small improvements to the Spring Expression Language ([SpEL](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/expressions.html)), annotation support, and embedded databases. Be sure to read the Change Log for all the details.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.5.RELEASE) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.x/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.5.RELEASE/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

Don't forget that Spring users can ask questions in the [community forum](http://forum.springsource.org/forumdisplay.php?f=21) and identify issues in [JIRA](http://jira.springframework.org/browse/SPR) as well.
