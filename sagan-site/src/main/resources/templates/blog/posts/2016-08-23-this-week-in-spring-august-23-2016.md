# This Week in Spring - August 23, 2016

_Engineering | Josh Long |  August 23, 2016 | 0 Comments_

Welcome to another installation of *This Week in Spring*! This week I'm in [NYC (for the NYC Java SIG)](http://www.meetup.com/JavaSIG/events/233368316/), Austin and [San Francisco (for the Silicon Valley Spring User Group)](http://www.meetup.com/SV-SUG/events/231931166/) talking to customers and doing meetups! We've got a *lot* to cover, as usual, so let's get to it!

-   Spinnaker is a continuous delivery platform from Netflix built on Spring Boot. It supports a build-and-bake pipeline and works naturally with Spring Boot and various cloud platforms. It's enjoyed contributions from Microsoft, Google and Pivotal. In this post, [Spring ninja Greg Turnquist introduces Spring Cloud Spinnaker 1.0.0.M1](https://spring.io/blog/2016/08/19/spring-cloud-spinnaker-1-0-0-m1). Definitely worth a look!
-   this seemed interesting - a code generator [that supports Spring-based applications called Sifu](https://blog.codesifu.com/getting-started-with-Sifu-for-the-first-time.html)
-   JAX London just announced their top 20 social influencers for Java. [Spring Data lead Oliver Gierke and I made the list](https://jaxlondon.com/top-20-social-influencers-in-java/). This is awesome and I'm sure I speak for Oliver in saying we're honored to be included :-) There are a lot of other cool folks down there on that list, too, so I'm pretty stoked to be counted among them!
-   Marco Völz has been putting together a nice series on what's [new in BOSH, the system's orchestration engine](https://medium.com/@voelzmo)
-   Good news everybody! [Many of the SpringOne Platform videos and content are available now!](https://twitter.com/pivotal/status/767850955408805889)
-   Cool! [An ElasticSearch Beat implementation for Spring Boot applications](https://github.com/consulthys/springbeat)
-   I liked Luanne Misquitta's post on [building Spring Data Neo4J 4.1 applications](https://dzone.com/articles/build-spring-data-neo4j-41-applications-like-a-sup)
-   there's an interesting post on [Stormpath](http://twitter.com/gostormpath): [Using Spring Boot to Build and Deploy `.war` files](https://stormpath.com/blog/tutorial-spring-boot-war-files)
-   this is an interesting post on using Spring Boot and Let's Encrypt to seamlessly [setup encryption for a Spring Boot application](https://dzone.com/articles/spring-boot-secured-by-lets-encrypt)
-   Rafael Salerno's got a nice post up on [using Spring Cloud Netflix for service registration and discovery](https://dzone.com/articles/spring-cloud-netflix-how-works-service-registratio)
