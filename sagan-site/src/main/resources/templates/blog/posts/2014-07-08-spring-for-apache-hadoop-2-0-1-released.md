# Spring for Apache Hadoop 2.0.1 Released

_Releases | Thomas Risberg |  July 08, 2014 | 0 Comments_

We are pleased to announce that Spring for Apache Hadoop 2.0.1 has just been released and is now available from [maven central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.data%22%20%22hadoop%22) and [repo.spring.io](http://repo.spring.io/release/org/springframework/data/spring-data-hadoop/). This release updates some dependency versions. The Apache Hadoop 2.4 support (hadoop24) is updated to use version 2.4.1. The Spring Framework version is updated to 4.0.6.RELEASE, Spring Batch updated to 3.0.1.RELEASE and Spring Boot updated to 1.1.4.RELEASE.
