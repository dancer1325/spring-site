# Spring Security 5.5.0-M3 Released

_Releases | Eleftheria Stein-Kousathana |  March 15, 2021 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.5.0-M3! You can find the complete details in the [release notes](https://github.com/spring-projects/spring-security/releases/tag/5.5.0-M3) and the highlights below:

## [](#oauth-20)OAuth 2.0

[gh-8975](https://github.com/spring-projects/spring-security/issues/8975) - Add BearerTokenAuthenticationConverter

## [](#saml)SAML

[gh-9317](https://github.com/spring-projects/spring-security/issues/9317) - Add setMetadataFilename method to Saml2MetadataFilter

[gh-9310](https://github.com/spring-projects/spring-security/issues/9310) - Throw Saml2AuthenticationException in Saml2AuthenticationTokenConverter on deflation or decoding error

## [](#acl)ACL

[gh-9425](https://github.com/spring-projects/spring-security/issues/9425) - Allow ACL to be owned by GrantedAuthoritySid

## [](#kotlin)Kotlin

[gh-9319](https://github.com/spring-projects/spring-security/issues/9319) - Kotlin DSL extension for rememberMe

## [](#web)Web

[gh-9387](https://github.com/spring-projects/spring-security/issues/9387) - Improve HttpSessionSecurityContextRepository performance
