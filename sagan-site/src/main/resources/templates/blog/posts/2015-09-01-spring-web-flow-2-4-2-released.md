# Spring Web Flow 2.4.2 released

_Releases | Rossen Stoyanchev |  September 01, 2015 | 0 Comments_

A minor maintenance release of Spring Web Flow is now available for use. The release addresses compatibility issues against Spring Security 4.0 and the Mojarra and Apache MyFaces JSF implementations. For a list of the addressed issues see the [release notes](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10050&version=14796) JIRA report.
