# Spring Integration 1.0 Milestone 2 Released

_Releases | Mark Fisher |  February 28, 2008 | 0 Comments_

Dear Spring community,  

I am pleased to announce that Spring Integration 1.0.0.m2 has been released.  
[Download](http://static.springframework.org/downloads/nightly/milestone-download.php?project=INT) | [Reference Documentation](http://static.springframework.org/spring-integration/reference/htmlsingle/spring-integration-reference.html) | [JavaDoc](http://static.springframework.org/spring-integration/apidocs/)

This is the second milestone release of this addition to the Spring Portfolio. To see a list of the new features and improvements since Milestone 1, view the [changelog](http://static.springframework.org/spring-integration/changelog.txt). For more information, visit the [Spring Integration Home](/spring-integration). Also, stay tuned to the [SpringSource Team Blog](http://blog.springsource.com/) for a Spring Integration update early next week.

Mark Fisher  
Spring Integration Lead
