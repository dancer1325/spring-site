# Spring XD 1.0.0.M1 released

_Releases | Gunnar Hillert |  June 12, 2013 | 0 Comments_

Dear Spring Community,

The Spring XD team is pleased to announce that the first milestone of **Spring XD** is now [available for download](http://www.springsource.org/download/community?project=Spring%20XD&version=1.0.0.M1).

Spring XD makes it easy to solve common big data problems such as data ingestion and export, real-time analytics, and batch workflow orchestration. The first milestone implements many features and provides a sizable amount of [documentation](http://static.springsource.org/spring-xd/docs/1.0.0.M1/reference/html/).

For more information, please see the [Project Home Page](http://www.springsource.org/spring-xd), the [Release Notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=11401&version=14115) and the [**the blog posting**](http://blog.springsource.org/2013/06/12/spring-xd-1-0-milestone-1-released).

We would love to hear your feedback as we continue working hard towards the final Spring XD 1.0.0 release. If you have any questions, please use [Stackoverflow](http://stackoverflow.com/questions/tagged/springxd) (Tag: **springxd**), and to report any bugs or improvements, please use either the [Jira Issue Tracker](https://jira.springsource.org/browse/XD) or file a [GitHub issue](https://github.com/SpringSource/spring-xd/issues).
