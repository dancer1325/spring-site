# This Week in Spring - March 4th, 2014

_Engineering | Josh Long |  March 05, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*.

As usual, we've got a lot to cover so let's get to it!

1.  Spring Batch and Boot co-founder Dr. Dave Syer has announced [that Spring Boot RC4 is now available](http://spring.io/blog/2014/03/04/spring-boot-1-0-0-rc4-available-now). Check out the latest cut for all the goodies!
2.  There is a new guide that details [how to use Spring Data REST](http://spring.io/blog/2014/02/26/getting-started-with-spring-data-rest)
3.  Spring Data ninja Christoph Strobl has announced that [Spring Data Redis](http://spring.io/blog/2014/03/04/spring-data-redis-1-2-ga-released) 1.2 is now available.
4.  Don't miss Mattias Arthursson on a webinar March 18th, presenting [Spring LDAP 2.0.0](http://spring.io/blog/2014/02/11/webinar-spring-ldap-2-0-0-on-march-18-2014).
5.  Join Juergen Hoeller and the Spring Team for a webinar on [Java 8 and Spring Framework 4.0](https://spring.io/blog/2014/02/11/webinar-spring-framework-4-0-on-java-8-march-25) on March 25!
6.  David Turanski has put together a nice post on [how to use Groovy for bean configuration](http://spring.io/blog/2014/03/03/groovy-bean-configuration-in-spring-framework-4)
7.  Alvaro Videla's and Jan Machacek's talk from SpringOne2GX 2013, [*RabbitMQ is the new King*, is now available online](http://spring.io/blog/2014/03/04/springone2gx-2013-replay-rabbitmq-is-the-new-king)
8.  Gary Russell's SpringOne2GX 2013 talk [introducing Spring Integration's internals is now online](http://spring.io/blog/2014/03/04/springone2gx-2013-replay-spring-integration-internals)
9.  Pance Cavkovski has a nice post demonstrating [the web socket support from the JavaWebSocket project, Java EE 7 and Spring 4](http://hsilomedus.me/index.php/websockets-in-java/).
10.  Norris Shelton Jr. has a nice post on accessing the Spring Security principal from the [currently installed `Authentication` object using a custom annotation](http://norrisshelton.wordpress.com/2014/02/27/when-the-spring-security-principal-isnt-enough-in-a-spring-mvc-controller/)
11.  Quinten Krijger has a nice post on how to [manage session concurrency with Spring Security and Spring MVC](http://blog.trifork.com/2014/02/28/session-timeout-and-concurrent-session-control-with-spring-security-and-spring-mvc/)
12.  Tomasz Nurkiewicz writes about [how to build a custom Spring XML namespace](http://www.javacodegeeks.com/2014/03/custom-spring-namespaces-made-easier-with-jaxb.html). He makes a great point at the beginning, though: Spring doesn't *require* XML!
13.  The *Not Just Another Blog* blog looks at how to [override the Spring Security filter chain](http://shazsterblog.blogspot.com/2014/02/spring-security-custom-filterchainproxy.html)
14.  Thys Michels has a short-and-sweet post on how to configure [a Spring application using Java configuration to consume an ElasticSearch service on Heroku](http://thysmichels.com/2014/03/02/heroku-elastic-search-example/).
