# Spring for Apache Kafka 3.0.14, 3.1.2 and 3.2.0-M1 Available Now

_Releases | Soby Chacko |  February 20, 2024 | 0 Comments_

On behalf of the entire team and everyone in the community who contributed, we are pleased to announce that Spring for Apache Kafka `3.0.14` and `3.1.2` are generally available now.

Spring for Apache Kafka `3.0.14` includes a [few improvments and bug fixes](https://github.com/spring-projects/spring-kafka/releases/tag/v3.0.14). This release will be included as part of the upcoming [Spring Boot 3.1.9 release](https://github.com/spring-projects/spring-boot/milestone/335).

Spring for Apache Kafka `3.1.2` ships with [some new features, enhancements, bug fixes and documentation improvements](https://github.com/spring-projects/spring-kafka/releases/tag/v3.1.2). This release will be part of the upcoming [Spring Boot 3.2.3 release](https://github.com/spring-projects/spring-boot/milestone/336).

We are also happy to announce the first milestone release of Spring for Apache Kafka 3.2.0 (`3.2.0-M1`). This milestone release includes a number of new additions as [features, enhancements, bug fixes and documentation changes](https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.0-M1). This milestone release will be included with the upcoming [Spring Boot 3.3.0-M2 release](https://github.com/spring-projects/spring-boot/milestone/329).

We want to thank everyone who contributed to these releases.
