# Spring for Apache Pulsar 1.1.0 available now

_Releases | Chris Bono |  May 20, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.1.0` has been released and is now available from Maven Central.

This release will be included in the upcoming Spring Boot [3.3.0](https://github.com/spring-projects/spring-boot/milestones/3.3.0) release.

This release includes numerous enhancements, documentation improvements, bug fixes, and dependency upgrades. Notable new features include:

-   Support for Transactions

Please see the [release notes](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.0) for more details.
