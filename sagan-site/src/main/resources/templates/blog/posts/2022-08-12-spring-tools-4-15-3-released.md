# Spring Tools 4.15.3 released

_Releases | Martin Lippert |  August 12, 2022 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.15.3 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

#### [](#fixes-and-improvements)fixes and improvements

-   *(Spring Boot)* fixed: Spring Boot Tools slow down code complete ([#808](https://github.com/spring-projects/sts4/issues/808))
-   *(Language Server)* fixed: Copied files should retain copyright ownership ([#807](https://github.com/spring-projects/sts4/issues/807))

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Detailed changes can be found here: [https://github.com/spring-projects/sts4/wiki/Changelog#2022-08-11-4153-release-incl-language-servers-version-1380](https://github.com/spring-projects/sts4/wiki/Changelog#2022-08-11-4153-release-incl-language-servers-version-1380)

Spring Tools 4.16.0 is scheduled to be released in mid September 2022.

Enjoy!
