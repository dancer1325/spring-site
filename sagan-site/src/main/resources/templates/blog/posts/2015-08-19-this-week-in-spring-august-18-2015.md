# This Week in Spring - August 18, 2015

_Engineering | Josh Long |  August 19, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it! This week I'm back in São Paulo, Brazil, to visit customers, then it's off [to Rio for QCon Rio](http://qconrio.com/)! If you're in either city, message me on Twitter [and let's grab a coffee](http://twitter.com/starbuxman)!

-   Today, Pivotal CEO Paul Maritz stepped down \[and welcomed Pivotal Labs founder Rob Mee as the new CEO. There's a post by [Paul](http://blog.pivotal.io/pivotal/news-2/how-far-weve-come) and by [Rob](http://blog.pivotal.io/pivotal/news-2/we-transform-how-the-world-builds-software). Thank you, Paul, and welcome Rob!
-   the amazing Scott Frederick just announced the [general availability of Spring Cloud Connectors 1.2.0!](http://spring.io/blog/2015/07/29/spring-cloud-connectors-1-2-0-released) - check this out!
-   Spring ninja Stéphane Nicoll just announced Spring Boot 1.3M4, [with fixes and improvements aplenty](https://spring.io/blog/2015/08/18/spring-boot-1-3-0-m4-available-now)!
-   Spring Integration lead Gary Russell just announced a CRAZY packed new Spring Integration 4.2 release including, but not limited to, support for security Context Propagation, STOMP client channel adapters, metrics, new Spring Framework 4.2 event channel adapters, a process barrier component, last modified file list filters, codec, JMS shared subscriptions, (S)FTP improvements, SOAP Action Propagation, [and much more](http://spring.io/blog/2015/08/17/spring-integration-4-2-release-candidate-is-available)
-   Not resting on his laurels, Spring Integration and Spring AMQP lead Gary Russel [*also* announced Spring AMQP 1.5.0 has been released](http://spring.io/blog/2015/08/14/spring-amqp-1-5-0-release-candidate-available)!
-   the SPR blog has a nice post that looks at [internationalization in a Spring Boot application](http://spr.com/part-4-internationalization-in-spring-boot/)
-   Stephane Deraco put together [a nice introductory look at Spring Boot](http://stephane-deraco.github.io/intro-spring-boot.html)
-   Check out this implementation to have bean dependencies [resolved as Spring MVC handler parameters](https://github.com/karlkfi/spring-mvc-params)
-   Check out this two part [podcast I did with Michael Cote on going cloud-native with Spring Boot, Spring Cloud and Cloud Foundry](http://blog.pivotal.io/pivotal-cloud-foundry/features/going-all-microservices-with-spring-cloud-and-pivotal-cloud-foundry)
-   Pivotal Cloud Foundry *finally* has support for Apache Geode (the project formerly known as Gemfire)
-   Our pal Petri is back at it again, this time with a post on [using Gradle for a Spring Boot web application](http://www.petrikainulainen.net/programming/gradle/getting-started-with-gradle-creating-a-spring-boot-web-application-project/)
