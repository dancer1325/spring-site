# Spring Boot 2.0.6

_Releases | Andy Wilkinson |  October 16, 2018 | 0 Comments_

On behalf of the team and everyone that contributed, it is my pleasure to announce that Spring Boot 2.0.6 has been released and is is now available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

Spring Boot 2.0.6 includes [97 fixes, improvements, and dependency updates](https://github.com/spring-projects/spring-boot/releases/tag/v2.0.6.RELEASE) as well as [a Spring Framework CVE report](https://spring.io/blog/2018/10/16/spring-project-vulnerability-reports-published). Thanks to all that have contributed with issue reports and pull requests.
