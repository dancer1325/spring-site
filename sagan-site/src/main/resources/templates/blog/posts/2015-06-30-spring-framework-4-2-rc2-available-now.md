# Spring Framework 4.2.RC2 Available Now

_Releases | Stéphane Nicoll |  June 30, 2015 | 0 Comments_

On behalf of the team I am pleased to announce the second release candidate of Spring Framework 4.2 is now available from our [milestone repository](http://repo.spring.io/milestone/). This release includes [over 70 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=15096).
