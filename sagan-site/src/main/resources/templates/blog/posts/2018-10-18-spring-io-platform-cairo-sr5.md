# Spring IO Platform Cairo-SR5

_Releases | Andy Wilkinson |  October 18, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Cairo-SR5 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Reactor Bismuth-SR12
-   Spring AMQP 2.0.8
-   Spring Boot 2.0.6
-   Spring Data Kay SR11
-   Spring Framework 5.0.10
-   Spring Integration 5.0.9
-   Spring Security 5.0.9
-   Spring Security OAuth 2.2.3
-   Spring Session Apple SR6
-   Spring Web Flow 2.5.1
-   Spring Web Services 3.0.4

The versions of a number of third-party dependencies have also been updated.
