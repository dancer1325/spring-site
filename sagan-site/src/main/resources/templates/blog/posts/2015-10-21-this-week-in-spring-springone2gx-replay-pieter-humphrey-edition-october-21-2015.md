# This Week in Spring ( SpringOne2GX replay / Pieter Humphrey edition!) - October 21, 2015

_Engineering | Josh Long |  October 21, 2015 | 3 Comments_

Welcome to another installation of *This Week in Spring*! This week is the first week (of many) where the *crazy awesome* \[[https://twitter.com/pieterhumphrey\](Pieter](https://twitter.com/pieterhumphrey%5D\(Pieter) Humphrey) has released scores of [SpringOne2GX](http://SpringOne2gx.com) 2015 videos for our binge watching enjoyment! Most of this week's bountiful roundup is thanks to Pieter's meticulous transcoding, uploading and posting videos from SpringOne2GX 2015 - thanks Pieter! So without further ado, let's take a look at some of this week's selection including a LOT of content from SpringOne2GX, including..

-   ..[Juergen Holler and Stephane Nicoll's talk presenting the updates to Spring framework 4.2](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-modern-java-component-design-with-spring-framework-4-2)
-   ..[the SpringOne2GX day 1 keynote](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-day-1-keynote-cloud-native-enterprise) featuring James Watters, Dr. David Syer, Scott Frederick, Juergen Hoeller and yours truly
-   ..[the SpringOne2GX day 2 keynote](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-day-2-technical-keynote-cloud-native-enterprise) featuring Guillaume LaForge, Graeme Rocher, Dr. David Syer, Mark Fisher, Scott Frederick and yours truly
-   ..the talk that Spring Boot and Spring Cloud co-founder Dr. Dave Syer and I, aided by a *VERY* gracious guest appearance by Spring creator and hero [Dr. Rod Johnson](http://twitter.com/springrod), gave [introducing Spring Cloud](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-getting-started-with-spring-cloud) (I've *never been so privileged* as I was to be a part of this talk and I *truly* hope you enjoy this one)
-   ..Netflix's Jon Schneider and Taylor Wicksell [showing - live - how they build cloud native applications using Spring Boot and Spring Cloud at Netflix](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-spring-cloud-at-netflix)
-   ..Matt Stine's talk introducing [some of the powerful new features in Lattice](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-lattice-a-cloud-native-platform-for-your-spring-applications)
-   .. Grails lead [Graeme Rocher's Grails 3 update](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-grails-3-x-update)
-   ..Spencer Gibb's talk [introducing some of the various extensibility planes of Spring Cloud](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-spring-cloud-exploring-alternative-spring-cloud-implementations)
-   ..John Field and Shawn McKinney's talk [discussing some of the security concerns when moving applications to a PaaS](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-a-how-to-guide-to-security-in-the-paas-cloud)
-   ..Will Tran's talk on [securing microservices with Spring Cloud Security](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-securing-microservices-with-spring-cloud-security)
-   ..Google's Ray Tsang introducing how [to run Spring Boot-based microservices on Kubernetes](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-spring-boot-micro-services-containers-and-kubernetes-how-to)
-   ..Nicholas Frankel's [talk introducing devops with Spring Boot](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-spring-boot-for-devops)
-   ..Cornelia Davis' talk on [12 factor Spring applications](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-12-factor-or-cloud-native-apps-for-spring-developers)
-   ..AppDynamics's Dustin Whittle's talk [about modern frontend engineering](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-modern-frontend-engineering)
-   ..VMWare's Emad Benjamin's talk [about building and tunng high performance Java platforms](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-building-and-tuning-high-performance-java-platforms) in a world of virtual machines and the cloud
-   ..Colin Harrington's talk on using Grails to [support Internet-of-Things use cases](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-booting-iot-with-grails)
-   ..the amazing [incomparable Jeff Brown's talk introducing `javax.cache`](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-bringing-javax-cache-ing-to-your-application)
-   ..and Jeff's [talk on RESTful Grails 3](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-restful-grails-3)
-   ..and Jeff's talk on building a [microservice architecture with Grails](http://spring.io/blog/2015/10/19/springone2gx-2015-replay-microservice-architecture-with-grails)

Whew! That's *at least* a week's worth of watching! That's all thanks to Pieter Humphrey working overtime to get this all available as early and abundantly as possible. Pieter is *also* responsible for working very hard over the last few years to ensure that *all* talks are recorded and available so to be able to binge watch everything so quickly after the show is a *true* privilege.
