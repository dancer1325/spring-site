# Spring LDAP 2.0.1 Released

_Releases | Rob Winch |  February 19, 2014 | 0 Comments_

I'm pleased to announce the release of Spring LDAP 2.0.1. This release can be found in Maven Central and contains a number of bug fixes including a fix to work with Spring Security 3.2. For additional details, please see the [changelog](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10071&version=14318).
