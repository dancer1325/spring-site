# This Week in Spring - April 11th, 2017

_Engineering | Josh Long |  April 11, 2017 | 0 Comments_

Welcome to another installation of *This Week in Spring*!

-   Spring Integration ninja Artem Bilan [just announced Spring Integration 5.0.M3](https://spring.io/blog/2017/04/05/spring-integration-5-0-milestone-3-available). The new release includes an integration test framework, speed improvements, support for the Spring 5 reactive `WebClient`, retroactive restoration of file trees when using the (S)FTP and AWS S3 inbound adapters, a smarter SOAP web-services gateway, and much more.
-   Spring Integration lead Gary Russell just announced [Spring for Apache Kafka 1.2.0 and 1.1.4](https://spring.io/blog/2017/04/07/spring-for-apache-kafka-1-2-0-and-1-1-4-available)
-   Andy Wilkinson just debuted a look at [the new Gradle Spring Boot 2.0 plugin](https://spring.io/blog/2017/04/05/spring-boot-s-new-gradle-plugin)
-   in last week's *Spring Tips* I looked at [how to standup a Spring Cloud Security-powered OAuth auth service](https://spring.io/blog/2017/04/05/spring-tips-creating-a-spring-security-oauth-auth-service)
-   [Spring AMQP lead Gary Russell just announced Spring AMQP 2.0.0M3](https://spring.io/blog/2017/04/04/spring-amqp-2-0-0-milestone-3-is-available), *packed* with new features supporting testing, smarter handling for `null` messages, and so much more.
-   Are you going to be at not-to-miss OSCON event in Austin, TX? [We'd love to see you](https://spring.io/blog/2017/04/03/spring-team-and-pivotal-at-oscon-austin-2017)! I'll be there, too!
-   I enjoyed revisting this [video from the Cloud Foundry launch event](https://www.youtube.com/watch?v=Uiqb8q02ZSw). My memory is hazy, but that event - I think - was April 11th, and uploaded on April 12th. Either way, happy birthday Cloud Foundry! You've changed the world with style.
-   Jared Ruckle has a great look all [the new features in PCF 1.10](https://content.pivotal.io/blog/always-be-shipping-with-pivotal-cloud-foundry-1-10-go-fast-go-secure-and-go-net) - check it out!
-   if you want to learn more about our thinking about taking Cloud Foundry to market, you'll love this a16z podcast with one of Pivotal's visionaries, [James Watters](http://a16z.com/2017/04/10/monetizing-open-source-enterprise-software/)
-   I liked this quick look [by Gaurav Rai Mazra at using Spring Data JPA with an embedded database](https://dzone.com/articles/spring-data-jpa-with-an-embedded-database-and-spring-boot)
-   Andreas Falk has a nice article on [securing Spring and Angular.js applications](http://www.novatec-gmbh.de/aktuelles/news/detailansicht-nachrichten/veroeffentlichung-von-andreas-falk-im-java-magazin-52017-sicher-in-die-cloud-mit-angular-2-und-spring/)
-   the Baeldung site has done yet another very interesting look at [the trends in Java, pointing again to the continued growth of Spring Boot and Java 8](http://www.baeldung.com/java-in-2017) - good news!
-   just a friendly reminder, Alibaba uses Spring throughout [and AliExpress, which serves 100+ million uses, is using Spring Boot and Spring Cloud](https://twitter.com/linux_china/status/851261770010210305)
-   I hope you'll join me and [many others at Voxxed Days Singapore](https://voxxeddays.com/singapore/)
-   Alex Falappa‏ hints at some of the [very cool new support for Apache NetBeans](https://twitter.com/aless_falappa/status/850771695656980480)
-   Stefan Rempfer put together a nice example [of using Spring Cloud Zuul with Vaadin](https://github.com/srempfer/spring-cloud-vaadin-proxy-sample)
-   I *love* this blog by Robert Kapala on [how they migrated their investment bank to Spring Boot](https://blog.gft.com/blog/2017/04/07/migrating-to-spring-boot-in-an-investment-bank)
-   Jonas Hecht put together a nice post on running [Spring Boot applications on Windows using Ansible](https://blog.codecentric.de/en/2017/04/ansible-docker-windows-containers-spring-boot/)
