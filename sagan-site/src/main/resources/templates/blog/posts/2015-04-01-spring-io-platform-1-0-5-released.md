# Spring IO Platform 1.0.5 released

_Releases | Andy Wilkinson |  April 01, 2015 | 0 Comments_

We are pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 1.0.5.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform). This release is intended for users who remain on the 1.0.x line; where possible, upgrading to 1.1.x is recommended.

This maintenance release upgrades the versions of a number of the projects in the Platform to pick up their latest maintenance releases:

-   Spring AMQP 1.3.7
-   Spring Batch 3.0.3
-   Spring Boot 1.1.12
-   Spring Data Dijkstra SR5
-   Spring Framework 4.0.9
-   Spring Integration 4.0.6
-   Spring Security 3.2.7
-   Spring Web Services 2.2.1

The versions of many third-party dependencies have also been updated.
