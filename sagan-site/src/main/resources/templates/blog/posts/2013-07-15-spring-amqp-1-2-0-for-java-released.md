# Spring AMQP 1.2.0 for Java Released

_Releases | Gary Russell |  July 15, 2013 | 0 Comments_

The Spring AMQP project applies core Spring concepts to the development of AMQP-based messaging solutions.

We are pleased to announce that the 1.2.0.RELEASE is now available for Java.

Following the [Release Candidate](http://www.springsource.org/node/22635), this GA release includes a minor bug fix and updates the default [RabbitMQ client](http://www.rabbitmq.com/) to 3.1.3. See the [Release Notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10450&version=13906) for details of these changes.

Major changes over and above the 1.1.x release are highlighted in the [What's New section of the reference manual](http://static.springsource.org/spring-amqp/reference/html/whats-new.html).

Please see the [Project page](http://www.springsource.org/spring-amqp) for more information including links to documentation and downloads.
