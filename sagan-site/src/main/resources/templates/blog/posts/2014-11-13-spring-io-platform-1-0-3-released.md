# Spring IO Platform 1.0.3 released

_Releases | Andy Wilkinson |  November 13, 2014 | 2 Comments_

We are pleased to announce that the latest maintenance release, 1.0.3.RELEASE, of Spring IO Platform is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This release upgrades the versions of a number of the projects in the Platform to pick up their latest maintenance releases:

-   Groovy 2.3.7
-   Reactor 1.1.5
-   Spring AMQP 1.3.7
-   Spring Batch 3.0.2
-   Spring Boot 1.1.9
-   Spring for Apache Hadoop 2.0.3
-   Spring Framework 4.0.8
-   Spring Integration 4.0.5
-   Spring Mobile 1.1.3
-   Spring Retry 1.1.2
-   Spring Security OAuth 2.0.4

The versions of many third-party dependencies have also been updated.
