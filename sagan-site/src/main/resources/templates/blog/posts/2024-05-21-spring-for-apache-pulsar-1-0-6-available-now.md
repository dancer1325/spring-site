# Spring for Apache Pulsar 1.0.6 available now

_Releases | Chris Bono |  May 21, 2024 | 0 Comments_

I’m happy to announce that Spring for Apache Pulsar `1.0.6` has been released and is now available from Maven Central.

This release will be included in the upcoming Spring Boot [3.2.6](https://github.com/spring-projects/spring-boot/milestones/3.2.6) release.

This release includes documentation improvements and dependency upgrades.

Please see the [release notes](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.0.6) for more details.
