# Spring IO Platform Athens-SR4

_Releases | Andy Wilkinson |  March 07, 2017 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Athens-SR4 is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](https://search.maven.org/#search%7Cgav%7C1%7Cg%3A%22io.spring.platform%22%20AND%20a%3A%22platform-bom%22).

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.6.8
-   Spring Boot 1.4.5
-   Spring Data Hopper SR8
-   Spring Framework 4.3.7
-   Spring Integration 4.3.8
-   Spring Security OAuth 2.0.13

The versions of a number of third-party dependencies have also been updated.
