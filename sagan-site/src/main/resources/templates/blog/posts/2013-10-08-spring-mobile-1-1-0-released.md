# Spring Mobile 1.1.0 Released

_Releases | Roy Clarkson |  October 08, 2013 | 0 Comments_

Dear Spring Community,

We are happy to announce the GA release of Spring Mobile 1.1.0! This release includes all the changes from the previous milestones, including the following:

-   Improved device detection in LiteDeviceResolver
-   Tablet support in SitePreference and SiteSwitcher
-   Java-based configuration to complement the traditional XML configuration
-   LiteDeviceDelegatingViewResolver for managing mobile and tablet views

See the [Changelog](http://docs.spring.io/spring-mobile/docs/1.1.0.RELEASE/changelog.txt), [JavaDoc](http://docs.spring.io/spring-mobile/docs/1.1.0.RELEASE/api/), and [Reference Documentation](http://docs.spring.io/spring-mobile/docs/1.1.0.RELEASE/reference/html/) for more details. Information about obtaining the artifacts can be found on the [Spring Mobile](http://projects.spring.io/spring-mobile/) project page.
