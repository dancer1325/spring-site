# Spring Framework 5.3.3 available now

_Releases | Stéphane Nicoll |  January 12, 2021 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.3` is available now.

Spring Framework `5.3.3` includes [40 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.3).
