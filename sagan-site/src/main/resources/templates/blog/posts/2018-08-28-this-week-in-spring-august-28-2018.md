# This Week in Spring - August 28, 2018

_Engineering | Josh Long |  August 28, 2018 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! This week I am in Chicago for the SpringOne Tour Chicago event. If you're around, then of [course don't hesitate to say hi](http://twitter.com/starbuxman)! Mostly, though, I've been busily preparing for the [epic SpringOnePlatform event](http://springoneplatform.io) event being held in historic Washington DC in September. I hope to see you there!

We've got a lot to get to so let's get to it!

-   [Spring Cloud Services for PCF Turns 2.0](https://content.pivotal.io/blog/spring-cloud-services-for-pcf-turns-2-0-adds-support-for-your-spring-boot-2-apps), adds support for your Spring Boot 2 applications
-   Nice article on the Pivotal blog on how to build [modern data pipelines with Pivotal GemFire and Spring Cloud Data Flow](https://content.pivotal.io/pivotal-blog/how-to-build-modern-data-pipelines-with-pivotal-gemfire-and-spring-cloud-data-flow)
-   [Spring Cloud Stream Fishtown.M2 /2.1.0.M2 Release Announcement](https://spring.io/blog/2018/08/28/spring-cloud-stream-fishtown-m2-2-1-0-m2-release-announcement)
-   In this installment of my new eight part series introducing Spring Cloud GCP, *Bootiful GCP*, I [look at globally consistent data access with Spring Cloud GCP and Google Cloud Spanner](https://spring.io/blog/2018/08/27/bootiful-gcp-globally-consistent-data-access-with-spanner-3-8)
-   In this, the second of an eight part series looking at Spring Cloud GCP and Google Cloud GCP, I [introduce RDBMS-based data access with Spring Cloud GCP](https://spring.io/blog/2018/08/23/bootiful-gcp-relational-data-access-with-spring-cloud-gcp-2-8)
-   [Love Spring and Spring Boot? Then, You’re Going to Love these Projects, too.](https://content.pivotal.io/blog/love-spring-and-spring-boot-then-you-re-going-to-love-these-projects-too)
-   [Spring Integration for AWS 2.0 GA and Spring Cloud Stream Kinesis Binder 1.0 GA](https://spring.io/blog/2018/08/21/spring-integration-for-aws-2-0-ga-and-spring-cloud-stream-kinesis-binder-1-0-ga)
-   Nice post over on the Stark and Wayne blog [on how to reload buildpacks in Cloud Foundry](https://starkandwayne.com/blog/how-to-reload-buildpacks-in-cf/)
-   The Cloud Foundry foundation [ranked programming languages most used for enterprise application development](https://sdtimes.com/java/report-finds-java-and-javascript-remain-the-top-languages-for-enterprise-development/). No surprise here: Java ranks first.
-   Join us at the [Cloud Foundry Summit in Philadelphia](https://www.cloudfoundry.org/blog/cloud-foundry-north-american-summit-coming-to-philadelphia-in-april-2019/), PA on April 2nd-4th, 2019
-   This is an interesting post on how to deploy [a Spring Cloud Function-based application using the `npm`\-based Serverless framework to deploy to AWS Lambda](https://www.linkedin.com/pulse/deploying-spring-cloud-function-aws-lambda-platform-madhu-babu/)
-   Check out this guide on things to [see on day 2 of SpringOne Platform](https://www.linkedin.com/pulse/springone-platform-my-guide-spring-developers-day-2-guedes-rettori/)
-   Check out this insightful post on using [Spring Security with Microsoft Azure B2C](https://www.lionmint.com/en/spring-boot-security-with-azure-b2c-1/amp/?__twitter_impression=true)
-   Benjamin Wilms [updated the docs of Chaos Monkeys for Spring Boot](https://twitter.com/mrbwilms/status/1032945865898569728?s=12) and described how to use it without adding it to your dependencies.
-   [Spring Framework 5.1 will no longer proxy your test instances unnecessarily](https://twitter.com/sam_brannen/status/1032950245842399233?s=12), thanks to Juergen Hoeller who is so amazing he sometimes *unwittingly* resolves issues! In this case, the fix arrived in [SPR-9478](https://jira.spring.io/browse/SPR-9478) while tackling a Kotlin-related issue [SPR-17137](https://jira.spring.io/browse/SPR-17137).
-   The first [JDK 11 Release Candidate is now available](http://mail.openjdk.java.net/pipermail/jdk-dev/2018-August/001844.html)
-   Check out the Netifi platform, a new platform based on RSocket, which [is in turn based on Reactor](https://devops.com/netifi-rises-to-microservices-integration-challenge/)
-   This is pretty cool! The concept of buildpacks, which spun out of Heroku and have been embraced by the whole Cloud Foundry community, was [presented recently to the Cloud Native Foundation](https://twitter.com/rettori/status/1031931357621104640)
-   [Do *not* miss SpringOne Platform](https://twitter.com/springcentral/status/1032344392772726785)! The agenda is up and waiting!
-   Check out this next nice installment of [a video series looking at Apache Geode](https://www.youtube.com/watch?v=g59Q529fna0&feature=youtu.be)
-   [The latest release of the Assembler project](https://twitter.com/sebastien_pel/status/1031884585829888001?s=12), which is a functional, type-safe and stateless Java API for solving the N + 1 query problem in multi-databases and microservices aggregation, is now available. Of course, it also supports Reactor for composition!
