# Spring Security 6.3.7 and 6.4.3 are now available

_Releases | Josh Cummings |  February 18, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security 6.3.7 and 6.4.3 are out! In all cases, the releases are mostly composed of bug fixes, dependency upgrades, and documentation improvements.

To learn more, please visit the [6.3.7](https://github.com/spring-projects/spring-security/releases/tag/6.3.7) and [6.4.3](https://github.com/spring-projects/spring-security/releases/tag/6.4.3) release summaries.

Commercial customers using Spring Boot 2.7, 3.0, 3.1, or 3.2 will be able to update to Spring Boot 2.7.24, 3.0.19, 3.1.15, or 3.2.13 respectively to receive the corresponding Security releases 5.7.15, 6.0.15, 6.1.13, and 6.2.9. These Security versions are available now on the Spring commercial artifact repository and can be accessed with a [Spring Enterprise Subscription](https://spring.vmware.com).
