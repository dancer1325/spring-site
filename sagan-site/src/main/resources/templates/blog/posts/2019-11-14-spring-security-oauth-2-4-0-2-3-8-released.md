# Spring Security OAuth 2.4.0, 2.3.8 Released

_Releases | Joe Grandja |  November 14, 2019 | 0 Comments_

I’m pleased to announce the releases of Spring Security OAuth 2.4.0 and 2.3.8.

### [](#deprecation-notice)[](#deprecation-notice)**Deprecation Notice**

The 2.4.0 release officially deprecates all classes. The latest OAuth 2.0 support is provided by [Spring Security](https://spring.io/projects/spring-security). See the [announcement](https://spring.io/blog/2019/11/14/spring-security-oauth-2-0-roadmap-update) for further details.

For a complete list of changes, please refer to:

-   [2.4.0 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/55?closed=1)
    
-   [2.3.8 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/56?closed=1)
