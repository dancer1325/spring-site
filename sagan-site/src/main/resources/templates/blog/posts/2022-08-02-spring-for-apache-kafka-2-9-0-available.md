# Spring for Apache Kafka 2.9.0 Available

_Releases | Gary Russell |  August 02, 2022 | 1 Comment_

I am pleased to announce that Spring for Apache Kafka 2.9.0 is now available.

This version is not provided by Spring Boot dependency management, but it can be used with Boot 2.7 (or 2.6), as long as you override the Kafka dependencies to 3.2.0 as described in [this appendix](https://docs.spring.io/spring-kafka/docs/2.9.0/reference/html/#update-deps), especially if you are using the embedded Kafka broker.

Please see the [release candidate announcement](https://spring.io/blog/2022/07/07/spring-for-apache-kafka-2-9-release-candidate-available) for notable changes in this release.

For information about all changes in this release, see [What’s New](https://docs.spring.io/spring-kafka/docs/2.9.0/reference/html/#whats-new-part).
