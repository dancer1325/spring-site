# This Week in Spring - July 4th, 2017

_Engineering | Josh Long |  July 05, 2017 | 0 Comments_

Welcome to another installment of *This Week in Spring*! If you're in the US, then let me wish you a happy 4th of July! We've got a lot to cover so let's get to it!

-   Spring Data ninja Thomas Risberg has just [announced Spring Cloud Data Flow 1.2.2](https://spring.io/blog/2017/06/29/spring-cloud-data-flow-1-2-2-released). This new version improves compatibility with Spring Boot 1.5.4 which in turn brought in a new version of Spring Data.
-   The [*Spring Framework Guru* web portal](https://springframework.guru/what-is-new-with-spring-framework-5/) has an interesting look at what's new in Spring Framework 5! Check it out!
-   I liked this post on *Bozho's Tech Blog* on setting up the epic [Spring Boot Admin](https://techblog.bozho.net/simple-spring-boot-admin-setup/) for monolithic applications, eschewing some of the more common, albeit more complex, setups.
-   our friend Petri Kainulainen is back at it again [this time with a new newsletter](https://www.petrikainulainen.net/weekly/java-testing-weekly-27-2017/) that looks to capture discussions of testing in Java. This seems like a no-brainer thing to follow for interesting content! Nice job, as usual, Petri!
-   I love [this nicely illustrated look at what constitutes a Spring bean](https://firstinthecloud.com/spring-boot-101).
-   [Baeldung has a nice Spring Cloud tutorial](http://www.baeldung.com/spring-cloud-tutorial) that's worth checking out.
-   I was very honored to be nominated as one of the [top java influencers by JAX 2017](https://jaxlondon.com/blog/java-core-languages/top-20-java-influencers-of-2017/) along with Spring Data lead Oliver Gierke. This is an honor. Thank you. :)
-   don't miss reactive web ninja Arjen Poutsma's [talk introducing functional reactive Spring](https://content.pivotal.io/webinars/jun-14-new-in-spring-framework-5-0-functional-web-framework-webinar)!
-   Speaking of *reactive*, this upcoming webinar on [reactive Spring Data Couchbase looks promising!](https://content.pivotal.io/spring/jul-13-full-stack-reactive-nosql-spring-data-and-couchbase-webinar) - check it out!
-   I liked this [demo of a (reactive) Spring WebFlux application](https://github.com/orlandovald/webflux-twitter-demo) that connects to Twitter's streaming API
