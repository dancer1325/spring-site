# Spring Security 5.4.5 released

_Releases | Josh Cummings |  February 18, 2021 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.4.5 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.4.5)). This release delivers bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
