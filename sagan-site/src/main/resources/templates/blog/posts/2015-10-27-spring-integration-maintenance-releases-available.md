# Spring Integration Maintenance Releases Available

_Releases | Gary Russell |  October 27, 2015 | 0 Comments_

We are pleased to announce the following release versions are now available. These versions include important bug fixes and users should upgrade as soon as possible. Click the version to see the appropriate JIRA release notes.

[4.2.1.RELEASE](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=15324) [4.1.7.RELEASE](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=15242) [4.0.8.RELEASE](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=14991) [3.0.8.RELEASE](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=14992)

Please note that, unless some compelling reason arises, it is anticipated that the 4.1.7 and 4.0.8 releases will be the last in those lines; 4.x users are encouraged to upgrade to 4.2.1, which is the current release for the 4.x line.

We expect to continue to make available further 3.0.x releases to address important bug fixes, as necessary, at least through the end of 2016.

Please refer to the [project page](http://projects.spring.io/spring-integration/) for links to documentation, downloads etc.
