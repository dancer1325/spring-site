# Spring Data Fowler SR 1 released

_Releases | Christoph Strobl |  July 01, 2015 | 0 Comments_

Hot on the heels of the [final service updates](https://spring.io/blog/2015/07/01/final-spring-data-service-releases-evans-sr3-and-dijkstra-sr6-released) for the release trains Dijkstra and Evans, the first service release for the Spring Data release train Fowler has just been shipped to [repo.spring.io](http://repo.spring.io/) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.data%22). The Fowler SR1 maintenance release includes a total of [119 issues](https://jira.spring.io/issues/?filter=15140) solved and is a recommended upgrade for all users.
