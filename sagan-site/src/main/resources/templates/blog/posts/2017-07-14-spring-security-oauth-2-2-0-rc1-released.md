# Spring Security OAuth 2.2.0.RC1 Released

_Releases | Joe Grandja |  July 14, 2017 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security OAuth 2.2.0.RC1.

The [2.2.0.RC1 release](https://github.com/spring-projects/spring-security-oauth/milestone/27?closed=1) includes numerous improvements to the JSON Web Key (JWK) feature contained in `JwkTokenStore` along with the addition of `JwtClaimsSetVerifier` that provides the capability of custom JWT Claim(s) verification. See the [GitHub Issue](https://github.com/spring-projects/spring-security-oauth/issues/1111) for more details.

This release also includes a small number of bug fixes and minor enhancements.
