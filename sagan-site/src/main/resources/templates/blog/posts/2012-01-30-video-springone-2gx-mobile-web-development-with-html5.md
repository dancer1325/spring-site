# VIDEO: SPRINGONE 2GX - Mobile Web Development with HTML5

_News | Josh Long |  January 30, 2012 | 0 Comments_

[![SpringOne 2GX Video: Mobile Web Development with HTML5](http://www.springsource.org/files/u41069/roy-and-josh-springone_0.png)](http://www.infoq.com/presentations/Mobile-Web-Development-with-HTML5)

[SpringOne 2GX 2011](http://www.springone2gx.com) was filled with great content on building better mobile and web applications. In this presentation, [Mobile Web Development with HTML5](http://www.infoq.com/presentations/Mobile-Web-Development-with-HTML5), we get to learn about building compelling mobile applications with HTML5 and technologies like PhoneGap.

This presentation is given by [Spring Android](http://www.springsource.org/spring-android) lead and mobile expert [Roy Clarkson](http://www.springone2gx.com/conference/speaker/roy_clarkson) and Spring Developer Advocate [Josh Long](http://www.springone2gx.com/conference/speaker/josh_long).

Roy and Josh introduce the innate capabilities of modern HTML 5 mobile browsers and introduce techniques and technologies  
like PhoneGap, a shell for web applications on mobile clients, and Jo, a widget framework,  
to simplify and augment mobile development.

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
