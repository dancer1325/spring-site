# SpringOne 2GX 2010 Tech Talk: Configuration Enhancements in Spring 3.1

_News | Adam Fitzgerald |  February 14, 2011 | 0 Comments_

[![SpringOne 2GX 2010 Tech Talk: Configuration Enhancements in Spring 3.1](/files/SpringOne2GX2010ChrisBeams.png)](http://www.infoq.com/presentations/Configuration-in-Spring-3-1)

After last week's [release of Spring 3.1](http://www.springsource.org/node/3026), there is some great material to help you get up to speed with the new enhancements. [InfoQ](http://www.infoq.com/SpringOne_2010) has published another talk from [SpringOne 2GX 2010](http://www.springone2gx.com), this one covers the [Configuration Enhancements in Spring 3.1](http://www.infoq.com/presentations/Configuration-in-Spring-3-1). [Chris Beams](http://blog.springsource.com/author/cbeams/) covers unified property management, bean definition profiles, nested elements, c: namespace, Grails BeanBuilder integration.

Also last week [Juergen Hoeller](http://blog.springsource.com/author/juergenh/) gave a webinar on the Spring 3.1 milestone release. In case you can were not able to attend in person, a full [recording of the webinar](http://www.springsource.com/webinar/introducing-spring-framework-31) has now been posted so you can get all of the details from the source!

Also don't forget to read Chris's recent blog post about [Bean Definition Profiles](http://blog.springsource.com/2011/02/11/spring-framework-3-1-m1-released/). This is the first in a [short series](http://blog.springsource.com/category/spring/31/) that will be covering the details of the new release.
