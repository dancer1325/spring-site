# Spring AMQP 1.4.4 and Spring Integration 3.0.7, 4.0.7 and 4.1.3 are available

_Releases | Artem Bilan |  April 07, 2015 | 0 Comments_

Dear Spring community,

On behalf of the Spring Integration/AMQP team I'm pleased to announce that we have released a number of maintenance releases (click the links in the following bullets to see the corresponding release notes):

-   Spring AMQP [1.4.4](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10450&version=14927).
-   Spring Integration [3.0.7](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10121&version=14823).
-   Spring Integration [4.0.7](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10121&version=14860).
-   Spring Integration [4.1.3](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10121&version=14897).

All of these releases are available from the [Release Repository](http://repo.springsource.org/release), as well as from Maven Central.

Thank you very much for everyone involved!

We are working on the first milestone for the Spring AMQP `1.5` release in the next several weeks, which will have some new features such as support for the RabbitMQ Management HTTP API, RabbitMQ `3.5` compatibility and other improvements.

Spring Integration `4.2` Milestone 1 is planned to be released in a month or so with full support for Spring Framework `4.2`, with feature candidates such as STOMP and Zookeeper adapters, as well as JMX metrics improvements and many others.

Stay in touch and don't hesitate to share any ideas or issues with us!
