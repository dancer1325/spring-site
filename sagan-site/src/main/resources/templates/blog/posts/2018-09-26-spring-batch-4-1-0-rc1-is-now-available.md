# Spring Batch 4.1.0.RC1 is now available

_Releases | Mahmoud Ben Hassine |  September 26, 2018 | 4 Comments_

On behalf of the Spring Batch team, I’m pleased to announce that Spring Batch 4.1.0.RC1 is now available on Github and the Pivotal download repository.

# [](#whats-new)What’s new?

In this release, we worked mainly on making Spring Batch build and run correctly on Java 8, 9, 10 and 11! This release is based on [Spring Framework 5.1 GA](https://spring.io/blog/2018/09/21/spring-framework-5-1-goes-ga) as well as the latest versions of Spring Integration, Spring AMQP and Spring Data. For a complete list of changes, please refer to the [change log](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10090&version=16996).

# [](#whats-next)What’s next?

Our plan is to release Spring Batch 4.1 GA by the end of October right in time for Spring Boot 2.1 GA. The focus will be on making this release candidate as stable as possible so please help us by testing new features and submitting your feedback on [JIRA](https://jira.spring.io/projects/BATCH/summary), [StackOverflow](https://stackoverflow.com/questions/tagged/spring-batch) or [Gitter](https://gitter.im/spring-batch/Lobby). You can consume Spring Batch 4.1.0.RC1 with [Spring Boot 2.1.0.M4](https://spring.io/blog/2018/09/25/spring-boot-2-1-m4-available-now).

# [](#springone-platform-2018)SpringOne Platform 2018

The Spring Batch team is present at [SpringOne Platform](https://springoneplatform.io) conference. [`@michaelminella`](https://www.twitter.com/michaelminella) and [`@_benas_`](https://www.twitter.com/_benas_) will be presenting a session on [high performance batch processing](https://springoneplatform.io/2018/sessions/high-performance-batch-processing) on Thursday September 27, 2018. You are welcome to this session to see this latest 4.1.0.RC1 release in action!
