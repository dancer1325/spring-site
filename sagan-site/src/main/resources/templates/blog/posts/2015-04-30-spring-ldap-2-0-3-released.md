# Spring LDAP 2.0.3 Released

_Releases | Rob Winch |  April 30, 2015 | 2 Comments_

I’m pleased to announce the release of Spring LDAP 2.0.3.RELEASE. The highlights of this release include:

-   [LDAP-330](https://jira.spring.io/browse/LDAP-330) - Support for Spring Data Commons 1.10 (Spring Data Fowler)
-   [LDAP-304](https://jira.spring.io/browse/LDAP-304) - NullPointerException DirContextAdapter.collectModifications
-   [LDAP-314](https://jira.spring.io/browse/LDAP-314) - repository methods ignoring @Entity(base=)
-   [LDAP-317](https://jira.spring.io/browse/LDAP-317) - ldap:context-source/url not parsing properties #{}
-   [LDAP-321](https://jira.spring.io/browse/LDAP-321) - IllegalStateException: No value for key PoolingContextSource

For additional information on the release, refer to the [changelog](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10071&version=14532).
