# Spring Framework 5.3.1 and 5.2.11 available now

_Releases | Stéphane Nicoll |  November 10, 2020 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.1` and `5.2.11` are available now.

Spring Framework `5.3.1` includes [29 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.1). Spring Framework `5.2.11` includes [10 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.11.RELEASE).

Stay tuned for follow-up Spring Boot releases later this week, including Spring Boot 2.4 GA!
