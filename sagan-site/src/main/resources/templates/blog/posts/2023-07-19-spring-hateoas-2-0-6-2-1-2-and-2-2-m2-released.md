# Spring HATEOAS 2.0.6, 2.1.2 and 2.2 M2 released

_Releases | Oliver Drotbohm |  July 19, 2023 | 0 Comments_

I am happy to announce the availability of Spring HATEOAS [2.0.6](https://github.com/spring-projects/spring-hateoas/releases/tag/2.0.6), [2.1.2](https://github.com/spring-projects/spring-hateoas/releases/tag/2.1.2) and [2.2 M2](https://github.com/spring-projects/spring-hateoas/releases/tag/2.2.0-M2). All releases solely ship a fix for a regression in our AOT reflection metadata generation so that applications building native images continue to work on the upcoming Spring Boot releases.
