# This Week in Spring - March 3, 2015

_Engineering | Josh Long |  March 03, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in Kansas City, Kansas, hanging out with my pal and Spring Security lead Rob Winch. This week, I'm speaking at the local User Group and the Spring User Group, and I'll be talking to Cerner and a few other large businesses about Spring, Spring Boot and Spring Cloud.

As usual, we've got a lot of great stuff to talk about, so let's go to it!

1.  Spring Boot co-lead Phil Webb [just announced Spring Boot 1.2.2](http://spring.io/blog/2015/02/27/spring-boot-1-2-2-released) and [Spring Boot 1.1.11](http://spring.io/blog/2015/02/27/spring-boot-1-1-11-released). Both releases are recommended upgrades, and Spring Boot 1.2.2 even includes new support for the Mustache templating library. Nice!
2.  Check out John Hann's [SpringOne2GX 2014 talk, introducing Rave.js, which brings Spring Boot-concepts to JavaScript](http://spring.io/blog/2015/03/03/springone2gx-2014-replay-ravejs-spring-boot-concepts-for-javascript-applications)
3.  I put together a blog on building [applications that use the Servlet HTTP session in a scalable manner and in a portable way - from application server, web server, and cloud - with ease](http://spring.io/blog/2015/03/01/the-portable-cloud-ready-http-session).
4.  Chris Beams's SpringOne2GX 2014 talk, [*The Revolution will not be Centralized*](http://spring.io/blog/2015/03/03/springone2gx-2014-replay-the-revolution-will-not-be-centralized) is now available online.
5.  Want to learn more about caching in Spring? Check out this replay [of Michael Plod's SpringOne2GX 2014 talk introducing advanced caching improvements in Spring framework](http://spring.io/blog/2015/03/03/springone2gx-2014-replay-caching-with-spring-advanced-topics-and-best-practices)
6.  Brian Clozel and Rossen Stoyanchev's SpringOne2GX 2014 talk [introducing resource handling in Spring MVC 4.1](http://spring.io/blog/2015/03/03/springone2gx-2014-replay-resource-handling-in-spring-mvc-4-1) is *awesome* check it out!
7.  Spring Security lead Rob Winch [just announced Spring Security 4.0 RC2](http://spring.io/blog/2015/02/26/spring-security-4-0-0-rc2-released) which includes, among other things, support for websockets! Check it out and try the bits out now!
8.  Rob Winch also just announced [Spring Security 3.2.6 which improves the Java configuratino and Spring Boot integration](http://spring.io/blog/2015/02/26/spring-security-3-2-6-released), so check it out!
9.  This isn't specific to Spring, or any tech, per se, but it's inspiring an inspiring at [how machine learning used to predict clinical response to anti-cancer drugs](http://www.cio.com.au/article/569280/machine-learning-used-predict-clinical-response-anti-cancer-drugs/)
10.  Speaking of Rob Winch, and education ninja Ken Kreuger, check out their SpringOne2GX 2014 talk on integration testing, [now available on InfoQ](http://www.infoq.com/presentations/spring-mvc-test-htmlunit).
11.  Alex Zhitnitsky put together a [nice comparison of Spring Boot and Dropwizard](http://blog.takipi.com/java-bootstrap-dropwizard-vs-spring-boot/)
12.  Check out these [Spring Cloud PaaS connectors for IBM's Bluemix, which is based on Cloud Foundry](https://t.co/usPYXAXWT3)
13.  Rahman Usta put together an [*amazing* cool demo of a web-based SSH client (with Spring Boot in the backend)](https://github.com/rahmanusta/ssh-on-web)
14.  This video demonstrates how [to use Codeenvy and Spring Boot](https://www.youtube.com/watch?v=9JTR7Y_aHuw)
