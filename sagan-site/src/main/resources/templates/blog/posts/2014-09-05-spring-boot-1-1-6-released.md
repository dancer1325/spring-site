# Spring Boot 1.1.6 released

_Releases | Phil Webb |  September 05, 2014 | 1 Comment_

We are pleased to announce that Spring Boot 1.1.6 has been released and is available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This release closes almost 50 reported issues and is a recommended upgrade for all Spring Boot users. For a complete list of changes please refer to the [issue tracker](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.1.6).
