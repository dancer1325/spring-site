# Spring for GraphQL 1.2.2 released

_Releases | Brian Clozel |  July 18, 2023 | 5 Comments_

I'm pleased to announce that Spring for GraphQL 1.2.2 is now available on Maven Central.

This maintenance release in the 1.2.x generation includes [13 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.2.2), including [1 regression](https://github.com/spring-projects/spring-graphql/issues?q=label%3A%22type%3A+regression%22+milestone%3A1.2.2+). This version will be shipped with Spring Boot 3.1.2, [to be released later this week](https://calendar.spring.io/).
