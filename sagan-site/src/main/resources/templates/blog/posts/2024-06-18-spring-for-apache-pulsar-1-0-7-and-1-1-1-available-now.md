# Spring for Apache Pulsar 1.0.7 and 1.1.1 available now

_Releases | Chris Bono |  June 18, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.0.7` and `1.1.1` have been released and are now available from Maven Central.

The releases will be included in the upcoming Spring Boot [3.2.7](https://github.com/spring-projects/spring-boot/milestones/3.2.7) and [3.3.1](https://github.com/spring-projects/spring-boot/milestones/3.3.1) releases, respectively.

The releases include dependency upgrades.

Please see the release notes ([1.0.7](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.0.7) and [1.1.1](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.1)) for more details.
