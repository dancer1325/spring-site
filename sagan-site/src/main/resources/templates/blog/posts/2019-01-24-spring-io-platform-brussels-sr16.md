# Spring IO Platform Brussels-SR16

_Releases | Andy Wilkinson |  January 24, 2019 | 0 Comments_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Brussels-SR16 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.12
-   Spring Batch 3.0.10
-   Spring Boot 1.5.19
-   Spring Data Ingalls SR18
-   Spring Framework 4.3.22
-   Spring Integration 4.3.19
-   Spring Security 4.2.11
-   Spring Session 1.3.5
-   Spring Web Services 2.4.4

The versions of a number of third-party dependencies have also been updated.
