# Spring Framework 2.0.1 Released

_Releases | Juergen Hoeller |  November 23, 2006 | 0 Comments_

Dear Spring community,  
  
We are pleased to announce that [Spring 2.0.1](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=173644) has been released. [Download](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=173644&release_id=465856) | [Docs](http://static.springframework.org/spring/docs/2.0.1/reference/index.html) | [Changelog](http://static.springframework.org/spring/docs/2.0.1/changelog.txt).  
  
This is the first maintenance release for Spring 2.0, including fixes for most bugs that have been reported since 2.0 final as well as many minor enhancements. This release also introduces OpenJPA support and GlassFish weaving support. Furthmermore, it features redesigned AspectJ support internals and an improved bean definition tooling SPI.  
  
As always, please see the [changelog](http://static.springframework.org/spring/docs/2.0.1/changelog.txt) for the details.

If you are new to the Spring 2.0 series, please also see the original [Spring 2.0 announcement](/go-2.0) for more information.  
  
FYI, the Spring 2.0.2 release is scheduled for end of December. Our JIRA road map outlines the issues that are planned to be addressed in that release.
