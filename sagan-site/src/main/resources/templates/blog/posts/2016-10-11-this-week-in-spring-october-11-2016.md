# This Week in Spring - October 11, 2016

_Engineering | Josh Long |  October 11, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! It's already almost mid October and I can't believe we're almost in 2017! Time is sure flying quickly!

-   It's an honor that today at the JAX London event, the [JAX community **voted Spring Boot as most innovative contribution to the Java ecosystem**](https://twitter.com/jaxenter/status/785897525488254976). I'm sure I speak for the whole Spring team when I say, Thank You very much!
-   Martin Lippert just announced Spring Tool Suite 3.8.2 featuring smarter integration with [the Spring Initializr and updates to Eclipse Neon 1](https://spring.io/blog/2016/10/07/spring-tool-suite-3-8-2-released) as well as bugfixes.
-   I published the first installment of what I hope will be a recurring series of hot-take, live-coded looks at different aspects of the Spring and Pivotal ecosystem.
-   Spring Cloud Data Flow ninja Eric Bottard just announced that [Spring Cloud Data Flow for Cloud Foundry 1.1M1 and 1.0.1 have ben released](https://spring.io/blog/2016/10/06/spring-cloud-data-flow-for-cloud-foundry-1-1m1-and-1-0-1-released)
-   Spring founder and Atomist CEO Dr. Rod Johnson has a new tool called Atomist which seeks to make your project development lifecycle easier. In this post, [Rod looks at how to use Atomist with Spring Boot code](https://medium.com/the-composition/evolving-projects-using-atomist-editors-fd3b1f057c86#.xfrsbtxbn)
-   RedMonk analyst James Governor has some interesting insights in his JAX London talk on the new world of Java, which [are summarized in this nice summary blog from JAX](https://jaxenter.com/129566-129566.html)
-   I did an interview [with JAXenter talking about the motivations behind microservices](https://jaxenter.com/most-important-benefit-microservices-is-agility-129472.html)
-   I liked this post by Biju Kunjummen on how to [customize Spring web suppports `HttpMessageConverters`](https://dzone.com/articles/customizing)
-   Boom! Save the date my friends! *SpringOne Platform 2017* will be held [December 4-7, 2017, at Moscone West in San Francisco](https://springoneplatform.io/)! I, for one, can't freaking wait!
-   MyBatis is up and available on the Spring Initializr, [announced Spring ninja Stephane Nicoll](https://twitter.com/snicoll/status/785386501328760832). Enjoy!
-   Cool! InfoQ has a post [on the latest Spring Boot releases](https://www.infoq.com/news/2016/10/spring-releases-boot)
-   This post on [Spring Boot's embedded web containers](http://nixmash.com/java/know-your-spring-boot-embedded-servlet-container/), by Dave Burke, is a good read.
