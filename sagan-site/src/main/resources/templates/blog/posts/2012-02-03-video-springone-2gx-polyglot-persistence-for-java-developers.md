# Video: SpringOne 2GX - Polyglot Persistence for Java Developers

_News | Adam Fitzgerald |  February 03, 2012 | 0 Comments_

[![SpringOne 2GX Video: Polyglot Persistence for Java Developers - Moving out of the Relational Comfort Zone](http://www.springsource.org/files/SpringOne2GX2012PolyglotPersistenceForJavaDevelopers.png)](http://www.infoq.com/presentations/Polyglot-Persistence-for-Java-Developers)

One of the top rated sessions from SpringOne 2GX 2011 was [Chris Richardson](http://blog.springsource.org/author/crichardson/)'s talk about [Polyglot Persistence for Java Developers - Moving out of the Relational Comfort Zone](http://www.infoq.com/presentations/Polyglot-Persistence-for-Java-Developers). In this talk, Chris provides a great overview of the new non-relational storage options available to Java developers, including Cassandra, Redis and MongoDB. He then talks about the practical matters of when and how to incorporate them into your applications. If you haven't already been experimenting with noSQL datastores then this presentation provides the perfect introduction for you.

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
