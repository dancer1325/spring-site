# Spring IO Platform 1.1.3 released

_Engineering | Andy Wilkinson |  July 13, 2015 | 2 Comments_

We are pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 1.1.3.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform to pick up their latest maintenance releases:

-   Spring AMQP 1.4.5
-   Spring Batch 3.0.4
-   Spring Boot 1.2.5
-   Spring Data Evans SR3
-   Spring Framework 4.1.7
-   Spring Integration 4.1.5
-   Spring LDAP 2.0.3
-   Spring Mobile 1.1.4
-   Spring Session 1.0.1
-   Spring Social 1.1.2
-   Spring Social Facebook 2.0.1

The versions of many third-party dependencies have also been updated.
