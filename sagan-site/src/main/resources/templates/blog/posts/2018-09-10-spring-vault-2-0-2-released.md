# Spring Vault 2.0.2 released

_Releases | Mark Paluch |  September 10, 2018 | 0 Comments_

On behalf of the community, I’d like to announce the availability of the Spring Vault service release 2.0.2.RELEASE, available from Maven Central.

This release primarily delivers bug fixes and dependency version updates along with some minor improvements.

For a complete list of changes see the [changelog](http://docs.spring.io/spring-vault/docs/2.0.2.RELEASE/changelog.txt).
