# Spring Boot 3.5.0 available now

_Releases | Stéphane Nicoll |  May 22, 2025 | 7 Comments_

On behalf of the Spring Boot team and everyone that has contributed, I am pleased to announce that Spring Boot 3.5.0 has been released and is available from Maven Central.

This release adds a significant number of new features and improvements. For full [upgrade instructions](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-3.5-Release-Notes#upgrading-from-spring-boot-34) and [new and noteworthy](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-3.5-Release-Notes#new-and-noteworthy) features please see the [release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-3.5-Release-Notes).

## [](#whats-new-in-35)What's new in 3.5

The highlights of the 3.5 release include:

-   Structured Logging Improvements
-   SSL Support for Service Connections
-   Load Properties From Environment Variables
-   AsyncTaskExecutor with Custom Executor
-   Auto-configuration for Bean Background Initialization
-   Annotations to Register Filter and Servlet

### [](#dependency-upgrades)Dependency upgrades

Spring Boot 3.5 moves to new versions of several Spring projects and we’ve also upgraded to the latest stable releases of other third-party libraries wherever possible. Please see the [release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-3.5-Release-Notes#dependency-upgrades) for details.

### [](#other-changes)Other changes

There are many other changes and improvements that are documented in the [release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-3.5-Release-Notes). You can also find a list of deprecated classes and methods that we plan to remove in the future.
