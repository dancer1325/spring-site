# Spring Batch 3.0.4.RELEASE is now available

_Releases | Michael Minella |  May 19, 2015 | 0 Comments_

We are pleased to announce that Spring Batch 3.0.4.RELEASE is now available via Maven Central, Github and the SpringSource download repository. This is the 4th maintenance release for the 3.0.x branch of Spring Batch and addresses a number of bugs and a few minor enhancements. Many thanks to all of those who submitted the many [pull requests](https://github.com/springsource/spring-batch/pulls) that went into this release.

[Spring Batch Home](http://spring.io/projects/spring-batch) | [Source on GitHub](https://github.com/spring-projects/spring-batch) | [Reference Documentation](http://docs.spring.io/spring-batch/)

We look forward to your feedback in [Jira](https://jira.spring.io/browse/BATCH), [StackOverflow](http://stackoverflow.com/questions/tagged/spring-batch), or to me directly via Twitter [@michaelminella](twitter.com/michaelminella).
