# This Week in Spring - September 24th, 2019

_Engineering | Josh Long |  September 24, 2019 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*, a weekly roundup where I get to swim through the great content in the ecosystem and learn things and hopefully share interesting tidbits with you all. I mark my calendar by the number of *This Week in Springs*, so imagine my elation to realize looking at the calendar that we're *really* close to the [epic SpringOne Platform 2019](https://springoneplatform.io)! I can't believe it!

The event takes place soon - October 7–10, 2019 - in Austin, Texas. That's just around the corner, but I've still got a few places to get to. I'm in Madrid, Spain, for the epic CodeMotion Madrid show. Then it's off to Guadalajara, Mexico for the Java day Mexico show; and then it's off to Taipei, Taiwan for the JCConf lead by the Taiwan JUG. Then, finally, it's off to the land of BBQ and Spring, SpringOne Platform! I can't wait to see you all there.

(Hopefully, by then, I'll have another chapter in my book [*Reactive Spring*](http://ReactiveSpring.io)!)

We've nary an instant to tarry: let's get to it!

-   [Java 13](https://openjdk.java.net/projects/jdk/13/) was released! Multiline strings, improved performance, a majorly rewrittern `Socket` API, and so much more. Get [the bits](https://jdk.java.net/13/) now! If you're on macOS, and have the venerable `sdkman` installed, you can also use `sdk install java 13.0.0-open`.
-   As we welcome Java 13, [A Bootiful Podcast: Nicolai Parlog on Java Modularity](https://spring.io/blog/2019/09/19/a-bootiful-podcast-nicolai-parlog-on-java-modularity)
-   [Spring Tools 4.4.0 released](https://spring.io/blog/2019/09/19/spring-tools-4-4-0-released)
-   [The Spanner-inspired Yugabyte DB just shipped their 2.0 release](https://twitter.com/SpringData/status/1176422666363973632) and there's even a Spring Data module and they're working on R2DBC support-nice!
-   I love this story of the path [of a contribution to the Spring Data MongoDB documentation](https://github.com/FrVaBe/blog/blob/master/2019-09-13_ObviousPullRequest/README.md)
-   Software Engineering Radio interviewed me for their podcast in [episode 381 on Spring Boot](https://www.se-radio.net/2019/09/episode-381-john-long-on-spring-boot/?tdest_id=1520171)
-   Heads-up from Spring Data lead Mark Paluch: [we're preparing a @R2DBC 0.8.0.RC1 release for late next week. Follow GitHub milestones for our release schedule](https://twitter.com/mp911de/status/1174938221442035712)
-   I love this post from Netifi on the [differences between gRPC and RSocket - Netifi - Medium](https://medium.com/netifi/differences-between-grpc-and-rsocket-e736c954e60)
-   It was a lot of fun being in the [Oracle CodeOne community keynote last week](https://twitter.com/afitnerd/status/1174364778803924993).
-   A nice article on using Reactor Blockhound to find [the code that's blocking threads](https://medium.com/@domenicosibilio/blockhound-detect-blocking-calls-in-reactive-code-before-its-too-late-6472f8ad50c1)
-   The ingenious Toshiaki Maki has added support for 8-bit ASCII art to Spring Boot. [So, you're upgrading, right?](https://twitter.com/making/status/1175677366120505344)
-   Did you know that, in addition to the recently added Dark Mode on the [Spring Initializr](http://start.Spring.io), there's also a way to "share" a configuration so that others can click the link and get the same configuration?
