# This Week in Spring - August 11, 2015

_Engineering | Josh Long |  August 12, 2015 | 0 Comments_

Welcome to another installation of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it!

-   Spring ninja Stephane Nicoll just announced [Spring Boot 1.3.0.M3](http://spring.io/blog/2015/08/11/spring-boot-1-3-0-m3-available-now), chock full of great features including HTTP session persistence, JMS and AMQP improvements, logging improvements, and auto-configurations for the H2 Database console and embedded MongoDB.
-   Spring Batch lead [Michael Minella just announced Spring Batch 3.0.5](http://spring.io/blog/2015/08/10/spring-batch-3-0-5-release-is-now-available)
-   Check out this cool look [at using Spring Boot to process Twitter feeds with Neo4j](http://www.javacodegeeks.com/2015/08/processing-twitter-feed-using-spring-boot.html)
-   check out this interesting Chinese-language [post on using `.yml` with Spring Boot](http://samchu.logdown.com/posts/290211-spring-boot-yaml-uses)
-   Disid corporation has [just announced a maintenance release for Spring Roo](http://spring.io/blog/2015/08/11/spring-roo-1-3-2-rc1-maintenance-release-now-available)
-   Just saw this post. Our very own [Cornelia Davis, director of platform engineering, was recognized as one of the top 10 women in cloud](http://blog.pivotal.io/pivotal-cloud-foundry/pivotal-people/meet-cornelia-davis-pivotals-role-model-for-women-in-cloud) by CloudNOW! As she has for years now, [she'll be at SpringOne2GX](https://2015.event.springone2gx.com/presenters/cornelia_davis.html)..
-   the All & Sundry blog has a cool post looking at how [to use Netflix's Archaius with Spring Cloud](http://www.java-allandsundry.com/2015/08/working-with-spring-cloud-and-netflix_8.html)
-   Speaking of configuration management, check out this post on [using configuration servers, including the Spring Cloud Config Server, on Cloud Foundry](http://blog.pivotal.io/pivotal-cloud-foundry/products/app-configuration-management-with-config-servers-in-cloud-foundry)
-   i dig this post on using [straight Java configuration to configure Spring Security roles](http://www.journaldev.com/8748/spring-4-mvc-security-managing-roles-example-with-in-memorywithout-web-xml-and-without-spring-xml-configuration) in a Spring MVC web application.
-   our very own Michael Cote asked me to record a podcast or two with him. [The first installment is now available](http://blog.pivotal.io/pivotal-conversations/features/injecting-cloud-native-thinking-into-java-with-spring-boot?utm_source=social&account_id&utm_medium=TWITTER&pivotal&utm_campaign=Products&20150808).
