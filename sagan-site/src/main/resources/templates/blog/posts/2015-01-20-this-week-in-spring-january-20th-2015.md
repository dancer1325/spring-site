# This Week in Spring - January 20th, 2015

_Engineering | Josh Long |  January 20, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week's roundup could've just as easily been titled, *This Week in Spring Videos and Webinars*, because we've got a *lot* of them!

1.  The good Dr. Dave Syer continues his series [introducing Spring Security and Angular.js integration, this time looking at breaking apart the resource server from the authorization server](https://spring.io/blog/2015/01/20/the-resource-server-angular-js-and-spring-security-part-iii). 2.  Our pal Nicoalas Frankel is back at it! Nicolas lives in both the [Vaadin](http://vaadin.com) and the Spring communities. So, naturally, when Petter Holmström and I started the [Vaadin4Spring project](https://github.com/peholmst/vaadin4spring) at the beginning of 2014, we weren't surprised when Nicolas started making great contributions. His latest contribution introduces an elegant [implementation of the MVP pattern for Vaadin and he's written a very nice introduction to it](http://blog.frankel.ch/improving-the-vaadin-4-spring-project-with-a-simpler-mvp) - check it out!
3.  I'll be speaking at the Oakland JUG on the 28th [on building microservices with Spring Cloud and Cloud Foundry](http://www.meetup.com/eastbayjug/events/219652953/).
4.  Tadaya Tsuyukubo just announced the [first 1.0.0 release of the Spring Social Slideshare project](https://github.com/ttddyy/spring-social-slideshare). I haven't had a chance to play with it yet, but it looks like it could scratch a few itches I've got! Nice work!
5.  I hope you'll join me next month for [a webinar introducing the Spring Session project](https://spring.io/blog/2015/01/20/webinar-introducing-spring-session).
6.  Speaking of Vaadin and Spring, I'll be co-presenting this week at the GWT.create conference. [Our talk](http://gwtcreate.com/#agenda-us_room1_event7) will look at controlling a Parrot AR Drone using a Vaadin-based HTML5 frontend and a Spring Boot-powered backend and drone client. Come join us, it's sure to be fun!
7.  InfoQ did an interview with Spring Boot co-founder [Phil Webb on all the new features in Spring Boot 1.2.1](http://www.infoq.com/news/2015/01/Spring-Boot-1.2.1-Released)
8.  Alexandre Lourenco put together a post demonstrating how easy it is to get a REST API up and running [with Spring Boot in this quick post on DZone](http://java.dzone.com/articles/spring-boot-creating)
9.  Be sure to check out Spring XD's [Sabby Anandan in this webinar on Spring XD.next on February 17th](https://spring.io/blog/2015/01/20/webinar-spring-xd-a-platform-for-data-at-scale-and-developer-productivity)
10.  The incomparable, amazing Andy Wilkinson will [be talking about how to document REST APIs](https://spring.io/blog/2015/01/20/webinar-documenting-restful-apis) in a February 3rd webinar.
11.  I put together a post that introduces service registries like Consul and Eureka [for more resilient service-to-service calls](https://spring.io/blog/2015/01/20/microservice-registration-and-discovery-with-spring-cloud-and-netflix-s-eureka) in a Spring Cloud-based system.
12.  Our pal Jakub Kubrynski has put together a *wonderful* [post on including the NewRelic Java agent in a Spring Boot uber-jar](http://www.dzone.com/links/r/including_java_agent_in_standalone_spring_boot_ap.html)
13.  In putting together the post on service discovery and registration with Spring Cloud, I stumbled upon [Jason Wilder's fantastic look at service registries](http://jasonwilder.com/blog/2014/02/04/service-discovery-in-the-cloud/) and really liked it, even though it doesn't have anything to do with Spring, per se.
14.  Check out this replay of Rossen Stoyanchev's SpringOne2GX 2014 talk [introducing Spring 4's support for web applications](https://spring.io/blog/2015/01/20/springone2gx-2014-replay-spring-4-web-applications)
15.  Check out this SpringOne2gx 2014 talk from Matthew Adams (SCI Spike) and Michael Nitschinger (Couchbase) on using [Spring Data Cassandra and Spring Data Couchbase in the enterprise](https://spring.io/blog/2015/01/20/springone2gx-2014-replay-cassandra-couchbase-and-spring-data-in-the-enterprise)
16.  Garrett Wampole's [SpringOne2GX 2014 talk on enterprise integration patterns as applied to real-time data processing](https://spring.io/blog/2015/01/20/springone2gx-2014-replay-applications-of-enterprise-integration-patterns-to-near-real-time-radar-data-processing) is worth a watch!
17.  Check out our pal and the Spanish Spring I/O conference's organizer Sergi Almar's [talk deep-diving into websockets](https://spring.io/blog/2015/01/20/springone2gx-2014-replay-deep-dive-into-spring-websockets)
18.  Speaking of super awesome Spaniards, [the Disid team just announced Spring Roo 1.3.1 RC2](https://spring.io/blog/2015/01/19/spring-roo-1-3-1-rc2-available-now)
19.  [Spring Tool Suite and Groovy Tool Suite lead Martin Lippert just announced the latest releases, 3.6.3 SR1](https://spring.io/blog/2015/01/19/spring-tool-suite-and-groovy-grails-tool-suite-3-6-3-sr1-released).
20.  My pal Luke Shannon did a [fantastic webinar introducing Spring Data Gemfire](https://spring.io/blog/2015/01/19/webinar-replay-building-scalable-data-applications-with-spring-and-gemfire), a world-class integration for a world-class data-grid.
21.  [Spring Batch and Spring Batch Admin lead Michael Minella has just announced the latest release of both projects](https://spring.io/blog/2015/01/16/spring-batch-and-spring-batch-admin-releases). The new Spring Batch revision includes support for JSR 352 custom infrastructures, polling for partitioned results, and more. The Spring Batch Admin revision integrates the latest and greatest in Spring Batch, including JSR 352 and Java configuration support. Check it out!
22.  Spring IO Platform lead Andy Wilkinson has announced [Spring IO Platform 1.1.1, which includes the latest cuts of Spring AMQP, Boot, and Integration](https://spring.io/blog/2015/01/15/spring-io-platform-1-1-1-released)
23.  Spring Integration lead Gary Russell's SpringOne2GX 2014 talk on Spring Integration's new Java configuration support is *not to be missed*!
24.  Check out Spring framework [lead Juergen Hoeller and Stephane Nicoll's talk on caching and messaging improvements in Spring 4.1](https://spring.io/blog/2015/01/14/springone2gx-2014-replay-caching-and-messaging-improvements-in-spring-framework-4-1)
25.  The sleepless Spring Security lead Rob Winch's [SpringOne2GX 2014 introduction talk on Spring Security is also available](https://spring.io/blog/2015/01/14/springone2gx-2014-replay-from-0-to-spring-security-4-0)
26.  Spring framework lead Juergen Hoeller's SpringOne2GX 2014 talk on Spring framework and Java 8 is not-to-be-missed primer for those looking at [the next-gen Spring 4 and 4.1](https://spring.io/blog/2015/01/14/springone2gx-2014-replay-spring-framework-on-java-8)
27.  Another nice post on using [`SimpleJdbcInsert` to handle JDBC data insertion](http://devilspace.org/2015/01/19/data-insertion-in-to-tables-by-using-simplejdbcinsert-class-of-spring-framework/)
28.  Alvin Henrick put together a nice post on using [Spring Boot and Docker together](http://alvinhenrick.com/2015/01/16/spring-boot-app-deployed-with-docker-and-data-only-container-pattern-explained/)
29.  I found [this interesting Q&A post on using Groovy with Spring Boot](http://hiteshtwo.org/asley/2015/01/20/spring-boot-with-groovy-and-external-libraries/)
30.  Alex Collins' post on using [Docker, Elastic Search, and Selenium to create a search-in-a-box](http://www.alexecollins.com/elastic-search-in-a-box-with-docker/) is *epic*! Check it out!
31.  This Japanese-language post on [using the new Spring Boot 1.2.1's Jersey support](http://backpaper0.github.io/2015/01/14/spring_boot_jersey.html) looked pretty good (via Google Translate!)
