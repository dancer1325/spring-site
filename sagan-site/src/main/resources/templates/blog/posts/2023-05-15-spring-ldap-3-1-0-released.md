# Spring LDAP 3.1.0 released

_Releases | Josh Cummings |  May 15, 2023 | 0 Comments_

On behalf of the team everyone who has contributed, I am pleased to announce that the Spring LDAP 3.1.0 is available now!

Please refer to the [releases page](https://github.com/spring-projects/spring-ldap/releases) for more detail on what is included in this and the preceding milestone releases.

Of note in this release is a new [LDAP client](https://docs.spring.io/spring-ldap/docs/3.1.0/reference/#spring-ldap-traditional-ldap-vs-ldaptemplate) that offers a fluent API similar to `WebClient` as well as first-class Stream support.
