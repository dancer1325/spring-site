# Spring Security 5.0.8 and 4.2.8 Released

_Releases | Josh Cummings |  September 11, 2018 | 0 Comments_

On behalf of the community, I am pleased to announce that the Spring Security 5.0.8 ([changelog](https://github.com/spring-projects/spring-security/milestone/120?closed=1)) and 4.2.8 ([changelog](https://github.com/spring-projects/spring-security/milestone/118?closed=1)) have been released. The releases primarily deliver bug fixes and dependency version updates along with some minor improvements. The releases will be found in the upcoming Spring Boot maintenance releases coming this week.
