# Spring Framework 6.0.15 and 6.1.2 available now

_Releases | Brian Clozel |  December 14, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.0.15` and `6.1.2` are available now.

Spring Framework `6.0.15` ships with [25 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.15), including [7 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.0.15+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.1.7, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.1.2` ships with [119 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.2), including [14 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.1.2+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.2.1, [to be released next week](https://calendar.spring.io/).
