# Spring IO Platform Athens-SR1

_Releases | Andy Wilkinson |  November 14, 2016 | 4 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Athens-SR1 is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.6.5.RELEASE
-   Spring Boot 1.4.2.RELEASE
-   Spring Data Hopper SR5
-   Spring Framework 4.3.4.RELEASE
-   Spring Integration 4.3.5.RELEASE
-   Spring Security OAuth 2.0.12.RELEASE

The versions of a number of third-party dependencies have also been updated.
