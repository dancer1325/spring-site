# This Week in Spring - February 23, 2016

_Engineering | Josh Long |  February 23, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! We're already staring down the last weeks of February!

-   Spring Data lead Oliver Gierke [has just announced Spring Data Gosling SR4](https://spring.io/blog/2016/02/23/spring-data-gosling-sr4-released) containing 45 bugfixes. Get it while it's hot!
-   Want to learn more about Spring Data Gosling? Check out this replay of Spring Data lead [Oliver Gierke's introduction too the new bits](https://spring.io/blog/2016/02/18/webinar-replay-what-s-new-in-spring-data-gosling-fowler)
-   the first milestone of Spring Integration 4.3, the last in the 4.x series, has arrived! Lots of small new features to tide you over before [Spring framework 5 and the reactive support there become the baseline for Spring integration 5](https://spring.io/blog/2016/02/17/spring-integration-4-3-m1-is-available)
-   Let's hear it for women in tech! Ludwine Probst & Ilayaperumal Gopinathan bring you this SpringOne2GX 2015 replay: [Apache Spark for Big Data Processing](http://spring.io/blog/2016/02/23/springone2gx-2015-replay-apache-spark-for-big-data-processing)
-   Our pal Sergi Almar, organizer of the [Spring IO conference](http://www.springio.net/) in beautiful Barcelona this May 19-20, 2016 did another great talk on how to go [From Zero to Hero with Spring WebSocket](http://spring.io/blog/2016/02/23/springone2gx-2015-replay-from-zero-to-hero-with-spring-websocket)
-   Our Securitay ninja, Rob Winch, starts with the basics in this SpringOne2GX 2015 replay: [Hands on Spring Security 4.1](http://spring.io/blog/2016/02/23/springone2gx-2015-replay-hands-on-spring-security-4-1)
-   Craig Walls' epic *Spring Boot in Action* is available and temporarily at a discount! [Get it while it's hot!](https://spring.io/blog/2016/02/18/discount-on-spring-boot-in-action-from-craig-walls)
-   Our pal Eberhard Wolff gave a good talk at JAX on building microservices [with Spring Boot and Spring Cloud: check it out](https://jaxenter.de/microservices-mit-spring-boot-und-spring-cloud-35223)!
-   I wanted to take a moment to warn you of the dangers of the internet [and posts like this](http://www.ekiras.com/2016/02/how-to-disable-application-banner-in-spring-boot.html) that seek only to *encourage* disabling the banners of Spring Boot-based applications. This is a *terrible* idea! How will you ever get Amazing banners like [this Nyan Cat banner](http://i.stack.imgur.com/SgPGS.png) and [the `ImageBanner` PR proposed here](https://github.com/spring-projects/spring-boot/pull/4647)? I wasn't originally going to include this article in *This Week in Spring* (as a matter of principle!) but was consoled upon [this epic response](https://twitter.com/yanncebron/status/702216038344220672) from [the *epic* IntellIJ IDEA developer Yann Cébron](http://twitter.com/yanncebron) demonstrating that the checkbox that previously disabled the Spring Boot banner was no longer enabled! :D
-   there's an interesting example of [configuring OpenUI5 based on Spring Boot](http://www.penninkhof.com/2016/02/openui5-boilerplate-based-on-olingo-jpa-and-spring-boot/)
-   Our pal Nicolas Frankel's written up a nice blog on managing [logs with Spring Boot](https://blog.frankel.ch/log-management-in-spring-boot)
-   Want to use [Apache CXF with Spring Boot (in 2016)](https://blog.codecentric.de/en/2016/02/spring-boot-apache-cxf/) to build a SOAP web service?
-   I did an interview [with InfoQ](http://twitter.com/infoq) on [Spring Boot, Spring Cloud and microservices in general](http://www.infoq.com/interviews/long-spring-boot-cloud)
-   Speaking of IntellIJ, the first preview of the new version 16 is out and contains updated support for Spring MVC, Spring framework, and Spring Boot as well as updated Kotlin support! [Check out the bits and file issues if you discover anything](http://blog.jetbrains.com/idea/2016/02/intellij-idea-16-public-preview/#more-13998)
