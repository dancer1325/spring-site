# Spring 3.0.4 is Now Available

_Releases | Adam Fitzgerald |  August 19, 2010 | 0 Comments_

![](/files/Logo_Spring_258x151.png)

[Arjen Poutsma](http://blog.springsource.com/author/arjenp/) has just [announced](http://twitter.com/poutsma) that Spring 3.0.4 is now available. This release addresses over 80 minor issues and updates Spring's support for some recent third-party releases including Hibernate Core 3.6, Hibernate Validator 4.1, EclipseLink 2.1, and EHCache 2.2.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.4.RELEASE) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.x/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.4.RELEASE/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

All the latest features for Spring as well as advanced technical sessions and roadmap discussion will be happening at [SpringOne 2GX 2010](http://www.springone2gx.com) in Chicago, October 19th-21st. Be sure to [register now](http://www.springone2gx.com/conference/chicago/2010/10/register) to lock in an early registration discount.

Don't forget that Spring users can ask questions in the [community forum](http://forum.springsource.org/forumdisplay.php?f=21) and identify issues in [JIRA](http://jira.springframework.org/browse/SPR) as well.
