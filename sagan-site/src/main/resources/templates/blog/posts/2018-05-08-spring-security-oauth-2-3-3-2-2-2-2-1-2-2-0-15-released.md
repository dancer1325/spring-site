# Spring Security OAuth 2.3.3, 2.2.2, 2.1.2, 2.0.15 Released

_Releases | Joe Grandja |  May 08, 2018 | 1 Comment_

I’m pleased to announce the releases of Spring Security OAuth 2.3.3, 2.2.2, 2.1.2 and 2.0.15. These maintenance releases primarily deliver bug fixes.

For a complete list of changes, please refer to:

-   [2.3.3 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/37?closed=1)
    
-   [2.2.2 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/32?closed=1)
    
-   [2.1.2 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/29?closed=1)
    
-   [2.0.15 changelog](https://github.com/spring-projects/spring-security-oauth/milestone/25?closed=1)
    

**2018-05-09 Update:** The releases address a vulnerability. Please see [this blog post](https://spring.io/blog/2018/05/09/spring-project-vulnerability-reports-published) published after the associated Spring Boot [1.5.13](https://spring.io/blog/2018/05/09/spring-boot-1-5-13) release.
