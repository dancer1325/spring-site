# Spring Framework 5.3.28 and 6.0.10 available now

_Releases | Brian Clozel |  June 15, 2023 | 2 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.28` and `6.0.10` are available now.

Spring Framework `5.3.28` ships with [17 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.28), including [3 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A5.3.28+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 2.7.13, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.0.10` ships with [64 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.10), including [7 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.0.10+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.0.8 and 3.1.1, [to be released next week](https://calendar.spring.io/).

The next Spring Framework releases are scheduled for July 13th, with the next 6.0.x maintenance release and the second 6.1.0 milestone.
