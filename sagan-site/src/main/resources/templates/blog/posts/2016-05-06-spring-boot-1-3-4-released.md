# Spring Boot 1.3.4 released

_Releases | Stéphane Nicoll |  May 06, 2016 | 0 Comments_

Spring Boot 1.3.4 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.4+is%3Aclosed) and 3rd party dependency updates.
