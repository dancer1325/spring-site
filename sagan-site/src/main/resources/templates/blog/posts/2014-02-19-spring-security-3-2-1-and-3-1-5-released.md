# Spring Security 3.2.1 and 3.1.5 Released

_Engineering | Rob Winch |  February 19, 2014 | 0 Comments_

Spring Security 3.2.1 ([changelog](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10040&version=14310)) and 3.1.5 ([changelog](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10040&version=14135)) have been released and are now available from Maven Central.

Spring Security 3.2.1 brings a number of bug fixes including

-   fixing ordering issues with Java Configuration
-   changes to support Spring LDAP 2.0.1.RELEASE
