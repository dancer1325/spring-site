# This Week in Spring - February 25th, 2014

_Engineering | Josh Long |  February 25, 2014 | 0 Comments_

Welcome back to another installment of *This Week in Spring*.

Yesterday I was in Atlanta, GA for the [amazing DevNexus conference](http://devnexus.com) talking to developers about the latest and greatest in the wide world of Spring, and about REST API development with Spring. Today I am in Montreal, Canada, for [the ConFoo conference](http://confoo.ca). I'll speak to developers this week about Spring's rich NoSQL and big-data support and building REST APIs with Spring.

As usual, we've got a lot to cover, so let's get into it!

1.  Spring Batch lead Michael Minella has announced that [Spring Batch 3.0 M3 is now available](https://spring.io/blog/2014/02/24/spring-batch-3-0-milestone-3-released). My favorite new feature? It implements all of the required pieces for JSR 352, the Batch JSR! Congratulations Michael and team, this is a *huge* milestone!
2.  Spring Mobile lead Roy Clarkson has [announced that Spring Mobile 1.1.1 is now available](http://spring.io/blog/2014/02/21/spring-mobile-1-1-1-released).
3.  Spring Security lead Rob Winch announced [that Spring Security 3.2.1, and 3.1.5, are now available](http://spring.io/blog/2014/02/19/spring-security-3-2-1-and-3-1-5-released)
4.  Rob *also* announced that [Spring LDAP 2.0.1 is now available](http://spring.io/blog/2014/02/19/spring-ldap-2-0-1-released).
5.  Christoph Strobl has [announced that Spring Data Redis 1.2.0](http://spring.io/blog/2014/02/19/spring-data-redis-1-2-0-rc1-released) is now available, also!
6.  Join Juergen Hoeller and the Spring Team to learn about [Java 8 and Spring Framework 4.0 on March 25](https://spring.io/blog/2014/02/11/webinar-spring-framework-4-0-on-java-8-march-25).
7.  Don't miss Mattias Arthursson on [March 18th, presenting Spring LDAP 2.0.0](http://spring.io/blog/2014/02/11/webinar-spring-ldap-2-0-0-on-march-18-2014).
8.  Tomcat fans should check out the webinar replay from Apache Committer Stuart Williams and Daniel Mikusa: [Introduction to Apache Tomcat 8](http://spring.io/blog/2014/02/18/webinar-replay-introduction-to-apache-tomcat-8).
9.  The replay of the [Spring Data Community Lightning Talks from SpringOne2GX 2013 are now up](http://spring.io/blog/2014/02/25/springone2gx-2013-replay-spring-data-community-lightning-talks)
10.  The replay of the SpringOne2GX 2013 talk, [\_SpringOne2GX 2013 Replay: Researching Cancer In the Cloud Using Spring, Neo4J, Mongo and Redis \_](http://spring.io/blog/2014/02/25/springone2gx-2013-replay-researching-cancer-in-the-cloud-using-spring-neo4j-mongo-and-redis)
11.  Spring Data lead Oliver Gierke has announced that [Spring Data Codd is now available and GA](http://spring.io/blog/2014/02/24/spring-data-release-train-codd-goes-ga)! Check out the latest and greatest bits!
12.  The latest cut of the [Axon CQRS framework, version 2.1, is now available](http://www.infoq.com/news/2014/02/axon-framework-2-1).
13.  Pivotal CEO Paul Maritz has penned a nice post on [the formation of a foundation around Cloud Foundry](http://blog.gopivotal.com/news-2/pivotal-moves-to-establish-open-governance-model-for-cloud-foundry)
14.  *El Reg* also has a nice post on the recent [formation of the Cloud Foundry foundation](http://www.theregister.co.uk/2014/02/24/cloud_foundry_foundation/) in a post titled, *Cloud Foundry laps up attention in EMC, IBM, HP, Rackspace, SAP, VMware love-in*.
15.  The Zoltan blog has a nice post on [using two-factor authentication with Spring Security](http://altfatterz.blogspot.com/2014/02/two-factor-authentication-with-spring.html)
16.  Biju Kunjummen on the *the Java Code Geek's blog* has put together a nice nice post on his first steps [with Spring Boot and Spring Integration](http://www.javacodegeeks.com/2014/02/spring-integration-standalone-application-with-spring-boot.html).
17.  Xavier Padró' has published his followup post to last week's post introducing [Spring and Thymeleaf](http://xpadro.blogspot.com/2014/02/thymeleaf-integration-with-spring-part-1.html). The new post looks at how [to build a form, and how to use Ajax with Thymeleaf](http://xpadro.blogspot.com/2014/02/thymeleaf-integration-with-spring-part-2.html).
18.  Speaking of Spring Boot, the *OpenCredo blog* has a nice post on their first steps with Spring Boot, as well! Check [it out](http://www.opencredo.com/2014/02/24/experiences-with-spring-boot/).
19.  JavaLobby has a nice post on using [custom `ViewResolver`s with Spring MVC](http://java.dzone.com/articles/chaining-url-view-resolvers).
