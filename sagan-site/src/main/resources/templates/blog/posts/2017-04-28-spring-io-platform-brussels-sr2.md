# Spring IO Platform Brussels-SR2

_Releases | Andy Wilkinson |  April 28, 2017 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR2 is now available from both [repo.spring.io](http://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   AspectJ 1.8.10
-   Spring AMQP 1.7.2
-   Spring Boot 1.5.3
-   Spring Data Ingalls SR 3
-   Spring Framework 4.3.8
-   Spring Integration 4.3.9
-   Spring Kafka 1.1.4

The versions of a number of third-party dependencies have also been updated.
