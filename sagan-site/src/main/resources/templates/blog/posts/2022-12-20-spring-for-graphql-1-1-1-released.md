# Spring for GraphQL 1.1.1 released

_Releases | Brian Clozel |  December 20, 2022 | 0 Comments_

I'm pleased to announce that Spring for GraphQL 1.1.1 is now available on Maven Central.

This first maintenance release in the 1.1.x generation includes [19 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.1.1). This version will be shipped with Spring Boot 3.0.1, [to be released later this week](https://calendar.spring.io/).

### [](#how-can-you-help)How can you help?

If you have general questions, please ask on [stackoverflow.com](https://stackoverflow.com) using the [`spring-graphql` tag](https://stackoverflow.com/tags/spring-graphql).
