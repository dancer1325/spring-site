# Spring Framework 5.3.13 available now

_Releases | Stéphane Nicoll |  November 11, 2021 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.13` is available now.

Spring Framework `5.3.13` includes [16 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.13).

Stay tuned for follow-up Spring Boot releases next week.
