# Spring Security SAML DSL 1.0.5.RELEASE

_Releases | Filip Hanik |  March 26, 2018 | 4 Comments_

The Spring Security team is proud to announce our latest 1.0.5.RELEASE of our SAML DSL project. This release works against Spring Security SAML 1.0.4.RELEASE.

This release adds a simple [Java configuration option](https://github.com/spring-projects/spring-security-saml-dsl/blob/929b94b349e894f17a7ff89375446b48f1c7f637/samples/spring-security-saml-dsl-sample/src/main/java/com/example/SecurityConfiguration.java#L21-L41) to your existing Spring Boot applications

This release requires an additional [Maven repository](http://repository.mulesoft.org/releases/) to download the latest version of the not-yet-commons-ssl dependency.

Available on [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Ca%3A%22spring-security-saml-dsl-core%22)

Feedback and contributions are always appreciated. Stay tuned.
