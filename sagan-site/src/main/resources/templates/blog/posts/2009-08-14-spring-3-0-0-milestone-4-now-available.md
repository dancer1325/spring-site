# Spring 3.0.0 Milestone 4 Now Available

_Releases | Adam Fitzgerald |  August 14, 2009 | 0 Comments_

This release got overshadowed by some of the recent news this week, but I want to tell the community that Spring 3.0.0.M4 is now available.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework) | [Reference Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html) | [Change Log](http://static.springsource.org/spring/docs/3.0.x/changelog.txt)

This release includes some documentation changes, updated OSGi manifest information, a number of other improvements, along with some small revisions to supported frameworks like Hibernate, Tiles and others. Thank you to all of the dedicated community members that have given [feedback](http://forum.springsource.org/forumdisplay.php?f=21) and identified [issues](http://jira.springframework.org/browse/SPR).
