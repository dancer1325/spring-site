# Spring Shell 1.0.0.RC1 Released

_Releases | Costin Leau |  September 20, 2012 | 0 Comments_

Dear Spring Community,

I am pleased to announce the first release candidate for Spring Shell project! Spring Shell is an interactive shell that can be easily extended with commands using a Spring based programming model, extracted from the Spring Roo project.

[Downloads](http://www.springsource.com/download/community?project=Spring%20Shell&version=1.0.0.RC1) | [JavaDocs](http://static.springsource.org/spring-shell/docs/1.0.0.RC1/api/) | [Reference Documentation](http://static.springsource.org/spring-shell/docs/1.0.0.RC1/reference/html) | [Changelog](http://static.springsource.org/spring-shell/docs/1.0.0.RC1/changelog.txt)

This release contains various fixes related to JLine/Jansi. See the [changelog](http://static.springsource.org/spring-shell/docs/1.0.0.RC1/changelog.txt) for detailed info.

We look forward to your feedback on the [forum](http://forum.springsource.org/forum.php) or in the [issue tracker](https://jira.springsource.org/browse/SHL).
