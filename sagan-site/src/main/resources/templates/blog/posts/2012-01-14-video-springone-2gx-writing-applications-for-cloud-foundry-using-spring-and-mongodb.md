# Video: SpringOne 2GX - Writing Applications for Cloud Foundry Using Spring and MongoDB

_News | Adam Fitzgerald |  January 14, 2012 | 0 Comments_

[![SpringOne 2GX Video: Writing Applications for Cloud Foundry Using Spring and MongoDB](http://www.springsource.org/files/SpringOne2GX2012CloudFoundrySpringMongoDB.png)](http://www.infoq.com/presentations/Writing-Applications-for-Cloud-Foundry-Using-Spring-and-MongoDB)

Here is another session from SpringOne 2GX 2011. This one gives instructions for [Writing Applications for Cloud Foundry Using Spring and MongoDB](http://www.infoq.com/presentations/Writing-Applications-for-Cloud-Foundry-Using-Spring-and-MongoDB). The Spring material is presented by data expert [Thomas Risberg](http://blog.springsource.org/author/trisberg/) and the Mongo material is presented by [Jared Rosoff](https://twitter.com/#!/forjared) from 10gen. The presentation provides a technical introduction to [Cloud Foundry](http://cloudfoundry.com/signup?utm_source=springsource_web&utm_medium=banner&utm_campaign=cloudfoundry-signup), [MongoDB](http://www.mongodb.org/) and [Spring Data for MongoDB](/spring-data/mongodb) (including Mongo Template and Mongo Repository support). This is a great way to dive into noSQL and cloud at the same time!

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
