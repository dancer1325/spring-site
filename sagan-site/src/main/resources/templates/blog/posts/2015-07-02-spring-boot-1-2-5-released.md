# Spring Boot 1.2.5 released

_Releases | Phil Webb |  July 02, 2015 | 13 Comments_

Spring Boot 1.2.5 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.5) and most importantly updates the managed Spring Framework dependency to [version 4.1.7](https://spring.io/blog/2015/06/30/spring-framework-4-1-7-3-2-14-released) (which includes an important [security fix](http://pivotal.io/security/cve-2015-3192)).

We recommend that all Spring Boot users upgrade.
