# Spring IO Platform Brussels-SR17

_Releases | Andy Wilkinson |  April 04, 2019 | 1 Comment_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Brussels-SR17 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.13
-   Spring Boot 1.5.20
-   Spring Cloud Connectors 1.2.8
-   Spring Data Ingalls SR20
-   Spring Framework 4.3.23
-   Spring Integration 4.3.20
-   Spring Retry 1.2.4
-   Spring Security 4.2.12
-   Spring Security JWT 1.0.10
-   Spring Security OAuth 2.0.17
-   Spring Web Services 2.4.5

The versions of a number of third-party dependencies have also been updated.
