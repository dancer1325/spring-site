# Spring Integration 3.0.1 Release is Now Available

_Engineering | Gary Russell |  February 04, 2014 | 0 Comments_

This release contains a small number of important fixes for the [recent 3.0.0 release](https://spring.io/blog/2013/12/16/spring-integration-3-0-release-is-now-available). 3.0 users are encouraged to upgrade to this release as soon as possible. Please see the [3.0.1 release notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=14309) and [project page](http://projects.spring.io/spring-integration/) for more information.
