# Gemini Web milestone 4

_Engineering | Glyn Normington |  September 08, 2010 | 0 Comments_

Gemini Web milestone 4 (1.1.0.M04-incubation) is available for [download](http://www.eclipse.org/gemini/web/download/).

Notable changes:

-   Upgrade to Tomcat 6.0.29 (plus SpringSource mods) to close some [security vulnerabilities](http://tomcat.apache.org/security-6.html#Fixed_in_Apache_Tomcat_6.0.28)
-   Restructure Tomcat bundles to enable clustering to work - see [bug 322983](https://bugs.eclipse.org/bugs/show_bug.cgi?id=322983)
-   Various test and build improvements
