# Spring Framework 4.0.2 & 3.2.8 Released

_Releases | Phil Webb |  February 18, 2014 | 4 Comments_

Spring Framework 4.0.2 and 3.2.8 have been released and are now available from [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aorg.springframework) and [repo.spring.io](https://repo.spring.io/release/org/springframework/).

We have brought forward both releases due to [SPR-11376](https://jira.springsource.org/browse/SPR-11376) which strengthens Spring OXM to protect against a [XXE vulnerability](https://www.owasp.org/index.php/XML_External_Entity_%28XXE%29_Processing).

For a complete list of changes please refer to [JIRA](https://jira.springsource.org/browse/SPR#selectedTab=com.atlassian.jira.plugin.system.project%3Achangelog-panel)
