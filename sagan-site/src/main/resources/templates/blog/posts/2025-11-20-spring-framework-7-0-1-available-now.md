# Spring Framework 7.0.1 Available Now

_Releases | Brian Clozel |  November 20, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `7.0.1` is available now.

Spring Framework `7.0.1` ships with [15 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v7.0.1). This version will be shipped today with Spring Boot `4.0.0`.
