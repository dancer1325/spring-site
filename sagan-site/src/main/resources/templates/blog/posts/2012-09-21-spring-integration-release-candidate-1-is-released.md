# Spring Integration Release Candidate 1 is Released!

_Releases | Gary Russell |  September 21, 2012 | 0 Comments_

**Spring Integration 2.2.0.RC1** - the release candidate for the 2.2 stream, including a number of important features and improvements, is now available. Barring major issues, we expect to make the final release shortly, so users are encouraged to try out the new features, some of which are described [here](http://static.springsource.org/spring-integration/docs/2.2.0.RC1/reference/htmlsingle/#whats-new). Over the next week or so, we intend to release a number of [blog posts](http://blog.springsource.com/) highlighting some of these features and others.

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=12602). The issues listed there are closed, in addition to those in milestones 1 through 4.
-   Artifacts are available in the [SpringSource repo](http://repo.springsource.org/simple/libs-milestone-local/org/springframework/integration/) and [the community download page](http://www.springsource.org/download/community?project=Spring%2520Integration).

More information is available on the project's [home page](http://www.springsource.org/spring-integration).
