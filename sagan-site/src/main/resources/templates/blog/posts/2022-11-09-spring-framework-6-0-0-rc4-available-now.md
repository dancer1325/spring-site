# Spring Framework 6.0.0-RC4 available now

_Releases | Stéphane Nicoll |  November 09, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.0.0-RC4` is available now.

Spring Framework `6.0.0-RC4` includes [12 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.0-RC4). Stay tuned for the announcement of Spring Boot `3.0.0-RC2` later this week!
