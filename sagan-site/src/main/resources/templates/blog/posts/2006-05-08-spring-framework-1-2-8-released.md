# Spring Framework 1.2.8 Released

_Releases | Juergen Hoeller |  May 08, 2006 | 0 Comments_

Dear Spring community,  
  
**We are pleased to announce that [Spring 1.2.8](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=73406) has been released. [Download](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=73406&release_id=415255) | [Docs](http://static.springframework.org/spring/docs/1.2.8/reference/index.html) | [Changelog](http://static.springframework.org/spring/docs/1.2.8/changelog.txt)  
**  
This is a maintenance release, fixing a number of issues found in previous 1.2.x releases. Most of these fixes have been backported from 2.0 M4, so have already been available there. Please see the [changelog](http://static.springframework.org/spring/docs/1.2.8/changelog.txt) for details.
