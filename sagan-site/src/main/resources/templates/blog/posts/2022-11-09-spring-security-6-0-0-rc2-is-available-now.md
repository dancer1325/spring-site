# Spring Security 6.0.0-RC2 is available now

_Releases | Marcus Hert Da Coregio |  November 09, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security `6.0.0-RC2` is available now.

Spring Security `6.0.0-RC2` includes [fixes and improvements to the documentation](https://github.com/spring-projects/spring-security/releases/tag/6.0.0-RC2). Stay tuned for the announcement of Spring Boot `3.0.0-RC2` later this week!
