# Spring Security 5.1.6 and 5.0.13 Released

_Releases | Josh Cummings |  August 05, 2019 | 0 Comments_

On behalf of the community I am pleased to announce the release of Spring Security 5.1.6 ([changelog](https://github.com/spring-projects/spring-security/milestone/144?closed=1)) and 5.0.13 ([changelog](https://github.com/spring-projects/spring-security/milestone/145?closed=1)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
