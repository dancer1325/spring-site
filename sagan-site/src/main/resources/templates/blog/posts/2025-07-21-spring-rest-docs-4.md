# Spring REST Docs 4.0.0-M1

_Releases | Andy Wilkinson |  July 21, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring REST Docs `4.0.0-M1` is now available from Maven Central.

This initial 4.0 milestone contains [13 enhancements, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-restdocs/releases/tag/v4.0.0-M1). It upgrades to Spring Framework 7 and Hibernate Validator 9. It also provides null safety through JSpecify annotations as part of the [previously announced](https://spring.io/blog/2025/03/10/null-safety-in-spring-apps-with-jspecify-and-null-away) effort across the whole Spring portfolio.
