# Spring ActionScript 1.0 is Now Available

_Releases | Adam Fitzgerald |  September 24, 2010 | 0 Comments_

Dear community,

We're pleased to announce that Spring ActionScript 1.0 is now available.

[Download](http://www.springactionscript.org/downloads/spring-actionscript-1.0-distribution-with-dependencies.zip) | [API Documentation](http://www.springactionscript.org/asdoc/index.html) | [HTML Docs](http://www.springactionscript.org/docs/reference/html/springactionscript.html) | [Changelog](http://www.springactionscript.org/changes-report.html)

Spring ActionScript is an offshoot of the Java Spring Framework written in ActionScript 3.0. The framework contains the following core features:

-   Inversion of Control container, configurable with [XML](http://www.springactionscript.org/docs/reference/html/container-documentation.html#the_container) or [MXML](http://www.springactionscript.org/docs/reference/html/container-documentation.html#composing_mxml_based_configuration_metadata)
-   [EventBus](http://www.springactionscript.org/docs/reference/html/the_eventbus.html) supporting loose communication between application components
-   Metadata/annotation processing used for autowiring, event handling, ...
-   [Operation](http://www.springactionscript.org/docs/reference/html/the_operation_api.html), [Command](http://www.springactionscript.org/docs/reference/html/the_operation_api.html#commands) and [Task](http://www.springactionscript.org/docs/reference/html/the_operation_api.html#tasks) API
-   Utilities for abstracting server communication
-   Support for Modules
-   Extensions for the Cairngorm and PureMVC frameworks

The Spring Actionscript team would like to take this opportunity to thank everyone who took the time to report bugs, give suggestions and help out in any other kind of way. Without the dedicated Spring community behind us, the framework would not be where it is today. Thank all of you!

As always, we're on the look out for new members to join the team. If you are willing to invest some of your spare time to write documentation, sample applications, fix bugs or develop new functionality then please don't hesitate to get in touch with us!

For the full list of changes please check the [changelog](http://www.springactionscript.org/changes-report.html#a1.0). If you have any suggestions for new functionality or improvements to the existing code base, then please use JIRA to submit a feature request.
