# Spring IDE 2.0.2 released

_Releases | Christian Dupuis |  December 15, 2007 | 0 Comments_

Dear Spring Community,

we are pleased to announce that Spring IDE 2.0.2 has been released today. 2.0.2 is basically a bug fix and enhancement release, but finally adds tooling support for missing Spring 2.5 features like <context:\* /> and <jms:\* /> namespaces and the component scan facility.  

![Spring IDE 2.0 Logo](http://springide.org/blog/wp-content/uploads/2007/06/springide_logo.png "Spring IDE 2.0 Logo")

[Download](http://springide.org/updatesite) | [Documentation](http://springide.org/project/wiki) | [Changelog](http://springide.org/jira/secure/IssueNavigator.jspa?reset=true&&pid=10000&fixfor=20001&fixfor=10025&sorter/field=issuekey&sorter/order=DESC)

The release is available from our [release update site](http://springide.org/updatesite). Spring IDE 2.0.2 is compatible with current milestone builds of upcoming Eclipse 3.4 (aka Eclipse Ganymede).
