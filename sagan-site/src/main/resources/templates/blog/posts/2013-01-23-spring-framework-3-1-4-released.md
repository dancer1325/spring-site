# Spring Framework 3.1.4 Released

_Releases | Chris Beams |  January 23, 2013 | 0 Comments_

The fourth maintenance release in the Spring 3.1.x line is now available via [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22), the [SpringSource repository](https://github.com/SpringSource/spring-framework/wiki/Downloading-Spring-artifacts#wiki-resolving-spring-artifacts), or for direct download from our [community download page](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.1.4.RELEASE). This release includes over 50 [bugfixes and improvements](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10000&version=13625) and is a recommended upgrade for all users.
