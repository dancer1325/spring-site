# Spring for GraphQL 1.3.4 released

_Releases | Brian Clozel |  February 18, 2025 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.3.4 is now available on Maven Central.

1.3.4 closes [14 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.3.4). This version will ship with Spring Boot 3.3.9 and 3.4.3, [to be released this week](https://calendar.spring.io/).

### [](#how-can-you-help)How can you help?

If you have general questions, please ask on [stackoverflow.com](https://stackoverflow.com) using the [`spring-graphql` tag](https://stackoverflow.com/tags/spring-graphql).
