# Spring Cloud Data Flow 2.7.1 Released

_Releases | Janne Valkealahti |  January 25, 2021 | 0 Comments_

Spring Cloud Data Flow team is pleased to announce the release of *2.7.1*.

*2.7.1* is a maintenance release fixing following issues:

-   UI bug fixes
-   *Cloudfoundry Java Client* has been updated to *4.13.0* fixing its memory leak
-   Using *Spring Boot 2.3.7* and *Cloud Hoxton.SR9*
-   *Spring Cloud Task* updated to *2.2.5*
-   Dataflow server contains fixes for docker, metrics and some endpoints
-   More detailed list can be found from [GitHub 2.7.1](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.7.1)

There is also releases for [GitHub 2.6.5](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.6.5) and [GitHub 2.5.4](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.5.4.RELEASE) with updated *Spring Cloud Task*.

These releases fixes [CVE-2020-5427](https://tanzu.vmware.com/security/cve-2020-5427).

We've switched [Pre-packaged Applications](https://dataflow.spring.io/docs/applications/) in our import links to point to *3.x* versions.
