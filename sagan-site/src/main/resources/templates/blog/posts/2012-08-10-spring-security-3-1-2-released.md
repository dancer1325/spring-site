# Spring Security 3.1.2 Released

_Releases | Rob Winch |  August 10, 2012 | 0 Comments_

We're pleased to an announce the GA release of Spring Security 3.1.2

The release is available from the [Spring Community Downloads area](http://www.springsource.com/download/community?project=Spring%20Security) and jars are available from Maven central repository. See the [project website](http://www.springsource.org/spring-security) for more information.

Special thanks to Chris Beams for all his help and guidance with updating the Spring Security build to use the [Spring release process](https://github.com/SpringSource/spring-framework/wiki/Gradle-build-and-release-FAQ#wiki-release_process).
