# This Week in Spring - March 24th, 2015

_Engineering | Josh Long |  March 24, 2015 | 1 Comment_

Welcome to another installment of *This Week in Spring*! This week, I'm in São Paulo, Brazil, where I presented a workshop on building cloud-native applications - specifically microservices - with [Spring Boot](http://start.spring.io), [Spring Cloud](http://cloud.spring.io) and [Cloud Foundry](http://cloudfoundry.org) (via [Pivotal Web Services](http://run.pivotal.io)).

-   Spring Data lead Oliver Gierke has just announced Spring Data Fowler has gone GA, [packed with lots of advanced Java 8 features, as well](http://spring.io/blog/2015/03/23/spring-data-release-train-fowler-goes-ga)!
-   The agenda for the [Spring IO show in Barcelona, Spain is now available](http://www.springio.net/agenda/)! Check it out!
-   Done with the microservices hype? Then let's get practical [on April 21st](https://spring.io/blog/2015/03/17/webinar-smarter-service-to-service-invocations-with-spring-cloud). I'll be talking about how Spring Cloud integrates service registration (e.g.: Eureka, Consul, or Zookeeper), declarative REST clients (with Netflix's Feign), reactive programming and the circuit breaker pattern with Hystrix to support easy, robust service-to-service invocations.
-   Marius Bogoevici and Mark Pollack are teaming up for a webinar on Tues April 28th about [Reactive data-pipelines with Spring XD and Kafka](https://spring.io/blog/2015/03/17/webinar-reactive-data-pipelines-with-spring-xd-and-kafka), a must for any big / fast data enthusiasts.
-   Colin Harrington's SpringOne2GX 2014 Replay is now available: [Groovy Browser Automation](https://spring.io/blog/2015/03/24/springone2gx-2014-replay-groovy-browser-automation)
-   Peter Ledbrook's SpringOne2GX 2014 Replay is now available: [Groovy for Java Developers](https://spring.io/blog/2015/03/24/springone2gx-2014-replay-groovy-for-java-developers)
-   Likely the coolest demo of SpringOne2GX 2014, brought to you by the dream team of Pivotal Ninjas Derek Beauregard, Phil Berman, Michael Minella, Darrel Sharpe

is now available: [IoT Realized - The Connected Car](https://spring.io/blog/2015/03/24/springone2gx-2014-replay-iot-realized-the-connected-car) showing you just how powerful Spring XD truly is for Big Data and IoT.

-   Frustrated with Java serialization performance? Are you a Spring Integration peformance fanatic? Or perhaps you're just trying to shave milliseconds and microseconds off your Java network calls? You won't want to miss John Davies from C24 as he shows you how to reclaim performance with Java serialzation in this mind blowing SpringOne2GX 2014 Replay: [Big Data in Memory](https://spring.io/blog/2015/03/24/springone2gx-2014-replay-big-data-in-memory)
-   The good Dr. Syer is back at it again with a followup to his epic blog series introducing single page applications [using Spring Security OAuth, Spring Cloud Zuul, and more](http://spring.io/blog/2015/03/23/multiple-ui-applications-and-a-gateway-single-page-application-with-spring-and-angular-js-part-vi).
-   I put together a look at [using Google Protocol Buffers to build efficient Spring-based REST services](http://spring.io/blog/2015/03/22/using-google-protocol-buffers-with-spring-mvc-based-rest-services)
-   Spring Tool Suite ninja Kris De Volder put together a great tutorial introducing some of the new features in [STS 3.6.4 that support building Spring Boot applications](http://spring.io/blog/2015/03/18/spring-boot-support-in-spring-tool-suite-3-6-4). Check it out!
-   I'll be doing another vJUG session, this time on [using Spring Cloud to build "Bootiful" Microservices](http://www.meetup.com/virtualJUG/events/221218834/) - I hope you'll join me there! Let's make this a super well attended event!
-   This post on [using Spring Social and Spring Security to secure GWT applications is kind of interesting](https://ngjweb.wordpress.com/2015/03/23/securing-gwt-apps-using-spring-oauth-spring-social/)
-   Hey, did you miss Adrian Cockroft's talk at O'Reilly's Software Architecture Summit? Check out this [blog summarizing the key takeaways](http://java.dzone.com/articles/key-takeaways-adrian-cockrofts)
-   I enjoyed the ideas raised in this blog post on [possible variants of microservices](http://java.dzone.com/articles/microservice-hybrids-and)
-   and, a *super* simple example of using [Spring Security with Spring Boot](https://gerrydevstory.com/2015/03/19/setting-up-spring-security-on-spring-boot-project/)
