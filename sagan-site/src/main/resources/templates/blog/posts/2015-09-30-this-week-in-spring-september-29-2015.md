# This Week in Spring - September 29, 2015

_Engineering | Josh Long |  September 30, 2015 | 1 Comment_

Welcome to another installation of *This Week in Spring*! To those of you reading from the US, let me be the first to wish you a [very happy national coffee day](https://twitter.com/search?q=national+coffee+day&ref_src=twsrc%5Egoogle%7Ctwcamp%5Eserp%7Ctwgr%5Esearch)! As usual, we've got a lot to cover so let's get to it!

-   have you been checking [out the Annotated Spring site](http://www.annotatedspring.com/)? It's *wonderful*! Packed with easily digsted screencasts on getting started with Spring (sort of like the Rails Casts of yore)
-   our pal Greg Turnquist is at it again, with part 3 of his [epic React.js and Spring Data REST tutorial](http://spring.io/blog/2015/09/29/react-js-and-spring-data-rest-part-3-conditional-operations)
-   not one to relax, Greg *also* posted an awesome new guide on building [CRUD applications with Vaadin and Spring Boot](http://spring.io/blog/2015/09/28/check-out-the-new-creating-crud-ui-with-vaadin-guide)
-   Spring Data ninja [Thomas Risberg has just announced Spring for Apache Hadoop 2.3 M3](http://spring.io/blog/2015/09/22/spring-for-apache-hadoop-2-3-milestone-3-released)
-   Spring Integration Java Config DSL lead Artem Bilan just announced the 1.1 GA release! [This is a *huge* release with lots of good stuff, so check it out](http://spring.io/blog/2015/09/22/java-dsl-for-spring-integration-1-1-ga-is-available)
-   I just noticed this book by Geoffroy Warin, [*Mastering Spring MVC 4*](http://www.amazon.fr/Mastering-Spring-MVC-Geoffroy-Warin-ebook/dp/B00ZCECXPY), is now available
-   our pal Nicolas Frankel has put together [a first look at understandig microservices](https://blog.frankel.ch/going-the-microservices-way-part-1) with Spring Boot
-   check out this 9 minute video clip [on monitoring microservices from Phil Webb and my training video, *Building Microservices with Spring Boot*](http://www.informit.com/articles/article.aspx?p=2437436)
-   Rob Hinds has put together an introductory post [to the Netflix OSS stack with Spring Cloud](https://dzone.com/articles/spring-boot-netflix-oss-an-adventure-into-microser-1)
-   [Want to get JSF running on Spring Boot](http://www.oakdalesoft.com/2015/09/spring-boot-with-jsfprimefaces/)?
-   Boyko Dimitrov put together a nice post on using [Spring Boot's OAuth2 support with Google as a provider](http://blog.dreamix.eu/webothers/configuring-google-as-oauth2-authorization-provider-in-spring-boot)
-   Patrick Grimard has a *wonderful* post on using Spring MVC's new server-side [JavaScript view renderer to render EJS script templates](http://patrickgrimard.com/2015/09/27/ejs-script-templates-with-spring-boot/)
-   Christophe Labouisse is back with a [post on managing Spring Boot logs with ELK and Docker](http://www.labouisse.com/how-to/2015/09/23/elk-docker-and-spring-boot/)
