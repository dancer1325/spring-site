# SpringSource dm Server 2.0.1 Released

_Engineering | Glyn Normington |  April 01, 2010 | 0 Comments_

SpringSource dm Server 2.0.1 is [released](http://forum.springsource.org/showthread.php?t=87176) today.

The release fixes a few minor bugs and one security issue. The Admin Console's version of dojo is upgraded from 1.3.2 to 1.3.3 to prevent an [open redirect attack](http://dojotoolkit.org/blog/post/dylan/2010/03/dojo-security-advisory/).

The release also includes the final Reference Implementation of the OSGi Web Container. The OSGi Web Container specification was recently released by the OSGi Alliance as part of the [OSGi 4.2 Enterprise Specification](http://www.osgi.org/Download/Release4V42).

No further releases of dm Server are planned as the project is being [donated](http://blog.springsource.com/2010/01/12/dm-server-project-moves-to-eclipse-org/) to the Eclipse Foundation as [Virgo](http://www.eclipse.org/virgo/). We aim to ship a baseline release of Virgo in due course which will be functionally equivalent to dm Server 2.0. SpringSource will offer commercial support for Virgo.
