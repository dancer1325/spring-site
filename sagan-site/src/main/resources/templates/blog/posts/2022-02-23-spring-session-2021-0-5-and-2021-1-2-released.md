# Spring Session 2021.0.5 and 2021.1.2 released

_Releases | Eleftheria Stein-Kousathana |  February 23, 2022 | 0 Comments_

On behalf of the team, I’m pleased to announce the release of Spring Session `2021.0.5` and `2021.1.2`. These releases deliver bug fixes and dependency upgrades. For your convenience, Spring Boot will pick up these artifacts with its upcoming releases.

The following modules were updated as part of `2021.0.5`:

-   Spring Session Core `2.5.5` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.5.5)
-   Spring Session Data Redis `2.5.5` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.5.5)
-   Spring Session JDBC `2.5.5` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.5.5)
-   Spring Session Hazelcast `2.5.5` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.5.5)

The following modules were updated as part of `2021.1.2`:

-   Spring Session Core `2.6.2` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.2)
-   Spring Session Data Redis `2.6.2` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.2)
-   Spring Session JDBC `2.6.2` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.2)
-   Spring Session Hazelcast `2.6.2` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.2)
-   Spring Session MongoDB `2.6.2` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.2)
