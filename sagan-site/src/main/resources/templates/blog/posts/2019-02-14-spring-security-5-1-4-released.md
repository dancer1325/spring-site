# Spring Security 5.1.4 Released

_Releases | Josh Cummings |  February 14, 2019 | 0 Comments_

On behalf of the community I am pleased to announce the release of Spring Security 5.1.4 ([changelog](https://github.com/spring-projects/spring-security/issues?q=is%3Aclosed+milestone%3A5.1.4+sort%3Aupdated-desc)). This release provides a round of bug fixes and users are encouraged to update to the latest patch release.
