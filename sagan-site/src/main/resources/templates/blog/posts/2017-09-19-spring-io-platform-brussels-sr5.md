# Spring IO Platform Brussels-SR5

_Releases | Andy Wilkinson |  September 19, 2017 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR5 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.4
-   Spring Boot 1.5.7
-   Spring Data Ingalls SR7
-   Spring Framework 4.3.11
-   Spring Integration 4.3.12
-   Spring Loaded 1.2.8
-   Spring Web Flow 2.4.6

The versions of a number of third-party dependencies have also been updated.
