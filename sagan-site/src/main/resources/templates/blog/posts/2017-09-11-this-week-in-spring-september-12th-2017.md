# This Week in Spring - September 12th, 2017

_Engineering | Josh Long |  September 11, 2017 | 1 Comment_

Hi Spring fans! This week I'm in Singapore [for the *amazing* YOW! Singapore event](http://www.yowconference.sg/program/) and talking to customers.

Also, I'll be doing [a meetup on testing](https://www.meetup.com/singasug/events/243106673/) - don't miss it!

-   It's almost time for Spring 5! Spring Framework lead Juergen Hoeller just [announced Spring Framework 5.0.RC4](https://spring.io/blog/2017/09/11/spring-framework-5-0-rc4-available-now), which is aligned with Spring Boot 2.0.M4, JUnit 5, and Reactor 3.1 RC1. This might be your last chance to kick the tires before the big release! There's *so* much new stuff in this release!
-   This is pretty interesting! There's a [Spring Cloud Config Server MongoDB project](https://github.com/spring-cloud-incubator/spring-cloud-config-server-mongodb) in the Spring Cloud incubator. Try it out!
-   Spring framework ninja Stéphane Nicoll just announced [Spring Framework 4.3.11](https://spring.io/blog/2017/09/11/spring-framework-4-3-11-available-now).
-   Spring Cloud and Spring Data ninja Mark Paluch just announced [Spring Cloud Vault 1.1.0.RC1](https://spring.io/blog/2017/09/11/spring-vault-1-1-0-rc1-available). This release includes pull-mode support for `AppRole` authentication and new HTML styling for the documentation.
-   Spring Data ninja Mark Paluch has just announced the availability of [Spring Data Ingalls Sr7 and Kay RC3](https://spring.io/blog/2017/09/11/spring-data-ingalls-sr7-and-kay-rc3-released). The new releases are both recommended updates.
-   Want to learn more about the reactive support in Spring Framework 5.0.GA? [Do not miss this forthcoming October 4 webinar](https://content.pivotal.io/webinars/oct-4-getting-reactive-with-spring-framework-5-0-s-ga-release-webinar?_lrsc=987dec05-e850-4874-9f15-2a2b5f8056ac)!
-   Our buddy Matt Raible did a nice roundup over [on the Okta blog of NoSQL options for Java developers](https://developer.okta.com/blog/2017/09/08/nosql-options-for-java-developers). The post is mostly a very interestingg discussion of technology uptake, but it does also highlight which modules are supported in Spring.
-   Hot on the trail of [JUnit 5.0.GA, JUnit and Spring Framework](https://twitter.com/sam_brannen/status/906992590356205568) contributor Sam Brannen just updated a sample application to use JUnit 5.0. Check it out!
-   At long last, [JUnit 5 is now here](https://t.co/eQsLWvsL6d?ssr=true)! There's a lot of good stuff in this release so be sure to check out [the release notes](http://junit.org/junit5/docs/5.0.0/user-guide/#release-notes-5.0.0)!
-   Mark Reinhold, chief architect of the Java platform, [has proposed moving Java to a predictable, and more frequent, release cycle](https://t.co/onLM101gmk?ssr=true). Read more about this proposal here.
-   I loved this installment of Microsoft Azure Fridays where [Sean McKenna introduces Cloud Foundry to Microsoft legend Scott Hanselman](https://t.co/JRPObzeRHO?ssr=true).
