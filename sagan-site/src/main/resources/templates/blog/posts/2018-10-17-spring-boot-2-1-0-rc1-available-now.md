# Spring Boot 2.1.0 RC1 available now

_Releases | Stéphane Nicoll |  October 17, 2018 | 0 Comments_

On behalf of the team, it is my pleasure to announce that Spring Boot 2.1.0.RC1 has been released and is now available from [our milestone repository](http://repo.spring.io/milestone/).

This release closes [over 100 issues and pull requests](https://github.com/spring-projects/spring-boot/milestone/107?closed=1), and is our first release candidate. At this point we're not anticipating that any major API changes or new features will be added before our final 2.1 GA release.

We've refined a number of items from previous milestone, and provide a number of notable new features including:

-   Auto-configuration for additional Micrometer registries: AppOptics, Humio and KairosDB
-   Support for Jersey metrics (Thread pool and HTTP request)
-   Integration with Hibernate's `BeanContainer` API

This release also picks up [97 issues](https://github.com/spring-projects/spring-boot/milestone/121?closed=1) that were closed for the [recent 2.0.6 release](https://spring.io/blog/2018/10/16/spring-boot-2-0-6).

For a complete list of changes, and upgrade instructions, see the [Spring Boot 2.1.0.RC1 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.1.0-RC1-Release-Notes) on the WIKI.

To get started with Spring Boot 2.1.0 RC1 and discover those new features, you can easily bootstrap a new project on [start.spring.io](https://start.spring.io).

Thanks to everyone that has contributed to Spring Boot 2.1 so far!
