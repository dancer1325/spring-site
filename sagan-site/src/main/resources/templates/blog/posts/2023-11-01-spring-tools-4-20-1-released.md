# Spring Tools 4.20.1 released

_Releases | Martin Lippert |  November 01, 2023 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.20.1 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

#### [](#important-highlights)important highlights

-   *(Spring Boot)* Completions for Spring Properties show up 10 times faster initially (as fast as on the consecutive invocations)
-   *(Spring Boot)* Various performance optimizations around symbols scanning and Java reconciling
-   *(Eclipse)* Early access builds available for upcoming Eclipse 2023-12 release

Detailed changes can be found in the release notes: [https://github.com/spring-projects/sts4/releases/tag/4.20.1.RELEASE](https://github.com/spring-projects/sts4/releases/tag/4.20.1.RELEASE)

#### [](#downloads)downloads

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Spring Tools 4.21.0 is scheduled to be released in mid December 2023.

Enjoy!
