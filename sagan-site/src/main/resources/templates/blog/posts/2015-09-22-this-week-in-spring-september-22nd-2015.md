# This Week in Spring - September 22nd, 2015

_Engineering | Josh Long |  September 22, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week, fresh after an *incredible* SpringOne2GX 2015, I'm.. a little burnt if I'm honest! :D BUT, Spring endures and I'm this week in Amsterdam [with our pal Martin Deinum](http://twitter.com/mdeinum) helping a large group of developers make the cloud native journey with Spring Boot, Spring Cloud and Cloud Foundry.

-   Spring Boot co-lead Phillip Webb [announced that Spring Boot 1.2.6 is now available!](http://spring.io/blog/2015/09/20/spring-boot-1-2-6-released). Also interesting: this release was done *live* on stage at SpringOne2GX 2015! (Thanks JFrog!)
-   our pal Matti Tahvonen has just posted guest blog on the Spring IO blog on how to use [the recently 1.0 Spring Boot & Vaadin integration](http://spring.io/blog/2015/09/17/single-page-web-apps-with-vaadin-spring-1-0)
-   the first Spring Cloud Brixton milestone is [now available complete with support for Zookeeper Discovery, Consul Discovery, Lattice Discovery, cluster leadership election and locks, and distributed tracing support](http://spring.io/blog/2015/09/16/first-milestone-of-spring-cloud-brixton-release-train-is-available)
-   Niklas Heidloff has put together a post on deploying [a Spring Boot application to Bluemix using Docker in 20 minutes](http://heidloff.net/nh/home.nsf/article.xsp?id=17.09.2015103506NHEC42.htm).
-   Heroku have updated their article [on deploying Spring Boot applications to their PaaS](https://devcenter.heroku.com/articles/deploying-spring-boot-apps-to-heroku)
-   The Mr. Haki blog has a great post [on reloading Java classes using Spring Loaded](http://mrhaki.blogspot.nl/2015/09/spring-sweets-reload-classes-spring.html)
-   The Mr. Haki blog has another post on reporting the applied [auto-configuration with the Spring Boot Actuator](http://mrhaki.blogspot.nl/2015/09/spring-sweets-report-applied-auto.html)
-   Spring Cloud Dataflow product manager Sabby Anandan put together a nice [post on the new Spring Cloud Dataflow](http://blog.pivotal.io/pivotal-cloud-foundry/products/introducing-spring-cloud-data-flow)
-   Biju Kunjummen has a nice post [on the rarely talked-about Spring Cloud Sidecar](https://dzone.com/articles/spring-cloud-sidecar) which lets other languages and technologies connect to Spring Cloud infrastructure
-   Jakub Narloch put together a nice [post on error handling Spring Cloud and Zuul](https://jmnarloch.wordpress.com/2015/09/16/spring-cloud-zuul-error-handling/)
