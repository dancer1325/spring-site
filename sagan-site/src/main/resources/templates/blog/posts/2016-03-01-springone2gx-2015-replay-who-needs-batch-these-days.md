# SpringOne2GX 2015 replay: Who needs batch these days?

_News | Pieter Humphrey |  March 01, 2016 | 0 Comments_

Recorded at SpringOne2GX 2015. Speakers: Gunnar Hillert, Michael Minella Data / Integration Track Slides: [http://www.slideshare.net/SpringCentral/who-needs-batch](http://www.slideshare.net/SpringCentral/who-needs-batch) Stream processing gets quite a bit of the press these days. Projects like Spark, Samza/Kafka, Google Cloud Dataflow, Apache Flink, and Amazon Kinesis bring the question of can we accomplish what we need without a batch component. This talk will take a hard look at if we still need batch these days, what that might look like in a modern development platform, and where Spring fits into it all.
