# New in STS 3.7.0: Spring Boot Yaml Editor

_Engineering | Kris De Volder |  May 11, 2015 | 7 Comments_

In STS 3.6.4 we introduced [".properties" editor support](https://drive.google.com/open?id=0B-E3P0EgKBcKdE9GY3RiMDZNb0E&authuser=1), for editing Spring Boot configuration properties.

Soon thereafter, people started asking about support for ".yml" files as well. We are proud to say that the ".yml" support is now reality.

Check out the [5 minute video demo](https://docs.google.com/file/d/0B-E3P0EgKBcKdE9GY3RiMDZNb0E/edit) to see this in action:

The Spring Boot Yaml editor is built on top of [YEdit](https://github.com/oyse/yedit). STS adds boot-specific content-assist, validation, hover-infos and hyperlink detectors.

It can be tried out today from STS's [nightly update site](https://github.com/spring-projects/spring-ide#installation-ci-builds) and is planned to be released as part of STS 3.7.0 near the end of June 2015.
