# This Week in Spring - February 28th, 2017

_Engineering | Josh Long |  February 28, 2017 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it! My friend Kenny Bastani and I will be doing a training for O'Reilly on the first and second of March introducing [all things Spring, Spring Boot, Spring Cloud and Cloud Foundry](https://www.safaribooksonline.com/live-training/courses/building-microservices-with-spring-boot-spring-cloud-and-cloud-foundry/0636920071181/) - join us!

The long awaited public beta of [Spring Cloud Services](https://content.pivotal.io/blog/now-available-spring-cloud-services-for-pivotal-cloud-foundry) is [now live on our public Cloud Foundry, PIvotal Web Services](https://content.pivotal.io/blog/building-spring-microservices-with-cloud-foundrys-new-container-networking-stack)!

-   Spring web ninja Rossen Stoyanchev just announced the the M5 milestone release of Spring 5, complete with [support for a WebClient testing mechanism](https://spring.io/blog/2017/02/23/spring-framework-5-0-m5-released)
-   Eric Bottard has just announced [Spring Cloud Data Flow for Cloud Foundry 1.1.1](https://spring.io/blog/2017/02/22/spring-cloud-data-flow-for-cloud-foundry-1-1-1-maintenance-release-available) which includes stability improvements, better error handling, network call optimizations and better load performance characteristics
-   Reactor project lead [Stephane Maldini just announced Reactor Aluminium SR1](https://spring.io/blog/2017/02/22/reactor-aluminium-sr1-has-landed)
-   Pivotal's own Chris Sterling explains Cloud Foundry's new [container-networking stack and how it applies to Spring Cloud-based applications](https://content.pivotal.io/blog/building-spring-microservices-with-cloud-foundrys-new-container-networking-stack).
-   Last week, continuing the Spring Tips series, I looked at [how to use Spring Boot with Apache MyBatis](https://spring.io/blog/2017/02/22/spring-tips-apache-mybatis), a technology that is commonly used in high performance organizations like Chinese's mega-merchant Alibaba
-   The Spring Boot team's Madhura Bhave nd Phillip Webb did [a great talk at DevNexus demystifying the inner workings of Spring Boot](https://speakerdeck.com/philwebb/breaking-the-magicians-code-diving-deeper-into-spring-boot-internals).
-   I loved Adib Saikali's [look at consumer driven contracts with Spring Cloud Contract at the amazing Toronto Spring User Group](https://www.youtube.com/watch?v=iyNzYOcuU4I)
-   This isn't specific to Java or any particular technology, [but it's an interesting post on how to unit test database indexes](http://databaseblog.myname.nl/2014/02/unittesting-your-indexes.html). Be sure to read the comments, too.
-   I was pointed to this post on Java 8's `CompletableFuture` the other day. [It's an oldie-but-a-goodie](https://www.slideshare.net/jpaumard/asynchronous-api-in-java8-how-to-use-completablefuture).
-   Are you using Neo4j and want to add some niceties to the Neo4j browser? Check out this project, [Spoon, by Neo4j's very own Michael Hunger](https://github.com/jexp/spoon-neo4j/)
-   I loved this video introducing [some of the highlights of the Spring Initializr](https://twitter.com/snicoll/status/835405736117170177) in Apache NetBeans
-   [Chris Richardson and Kenny Bastani gave a great talk on eventual consistency for distributed systems](https://vimeo.com/205223073) a few weeks ago, and it's now available online
-   Spring Data ninja Mark Paluch gave an *awesome* talk introducing [secret management with Hashicorp's Vault, and the talk is now available online](https://speakerdeck.com/mp911de/securely-managing-secrets-with-vault)
