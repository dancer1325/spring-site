# This Week in Spring - January 14th, 2014

_Engineering | Josh Long |  January 15, 2014 | 0 Comments_

Welcome back to another installment of *This Week in Spring*!

As usual, we've got a lot to cover, so let's dive right into it!

By the way, due to *overwhelming demand*, we're going to repeat the webinar introducing Spring 4 with Juergen Hoeller on January 23rd. Watch this space for when we open up registration. The expected times are:  

Thursday, January 23, 2014 - 3:00pm GMT Time (London GMT)

-   closed as of Jan 20th  
    

Thursday, January 23, 2014 - 10:00am PST (San Francisco, GMT-08:00)

-   [registration open](http://play.gopivotal.com/OSS_Webinar_MORESpringFramework4.0.17AM_Register.html)
    

1.  The replay of [Ben Hale](http://spring.io/team/bhale)'s talk on [RESTful API evolution from SpringOne2GX 2013 is now available online](http://spring.io/blog/2014/01/14/springone2gx-2013-replay-rest-ful-api-evolution)
2.  [Spring and Groovy/Grails Tool Suite lead Martin Lippert](http://spring.io/team/mlippert) just refreshed the most popular article ever written on JavaLobby, [*Spring IDE and the Spring Tool Suite - Using Spring in Eclipse*](http://eclipse.dzone.com/news/springide-using-spring-eclipse). Check it out!
3.  [Spring XD lead Dr. Mark Pollack](http://spring.io/team/mpollack) has just announced [that Spring XD 1.0.0.M5 is now available](http://spring.io/blog/2014/01/10/spring-xd-1-0-0-m5-released)
4.  Oliver Gierke's talk from SpringOne2GX ["Spring RESTBucks: a Hypermedia Driven REST webservice"](http://spring.io/blog/2014/01/08/springone2gx-2013-replay-spring-restbucks-a-hypermedia-driven-rest-webservice) is now available online.
5.  Following the *crazy* success of the Spring 4 webinar on the 9th, Spring project lead [Juergen Hoeller](http://spring.io/team/jhoeller) has just written a blog detailing the [next steps for the framework, including Spring 4.1 and Spring 3.2.7](http://spring.io/blog/2014/01/14/the-spring-framework-4-1-plan-and-4-0-1-3-2-7-releases-coming-up-next-week).
6.  Have you tried the crazy [cool shell built into Spring Boot](http://www.joshlong.com/jl/blogPost/spring_boot_m6s_new_security_featur.html)? It's powered by [CRaSH](http://www.crashub.org/), which you can learn [more about in this video](http://www.java-tv.com/2014/01/09/crash-a-shell-for-the-java-platform/)
7.  The replay from [Gunnar Hillert](http://spring.io/team/ghillert) and [Michael Minella](http://spring.io/team/mminella)'s talk from SpringOne2GX 2013, [*Integrating Spring Batch and Spring Integration*](http://spring.io/blog/2014/01/14/springone2gx-2013-replay-integrating-spring-batch-and-spring-integration), is now available online.
8.  Matt Stine webinar, [*Spring with Immutability*](http://spring.io/blog/2014/01/14/webinar-replay-spring-with-immutability), is now available online.
9.  [Rob Winch](http://spring.io/team/rwinch) just announced that [Spring LDAP 2.0.0 is now available](http://spring.io/blog/2014/01/13/spring-ldap-2-0-0-release-released) online!
10.  Our pal Eugen Paraschiv is back at it again, this time with a post [on Spring's `HttpMessageConverter`s](http://www.baeldung.com/spring-httpmessageconverter-rest).
11.  Jakub Kubrynski has a nice post on [integration testing Spring Integration and Spring 4](http://java.dzone.com/articles/spring-integration-testing)
12.  Gary Russell has just announced that [Spring AMQP 1.3.0.M1 and 1.2.1 RELEASE are now available](https://spring.io/blog/2014/01/13/spring-amqp-1-3-0-m1-and-1-2-1-release-available). The new release is *very* meaty, so if you're using it, I strongly suggest upgrading!
13.  Sproogle 0.3.0, which is an integration [with Spring and parts of the Google stack (that don't so far as I can tell include the OAuth pieces that Spring Social Google takes care of), is now available](http://unnaturalcode.blogspot.com/2014/01/sproogle-003-released.html).
14.  On the GoPivotal blog, Steve Greenberg has a nice post on how to [add a *service* to your Cloud Foundry architecture using Spring](http://blog.gopivotal.com/features/how-to-add-your-service-to-cloud-foundry-architecture-code-boilerplate-and-mongodb-example). Nice! I've been waiting for something like this for a *long* time!
15.  Our pals on the Vaadin team are doing a webinar on [Spring and Vaadin integration best-practices on January 30th, 2014](https://docs.google.com/forms/d/1Kh5WeO96WcQYpkuurJgIcSoD6FnNB5z32kg-MSjJSOk/viewform?edit_requested=true). Check it out!
16.  A little late, but RestHub, which integrates a full Spring-powered REST and web application stack, has just released [RESTHub 2.1.4](https://github.com/resthub/resthub-spring-stack/blob/master/CHANGELOG.rst#214-version-12-20-2013). Check it out!
17.  Rest Assured, which provides an alternative testing API for REST services, has [just released a new version that supports Spring MVC](http://code.google.com/p/rest-assured/wiki/ReleaseNotes22).
18.  [New Relic has just announced dedicated Grails support](http://blog.newrelic.com/2014/01/08/new-relic-supports-grails/).
19.  Zan Thrash did a nice talk on InfoQ about [using Node.js ecosystem tools for client-side development along with Grails](http://www.infoq.com/presentations/grails-nodejs). The talk has very little to do with Grails, actually, and could as easily be applied to Java and Spring MVC development. These tools are front-and-center for many different types of web application developers, and this talk is a good primer.
