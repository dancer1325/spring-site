# Spring Shell 1.0.0 Released

_Releases | Costin Leau |  October 03, 2012 | 0 Comments_

Dear Spring Community,

I am pleased to announce the first GA release for Spring Shell project! Spring Shell is an interactive shell that can be easily extended with commands using a Spring based programming model, extracted from the Spring Roo project.

[Downloads](http://www.springsource.com/download/community?project=Spring%20Shell&version=1.0.0.RELEASE) | [JavaDocs](http://static.springsource.org/spring-shell/docs/1.0.0.RELEASE/api/) | [Reference Documentation](http://static.springsource.org/spring-shell/docs/1.0.0.RELEASE/reference/html) | [Changelog](http://static.springsource.org/spring-shell/docs/1.0.0.RELEASE/changelog.txt)

We look forward to your feedback on the forum or in the issue tracker.
