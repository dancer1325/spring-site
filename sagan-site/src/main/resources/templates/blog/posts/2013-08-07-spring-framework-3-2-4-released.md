# Spring Framework 3.2.4 Released

_Releases | Phil Webb |  August 07, 2013 | 0 Comments_

The fourth maintenance release in the Spring 3.2.x line is now available via [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22), the [SpringSource repository](https://github.com/SpringSource/spring-framework/wiki/Downloading-Spring-artifacts#wiki-resolving-spring-artifacts), or for direct download from our [community download page](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.2.4.RELEASE). This release includes numerous [bugfixes and improvements](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10000&version=14143) including an [important security fix](https://jira.springsource.org/browse/SPR-10806) for Spring OXM. We recommend that all all users upgrade.
