# Spring for Apache Kafka, Spring for RabbitMQ 3.1 Releases Available

_Releases | Gary Russell |  November 21, 2023 | 0 Comments_

Hot on the heels of Spring Framework 6.1, I am pleased to announce the 3.1 releases of Spring for Apache Kafka and Spring AMQP (Spring for RabbitMQ).

See the release notes for the 3.1.0, 3.1.0-RC1 and 3.1.0-M1 releases of each project - [Spring for Apache Kafka](https://github.com/spring-projects/spring-kafka/releases) and [Spring AMQP](https://github.com/spring-projects/spring-amqp/releases) for detailed content.

Also see the What's New documentation sections for highlights: [Spring for Apache Kafka](https://docs.spring.io/spring-kafka/reference/whats-new.html) [Spring AMQP](https://docs.spring.io/spring-amqp/docs/current/reference/html/#whats-new).
