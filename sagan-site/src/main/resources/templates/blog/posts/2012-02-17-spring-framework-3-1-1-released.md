# Spring Framework 3.1.1 Released

_Releases | Chris Beams |  February 17, 2012 | 6 Comments_

The first maintenance release in the Spring 3.1.x line is now available via Maven Central, the [SpringSource repository](http://repo.springsource.org/release), or for direct download from our [community download page](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.1.1.RELEASE). This release includes many [important bugfixes and minor improvements](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10000&version=12706) and is a recommended upgrade.

Cheers!
