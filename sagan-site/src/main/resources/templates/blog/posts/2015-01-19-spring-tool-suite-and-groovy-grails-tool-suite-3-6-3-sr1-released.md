# Spring Tool Suite and Groovy/Grails Tool Suite 3.6.3.SR1 released

_Releases | Martin Lippert |  January 19, 2015 | 21 Comments_

Dear Spring Community,

We have released a minor service release for STS/GGTS 3.6.3 on Eclipse 4.4.1 to address a security vulnerability that was found in the JGit client. More details on this can be found here: [JGit client (CVE-2014-9390)](https://bugs.eclipse.org/bugs/show_bug.cgi?id=456947)

To download the updated distributions, please go visit:

-   Spring Tool Suite: [https://spring.io/tools/sts/all](https://spring.io/tools/sts/all)
-   Groovy/Grails Tool Suite: [https://spring.io/tools/ggts/all](https://spring.io/tools/ggts/all)

Detailed new and noteworthy notes can be found here: [STS/GGTS 3.6.3.SR1 New & Noteworthy](http://docs.spring.io/sts/nan/v363sr1/NewAndNoteworthy.html).

STS/GGTS 3.6.4 is scheduled to ship in March 2015, shortly after the Eclipse Luna SR2 release.

Enjoy!
