# New Milestone Update for SpringSource Tool Suite: 2.3.2.M2

_Releases | Adam Fitzgerald |  July 16, 2010 | 0 Comments_

SpringSource has just released a new milestone version of [SpringSource Tool Suite: 2.3.2.M2](http://www.springsource.com/products/springsource-google-download). STS provides the best Eclipse-powered development environment for building Spring and Grails powered enterprise applications. Here are some highlights from the release:

-   integration with [GWT](http://code.google.com/webtoolkit/) and [Google App Engine](http://code.google.com/appengine/)
-   visual editor for [Spring Integration](/spring-integration)
-   support for [Spring Roo](/roo) 1.1.0.M2
-   non-locking class loader infrastructure
-   new [Grails](http://www.grails.org) Project Explorer, Perspective and Plugin Manager
-   new Subclipse and Subversive extension installs

This is also the first time we are providing bundles based on Eclipse 3.5 *and* 3.6. If you aren't quite ready to try the cutting edge [milestone release](http://www.springsource.com/products/springsource-google-download), then you can always download the [2.3.2.RELEASE](http://www.springsource.com/products/springsource-tool-suite-download) version.

You can also get community support in the [Spring Development Tools](http://forum.springsource.org/forumdisplay.php?f=32) support forums. If you want to provide direct technical feedback, open a [JIRA](https://issuetracker.springsource.com/browse/STS) issue against the SpringSource Tool Suite project.
