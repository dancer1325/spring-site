# Spring Session Dragonfruit-M1, Corn-SR1 and Bean-SR9 Released

_Releases | Eleftheria Stein-Kousathana |  February 04, 2020 | 0 Comments_

On behalf of the community I’m pleased to announce the releases of Spring Session `Dragonfruit-M1`, `Corn-SR1` and `Bean-SR9`.

## [](#spring-session-dragonfruit-m1)[](#spring-session-dragonfruit-m1)Spring Session `Dragonfruit-M1`

The `Dragonfruit-M1` release is based on:

-   Spring Session core modules `2.3.0.M1`
    
-   Spring Session Data Geode `2.3.0.M1`
    
-   Spring Session Data MongoDB `2.3.0.M1`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/71?closed=1).

## [](#spring-session-corn-sr1)[](#spring-session-corn-sr1)Spring Session `Corn-SR1`

The `Corn-SR1` release is based on:

-   Spring Session core modules `2.2.1.RELEASE`
    
-   Spring Session Data Geode `2.2.2.RELEASE`
    
-   Spring Session Data MongoDB `2.2.2.RELEASE`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/70?closed=1).

## [](#spring-session-bean-sr9)[](#spring-session-bean-sr9)Spring Session `Bean-SR9`

The `Bean-SR9` release is based on:

-   Spring Session core modules `2.1.10.RELEASE`
    
-   Spring Session Data Geode `2.1.8.RELEASE`
    
-   Spring Session Data MongoDB `2.1.6.RELEASE`
    

Additional details of these releases can be found in [the changelog](https://github.com/spring-projects/spring-session/milestone/69?closed=1).
