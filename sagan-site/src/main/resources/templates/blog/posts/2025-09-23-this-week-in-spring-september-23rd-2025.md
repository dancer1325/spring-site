# This Week in Spring - September 23rd, 2025

_Engineering | Josh Long |  September 23, 2025 | 0 Comments_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*! I'm preparing my talks for several amazing shows including: *Commit Your Code* conference in Plano, Texas (starting tomorrow); *Dev2Next* in Colorado; Devoxx Belgium in Antwerp, Belgium; and CloudFoundry Days in Germany. So much good stuff comin' up! And so much good stuff in the community this week, as well, so let's dive right into it!

-   I love this! One of my favorite new features in Spring Framework 7 is [the new HTTP Service Clients](https://spring.io/blog/2025/09/23/http-service-client-enhancements)
-   there's so much good stuff coming in [Spring AI 1.0.2 and improved MCP support](https://spring.io/blog/2025/09/19/spring-ai-1-1-0-M2-mcp-focused)
-   [Spring Modulith 2.0 M3 released](https://spring.io/blog/2025/09/19/spring-modulith-2-0-m3-released). This new release includes updated event publication repository implementations for JPA, Jackson 3 support, and so much more.
-   in last week's installment of *A Bootiful Podcast*, I talked to the amazing [Spring Cloud lead Spencer Gibb](https://spring.io/blog/2025/09/18/a-bootiful-podcast-spencer-gibb)
-   [Spring Boot 3.4.10 is available now](https://spring.io/blog/2025/09/18/spring-boot-3-4-10-available-now)
-   [Spring Boot 3.5.6 available now](https://spring.io/blog/2025/09/18/spring-boot-3-5-6-available-now)

-   this is a nice introduction to streaming [multipart data in a Spring web application](https://www.baeldung.com/spring-streaming-multipart-data)
