# Spring Boot for Apache Geode & Pivotal GemFire 1.1.6.RELEASE & 1.2.5.RELEASE available!

_Releases | John Blum |  March 09, 2020 | 0 Comments_

On behalf of the Spring, Apache Geode and Pivotal GemFire communities, it is my pleasure to announce the release of *Spring Boot for Apache Geode & Pivotal GemFire* (SBDG) `1.1.6.RELEASE` and `1.2.5.RELEASE`.

SBDG `1.1.6.RELEASE` builds on Spring Boot 2.1.13.RELEASE, which is based on Spring Framework 5.1.14.RELEASE, Spring Data Lovelace-SR16, Spring Session Bean-SR9 with SSDG 2.1.9.RELEASE, Apache Geode 1.6.0 and Pivotal GemFire 9.5.4. [Bits](https://search.maven.org/artifact/org.springframework.geode/spring-geode-starter/1.1.6.RELEASE/jar) are available in *Maven Central*. See [changelog](https://github.com/spring-projects/spring-boot-data-geode/blob/1.1.6.RELEASE/spring-geode/src/main/resources/changelog.txt#L7-L20) for more details.

SBDG `1.2.5.RELEASE` builds on Spring Boot 2.2.5.RELEASE, which is based on Spring Framework 5.2.4.RELEASE, Spring Data Moore-SR5, Spring Session Corn-SR1 with SSDG 2.2.3.RELEASE, Apache Geode 1.9.2 and Pivotal GemFire 9.8.6. [Bits](https://search.maven.org/artifact/org.springframework.geode/spring-geode-starter/1.2.5.RELEASE/jar) are available in *Maven Central*. See [changelog](https://github.com/spring-projects/spring-boot-data-geode/blob/1.2.5.RELEASE/spring-geode/src/main/resources/changelog.txt#L7-L18) for more details.

Additionally, you can start building Spring for Apache Geode applications using Spring Boot, and specifically, SBDG `1.2.5.RELEASE`, from [start.spring.io](https://start.spring.io/#!platformVersion=2.2.5.RELEASE&dependencies=geode).
