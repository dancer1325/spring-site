# Spring Shell 1.1.0.M1 Released

_Releases | Mark Pollack |  July 26, 2013 | 0 Comments_

Dear Spring Community,

I am pleased to announce the first milestone release Spring Shell 1.1. Spring Shell is an interactive shell that can be easily extended with commands using a Spring based programming model. This release adds support for [testing of commands](http://static.springsource.org/spring-shell/docs/1.1.0.M1/reference/html/dev-shell.html#d4e249) as well as several bug fixes and general improvements. Many thanks to to those who [submitted pull-requests](https://github.com/springsource/spring-shell/pulls)

We look forward to your feedback on the forum or in the issue tracker.
