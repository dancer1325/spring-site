# This Week in Spring - July 31st, 2018

_Engineering | Josh Long |  July 31, 2018 | 1 Comment_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! I just finished an online training today, *Bootiful Kotlin*, for O'Reilly today, and I'm doing another one on the 2nd of August (this Thursday), [*Reactive Spring*](https://www.safaribooksonline.com/live-training/courses/reactive-spring-boot/0636920198048/), because the demand for the first one was so high. Don't miss it!

-   Spring Boot legend Andy Wilkinson has just [announced Spring Boot 2.1.M1](https://spring.io/blog/2018/07/30/spring-boot-2-1-m1). This release includes servlet 4 support with Undertow 2.0 and Tomcat 9.0, a caches Actuator endpoint, support for querying an individual component’s health, support for profile expressions in multi-profile YAML documents and `<springProfile>` Logback configuration.

-   This news is encouraging as it gets: the [team at Oracle working on the reactive JDBC-like API, called ADBA, have decided to incorporate the reactive types in Java 9+, `Publisher.Flow`](http://mail.openjdk.java.net/pipermail/jdbc-spec-discuss/2018-July/000378.html). They have hitherto resisted incorporating the `Flow.Publisher` types in the design of the reactive API. This was a pity since the `Publisher.Flow` types mirror, directly, the types in the Reactive Streams specification and are purpose built to support this use case.
