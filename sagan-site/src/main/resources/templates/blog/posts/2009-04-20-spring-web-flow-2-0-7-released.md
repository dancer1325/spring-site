# Spring Web Flow 2.0.7 Released

_Releases | Keith Donald |  April 20, 2009 | 0 Comments_

Dear Spring Community,

  

We are pleased to announce that Spring Web Flow 2.0.7 has been released. [Download](/download#webflow) | [Maven](http://static.springframework.org/spring-webflow/docs/2.0.x/reference/html/ch01s05.html) | [Documentation](/documentation#webflow) | [Changelog](http://static.springframework.org/spring-webflow/docs/2.0.x/changelog.txt)

This is a bugfix and minor improvement release addressing all known issues reported against previous 2.0.x releases; see the Changelog for all the details. **We recommend upgrading to 2.0.7 from previous 2.0.x versions.** Enjoy! Keith Donald Web Flow Project Lead
