# Spring for Apache Hadoop 1.0.0 M1 Released

_Releases | Costin Leau |  February 29, 2012 | 0 Comments_

Dear Spring Community, I am pleased to announce the release of Spring for Apache Hadoop 1.0 M1:

[Downloads](http://www.springsource.com/download/community?project=Spring Data Hadoop&version=1.0.0.M1) | [JavaDocs](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.M1/api/) | [Reference Documentation](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.M1/reference/html/) | [Changelog](http://static.springframework.org/spring-data/hadoop/docs/1.0.0.M1/changelog.txt)

The Spring for Apache Hadoop features include support for:

-   Apache Hadoop configuration
-   MapReduce, Streaming Jobs and Tool
-   HBase configuration
-   Hive server and thrift client
-   Pig configuration
-   Embedded API for Apache Hadoop FsShell and DistCp
-   JSR-223/javax.scripting for HDFS interaction
-   Spring Batch tasklets for vanilla Apache Hadoop, Hive, Pig

For a quick introduction to Spring for Apache Hadoop, see this blog [entry](http://blog.springsource.org/2012/02/29/introducing-spring-hadoop/) or the reference documentation.

We look forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?80-NoSQL) or in the [issue tracker](https://jira.springsource.org/browse/SHDP).
