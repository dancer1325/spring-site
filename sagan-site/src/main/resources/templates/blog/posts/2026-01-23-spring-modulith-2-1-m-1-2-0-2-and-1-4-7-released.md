# Spring Modulith 2.1 M1, 2.0.2, and 1.4.7 released

_Releases | Oliver Drotbohm |  January 23, 2026 | 0 Comments_

On behalf of the community I am happy to announce the availability of Spring Modulith 2.1 M1, 2.0.2, and 1.4.7. The bug fix releases primarily ship the usual dependency upgrades. The first milestone of 2.1 upgrades to Boot 4.1's first milestone and includes a few minor additions accumulated after the 2.0 GA release.

-   Allow resetting the shift in `TimeMachine` [#1490](https://github.com/spring-projects/spring-modulith/issues/1490)
-   Allow skipping all tests if no changes are detected [#1438](https://github.com/spring-projects/spring-modulith/issues/1438)

Find more details about the releases in the full changelog for [2.1 M1](https://github.com/spring-projects/spring-modulith/releases/tag/2.1.0-M1), [2.0.2](https://github.com/spring-projects/spring-modulith/releases/tag/2.0.2), and [1.4.7](https://github.com/spring-projects/spring-modulith/releases/tag/1.4.7).
