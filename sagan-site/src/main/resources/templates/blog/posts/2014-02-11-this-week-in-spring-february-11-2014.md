# This Week in Spring - February 11, 2014

_Engineering | Josh Long |  February 11, 2014 | 0 Comments_

Welcome back to another installation of *This Week in Spring*! I'm hurriedly preparing for a busy few weeks. I'll be giving a tech talk at our office (875 Howard St, San Francisco, CA) on the 18th (and you're more than welcome! Just turn up around 12:30 PST on Tuesday the 18th and join us as we talk Boot, baby!), some training on the 19th, then I'll be in Atlanta for the epic DevNexus show and *then*, the very next day, off to Montreal, Canada for the ConFoo conference. I hope to see you here, [at the GoPivotal offices for the tech talk](http://pivotallabs.com/), or in [Atlanta](http://devnexus.com), or in [Montreal](http://confoo.ca)!

This week we've got *lots* of great stuff, particularly concerning using Spring for building data-driven applications. Check it out!

-   Want to learn more about Spring 4's rich support for Java 8? Check out [Spring lead Juergen Hoeller's upcoming webinar on Spring 4's support for Java 8](http://spring.io/blog/2014/02/11/webinar-spring-framework-4-0-on-java-8-march-25)!
-   We're excited to announce a webinar on the frequently-asked about [Spring LDAP project](http://spring.io/blog/2014/02/11/webinar-spring-ldap-2-0-0-on-march-18-2014)
-   Check out project lead Michael Hunger's [epic SpringOne2GX 2013 talk on Spring Data Neo4j](http://spring.io/blog/2014/02/11/springone2gx-2013-replay-real-world-applications-with-spring-data-neo4j)
-   Jeremy Grelle, Mattias Niemela, and Tim Branyen's talk from SpringOne2GX 2013 on AngularJS, Backbone.js and client-centric design [from SpringOne2GX 2013 is now available online](http://spring.io/blog/2014/02/11/springone2gx-replay-angularjs-backbone-js-and-client-centric-design)
-   Spring Data lead Oliver Gierke has just [announced that Spring Data Elasticsearch 1.0M1 has been released](http://spring.io/blog/2014/02/11/spring-data-elasticsearch-1-0-m1-released)
-   Speaking of, check out Oliver's post introducing some [of the new features in Spring Data Codd RC1](http://spring.io/blog/2014/02/07/what-s-new-in-spring-data-codd-rc1)
-   JANNE VALKEALAHTI has a nice post on using the [Spring Data YARN project](http://spring.io/blog/2014/02/06/spring-yarn-application-model)
-   Thomas Risberg has announced that [Spring Hadoop M5 is now available](http://spring.io/blog/2014/02/06/spring-for-apache-hadoop-2-0-m5-released)
-   Check out this introduction [to Apache Hadoop and Spring Data Hadoop by Hadoop expert Kailash Kutti (video from the Singapore Spring User Group)](http://spring.io/blog/2014/02/06/introduction-to-apache-hadoop-and-spring-data-hadoop)
-   Also from the Singapore Spring User Group, [a presentation introducing Spring Data JPA and Spring Data MongoDB](http://spring.io/blog/2014/02/06/introduction-to-spring-data-jpa-and-spring-data-mongodb)
-   Oliver and company keep it coming [with a release of Spring Data Couchbase 1.0 RC1](http://spring.io/blog/2014/02/06/spring-data-couchbase-1-0-rc1-released)
-   Martin Lippert has announced that [Spring Tool Suite and Groovy/Grails Tool Suite 3.5.0.M2 are now available](http://spring.io/blog/2014/02/05/spring-tool-suite-and-groovy-grails-tool-suite-3-5-0-m2-released)
-   Norris Shelton, Jr has written up a nice post on how to get [the current, active Spring profile](http://norrisshelton.wordpress.com/2014/02/11/get-the-active-spring-profiles/)
-   Jan Philipp has a nice post on how to use [Spring Integration's *fabulous* gateway support to expose a facade for a messaging system based on an interface, in this case one that ultimately speaks to RabbitMQ](http://www.knallisworld.de/blog/2014/02/05/expose-a-java-method-with-amqp-using-spring-integration/)
-   Theodora Fragkouli put together a nice [post on Spring for Apache Hadoop 2.0 M5](http://www.javacodegeeks.com/2014/02/spring-for-apache-hadoop-2-0-m5.html). Check it out!
