# This Week in Spring - November 18th, 2025

_Engineering | Josh Long |  November 18, 2025 | 0 Comments_

# [](#this-week-in-spring---november-18th-2025)This Week in Spring - November 18th, 2025

Hi, Spring fans! I'm thrilled to be in New York City for an exciting week of joint presentations on Spring AI + Bedrock and Spring Boot with the legendary James Ward.

First up: we'll present a workshop at the AI Native Dev Conf today, then speak at the Garden State JUG tonight, then deliver a talk tomorrow at the conference, and finally speak at the New York Java SIG tomorrow night. It might be cold out, but we're bringing the heat!

You know who else is bringing the heat? The Spring team, of course! This past week saw Spring Framework 7 go GA, and with it a whole release train of new releases that will eventually culminate in Spring Boot 4. Once Spring Boot 4 is released, Spring AI, Spring Modulith, and Spring Cloud will all be able to release as well. Watch this space — spring.io/blog — for a ton of new releases this week culminating on the 20th with **Spring Boot 4.0.0**! (For more on what comes when, check out [the Spring Calendar](https://calendar.spring.io).)

As you can imagine, we've got a ton of stuff to dive into, so let's get to it!

-   **Spring Framework 7**, the new baseline for a [whole new generation of Spring, is available now!](https://spring.io/blog/2025/11/13/spring-framework-7-0-general-availability)
-   Want to learn about how Spring Framework 7 and Spring Boot 4 help deliver null-safety to the Java language? [Check this out](https://spring.io/blog/2025/11/12/null-safe-applications-with-spring-boot-4)!
-   [Spring Framework 6.2.13 is available now](https://spring.io/blog/2025/11/13/spring-framework-6-2-13-available-now)
-   [Spring Data 2025.1.0 goes GA](https://spring.io/blog/2025/11/14/spring-data-2025-1-goes-ga)
-   [Spring Data 2025.0.6 and 2024.1.12 released](https://spring.io/blog/2025/11/14/spring-data-2025-0-6-and-2024-1-12-released)
-   [Spring for Apache Pulsar 1.2.12 and 2.0.0 are available now](https://spring.io/blog/2025/11/17/spring-for-apache-pulsar-1-2-12-and-2-0-0-are-now-available). Did you catch that? Spring for Apache Pulsar **2.0.0**!
-   [Spring Cloud 2025.1.0-RC1 (aka Oakwood) has been released](https://spring.io/blog/2025/11/13/spring-cloud-2025-1-0-RC1-aka-oakwood-has-been-released)
-   Spring Security 2025-11 releases — [7.0.0, 6.5.7, and 6.4.13 are all available now!](https://spring.io/blog/2025/11/17/spring-security-releases) Yes, that's right, **Spring Security 7** is available now!
-   [Spring Vault 4.0 goes GA](https://spring.io/blog/2025/11/18/spring-vault-4-0-goes-ga)
-   One of the things I love in the new release is the possibility of making even more use [of advisors for control flow](https://spring.io/blog/2025/11/10/spring-ai-llm-as-judge-blog-post); in this blog we look at using LLM response evaluation with Spring AI.

-   James Ward, DaShaun Carter, and I looked at the amazing new Spring AI 1.1 release and [its bountiful new MCP support](https://youtu.be/qAOA6PKmrnA?si=m0lzv67IqRhHo2bB)
-   From data to insights: [building a Bluesky bot powered by Spring AI](https://www.youtube.com/watch?v=uGgK0F3cLEI)
-   An interesting (Chinese-language) [article about Spring Boot 4.0](https://didispace.com/posts/171)
-   Spring team contributor and legend Sebastien Deleuze has a nice post on some of [the awesome features to expect in Spring Boot 4](https://www.linkedin.com/posts/deleuze_java-spring-activity-7392877857284444161-faOk?rcm=ACoAAACXdIABb8VbfXttpb7WA9oliQsXOS5rEgw)
-   In last week's installment of *A Bootiful Podcast*, [I talked to legendary author Bruce Eckel about just about everything!](https://spring.io/blog/2025/11/13/a-bootiful-podcast-bruce-eckel)
