# Spring IO Platform Cairo SR4

_Releases | Andy Wilkinson |  September 13, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Cairo-SR4 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 2.0.6
-   Spring Boot 2.0.6
-   Spring Cloud Connectors 2.0.3
-   Spring Data Kay SR10
-   Spring Framework 5.0.9
-   Spring Integration 5.0.8
-   Spring Kafka 2.1.10
-   Spring Security 5.0.8
-   Spring Session Apple SR5

The versions of a number of third-party dependencies have also been updated.
