# Spring IDE 1.3.1 Released

_Releases | Torsten Juergeleit |  June 08, 2006 | 0 Comments_

Dear Spring Community,  
  
We are pleased to announce that Spring IDE 1.3.1 has been released.  
  
This release of Spring IDE provides a bunch of bugfixes. For a complete list visit [](http://springide.org/project/milestone/Release%201.3.1)[http://springide.org/project/milestone/Release%201.3.1](http://springide.org/project/milestone/Release%201.3.1)

Changes:

-   Now [Web Tools](http://www.eclipse.org/webtools/) is only required if you want to use the [BeansXmlEditor](http://springide.org/project/wiki/BeansXmlEditor)
-   Fixed resource leak in image caching
-   Updated to Spring 1.2.8

More details are available from the project's site [](http://springide.org/)[http://springide.org/](http://springide.org/)  
  
Use Eclipse's update manager to [install](http://springide.org/project/wiki/SpringideInstall) this release from the project's update site [](http://springide.org/updatesite/)[http://springide.org/updatesite/](http://springide.org/updatesite/)  
  
For bug reports tickets can be opened via [](http://springide.org/project/newticket)[http://springide.org/project/newticket](http://springide.org/project/newticket)
