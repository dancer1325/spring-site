# Spring Batch 5.0.2 available now!

_Releases | Mahmoud Ben Hassine |  May 17, 2023 | 0 Comments_

I am pleased to announce that Spring Batch 5.0.2 is now available from Maven Central.

This release resolves 25 issues, including bug fixes, enhancements, documentation updates and dependency updates. For more details about the changes, please refer to the [change log](https://github.com/spring-projects/spring-batch/releases/tag/v5.0.2).

I would like to thank all contributors who had a role in this release!
