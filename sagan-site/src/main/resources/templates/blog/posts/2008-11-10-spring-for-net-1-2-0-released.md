# Spring for .NET 1.2.0 Released

_Releases | Mark Pollack |  November 10, 2008 | 0 Comments_

We are pleased to announce that Spring for .NET 1.2.0 is now available.  
  
[Download](http://www.springframework.net/download.html) | [Support](http://www.springsource.com/) | [Documentation](http://www.springframework.net/documentation.html#Reference_Manual_and_API)| [Community](http://forum.springframework.net/)  
  
This release contains the following new major features:  

-   [WCF integration](http://www.springframework.net/docs/1.2.0/reference/html/wcf.html) - Configure WCF services using dependency injection. Apply AOP advice to WCF services.
-   [MSMQ integration](http://www.springframework.net/docs/1.2.0/reference/html/msmq.html) - MSMQ helper classes to increase your productivity developing messaging applications. Provides integration with Spring's transaction management features.
-   [Apache ActiveMQ integration](http://www.springframework.net/docs/1.2.0/reference/html/messaging.html) - Helper classes to increase your productivity developing messaging applications with ActiveMQ/NMS
-   [Quartz integration](http://www.springframework.net/docs/1.2.0/reference/html/scheduling.html) - Configure Quartz jobs, schedulers, triggers using dependency injection. Convenience classes for implementing Quartz Jobs.
-   [AOP](http://www.springframework.net/docs/1.2.0/reference/html/aop.html#aop-inheritancebasedaopconfigurer)\- New inheritance based AOP proxy generation
-   NHibernate 2.0.1 support.

This release includes approximately 100 bug fixes and enhancements since the 1.1.2 release.  
  
Please refer to the [changelog](http://www.springframework.net/docs/1.2.0/changelog.txt) for additional details.  
  
Enjoy!
