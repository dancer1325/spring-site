# Spring Boot 2.2.4 released

_Releases | Stéphane Nicoll |  January 20, 2020 | 5 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot 2.2.4 has been released and is now available from [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This release includes [6 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-boot/releases/tag/v2.2.4.RELEASE) and was released shortly after Spring Boot 2.2.3 to [work around a dependency resolution problem with Gradle 5.3 to 5.6](https://github.com/spring-projects/spring-boot/issues/19783).
