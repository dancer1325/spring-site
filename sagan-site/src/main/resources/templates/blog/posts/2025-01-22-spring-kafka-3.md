# Spring for Apache Kafka 3.3.2 is available

_Releases | Soby Chacko |  January 22, 2025 | 0 Comments_

We are pleased to announce the release of Spring for Apache Kafka version `3.3.2`

#### [](#release-notes)Release Notes

For details about what's included in this release, please refer to the release notes:

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.2](https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.2)

#### [](#kafka-client-39x-compatibility)Kafka Client 3.9.x compatibility

This release supports Kafka Client versions 3.8.x and 3.9.x. If your application requires Kafka Client 3.9.x, you'll need to manually upgrade the client version. For instructions on overriding Kafka client versions in Spring Boot, please refer to our [documentation](https://docs.spring.io/spring-kafka/reference/appendix/override-boot-dependencies.html).

#### [](#spring-boot-integration)Spring Boot Integration

The `3.3.2` release will be incorporated into the upcoming Spring Boot `3.4.2`. Additionally, this version will be included in the Spring Boot `3.5.x` line and will ship with the upcoming Spring Boot `3.5.0-M1` release.

#### [](#looking-ahead-to-the-next-major-version-of-spring-for-apache-kafka)Looking Ahead to the Next Major Version of Spring for Apache Kafka

Development of Spring for Apache Kafka 4.0.x will commence in the near future. This major version will be built on Kafka Client 4.0.x, Spring Framework 7.0.x and will be compatible with Spring Boot 4.0.x. We encourage community involvement through feature requests, bug reports, and general feedback.

You can engage with the project through these channels:
