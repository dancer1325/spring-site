# Spring Web Flow 2.3.0 Released

_Releases | Rossen Stoyanchev |  March 01, 2011 | 0 Comments_

Spring Web Flow 2.3 is now available for download. You can review [what's new](http://static.springsource.org/spring-webflow/docs/2.3.x/reference/htmlsingle/spring-webflow-reference.html#whatsnew) in the reference documentation, see the complete [changelog](http://static.springsource.org/spring-webflow/docs/2.3.x/changelog.txt), or [download](http://www.springsource.com/download/community?project=Spring%20Web%20Flow) the release.

A few of the highlights include:

-   Launching a flow in a page embedded mode.
-   JSR-303 bean validation support.
-   Propagation of a flow-managed persistence context to sub-flows.
-   Support for partial rendering with Portlet resource requests.
