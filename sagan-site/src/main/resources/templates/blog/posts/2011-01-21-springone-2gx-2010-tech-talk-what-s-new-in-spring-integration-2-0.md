# SpringOne 2GX 2010 Tech Talk: What's New in Spring Integration 2.0?

_News | Adam Fitzgerald |  January 21, 2011 | 0 Comments_

[![SpringOne 2GX 2010 Tech Talk: What's new in Spring Integration 2.0](/files/SpringOne2GX2010MarkFisher.png)](http://www.infoq.com/presentations/Spring-Integration-2)

Yet another session from [SpringOne 2GX 2010](http://www.springone2gx.com) has been posted by [InfoQ](http://www.infoq.com/SpringOne_2010). This time it is [Mark Fisher](http://blog.springsource.com/author/markf/) and [Oleg Zhurakousky](http://blog.springsource.com/author/ozhurakousky/) who are presenting on [What's New in Spring Integration 2.0](http://www.infoq.com/presentations/Spring-Integration-2).

Mark and Oleg use code-centric demos to describe how Spring 3.0 features have been integrated in Spring Integration, what enterprise integration patterns have been implemented and where channel adapters and gateways have been updated. They demonstrate the new [STS](http://www.springsource.com/developer/sts) tooling support for Spring Integration and discuss some of the future features and direction for the project.

This presentation is a great way to get introduced to [Spring Integration 2.0](/spring-integration). The [GA release of 2.0](http://blog.springsource.com/2010/11/23/spring-integration-2-0-ga-released/) followed less than one month after this talk so all the topics covered are up to date with the current release. Again many thanks to InfoQ for [covering SpringOne 2GX 2010](http://www.infoq.com/SpringOne_2010).
