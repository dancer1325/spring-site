# Spring Data Lovelace SR11 released

_Releases | Mark Paluch |  September 30, 2019 | 0 Comments_

On behalf of the community, I'm pleased to announce that Spring Data `Lovelace SR11` is now available.

This release ships with [72 tickets fixed](https://jira.spring.io/issues/?filter=16438), mostly bug fixes and dependency upgrades. `Lovelace SR11` is built on top the recently released [Spring Framework `5.1.10`](https://spring.io/blog/2019/09/28/spring-framework-5-1-10-available-now) and will be picked up by Spring Boot `2.1.9` for easier consumption.
