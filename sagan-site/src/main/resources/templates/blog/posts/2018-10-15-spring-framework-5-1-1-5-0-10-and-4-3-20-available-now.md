# Spring Framework 5.1.1, 5.0.10 and 4.3.20 available now

_Releases | Stéphane Nicoll |  October 15, 2018 | 5 Comments_

On behalf of the team, I am pleased to announce that Spring Framework 5.1.1, 5.0.10 and 4.3.20 [are available now](http://projects.spring.io/spring-framework).

The first maintenance release of the 5.1 line includes [over 30 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=17148). Spring Framework 5.0.10 [also includes a wide range of 20 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=17115) while Spring Framework 4.3.20 [is a rather minimal patch release](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=17116).

The first release candidate of Spring Boot 2.1, based on Spring Framework 5.1.1, is around the corner! We'll also follow up with corresponding Spring Boot 2.0.6 and 1.5.17 releases later this week.

[Project Page](https://spring.io/projects/spring-framework) | [GitHub](https://github.com/spring-projects/spring-framework) | [Issues](https://jira.spring.io/browse/SPR) | [Documentation](https://docs.spring.io/spring-framework/docs/5.1.1.RELEASE/spring-framework-reference)

**2018-10-16 UPDATE:** The releases address a vulnerability. Please see [this blog post](https://spring.io/blog/2018/10/16/spring-project-vulnerability-reports-published) published after the associated Spring Boot 2.0.6 and 1.5.17 releases.
