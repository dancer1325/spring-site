# Spring Cloud Task 2.0.0.M2 is now available

_Releases | Glenn Renfro |  November 10, 2017 | 0 Comments_

We are pleased to announce that Spring Cloud Task 2.0.0.M2 is now available to download from Spring’s [milestone repository](https://repo.spring.io/libs-milestone-local/org/springframework/cloud/spring-cloud-task-core/2.0.0.M2/). Many thanks to all of those who contributed to the release.

This release brings Spring Boot 2.0 M5 compatibility. With this upgrade, all dependencies are also upgraded (Spring Cloud, Spring Cloud Stream, Spring Batch, etc).
