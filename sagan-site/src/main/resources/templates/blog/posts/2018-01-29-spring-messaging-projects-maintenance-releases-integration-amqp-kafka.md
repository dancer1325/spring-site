# Spring Messaging Projects Maintenance Releases - Integration, AMQP, Kafka

_Releases | Gary Russell |  January 29, 2018 | 0 Comments_

We are pleased to announce the availability of the following maintenance releases; users are encouraged to upgrade at the earliest opportunity.

-   Spring Integration - [5.0.1](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=16627), [4.3.14](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=16631) and [4.2.13](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10121&version=15806)
    
-   Spring Integration for Apache Kafka [3.0.1](https://github.com/spring-projects/spring-integration-kafka/releases/tag/v3.0.1.RELEASE)
    
-   Spring for Apache Kafka - [2.1.2](https://github.com/spring-projects/spring-kafka/releases/tag/v2.1.2.RELEASE), [2.0.3](https://github.com/spring-projects/spring-kafka/releases/tag/v2.0.3.RELEASE) and [1.3.3](https://github.com/spring-projects/spring-kafka/releases/tag/v1.3.3.RELEASE)
    
-   Spring AMQP - [2.0.2](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=16625) and [1.7.6](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=16624)
    

Spring Integration `4.3.13`, Spring AMQP `1.7.6` will be used in the upcoming Spring Boot `1.5.10` release. Spring Integration `5.0.1`, Spring AMQP `2.0.2` and Spring for Apache Kafka `2.1.2` will be used in the upcoming release candidate for Spring Boot `2.0`.

See the respective project pages at [spring.io/projects](https://spring.io/projects) for more information.

The Spring Integration for Apache Kafka documentation is in the Spring for Apache Kafka Reference Manual.
