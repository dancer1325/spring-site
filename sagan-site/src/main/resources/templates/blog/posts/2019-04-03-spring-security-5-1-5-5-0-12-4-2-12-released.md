# Spring Security 5.1.5, 5.0.12, 4.2.12 Released

_Releases | Joe Grandja |  April 03, 2019 | 1 Comment_

On behalf of the community I am pleased to announce the release of Spring Security 5.1.5 ([changelog](https://github.com/spring-projects/spring-security/milestone/140?closed=1)), 5.0.12 ([changelog](https://github.com/spring-projects/spring-security/milestone/138?closed=1)), and 4.2.12 ([changelog](https://github.com/spring-projects/spring-security/milestone/139?closed=1)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
