# Spring Batch 4.2.2 and 4.1.4 available now!

_Releases | Mahmoud Ben Hassine |  April 01, 2020 | 0 Comments_

I am pleased to announce the release of Spring Batch 4.2.2 and 4.1.4 with bug fixes and documentation updates. Please find the complete list of changes in the release notes: [4.2.2](https://github.com/spring-projects/spring-batch/releases/tag/4.2.2.RELEASE) and [4.1.4](https://github.com/spring-projects/spring-batch/releases/tag/4.1.4.RELEASE).

As we [announced earlier](https://spring.io/blog/2019/12/04/spring-batch-4-0-4-4-1-3-and-4-2-1-available-now), version 4.1.4 is the last release of the 4.1 line. Please upgrade to version 4.2.2 at your earliest convenience as this is the primary active branch for the moment and which will be supported until the end of 2020.

The next feature release will be 4.3, with a GA planned for October 2020, aligned with Spring Framework 5.3 and Spring Boot 2.4. This release is expected to be the last feature branch of Spring Batch 4.
