# This Week in Spring - August 26th, 2014

_Engineering | Josh Long |  August 26, 2014 | 0 Comments_

Holy cats! Can you believe how close we are to [SpringOne2GX](http://springone2gx.com/)? Last year we made a huge splash with the announcements of Spring Boot and Spring XD, both of which have recently hit 1.0 releases. I happen to know the next level looks like, and you're going to want to see it. Register now if you haven't already!

-   Spring ninja Greg Turnquist wanted to [replace one of the services used in the guides with something more family-friendly and so stood up a bespoke substitute using Spring](https://spring.io/blog/2014/08/21/building-a-restful-quotation-service-with-spring)! This is a really nice read, check it out!
-   I hope you'll [join me September 16th for a webinar looking at building microservices with Spring](https://spring.io/blog/2014/07/29/webinar-microservices-with-spring-boot-sept-16th)
-   On September 23rd, join Joram Barrez and I for a webinar looking at [how to build process-centric Spring applications with the open-source BPMN2 engine, Activiti](https://spring.io/blog/2014/07/29/webinar-process-driven-spring-applications-with-activiti-sept-23rd)
-   Patrick Grimard did a really nice job introducing [how to build Spring Boot applications using IntelliJ IDEA](http://patrickgrimard.com/2014/08/14/how-to-build-a-spring-boot-application-using-intellij-idea/). Pretty obvious, I hope, but it's nice to have it in one place.
-   The *GeekAbyte* shows [how to implement a `HandlerMethodArgumentResolver` in other to resolve and inject objects into Spring MVC controllers](http://geekabyte.blogspot.sg/2014/08/how-to-inject-objects-into-spring-mvc.html). This is a *really* powerful technique if you need it.
-   This looks interesting - a (Swiss?) German workshop on [building modern web applications using Spring IO and Angular.js](http://techup.ch/1647/build-modern-web-apps-with-spring-io-and-angularjs). I wish I could go! Check it out if you're in the area - I'd love to know how it went and if the recording and/or slides are going to be available.
-   Our pal Tomasz Nurkiewicz is at it again, this [time with a look on building a URL shortner ("the new 'Hello, world!' in the IoT/microservice/era world" - ha!) in 42 lines of code](http://www.javacodegeeks.com/2014/08/url-shortener-service-in-42-lines-of-code-in-java-spring-boot-redis.html). So... go, check it out!
-   (Aforementioned Spring ninja) Greg Turnquist's writing a book on Spring Boot for PACKT, [and he's just posted a summary of the latest chapter (on security) on his blog](http://blog.greglturnquist.com/2014/08/chapter-5-of-learning-springboot-almost-done.html). This is exciting stuff - definitely intriguing, check it out! Rock on, Greg! (Did I mention he'll be [at SpringOne2GX, too](http://springone2gx.com/)?)
-   Pankaj Kumar introduces [a simple Spring Data MongoDB example in this post](http://www.journaldev.com/4144/spring-data-mongodb-example-tutorial)
-   This (Japanese-language) post (appears) to [introduce basic dependency injection with Spring](http://code-examples-ja.hateblo.jp/entry/2014/08/23/Spring_Framework_Dependency_Injection\(%E4%BE%9D%E5%AD%98%E6%80%A7%E6%B3%A8%E5%85%A5\)%E3%81%A8%E3%81%AF)). Could do...
-   This (Portuguese-language) post introduces how to create [a Spring Security context in unit tests](http://rodrigoportela.blog.br/2014/08/criando-um-securitycontext-para-testes-unitarios/). This approach works, but readers should *definitely* check out some of the cool stuff Spring Security lead [Rob Winch](http://spring.io/team/rwinch) has [been unveiling to support unit-testing secure applications recently](https://spring.io/blog/2014/05/23/preview-spring-security-test-web-security)...
-   A few years ago I blogged about the amazing [things the Alibaba group is doing to meet China-scale demand with Spring](https://spring.io/blog/2013/03/04/spring-at-china-scale-alibaba-group-alipay-taobao-and-tmall/). They built an (open-source) framework called Dubbo (which sits on top of Spring), and this [(Mandarin-language) post has a (simple) example demonstrating its use](http://www.kukujia.net/archives/389). *Very* cool!

I am personally *super* excited to [see Greg, Rob, the rest of the Spring team, and of course all of you, at SpringOne2GX](http://springone2gx.com)! Register now if you haven't!
