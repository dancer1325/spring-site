# Spring Framework 6.1.10 available now

_Releases | Brian Clozel |  June 19, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.10` is available now. This 6.1.x release fixes [a few regressions that were shipped in 6.1.9 last week](https://github.com/spring-projects/spring-framework/issues?q=label%3A%22type%3A+regression%22+milestone%3A6.1.10+).

Spring Framework `6.1.10` ships with [12 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.10). This version will be shipped with [Spring Boot 3.2.7](https://github.com/spring-projects/spring-boot/milestone/344) and [Spring Boot 3.3.1](https://github.com/spring-projects/spring-boot/milestone/345) tomorrow.
