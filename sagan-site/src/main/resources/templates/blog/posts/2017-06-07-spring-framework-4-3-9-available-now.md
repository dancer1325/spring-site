# Spring Framework 4.3.9 available now

_Releases | Stéphane Nicoll |  June 07, 2017 | 0 Comments_

On behalf of the team, I am pleased to announce that Spring Framework [4.3.9](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15928) [is available now](http://projects.spring.io/spring-framework).

This maintenance release is the base for the upcoming Spring Boot 1.4.7 and 1.5.4 maintenance releases and includes 30 fixes and selected improvements.
