# Dependency Management Plugin 1.0.6.RELEASE

_Releases | Andy Wilkinson |  July 18, 2018 | 0 Comments_

I am pleased to announce that [Gradle dependency management plugin](https://github.com/spring-gradle-plugins/dependency-management-plugin) 1.0.6.RELEASE is now available from Maven Central, Bintray, and the [Gradle Plugin Portal](https://plugins.gradle.org/plugin/io.spring.dependency-management). This maintenance release fixes [four issues](https://github.com/spring-gradle-plugins/dependency-management-plugin/milestone/26?closed=1).

For Spring Boot users, the new version will be included in the Spring Boot 1.5.15 and 2.0.4 releases. For all other users of the plugin, 1.0.6 is a recommended manual upgrade.
