# Spring Security 5.3.2, 5.2.4, 5.1.10, 5.0.16, 4.2.16 Released

_Releases | Rob Winch |  May 07, 2020 | 1 Comment_

**UPDATE 2020-05-13:** The following versions of Spring Security address [CVE-2020-5407 and CVE-2020-5408](https://spring.io/blog/2020/05/13/cve-reports-published-for-spring-security)

On behalf of the community, I’m pleased to announce the release of Spring Security 5.3.2 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.2.RELEASE)), 5.2.4 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.4.RELEASE)), 5.1.10 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.10.RELEASE)) , 5.0.16 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.16.RELEASE)), 4.2.16 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.16.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
