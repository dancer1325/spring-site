# Spring for GraphQL 2.0.0 goes GA

_Releases | Brian Clozel |  November 18, 2025 | 0 Comments_

I am pleased to announce that Spring for GraphQL 2.0 is now generally available and ships new features:

-   a new GraphQL Java 25 baseline
-   Nullability and JSpecify support in the project API
-   Nullability checks for your application during Schema Mapping Inspection
-   Flexible binding for input types
-   `ArgumentValue` support for client input
-   Multiple queries with `DgsGraphQlClient`
-   Kotlin extensions for `GraphQlClient` and `GraphQlTester`

You can get a full picture of this new major release with [our release notes on the wiki](https://github.com/spring-projects/spring-graphql/wiki/Spring-for-GraphQL-2.0).

Thanks to the community for your feedback and for testing the release candidates, this helped us to shape this new generation. Collaborating with the GraphQL Java team is always a pleasure, and we're sure that [you will love the new GraphQL Java 25.0 features](https://github.com/graphql-java/graphql-java/releases/tag/v25.0).
