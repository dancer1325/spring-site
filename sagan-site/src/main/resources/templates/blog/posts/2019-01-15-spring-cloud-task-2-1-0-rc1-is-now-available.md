# Spring Cloud Task 2.1.0.RC1 is now available

_Releases | Glenn Renfro |  January 15, 2019 | 0 Comments_

We are pleased to announce that Spring Cloud Task 2.1.0.RC1 is now available on Github and the Pivotal download repository. Many thanks to all of those who contributed to this release.

## [](#whats-new)What's New?

As a follow up to 2.1.0.M2, this release primarily encompasses minor enhancements and bug fixes.

## [](#what-do-you-think)What Do You Think?

We look forward to your feedback on these new features in [Github](https://github.com/spring-cloud/spring-cloud-task/issues), [StackOverflow](http://stackoverflow.com/tags/spring-cloud-task), [Gitter](https://gitter.im/spring-cloud/spring-cloud-task), or directly via Twitter to [@michaelminella](https://twitter.com/michaelminella) or [@cppwfs](https://twitter.com/cppwfs)!
