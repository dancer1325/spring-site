# Spring Vault 2.2.1.RELEASE and 2.1.5.RELEASE available

_Releases | Mark Paluch |  January 17, 2020 | 0 Comments_

On behalf of the community, I’d like to announce the availability of Spring Vault releases `2.2.1.RELEASE` and `2.1.5.RELEASE`.

Both releases ship with mostly bugfixes and dependency upgrades. Spring Vault `2.2.1.RELEASE` ships also support to disable authentication and session management so it can be used with Vault Agent’s cached authentication.

For a complete list of changes, see the [2.2.1.RELEASE changelog](http://docs.spring.io/spring-vault/docs/2.2.1.RELEASE/changelog.txt) respective [2.1.5.RELEASE changelog](http://docs.spring.io/spring-vault/docs/2.1.5.RELEASE/changelog.txt).

With this release, we’d also like to announce that Spring Vault 2.1.x is going to end of life in November 2020. We encourage all users to upgrade to the latest release of the 2.2.x line. You can find more details about the supported versions on our [versions overview](https://github.com/spring-projects/spring-vault/wiki/Spring-Vault-Versions) in the GitHub repository.
