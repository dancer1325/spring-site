# Spring Framework 6.2.0-RC3 available now

_Releases | Brian Clozel |  October 24, 2024 | 0 Comments_

We are happy to announce the availability of the last release candidate of Spring Framework 6.2. We have published this version just in time for the Spring Boot RC today. We'll be collecting feedback for the [6.2.0 GA release in November](https://github.com/spring-projects/spring-framework/milestone/384).

Spring Framework 6.2.0-RC3 is available from [repo.spring.io/milestone](https://repo.spring.io/milestone) now, check out the [detailed changelog for this version](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.0-RC3).
