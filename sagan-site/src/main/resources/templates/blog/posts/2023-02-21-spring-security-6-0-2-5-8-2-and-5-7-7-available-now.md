# Spring Security 6.0.2, 5.8.2, and 5.7.7 available now

_Releases | Josh Cummings |  February 21, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security 6.0.2, 5.8.2, and 5.7.7 are available now. In all cases, the releases are mostly composed of bug fixes, dependency upgrades, and documentation improvements.

To learn more, please visit the [6.0.2](https://github.com/spring-projects/spring-security/releases/tag/6.0.2), [5.8.2](https://github.com/spring-projects/spring-security/releases/tag/5.8.2), and [5.7.7](https://github.com/spring-projects/spring-security/releases/tag/5.7.7) release summaries.
