# Spring Boot 2.4.0-M1 is now available

_Releases | Andy Wilkinson |  June 29, 2020 | 0 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the first milestone of Spring Boot 2.4 has been released and is available from [our milestone repository](http://repo.spring.io/milestone/). This release closes [over 120 issues and pull requests](https://github.com/spring-projects/spring-boot/releases/tag/v2.4.0-M1).

Please note the slight change in the format of the version. 2.4.0-M1 is the first Spring Boot release to use the [new versioning scheme](https://spring.io/blog/2020/04/30/updates-to-spring-versions).

With Spring Boot 2.4, we have switched to a 6-month release cadence. All being well, you can expect to see 2.4.0 reach general availability in October or November.

Highlights of this first milestone include:

-   Preliminary support for JDK 15
-   [Spring Batch 4.3 M1](https://spring.io/blog/2020/06/26/spring-batch-4-3-0-m1-is-released-now)
-   [Spring Data 2020.0 M1](https://spring.io/blog/2020/06/25/first-milestone-of-spring-data-2020-0-available)
-   [Spring Framework 5.3 M1](https://spring.io/blog/2020/06/25/first-spring-framework-5-3-milestone-released)
-   [Spring Security 5.4 M1](https://spring.io/blog/2020/05/07/spring-security-5-4-0-m1-released)
-   Filtered metric scraping with Prometheus
-   Numerous other [dependency upgrades](https://github.com/spring-projects/spring-boot/issues?utf8=%E2%9C%93&q=label%3A%22type%3A+dependency-upgrade%22+milestone%3A2.4.0-M1)

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.4 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.4.0-M1-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.4.0-M1/reference/html/).

If you want to get started with 2.4 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).
