# This Week in Spring

Welcome to another installment of *This Week in Spring*! We are finally running out of SpringOne2GX video recordings.. this is the last week in Spring that you'll see them, so refer back to the [replays page](https://springone2gx.com/conference/washington/2012/10/video_list) for an index. As usual, we've got a lot to cover so let's get to it!

1.  Spring lead [Juergen Hoeller](http://twitter.com/springjuergen) just announced the release of [Spring Framework 4.0 M1 and 3.2.3.RELEASE](http://blog.springsource.org/2013/05/21/spring-framework-4-0-m1-3-2-3-available/) The 3.2.3 update mainly includes updates and fixes related to Java 8 support. The 4.0 milestone, on the other hand, is a look ahead to the many awesome features in Spring 4.0 including support for web sockets, Java EE 7, the `@Conditional` annotation (in the same vein as the `@Profile` annotation) and much more, so be sure to check it out!
2.  [TcServer 2.9.2 is now available!](https://my.vmware.com/web/vmware/info/slug/application_platform/vmware_vfabric_tc_server/2_9) The release contains security fixes and updates, for more details check out the [release notes](https://www.vmware.com/support/vfabric-tcserver/doc/vfabric-tcserver-rn-2.9.1.html).
3.  Jennifer Hickey's talk -- [Thinking outside the container - Standalone Applications on CloudFoundry](http://www.springsource.org/node/22610) has been released in HD on YouTube.
4.  Stephen Bohlen's talk --[An Introduction to Spring.NET for Java developers](http://www.springsource.org/node/22610), has been released in HD on YouTube.
5.  [InfoQ has done a nice writeup of Spring HATEOAS](http://www.infoq.com/news/2013/05/spring-hateoas-rest)
6.  Johnathan Mark Smith has put together a nice post on [how to use Spring's Java configuration style](http://johnathanmarksmith.com/spring/java/javaconfig/programming/2013/04/25/how-to-use-spring-javaconfig-and-not-xml-files-for-configuation/).
7.  Xavier Padró has put together a nice post on [how to communicate within a Spring Web Flow `flow`](http://xpadro.blogspot.com.es/2013/04/communication-in-spring-webflow-2.html)
8.  [Static.com](http://static.com) has announced [their Hadoop and Cloud Foundry-powered service](http://blog.cloudfoundry.com/2013/05/20/static-com-adds-hadoop-support-for-cloud-foundry/). Frankly, it looks *really* cool and cost-effective as a public platform on which to host applications that need a backoffice Hadoop solution.
9.  I can't believe I missed this! RabbitMQ 3.1.0 is out (slightly old news) and, to introduce it, you should check [out this amazing RabbitMQ 3.1.0 in pictures](http://www.rabbitmq.com/blog/2013/05/01/rabbitmq-3-1-0-in-images/).
10.  The HMKCode blog has a nice [post on doing the not-so-well-documented, but common, things with a MyBatis, Spring and jUnit integration](http://hmkcode.com/mybatis-spring-junit-the-missing-part/).
11.  The Java Code Geeks have put together a nice tutorial [on how to process radio buttons in a `form`](http://www.javacodegeeks.com/2013/05/spring-mvc-form-handling-vol-4-radiobuttons.html) in Spring MVC.  
     
12.  Gerry Tan has put together a nice blog on how to bind [form date values with Spring MVC](http://gerrytan.wordpress.com/2013/05/21/binding-date-form-input-on-spring-mvc/).
