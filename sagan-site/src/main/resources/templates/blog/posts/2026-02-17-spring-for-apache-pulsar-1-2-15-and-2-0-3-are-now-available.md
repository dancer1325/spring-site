# Spring for Apache Pulsar 1.2.15 and 2.0.3 are now available

_Releases | Chris Bono |  February 17, 2026 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar `1.2.15` and `2.0.3` have been released and are now available from Maven Central.

The `1.2.15` release will be included in the upcoming Spring Boot [3.5.11](https://github.com/spring-projects/spring-boot/milestones/3.5.11) releases. The `2.0.3` release will be included in the upcoming Spring Boot [4.0.3](https://github.com/spring-projects/spring-boot/milestones/4.0.3) release.

Please see the release notes ([1.2.15](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.15) and [2.0.3](https://github.com/spring-projects/spring-pulsar/releases/tag/v2.0.3)) for more details.
