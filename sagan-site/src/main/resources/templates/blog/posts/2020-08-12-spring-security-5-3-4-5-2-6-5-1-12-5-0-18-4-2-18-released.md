# Spring Security 5.3.4, 5.2.6, 5.1.12, 5.0.18, 4.2.18 Released

_Releases | Eleftheria Stein-Kousathana |  August 12, 2020 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.3.4 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.4.RELEASE)), 5.2.6 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.6.RELEASE)), 5.1.12 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.12.RELEASE)) , 5.0.18 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.18.RELEASE)), 4.2.18 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.18.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
