# Spring Boot 1.3.6 available now

_Releases | Stéphane Nicoll |  July 04, 2016 | 0 Comments_

Right after [Spring Framework 4.2.7](https://spring.io/blog/2016/07/04/spring-framework-4-3-1-and-4-2-7-available-now), Spring Boot 1.3.6 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.3.6+is%3Aclosed) and 3rd party dependency updates.
