# Spring Web Flow 3.0.0 Released

_Releases | Rossen Stoyanchev |  July 13, 2023 | 0 Comments_

I'm pleased to announce the availability of Spring Web Flow 3.0.0. This release is all about compatibility with Spring Framework 6 and Jakarta EE. The [spring-webflow-sampes](https://github.com/spring-projects/spring-webflow-samples) have been updated, and the commit history provides an example of changes necessary to upgrade. One significant change is the need to remove Tiles which has not migrated to Jakarta EE. The booking-mvc sample now uses Thymeleaf Layouts instead.

Once again special thanks to Ian Young, Scott Cantor, and Gábor Lipták for your help to make these updates available to the community!
