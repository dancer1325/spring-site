# Spring AMQP 4.0 Milestone 1 Available

_Releases | Artem Bilan |  February 19, 2025 | 0 Comments_

On behalf of the team and everyone who contributed, I am pleased to announce the first Milstone for `4.0.0` of Spring AMQP.

The patch versions `3.1.9` and `3.2.3` also have been released.

The notable changes in `4.0.0` generation so far are:

-   Migration null-safety to [JSpecify and NullAway](https://docs.spring.io/spring-framework/reference/7.0-SNAPSHOT/core/null-safety.html);
-   Removal of previously deprecated API;
-   Migrate some data classes to Java `record`.

The `4.0.0-M1` is also the first Milestone for this project which, according to our new policy, has been published to [Maven Central](https://repo1.maven.org/maven2/org/springframework/amqp/spring-amqp-bom/4.0.0-M1).

Many thanks to community for feedback and prompt fixes!

See more info in the [Release Notes](https://github.com/spring-projects/spring-amqp/releases/tag/v4.0.0-M1).

Don't hasitate to reach us out in GitHub issues for the project!

Cheers,   
Artem
