# Spring Social 1.0.0 Released

_Releases | Craig Walls |  September 08, 2011 | 0 Comments_

Dear Spring Community,

We are pleased to announce *general availability* of [Spring Social 1.0](http://www.springsource.org/spring-social)!

> **Spring Social is an extension of the Spring Framework that enables you to connect your Java applications to Software-as-a-Service (SaaS) providers such as Facebook and Twitter.**

The big 1.0 release includes fixes for all bugs reported since 1.0.0.RC3. See the change log for all the details ([Core](http://static.springsource.org/spring-social/docs/1.0.0.RELEASE/changelog.txt) | [Facebook](http://static.springsource.org/spring-social-facebook/docs/1.0.0.RELEASE/changelog.txt) | [Twitter](http://static.springsource.org/spring-social-twitter/docs/1.0.0.RELEASE/changelog.txt))

To get the software, download the release distribution ([Core](http://www.springsource.com/download/community?project=Spring%20Social&version=1.0.0.RELEASE) | [Facebook](http://www.springsource.com/download/community?project=Spring%20Social%20Facebook&version=1.0.0.RELEASE) | [Twitter](http://www.springsource.com/download/community?project=Spring%20Social%20Twitter&version=1.0.0.RELEASE)) or simply add the [maven artifacts](http://static.springsource.org/spring-social/docs/1.0.0.RELEASE/reference/html/overview.html#overview-howtoget) to your project. To see it live, try out the [quickstart](https://github.com/SpringSource/spring-social/wiki/Quick-Start) and fire up the [samples](https://github.com/SpringSource/spring-social-samples). Supplement as you go with information from the [reference manual](http://static.springsource.org/spring-social/docs/1.0.x/reference/html/).

Spring Social requires Spring Framework 3.0.5 or > to run. We recommend [Spring 3.1](http://blog.springsource.com/2011/06/09/spring-framework-3-1-m2-released/) for new applications to take advantage of the latest advances in the core framework. See the reference manual for a full description of dependencies.

It has been an awesome year leading up to this release. We are thankful for all of the community involvement that has helped make this a useful, quality project. Special thanks go to Matt Wright, Morton Andersen-Gott, Stefan Fussenegger, Bryce Fischer, Gabriel Axel, Marc Schipperheyn, Domingo Suarez, Gordon Dickens, Arjen Poutsma, and Juergen Hoeller for their support. I've taken the opportunity to reflect on our work in the last year at our [team blog](http://blog.springsource.com/2011/09/08/spring-social-1-0-what-a-year-makes).

Join me for the webinar that I'll be hosting on September 29th.

We hope you enjoy using Spring Social!
