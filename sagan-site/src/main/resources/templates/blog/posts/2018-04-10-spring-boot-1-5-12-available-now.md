# Spring Boot 1.5.12 available now

_Releases | Phil Webb |  April 10, 2018 | 5 Comments_

CVE-2018-1270 [reported last week](https://spring.io/blog/2018/04/05/multiple-cve-reports-published-for-the-spring-framework) was unfortunately not fully addressed in the 4.3.x branch of the Spring Framework. Spring Framework [4.3.16 has been released](https://spring.io/blog/2018/04/09/cve-2018-1275-address-partial-fix-for-cve-2018-1270) today, so we've decided to also release Spring Boot 1.5.12 to help people upgrade easily.

This release includes just [3 fixed issues](https://github.com/spring-projects/spring-boot/milestone/108?closed=1), but should be considered a priority upgrade for all existing Spring Boot 1.5 users.

Spring Boot 2.0 users are not affected and should use the existing 2.0.1 release.
