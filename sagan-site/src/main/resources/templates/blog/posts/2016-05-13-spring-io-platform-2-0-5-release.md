# Spring IO Platform 2.0.5.RELEASE

_Releases | Andy Wilkinson |  May 13, 2016 | 1 Comment_

Following on from Spring Boot 1.3.5.RELEASE [which fixed a regression](https://spring.io/blog/2016/05/10/spring-boot-1-3-5-released), [Spring IO Platform](http://platform.spring.io/platform) 2.0.5.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

This maintenance release upgrades the versions of a number of the projects in the Platform to pick up their latest maintenance releases:

-   Spring AMQP 1.5.6.RELEASE
-   Spring Boot 1.3.5.RELEASE
