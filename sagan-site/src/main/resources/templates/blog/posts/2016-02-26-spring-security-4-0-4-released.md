# Spring Security 4.0.4 Released

_Releases | Rob Winch |  February 26, 2016 | 0 Comments_

I'm pleased to announce the release of Spring Security 4.0.4.RELEASE. This release is the fourth maintenance release of the 4.0 line. It provides minor fixes along with a fix for backwards compatibility of Spring Framework 4.1.x. For complete details on the release, refer to the [Change Log](https://github.com/spring-projects/spring-security/issues?utf8=%E2%9C%93&q=milestone%3A4.0.4).
