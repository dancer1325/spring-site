# This Week in Spring - May 10th, 2016

_Engineering | Josh Long |  May 10, 2016 | 0 Comments_

Auntie Em! Time sure is flying! Welcome to another installment of *This Week in Spring!* This week I'm in Warsaw for a Cloud Native meetup and Krakow, Poland for the spectacular Geecon event! [If you're around, don't hesitate to say hi](http://twitter.com/starbuxman) for Also, some *very* big news coming! Stay tuned [to the Spring.io blog tomorrow](http://spring.io/blog)! Anyway, as usual, we've got a lot to look at so let's get to it!

-   Spring Security lead Rob Winch [just announced Spring Security 4.1.0 RELEASE](https://spring.io/blog/2016/05/05/spring-security-4-1-0-released)! This is a big release with [a lot of cool stuff](http://docs.spring.io/spring-security/site/docs/4.1.x/reference/htmlsingle/#new)!
-   Spring IO Platform lead [Andy Wilkinson just announced Spring IO Platform 2.0.4 release](https://spring.io/blog/2016/05/09/spring-io-platform-2-0-4-release)
-   Spring AMQP and messaging guru Gary Russell just [announced Spring AMQP 1.6.0 RC (and 1.5.6)](https://spring.io/blog/2016/05/06/spring-amqp-1-6-0-release-candidate-and-1-5-6-available)
-   Spring framework [ninja Stéphane Nicoll just announced Spring Framework 4.3 RC2, 4.2.6 and 3.2.17 are now available](https://spring.io/blog/2016/05/06/spring-framework-4-3-rc2-4-2-6-and-3-2-17-available-now)
-   not one to sit on his laurels, [Stéphane *also* announced Spring Boot 1.3.5](https://spring.io/blog/2016/05/10/spring-boot-1-3-5-released)
-   This [blog has a really nice look at some of the players in the microservice ecosystem](https://www.datawire.io/continuous-delivery-ecosystem-for-microservices/), albeit a high level one.
-   Want to learn about the service package manager that underpins Cloud Foundry, BOSH? [This blog's a *very* fine start](http://www.engineerbetter.com/bosh-concourse.html)!
-   Jessica Kerr and Continuous Delivery luminary Jez Humble had a nice discussion at Craft Conf, it seems, and [Jessica's resulting blog post on tradeoffs in coordination among teams](http://blog.jessitron.com/2016/05/tradeoffs-in-coordination-among-teams.html) is well worth a read
-   [Using Data Science to Make Sense of Unstructed Text](https://blog.pivotal.io/data-science-pivotal/products/using-data-science-to-make-sense-of-unstructured-text)
-   I enjoyed this nice post [on Spring Boot and HTTP 2](https://vanwilgenburg.wordpress.com/2016/04/01/spring-boot-http2/)
-   a nice post from my buddy Markus Eisele on our joint talk at O'Reilly's Software Architecture show where we [discussed Spring and Java EE integration patterns](http://blog.eisele.net/2016/04/integration-architecture-with-java-ee-and-spring.html).
