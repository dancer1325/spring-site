# This Week in Spring - June 5th, 2018

_Engineering | Josh Long |  June 06, 2018 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! It's been a wild and wonderful week. I've just returned from the epic VOXXED Singapore event, just intime to enjoy a wonderful week home in Spring-time and sun-filled San Francisco. We've got a *ton* of things to cover, so let's get to it!

-   Pivotal Cloud Foundry [now supports Minio](https://pivotal.io/platform/services-marketplace/data-management/minio), a highly available S3-like HA enterprise grade object storage server for developers.
-   [Spring Cloud Finchley GA is scheduled for June 15](https://github.com/spring-cloud/spring-cloud-release/milestones?direction=asc&sort=due_date)
-   In Part 4 of "Should that be a Microservice?" Nate Schutta [talks Independent Scalability and why it's important](https://content.pivotal.io/blog/should-that-be-a-microservice-part-4-independent-scalability)
-   Spring Batch ninja Mahmoud Ben Hassine just announced Spring Batch 4.1.0.M1 which includes, among other thing, [simplified testing and easier remote chunking](https://spring.io/blog/2018/05/31/spring-batch-4-1-0-m1-released). Get the bits now!
-   In last week's *Spring Tips* we looked at JDBC, which is much more than just the venerable `JdbcTemplate`! In this post we look at the `JdbcTemplate`, various SQL command objects, `SimpleFlatMapper`, and so much more. [Check it out!](https://spring.io/blog/2018/05/30/spring-tips-jdbc)
-   Spring Cloud teammember Ryan Baxter has just announced [that Spring Cloud Finchley.RC2 was released!](https://spring.io/blog/2018/05/29/spring-cloud-finchley-rc2-has-been-released)
-   Join Madhura Bhave and Stephane Maldini on June 20th for [this](https://www.meetup.com/sfjava/events/251270797/) San Francisco JUG meetup!
-   This is kind of interesting! It's a [rules-platform that targets Spring Boot, Spring Cloud and other frameworks](https://www.blueriq.com/blueriq-10/)
-   The latest release [of jQAssistant now supports quality control rules for Spring and Spring Boot-based applications](https://jqassistant.org/jqassistant-1-4-0-and-sonarqube-plugin-released/) with the inclusion of the Spring plugin.
-   The Spring I/O team (behind the *epic* Spring I/O conference in Barcelona, Spain, are now [publishing their talks and you should follow that account to get the latest videos](http://twitter.com/spring_io)
-   I really like this post by Spring creator Rod Johnson that describes the spirit of Spring and speaks to the [recent acquisition of Github by Microsoft](https://the-composition.com/developers-developers-developers-microsoft-github-and-the-ascendancy-of-code-7e6ceb51635f?source=userActivityShare-a17df5ec14a4-1528236113)
-   Check out the Vedran Pavic's Spring I/O 2018 talk on Spring Session\]([https://www.youtube.com/watch?v=sJcVaOERo6M&feature=youtu.be](https://www.youtube.com/watch?v=sJcVaOERo6M&feature=youtu.be))

-   In London? You should be at the SpringOne Tour event on 21-22 June in London! Learn about all [things Cloud Native and Spring Boot](https://twitter.com/springcentral/status/1003701222707118080)
-   Apache Kafka 2.0 just had a feature-freeze! [Congrats to the Apache Kafka team](https://twitter.com/matthiasjsax/status/1002077052307095553?s=12)!

BOSH is the driftless package manager distributed systems. [It](http://bosh.io) manages the lifecycle of infrastructure and surfaces a portable API that works, agnostic of the underlying IaaS.
