# Spring for Apache Pulsar 1.2.0 is now available

_Releases | Chris Bono |  November 18, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring for Apache Pulsar 1.2.0 has been released and is now available from Maven Central.

The release will be included in the upcoming Spring Boot [3.4.0](https://github.com/spring-projects/spring-boot/milestones/3.4.0) release.

This release includes numerous enhancements and dependency upgrades.

Please see the release notes for [more details](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.2.0).
