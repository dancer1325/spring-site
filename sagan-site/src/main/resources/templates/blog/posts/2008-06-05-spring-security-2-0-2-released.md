# Spring Security 2.0.2 Released

_Releases | Luke Taylor |  June 05, 2008 | 0 Comments_

Spring Security 2.0.2 is now available.

-   [Download](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=270072&release_id=604674) | [Changelog](http://jira.springframework.org/browse/SEC?report=com.atlassian.jira.plugin.system.project:changelog-panel) | [Announcement](http://forum.springframework.org/showthread.php?t=55434) | [Web Site](http://www.springframework.org/spring-security/)  
    

Spring Security 2.0.2 is mainly a bugfix release and upgrading is recommended for users of earlier versions. There are also some minor extensions to the security namespace to support additional configuration options requested by the community.
