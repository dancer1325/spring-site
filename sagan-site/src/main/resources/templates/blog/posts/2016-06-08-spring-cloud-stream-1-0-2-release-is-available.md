# Spring Cloud Stream 1.0.2.RELEASE is available

_Releases | Marius Bogoevici |  June 08, 2016 | 0 Comments_

Shortly after the [1.0.1.RELEASE announcement](https://spring.io/blog/2016/06/03/spring-cloud-stream-1-0-1-release-is-available) last week we’ve identified a critical and very specific [issue](https://github.com/spring-cloud/spring-cloud-stream/issues/564) concerning multi-binder configurations in cloud environments. As a result, we have decided to follow up with a 1.0.2.RELEASE to handle that and ensure that the fix is available in the upcoming Brixton.SR1 release. The artifacts can be found in [Spring Repository](https://repo.spring.io) as well as [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cspring-cloud-stream).
