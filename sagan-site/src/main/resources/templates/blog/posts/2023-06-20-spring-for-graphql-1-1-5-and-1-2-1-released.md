# Spring for GraphQL 1.1.5 and 1.2.1 released

_Releases | Brian Clozel |  June 20, 2023 | 0 Comments_

I'm pleased to announce that the Spring for GraphQL 1.1.5 and 1.2.1 maintenance versions are now available on Maven Central.

Spring for GraphQL 1.1.5 includes [11 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.1.5). This version will be shipped with Spring Boot 3.0.8, [to be released this Thursday](https://calendar.spring.io/).

Spring for GraphQL 1.2.1 includes [10 fixes and documentation improvements](https://github.com/spring-projects/spring-graphql/releases/tag/v1.2.1). This version will be shipped with Spring Boot 3.1.1, [to be released this Thursday](https://calendar.spring.io/) as well.
