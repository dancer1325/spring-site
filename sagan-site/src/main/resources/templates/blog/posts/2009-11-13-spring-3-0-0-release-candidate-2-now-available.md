# Spring 3.0.0 Release Candidate 2 Now Available

_Releases | Adam Fitzgerald |  November 13, 2009 | 0 Comments_

Fresh from [Arjen Poutsma](http://blog.springsource.com/author/arjenp/)'s [twitter feed](http://twitter.com/poutsma): Spring 3.0.0.RC2 is now available. The release candidate contains lots of fixes to issues raised by community members and has also been updated to support the final versions of JSR-330 "javax.inject" and JSR-303 "javax.validation" APIs.

  

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.0.RC2) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.0.RC2/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.0.RC2/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

  

Thank you to all of the dedicated community members that have given [feedback](http://forum.springsource.org/forumdisplay.php?f=21) and identified [issues](http://jira.springframework.org/browse/SPR). Please keep up the good work as we move closer to GA.
