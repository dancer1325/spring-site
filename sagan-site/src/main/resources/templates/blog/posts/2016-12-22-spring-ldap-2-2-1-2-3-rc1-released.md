# Spring LDAP 2.2.1 & 2.3 RC1 Released

_Releases | Rob Winch |  December 22, 2016 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring LDAP 2.2.1 and 2.3 RC1.

A special thanks to [Mark Paluch](https://twitter.com/mp911de) for getting spring-data-ldap aligned on the Spring Data side!

## [](#spring-ldap-23-rc1)[](#spring-ldap-2-3-rc1)Spring LDAP 2.3 RC1

This release brings in a new era for Spring Data compatibility. Integration for Spring LDAP and Spring Data has been moved to [spring-data-ldap](http://projects.spring.io/spring-data-ldap/) so that it can partake in the Spring Data release train and ensure compatibility with the latest and greatest Spring Data code base. For additional details refer to the [changelog](https://github.com/spring-projects/spring-ldap/milestone/25?closed=1)

## [](#spring-ldap-221)[](#spring-ldap-2-2-1)Spring LDAP 2.2.1

This release contains some minor bug fixes along with an update to make updates to newer versions of Spring Data more seamless. For additional details refer to the [changelog](https://github.com/spring-projects/spring-ldap/milestone/23?closed=1).
