# Spring Security OAuth 2.3.0.RC1 Released

_Releases | Joe Grandja |  February 27, 2018 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security OAuth 2.3.0.RC1.

The [2.3.0.RC1 release](https://github.com/spring-projects/spring-security-oauth/milestone/30?closed=1) adds new support for Elliptic Curve signature verification in `JwkTokenStore`. This release also includes a few minor enhancements and bug fixes.
