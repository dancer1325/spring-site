# This Week in Spring - June 7th, 2016

_Engineering | Josh Long |  June 07, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! It's *already* June! Where. Does. The. Time. GO?? This week I'm in Chicago, IL, for the Chicago Coder Conference, Boston and New Hampshire for customer visits, London, England for Devoxx UK and Talin, Estonia for Geekout EE. If you're around be [sure to say hi](http://twitter.com/starbuxman)! Now then, we've got a lot to cover this week so let's get to it!

-   Spring Cloud Services ninja Scott Frederick just announced the Spring Cloud Cloud Foundry Service Broker, which is a [nice framework for building service brokers for the Cloud Foundry platform](https://spring.io/blog/2016/06/07/introducing-spring-cloud-cloud-foundry-service-broker)
-   Spring Boot and Spring Cloud co-founder (the good) Dr. Dave Syer just put published the [first post in a series on the reactive programming landscape](https://spring.io/blog/2016/06/07/notes-on-reactive-programming-part-i-the-reactive-landscape)
-   Spring Integration ninja Artem Bilan [just announced that Spring Integration Kafka 2.0.0.RC1 is now available](https://spring.io/blog/2016/06/07/spring-integration-kafka-support-2-0-0-release-candidate-is-now-available)
-   not one to rest on his laurels, Artem *also* [announced the Spring for Apache Kafka 1.0. RC1](https://spring.io/blog/2016/06/07/spring-for-apache-kafka-1-0-release-candidate-1-available)
-   Spring Cloud Streams lead [Marius Bogoevici has just announced Spring Cloud Stream 1.0.1.RELEASE](https://spring.io/blog/2016/06/03/spring-cloud-stream-1-0-1-release-is-available). It provides additional configuration options and the ability to contribute bindings without overriding defaults and a slew of small updates.
-   the Reactor project has come a long way, and [we've deprecated some of the older Reactor guides accordingly](https://spring.io/blog/2016/06/02/reactor-guides-deprecated)
-   Spring Integration and Spring AMQP ninja Artem Bilan *also* just announced that [the Spring AMQP 1.6 GA (1.6.0.RELEASE) is now available](https://spring.io/blog/2016/06/01/spring-amqp-1-6-release-available)
-   This blog on [integrating Hibernate to use Spring Expression Language for class-level and cross-parameter constraints](http://javatar81.blogspot.com/2016/06/hibernate-validator-spring-expression.html) is *very interesting*.
-   I really liked this InfoQ post [by Michael Redlich on Spring Statemachine 1.1](https://www.infoq.com/news/2016/06/Spring-Releases-Statemachine)
-   the New Stack has a nice post called [*Cloud Foundry: Built to Survive*](http://thenewstack.io/cloud-foundry-wants-help-destroy-architecture-seriously/)
-   want [a quick introduction to MQTT](https://blog.se.com/industrial-software/2016/06/03/basics-mqtt-messaging-protocol-internet-things/)?
-   this blog on [developing document-centric JAX-WS SOAP-based services with Spring Boot and CXF is interesting](https://blog.codecentric.de/en/2016/06/spring-boot-apache-cxf-testing-soap-webservices/)
