# Spring Boot 2.5.0-M2 available now

_Releases | Phil Webb |  February 19, 2021 | 0 Comments_

Continuing our monthly milestone release cadence, I am pleased to announce that the second milestone of Spring Boot 2.5 has been released and is available from our [milestone repository](https://repo.spring.io/milestone/). This release [adds a number of new features and bug fixes](https://github.com/spring-projects/spring-boot/releases/tag/v2.5.0-M2).

Highlights of this milestone include:

-   Layered WARs support for use with Docker
-   Custom Buildpack Builder Support
-   Jetty 10 Support
-   Early Support for Gradle 7

We've also made some significant behind-the-scenes changes to do with the way that `schema.sql` and `data.sql` files are processed. If you use those features, please try the milestone and let us know if you find any issues.

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.5-M2 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.5.0-M2-Release-Notes) on the wiki and the [updated reference documentation](https://docs.spring.io/spring-boot/docs/2.5.0-M2/reference/html/).

If you want to get started with 2.5 and try out the new features, you can bootstrap a new project on [start.spring.io](https://start.spring.io/#!type=maven-project&language=java&platformVersion=2.5.0.M2&packaging=jar&jvmVersion=11&groupId=com.example&artifactId=demo&name=demo&description=Demo%20project%20for%20Spring%20Boot&packageName=com.example.demo).
