# Spring Security 3.2.7 Released

_Releases | Rob Winch |  March 26, 2015 | 1 Comment_

I'm pleased to announce the release of Spring Security 3.2.7.RELEASE. This release brings a number of [bug fixes](https://jira.spring.io/issues/?jql=project%20%3D%20SEC%20AND%20fixVersion%20%3D%203.2.7) including a fix for a regression in the Active Directory support.
