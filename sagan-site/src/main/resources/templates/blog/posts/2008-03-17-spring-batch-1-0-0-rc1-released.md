# Spring Batch 1.0.0.rc1 Released

_Releases | Ben Hale |  March 17, 2008 | 0 Comments_

I'm pleased to announce that Spring Batch 1.0.0.rc1 has been released.  You can access this release via the Spring Maven Milestone Repository ([browse](http://s3browse.com/explore/maven.springframework.org/milestone/org/springframework/batch)) or via the [Download Page.](http://static.springframework.org/spring-batch)  
  
This is the first release candidate for the Spring Batch 1.0.0 release with an anticipated final release on 28 March.  The major changes for this release are:  

-   Improvements in [Reference Documentation](http://static.springframework.org/spring-batch/spring-batch-docs/index.html)
-   Reorganization of packaging structure in spring-batch-infrastructure and spring-batch-core
-   Merging of the spring-batch-core and spring-batch-execution modules

Please see the [changelog](http://jira.springframework.org/browse/BATCH/fixforversion/10630) for details.
