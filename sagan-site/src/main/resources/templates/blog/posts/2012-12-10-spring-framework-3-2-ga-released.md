# Spring Framework 3.2 GA Released

_Releases | Chris Beams |  December 10, 2012 | 0 Comments_

The Spring team are very happy to announce that Spring Framework 3.2 GA is now released and available via both the [SpringSource repository](https://github.com/SpringSource/spring-framework/wiki/Downloading-Spring-artifacts#wiki-resolving-spring-artifacts) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22).

See Juergen's [blog post](http://bit.ly/VCH9Wh) for complete details!
