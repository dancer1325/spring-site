# This Week in Spring - April 19th, 2016

_Engineering | Josh Long |  April 19, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in Stuttgart and Mainz, Germany and Paris, France, speaking to customers and at the [JAX conference](http://jax.de), then it's off to Paris, France, to [speak at the Devoxx France edition](http://www.devoxx.fr/). If you're around in any of these places don't hesitate to reach out I'd love to say hello.

Also, tonight, I'll be joining my friends from industry (Daniel Bryant, Markus Eisele, and Simon Maple) for the [ZeroTurnaround](http://twitter.com/zeroturnaround) webinar, [*Microservices for the Enterprise*](http://zeroturnaround.com/webinar). There are already more than 3,000 people registered and I hope to see you there, as well!

As usual, we've got a lot to look at today, so let's get started!

-   the BIG news this week: [**Spring Boot 1.4.0.M2 is now available**](https://spring.io/blog/2016/04/13/spring-boot-1-4-0-m2-available-now)! This release is *packed* with new features (some of which we talk about in other blogs this week), so make sure to get the bits, kick the tires and feedback!
-   the biggest event of the year is, without a doubt, [SpringOne Platform](https://spring.io/blog/2016/04/15/springone-platform), and we want to see you there!
-   Spring Integration lead Artem Bilan has just announced Spring Integration AWS 1.0.0.M1, [supporting Amazon Web Services integrations](https://spring.io/blog/2016/04/14/spring-integration-aws-1-0-0-m1-is-now-available)
-   Our pal Simon Baslé, [from Couchbase](http://twitter.com/couchbase) - makers of the epic document database, cross-posted a *very* [cool look at using Spring Data Couchbase, a first-class citizen in Spring Boot 1.4](https://spring.io/blog/2016/04/14/couchbase-as-a-first-class-citizen-of-spring-boot-1-4)
-   the good [Dr. David Syer looks at how to override platform-and-framework specified versions](https://spring.io/blog/2016/04/13/overriding-dependency-versions-with-spring-boot) in Spring Boot and Spring Cloud in this really fascinating article that plumbs the depths of dependency management so that you don't have to!
-   Spring Boot lead and testing enthusiast Phil Webb [debuts some of the upcoming Spring Boot 1.4's *numerous* new features supporting ease-of-testing](https://spring.io/blog/2016/04/15/testing-improvements-in-spring-boot-1-4)
-   our very own Jared Gordon put together a nice case-study [on refactoring a monolith to a microservices-based system using Spring Boot and Spring Cloud](https://blog.pivotal.io/pivotal-cloud-foundry/features/case-study-refactoring-a-monolith-into-a-cloud-native-app-part-4)
-   Alexander Lukyanchikov put together an interesting [post on building a microservices-based system using Spring Cloud and Docker](https://dzone.com/articles/microservice-architecture-with-spring-cloud-and-do)
-   Zoltan Altfatter is back at it again, [this time with a look at the features in Spring Boot 1.4](http://zoltanaltfatter.com/2016/04/16/trying-out-spring-boot-1.4.0-new-features-and-enhancements/)
-   our other pal Michael Simons is back at it again this week, *also* [with a look at the features in Spring Boot 1.4](http://info.michael-simons.eu/2016/04/17/8-new-features-youll-get-with-spring-boot-1-4/)
