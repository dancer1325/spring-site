# Spring Framework 3.2.16 available now

_Releases | Stéphane Nicoll |  December 17, 2015 | 0 Comments_

After [4.2.4 and 4.1.9 earlier today](https://spring.io/blog/2015/12/17/spring-framework-4-2-4-4-1-9-released), it is my pleasure to announce that Spring Framework [3.2.16 is now available](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=15402).
