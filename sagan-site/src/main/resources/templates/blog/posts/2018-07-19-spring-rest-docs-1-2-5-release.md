# Spring REST Docs 1.2.5.RELEASE

_Releases | Andy Wilkinson |  July 19, 2018 | 0 Comments_

It's my pleasure to announce that [Spring REST Docs](https://github.com/spring-projects/spring-restdocs) 1.2.5.RELEASE is available from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release). My thanks to everyone who contributed to this release by reporting bugs and opening pull requests.

This maintenance release includes a [handful of bug fixes and documentation improvements](https://github.com/spring-projects/spring-restdocs/issues?q=milestone%3A1.2.5.RELEASE). It is a recommend upgrade for all Spring REST Docs 1.x users.
