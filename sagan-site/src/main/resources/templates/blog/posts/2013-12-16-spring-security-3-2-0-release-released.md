# Spring Security 3.2.0.RELEASE Released

_Releases | Rob Winch |  December 16, 2013 | 0 Comments_

I am pleased to announce the release of Spring Security 3.2.0.RELEASE. You can view the highlights of this release within the [What’s new in Spring Security 3.2](http://docs.spring.io/spring-security/site/docs/3.2.x/reference/htmlsingle/#new) section of the reference. A list of [changes since 3.1.x](https://jira.springsource.org/issues/?jql=project%20%3D%20SEC%20AND%20fixVersion%20in%20\(%223.2.0%22%2C%20%223.2.0.RC2%22%2C%20%223.2.0.RC1%22%2C%20%223.2.0.M2%22%2C%20%223.2.0.M1%22\)%20ORDER%20BY%20priority%20DESC%2C%20reporter%20ASC%2C%20issuetype%20DESC) and [since the last release](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10040&version=11503) can be found within JIRA's change logs.

To learn more about Spring Security 3.2, I'd like to encourage you to view the new [Spring Security guides](http://docs.spring.io/spring-security/site/docs/3.2.x/guides/) and to attend the free [Spring Security 3.2 Webniar](https://spring.io/blog/2013/12/06/webinar-spring-security) on January 16th.

Please consider helping us [spread the word on Twitter](https://twitter.com/rob_winch/status/412701647190491136), ask questions on [Stackoverflow with the spring-security tag](http://stackoverflow.com/questions/tagged/spring-security) and log any issues to the [Spring Security JIRA](jira.springsource.org/browse/SEC).
