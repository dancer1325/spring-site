# Spring Tools 4.4.1 released

_Releases | Martin Lippert |  October 24, 2019 | 0 Comments_

Dear Spring Community,

I am happy to announce the 4.4.1 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

Highlights from this release include:

-   *(Spring Boot)* introducing the all-new mechanism to connect and disconnect to/from running Spring applications to show live data - take a look at the [wiki](https://github.com/spring-projects/sts4/wiki/Live-Application-Information#managing-live-data-connections-to-local-processes) for details
-   *(Spring Boot)*, *(Eclipse)* enhancement: Goto Symbol now also works from XML bean files
-   *(Spring Boot)*, *(Eclipse)* fixed: Completion in `application.properties` sometimes leaves some 'garbage' text
-   *(Spring Boot)*: improve performance of xml symbol scanning
-   *(Spring Boot)*: updated lsp4xml to 0.9.1 to address ([CVE-2019-18212](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2019-18212)) + ([CVE-2019-18213](https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2019-18213))
-   *(Spring Boot)*: improve performance of xml symbol scanning
-   *(Concourse)* enhancement: added support for `registry-image` resource type ([#380](https://github.com/spring-projects/sts4/issues/380))

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Detailed changes can be found here: [https://github.com/spring-projects/sts4/wiki/Changelog#2019-10-24-441-release](https://github.com/spring-projects/sts4/wiki/Changelog#2019-10-24-441-release)

Spring Tools 4.4.2 is scheduled to be released in late November 2019.

Enjoy!
