# Spring Shell 4.0.1 is out!

_Releases | Mahmoud Ben Hassine |  January 31, 2026 | 0 Comments_

On behalf of the team and all contributors, I am pleased to announce that Spring Shell `4.0.1` is now available from Maven Central!

## [](#whats-new-in-spring-shell-401)What's New in Spring Shell 4.0.1

Spring Shell 4.0.1 is a patch release that addresses several issues reported by the community since the release of Spring Shell 4.0.0. This release includes important bug fixes and improvements to enhance the overall stability and performance of the framework.

You can find the complete list of changes in the [4.0.1 release notes](https://github.com/spring-projects/spring-shell/releases/tag/v4.0.1).

## [](#feedback)Feedback

Many thanks to all contributors who made this release possible! We encourage you to try Spring Shell 4.0.1 out and provide your feedback on [GitHub Issues](https://github.com/spring-projects/spring-shell/issues) and [GitHub Discussions](https://github.com/spring-projects/spring-shell/discussions).
