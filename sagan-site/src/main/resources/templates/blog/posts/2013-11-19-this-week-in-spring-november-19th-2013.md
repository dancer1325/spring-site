# This Week in Spring - November 19th, 2013

_Engineering | Josh Long |  November 19, 2013 | 0 Comments_

Welcome to another edition of *This Week in Spring*! As usual, we've got a lot to cover, so let's get to it!

I want to encourage everyone to join me on the 21st (this Thursday!) for my webinar, [*Have You Seen Spring Lately?*](http://spring.io/blog/2013/10/16/nov-21st-webinar-have-you-seen-spring-lately). If you haven't been paying very deep attention to everything that's happening in the wide world of Spring since we became part of the [Pivotal adventure](http://gopivotal.com), this is the one to watch! And even if you have, I'll bet you'll learn about something new and exciting, anyway. :) C'ya there!

1.  We published an article - [*Have You Seen Spring Lately?*](http://blog.gopivotal.com/products/have-you-seen-spring-lately) - to accompany the webinar today on the Pivotal blog.
2.  Mark Baars has just announced that we've just added [*Big Data with Hadoop and Pivotal HD*](http://spring.io/blog/2013/11/19/processing-big-data-with-hadoop-and-pivotal-hd) to our training. Awesome!
3.  Spring Data ninja Thomas Risberg has just announced that the [1.0.2.GA release of Spring for Apache Hadoop](http://spring.io/blog/2013/11/14/spring-for-apache-hadoop-1-0-2-ga-and-2-0-m2-released) is now available!
4.  Spring XD ninja Gunnar Hillert has just announced that [Spring XD 1.0.0.M4 has just been released](http://spring.io/blog/2013/11/14/spring-xd-1-0-0-m4-released). The new release is packed with a *lot* of new things, so definitely check this release out!
5.  Spring Data lead Oliver Gierke and Spring Data Couchbase lead Michael Nitschinger have just announced the availability [of milestone two for the Spring Data Couchbase module](http://spring.io/blog/2013/11/18/spring-data-couchbase-1-0-m2-released)! Michael also followed up with an initial, but promising, pull request for [Spring Boot support for Spring Data Couchbase, too](https://github.com/spring-projects/spring-boot/pull/124)! :)
6.  Excited about the [recent Reactor 1.0 release](http://spring.io/blog/2013/11/12/it-can-t-just-be-big-data-it-has-to-be-fast-data-reactor-1-0-goes-ga) and want to learn more? Check out [the upcoming webinar featuring Reactor lead Jon Brisbin on December 3, 2013](http://spring.io/blog/2013/11/13/webinar-reactor)! I'll be there, eagerly listening, and I hope to see you there, too.
7.  Spring and Cloud Foundry ninja Matt Stine's doing a webinar on December 10th called *Spring with Immutability* that will introduce how [Spring can help with item #15 in Josh Bloch's canonincal *Effective Java* book](http://spring.io/blog/2013/11/13/webinar-spring-with-immutability): **minimize mutability**.
8.  The replay for the SpringOne2GX2013 session from New Relic: [*Troubleshooting Live Java Web Applications* is now available online](http://spring.io/blog/2013/11/13/springone2gx-2013-replay-troubleshooting-live-java-web-applications).
9.  [Spring Data GemFire 1.3.3 has been released!](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10462&version=14257). Don't quite have a release note for it, yet, though.
10.  Check out this [demo of Pivotal CF - the turnkey Cloud Foundry in a box for on-premises deployment - in action](https://www.youtube.com/watch?v=TLhST6Hmiso&utm_content=buffer943d3&utm_source=buffer&utm_medium=twitter&utm_campaign=Buffer).
11.  Using Redis? We love Redis! We found this great post from a few years ago with [various uses cases readily solved with Redis](http://highscalability.com/blog/2011/7/6/11-common-web-use-cases-solved-in-redis.html?utm_content=buffer7cadf&utm_source=buffer&utm_medium=twitter&utm_campaign=Buffer). Enjoy!
12.  Pinterest [just launched their first web API](http://techcrunch.com/2013/11/14/pinterest-launches-its-first-apis-partners-with-zappos-walmart-disney-nestle-random-house-hearst-on-first-rollout/). There's nothing Spring-related about this mention, yet, but I'm just throwing this out there because I'm taking bets on how soon we'll see someone in the community create a Spring Social Pinterest binding! :)
13.  Friendly reminder: you can get a *lot* of cool Spring / Pivotal stuff from [Homebrew](http://brew.sh/) to flesh out your Java stack:
     
     `    $ brew tap pivotal/tap;    $ brew install springboot springxd rest-shell maven groovy redis rabbitmq tomcat  `
     
14.  I really enjoyed this sort of [look behind the curtain with Spring team engineer Greg Turnquist](http://blog.greglturnquist.com/2013/11/a-day-in-life-of-one-spring-developer.html).
15.  The Spring Exchange, put on by our friends at Skills Matter, took place in in London recently. There were many great talks, and most (all?) of [it has been recorded and put online](http://skillsmatter.com/event/java-jee/spring-exchange-1724)! Enjoy!
