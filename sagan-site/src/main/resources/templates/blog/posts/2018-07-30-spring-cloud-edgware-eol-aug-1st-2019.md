# Spring Cloud Edgware EOL Aug 1st 2019

_News | Spencer Gibb |  July 30, 2018 | 0 Comments_

Spring Cloud Edgware will follow the [Spring Boot 1.x EOL announcement](https://spring.io/blog/2018/07/30/spring-boot-1-x-eol-aug-1st-2019). We will cease maintenance of the Edgware release train twelve months from today, on Aug 1st 2019.

We will keep publishing occasional Edgware service releases up until that point and will then end the release train.

As [a reminder](https://spring.io/blog/2018/06/19/spring-cloud-finchley-release-is-available#lifecycle-announcements-for-release-train-versions), the Camden release train has reached EOL, and Dalston will reach EOL at the end of December 2018.

Existing Spring Cloud users should plan accordingly to ensure that they have upgraded to the latest Finchley+ version before that date. Also, please note that Spring Cloud Finchley and later requires Spring Boot 2.x, which requires Java 8+. So, if you're running Java 6 or 7 you should also plan on upgrading your JDK.
