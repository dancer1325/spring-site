# SpringOne 2GX 2010 Tech Talk: Developing Social-Ready Web Applications

_News | Adam Fitzgerald |  February 25, 2011 | 0 Comments_

[![SpringOne 2GX 2010 Tech Talk: Developing Social-Ready Web Applications](/files/SpringOne2GX2010CraigWalls.png)](http://www.infoq.com/presentations/Developing-Social-Ready-Web-Apps)

More content has been posted by [InfoQ](http://www.infoq.com/SpringOne_2010) covering the talks from [SpringOne 2GX 2010](http://www.springone2gx.com). This time is it [Spring in Action](http://www.manning.com/walls2/) author, [Craig Walls](http://blog.springsource.com/author/cwalls/), talking about [Developing Social-Ready Web Applications](http://www.infoq.com/presentations/Developing-Social-Ready-Web-Apps).

Craig covers:

-   The types of social integration possible for your Spring applications
-   How to access social data via REST
-   How to secure social data in appropriate ways to protect your users
-   The features available in the [Spring Social](http://www.springsource.org/spring-social) project to support social integration within the Spring idiom.

Once again, many thanks to InfoQ for [covering SpringOne 2GX 2010](http://www.infoq.com/SpringOne_2010) and providing this outstanding presentations to the community.
