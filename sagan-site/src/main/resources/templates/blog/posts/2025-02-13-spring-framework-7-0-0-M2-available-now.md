# Spring Framework 7.0.0-M2 Available Now

_Releases | Stéphane Nicoll |  February 13, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce the second milestone of the next Spring Framework generation.

If [you've missed it](https://spring.io/blog/2025/01/21/milestones-to-central), we now ship milestones to the usual repo.spring.io artifact repository, but also to Maven Central.

The second milestone refines the work we've devlived in [7.0.0-M1](https://spring.io/blog/2025/01/23/spring-framework-7-0-0-M1-available-now), in particular arround our null safety support using JSpecify. Check [the changelog](https://github.com/spring-projects/spring-framework/releases/tag/v7.0.0-M2) for more details.
