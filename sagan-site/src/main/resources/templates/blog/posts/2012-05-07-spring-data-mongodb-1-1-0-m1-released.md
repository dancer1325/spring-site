# Spring Data MongoDB 1.1.0 M1 released

_Releases | Oliver Drotbohm |  May 07, 2012 | 0 Comments_

I'd like to announce the availability of Spring Data MongoDB 1.1.0 M1. This is the first milestone of the 1.1.0 branch and comes with a hand full of new features:

-   Support for GridFS ([DATAMONGO-6](http://jira.springsource.org/browse/DATAMONGO-6))
-   Support for JSR-303 validation ([DATAMONGO-36](http://jira.springsource.org/browse/DATAMONGO-36) - community contribution)
-   Support for new repository keywords: StartsWith, EndsWith, Contains, After, Before ([DATAMONGO-418](http://jira.springsource.org/browse/DATAMONGO-418), [427](http://jira.springsource.org/browse/DATAMONGO-427))
-   CDI integration for repositories ([DATAMONGO-356](http://jira.springsource.org/browse/DATAMONGO-356))

[Downloads](http://www.springsource.com/download/community?project=Spring%20Data%20MongoDB&version=1.1.0.M1) | [JavaDocs](http://static.springsource.org/spring-data/data-mongodb/docs/1.1.0.M1/api/) | [Reference Documentation](http://static.springsource.org/spring-data/data-mongodb/docs/1.1.0.M1/reference/html/) | [Changelog](http://static.springframework.org/spring-data/data-mongodb/docs/1.1.0.M1/changelog.txt)

The release is available from our [Maven repository](http://repo.springsource.org/milestone/org/springframework/data/spring-data-mongodb/1.1.0.M1). To learn more about the project, visit the Spring Data [MongoDB Page](http://www.springsource.org/spring-data/mongodb). Looking forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?f=80) or in the [issue tracker](https://jira.springsource.org/browse/DATAMONGO).
