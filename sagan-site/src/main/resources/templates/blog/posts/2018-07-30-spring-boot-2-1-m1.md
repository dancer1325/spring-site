# Spring Boot 2.1 M1

_Releases | Andy Wilkinson |  July 30, 2018 | 0 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the first milestone of Spring Boot 2.1 has been released and is available from [our milestone repository](http://repo.spring.io/milestone/). This release closes over [180 issues and pull requests](https://github.com/spring-projects/spring-boot/milestone/99?closed=1)!

This first milestone builds on and requires [Spring Framework 5.1.RC1](https://spring.io/blog/2018/07/26/spring-framework-5-1-goes-rc1). It also builds on recent releases of a number of other Spring projects including [Spring Data Lovelace RC1](https://spring.io/blog/2018/07/26/spring-data-lovelace-rc1-available) and [Spring Security 5.1 M2](https://spring.io/blog/2018/07/30/spring-security-5-1-0-m2-released) to name just two.

Other highlights of this first milestone include:

-   Servlet 4 support with Undertow 2.0 and Tomcat 9.0
-   Caches endpoint
-   Support for querying an individual component's health
-   Support for [profile expressions](https://docs.spring.io/spring-framework/docs/5.1.0.RC1/spring-framework-reference/core.html#beans-definition-profiles) in multi-profile YAML documents and `<springProfile>` Logback configuration.

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.1 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.1-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.1.0.M1/reference/htmlsingle/).

If you want to get started with 2.1 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).
