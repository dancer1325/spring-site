# Virgo Milestone M03 Ships

_Engineering | Christopher Frost |  August 10, 2010 | 0 Comments_

Virgo 2.1.0.M03-incubator is available from the Virgo [website](http://www.eclipse.org/virgo/download/). The changes include:

-   Further performance improvements, especially for Windows users.
-   Update the version of Logback to 0.9.24 and SLF4J to 1.6.1
-   Minor bug fixes

The performance improvement was achieved by adjusting the log level of entry exit tracing.

As a result of the new version of Logback the syntax of the configuration file 'serviceability.xml' has changed slightly. More information can be found [here](http://logback.qos.ch/codes.html#layoutInsteadOfEncoder).

We have now published a [plan](http://wiki.eclipse.org/Virgo/Future#Virgo_Plan) outline for the future of Virgo so you have a better idea of where we are going.
