# Spring Framework 6.1.3 and 6.0.16 available now

_Releases | Stéphane Nicoll |  January 11, 2024 | 0 Comments_

**Updates**

-   **\[01-22\]** These versions include a fix for [CVE-2024-22233](https://spring.io/security/cve-2024-22233).

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.3` and `6.0.16` are available now.

Spring Framework `6.1.3` ships with [87 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.3), including [8 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.1.3+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.2.2, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.0.16` ships with [14 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.16). This version will be shipped with Spring Boot 3.1.8, [to be released next week](https://calendar.spring.io/).
