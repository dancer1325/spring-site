# SPRING INTEGRATION 2.1.3.RELEASE AND 2.2.0.M3 ARE RELEASED!

_Releases | Gary Russell |  July 02, 2012 | 0 Comments_

The Spring Integration team is pleased to announce the release of:

**Spring Integration 2.1.3.RELEASE** - a small maintenance release with some important bug fixes

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=13002)
-   Artifacts are available in the [SpringSource repo](http://repo.springsource.org/simple/libs-release-local/org/springframework/integration/), [the community download page](http://www.springsource.org/download/community?project=Spring%2520Integration) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Corg.springframework.integration)

**Spring Integration 2.2.0.M3** - The third milestone release of the 2.2 stream, including a number of important features and improvements

-   Release notes are available [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=13000)
-   Artifacts are available in the [SpringSource repo](http://repo.springsource.org/simple/libs-release-local/org/springframework/integration/) and [the community download page](http://www.springsource.org/download/community?project=Spring%2520Integration)

More information is available on the project's [home page](http://www.springsource.org/spring-integration)
