# First Virgo Milestone Ships

_Engineering | Glyn Normington |  June 17, 2010 | 0 Comments_

[![Virgo from EclipseRT](http://www.eclipse.org/virgo/images/virgo-logo.png)](http://www.eclipse.org/virgo/) The first milestone (2.1.0.M01) of Eclipse [Virgo](http://www.eclipse.org/virgo/) is now [available for download](http://eclipse.org/virgo/download/) under the [Eclipse Public License](http://www.eclipse.org/legal/epl-v10.html). This consists of an application server, known as the Virgo Web Server, and a stand-alone kernel.

The goal of this milestone is for [dm Server](http://www.springsource.org/dmserver) 2.0.x users to be able to migrate to it relatively painlessly and have an equally stable environment. SpringSource is offering commercial support for Virgo and we encourage all dm Server users to migrate to Virgo. The main communication channel with users is now the [Virgo forum](http://www.eclipse.org/forums/index.php?t=thread&frm_id=159). There is also a [Virgo developer](https://dev.eclipse.org/mailman/listinfo/virgo-dev) mailing list and a weekly [Virgo community call](http://wiki.eclipse.org/Virgo/Discussion/Meetings).

This is a major step for the Virgo project and demonstrates that the dm Server code contribution is complete. Users who want to package and distribute Virgo can now do so under the terms of the Eclipse Public License.

Documentation is [available online](http://www.eclipse.org/virgo/documentation/) in the form of a [User Guide](http://www.eclipse.org/virgo/documentation/2.1.0.M01/user-guide/index.html), a [Programmer Guide](http://www.eclipse.org/virgo/documentation/2.1.0.M01/programmer-guide/index.html), and a draft ["getting started guide"](http://www.eclipse.org/virgo/documentation/2.1.0.M01/getting-started-guide/index.html). Then there is a growing body of developer-oriented information -- including [a conceptual overview](http://wiki.eclipse.org/Virgo/Concepts), [a source code directory](http://wiki.eclipse.org/Virgo/Source), and [build tips](http://wiki.eclipse.org/Virgo/Build) -- in the [Virgo wiki](http://wiki.eclipse.org/Virgo).

Although the Eclipse-based dm Server application development tools will be contributed to Eclipse in due course, Virgo support has been added to the [SpringSource Tool Suite](http://www.springsource.com/products/sts) in the interim. This is available by downloading STS 2.3.2 and installing the latest dm Server tools from the update site at [http://www.springsource.com/snapshot/e3.5](http://www.springsource.com/snapshot/e3.5).

##### Background

In January Adrian Colyer [announced](http://blog.springsource.com/2010/01/12/dm-server-project-moves-to-eclipse-org/) our intention to donate SpringSource dm Server to Eclipse. The [project proposal](http://www.eclipse.org/proposals/virgo/) which was approved in February and the Virgo project created. Since then we have set up source code repositories, created a wiki, and tweaked the code in preparation for a formal release with full Eclipse legal approval. In May dm Server won the Eclipse Community Award for [Best EclipseRT application](http://www.springsource.org/node/2480), which is particularly fitting since Virgo is a subproject of EclipseRT.

Today's milestone demonstrates our continuing commitment to OSGi and to Virgo, the future of the dm Server project.
