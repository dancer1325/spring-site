# Spring for Apache Kafka 3.3.7 and 3.2.10 are Available Now

_Releases | Soby Chacko |  June 18, 2025 | 0 Comments_

On behalf of the team and everyone who contributed, we are pleased to announce that Spring for Apache Kafka `3.3.7` and `3.2.10` have been released and are available from Maven Central.

We extend our gratitude to all contributors who made these releases possible.

With the `3.2.10` release, we have reached the Open Source End of life for the `3.2.x` generation of Spring for Apache Kafka. Please see the [support timeline](https://spring.io/projects/spring-kafka#support) for more details. Please consider upgrading to the latest `3.3.x` version for ongoing OSS support.

#### [](#release-notes)Release Notes

See the release notes for more details on what's included in these releases:

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.7](https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.7)

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.10](https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.10)

#### [](#spring-boot-integration)Spring Boot Integration

The `3.3.7` and `3.2.10` releases will be integrated into the upcoming Spring Boot `3.4.7` and `3.3.13` releases respectively. In addition, version `3.3.7` will also be included in the upcoming Spring Boot `3.5.1` release.
