# Spring Cloud Square 0.4.0-M2 is available.

_Engineering | Olga Maciaszek-Sharma |  October 20, 2021 | 0 Comments_

On behalf of the community, I am pleased to announce that the Milestone 2 (M2) of the [Spring Cloud Square](https://cloud.spring.io) 0.4.0 is available today. The release can be found in [Spring Milestone](https://repo.spring.io/milestone/) repository. You can check out the 0.4.0-M2 [release notes for more information](https://github.com/spring-cloud-incubator/spring-cloud-square/releases/tag/v0.4.0-M2).

See all issues included in this release [here](https://github.com/spring-cloud-incubator/spring-cloud-square/milestone/1?closed=1).

## [](#notable-changes-in-040-m2)Notable Changes in 0.4.0-M2

-   Spring Native support adjustments for non-load-balanced use-cases ([PR](https://github.com/spring-cloud-incubator/spring-cloud-square/pull/23))
-   Support for configuring different WebClient.Builder beans (not shared) for each `@RetrofitClient` instance ([PR](https://github.com/spring-cloud-incubator/spring-cloud-square/pull/30))

The documentation for this milestone can be found [here](https://spring-cloud-incubator.github.io/spring-cloud-square/docs/current/reference/html/index.html).
