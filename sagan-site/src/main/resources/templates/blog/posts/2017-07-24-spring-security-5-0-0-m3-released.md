# Spring Security 5.0.0 M3 Released

_Releases | Joe Grandja |  July 24, 2017 | 3 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.0.0 M3. This release includes bug fixes, new features, and is based off of Spring Framework 5.0.0 RC3.

The new features included in this milestone are:

-   Support added for [JSON Web Token (JWT)](https://tools.ietf.org/html/rfc7519) and [JSON Web Signature (JWS)](https://tools.ietf.org/html/rfc7515) via the new module ***spring-security-jwt-jose***
    
-   Integrated the [ID Token](http://openid.net/specs/openid-connect-core-1_0.html#IDToken) for OpenID Connect authentication flows
    

Please refer to the [change log](https://github.com/spring-projects/spring-security/milestone/100?closed=1) for further details on this release.
