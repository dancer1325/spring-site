# This Week in Spring, May 1, 2012

_Engineering | Josh Long |  May 01, 2012 | 0 Comments_

Welcome to another installment of *This Week in Spring*! I'm writing the back of the room during [Adrian Colyer](http://blog.springsource.org/author/adrianc/)'s amazing keynote at [SpringOne On The Road - London](http://www.springsource.org/springone-on-the-road/london) event. ![](http://www.springsource.org/files/adrian_keynote_0.jpg)

    Also, be sure to check out part 2, this Wednesday, May 3rd for both [Europe](http://www.springsource.org/node/3525) and [North America](http://www.springsource.org/node/3526)!

2.  Blogger *Daniel Sawano* has put together two blogs introducing how to use Spring MVC with Scala. The first blog introduces a simple Scala-Java hybrid [Spring MVC application setup](http://software.sawano.se/2012/04/scala-spring-mvc-true.html), and the second blog introduces how to [build a more advanced Spring MVC example with RESTful endpoints using Scala](http://software.sawano.se/2012/04/scala-spring-mvc-true-part-2.html). Nice job, Daniel!
3.  Our pal *Jan Machacek* over at Cake Solutions has put together a *really* cool blog on how to use [Akka (an actor framework), Spray (an HTTP framework on top of Akka) and Spring together](http://www.cakesolutions.net/teamblogs/2012/04/28/akka-spray-and-spring-framework/).
4.  Blogger *mkyong* has a nice introduction on how to use Google App Engine to setup [a RESTful Spring MVC application on Google App Engine](http://www.mkyong.com/google-app-engine/google-app-engine-spring-3-mvc-rest-example/).

5.  Blogger *Maciej Walkowiak* has a fanastic blog on how to force Spring Data MongoDB to cascade saves [to related document objects annotated with `@DbRef`](http://maciejwalkowiak.pl/blog/2012/04/30/spring-data-mongodb-cascade-save-on-dbref-objects/).
6.  Over on the Rai Studies blog, there's a few good blog posts on setting up [Spring MVC3 CRUD applications using MyBatis 3](http://www.raistudies.com/spring/spring-mvc/spring-mvc-3-crud-example-with-mybatis-3/).
7.  Our friend *Tobias Trelle* has another installment of his fantastic series introducing Spring Data. This [post introduces Redis](http://blog.codecentric.de/en/2012/04/spring-data-redis/).
8.  The Thymeleaf project, and *Soraya Sánchez*, in particular, has put together a wonderful example showing [how to rework the classic Spring PetClinic sample application to use the powerful Thymeleaf templating engine](http://www.thymeleaf.org/petclinic.html).
9.  Tomcat engineer [Filip Hanik](http://www.tomcatexpert.com/users/fhanik) has put together a wonderful blog introducing how to [use WebSockets in Apache Tomcat 7](http://www.tomcatexpert.com/blog/2012/04/24/websockets-tomcat-7).

11.  Our friend *Willie Wheeler*, author of Manning's *Spring in Practice*, has a great post on using [Spring Data JPA to create dynamic DAOs and queries](http://springinpractice.com/2012/04/24/autogenerate-daos-and-queries-using-spring-data-jpa/).
12.  *Billy Sjöberg* has a great blog on using [Spring Integration to bridge AMQP and JMS](http://java.dzone.com/articles/bridging-between-jms-and).
13.  Over on Xebia's Xebee blog, *Anita Parida* has put together a great blog introducing how to setup [LDAP authentication using Spring Security](http://xebee.xebia.in/2012/04/30/ldap-authentication-using-spring-security/).
14.  The CandidJava blog has a very... *candid* look at how to setup [file uploads in Spring MVC that map to a domain model in the backend](http://candidjava.com/spring-mvc-fileupload-example-prgram-eclipse).
15.  Check out this blog if you want a narrated example introducing [how to do Spring MVC 3.0, and Hibernate, with annotations](http://chenying99.iteye.com/blog/1498054).
16.  AMQP's a powerful technology, and RabbitMQ, the leading AMQP-based message broker, is even more so. However, one common question I get is: what other implementations are there? This blog [has the top 5 implementations](http://www.fromdev.com/2012/04/top-5-open-source-amqp-advanced-message.html) (though, clearly not in order, because [RabbitMQ](http://www.rabbitmq.com/) would have to be #1, otherwise!)
17.  Want to manage JTA transactions on JBoss AS 7 using Spring and Hibernate? This *gist* by [ggerard](https://github.com/ggerard) shows [one possible configuration](https://gist.github.com/2484231).
