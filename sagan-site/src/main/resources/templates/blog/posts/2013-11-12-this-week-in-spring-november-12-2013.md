# This Week in Spring - November 12, 2013

_Engineering | Josh Long |  November 12, 2013 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week, I'm in Antwerp, Belgium talking to Spring, RabbitMQ, Tomcat, Redis, and Cloud Foundry users at [Devoxx Belgium](http://devoxx.be). We at Pivotal have a booth and are showing Spring Boot and giving away t-shirts. Be sure to stop by and say hi!

1.  [Spring Boot 0.5.M6 is now available](https://spring.io/blog/2013/11/10/spring-boot-0-5-0-m6-released)! The new release is full of amazing new features and you can use it from `gvm` and Homebrew, too.
2.  [The Reactor project 1.0 has just been released](https://spring.io/blog/2013/11/12/it-can-t-just-be-big-data-it-has-to-be-fast-data-reactor-1-0-goes-ga)! Project lead Jon Brisbin has done an amazing job with the new release and the blog is full of win, so be sure to check it out.

-   In that vein, I put together a post [introducing Spring Boot M6's new *awesome* SSH support](http://www.joshlong.com/jl/blogPost/spring_boot_m6s_new_security_featur.html), complete with sample code and a screencast.
-   [Spring Framework 3.2.5 is now available!](https://spring.io/blog/2013/11/07/spring-framework-3-2-5-released)
-   the replay of the webinar, [*Inside Cloud Foundry - an Arcitectural Review*](https://spring.io/blog/2013/11/08/springone2gx-2013-replay-inside-cloud-foundry-an-architectural-review) from SpringOne2GX 2013 is now online!
-   Today [we took the wraps off of Pivotal One](http://blog.gopivotal.com/news-2/pivotal-one-launches-now-every-enterprise-can-develop-like-an-internet-giant) - the cohesive set of Pivotal technologies that aims to significantly accelerate how companies build big data apps - and [Pivotal CF](http://blog.cloudfoundry.com/2013/11/12/the-platform-for-building-great-software/), the turnkey, integrated version of Cloud Foundry. Definitely exciting news and this all, of course, bodes well for Spring users who want an open, portable cloud! I [liked Wired's coverage of the event](http://www.wired.com/wiredenterprise/2013/11/pivotal-one/all/), too.
-   The JavaBeat blog has a nice post [on bean inheritance in Spring](http://www.javabeat.net/2013/11/bean-definition-inheritance-spring/)
-   Spring MVC ninja Rossen Stoyanchev just put up a fantastic post [on Spring 4's websockets support](http://blog.gopivotal.com/products/websocket-architecture-in-spring-4-0). Defintely worth a read!
-   Also, be sure to see Rossen, Dr. David Syer, Juergen Hoeller, Rob Harrop, Oliver Gierke, and more at the [Spring eXchange](http://skillsmatter.com/event/java-jee/spring-exchange-1724) in London Thursday the 14th, 2013.
-   Have you seen our new [Spring Training Page](http://gopivotal.com/training)? It features all the available Spring trainings and you can see a list of upcoming trainings.
