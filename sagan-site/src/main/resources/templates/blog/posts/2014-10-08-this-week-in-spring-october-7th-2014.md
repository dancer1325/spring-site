# This Week in Spring  - October 7th, 2014

_Engineering | Josh Long |  October 08, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week, I've been at the Couchbase Connect conference talking to developers about Spring Data Couchbase (and Spring Boot, Spring XD, Spring Session, and more..). Fun show, fun crowd! Next week, I'll be at [JAX London](http://jaxlondon.com/2014/speakers/josh-long) and [Geekout UK](http://uk.geekout.ee/speakers/josh-long/), so be sure to say hi if you're around.

Now, with that, let's get on to the roundup..

1.  The good Dr. Pollack just announced [the 1.0.1 release of Spring XD](http://spring.io/blog/2014/10/02/spring-xd-1-0-1-released), the easiest way to build data-integration and ingestion solutions. It may be only a 1.0.1 release, but make no mistake: there are some very cool features in this release, including improved administrative and administrative UI features.

-   Federated security is an important part of any distributed, multi-client, (micro)service-oriented system. Spring Security and Spring Security OAuth, in particular, provide a compelling solution. Join the good Dr. Dave Syer for this [upcoming webinar on microservice security using OAuth2](https://spring.io/blog/2014/10/01/webinar-security-for-microservices-with-spring-and-oauth2)
-   Matt Stine will be doing a webinar introducing [how Pivotal Cloud Foundry and Spring Cloud make for a natural solution to building microservices](https://spring.io/blog/2014/10/01/webinar-architecting-for-continuous-delivery-microservices-with-pivotal-cf-and-spring-cloud). Don't miss this one!
-   Spring and JavaScript ninja Brian Cavalier has written a *very* interesting post on some of [the work](http://github.com/spring-projects/spring-sync/) we're doing towards enabling [easier differential sync between JavaScript clients and services](http://spring.io/blog/2014/10/08/streaming-json-patch-from-spring-to-a-react-ui). This post not only introduces all these exciting new stuff, but also builds on RaveJS, the Spring Boot-inspired approach to JavaScript.
-   I hope you'll join the Vaadin team and I on October 23rd for a [look how to build Spring Boot-powered Vaadin applications](http://spring.io/blog/2014/10/01/webinar-building-bootful-uis-with-spring-boot-and-vaadin).
-   Justin Lee Grant has put together [a good introductory look at Spring Boot](http://www.justinleegrant.com/?p=489).
-   Spring lead Juergen Hoeller just [announced the 4.1.1 release of Spring framework](http://spring.io/blog/2014/10/01/spring-framework-4-1-1-released). This release includes improved `Optional` handling, a Jackson `ObjectMapper` builder, some requested JDBC improvements, and much more.
-   The good Dr. Syer just announced [the first milestone release of the Spring Cloud project](http://spring.io/blog/2014/10/07/spring-cloud-1-0-0-m1-available-now). Spring Cloud provides primatives for building microservice and PaaS-centric applications. It builds on top of the Netflix OSS stack and Spring Boot.
-   Karthik Abram put together [a nice post on a hidden gem in Spring: the `ClassPathScanningCandidateComponentProvider`](http://www.eclecticlogic.com/2014/09/01/classpath-scanning/) class, which makes it easy to discover beans of a certain type.
-   Nicolas Lejeune has written up a fascinating look at how he managed [to convert a legacy application (it uses EJB3) and move it to a Cloud Foundry environment](http://labs.bsb.com/2014/09/turn-a-legacy-app-into-a-cf-service/)
-   Our pal Toshiaki Maki put together [his ideal Spring Boot Maven archetype](http://github.com/making/spring-boot-blank), and [shared it](http://twitter.com/making/status/519626091934134272)!
-   Are you a Groovy fan? (who isn't??) Check out Groovy project lead Guillaume LaForge's weekly roundup, [*This Week in Groovy*](http://t.co/AiDLC6novd)!
-   Do you want [to debug applications deployed on Cloud Foundry](http://mikusa.blogspot.com/2014/08/debugging-java-applications-on.html)? Daniel Mikusa put together a very nice post explaining one approach.
