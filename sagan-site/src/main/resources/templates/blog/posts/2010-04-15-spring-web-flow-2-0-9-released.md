# Spring Web Flow 2.0.9 Released

_Releases | Keith Donald |  April 15, 2010 | 0 Comments_

Dear Spring Community,

We are pleased to announce that Spring Web Flow 2.0.9 is now available. [Download](http://www.springsource.com/download?project=Spring%20Web%20Flow) | [Reference Documentation](http://static.springframework.org/spring-webflow/docs/2.0.x/reference/htmlsingle/spring-webflow-reference.html) | [Change Log](http://static.springframework.org/spring-webflow/docs/2.0.x/changelog.txt)

This release includes a number of minor bug fixes, as well as an update to the version of Dojo that addresses a Dojo [security vulnerability](http://www.springsource.com/security/spring-web-flow).

In addition to 2.0.x maintenance, the Web Flow project team is currently working on a [2.1](https://jira.springsource.org/browse/SWF/fixforversion/11441) release that provides a current generation version of Web Flow that builds on Spring 3. At the same time, work is continuing towards the first [3.0 milestone](https://jira.springsource.org/browse/SWF/fixforversion/11144) for the next generation Web Flow. Watch for future announcements in these areas soon.

Keith Donald Spring Web Flow Project Lead
