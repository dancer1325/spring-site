# Spring Data release train Hopper SR2 released

_Releases | Mark Paluch |  June 15, 2016 | 0 Comments_

On behalf of the Spring Data team I’d like to announce the availability of the second service release of the Spring Data Hopper release train. The release ships [103 issues](https://jira.spring.io/issues/?filter=15382) fixed. We fixed a couple of bugs in the area of repository projections, especially for JPA users and introduce Hibernate 5.2 compatibility with this release (also already back-ported to the Gosling release train for inclusion in the upcoming service release). Hopper SR2 is a recommended upgrade for all Hopper users and also users of previous release trains.
