# Spring Security 5.8.0-M1 and 6.0.0-M6 are released

_Engineering | Josh Cummings |  July 18, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Security `5.8.0-M1` and `6.0.0-M6` are available now.

This release includes dependency upgrades, bug fixes, and enhancements. Here are a few noteworthy changes:

-   [Deferred SecurityContext lookup](https://github.com/spring-projects/spring-security/issues/10913)
    
-   [SecurityContextHolderStrategy Bean lookup](https://github.com/spring-projects/spring-security/issues/10973)
    
-   [AuthorizationManager support](https://github.com/spring-projects/spring-security/issues/9290)
    
-   [SHA-256 RememberMe support](https://github.com/spring-projects/spring-security/issues/8549)
    
-   [Release automation improvements](https://github.com/spring-projects/spring-security/issues/10451)
    

See the [5.8.0-M1](https://github.com/spring-projects/spring-security/releases/tag/5.8.0-M1) and [6.0.0-M6](https://github.com/spring-projects/spring-security/releases/tag/6.0.0-M6) release notes for more details.
