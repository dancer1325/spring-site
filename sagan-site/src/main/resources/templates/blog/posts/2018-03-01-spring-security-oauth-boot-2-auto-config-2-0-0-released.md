# Spring Security OAuth Boot 2 Auto-config 2.0.0 Released

_Releases | Joe Grandja |  March 01, 2018 | 1 Comment_

I’m pleased to announce the release of [Spring Security OAuth Boot 2 Auto-config](https://github.com/spring-projects/spring-security-oauth2-boot) 2.0.0.

This project is intended to be used to help users transition between the old Spring Security OAuth 2.x support and the [Next Generation OAuth 2.0 Support in Spring Security 5](https://spring.io/blog/2018/01/30/next-generation-oauth-2-0-support-with-spring-security). It provides users of Spring Security OAuth 2.x the same auto-configuration capabilities in a Spring Boot 2.0 based application that is currently available in Spring Boot 1.5.x. For more details please refer to the [documentation](https://docs.spring.io/spring-security-oauth2-boot/docs/current-SNAPSHOT/reference/htmlsingle/).
