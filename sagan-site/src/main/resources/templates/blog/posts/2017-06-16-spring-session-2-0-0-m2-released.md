# Spring Session 2.0.0 M2 Released

_Releases | Rob Winch |  June 16, 2017 | 0 Comments_

On behalf of the community I’m pleased to announce the release of [Spring Session 2.0.0.M2](http://docs.spring.io/spring-session/docs/2.0.0.M2/reference/html5/). This release is focused primarily on ensuring compatibility with Spring Framework 5.0.0.RC2 and Spring Data Kay-M4 which is the minimum Spring version required.

We expect that Spring Session 2.0.0.M3 will have a new Java 8 friendly API along with Support for Spring WebFlux.

# [](#supported-data-stores)[](#supported-data-stores)Supported Data Stores

As an update to our new story for [supported repositories](https://spring.io/blog/2017/05/11/spring-session-2-0-0-m1-released), we now have sub projects for [Spring Session Geode (GemFire)](https://github.com/spring-projects/spring-session-data-geode) and [Spring Session MongoDB](https://github.com/spring-projects/spring-session-data-mongodb). You can find the Spring Session MongoDB release announcement [here](https://spring.io/blog/2017/06/16/spring-session-mongodb-2-0-0-m1-released).
