# Spring Framework 6.1.8, 6.0.21 and 5.3.36 available now

_Releases | Brian Clozel |  May 22, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.8`, `6.0.21` and `5.3.36` are available now.

We are releasing out of our usual cycle to [fix an AOP regression](https://github.com/spring-projects/spring-framework/issues/32838) that was shipped last week.

-   Spring Framework `6.1.8` ships with [10 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.8). This version will be shipped with [Spring Boot 3.2.6](https://github.com/spring-projects/spring-boot/milestone/342), to be released tomorrow.
-   Spring Framework `6.0.21` ships with [6 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.21). This version will be shipped with [Spring Boot 3.1.12](https://github.com/spring-projects/spring-boot/milestone/341), to be released tomorrow.
-   Spring Framework `5.3.36` ships with [3 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.36).
