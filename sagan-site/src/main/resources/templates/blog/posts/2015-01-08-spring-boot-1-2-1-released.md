# Spring Boot 1.2.1 released

_Releases | Phil Webb |  January 08, 2015 | 0 Comments_

Spring Boot 1.2.1 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes over [70 fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.1) and enhancements from 1.2.0 and fixes a number of bugs and regressions. If you are still using Spring Boot 1.1, now would be an excellent time to upgrade!
