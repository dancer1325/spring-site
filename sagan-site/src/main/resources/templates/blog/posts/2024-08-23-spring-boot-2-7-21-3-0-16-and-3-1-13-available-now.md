# Spring Boot 2.7.21, 3.0.16, and 3.1.13 available now

_Releases | Andy Wilkinson |  August 23, 2024 | 0 Comments_

On behalf of the team, I am pleased to announce that Spring Boot 2.7.22, 3.0.17, and 3.1.13 have been released for [support](https://tanzu.vmware.com/spring) customers. The releases are available from [https://packages.broadcom.com](https://packages.broadcom.com).

-   Spring Boot [2.7.22](https://enterprise.spring.io/projects/spring-boot/changelog/2.7.22) ships with 17 bug fixes and dependency upgrades.
-   Spring Boot [3.0.17](https://enterprise.spring.io/projects/spring-boot/changelog/3.0.17) ships with 35 bug fixes and dependency upgrades.
-   Spring Boot [3.1.13](https://enterprise.spring.io/projects/spring-boot/changelog/3.1.13) ships with 46 bug fixes and dependency upgrades.

The next 2.7.x, 3.0.x, and 3.1.x releases for support customers are planned for 3 months' time.
