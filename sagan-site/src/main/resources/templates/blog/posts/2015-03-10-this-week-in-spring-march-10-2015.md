# This Week in Spring - March 10, 2015

_Engineering | Josh Long |  March 10, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring!* This week [I am at DevNexus in Atlanta, GA](https://devnexus.com/s/devnexus2015/speakers#Josh_Long) along with a handful of Spring team members and Spring community luminaries. I'll be presenting a talk on getting started [with Spring Boot](https://devnexus.com/s/devnexus2015/speakers#Josh_Long) and I hope you'll join me, or one of the many other talks on Spring team:

-   [Spring 4 Web Applications](https://devnexus.com/s/devnexus2015/presentations#id-4156) by Rossen Stoyanchev
-   [Cloud Native Apps with Spring Cloud](https://devnexus.com/s/devnexus2015/presentations#id-4446) by Spencer Gibb
-   [Building microservices with Scala, functional domain models and Spring Boot](https://devnexus.com/s/devnexus2015/presentations#id-4765) by Chris Richardson
-   [What is new in Spring Integration 4?](https://devnexus.com/s/devnexus2015/presentations#id-4227) by Marius Bogoevici
-   [Spring Data REST - Data Meets Hypermedia](https://devnexus.com/s/devnexus2015/presentations#id-4417) by Greg Turnquist and Roy Clarkson
-   [IoT Realized - The Connected Car](https://devnexus.com/s/devnexus2015/presentations#id-4444) by Michael Minella
-   [Keeping It Clean - Lessons learned from Open Source on how to polish code](https://devnexus.com/s/devnexus2015/presentations#id-4409) by Phillip Webb
-   [Spring Integration Java DSL - A Deep Dive](https://devnexus.com/s/devnexus2015/presentations#id-4448) by Gary Russell

We'll also be at [the Spring User Group meeting](http://www.meetup.com/AtlantaSpring/events/219263159/). Come out and see Rossen Stoyanchev talk about what's new in Spring MVC, and see Michael Minella talk about what's new in Spring Batch.

Anyway, with that out of the way, let's get to it!

1.  The amazing Dr. Syer just announced [Spring Cloud 1.0.0.RELEASE](https://spring.io/blog/2015/03/04/spring-cloud-1-0-0-available-now)! This is the culmination of a long, frenzied development cycle and includes a *lot* of great stuff. Spring Cloud is delivered as a set of Spring Boot-style auto-configurations and starter dependencies for infrastructure supporting microservices.
2.  Spring Data ninja Christoph Strobl just announced [RC1 for the Spring Data Fowler release train](https://spring.io/blog/2015/03/05/spring-data-release-train-fowler-rc1-available). This new release adds many new features including support for the `@Score`\-annotation in Solr, updated MongoDB 3 support (including support for both types of MongoDB GIS indexes), and much more.
3.  Javascript month continues with our remaining March Webinars. Scott Deeg will examine Polymer, Spring Security, and Spring Cloud for a lightweight API gateway on [March 24th](https://spring.io/blog/2015/02/24/webinar-building-a-secure-polymer-app-with-spring-backend). Lastly, Julien Dubois will introduce the AngularJS+Boot application generator, JHipster on [March 17th, 2015](https://spring.io/blog/2015/02/24/webinar-jhipster-for-spring-boot).
4.  This week's SpringOne2GX 2014 replays are out! Check out Gunnar's excellent talk on [Creating modular test driven SPAs with Spring and AngularJS](http://spring.io/blog/2015/03/10/springone2gx-2014-replay-creating-modular-test-driven-spas-with-spring-and-angularjs). Rob Winch and Ken Krueger cover Spring MVC Test in [The Quest for the Holy Integration Test](https://spring.io/blog/2015/03/10/springone2gx-2014-replay-the-quest-for-the-holy-integration-test) - a less known and super helpful part of Spring MVC. Scott Frederick, Ramnivas Laddad cover the original Spring Cloud (now called Spring Cloud Connectors) in their talk [Simplify Cloud Applications using Spring Cloud Connectors](http://spring.io/blog/2015/03/10/springone2gx-2014-replay-simplify-cloud-applications-using-spring-cloud-connectors), and one from RedHat/OpenShift about [Spring your apps into the cloud with a PaaS](https://spring.io/blog/2015/03/10/springone2gx-2014-replay-spring-your-apps-into-the-cloud-with-a-paas).
5.  EMC Academics have just released their first of [their Flipboard Spring magazine](https://flipboard.com/@emcacademics/spring-bn5acksky) - check it out!
6.  [JAXenter had an nice writeup on the recently released Spring Cloud 1.0.0](http://jaxenter.com/spring-cloud-1-0-0-now-available-115269.html).
7.  My (super highly rated, owing I think to the subject..) talk from [Devoxx UK 2014 on Spring Boot](https://www.parleys.com/talk/building-bootiful-applications-spring-boot) is available on Parleys.com! Check it out!
8.  I had the pleasure of doing two webinars for O'Reilly whose recordings I just learned are now online, one on building [applications with Spring Boot](http://www.oreilly.com/pub/e/3282) and [microservices with Spring Cloud](http://www.oreilly.com/pub/e/3283). O'Reilly are publishing my book on Spring and Cloud Foundry-based microservices.
9.  Our pal and community member Joram Barrez, co-founder of the Activiti BPMN workflow engine, [put together a nice post introducing the Spring Boot and Activiti integration](https://spring.io/blog/2015/03/08/getting-started-with-activiti-and-spring-boot).
10.  InfoQ interviewed Spring framework lead Juergen Hoeller on a wide [variety of things at the recent Voxxed Days Vienna event](http://www.infoq.com/interviews/spring-today-past-and-future).
