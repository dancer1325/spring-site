# Spring IO Platform Brussels-SR14

_Releases | Andy Wilkinson |  October 18, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](https://spring.io/projects/platform) Brussels-SR14 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring AMQP 1.7.11
-   Spring Boot 1.5.17
-   Spring Data Ingalls SR16
-   Spring Framework 4.3.20
-   Spring Security 4.2.9
-   Spring Security OAuth 2.0.16
-   Spring Web Flow 2.4.8
-   Spring Web Services 2.4.3

The versions of a number of third-party dependencies have also been updated.
