# Spring for Apache Pulsar 1.1.0-M1 available now

_Releases | Chris Bono |  February 21, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring for Apache Pulsar 1.1.0-M1 has been released and is now available from [https://repo.spring.io/milestone](https://repo.spring.io/milestone).

This milestone release will be included with the upcoming Spring Boot [3.3.0-M2](https://github.com/spring-projects/spring-boot/milestone/329) release.

This release includes [enhancements, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.0-M1). Notable new features include:

-   Auto Schema support
-   Default topic/schema via message annotation

Please see the [release notes](https://github.com/spring-projects/spring-pulsar/releases/tag/v1.1.0-M1) for more details.

A great place to get started is the [Quick Tour](https://docs.spring.io/spring-pulsar/docs/1.1.0-M1/reference/reference/pulsar.html#quick-tour) in the reference docs.
