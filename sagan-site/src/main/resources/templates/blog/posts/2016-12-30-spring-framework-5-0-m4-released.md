# Spring Framework 5.0 M4 released

_Releases | Juergen Hoeller |  December 30, 2016 | 9 Comments_

Dear Spring community,

It is my pleasure to announce that [the fourth Spring Framework 5.0 milestone](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15721) is available now, right in time to wrap up 2016! Once again, please check [the project page](http://projects.spring.io/spring-framework/) for how to obtain it from our milestone repository and the updated ["Reactive Web Applications"](http://docs.spring.io/spring/docs/5.0.0.M4/spring-framework-reference/html/web-reactive.html) documentation.

This milestone brings our reactive efforts close to completion and introduces a functional bean registration facility as well as comprehensive support for the Kotlin language. Stay tuned for a dedicated blog post on Kotlin...

Cheers, Juergen
