# Spring Roo 1.2.3.RELEASE available

_Engineering | Alan Stewart |  December 18, 2012 | 0 Comments_

The Spring Roo team is pleased to announce the availability 1.2.3.RELEASE.  This is the third maintenance release for 1.2 and includes fixes for a number of issues and includes support for Spring Framework 3.2.0. Please see the [change log](https://jira.springsource.org/issues/?jql=project%20%3D%20ROO%20AND%20resolution%20%3D%20Complete%20AND%20fixVersion%20%3D%20%221.2.3.RELEASE%22%20AND%20status%20%3D%20Resolved) for a list of the bugs and improvements completed.

I hope you enjoy this new release!

Alan

Spring Roo Project Lead
