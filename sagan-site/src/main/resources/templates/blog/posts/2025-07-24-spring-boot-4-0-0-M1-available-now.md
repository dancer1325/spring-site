# Spring Boot 4.0.0-M1 available now

_Releases | Phil Webb |  July 24, 2025 | 6 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot `4.0.0-M1` has been released and is now available from **Maven Central!!**

This release includes is the first milestone of the 4.x generation and is the first release including our efforts to modularize the codebase. It is also the first milestone that we are publishing to Maven central.

Please see the [release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0.0-M1-Release-Notes) and the [draft migration guide](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0-Migration-Guide) for more details and upgrade instructions.

We are particularly interested in getting early feedback on the new modular design and hearing about any upgrade problems that you encounter. Issues can be raised [on the GitHub project page](https://github.com/spring-projects/spring-boot/issues).

Thanks to all those who have contributed to this release.
