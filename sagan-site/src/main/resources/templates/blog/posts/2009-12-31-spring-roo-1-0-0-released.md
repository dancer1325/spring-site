# Spring Roo 1.0.0 Released

_Engineering | Ben Alex |  December 31, 2009 | 0 Comments_

We're delighted to announce the general availability (GA) of [Spring Roo](http://www.springsource.org/roo) 1.0.0.

Spring Roo is a next-generation rapid application development tool for Java developers. With Roo you can easily build full Java applications in minutes. It differs from other tools by focusing on:

-   **Higher Java productivity**: Try the ten minute test and see for yourself.
-   **Stock-standard Java**: Roo uses the Java APIs and standards you already know and trust.
-   **Usable and learnable**: Roo features an extremely high level of usability and an advanced shell.
-   **No engineering trade-offs**: Roo has no runtime portion and does not impose any CPU, RAM or disk storage cost.
-   **Easy Roo removal**: Roo can be easily removed from a user project in under five minutes.

This release follows some eight community releases (A1, A2, M1, M2, RC1, RC2, RC3, RC4) and extensive feedback. Before reaching Roo 1.0.0 we already had [over 1,000 forum posts](http://forum.springsource.org/forumdisplay.php?f=67) and some [500 Jira issues](http://jira.springsource.org/browse/ROO) logged. There's already been [dozens](http://www.jaxitalia.com/konferenzen/jaxitalia09/speaker/#4566) [of](http://www.slideshare.net/krimple/philly-spring-ug-roo-overview) [conference](http://europe.springone.com/europe-2009/presentation/Extreme+Productivity+in+Application+Development) [and](http://www.springone2gx.com/conference/new_orleans/2009/10/session?id=15641) [JUG](http://www.springone2gx.com/conference/new_orleans/2009/10/session?id=15644) [presentations](http://thestrangeloop.com/sessions/extreme-web-productivity-spring-roo) [on](http://www.therichwebexperience.com/conference/orlando/2009/12/session?id=16629) [Roo](http://qconlondon.com/london-2010/speaker/Rod+Johnson), several third-party add-ons and there's even people already writing a commercial book on Roo.

Spring Roo is thoroughly documented, with around 100 pages of detailed [documentation](http://static.springsource.org/spring-roo/reference/index.html) available online and in the download ZIP. You can read a [three-part](http://blog.springsource.com/2009/05/01/roo-part-1/) [blog](http://blog.springsource.com/2009/05/27/roo-part-2/) [series](http://blog.springsource.com/2009/06/18/roo-part-3/) introducing Roo, plus view recently-updated conference presentations ([Introduction to Roo](http://www.slideshare.net/benalexau/introduction-to-spring-roo-100-2805183) and [Roo Technical Deep Dive](http://www.slideshare.net/benalexau/spring-roo-100-technical-deep-dive)).

You can [download](http://www.springsource.com/download/community?project=Spring%20Roo) this new version of Roo, plus access key links at:

-   **[http://www.springsource.org/roo](http://www.springsource.org/roo)**

We hope you that enjoy using Roo. The Roo team will be taking a short break to celebrate the New Year, and we look forward to hearing about your Roo experiences and following up with more Roo resources.

Happy New Year!
