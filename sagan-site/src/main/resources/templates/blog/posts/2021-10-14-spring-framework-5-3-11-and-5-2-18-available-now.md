# Spring Framework 5.3.11 and 5.2.18 available now

_Releases | Stéphane Nicoll |  October 14, 2021 | 0 Comments_

**Update (2021-10-26):** These releases contain fixes for a security issue. Please, read the related [announcement](https://spring.io/blog/2021/10/26/security-reports-for-spring-framework-spring-data-rest-spring-amqp-and-spring-cloud-openfeign).

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.11` and `5.2.18` are available now.

Spring Framework `5.3.11` includes [32 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.11). Spring Framework `5.2.18` includes [10 selected fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.18.RELEASE).

Stay tuned for follow-up Spring Boot releases next week.
