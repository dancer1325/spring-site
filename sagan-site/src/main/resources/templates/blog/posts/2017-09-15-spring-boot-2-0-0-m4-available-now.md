# Spring Boot 2.0.0 M4 Available Now

_Releases | Stéphane Nicoll |  September 15, 2017 | 1 Comment_

Hot on the heels of [the latest Spring Framework 5 release candidate](https://spring.io/blog/2017/09/11/spring-framework-5-0-rc4-available-now), Spring Boot 2.0 M4 is now available from [our milestone repository](http://repo.spring.io/milestone/). This release closes [150 issues and pull requests](https://github.com/spring-projects/spring-boot/milestone/78?closed=1) and is a major step towards 2.0 GA. Thanks to all that contributed!

This milestone provides a host of minor tweaks and enhancements along with three major changes:

-   A new [Actuator endpoint](https://spring.io/blog/2017/08/22/introducing-actuator-endpoints-in-spring-boot-2-0) infrastructure
-   An [overhaul of security](https://spring.io/blog/2017/09/15/security-changes-in-spring-boot-2-0-m4)
-   [Micrometer](http://micrometer.io/) metrics support

For a complete list of changes, and upgrade instructions, see the [Spring Boot 2.0.0.M4 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.0.0-M4-Release-Notes) on the WIKI. We are a bit behind with regards to updating the reference documentation, so please consider using [the snapshot version](http://docs.spring.io/spring-boot/docs/current-SNAPSHOT/reference/htmlsingle/) which we'll continue to update.

We're especially keen to get feedback on any core Spring Framework issues that you might encounter with this release. The final version of Spring Framework 5.0 is due to be released in two weeks time. If you find any issues with the framework, please raise them on [their JIRA](https://jira.spring.io/browse/SPR).

To get started with Spring Boot 2.0 M4 and discover those new feature, you can easily bootstrap a new project on [start.spring.io](https://start.spring.io).
