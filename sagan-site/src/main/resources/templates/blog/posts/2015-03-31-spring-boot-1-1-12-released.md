# Spring Boot 1.1.12 released

_Engineering | Phil Webb |  March 31, 2015 | 0 Comments_

Spring Boot 1.1.12 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [small number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.1.12) and is recommended for anyone still on the 1.1.x line.

This the last planned release from the 1.1.x branch so please consider [upgrading to 1.2.3](http://spring.io/blog/2015/03/31/spring-boot-1-2-3-released) if you haven't already.
