# Spring 3.0.0 is Now Available

_Releases | Adam Fitzgerald |  December 16, 2009 | 0 Comments_

It's here just in time for the holidays! [Arjen Poutsma](http://blog.springsource.com/author/arjenp/) has just [announced](http://twitter.com/poutsma) that Spring 3.0.0 is now final and [Juergen Hoeller](http://blog.springsource.com/author/juergenh/) has [blogged](http://blog.springsource.com/2009/12/16/spring-framework-3-0-goes-ga/) about the features in the release.

[Download](http://www.springsource.com/download/community?project=Spring%20Framework&version=3.0.0.RELEASE) | [Documentation](http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/) | [Javadoc API](http://static.springsource.org/spring/docs/3.0.x/javadoc-api/) | [Change Log](http://static.springsource.org/spring/docs/3.0.0.RELEASE/changelog.txt) | [JIRA](http://jira.springframework.org/browse/SPR)

Congratulations to Juergen, Arjen and all the other SpringSource engineers that worked so hard on the release. Also a huge thank you to all of the dedicated community members that have given [feedback](http://forum.springsource.org/forumdisplay.php?f=21) and identified [issues](http://jira.springframework.org/browse/SPR) along the way. Please keep up the good work so that we can continue to make all the Spring projects better and better.
