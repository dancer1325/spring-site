# Spring Batch 2.1.9.RELEASE is available

_Releases | Dave Syer |  August 28, 2012 | 0 Comments_

#spring #batch 2.1.9.RELEASE is available (download github [http://bit.ly/NYXItL](http://bit.ly/NYXItL) or Maven central). Thanks to all contributors!

It's mostly bugfixes for 2.1.8, plus a few interesting additions (e.g. nested tasklets from any namespace to support Spring Hadoop): [http://static.springsource.org/sprin...1.8-2.1.9.html](http://static.springsource.org/sprin...1.8-2.1.9.html).

The first real commit for 2.2 came in as a pull request. Nice work! Please keep them coming.
