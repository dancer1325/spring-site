# This Week in Spring - June 27th, 2017

_Engineering | Josh Long |  June 27, 2017 | 5 Comments_

Hi Spring fans and welcome to another installment of *This Week in Spring*! This week I'm in Chicago, New York City and Denver talking to Pivotal customers. We've got a lot of good stuff to look at this week so let's get to it!

-   Spring Data ninja Thomas Risberg just announced Spring for Apache Hadoop 2.5.0.RC1. The new release is primarily a bugfix. [The post looks forward to Spring for Apache Hadoop 3.0](https://spring.io/blog/2017/06/23/spring-for-apache-hadoop-2-5-0-rc1-released), which will be based on Spring Boot, a Java 8 baseline, Spark 2.0 integration, and Spring Batch 4.0 readers and writers.
-   Spring Data ninja Mark Paluch looks [at a lot of the new features in Spring Data Kay](https://spring.io/blog/2017/06/20/a-preview-on-spring-data-kay) including support for fluent MongoOperations, Kotlin,smarter MongoDB collation, Java 9 compatability, and so much more.
-   I liked Arnold Galovics's post [on how to configure a `DataSource` proxy](http://blog.arnoldgalovics.com/2017/06/26/configuring-a-datasource-proxy-in-spring-boot/) in Spring Boot.
-   [Kotlin's hidden costs - Benchmarks - Renato Athaydes](https://sites.google.com/a/athaydes.com/renato-athaydes/posts/kotlinshiddencosts-benchmarks) -
-   [Bytecode generation on the JVM and Kotlin](https://t.co/wWlIQQX4PH?ssr=true) - This is a deep-dive into the basics of how the Kotlin-language interacts with the JVM and the compiler. Worth a read if you're diving into Kotlin (as I have been for the last few years) even if the post doesn't have a lot to do with Spring perse.
-   [A Spring Boot application that talks to Ethereum or Quorum network](https://github.com/blk-io/erc20-rest-service) - This was an interesting Spring Boot application that talks to an Ethereum or Quorum network.
-   In [Redmonk's latest look at Java language and framework popularity](https://redmonk.com/fryan/2017/06/22/language-framework-popularity-a-look-at-java-june-2017/), Spring and Spring Boot seem to be doing quite well!
-   Mark Heckler looks at serving XML in REST APIs [in this useful post](http://www.thehecklers.org/2017/06/21/t4sd-tips-4-spring-devs-how-to-get-spring-boot-rest-endpoints-to-speak-xml/)
-   I liked [this post on using the Quartz scheduling API](https://t.co/5iIp2tLCcC?ssr=true). It starts from the basic usage and eventually gets all the way through to the Spring Boot experience. Nice!
-   I liked this presentation on the [basics of Chaos Engineering by Tammy Butow](https://t.co/DgzjCrWM8h?ssr=true).
-   What if an application could automatically repel attackers by rotating its access credentials every day, thereby making every stolen password useless in a short amount of time? That’s the idea behind CredHub, a [new feature introduced for Pivotal Cloud Foundry today](https://venturebeat.com/2017/06/16/pivotal-cloud-foundry-aims-to-thwart-hackers-with-ever-changing-passwords/). Read the rest of this VentureBeat article for more.
-   Dormain Drewitz chimes in with a look at some Secrets of successful Cloud Foundry users from the [recent Cloud Foundry Summit](https://medium.com/p/secrets-of-successful-cloud-foundry-adopters-232193111b18)
