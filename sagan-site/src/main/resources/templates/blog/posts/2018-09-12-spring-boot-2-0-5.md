# Spring Boot 2.0.5

_Releases | Andy Wilkinson |  September 12, 2018 | 0 Comments_

It is my pleasure to announce that Spring Boot 2.0.5 has been released and is is now available from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

Spring Boot 2.0.5 includes [87 fixes, improvements and dependency updates](https://github.com/spring-projects/spring-boot/releases/tag/v2.0.5.RELEASE) (including [an important security fix for Spring AMQP](https://pivotal.io/security/cve-2018-11087)). Thanks to all that have contributed with issue reports and pull requests.
