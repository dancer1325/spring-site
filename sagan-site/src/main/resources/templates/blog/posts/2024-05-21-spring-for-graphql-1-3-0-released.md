# Spring for GraphQL 1.3.0 Released

_Releases | Rossen Stoyanchev |  May 21, 2024 | 0 Comments_

I'm pleased to announce the release of Spring for GraphQL 1.3.0. In addition to the [M1](https://spring.io/blog/2024/02/21/spring-for-graphql-1-3-m1-released) and the [RC1](https://spring.io/blog/2024/04/16/spring-for-graphql-1-3-rc1-released) releases, the final release adds the following features:

-   Virtual thread support for controller methods.
-   GraphQL over WebSocket authentication via `"connect_init"` message.
-   Interceptors in `WebSocketGraphQlTester`.
-   Kotlin `Flow` as controller method return value.

See the [Versions page](https://github.com/spring-projects/spring-graphql/wiki/Spring-for-GraphQL-Versions) on the wiki for a list of features.
