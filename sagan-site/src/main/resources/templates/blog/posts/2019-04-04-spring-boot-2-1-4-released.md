# Spring Boot 2.1.4 released

_Releases | Phil Webb |  April 04, 2019 | 1 Comment_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Boot 2.1.4 has been released and is now available from [repo.spring.io](http://repo.spring.io/release) and Maven Central.

This is a maintenance release that includes [a number of important dependency updates and bug fixes](https://github.com/spring-projects/spring-boot/releases/tag/v2.1.4.RELEASE).

Since 1.5 will be [end of life in august](https://spring.io/blog/2018/07/30/spring-boot-1-x-eol-aug-1st-2019), all users should now be considering an upgrading to Spring Boot 2.1.
