# Spring for GraphQL 1.2.7 and 1.3.1 released

_Releases | Brian Clozel |  June 18, 2024 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.2.7 and 1.3.1 are now available on Maven Central.

1.2.7 closes [11 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.2.7). This version will ship with Spring Boot 3.2.7, [to be released this week](https://calendar.spring.io/).

1.3.1 closes [11 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.3.1). This version will ship with Spring Boot 3.3.1, [to be released this week](https://calendar.spring.io/).
