# Spring Boot 4.0.0-RC1 available now

_Releases | Phil Webb |  October 23, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot `4.0.0-RC1` has been released and is now available from Maven Central.

This release includes [176 enhancements, documentation improvements, dependency upgrades, and bug fixes](https://github.com/spring-projects/spring-boot/releases/tag/v4.0.0-RC1). Notable new features include:

-   Completion of the modularization effort (including new test starters).
-   Support for `RestTestClient`.
-   Redis observability improvements

Please see the [release notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-4.0.0-RC1-Release-Notes) for more details and upgrade instructions.

Thanks to all those who have contributed with issue reports and pull requests.
