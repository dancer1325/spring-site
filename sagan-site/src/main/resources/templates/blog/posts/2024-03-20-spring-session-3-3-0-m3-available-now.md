# Spring Session 3.3.0-M3 available now

_Releases | Marcus Hert Da Coregio |  March 20, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, it is my pleasure to announce the release of Spring Session 3.3.0-M3! The milestone release of Spring Session contains a few noteworthy new features:

-   Add Spring Session Backed implementation of `ReactiveSessionRegistry` [#2824](https://github.com/spring-projects/spring-session/issues/2824)
-   Allow `PlatformTransactionManager` to be specified using `@SpringSessionTransactionManager` [#2821](https://github.com/spring-projects/spring-session/issues/2821)

See the [3.3.0-M3](https://github.com/spring-projects/spring-session/releases/tag/3.3.0-M3) release notes for complete details.
