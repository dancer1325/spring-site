# Apache Tomcat 7.0.0 beta is Now Available

_Releases | Adam Fitzgerald |  June 29, 2010 | 0 Comments_

The first [Apache Tomcat 7](http://tomcat.apache.org) release, Tomcat 7.0.0 beta, is now available from the Tomcat 7 [download page](http://tomcat.apache.org/download-70.cgi) at the Apache Software Foundation.

SpringSource's Tomcat expert, [Mark Thomas](http://blog.springsource.com/author/mthomas/), describes the [details](http://www.tomcatexpert.com/blog/2010/06/29/apache-tomcat-7-has-been-released) of the release at [TomcatExpert.com](http://www.tomcatexpert.com).In addition to the implementation of the Servlet 3.0, JSP 2.2 and EL 2.2 specifications, Tomcat 7 boasts a number of new features:

-   memory leak prevention and detection
-   protection against session fixation attacks
-   a simple filter to add cross-site request forgery protection to an application
-   simplified embedding
-   alias support
-   better security for the Manager and Host Manager applications

Developers interested in using Tomcat should also check out [tc Server Developer Edition](http://www.springsource.com/products/tcserver/devedition) which incorporates the [Spring Insight](http://www.springsource.org/insight/) performance analysis console. The tc Server team are tracking the Tomcat beta process and will provide an update once the Tomcat 7 releases stabilize.
