# Spring Session 1.0.0.M1 Released

_Engineering | Rob Winch |  July 08, 2014 | 50 Comments_

I'm pleased to announce the release of [Spring Session](https://github.com/spring-projects/spring-session) 1.0.0.M1.

# [](#benefits)Benefits

This project provides a number of [benefits](https://github.com/spring-projects/spring-session#benefits) including:

-   Accessing a session from any environment (i.e. web, messaging infrastructure, etc)
-   In a web environment
    -   Support for clustering in a vendor neutral way
    -   Pluggable strategy for determining the session id
    -   Easily keep the HttpSession alive when a WebSocket is active

# [](#getting-started)Getting Started

Refer to the [Quick Start](https://github.com/spring-projects/spring-session#quick-start) section of the README to learn how to you can use Spring Session in your application.
