# Spring Framework 5.3.31 and 6.0.14 available now

_Releases | Brian Clozel |  November 16, 2023 | 0 Comments_

**Updates**

-   **\[11-27\]** The 6.0.14 release includes a fix for [CVE-2023-34053](https://spring.io/security/cve-2023-34053)

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.31` and `6.0.14` are available now.

Spring Framework `5.3.31` ships with [18 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.31), including [1 fix for a regression](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A5.3.31+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 2.7.18, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.0.14` ships with [38 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.14), including [4 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.0.14+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.0.13 and 3.1.6, [to be released next week](https://calendar.spring.io/).

Note, Spring Boot 2.7.18 and 3.0.13 will be [the last OSS support releases in the 2.7.x and 3.0.x lines respectively](https://spring.io/projects/spring-boot#support).
