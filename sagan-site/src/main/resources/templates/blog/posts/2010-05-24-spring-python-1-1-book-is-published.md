# Spring Python 1.1 book is published

_Engineering | Greg L. Turnquist |  May 24, 2010 | 0 Comments_

[![](https://www.packtpub.com/sites/default/files/imagecache/productview/bookimages/0660_MockupCover_0.jpg "0660_cover")](https://www.packtpub.com/sites/default/files/imagecache/productview/bookimages/0660_MockupCover_0.jpg)The [Spring Python 1.1 book](https://www.packtpub.com/create-powerful-versatile-spring-python-1-1-applications/book?mid=310510qhzlon&utm_source=springpythonbook.com&utm_medium=affiliate&utm_content=authorsite&utm_campaign=mdb_003537) is published! [Spring Python](http://springpython.webfactional.com) takes the concepts of the [Spring Framework](http://springframework.org) and [Spring Security](http://static.springsource.org/spring-security/site/), and applies them to the world of [Python](http://python.org).

In this book, you will be writing code using [dependency injection](http://springpython.webfactional.com/1.1.0/reference/html/objects.html), [aspect oriented programming](http://springpython.webfactional.com/1.1.0/reference/html/aop.html), [data access](http://springpython.webfactional.com/1.1.0/reference/html/dao.html), [transaction management](http://springpython.webfactional.com/1.1.0/reference/html/dao.html), [remoting](http://springpython.webfactional.com/1.1.0/reference/html/remoting.html) and [security](http://springpython.webfactional.com/1.1.0/reference/html/security.html). Later on in the first case study, you will be building a banking web app and see how using all these components in concert quickly meets your needs. In the second case study you will integrate Python and Java together as you build a flight reservation system.

The book is full of code samples, screenshots, and diagrams. It presents a pragmatic, no-nonsense approach to using the powerful concepts of Spring with the nimble language of Python. Its for Java developers for want to stir in some dynamic boosting Python. Its also for Python developers who want to take their applications to the next level.

Spring Python is the first Spring extension to go live. The project was started back in 2006 and has been growing ever since.
