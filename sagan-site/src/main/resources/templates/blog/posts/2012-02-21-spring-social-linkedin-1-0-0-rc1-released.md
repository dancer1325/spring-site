# Spring Social LinkedIn 1.0.0.RC1 Released

_Releases | Craig Walls |  February 21, 2012 | 0 Comments_

Dear Spring Community,

We are happy to announce that the first release candidate of Spring Social LinkedIn is now available!

Thanks to community contributions, the Spring Social LinkedIn project now has a complete API binding in addition to connection support through Spring Social's service provider connection framework.

To get Spring Social LinkedIn, download the [release distribution](http://www.springsource.com/download/community?project=Spring%20Social%20LinkedIn) or simply add the [Maven artifacts](http://static.springsource.org/spring-social-linkedin/docs/1.0.x/reference/html/overview.html#overview-howtoget) to your project.

As we push toward a GA release, we would like to hear what you think of the RC1 release. Give feedback in the [forum](http://forum.springsource.org/forumdisplay.php?82-Social) or, if you have any suggestions or find any bugs, please post them in the [issue tracker](https://jira.springsource.org/browse/SOCIALLI).
