# Spring for Apache Hadoop 1.0 RC2 Released

_Releases | Costin Leau |  January 21, 2013 | 0 Comments_

Dear Spring Community, I am pleased to announce the second, and last planned, release candidate, 1.0 RC2 for Spring for Apache Hadoop:

[Downloads](http://www.springsource.com/download/community?project=Spring%20Data%20Hadoop&version=1.0.0.RC2) | [JavaDocs](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.RC2/api/) | [Reference Documentation](http://static.springsource.org/spring-data/hadoop/docs/1.0.0.RC2/reference/html/) | [Changelog](http://static.springframework.org/spring-data/hadoop/docs/1.0.0.RC2/changelog.txt)

RC2 provides bug fixes and improvements and enhances the overall usability:

-   Introduced dedicated namespace for Cascading
-   Enhanced compatibility with Cloudera CDH3 and 4 and Greenplum HD 1.x
-   Cascading, Pig and MapReduce batch tasklet expose the execution stats
-   Refined threading for MapReduce jobs including safe job cancellation
-   Overhauled reference documentation and samples

For the full list of changes, refer to the [changelog](http://static.springframework.org/spring-data/hadoop/docs/1.0.0.RC2/changelog.txt). For more information on Spring for Apache Hadoop, see the project home page for reference documentation and the sample applications.

For a quick introduction to Spring for Apache Hadoop, see this <webinar|[http://www.youtube.com/watch?v=wlTnBzQ6KDU>](http://www.youtube.com/watch?v=wlTnBzQ6KDU%3E) or the reference documentation.

We look forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?80-NoSQL) or in the [issue tracker](https://jira.springsource.org/browse/SHDP).
