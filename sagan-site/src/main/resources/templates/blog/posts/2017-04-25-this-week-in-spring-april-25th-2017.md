# This Week in Spring - April 25th, 2017

_Engineering | Josh Long |  April 25, 2017 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in beautiful Bangalore, India, at the [10th edition of the great, *Great* Indian Developer Summit](http://www.developermarch.com/developersummit)! Then, I'm off to Perth, Australia, [for YOW! West](http://west.yowconference.com.au/). If you're around, [say hi!](http://twitter.com/starbuxman). We've got a lot to cover, so let's get to it!

-   Spring REST Docs lead Andy Wilkinson [just announced Spring REST Docs 1.2.0](https://spring.io/blog/2017/04/24/spring-rest-docs-1-2-0-release)! This new release includes smarter Asciidoctor integration, support for request-and-response snippets, and better support for large and complex payloads. At the same time, Andy *also* [released Spring REST Docs 1.1.3, a maintenance release](https://spring.io/blog/2017/04/24/spring-rest-docs-1-1-3-release)
-   Catch a replay of [Spring WebFlux ninja Sebastien Deleuze's epic Reactive Spring talk on ZTLive](http://pages.zeroturnaround.com/Reactive-Spring-Webinar_On-Demand-Gated-Landing-Page.html?utm_campaign=reactivespringwebinar&utm_medium=social&utm_source=twitter)
-   Spring Boot [ninja Stéphane Nicoll just announced Spring Boot 1.5.3](https://spring.io/blog/2017/04/21/spring-boot-1-5-3-available-now), a maintenance release, packed with lots of fixes and improvements. He [also announced Spring Boot 1.4.6](https://spring.io/blog/2017/04/20/spring-boot-1-4-6-available-now).
-   Spring Cloud Data Flow lead and co-founder Dr. Mark Pollack [just announced Spring Cloud Data Flow 1.2.RC1](https://spring.io/blog/2017/04/20/spring-cloud-data-flow-1-2-rc1-released). This release is *jam-packed* with new features including support for composed tasks, real-time metrics and monitoring, OAuth improvements, a new (deliciously named) release of Spring Cloud Stream app starters Bacon.RELEASE, and a new release of Spring Cloud Task App Starters Belmont.RC1. Get the bits while you can! Kick the tires and feedback!
-   Spring Cloud Pipelines lead and continuous delivery guru [Marcin Grzejszczak just announced Spring Cloud Pipelines 1.0.0.M4](https://spring.io/blog/2017/04/20/spring-cloud-pipelines-1-0-0-m4-released). Spring Cloud Pipelines is a ready-made continuous delivery pipeline that you can use in Jenkins or [Pivotal's Concourse](https://concourse.ci).
-   Not one to rest on his laurels, [Stéphane Nicoll also announced Spring Framework 4.3.8](https://spring.io/blog/2017/04/18/spring-framework-4-3-8-released)
-   The Baeldung blog has a nice post on [using the Spring AMAQP project's service invoker support](http://www.baeldung.com/spring-remoting-amqp)
-   Spring Cloud Zipkin lead and OpenZipkin project contributor Adrian Cole put together a very nice talk on [observability with logging, ,metrics, and tracing](https://speakerdeck.com/adriancole/observability-3-ways-logging-metrics-and-tracing)
-   This connector makes [it easy to connect RabbitMQ to Apache Kafka](https://github.com/jcustenborder/kafka-connect-rabbitmq) - looks interesting!
-   Spring Cloud Contract lead Marcin Grzejszczak (remember him? He's the lead of Spring Cloud Pipelines, too!) [did a nice interview on InfoQ](https://www.infoq.com/news/2017/04/spring-cloud-contract).
-   Bozho did a nice blog looking at use-cases for [Spring MVC in a Spring Boot application](https://techblog.bozho.net/spring-boot-enablewebmvc-common-use-cases/)
-   the Microsoft Azure service broker for Cloud Foundry is very interesting for a number of reasons: it lets you use Microsoft Azure backing services from Cloud Foundry (right on!) and it's written in Spring Boot, [so it serves as a very nice template](https://github.com/pivotal-partner-solution-architecture/azure-service-broker-client)
-   Want to learn what's happening in the Cloud Foundry ecosystem? [I love this roundup](https://www.cloudfoundry.org/cloud-foundry-technology-review-jan-march-2017/) from January to March 2017.
-   the Auth0 blog has a nice post [on using JWTs with Spring Boot](https://auth0.com/blog/securing-spring-boot-with-jwts)
-   Pierre Besson put together a *very* convenient JHipster [generator that generates a custom Spring Boot `banner.txt`](https://github.com/PierreBesson/generator-jhipster-banner)
-   Gerrit Meier put together a very cool look [at using Spring WebFlux with the Thymleaf templating technology](https://github.com/meistermeier/reactive-thymeleaf)
-   Subham Aggarwal put togeter [a nice post on caching with Spring](http://www.discoversdk.com/blog/caching-in-spring)
