# Spring Boot 1.1.8 released

_Releases | Phil Webb |  October 11, 2014 | 0 Comments_

Spring Boot 1.1.8 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [security patch](https://github.com/spring-projects/spring-boot/commit/3135c7f8aeb30b71eccc02bb0bb3817132c41060) to the "white label error" page which prevents potential cross-site scripting attacks; as such it is a recommended upgrade for all Spring Boot users. For a complete list of changes please refer to the [issue tracker](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.1.8).
