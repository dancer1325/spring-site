# Spring Framework 4.3.17 and 5.0.6 available now

_Releases | Stéphane Nicoll |  May 08, 2018 | 0 Comments_

It is my pleasure to announce that Spring Framework `4.3.17` and `5.0.6` have been released and are available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22).

Spring Framework 4.3.17 includes [15 fixes and a selection of improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16751). Spring Framework 5.0.6 contains [over 40 fixes and improvements](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=16720). Those maintenance releases will be used for the upcoming Spring Boot `1.5.13` and `2.0.2` releases coming up later this week.

**2018-05-09 Update:** The releases address a vulnerability. Please see [this blog post](https://spring.io/blog/2018/05/09/spring-project-vulnerability-reports-published) published after the associated Spring Boot [2.0.2](https://spring.io/blog/2018/05/09/spring-boot-2-0-2) and [1.5.13](https://spring.io/blog/2018/05/09/spring-boot-1-5-13) releases.
