# Spring Boot 2.3.0.M2 available now

_Releases | Stéphane Nicoll |  February 13, 2020 | 0 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the second milestone of Spring Boot 2.3 has been released and is available from [our milestone repository](https://repo.spring.io/milestone/).

This release closes [120 issues and pull requests](https://github.com/spring-projects/spring-boot/releases/tag/v2.3.0.M2). Thanks to all those who have contributed with issue reports and pull requests.

Highlights of this milestone include:

-   Improvement of our [Docker image support](https://spring.io/blog/2020/01/27/creating-docker-images-with-spring-boot-2-3-0-m1)
-   Upgrade to MongoDB 4 (still in beta)
-   Support of wildcard locations for configuration files
-   Numerous dependency upgrades

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.3.0-M2-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.3.0.M2/reference/html/).

If you want to get started with 2.3 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).
