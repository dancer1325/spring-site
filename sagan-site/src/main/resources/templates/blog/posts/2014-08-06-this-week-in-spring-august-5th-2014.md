# This Week in Spring - August 5th, 2014

_Engineering | Josh Long |  August 06, 2014 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it.

1.  Spring Cloud lead Ramnivas Laddad has written up a nice post that looks at how to extend [Spring Cloud to consume *other* services in a consistent manner](http://spring.io/blog/2014/08/05/extending-spring-cloud).

-   I liked Samer Abdelkafi's blog introducing [how to use Java configuration with Spring](http://samerabdelkafi.wordpress.com/2014/08/03/spring-mvc-full-java-based-config/). It's always a good time to review..
-   The Pragmatists blog has a detailed look at the sort of (tortured) steps involved in [using Spring with the Play! web framework](http://pragmatists.pl/blog/2014/07/integrating-spring-with-play-framework-1-0-jpa-management/)
-   Biju Kunjummen put together a great post that looks at [how to deploy a Spring Cloud enabled application to Cloud Foundry](http://www.javacodegeeks.com/2014/08/deploying-a-spring-boot-application-to-cloud-foundry-with-spring-cloud.html)
-   Thys Michels is back at it again, this time with a post on [building a ToDo application with Spring MVC and Angular.js](http://thysmichels.com/2014/08/04/spring-mvc-angularjs-todo-list/)
-   I am really happy that this link is in this week's roundup, the author -Priyadarshini - did an amazing job! Check out this really thoughtful introduction [to using Spring Boot to build a CRUD-capable Spring Boot-based application](http://www.programming-free.com/2014/07/spring-data-rest-with-angularjs-crud.html).
-   The `/dev/Kico` blog has a [nice Portguese-language post on what's new in Spring 4](http://www.itexto.net/devkico/?p=1919)
-   Ahmed Essam El-din Ahmed's roundup of the [integrations for security among the various web frameworks - including Spring MVC - is fantastic](http://blog.eramax.org/java-security-frameworks/).
-   Did you know that [Spring provides a `StoredProcedure` object that can be used to wrap and adapt the painful setup involved in invoking a stored procedure from JDBC](http://myshittycode.com/2014/08/01/spring-invoking-stored-procedure/).
-   Theodora Fragkouli put together a [nice post on how to handle Spring MVC form handling](http://examples.javacodegeeks.com/enterprise-java/spring/mvc/spring-mvc-form-handling-example/)
-   The ITEssays blog has a code recipe demonstrating [how to use Spring Data MongoDB with XML configuration](http://itessays.com/java-development-technology/spring-mongodb-mongodb.html).
