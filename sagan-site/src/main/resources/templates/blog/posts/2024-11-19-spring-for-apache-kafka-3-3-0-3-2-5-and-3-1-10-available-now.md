# Spring for Apache Kafka 3.3.0, 3.2.5 and 3.1.10 Available Now

_Releases | Soby Chacko |  November 19, 2024 | 0 Comments_

On behalf of everyone who contributed, we are pleased to announce the general availability of Spring for Apache Kafka `3.3.0`. This is the first release that is generally available in the `3.3.x` generation. In additions, the patch GA releases for `3.2.5` and `3.1.10` are also available now. We thank everyone who contributed to these releases.

Please note that the `3.1.10` release is the last OSS release in the `3.1.x` generation. Please see the [support timeline](https://spring.io/projects/spring-kafka#support) for more details.

#### [](#release-notes)Release Notes

See the release notes for more details on what's included in these releases:

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.0](https://github.com/spring-projects/spring-kafka/releases/tag/v3.3.0)

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.5](https://github.com/spring-projects/spring-kafka/releases/tag/v3.2.5)

[https://github.com/spring-projects/spring-kafka/releases/tag/v3.1.10](https://github.com/spring-projects/spring-kafka/releases/tag/v3.1.10)

#### [](#spring-boot-integration)Spring Boot Integration

The `3.3.0` release will be integrated with upcoming Spring Boot `3.4.0`.

The patch releases `3.2.5` and `3.1.10` will be included in the upcoming Spring Boot `3.2.12` and `3.3.6` releases respectively.
