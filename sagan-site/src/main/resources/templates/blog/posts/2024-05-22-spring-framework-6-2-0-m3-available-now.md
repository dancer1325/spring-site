# Spring Framework 6.2.0-M3 available now

_Releases | Brian Clozel |  May 22, 2024 | 0 Comments_

We are happy to announce the availability of the third milestone of Spring Framework 6.2. [We released M2 last week](https://spring.io/blog/2024/05/16/spring-framework-6-2-0-m2-available-now), but decided to ship M3 along [our latest set of maintenance releases](https://spring.io/blog/2024/05/22/spring-framework-6-1-8-6-0-21-and-5-3-36-available-now).

Spring Framework 6.2.0-M3 is available from [repo.spring.io/milestone](https://repo.spring.io/milestone) now, with [a few new features](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.0-M3).

## [](#62-features-recap)6.2 features recap

Check out our [What's New page](https://github.com/spring-projects/spring-framework/wiki/What's-New-in-Spring-Framework-6.x#whats-new-in-version-62) for details about the new features available at this point.
