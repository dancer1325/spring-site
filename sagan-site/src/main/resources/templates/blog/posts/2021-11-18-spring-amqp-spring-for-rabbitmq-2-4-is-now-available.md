# Spring AMQP (Spring for RabbitMQ) 2.4 is Now Available

_Releases | Gary Russell |  November 18, 2021 | 1 Comment_

I am pleased to announce that Spring AMQP 2.4.0 is now available.

### [](#notable-changes)[](#notable-changes)Notable Changes

-   Initial support for the [RabbitMQ Stream Plugin](https://docs.spring.io/spring-amqp/docs/2.8.x/reference/html/#stream-support)
    
-   Support for remoting using Spring Framework’s RMI support is deprecated
    

For information about all changes in this release, see [What’s New](https://docs.spring.io/spring-amqp/docs/2.4.x/reference/html/#whats-new-part).
