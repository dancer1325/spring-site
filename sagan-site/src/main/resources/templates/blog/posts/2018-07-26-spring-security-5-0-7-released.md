# Spring Security 5.0.7 Released

_Releases | Josh Cummings |  July 26, 2018 | 0 Comments_

On behalf of the community, I am pleased to announce that Spring Security 5.0.7 ([changelog](https://github.com/spring-projects/spring-security/milestone/119?closed=1)) has been released. The release primarily delivers bug fixes and dependency version updates along with some minor improvements. The releases will be found in the upcoming Spring Boot maintenance releases coming later this week.
