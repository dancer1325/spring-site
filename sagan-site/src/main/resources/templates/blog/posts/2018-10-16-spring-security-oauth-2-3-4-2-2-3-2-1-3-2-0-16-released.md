# Spring Security OAuth 2.3.4, 2.2.3, 2.1.3, 2.0.16 Released

_Releases | Joe Grandja |  October 16, 2018 | 0 Comments_

I’m pleased to announce the releases of Spring Security OAuth 2.3.4, 2.2.3, 2.1.3 and 2.0.16. The releases address a vulnerability. Please see [this blog post](https://spring.io/blog/2018/10/16/spring-project-vulnerability-reports-published) published after the associated Spring Boot 2.0.6 and 1.5.17 releases.
