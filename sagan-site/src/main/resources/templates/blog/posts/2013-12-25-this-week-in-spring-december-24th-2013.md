# This Week in Spring   - December 24th, 2013

_Engineering | Josh Long |  December 25, 2013 | 0 Comments_

Happy holidays! Hopefully with the holiday season comes some time off, and a chance to relax and more fully catch up on fun stuff you missed during a busy year. Readers of this column will know there are many channels for developers learning about Spring, and studying the field of technologies that Spring supports. Don't forget about our [SpringSource YouTube page](http://youtube.com/SpringSourceDev), [Twitter account](http://twitter.com/springcentral), our 15-30 minute ["Getting Started" guides](http://spring.io/guides), the [blog](http://spring.io/blog), and of course our [Facebook](https://www.facebook.com/springsource) and [Google+ pages](https://plus.google.com/+springframework/posts). I personally want to go back and watch as many SpringOne2GX talks on the YouTube channel as I can.

1.  Remeber [JHipster](http://jhipster.github.io/2013/12/23/jhipster-release-0.6.0.html)? [Julien Dubois's](https://twitter.com/juliendubois) [Yeoman](http://yeoman.io/)\-powered code generator for Spring applications? Well, 0.0.6 has been released and it has no required Spring XML (and Java EE's `web.xml`'s the last one!) and provides code-generation support for services.

-   You may have noticed we've got some new *Getting Started* guides, including one on [Consuming a RESTful Web Service with `rest.js`](http://bit.ly/1kYXK3y).
-   Want to use Zero Turnaround's JRebel, the live class reloading technology, with Spring `4.0.0.RELEASE`? Use [`JRebel 5.4.3-SNAPSHOT (201312032038)`](http://zeroturnaround.com/forums/topic/spring-framework-4-latest-snapshot-failure/).
-   *SD Times* has an interesting look at [the Spring 4.0 release](http://sdt.bz/content/article.aspx?ArticleID=67473&page=1). Mostly analysts perspectives, less tech.
-   [Roy Clarkson](http://twitter.com/royclarkson) put together an awesome *Getting Started* guide on how to use [Spring Boot](http://projects.spring.io/spring-boot/) (and by extension [Cloud Foundry](http://twitter.com/cloudfoundry)!) to effortlessly [host static files (like JavaScript, CSS and `.html` template](https://spring.io/blog/2013/12/19/serving-static-web-content-with-spring-boot).
-   Petri Kainulainen put together a very nice post on how to invoke a Spring Seurity-secured method from a [`@Scheduled`\-job in Spring](http://www.petrikainulainen.net/programming/spring-framework/spring-from-the-trenches-invoking-a-secured-method-from-a-scheduled-job/).
-   Spring Data lead [Oliver Gierke](http://twitter.com/olivergierke) has [moved Spring Data REST to the JSON representation](https://jira.springsource.org/browse/DATAREST-203) [HAL](http://tools.ietf.org/html/draft-kelly-json-hal-06) as the default format. HAL has become very popular since Spring HATEOAS's inception and is now a leading choice in the community.
-   Are you into machine-learning? Here at [GoPivotal](http://twitter.com/GoPivotal) there's a lot of that going around - especially with our open-source centric big-data toolset - and I've [found the Meta Optimize QA site to be very helpful in learning/getting answers.](http://metaoptimize.com/qa/) . This is just an FYI, and not strictly related to Spring...
-   A new cut of the [Thymeleaf templating engine is available that works with Spring 4](http://forum.thymeleaf.org/New-Thymeleaf-integration-module-for-Spring-4-td4027301.html). Nice job, check it out!
-   Viral Patel has a nice blog on [manipulating HTTP cookies with Spring MVC](http://viralpatel.net/blogs/spring-mvc-cookie-example/).
-   [Spring Mobile](http://spring.io/projects/spring-mobile) offers a convenient way to build mobile form-factor-specific versions of an application. Part of this support is a facility for *switching* between experiences and maintaining state about the users preferences for their experience (e.g.: "Switch to Mobile site"). Check out [this great *Getting Started* guide on the subject](https://spring.io/guides/gs/handling-site-preference/).
-   *Application Development Trends* has an interesting, analyst-y [piece on Spring 4.0's release recently](http://adtmag.com/articles/2013/12/17/spring-framework-4.aspx?m=1). Worth a look.
-   Spring Integration lead [Gary Russell](https://twitter.com/gprussell) has just announced the second milestone (first public one, too!) of [Spring Integration 4.0](http://spring.io/blog/2013/12/18/spring-integration-4-0-milestone-2-is-now-available). This release features, among other things, the first cut of the API to rebase on top of Spring Framework 4, and build on top of its Spring Integration-inspired messaging APIs (including `Message`, and `MessageChannel`).
-   Apache BigTop founder Roman Shaposhnik just [joined our team here at @GoPivotal](http://t.co/tWHnBWZIuW). Can't wait to see what's coming!

If you celebrate something in the next week, then happy and healthy holidays to you! C'ya next week, on New Year's eve!
