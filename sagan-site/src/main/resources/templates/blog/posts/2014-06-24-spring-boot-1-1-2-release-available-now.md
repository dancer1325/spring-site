# Spring Boot 1.1.2.RELEASE Available Now

_Releases | Phil Webb |  June 24, 2014 | 0 Comments_

The second maintenance release for [Spring Boot](http://projects.spring.io/spring-boot/) 1.1 has been released and is available now from [repo.spring.io](https://repo.spring.io/release/) and [maven central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This release fixes a number of issues and also offers some performance improvements when running "executable JARs". For a complete list of changes see [the issue tracker](https://github.com/spring-projects/spring-boot/issues?milestone=17&page=1&state=closed).

*(Windows users may want to use the [Spring Boot 1.1.3.RELEASE](http://spring.io/blog/2014/06/27/spring-boot-1-1-3-available-now) as it fixes some Windows specific issues.)*
