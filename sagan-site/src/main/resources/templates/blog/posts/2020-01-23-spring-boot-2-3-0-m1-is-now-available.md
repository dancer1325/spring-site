# Spring Boot 2.3.0.M1 is now available

_Releases | Madhura Bhave |  January 23, 2020 | 2 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the first milestone of Spring Boot 2.3 has been released and is available from [our milestone repository](http://repo.spring.io/milestone/). This release closes [167 issues and pull requests](https://github.com/spring-projects/spring-boot/releases/tag/v2.3.0.M1).

Highlights of this first milestone include:

-   Support for building Docker images using [Cloud Native Buildpacks](https://buildpacks.io/)
-   Support for building layered jars for building optimized Docker images
-   End-to-end traceability for Configuration Properties
-   Numerous [dependency upgrades](https://github.com/spring-projects/spring-boot/issues?utf8=%E2%9C%93&q=label%3A%22type%3A+dependency-upgrade%22+milestone%3A2.3.0.M1)
-   Support for [Spring Data Neumann M1](https://spring.io/blog/2020/01/16/spring-data-neumann-m1-released) which ships with major driver upgrades

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.3.0-M1-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.3.0.M1/reference/html/).

If you want to get started with 2.3 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).

Stay tuned for a blog post describing how to use the new Docker features!
