# STS 2.1.0.RC1 Updated for Eclipse 3.5

_Releases | Adam Fitzgerald |  June 26, 2009 | 0 Comments_

The latest release candidate for [SpringSource Tool Suite](http://www.springsource.com/products/sts) has been updated to run on the brand new [Eclipse Galileo release](http://www.eclipse.org/org/press-release/20090624_galileo.php). STS 2.1.0.RC1 includes support for [Spring Roo](/roo), SpringSource [dm Server](http://www.springsource.com/products/dmserver) and [tc Server](http://www.springsource.com/products/tcserver), enterprise OSGi tools, as well as all the usual great Spring development tools. This newest version also provides support for a Mac OS X 64 bit Cocoa client. Of course the [download](http://www.springsource.com/products/sts/registration) for SpringSource Tool Suite is **free** so get started with the best IDE for enterprise Java development today.
