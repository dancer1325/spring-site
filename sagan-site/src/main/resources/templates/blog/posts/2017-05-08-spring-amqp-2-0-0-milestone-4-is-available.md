# Spring AMQP 2.0.0 Milestone 4 is Available

_Releases | Gary Russell |  May 08, 2017 | 0 Comments_

I am pleased to announce that the fourth milestone of Spring AMQP 2.0 [2.0.0.M4](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=15926) is now available in the [Spring milestone repository](https://repo.spring.io/milestone/).

Features since [the last milestone](https://spring.io/blog/2017/04/04/spring-amqp-2-0-0-milestone-3-is-available) include…​

-   The new `DirectMessageListenerContainer` now supports configuring the number of messages to process between acks.
    
-   You can now specify container `concurrency` on the `@RabbitListener` annotation.
    

The release candidate is expected to be released in early June with the GA shortly after the Spring Framework 5.0 GA release.

For a complete list of changes in 2.0, see [What’s New in the reference manual](http://docs.spring.io/spring-amqp/docs/2.0.0.BUILD-SNAPSHOT/reference/html/_introduction.html#_changes_in_2_0_since_1_7); also see the release notes for [2.0.0.M1](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=15711), [2.0.0.M2](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=15775), [2.0.0.M3](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=15861), as well as [2.0.0.M4](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10450&version=15926).
