# Spring Security 4.0.0.M1 Released

_Engineering | Rob Winch |  May 05, 2014 | 3 Comments_

I'm pleased to announce the release of Spring Security 4.0.0.M1 available in the Spring [Milestone repository](http://docs.spring.io/spring-security/site/docs/4.0.0.M1/reference/htmlsingle/#get-spring-security). The primary component of this first milestone is testing support. Stay tuned to the [spring.io blog](http://spring.io/blog) for a [blog series](http://spring.io/blog/2014/05/07/preview-spring-security-test-method-security) introducing the test support. If you can't wait to try it out, see the tests in the [showcase package](https://github.com/spring-projects/spring-security/tree/4.0.0.M1/test/src/test/java/org/springframework/security/test/web/servlet/showcase) which demonstrate some of the highlights.
