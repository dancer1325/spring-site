# Spring Cloud Square 0.4.0-RC1 is available

_Releases | Olga Maciaszek-Sharma |  December 14, 2021 | 1 Comment_

On behalf of the community, I am pleased to announce that the Release Candidate 1 (RC1) of the [Spring Cloud Square](https://cloud.spring.io) 0.4.0 is available today. The release can be found in [Spring Milestone](https://repo.spring.io/milestone/) repository. You can check out the 0.4.0-RC1 [release notes for more information](https://github.com/spring-projects-experimental/spring-cloud-square/releases/tag/v0.4.0-RC1).

This is primarily a bugfix release. See all issues included in this release [here](https://github.com/spring-projects-experimental/spring-cloud-square/milestone/2?closed=1).

## [](#notable-changes-in-rc1)Notable changes in RC1:

-   Spring Cloud Sleuth integration for OkHttpClient [#36](https://github.com/spring-projects-experimental/spring-cloud-square/issues/36)

The documentation for this milestone can be found [here](https://spring-projects-experimental.github.io/spring-cloud-square/docs/current/reference/html/index.html).
