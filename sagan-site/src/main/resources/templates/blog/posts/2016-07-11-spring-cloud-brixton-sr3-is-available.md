# Spring Cloud Brixton.SR3 is available

_Releases | Ryan Baxter |  July 11, 2016 | 0 Comments_

On behalf of the team, I am pleased to announce that Service Release 3 of the [Spring Cloud Brixton](http://cloud.spring.io) Release Train is available today. The release can be found in our [Spring Release repository](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Ca%3A%22spring-cloud-dependencies%22).

Included in this release is the [GA release of Spring Cloud Task](https://spring.io/blog/2016/07/11/spring-cloud-task-1-0-1-release-is-now-available).

Other than the addition of Spring Cloud Task, this release includes primarily bug fixes.

#### [](#the-following-modules-were-updated-as-part-of-brixtonsr3)The following modules were updated as part of Brixton.SR3:

-   Spring Cloud Security [1.1.2.RELEASE](https://github.com/spring-cloud/spring-cloud-security/milestone/12?closed=1)
-   Spring Cloud Netflix [1.1.4.RELEASE](https://github.com/spring-cloud/spring-cloud-netflix/milestone/24?closed=1)
-   Spring Cloud Sleuth [1.0.3.RELEASE](https://github.com/spring-cloud/spring-cloud-sleuth/milestone/10?closed=1)
-   Spring Cloud Task [1.0.1.RELEASE](https://github.com/spring-cloud/spring-cloud-task/milestone/6?closed=1)

And, as always, we welcome feedback: either on [GitHub](https://github.com/spring-cloud), on [gitter](https://gitter.im/spring-cloud/spring-cloud), on [Stack Overflow](http://stackoverflow.com/questions/tagged/spring-cloud), or on [Twitter](https://twitter.com/SpringCloudOSS).
