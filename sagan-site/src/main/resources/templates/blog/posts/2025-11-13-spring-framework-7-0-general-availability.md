# Spring Framework 7.0 General Availability

_Releases | Juergen Hoeller |  November 13, 2025 | 9 Comments_

Dear Spring community,

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework 7.0 is generally available now! Spring Framework 7.0 is the beginning of a new framework generation with a focus on Java 25 and the latest open source ecosystem, serving as the foundation for Spring Boot 4.0.

This generation of Spring comes with a pragmatic baseline and dependency upgrade:

-   Embracing Java 25 (as the latest LTS) while retaining a Java 17 baseline
-   Jakarta EE 11 API level: Servlet 6.1, JPA 3.2, Bean Validation 3.1
-   [Comprehensive null safety through JSpecify](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#null-safety)
-   [Support for Jackson 3.0](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#jackson-3x-support) (retaining deprecated support for Jackson 2.x)
-   Kotlin 2.2
-   JUnit 6.0

Our initial Spring Framework 7.0 release ships with a wide range of new features, including:

-   [Programmatic bean registration](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#programmatic-bean-registration)
-   [Core resilience features](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#resilience-features-retrytemplate-retryable-concurrencylimit)
-   [JmsClient](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#introducing-jmsclient-and-revisiting-jdbcclient)
-   [API versioning](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#api-versioning)
-   [HTTP Interface Client configuration](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#http-interface-client-configuration)
-   [RestTestClient](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes#resttestclient)

Check out our [release notes](https://github.com/spring-projects/spring-framework/wiki/Spring-Framework-7.0-Release-Notes) for a comprehensive overview of enhancements and upgrade notes.

Cheers, Juergen
