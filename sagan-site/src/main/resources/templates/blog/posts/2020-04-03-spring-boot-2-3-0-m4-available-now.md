# Spring Boot 2.3.0.M4 available now

_Releases | Andy Wilkinson |  April 03, 2020 | 0 Comments_

On behalf of the team and everyone that contributed, I am pleased to announce that the fourth milestone of Spring Boot 2.3 has been released and is available from [our milestone repository](https://repo.spring.io/milestone/).

This release closes [99 issues and pull requests](https://github.com/spring-projects/spring-boot/releases/tag/v2.3.0.M4). Thanks to all those who have contributed.

Highlights of this milestone include:

-   Support for [liveness and readiness probes](https://spring.io/blog/2020/03/25/liveness-and-readiness-probes-with-spring-boot).
-   Improved support for [building OCI images](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.3.0-M4-Release-Notes#building-oci-images).
-   Improved support for [building layered jars](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.3.0-M4-Release-Notes#building-layered-jars-1), including customization when using Gradle.

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.3 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.3.0-M4-Release-Notes) on the wiki and the [updated reference documentation](http://docs.spring.io/spring-boot/docs/2.3.0.M4/reference/html/).

2.3.0.M4 is the last 2.3 milestone with 2.3.0.RC1 scheduled for release in three weeks' time. All being well, 2.3.0.RELEASE will then follow a couple of weeks after that in early May.

If you want to get started with 2.3 and try out the new features, you can bootstrap a new project on [https://start.spring.io](https://start.spring.io).
