# Second milestone of spring-test-mvc released

_Releases | Rossen Stoyanchev |  September 26, 2012 | 0 Comments_

A second milestone of the [Spring MVC Test](https://github.com/SpringSource/spring-test-mvc "spring-test-mvc") project is now available through the SpringSource [Artifactory](http://www.jfrog.com/home/ "JFrog") repository [](http://repo.springsource.org/milestone "SpringSource milestone Artifactory repository")[http://repo.springsource.org/milestone](http://repo.springsource.org/milestone).

There is much that's new. On the server side we've added support for servlet filters, basic support for Spring Security, the ability to define default request properties and expectations to re-use across every performed request, and much more. On the client side we've greatly expanded the options for request matching and response creation.

For further details and comments read [this blog post](http://rossenstoyanchev.org/blog/2012/09/25/second-milestone-of-spring-test-mvc/).
