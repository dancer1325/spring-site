# Spring Integration AWS 1.0.0 GA available now

_Releases | Artem Bilan |  July 26, 2016 | 0 Comments_

Dear Spring Community,

It is my pleasure to announce that the Spring Integration AWS `1.0.0.RELEASE` is available now.

As usual, thanks to the community for any feedback and contributions!

This release contains several critical bug fixes (mostly community driven) and aligns requirements of the upcoming [Spring Cloud Stream Application Starters](http://cloud.spring.io/spring-cloud-stream-app-starters/) release.

We haven’t added any new features since the [latest announcement](https://spring.io/blog/2016/04/14/spring-integration-aws-1-0-0-m1-is-now-available) and have concentrated mostly on improvements, bug fixes and integration testing, to deliver this product as soon as possible. For the next `1.1` version we would like to hear from you about priorities since there are still many [Amazon features](https://aws.amazon.com/) to cover.
