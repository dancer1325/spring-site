# Spring Security 5.5.5 and 5.6.2 released

_Releases | Eleftheria Stein-Kousathana |  February 21, 2022 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.5.5 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.5.5)) and Spring Security 5.6.2 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.6.2)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
