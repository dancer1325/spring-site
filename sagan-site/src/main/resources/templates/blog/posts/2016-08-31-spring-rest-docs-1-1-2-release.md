# Spring REST Docs 1.1.2.RELEASE

_Releases | Andy Wilkinson |  August 31, 2016 | 0 Comments_

It's my pleasure to announce that [Spring REST Docs](https://github.com/spring-projects/spring-restdocs) 1.1.2.RELEASE is available from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release). My thanks to everyone who contributed to this release by reporting bugs and opening pull requests.

This maintenance release includes a [number of bug fixes](https://github.com/spring-projects/spring-restdocs/issues?q=milestone%3A1.1.2.RELEASE) and is a recommend upgrade for all Spring REST Docs users.
