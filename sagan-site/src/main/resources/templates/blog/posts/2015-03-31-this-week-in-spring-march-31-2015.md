# This Week in Spring - March 31, 2015

_Engineering | Josh Long |  March 31, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it!

-   Spring Security lead Rob Winch [just announced the amazing Spring Security 4.0 release](http://spring.io/blog/2015/03/26/spring-security-4-0-0-released)! This one is packed with amazing features including websocket support, Spring Data integration, enhanced testing support, and much improved defaults
-   The [Cloud Foundry Java buildpack](https://github.com/cloudfoundry/java-buildpack#configuration-and-extension) now supports overriding inbuilt configuration through the use of well-named environment variables! This means you can tweak all manner of the default behavior without needing to fork the buildpack!
-   Good news! [Spring XD 1.1.1](http://spring.io/blog/2015/03/26/spring-xd-1-1-1-released) is released! It includes improved performance and offset management when using Kafka message bus, support for Spark's streaming reliable receiver, Kryo serialization improvements, RabbitMQ auto-cleanup options and a Sqoop example.
-   Graeme Rocher's amazing Groovy Web App framework, [Grails 3.0 has been released](https://github.com/grails/grails-core/releases/tag/v3.0.0) - now based on the Spring Boot!
-   Greg Turnquist just [announced Spring WS 2.2.1, with bugfixes, was just released](http://spring.io/blog/2015/03/26/spring-ws-2-2-1-is-released). Get it while it's hot!
-   Oh yah, [Spring Security 3.2.7](http://spring.io/blog/2015/03/26/spring-security-3-2-7-released) with bugfixes aplenty is also released but... download 4 (see above)!
-   speaking of allll the data, [Spring Integration Kafka 1.1](http://spring.io/blog/2015/03/26/spring-integration-kafka-support-1-1-ga-is-available) was just released! This is also included in Spring XD 1.1.1, so check it out!
-   Spring framework 4.1.6, with patches and bugfixes, was just released. [It is a recommended update for Spring 4.x users](http://spring.io/blog/2015/03/25/spring-framework-4-1-6-released).
-   I'll be doing [a webinar on smarter service-to-service invocations with Spring Cloud](https://spring.io/blog/2015/03/17/webinar-smarter-service-to-service-invocations-with-spring-cloud)
-   I'll be appearing at the Virtual JUG (vJUG) again, this time [to talk about Spring Cloud-based microservices](http://www.meetup.com/virtualJUG/events/221218834/). Join us, worldwide!
-   Marius Bogoevici and Mark Pollack are teaming up for a webinar on Tues April 28th about [Reactive data-pipelines with Spring XD and Kafka](https://spring.io/blog/2015/03/17/webinar-reactive-data-pipelines-with-spring-xd-and-kafka), a must for any big / fast data enthusiasts.
-   [The replay from Julien Dubois's recent webinar](https://spring.io/blog/2015/03/31/webinar-replay-jhipster-for-spring-boot) on the Spring Boot-based app generator, JHipster - used to generate AngularJS web apps - is now available!
-   Scott Deeg's excellent [Webinar Replay: Building a secure Polymer app with Spring backend](https://spring.io/blog/2015/03/31/webinar-replay-building-a-secure-polymer-app-with-spring-backend) is now available! This is a different frontend JavaScript take on Dr. Dave Syer's uber popular [Spring and Angular JS: A Secure Single Page Application](https://spring.io/blog/2015/01/12/spring-and-angular-js-a-secure-single-page-application) blog and webinar, that uses [Spring Cloud Netflix/Zuul](http://cloud.spring.io/spring-cloud-netflix/) to have a secure JS app on the frontend, a lightweight API gateway with Spring Cloud, and a secure Spring backend.
-   In celebration of the Grails 3.0 release - which is now based on Spring Boot - we'll be publishing all G&G talks from SpringOne2GX 2014 for a while. This week's replays are now up! Jeff Scott Brown talks about [GORM inside and out](https://spring.io/blog/2015/03/31/springone2gx-2014-replay-gorm-inside-and-out) Dr. Paul King shows us [Groovy AST Transformations](http://spring.io/blog/2015/03/31/springone2gx-2014-replay-groovy-ast-transformations), and Ken Kousen shares [Advanced Groovy Tips and Tricks](https://spring.io/blog/2015/03/31/springone2gx-2014-replay-advanced-groovy-tips-and-tricks) and [Distributed Platform Development with Groovy](https://spring.io/blog/2015/03/31/springone2gx-2014-replay-distributed-platform-development-with-groovy)
-   Check out Thomas Darimont's amazing post, [*What's New in Spring Data Fowler*](http://spring.io/blog/2015/03/26/what-s-new-in-spring-data-fowler)
-   The Disid corporation just posted an [update looking ahead to Spring Roo 2.0](http://spring.io/blog/2015/03/30/looking-ahead-to-spring-roo-2-0). Speaking of..
-   I, along with a great many other Spring team members, will be in Barcelona, Spain, for [the Spring I/O conference](http://www.springio.net/). Join us! (also, Sangria!)
