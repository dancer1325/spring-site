# Spring Data Gosling SR4 released

_Releases | Oliver Drotbohm |  February 23, 2016 | 2 Comments_

In preparation of a Spring Boot service release on the horizon, I am pleased to announce the fourth service release of the Spring Data release train Gosling. As usual, service releases contain bugfixes — [45 in total](https://jira.spring.io/issues/?filter=15356) this time —, so that an upgrade is highly recommended.
