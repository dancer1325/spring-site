# Spring Cloud Data Flow 2.10.1 Released

_Releases | Corneil du Plessis |  February 16, 2023 | 0 Comments_

On behalf of the team and everyone who has contributed, we are very excited to announce the `2.10.1` release of Spring Cloud Data Flow is now available from Maven Central. [Release Notes](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.10.1)

### [](#spring-updates)Spring updates

Updates to the following versions:

-   Spring Boot 2.7.8
-   Spring Framework 5.3.25
-   Spring Shell 2.1.5
