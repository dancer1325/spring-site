Hi, Spring fans! Welcome to another installment of *This Week in Spring*! It's Tuesday and work is well underway to prepare for the huge SpringOne event in Las Vegas in just a few short week's time! I'm elated! So, let's get this roundup on the road so I can get back to the preparation frenzy...

-   [Spring AI Embraces OpenAI's Structured Outputs: Enhancing JSON Response Reliability](https://spring.io/blog/2024/08/09/spring-ai-embraces-openais-structured-outputs-enhancing-json-response)
-   [A Bootiful Podcast: Spring Cloud Dataflow, Spring Cloud Task, and Spring Batch legend Glenn Renfro](https://spring.io/blog/2024/08/08/a-bootiful-podcast-spring-cloud-dataflow-spring-cloud-task-and-spring-batch)
-   [in last week's installment of *Spring Tips*, I looked at using Spring Boot and HTMX together](https://spring.io/blog/2024/08/07/spring-tips-htmx)
-   I love this: a bunch of opensource projects are working on a common set of annotations [to convey nullability and null-restricted types in Java](https://www.infoq.com/news/2024/08/jspecify-java-nullability/)
-   I love this: IntelliJ IDEA has a [new feature: multi-project workspaces](https://x.com/yuriy_artamonov/status/1822370489597133287?s=61), which adds support for opening multiple projects in one window. Supports Maven, Gradle and arbitrary projects in IntelliJ IDEA.
