# Groovy 2.3 Released

_Releases | Guillaume Laforge |  May 07, 2014 | 1 Comment_

The Groovy development team is pleased to announce the **release of Groovy 2.3.0**!

Groovy 2.3 is the new major release of the Groovy programming language for the JVM, featuring:

-   official **support for running Groovy on JDK 8**
-   a new [trait](http://beta.groovy-lang.org/docs/groovy-2.3.0/html/documentation/core-traits.html) keyword to define new units of code for composing behaviors
-   new and improved compile-time code transformations like:
    -   **@TailRecursive**: for transforming methods with tail recursion to avoid blowing the stack,
    -   [@Builder](http://beta.groovy-lang.org/docs/groovy-2.3.0/html/documentation/core-metaprogramming.html#xform-Builder): to easily implement fluent builders, generated by the Groovy compiler itself
    -   [@Sortable](http://beta.groovy-lang.org/docs/groovy-2.3.0/html/documentation/core-metaprogramming.html#xform-Sortable): to transform a class to implement Comparable using the various properties of the class
-   a new **NIO2 module** with Path support
-   **lightening fast JSON** parsing and building as covered in the [recent article on InfoQ](http://www.infoq.com/news/2014/04/groovy-2.3-json)
-   **closure parameter type inference**, to infer the type of parameters in closures when using the static type checker and static compiler
-   a [markup template engine](http://beta.groovy-lang.org/docs/groovy-2.3.0/html/documentation/template-engines.html#_the_markuptemplateengine)
-   Groovysh and GroovyConsole ease of use improvements
-   a new [GroovyAssert](http://beta.groovy-lang.org/docs/groovy-2.3.0/html/documentation/core-testing-guide.html#_junit_4) test utility
-   more @BaseScript class capabilities, and more.

If you want to learn more about the new features of Groovy 2.3, please reader the [detailed release notes](http://bit.ly/g230notes), and if you want to get some regular fresh news of the Groovy ecosystem, you can [subscribe to the Groovy Weekly newsletter](http://bit.ly/groovy-weekly-subscribe), delivering fresh news every Tuesday.

For more information about the issues closed, you can have a look at our [changelog](http://bit.ly/g230jira).

In addition to the Groovy library in [Bintray JCenter](https://bintray.com/groovyhttps://bintray.com/groovy) and [Maven Central](http://search.maven.org/#artifactdetails%7Corg.codehaus.groovy%7Cgroovy-all%7C2.3.0%7Cjar), you can download also [download a binary distribution of Groovy](http://groovy.codehaus.org/Download).
