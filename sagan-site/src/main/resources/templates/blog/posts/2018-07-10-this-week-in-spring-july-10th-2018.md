# This Week in Spring - July 10th, 2018

_Engineering | Josh Long |  July 10, 2018 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! This week I'm in sunny (warm!) Los Angeles for the epic SpringOne Tour Los Angeles event and then it's off to Boston for meetings. We've got a lot to cover, as usual, so let's get to it!

-   Pivotal Cloud Foundry 2.2 is out and [it has smarter metrics support](https://content.pivotal.io/blog/how-pivotal-cloud-foundry-2-2-helps-you-improve-the-metrics-that-matter)
-   Do you want to begin in open-source or, at least, begin contributing to Spring Boot? We have a *ton* of [open-issues where we'd welcome external contributions](https://github.com/spring-projects/spring-boot/issues?q=is%3Aopen+is%3Aissue+label%3A%22status%3A+ideal-for-contribution%22)!
-   [Spring Cloud Data Flow 1.6 M1 and 1.5.2 released](https://spring.io/blog/2018/07/05/spring-cloud-data-flow-1-6-m1-and-1-5-2-released)
-   [Spring Tool Suite 3.9.5 released](https://spring.io/blog/2018/07/05/spring-tool-suite-3-9-5-released)
-   Follow the [progress on Graal supporting Spring Boot-based](https://jira.spring.io/browse/SPR-16991) applications for native images
-   [Spring Social End of Life Announcement](https://spring.io/blog/2018/07/03/spring-social-end-of-life-announcement)
-   I love this post that looks at how e-Bay does faster integration testing [of Spring Boot-based applications](https://www.ebayinc.com/stories/blogs/tech/your-own-spring-test-context/)
-   Check out [the migration to v2 in Zipkin 2.10](https://github.com/openzipkin/zipkin/releases/tag/2.10.1).
-   This is an oldie-but-a-goodie that I found myself needing today: it looks at [the Solace messaging technology's Spring Boot starters](https://solace.com/blog/devops/solace-java-meet-spring-boot-starters)
-   The Clever Cloud blog has [a nice post on how to adapt a JHipster Spring Boot and Angular-based](https://www.clever-cloud.com/blog/engineering/2018/07/09/specifics-paas-deployment-jhipster/) application for deployment on their platform
-   [https://twitter.com/springcentral/status/1016747383483322368](https://twitter.com/springcentral/status/1016747383483322368)
-   Learn how to install multicloud, enterprise-grade CloudFoundry with HAProxy. [Watch the video now](https://www.cloudfoundry.org/blog/video-how-to-install-multi-cloud-enterprise-grade-cloud-foundry-haproxy/).
-   Community legend Martin Deinum has a nice [post on loading properties with `PropertySource` and `.yaml` files](https://mdeinum.github.io/2018-07-04-PropertySource-with-yaml-files/)
-   The Spring Cloud-inspired Steeltoe.NET library [supports microservices patterns for .NET-based applications](https://twitter.com/steeltoeoss/status/1016476508292911104?s=12) and it is *hot*! It's had over 500k downloads on Nuget!
-   I love this post on using the [Chaos Monkey for Spring Boot on AWS EKS](https://medium.com/chaosiq/exploring-spring-boot-resiliency-on-aws-eks-6afcd385b92d)
-   Spring legend Thomas Darimont just upgraded his [example](https://github.com/thomasdarimont/spring-boot-admin-keycloak-example) demonstrating how to protect Spring Boot Admin 2.0.1 UI and Spring Boot 2.0.3 Actuator endpoints with Keycloak 4.1.0 to the latest versions
-   In which community hero Nicolas Frankel brilliantly revisits a fundamental question that we might at this point take for granted: [*why* should we use a framework or a dependency injection container](https://blog.frankel.ch/on-containers-frameworks/) like Spring? Why use a framework or a dependency injection container at all?
-   [Apache Camel 2.2.2](http://www.davsclaus.com/2018/07/apache-camel-222-released-with-spring.html?m=1) - the first with support for Spring Boot 2! - is now available!
-   Check out our very own Mario Gray's look [at authentication with reactive Spring Security](https://medium.com/@mgray_94552/reactive-spring-security-authentication-ba6e38ce0ae7) . This is one in a series of posts that look at security. Check out [the last installment here](https://medium.com/@mgray_94552/reactive-authorization-in-spring-security-943e6534aaeb).
-   This Codecentric post, by Benjamin Wilms, is a must-read manifesto for chaos engineering in general and [a useful background for anybody considering the Chaos Monkey for Spring Boot](https://blog.codecentric.de/en/2018/07/chaos-engineering/)
-   This is an oldie-but-a-goodie: [Oracle's serverless project, Project Fn, supports Spring Cloud Function](https://mjg123.github.io/2017/12/02/FnProject-Spring-Cloud-Function.html).
-   The amazing Jonas Hecht's article [on Spring Boot and Vue.js](https://twitter.com/jonashackt/status/1014956570801819648?s=12) is now available in the latest installment of the German-language magazine JavaMagazin.
-   This pull-request from Hibernate developer advocate Vlad Mihalcea looks *very* promising *and* the discussion [therein is illuminating, too](https://twitter.com/vlad_mihalcea/status/1014949370528182272?s=12)
-   Graal developer advocate Oleg Šelajev has a great post demonstrating some unique possibilities when running Spring Boot-based applications on Graal: [*Enhance your Java Spring application with R data science*](https://medium.com/graalvm/enhance-your-java-spring-application-with-r-data-science-b669a8c28bea).
-   Wojciech Marusarz, on the Nexocode blog, has a great post introducing [reactive programming](https://www.nexocode.com/blog/posts/reactive-programming/) (in terms of the basics, but with brief examples using Reactor and Spring)
-   This is an [interesting nugget of knowledge](https://twitter.com/gprussell/status/1014930928664182784?s=12) from Spring Integration lead [Gary Russell](http://twitter.com/gprussell): did you know that you could use Spring Integration/JMS to talk to QPID (AMQP 1.0) using [the QPID JMS bridge](https://qpid.apache.org/components/jms/index.html) and this should be a meaningful channel to connect Spring Integration to other AMQP 1.0-compliant brokers?
-   Check out this post on how to build services for Cloud Foundry and Kubernetes using [the Open Service Broker API](https://www.cloudfoundry.org/blog/video-how-to-services-cloud-foundry-kubernetes-using-open-service-broker-api/)
-   This is an interesting post on [Blockchain with Java using Ethereum, web3j and Spring Boot](https://t.co/AeUTDKFEDa?ssr=true). Good job Piotr Mińkowski!
-   There's now [PHP support for Spring Cloud Pipelines!](https://twitter.com/springcloud/status/1014886163872649217)
-   Be sure to register for this July 18 webinar: [*Bring Your Own Code vs. Bring Your Own Container*](https://twitter.com/cloudfoundry/status/1014800807630385152)
-   Congratulations to Thibaud Lepretre who released the first version (beta) supporting Spring Boot 2 `cas-security-spring-boot-starter` during the recent Pivotal Paris event! [See the project for details!](https://github.com/kakawait/cas-security-spring-boot-starter/blob/master/CHANGELOG.md)
