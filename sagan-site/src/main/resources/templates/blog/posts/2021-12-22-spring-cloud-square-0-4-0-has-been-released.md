# Spring Cloud Square 0.4.0 has been released

_Releases | Olga Maciaszek-Sharma |  December 22, 2021 | 2 Comments_

On behalf of the community, I am pleased to announce that the 0.4.0 release of [Spring Cloud Square](https://github.com/spring-projects-experimental/spring-cloud-square) is available today. The release can be found in [Maven Central](https://repo.maven.apache.org/maven2/org/springframework/cloud/spring-cloud-square-dependencies/0.4.0/). You can check out the 0.4.0 [release notes for more information](https://github.com/spring-projects-experimental/spring-cloud-square/releases/tag/v0.4.0).

## [](#notable-changes-in-040)Notable changes in 0.4.0:

-   Spring Native support adjustments for non-load-balanced use-cases ([PR](https://github.com/spring-cloud-incubator/spring-cloud-square/pull/23))
-   Support for configuring different WebClient.Builder beans (not shared) for each `@RetrofitClient` instance ([PR](https://github.com/spring-cloud-incubator/spring-cloud-square/pull/30))
-   Spring Cloud Sleuth integration for OkHttpClient [#36](https://github.com/spring-projects-experimental/spring-cloud-square/issues/36)

The documentation for this release can be found [here](https://spring-projects-experimental.github.io/spring-cloud-square/docs/current/reference/html/index.html).
