# Spring Framework 3.2.5 Released

_Releases | Phil Webb |  November 07, 2013 | 4 Comments_

The fifth maintenance release in the [Spring Framework](http://projects.spring.io/spring-framework/) 3.2.x line is now available via [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework%22) or the [SpringSource repository](http://github.com/spring-projects/spring-framework/wiki/Downloading-Spring-artifacts#wiki-resolving-spring-artifacts). This release includes nearly 50 bug fixes and improvements and we recommend that all users upgrade.
