# This Week in Spring - April 2nd, 2013

_Engineering | Josh Long |  April 02, 2013 | 0 Comments_

Welcome to another installment of *This Week in Spring* ! I've just returned from [Devoxx UK](http://www.devoxx.com/display/UK13/Home) and [Devoxx France](http://www.devoxx.com/display/FR13/Home) where I was very happy to talk to developers using Spring from all walks of industry. I also spoke at Skills Matter in London [on building web applications using Spring](http://skillsmatter.com/podcast/java-jee/spring-user-group-march). Thanks to Skills Matter, the London Spring User Group, and to the amazing [Rob Harrop](https://twitter.com/robertharrop) for having me, it was such a pleasure! The video from that [session is available online](http://skillsmatter.com/podcast/java-jee/spring-user-group-march) if you're interested.

4.  [Arjen Poutsma has announced that Spring Scala 1.0.0.M2 is now available](http://forum.springsource.org/showthread.php?136036-Spring-Scala-1-0-0-M2-has-been-released). The new release is full of features, including an updated `JdbcTemplate` wrapper, rich wrappers for `BeanFactory`, `ListableBeanFactory` and `ApplicationContext`, AOP advice wrappers with implicit conversions, support for Spring 3.2.2, and updated support in `FunctionConfiguration` for eager initialization, along with many other bug fixes and changes.

7.  The JavaBeat blog has a nice post [on how to use Spring's scopes](http://www.javabeat.net/2013/03/spring-bean-scopes-singleton-prototype/)

10.  The TechRecipes blog has a nice post on basic configuration of [a Spring application using Spring's XML configuration format](http://www.tech-recipes.com/rx/39525/basic-configuration-of-spring-application-context-xml-file-in-eclipse-with-maven/). This is admittedly *very* introductory.
