# Spring Session 2021.2.0-RC1, 2021.1.3 and 2021.0.6 released

_Releases | Eleftheria Stein-Kousathana |  April 20, 2022 | 0 Comments_

On behalf of the team, I’m pleased to announce the release of Spring Session `2021.2.0-RC1`, `2021.1.3` and `2021.0.6`. The `2021.2.0` release train is entering the RC phase. If you haven’t done so yet, please give it a try! The `2021.1.3` and `2021.0.6` releases deliver bug fixes and dependency upgrades. For your convenience, Spring Boot will pick up these artifacts with its upcoming releases.

The following modules were updated as part of `2021.2.0-RC1`:

-   Spring Session Core `2.7.0-RC1` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.7.0-RC1)
-   Spring Session Data Redis `2.7.0-RC1` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.7.0-RC1)
-   Spring Session JDBC `2.7.0-RC1` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.7.0-RC1)
-   Spring Session Hazelcast `2.7.0-RC1` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.7.0-RC1)
-   Spring Session MongoDB `2.7.0-RC1` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.7.0-RC1)
-   Spring Session for Apache Geode `2.7.0-RC1`

-   Spring Session MongoDB `2.6.3` - [release notes](https://github.com/spring-projects/spring-session/releases/tag/2.6.3)
