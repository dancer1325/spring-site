# This Week in Spring - June 3rd, 2025

_Engineering | Josh Long |  June 03, 2025 | 0 Comments_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*! I just finished recording my session with IntelliJ IDEA project lead Aleksey Stukalov about all the amazing features coming to IntelliJ IDEA to better support Java, Kotlin, and Spring developers.

It went off without a hitch (even though we're using a few early features), and I want to thank the Jetbrains team for the chance to participate.

Tomorrow, I speak at the amazing JSpring conference (sibling conference to the amazing JFall conference), too.

-   [Spring Cloud 2022.0.11 (aka Kulburn) has been released](https://spring.io/blog/2025/06/02/spring-cloud-2022-0-11-aka-kilburn-has-been-released)
-   [Spring Cloud 2025.0.0 (aka Northfields) has been released](https://spring.io/blog/2025/05/29/spring-cloud-2025-0-0-is-abvailable)
-   [Spring Cloud Gateway 3.1.10, 4.0.12, 4.1.8, 4.2.3, 4.3.0 Released, including fixes for CVE-2025-41235](https://spring.io/blog/2025/05/29/spring-cloud-gateway-2025-05-29-releases)
-   in last week's installment of *A Bootiful Podcast*, I talked with [the amazing Victor Rentea](https://spring.io/blog/2025/05/29/a-bootiful-podcast-victor-rentea)
-   [Spring Modulith 1.4 GA, 1.3.6, and 1.2.13 released](https://spring.io/blog/2025/05/28/spring-modulith-1-4-1-3-6-and-1-2-13-released)
-   never one to rest on his laurels, Wim Deblauwe just released Vite Spring Boot 0.9.0, which improves [support for Java Template Engine and updates to use Spring Boot 3.3.1.2](https://github.com/wimdeblauwe/vite-spring-boot/releases/tag/0.9.0)
-   Speaking of Jetbrains, I am super pleased with the [forthcoming support for Spring Data AOT repositories in the IDE](https://x.com/belyaev_andrey/status/1927072372101656643?s=12)
-   I love the forthcoming support for [reverse engineering JPA entities within IntelliJ IDEA](https://www.linkedin.com/posts/joshlong_intellijideaconf-activity-7335632625954340864-CfJI?utm_source=share&utm_medium=member_desktop&rcm=ACoAAACXdIABb8VbfXttpb7WA9oliQsXOS5rEgw) which, while not Spring related *per se*, will surely benefit a bunch of folks.
-   Another interesting thing upon which the Jetbrains team is working - but has not yet released - [is this epic and forthcoming Spring Modulith support](https://www.linkedin.com/posts/joshlong_intellijideaconf-activity-7335633019946356736-FE_R?utm_source=share&utm_medium=member_desktop&rcm=ACoAAACXdIABb8VbfXttpb7WA9oliQsXOS5rEgw)
