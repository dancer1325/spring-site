# Spring Session 3.3 goes GA

_Releases | Marcus Hert Da Coregio |  May 21, 2024 | 0 Comments_

On behalf of the Spring Session team and everyone who contributed to this release, I am delighted to announce the general availability of Spring Session `3.3.0` from Maven Central!

The 3.3 release brings:

-   Spring Session Backed implementation of `ReactiveSessionRegistry` ([docs](https://docs.spring.io/spring-session/reference/configuration/common.html#spring-session-backed-reactive-session-registry))
-   Add ReactiveRedisIndexedSessionRepository ([docs](https://docs.spring.io/spring-session/reference/configuration/reactive-redis-indexed.html))
-   Allow `PlatformTransactionManager` to be specified using `@SpringSessionTransactionManager` ([docs](https://docs.spring.io/spring-session/reference/configuration/jdbc.html#customizing-transaction-operations))
-   Improvements [in the JDBC documentation](https://docs.spring.io/spring-session/reference/3.3/configuration/jdbc.html)

This release will be included in the upcoming Spring Boot 3.3 GA release. We'd like to hear from you, so keep the feedback coming.
