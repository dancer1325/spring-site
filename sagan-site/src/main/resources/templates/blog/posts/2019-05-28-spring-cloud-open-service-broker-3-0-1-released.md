# Spring Cloud Open Service Broker 3.0.1 Released

_Releases | Roy Clarkson |  May 28, 2019 | 2 Comments_

We are pleased to announce the 3.0.1 release of Spring Cloud Open Service Broker. This release is a maintenance release that includes the following [updates](https://github.com/spring-cloud/spring-cloud-open-service-broker/issues?q=is%3Aclosed+milestone%3A3.0.1.RELEASE):

-   Improve getting started experience
-   Update reference documentation

Include the following Spring Boot starter:

```
Copy<dependency>
  <groupId>org.springframework.cloud</groupId>
  <artifactId>spring-cloud-starter-open-service-broker</artifactId>
  <version>3.0.1.RELEASE</version>
</dependency>
```
