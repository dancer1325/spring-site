# Spring GemFire 1.2 M1 Released for Java

_News | Costin Leau |  March 20, 2012 | 0 Comments_

Dear Spring Community,

I am pleased to announce that the first milestone of the Spring GemFire 1.2 project for Java is now available!

[Downloads](http://www.springsource.com/download/community?project=Spring GemFire&version=1.2.0.M1) | [JavaDocs](http://static.springsource.org/spring-gemfire/docs/1.2.0.M1/api/) | [Reference Documentation](http://static.springsource.org/spring-gemfire/docs/1.2.0.M1/reference/) | [Changelog](http://static.springframework.org/spring-gemfire/docs/1.2.0.M1/changelog.txt)

This new milestone introduces:

-   POJO [mapping](http://static.springsource.org/spring-gemfire/docs/1.2.0.M1/reference/mapping.html)
-   Repository [support](http://static.springsource.org/spring-gemfire/docs/1.2.0.M1/reference/repositories.html)

For more information see the reference [documentation](http://static.springsource.org/spring-gemfire/docs/1.2.0.M1/reference/).

Looking forward to your feedback!
