# Spring Framework 2.0.4 Released

_Releases | Juergen Hoeller |  April 10, 2007 | 0 Comments_

Dear Spring Community,

We are pleased to announce that Spring 2.0.4 has been released.  This is a bugfix and enhancement release, addressing all reported bugs against the Spring 2.0 series and introducing significant performance improvements.  We recommend upgrading to Spring 2.0.4 from previous 2.0.x releases.

![Spring 2.0 Released](http://static.springframework.org/images/spring20.png "Spring 2.0 Released")

[Download](http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=173644&release_id=500032) | [Documentation](http://static.springframework.org/spring/docs/2.0.4/reference/index.html) | [Changelog](http://static.springframework.org/spring/docs/2.0.4/changelog.txt)

Regarding the performance improvements, repeated creation of Spring bean instances is up to **12 times faster** in this release than previous versions of Spring 2.0. AspectJ-based weaving performance has also increased by a significant factor.  

Please see the [changelog](http://static.springframework.org/spring/docs/2.0.x/changelog.txt) and [JIRA roadmap](http://opensource.atlassian.com/projects/spring/browse/SPR?report=com.atlassian.jira.plugin.system.project:roadmap-panel) for all the details of the 77 issues addressed in this release.
