# A Bootiful Podcast: SpringOne 2020, Josh's book "Reactive Spring," and Microsoft Java Architect and fellow Java Champion Jonathan Giles

_Engineering | Josh Long |  September 04, 2020 | 0 Comments_

Hi, Spring fans! Welcome to another installment of a Bootiful Podcast! In this episode [Josh Long (@starbuxman)](http://twitter.com/starbuxman) talks about the massive, virtual SpringOne 2020 event (tens of thousands of people in attendance!), his just-completed book ["Reactive Spring"](http://www.reactivespring.io), and then talks to Microsoft's [Jonathan Giles (@jonathangiles)](https://twitter.com/jonathangiles) .
