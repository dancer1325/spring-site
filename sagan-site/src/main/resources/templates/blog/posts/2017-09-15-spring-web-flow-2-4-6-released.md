# Spring Web Flow 2.4.6 released

_Releases | Rossen Stoyanchev |  September 15, 2017 | 0 Comments_

Spring Web Flow 2.4.6 was released earlier today containing a [security fix](https://pivotal.io/security/cve-2017-8039). Applications that use explicit data bindings through the `<binder>` element in flow definitions are not affected. Those that do rely on default bindings should upgrade as soon as possible.
