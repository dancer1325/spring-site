# Spring Framework 4.2.2, 4.1.8 and 3.2.15 available now

_Releases | Stéphane Nicoll |  October 15, 2015 | 2 Comments_

On behalf of the team I am pleased to announce that the Spring Framework [4.2.2](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=15267), [4.1.8](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=15227) and [3.2.15](https://jira.spring.io/secure/ReleaseNote.jspa?projectId=10000&version=15325) maintenance releases [are available now](http://projects.spring.io/spring-framework).

Aside from fixing various minor issues across the framework, these releases are designed to prevent [Reflected File Download](https://www.trustwave.com/Resources/SpiderLabs-Blog/Reflected-File-Download---A-New-Web-Attack-Vector/) attacks; check the corresponding [CVE report for details](http://pivotal.io/security/cve-2015-5211). We recommend an immediate upgrade for all Spring Framework 3.x and 4.x users.

Spring Framework 4.2.2 also provides WebSocket support for the recently released servers [Undertow 1.3 and GlassFish 4.1.1](https://spring.io/blog/2015/10/14/spring-on-glassfish-4-1-1-and-wildfly-10-undertow-1-3), as well as fine-tuned CORS and JavaScript support. This release serves as the foundation of the upcoming [Spring Boot 1.3](https://github.com/spring-projects/spring-boot/milestones); stay tuned!
