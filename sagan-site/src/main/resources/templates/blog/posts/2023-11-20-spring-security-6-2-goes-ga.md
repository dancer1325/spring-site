# Spring Security 6.2 goes GA

_Releases | Marcus Hert Da Coregio |  November 20, 2023 | 0 Comments_

On behalf of the Spring Security team, it is my pleasure to announce that Spring Security 6.2.0 is generally available from Maven Central now! The 6.2 generation comes with improvements that you can check on the [What's New](https://docs.spring.io/spring-security/reference/whats-new.html) section of the documentation.

Spring Security `6.2` has upgraded its Spring Framework baseline to `6.1` along with Project Reactor `2023.0.0` and Micrometer `1.12.0` while requiring Java 17 as minimum platform version and supporting up to Java 21 for Virtual Threads support.

This release will be included in the upcoming Spring Boot `3.2` GA release. We'd like to hear from you, so keep the feedback coming.
