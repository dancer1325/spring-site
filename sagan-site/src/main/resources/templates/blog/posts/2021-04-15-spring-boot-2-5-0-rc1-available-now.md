# Spring Boot 2.5.0-RC1 available now

_Releases | Phil Webb |  April 16, 2021 | 6 Comments_

In preparation for the GA release on May 20 2021, I am pleased to announce that Spring Boot 2.5.0-RC1 has been released and is available from our [milestone repository](https://repo.spring.io/milestone/). This release [adds a number of new features and bug fixes](https://github.com/spring-projects/spring-boot/releases/tag/v2.5.0-RC1).

Highlights of the RC include:

-   A new actuator endpoint for Quartz
-   Support for database initialization with R2DBC
-   Micometer metrics support for Spring Data repositories and MongoDB
-   Support for Gradle 7.0

For a complete list of changes and upgrade instructions, please see the [Spring Boot 2.5.0-RC1 Release Notes](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.5.0-RC1-Release-Notes) on the wiki and the [updated reference documentation](https://docs.spring.io/spring-boot/docs/2.5.0-RC1/reference/html/).

If you want to get started with 2.5 and try out the new features, you can bootstrap a new project on [start.spring.io](https://start.spring.io/#!type=maven-project&language=java&platformVersion=2.5.0.RC1&packaging=jar&jvmVersion=11&groupId=com.example&artifactId=demo&name=demo&description=Demo%20project%20for%20Spring%20Boot&packageName=com.example.demo).
