# Spring Web Flow 2.3.3 Released

_Releases | Rossen Stoyanchev |  January 27, 2014 | 0 Comments_

I'm pleased to announce the availability of Spring Web Flow 2.3.3. This is the third maintenance release of the 2.3.x branch extending compatibility to Spring Framework 4 and also JSF 2.2. The [spring-webflow-samples](https://github.com/spring-projects/spring-webflow-samples) Github repository has been updated to use the above versions.
