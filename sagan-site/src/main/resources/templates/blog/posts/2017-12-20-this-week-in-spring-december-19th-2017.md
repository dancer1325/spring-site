# This Week in Spring - December 19th, 2017

_Engineering | Josh Long |  December 20, 2017 | 0 Comments_

Hi Spring fans! Welcome to another installment of *This Week in Spring*! I've just returned from London where I spent Monday talking to Pivotal customers and helping teams build software in terms of Spring.

Now I'm home for the holidays. Can you believe we're only a few weeks away from 2018!? Where has the time gone! Chanukah, Christmas and Boxing Day will all be over and Kwanzaa will have started before next week's installment of *This Week in Spring*, so on behalf of the Spring team and Pivotal as a whole, let me wish you the warmest and happiest of holidays!

We've got a lot to tide you over if you happen to have some time off, so without further ado, let's get to it.

-   In this latest installment of [*Spring Tips*](http://twitter.com/SpringTipsLive), I look at one of the most important [features in Spring Boot 2.0](https://spring.io/blog/2017/12/15/spring-tips-bootiful-banners), and indeed in Spring, ever: the ASCII banners! Do NOT miss this one!
-   I loved this talk introducing [Consumer Driven Contract (CDC)](https://www.youtube.com/watch?v=JEmpIDiX7LU&t=2329s&list=PLAdzTan_eSPQ2uPeB0bByiIUMLVAhrPHL&index=37) by Spring Cloud Contract lead Marcin Grzejszczak and Adib Saikali.
-   I really liked this first blog post [for Project Riff](https://projectriff.io/blog/first-post-announcing-riff-0-0-2-release/), a serverless platform that we announced at SpringOne Platform 2017.
-   This is an oldie but might be useful to some. In this post, Redmonk's James Governor and [I look at *cloud-native* with Spring Boot](https://www.youtube.com/watch?v=sWbeG782Vec).
-   In this SpringOne Platform 2017 talk, Spring Framework Juergen Hoeller looks [at Spring Framework 5.0 on Java 9](https://www.youtube.com/watch?v=0-sbPBf81KA).
-   This is an interesting sample Spring Boot project [that uses the Giraphe CMS framework and Spring Boot](https://github.com/creactiviti/graffiti). Giraphe is a Spring Boot based headless CMS Framework.
-   Baeldung have [updated their REST with Spring tutorial](http://www.baeldung.com/rest-with-spring-series/), and it's always worth a look.
-   The good Dr. Syer has put together a very nice tutorial looking at [how to incept a Spring Boot project with Angular](https://github.com/dsyer/spring-boot-angular).
-   IntelliJ engineer Yann Cébron knows how to [start the journey to production](https://twitter.com/yanncebron/status/940680833731584000): *start.. dot.. spring.. dot.. io*!
-   It's that time of year again! All [the SpringOne Platform 2017 videos are up and available](https://springoneplatform.io/)! Binge watch 'em all and give yourself the gift of knowledge!
-   Spring Data ninja Mark Paluch, who did a lot of the work behind the reactive Spring Data implementations, gave a great SpringOne Platform 2017 talk, [*Under the Hood of Reactive Data Access*](https://www.youtube.com/watch?v=BKYXu25sziI)
-   Spring community legend Jenn Strater gave a talk on \_Documenting RESTful APIs with Spring REST Docs\]([https://www.youtube.com/watch?v=CaARz49u1Mc&list=PLSzU7snapMVZ4n5s5glQz4Dd\_C3572O\_2](https://www.youtube.com/watch?v=CaARz49u1Mc&list=PLSzU7snapMVZ4n5s5glQz4Dd_C3572O_2)) for SpringOne Platform 2017 that is well worth a watch.
-   The Linux Foundation has a nice course on building [cloud native software with Cloud Foundry](https://www.edx.org/node/33041.json#!?_lrsc=f867a0a2-befd-4cec-830e-4d6bafd2bf12&utm_source=employee-social&utm_medium=twitter&utm_campaign=employee_advocacy).
-   InfoQ's Victor Grazi looks at the [new Spring Security 5.0 release](https://www.infoq.com/news/2017/12/Spring-Security-5.0.0)
-   The epages blog has a nice post that looks at [RESTful API documentation with Spring REST Docs and RAML](https://developer.epages.com/blog/api-experience/restful-api-documentation-with-spring-rest-docs-and-raml/)
-   InfoQ's Victor Grazi had a nice interview with [Reactor engineer Simon Basle and marketing manager Pieter Humphrey](https://www.infoq.com/news/2017/12/s1p-Conference-reactor-webflux)
-   Spring ninja Thomas Risberg put together a [cool Project Riff - our recently announced serverless platform - demo that you can inspect here](https://github.com/trisberg/s1p2017-jdbc/tree/master/jdbc-writer)
-   the Open at Microsoft blog looks at Pivotal Cloud Foundry's latest [release supporting Microsoft Azure](https://open.microsoft.com/2017/12/13/microsoft-open-source-weekly-kubecon/)
