# Spring Cloud Brixton.RC2 now available

_Engineering | Dave Syer |  April 19, 2016 | 3 Comments_

The latest milestone version of Spring Cloud is "Brixton.RC2" and it is available now from the usual repositories (details in the [home page](http://projects.spring.io/spring-cloud/)). This is a bugfix release building on the RC1, and there are no notable new features. If all goes well we'd like to get to a RELEASE in a week or two.
