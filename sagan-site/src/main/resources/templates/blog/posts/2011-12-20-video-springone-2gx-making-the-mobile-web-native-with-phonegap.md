# Video: SpringOne 2GX - Making the Mobile Web Native with PhoneGap

_News | Adam Fitzgerald |  December 20, 2011 | 0 Comments_

[![SpringOne 2GX Video: Making the Mobile Web Native with PhoneGap](http://www.springsource.org/files/SpringOne2GX2011MakingtheMobileWebNativeWithPhoneGap.png)](http://www.infoq.com/presentations/Making-the-Mobile-Web-Native-with-PhoneGap)

Here is more great content from SpringOne 2GX 2011, this time a detailed follow-up to the mobile demonstrations in the [technical keynote](http://www.infoq.com/presentations/SpringOne-2GX-Keynote-Next-Generation-Applications). This video presentation is by [Keith Donald](http://blog.springsource.org/author/keithd/) and [Roy Clarkson](http://blog.springsource.org/author/rclarkson/), who have been doing the advanced web work for Spring. In [Making the Mobile Web Native with PhoneGap](http://www.infoq.com/presentations/Making-the-Mobile-Web-Native-with-PhoneGap) Keith and Roy cover:

-   The emergence of HTML 5 based applications as a way to do cross platform mobile development
-   PhoneGap APIs and Plugins for development that combine HTML 5 with native Android and iOS capabilities
-   PhoneGap build process and remote debugging
-   A complete walk-through for the [Spring html5expense sample application](https://github.com/SpringSource/html5expense)

Many thanks to [InfoQ](http://www.infoq.com) for coming to Chicago to record so many of the fantastic [SpringOne 2GX presentations](http://www.infoq.com/springone_2gx_2011/).
