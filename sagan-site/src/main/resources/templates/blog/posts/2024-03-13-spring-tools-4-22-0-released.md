# Spring Tools 4.22.0 released

_Releases | Martin Lippert |  March 13, 2024 | 1 Comment_

Dear Spring Community,

I am happy to announce the 4.22.0 release of the Spring Tools 4 for Eclipse, Visual Studio Code, and Theia.

#### [](#important-highlights)important highlights

-   *(Spring Boot)* **Add Spring Boot Starters...** InlayHint in Spring Boot projects `pom.xml` next to `<dependencies>` tag (VSCode only at the moment)
-   *(Spring Boot)* early TestJars support [#1197](https://github.com/spring-projects/sts4/issues/1197)
-   *(Eclipse)* fixed: Spring Tool Suite 17 with maven no longer copies application.properties file to the target folder [#929](https://github.com/spring-projects/sts4/issues/929)

#### [](#updates-to-the-spring-tools-4-for-eclipse-distribution)updates to the Spring Tools 4 for Eclipse distribution

-   updated to Eclipse 2024-03 release ([new and noteworthy](https://eclipseide.org/release/noteworthy/))

Detailed changes can be found in the release notes: [https://github.com/spring-projects/sts4/releases/tag/4.22.0.RELEASE](https://github.com/spring-projects/sts4/releases/tag/4.22.0.RELEASE)

#### [](#downloads)downloads

To download the distribution for Eclipse and find links to the marketplace entries for Visual Studio Code and Theia, please go visit:

-   Spring Tools 4: [https://spring.io/tools/](https://spring.io/tools/)

Spring Tools 4.22.1 is scheduled to be released in early May 2024.

Enjoy!
