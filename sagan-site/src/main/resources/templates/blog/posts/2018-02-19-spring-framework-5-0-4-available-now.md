# Spring Framework 5.0.4 available now

_Releases | Stéphane Nicoll |  February 19, 2018 | 0 Comments_

On behalf of the team, I am pleased to announce that Spring Framework 5.0.4 [is available now](http://projects.spring.io/spring-framework).

This maintenance release is the base for the upcoming second release candidate of Spring Boot 2 and [includes 59 fixes and selected improvements](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=16629).
