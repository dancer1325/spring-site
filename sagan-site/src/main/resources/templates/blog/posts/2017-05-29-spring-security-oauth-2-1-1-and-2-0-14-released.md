# Spring Security OAuth 2.1.1 and 2.0.14 Released

_Releases | Joe Grandja |  May 29, 2017 | 1 Comment_

On behalf of the community, I’m pleased to announce the releases of Spring Security OAuth 2.1.1 and 2.0.14.

Both are maintenance releases that primarily include bug fixes and minor improvements.
