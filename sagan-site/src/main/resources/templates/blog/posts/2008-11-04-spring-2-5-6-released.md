# Spring 2.5.6 Released

_Releases | Adam Fitzgerald |  November 04, 2008 | 5 Comments_

We are pleased to announce that Spring 2.5.6 has been released.  
[Download](http://www.springsource.com/download?project=Spring%20Framework) | [Support](http://www.springsource.com/products/enterprise) | [Documentation](http://static.springframework.org/spring/docs/2.5.x/reference/index.html) | [Changelog](http://static.springframework.org/spring/docs/2.5.x/changelog.txt)  
This release primarily fixes some bugs and also adds some new features:  

-   Over 100 [resolved issues](http://jira.springframework.org/browse/SPR/fixforversion/10968)
-   Updates to match up with latest version of some third party libraries, for example:
    -   [AspectJ](http://www.eclipse.org/aspectj/) 1.6.2
    -   [OpenJPA](http://openjpa.apache.org/) 1.1.0
    -   [EclipseLink](http://www.eclipse.org/eclipselink/) 1.0.1
-   Revised OSGi manifest headers

  
Please refer to the changelog for additional details.  

Enjoy!
