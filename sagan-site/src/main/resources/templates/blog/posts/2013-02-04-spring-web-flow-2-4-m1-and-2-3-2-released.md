# Spring Web Flow 2.4 M1 and 2.3.2 Released

_Releases | Rossen Stoyanchev |  February 04, 2013 | 0 Comments_

A minor maintenance release of Spring Web Flow 2.3.2 is now available via Maven and for download. See the [Changelog](http://static.springsource.org/spring-webflow/docs/2.3.x/changelog.txt) for the list of changes.

A first milestone of Spring Web Flow 2.4 is also available through the [SpringSource milestone repository](https://github.com/SpringSource/spring-framework/wiki/SpringSource-repository-FAQ). See the [Changelog](http://static.springsource.org/spring-webflow/docs/2.4.0.M1/changelog.txt) for the full list of changes.

Spring Web Flow samples have been separated from the distribution and into a separate [Github project](https://github.com/SpringSource/spring-webflow-samples). In addition the booking-mvc sample has been updated to use Thymeleaf thanks to Thymeleaf's project lead [Daniel Fernández](https://twitter.com/danfenz).
