# Feature Tour

_Releases | Chloe Jackson |  August 26, 2011 | 0 Comments_

Spring enables you to focus on your business problem rather than the plumbing that connects components and systems. Take a tour of Spring’s key features from the core framework to infrastructure and data services and learn how to build, run and manage your modern Java applications. [Getting started](/get-started) is easy and you can do it now.

 

[![](http://www.springsource.org/sites/all/themes/s2org11/images/button_getstarted_shadow.png)](/get-started)

## Spring Framework Features

The Spring Framework helps you build Java Applications faster because it allows you to focus on your business problem rather than the plumbing code that connects components and systems. The features of Spring include:

[Modern Web](/features/modern-web)

Complete support for modern applications including REST, HTML 5, conversations and AJAX.  
[*See more...*](/features/modern-web)

[Data Access](/features/data-access)

Supports traditional RDBMS as well as new NoSQL solutions, map-reduce frameworks and cloud based data services.  
[*See more...*](/features/data-access)

[Integration](/features/integration)

Enterprise orchestration and adapters for distributed applications, asynchronous message-based applications, and batch applications.  
[*See more...*](/features/integration)

[Mobile](/features/mobile)

Web support for mobile client platforms including Android and iPhone.  
[*See more...*](/features/mobile)

[Social](/features/social)

Integration with Facebook, Twitter, LinkedIn, and other prominent social networks.  
[*See more...*](/features/social)

[Security](/features/security)

Authorization control for all tiers and authentication integration to dozens of providers.  
[*See more...*](/features/security)

[Cloud Ready](/features/cloud-ready)

Spring applications are supported on all popular cloud platforms like Cloud Foundry, Google App Engine and Amazon EC2.  
[*See more...*](/features/cloud-ready)

 

 

## Spring Key Benefits

![](http://www.springsource.org/sites/all/themes/s2org11/images/FeatureTour_Benefit_Modularity.png)  

**Modularity**

Plain old Java Objects keep your code concise, simple and modular

 

![](http://www.springsource.org/sites/all/themes/s2org11/images/FeatureTour_Benefit_Productivity1.png)  

**Productivity**

Over 70% of developers report productivity gains and reduction in time to deploy with Spring

 

![](http://www.springsource.org/sites/all/themes/s2org11/images/FeatureTour_Benefit_Portability.png)  

**Portability**

Applications run on Tomcat, all Java EE servers as well as cloud platforms

 

![](http://www.springsource.org/sites/all/themes/s2org11/images/FeatureTour_Benefit_Testability.png)  

**Testability**

Cleanly expressed dependencies make unit and integration testing easier
