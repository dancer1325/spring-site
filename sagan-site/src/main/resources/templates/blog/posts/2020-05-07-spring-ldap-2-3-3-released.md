# Spring LDAP 2.3.3 Released

_Releases | Rob Winch |  May 07, 2020 | 1 Comment_

On behalf of the community, I’m pleased to announce the release of Spring LDAP 2.3.3 ([release notes](https://github.com/spring-projects/spring-ldap/releases/tag/2.3.RELEASE)). The release delivers bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
