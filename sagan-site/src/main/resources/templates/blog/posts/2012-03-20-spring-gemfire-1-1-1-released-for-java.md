# Spring GemFire 1.1.1 Released for Java

_Releases | Costin Leau |  March 20, 2012 | 0 Comments_

Dear Spring Community,

I am pleased to announce the GA release of Spring GemFire 1.1.1 project is now available!

[Downloads](http://www.springsource.com/download/community?project=Spring GemFire&version=1.1.1.RELEASE) | [JavaDocs](http://static.springsource.org/spring-gemfire/docs/1.1.1.RELEASE/api/) | [Reference Documentation](http://static.springsource.org/spring-gemfire/docs/1.1.1.RELEASE/reference/) | [Changelog](http://static.springframework.org/spring-gemfire/docs/1.1.1.RELEASE/changelog.txt)

To learn more about the project, visit the Spring GemFire [Homepage](http://www.springsource.org/spring-gemfire).

1.1.1 is mainly a maintenance release providing several enhancements on the configuration front. See the changelog for more details.
