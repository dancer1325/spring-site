# Now Available: SpringSource Tool Suite 2.2.1

_Releases | Adam Fitzgerald |  November 11, 2009 | 0 Comments_

Today [SpringSource](http://www.springsource.com) has released an update to [SpringSource Tool Suite](http://www.springsource.com/products/sts) (STS), the best Eclipse-powered development environment for building Spring, Groovy and Grails powered enterprise applications. The latest version (2.2.1) is now available for [download](http://www.springsource.com/products/springsource-tool-suite-download) and includes:

-   Hot on the heels of the latest [Spring Roo](/roo) release, STS now supports Spring Roo 1.0.0.RC3 and enhances the Roo shell with hyperlinking to application files letting you open them for editing with a single click.
-   The STS [Grails](http://www.grails.org) tools have been updated to include a GSP editor for easy page template editing. The Grails tools can be downloaded as an extension to STS.
-   Support for the latest milestones of [SpringSource dm Server](/dmserver): 2.0.0.M6.
-   Improvements to the [Cloud Foundry](http://www.cloudfoundry.com) integration that Charles Lee discussed in today's [blog post](http://blog.springsource.com/2009/11/10/cloud-foundry-integrated-into-developer-tools/).
