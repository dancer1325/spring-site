# Spring Framework 6.1.19 and 6.2.6 Available Now

_Releases | Brian Clozel |  April 17, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.1.19` and `6.2.6` are available now.

Spring Framework `6.1.19` ships with [11 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.1.19). This version will be shipped next week with Spring Boot `3.3.11`.

Spring Framework `6.2.6` ships with [35 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.6). This version will be shipped next week with Spring Boot `3.4.5` and `3.5.0-RC1`. It's very unusual for us to ship new features in maintenance versions, but this version also [brings first-class support for Bean Overrides with `@ContextHierarchy`](https://github.com/spring-projects/spring-framework/pull/34723). Please refer to the ["Context hierarchies with bean overrides" documentation section](https://docs.spring.io/spring-framework/reference/6.2/testing/testcontext-framework/ctx-management/hierarchies.html#testcontext-ctx-management-ctx-hierarchies-with-bean-overrides) for more details.
