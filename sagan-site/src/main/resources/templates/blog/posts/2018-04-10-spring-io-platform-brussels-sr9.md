# Spring IO Platform Brussels SR9

_Releases | Andy Wilkinson |  April 10, 2018 | 0 Comments_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR9 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Boot 1.5.12
-   Spring Framework 4.3.16
