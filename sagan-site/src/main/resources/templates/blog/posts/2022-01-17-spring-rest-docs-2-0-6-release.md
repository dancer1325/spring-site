# Spring REST Docs 2.0.6.RELEASE

_Releases | Andy Wilkinson |  January 17, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring REST Docs `2.0.6.RELEASE` has been released and is now available from Maven Central.

This release includes [18 bug fixes and documentation improvements](https://github.com/spring-projects/spring-restdocs/releases/tag/v2.0.6.RELEASE). Thanks to all those who have contributed with issue reports and pull requests.
