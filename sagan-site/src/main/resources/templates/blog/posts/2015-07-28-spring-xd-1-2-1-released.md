# Spring XD 1.2.1 Released

_Releases | Mark Pollack |  July 28, 2015 | 0 Comments_

On behalf of the Spring XD team, I am very pleased to announce the general availability of Spring XD 1.2.1!

This release includes [bug fixes and some small enhancements](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=11401&version=15199):

-   Support for registering custom Kryo Serializers
-   Configuration options to fine tune Kryo performance
-   Update several dependencies to latest versions - Spring Integration, Boot, reactor, Groovy
-   Add jars for Avro and Snappy compression to Sqoop job submission
-   Fix admin UI access when security is enabled
-   Improved pagination for containers and jobs

You can [download the zip distribution](http://repo.spring.io/release/org/springframework/xd/spring-xd/1.2.1.RELEASE/spring-xd-1.2.1.RELEASE-dist.zip) or install on OSX [using homebrew](http://docs.spring.io/spring-xd/docs/1.2.0.RELEASE/reference/html/#osx-homebrew-installation) On RHEL/CentOS you can [install using rpm](https://github.com/spring-projects/spring-xd/wiki/Installing-Spring-XD-using-RPM-on-RHEL-CentOS-v.-6.x).

Feedback is very important, so please get in touch with questions and comments via

-   [StackOverflow](http://stackoverflow.com/questions/tagged/spring-xd) `spring-xd` tag
