# Spring Integration 2.2.0.M1 is released

_Releases | Oleg Zhurakousky |  May 17, 2012 | 0 Comments_

We are pleased to announce the first milestone release of Spring Integration 2.2 stream - [Spring Integration 2.2.0.M1](http://static.springsource.org/spring-integration/docs/2.2.0.M1/reference/htmlsingle/). [Spring Integration](http://www.springsource.org/spring-integration) provides an extension of the Spring programming model to support the well-known [Enterprise Integration Patterns](http://www.eaipatterns.com/)

Aside from the usual bug fixes this release brings quite a few new features and upgrades:

-   Dependency upgrades such as Spring 3.1.0.RELEASE, Spring AMQP 1.1.0.RELEASE, Spring Data Mongo 1.1.0.M1, Spring Gemfire 1.1.1.RELEASE
-   JPA support
-   Initial support for graceful shutdown of integration context
-   Support for 'Publisher Confirms and Returns' based on new features of Spring AMQP
-   etc.

This release also contains many improvements. You can see full release notes [here](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=12343)

For more information about Spring Integration, refer to the project's [home page](http://www.springsource.org/spring-integration)
