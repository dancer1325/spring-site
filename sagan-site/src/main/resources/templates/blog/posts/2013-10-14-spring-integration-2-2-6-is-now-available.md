# Spring Integration 2.2.6 is Now Available

_Releases | Gary Russell |  October 14, 2013 | 0 Comments_

Spring Integration 2.2.6.RELEASE is now available.

This is the latest maintenance release in the current release (2.2.x) branch and includes a number of important bug fixes for (S)FTP, Scripting, TCP, Events, and Mongodb support.

Please see the [release notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=14236) and [the project page](http://projects.spring.io/spring-integration/) for more information.

The artifacts are in the [Spring Maven Repository](http://repo.springsource.org/release) and Maven Central.
