# Spring Security 4.1.3 Released

_Releases | Rob Winch |  August 24, 2016 | 1 Comment_

I’m pleased to announce the release of Spring Security 4.1.3.RELEASE which updates libraries & resolves some minor issues including fixes for the new [MvcRequestMatcher](http://docs.spring.io/spring-security/site/docs/4.1.x/reference/htmlsingle/#mvc-requestmatcher).

For details refer to the [changelog](https://github.com/spring-projects/spring-security/issues?q=is%3Aclosed+milestone%3A4.1.3).

# [](#contributions)[](#contributions)Contributions

Without the community we couldn’t be the successful project we are today. I’d like to thank everyone that created issues & provided feedback.
