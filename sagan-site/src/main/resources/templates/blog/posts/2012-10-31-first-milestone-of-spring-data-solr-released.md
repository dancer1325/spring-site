# First milestone of Spring Data Solr released

_Releases | Oliver Drotbohm |  October 31, 2012 | 0 Comments_

I am pleased to announce the availablity of the first milestone of a Spring Data Solr project. It's the first community-driven Spring Data module lead by [Christoph Strobl](https://twitter.com/stroblchristoph). It's essentially an implementation of the Spring Data repository API to access Solr stores and provides the same programming model you are already used to from Spring Data JPA, MongoDB, Neo4j and Gemfire. Beyond that Spring Data Solr provides a lower level `SolrTemplate` as well as abstractions to easily bootstrap an embedded Solr server.

[Maven artifacts](http://repo.springsource.org/libs-milestone/org/springframework/data/spring-data-solr/1.0.0.M1/) | [JavaDocs](http://static.springsource.org/spring-data/solr/docs/1.0.0.M1/api/) | [Reference Documentation](http://static.springsource.org/spring-data/solr/docs/1.0.0.M1/reference/html/) | [Changelog](http://static.springsource.org/spring-data/solr/docs/1.0.0.M1/changelog.txt) | [JIRA](https://jira.springsource.org/browse/DATASOLR)

We looking forward to community-driven Spring Data module implementations for other stores. Please get in touch with us through the forums in case you'd like to see a particular store supported and can commit to actively maintaining the project.
