# Spring Boot 3.0.7 available now, fixing CVE-2023-20883

_Releases | Andy Wilkinson |  May 18, 2023 | 1 Comment_

On behalf of the team and everyone who has contributed, I'm happy to announce that Spring Boot `3.0.7` has been released and is now available from Maven Central.

This release includes [54 bug fixes, documentation improvements, and dependency upgrades](https://github.com/spring-projects/spring-boot/releases/tag/v3.0.7). Thanks to all those who have contributed with issue reports and pull requests.

This release includes a fix for [CVE-2023-20883: Spring Boot Welcome Page DoS Vulnerability](http://spring.io/security/cve-2023-20883).
