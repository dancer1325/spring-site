# This Week in Spring - August 22, 2023

_Engineering | Josh Long |  August 23, 2023 | 0 Comments_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*! And, would you believe it, I'm writing this at SpringOne 2023, in sunny Las Vegas, Nevada. This is the *first* in-person SpringOne since 2019, and I'm so, so, so glad to be here!

We've got a *ton* of things to get into this week, let's dive right into it!

-   This year we're celebrating twenty (20!) years since the early builds of Spring Framework 1.0, and ten (10!!) years since the early builds of Spring Boot 1.0. Did you see this amazing easter-egg, [*The History of Spring?*](https://springone.io/history-of-spring)? It's amazing!
-   We announced a new project called Spring AI that provides a convenient abstraction to allow developers to work with Large Language Models like Azure OpenAI and regular OpenAI, and - hopefully - more in the future. I did a live stream with its founder, [Dr. Mark Pollack, here](https://www.youtube.com/watch?v=0P8UU5vkvI8), and we did [a demo you can look at here](https://github.com/coffee-software-show/spring-ai.git)
-   [Spring Authorization Server 1.1.2 available now](https://spring.io/blog/2023/08/22/spring-authorization-server-1-1-2-available-now)
-   [Spring Data 2023.1.0-M2, 2023.0.3, 2022.0.9, and 2021.2.15 released](https://spring.io/blog/2023/08/18/spring-data-2023-1-0-m2-2023-0-3-2022-0-9-and-2021-2-15-released)
-   [Spring Framework 6.1 M4 released](https://spring.io/blog/2023/08/17/spring-framework-6-1-m4-released)
-   [Spring Modulith 1.0 GA released](https://spring.io/blog/2023/08/21/spring-modulith-1-0-ga-released)
-   [Spring Modulith 1.0 RC2 released](https://spring.io/blog/2023/08/16/spring-modulith-1-0-rc2-released)
-   [Tackling the OAuth2 Client component model in Spring Security](https://spring.io/blog/2023/08/22/tackling-the-oauth2-client-component-model-in-spring-security)
-   [Oracle Backend for Spring Boot - Developer Preview](https://oracle.github.io/microservices-datadriven/spring/)
