# Spring IO Platform 1.1.1 released

_Releases | Andy Wilkinson |  January 15, 2015 | 0 Comments_

We are pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) 1.1.1.RELEASE is now available from both [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3Aio.spring.platform).

The main highlight of this release is the addition of [Spring Session 1.0.0.RELEASE](https://spring.io/blog/2015/01/08/spring-session-1-0-0-release) to the Platform.

In addition, this release also upgrades the versions of a number of the projects to pick up their latest maintenance releases:

-   Spring AMQP 1.4.2.RELEASE
-   Spring Boot 1.2.1.RELEASE
-   Spring Framework 4.1.4.RELEASE
-   Spring Integration 4.1.2.RELEASE

The versions of many third-party dependencies have also been updated.
