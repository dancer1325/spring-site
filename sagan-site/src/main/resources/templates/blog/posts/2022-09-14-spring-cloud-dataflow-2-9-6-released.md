# Spring Cloud Dataflow 2.9.6 Released

_Releases | Chris Bono |  September 14, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring Cloud Dataflow `2.9.6` has been released and is now available from Maven Central.

This release updates to Postgres SQL driver `42.2.26` to address [CVE-2022-31197](https://github.com/advisories/GHSA-r38f-c4h4-hqq2). See the [release notes](https://github.com/spring-cloud/spring-cloud-dataflow/releases/tag/v2.9.6) for more information.
