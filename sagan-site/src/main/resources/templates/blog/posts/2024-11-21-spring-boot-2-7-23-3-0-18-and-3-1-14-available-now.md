# Spring Boot 2.7.23, 3.0.18, and 3.1.14 available now

_Releases | Andy Wilkinson |  November 21, 2024 | 0 Comments_

On behalf of the team, I am pleased to announce that Spring Boot 2.7.23, 3.0.18, and 3.1.14 have been released for [support](https://www.vmware.com/products/app-platform/tanzu-spring) customers. The releases are available from [https://packages.broadcom.com](https://packages.broadcom.com).

-   Spring Boot [2.7.23](https://enterprise.spring.io/projects/spring-boot/changelog/2.7.23) ships with 15 bug fixes, dependency upgrades, and documentation improvements.
-   Spring Boot [3.0.18](https://enterprise.spring.io/projects/spring-boot/changelog/3.0.18) ships with 28 bug fixes, dependency upgrades, and documentation improvements.
-   Spring Boot [3.1.14](https://enterprise.spring.io/projects/spring-boot/changelog/3.1.14) ships with 34 bug fixes, dependency upgrades, and documentation improvements.

The next 2.7.x, 3.0.x, and 3.1.x releases for support customers are planned for 3 months' time. The first 3.2.x release for support customers is also planned for 3 months' time.
