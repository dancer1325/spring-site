# Spring Cloud Stream Applications 2022.0.1 GA Release now available

_Releases | Corneil du Plessis |  November 27, 2024 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring Cloud Stream Applications `2022.0.1` has been released.

This release is updates to Spring Boot `3.1.12` and Spring Cloud `2022.0.5` and contains some bug fixes - check out the [release notes](https://github.com/spring-cloud/stream-applications/releases/tag/v2022.0.1) for more information.

## [](#stay-in-touch)Stay in touch...

Please continue to engage with us by creating bug reports and feature requests on [Github](https://github.com/spring-cloud/stream-applications) and asking questions on StackOverflow using the [spring-cloud-stream](https://stackoverflow.com/questions/tagged/spring-cloud-stream) or [spring-cloud-dataflow](https://stackoverflow.com/questions/tagged/spring-cloud-dataflow) tags.
