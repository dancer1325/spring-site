# Spring Framework 5.3.0-M2 available now

_Releases | Stéphane Nicoll |  August 11, 2020 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.0-M2` is available now.

The second milestone of Spring Framework 5.3 includes [55 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.0-M2).

If you want to give the milestone a try, you can bootstrap a new application on [start.spring.io](https://start.spring.io) once Spring Boot `2.4.0-M2` is released later this week.
