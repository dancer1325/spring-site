# Spring Security 7.0.0-M2 available now

_Releases | Josh Cummings |  August 18, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce a new milestone for the next Spring Security generation. We encourage you to review the [Spring Security 6 to 7 migration guide](https://docs.spring.io/spring-security/reference/6.5/migration-7/index.html) to gauge your preparedness for the release. Please also check out the main feature set in our [What's New in Spring Security 7 page](https://docs.spring.io/spring-security/reference/7.0-SNAPSHOT/whats-new.html). Finally, for a detailed view of all changes, please see [the Spring Security 7.0.0-M2 release notes](https://github.com/spring-projects/spring-security/releases/tag/7.0.0-M2).

7.0.0-M2 is now available from [Maven Central](https://spring.io/blog/2025/01/21/milestones-to-central).
