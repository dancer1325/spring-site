# Spring Data MongoDB 1.0.0.M5 Released

_Releases | Thomas Risberg |  October 24, 2011 | 0 Comments_

Dear Spring Community,

I am pleased to announce that the Spring Data MongoDB 1.0 Milestone 5 release is now available!

The primary goal of the **Spring Data** project is to make it easier to build Spring-powered applications that use new data access technologies such as non-relational databases, map-reduce frameworks, and cloud based data services.

The MongoDB module provides integration with the [MongoDB](http://www.mongodb.org) document database.

[Downloads](http://www.springsource.com/download/community?project=Spring%20Data%20Mongo&version=1.0.0.M5) | [JavaDocs](http://static.springsource.org/spring-data/data-mongo/docs/1.0.0.M5/api/) | [Reference Documentation](http://static.springsource.org/spring-data/data-mongo/docs/1.0.0.M5/reference/html/) | [Changelog](http://static.springsource.org/spring-data/data-mongo/docs/1.0.0.M5/changelog.txt)

To learn more about the project, visit the Spring Data [MongoDB Page](http://www.springsource.org/spring-data/mongodb).

There are a number of bug fixes and minor changes to the way certain features work. See the [changelog](http://static.springsource.org/spring-data/data-mongo/docs/1.0.0.M5/changelog.txt) for more details.

Looking forward to your feedback on the [forum](http://forum.springsource.org/forumdisplay.php?f=80) or in the [issue tracker](https://jira.springsource.org/browse/DATAMONGO).
