# Spring Framework 5.3.12 available now

_Releases | Stéphane Nicoll |  October 21, 2021 | 0 Comments_

**Update (2021-10-26):** This release contains a fix for a security issue. Please, read the related [announcement](https://spring.io/blog/2021/10/26/security-reports-for-spring-framework-spring-data-rest-spring-amqp-and-spring-cloud-openfeign).

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.3.12` is available now.

Spring Framework `5.3.12` is a follow-up of the recent `5.3.11` release fixing [2 regressions](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.12) in time for the Spring Boot releases.
