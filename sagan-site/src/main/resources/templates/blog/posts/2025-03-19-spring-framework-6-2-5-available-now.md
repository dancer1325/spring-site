# Spring Framework 6.2.5 Available Now

_Releases | Brian Clozel |  March 19, 2025 | 0 Comments_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `6.2.5` is available now.

Spring Framework `6.2.5` ships with [14 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.2.5). This is an out of cycle release [fixing a JAR scanning regression introduced last week](https://github.com/spring-projects/spring-framework/issues/34607). This version is right on time for the Spring Boot `3.4.4` and `3.5.0-M3` releases tomorrow.
