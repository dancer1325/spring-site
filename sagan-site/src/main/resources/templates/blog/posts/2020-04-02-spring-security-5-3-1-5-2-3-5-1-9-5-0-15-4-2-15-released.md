# Spring Security 5.3.1, 5.2.3, 5.1.9, 5.0.15, 4.2.15 Released

_Releases | Rob Winch |  April 02, 2020 | 1 Comment_

On behalf of the community, I’m pleased to announce the release of Spring Security 5.3.1 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.3.1.RELEASE)), 5.2.3 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.2.3.RELEASE)), 5.1.9 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.1.9.RELEASE)) , 5.0.15 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/5.0.15.RELEASE)), 4.2.15 ([release notes](https://github.com/spring-projects/spring-security/releases/tag/4.2.15.RELEASE)). These releases deliver bug fixes along with some minor improvements. Users are encouraged to update to the latest patch release.
