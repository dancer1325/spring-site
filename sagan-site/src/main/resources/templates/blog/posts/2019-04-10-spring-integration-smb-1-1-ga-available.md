# Spring Integration SMB 1.1 GA Available

_Releases | Artem Bilan |  April 10, 2019 | 0 Comments_

On behalf of Spring Integration team I am pleased to announce that the `1.1.0.RELEASE` for the Spring Integration Extension for SMB is available.

It can be downloaded from Maven Central, JCenter, and our [release repository](https://repo.spring.io/release):

```
Copycompile "org.springframework.integration:spring-integration-smb:1.1.0.RELEASE"
```

I would like to thank [Gregory Bragg](https://github.com/GregBragg) for awesome contribution to this project in regards to update it to the latest client library version to support SMB v2 & 3!

You can learn more about this extension on its GitHub [page](https://github.com/spring-projects/spring-integration-extensions/tree/master/spring-integration-smb).

Any feedback, feature ideas, critics, bug reports and questions are welcome via appropriate communication channels:
