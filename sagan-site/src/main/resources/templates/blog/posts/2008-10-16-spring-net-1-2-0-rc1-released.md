# Spring.NET 1.2.0 RC1 Released

_Releases | Mark Pollack |  October 16, 2008 | 0 Comments_

We are pleased to announce that Spring .NET 1.2.0 RC1 has been released.  
[Download](http://www.springframework.net/download.html) | [Support](http://www.springsource.com/) | [Documentation](http://www.springframework.net/documentation.html#Reference_Manual_and_API) | [Changelog](http://www.springframework.net/docs/1.2.0-RC1/changelog.txt)  
This release contains the following features:  

-   [WCF Integration](http://www.springframework.net/docs/1.2.0-RC1/reference/html/wcf.html) - Configure WCF services using dependency injection. Apply AOP advice to WCF services.
-   [MSMQ integration](http://www.springframework.net/docs/1.2.0-RC1/reference/html/msmq.html) - MSMQ helper classes to increase your productivity developing messaging applications. Provides integration with Spring's transaction management features.
-   [Apache ActiveMQ integration](http://www.springframework.net/docs/1.2.0-RC1/reference/html/messaging.html) - Helper classes to increase your productivity developing messaging applications with ActiveMQ
-   [Quartz integration](http://www.springframework.net/docs/1.2.0-RC1/reference/html/scheduling.html) - Configure Quartz jobs, schedulers, triggers using dependency injection. Convenience classes for implementing Quartz Jobs and integration with Spring's transaction management features.
-   [AOP](http://www.springframework.net/docs/1.2.0-RC1/reference/html/aop.html#aop-inheritancebasedaopconfigurer) New inheritance based AOP proxy generation.
-   Performance Improvements in WebForm dependency injection.
-   NHibernate 2.0.1 support.

Other notable new features in 1.2.0 RC1  

-   Color code and XML listings in reference documentation.
-   Introduction of [sterotype attributes](http://www.springframework.net/docs/1.2.0-RC1/reference/html/objects.html#d4e2365) - for example \[Repository\] 
-   [WCF Service exporter](http://www.springframework.net/docs/1.2.0-RC1/reference/html/wcf.html#d4e7876).
-   [Extensible web navigation](http://www.springframework.net/docs/1.2.0-RC1/reference/html/web.html#web-resultmapping-custom).
-   [New QuickStart applications](http://www.springframework.net/docs/1.2.0-M1/reference/html/index-quickstarts.html) are available for WCF, Quartz, MQMQ, and Apache Active MQ integration.

Support for TIBCO EMS, Websphere MQ, and Progress SonicMQ will be made available commerically.  Contact [sales](mailto:insidesales@springsource.com) for more information.  
  
Please refer to the changelog for additional details.  

Enjoy!
