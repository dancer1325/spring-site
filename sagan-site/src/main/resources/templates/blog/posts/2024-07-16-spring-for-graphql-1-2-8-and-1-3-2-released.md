# Spring for GraphQL 1.2.8 and 1.3.2 released

_Releases | Brian Clozel |  July 16, 2024 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.2.8 and 1.3.2 are now available on Maven Central.

1.2.8 closes [2 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.2.8). This version will ship with Spring Boot 3.2.8, [to be released this week](https://calendar.spring.io/).

1.3.2 closes [10 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.3.2). This version will ship with Spring Boot 3.3.2, [to be released this week](https://calendar.spring.io/).
