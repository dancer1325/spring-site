# Spring Boot 1.2.2 released

_Releases | Phil Webb |  February 27, 2015 | 3 Comments_

Spring Boot 1.2.2 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [significant number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.2) and is a recommended upgrade for all users.

We've also managed to sneak support for JMustache into this release, see [this application](https://github.com/spring-projects/spring-boot/tree/v1.2.2.RELEASE/spring-boot-samples/spring-boot-sample-web-mustache) for an example of how to use it.
