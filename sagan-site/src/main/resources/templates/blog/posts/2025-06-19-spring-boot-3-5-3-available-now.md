# Spring Boot 3.5.3 available now

_Releases | Phil Webb |  June 19, 2025 | 0 Comments_

Spring Boot `3.5.3` has been released and is now available from Maven Central. This unscheduled release fixes a single nasty regression that was accidentally introduced in the Spring Boot 3.5.1 and not completely fixed in Spring Boot 3.5.2.

Please see [see the 3.5.1 blog annoucement](https://spring.io/blog/2025/06/19/spring-boot-3-5-1-available-now) and [3.5.1 release notes](https://github.com/spring-projects/spring-boot/releases/tag/v3.5.1) for details of the other issues fixed in this release.
