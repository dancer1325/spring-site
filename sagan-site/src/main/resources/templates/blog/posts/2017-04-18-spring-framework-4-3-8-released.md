# Spring Framework 4.3.8 released

_Releases | Stéphane Nicoll |  April 18, 2017 | 1 Comment_

It is my pleasure to announce that Spring Framework [4.3.8](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15892) [is available now](http://projects.spring.io/spring-framework).

This maintenance release is the base for the upcoming Spring Boot 1.4.6 and 1.5.3 maintenance releases and includes 40 fixes and selected improvements.
