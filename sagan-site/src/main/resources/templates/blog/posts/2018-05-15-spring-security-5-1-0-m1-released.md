# Spring Security 5.1.0.M1 Released

_Releases | Rob Winch |  May 15, 2018 | 4 Comments_

On behalf of the community I’m pleased to announce the release of Spring Securiity 5.1.0.M1. This release resolves over [80 tickets](https://github.com/spring-projects/spring-security/milestone/104?closed=1). The highlights can be seen below:

-   Spring Security OAuth2 Client Support for WebFlux. See [the sample](https://github.com/spring-projects/spring-security/tree/5.1.0.M1/samples/boot/oauth2login-webflux) for how to use it.
    
-   Numerous other enhancements to WebFlux Support
    
-   Added [OAuth2ClientArgumentResolver](https://docs.spring.io/spring-security/site/docs/5.1.0.M1/api/org/springframework/security/oauth2/client/web/method/annotation/OAuth2ClientArgumentResolver.html)
    
-   Implementation of the Authorization Code Grant. See [the sample](https://github.com/spring-projects/spring-security/tree/5.1.0.M1/samples/boot/oauth2/authcodegrant) for how to use it.
