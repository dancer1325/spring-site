# Spring Security 2.0.8, 3.0.8, & 3.1.3 Released

_Releases | Rob Winch |  October 09, 2012 | 0 Comments_

We're please to announce the relase of Spring Security 2.0.8, 3.0.8, and 3.1.3 which fixes [CVE-2012-5055](http://support.springsource.com/security/cve-2012-5055) and brings Spring Security's open bugs to 0. The releases are available from the [Spring Community Downloads area](http://www.springsource.com/download/community?project=Spring%20Security) and jars are available from Maven central repository. See the [project website](http://www.springsource.org/spring-security) for more information.

A special thanks to Nicholas Goodwin ([@NGoody](https://twitter.com/NGoody)) for disclosing the issue and keeping it private until a fix could be pushed out.
