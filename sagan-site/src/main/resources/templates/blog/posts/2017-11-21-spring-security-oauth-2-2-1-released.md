# Spring Security OAuth 2.2.1 Released

_Releases | Joe Grandja |  November 21, 2017 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security OAuth 2.2.1.RELEASE.

The [2.2.1.RELEASE](https://github.com/spring-projects/spring-security-oauth/milestone/28?closed=1) is a maintenance release that includes bug fixes and minor improvements.
