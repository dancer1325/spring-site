# Spring CredHub 2.0.0.M1 released

_Engineering | Scott Frederick |  November 05, 2018 | 0 Comments_

I’m pleased to announce the first milestone release of Spring CredHub 2.0.0, available now in the [Spring milestone repository](https://repo.spring.io/libs-milestone-local/org/springframework/credhub/).

## [](#credhub)[](#credhub)CredHub

[CredHub](https://docs.cloudfoundry.org/credhub/) provides centralized credential management to the Cloud Foundry platform. CredHub implements an HTTP API to securely store, generate, retrieve, and delete credentials of various types.

Spring CredHub provides a Java binding for the CredHub API, making it easy to integrate Spring applications with CredHub.

## [](#whats-new)[](#what-s-new)What’s New

The 2.0.0 release brings support for CredHub server versions 1.x and 2.x, and the CredHub v1 and v2 APIs. This release also provides compatibility with Spring Framework 5 and Spring Boot 2. The GA release of Spring CredHub 2.0.0 will add a reactive alternative to the existing interfaces.

## [](#getting-started)[](#getting-started)Getting Started

The Spring CredHub documentation contains a [Getting started](https://docs.spring.io/spring-credhub/docs/2.0.0.M1/reference/html5/#getting-started) section to help get you started using the library. Feedback can be provided via [GitHub issues](https://github.com/spring-projects/spring-credhub/issues).
