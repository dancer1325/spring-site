# Spring Vault 2.3.1 available

_Releases | Mark Paluch |  February 16, 2021 | 0 Comments_

On behalf of the community, I’d like to announce the availability of the Spring Vault service release `2.3.1`.

This release ships with mostly bugfixes, dependency upgrades, and selected features.

For a complete list of changes, see the [2.3.1 changelog](https://github.com/spring-projects/spring-vault/releases/tag/2.3.1).
