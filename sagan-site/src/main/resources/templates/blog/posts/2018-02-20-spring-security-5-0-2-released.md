# Spring Security 5.0.2 Released

_Releases | Rob Winch |  February 20, 2018 | 3 Comments_

I’m pleased to announce the release of Spring Security 5.0.2. This release fixes a number of bugs and updates to dependency versions to align with Spring Boot’s upcoming release. It also includes some changes to work with [Jackson 2.9.4](https://github.com/spring-projects/spring-security/issues/4985). For a complete list of changes, refer to the [changelog](https://github.com/spring-projects/spring-security/issues?page=1&q=milestone%3A5.0.2+is%3Aclosed).
