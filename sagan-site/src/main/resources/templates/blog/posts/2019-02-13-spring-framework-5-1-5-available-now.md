# Spring Framework 5.1.5 available now

_Releases | Stéphane Nicoll |  February 13, 2019 | 0 Comments_

On behalf of the team and everyone who has contributed, it is my pleasure to announce that Spring Framework 5.1.5 [is available now](http://projects.spring.io/spring-framework).

This maintenance release includes [over 40 fixes and improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.1.5.RELEASE). As usual, we'll follow-up later this week with a maintenance release for Spring Boot 2.1, stay tuned!
