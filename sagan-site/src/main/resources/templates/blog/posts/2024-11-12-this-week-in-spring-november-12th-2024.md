# This Week in Spring - November 12th, 2024

_Engineering | Josh Long |  November 12, 2024 | 0 Comments_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*!

-   [Spring Cloud 2024.0.0-RC1 (aka Moorgate) has been released](https://spring.io/blog/2024/11/08/spring-cloud-2024-0-0-rc1-aka-moorgate-has-been-released)
-   In this installment of *A Bootiful Podcast*, [I talk to Gradle developer advocate Baruch Sadogursky](https://spring.io/blog/2024/11/07/a-bootiful-podcast-baruch-sadogursky)
-   good news everybody! [GraalVM will now support `jcmd`](https://x.com/fniephaus/status/1856290749769281810), which allows you to bring diagnostics and monitoring of native images even closer to the Hotspot experience. JFR, heap dumps, thread dumps, JMX, and NMT are already available!
-   Modern Java is [*amazing*](https://gist.githubusercontent.com/joshlong/fcca98b7bbe0a09bd9927d138e970feb/raw/22df0a404fd836f1e21d7e1e5c65a3a0e0f46a58/DemoApplication.java)
-   I'll be speaking at VOXXED Days Amsterdam next year. [Will you be there?](https://x.com/VoxxedAmsterdam/status/1856297547523874961)
-   this French language post on how to [integrate Langchain4J with Spring is pretty amazing](https://javaetmoi.com/2024/11/integrer-un-chatbot-dans-une-webapp-java-avec-langchain4j/)
-   the latest Pkl release [supports Spring Boot 3!](https://x.com/_JamesWard/status/1854998612431737033)
-   three ways to [fake OAuth 2 single sign on in a Spring application](https://blogs.vmware.com/tanzu/faking-oauth2-single-sign-on-in-spring-3-ways/)
-   Data legend Gwen Shapira has a nice video on how to [get started with `pgvector`, the module for vector stores in PostgreSQL](https://www.youtube.com/watch?v=psxR23HOGbI)
-   [Spring AI has natiuve support for Kotlin! Nice!](https://github.com/spring-projects/spring-ai/pull/1666)
-   [JEP 483, landing in Java 24, introduces AOT class loading and linking](https://openjdk.org/jeps/483)
-   also coming in Java 24: [synchronize virtual threads without pinning](https://openjdk.org/jeps/491)
