# Spring Shell 4.0.0-M2 is out!

_Releases | Mahmoud Ben Hassine |  December 09, 2025 | 0 Comments_

On behalf of the team and all contributors, I am pleased to announce that Spring Shell `4.0.0-M2` is now available on Maven Central!

## [](#whats-new)What's new?

This second milestone release continues our efforts toward a modern version of Spring Shell, aligned with the rest of the Spring portfolio.

This milestone includes several bug fixes and improvements based on the feedback we received after the `4.0.0-M1` release.

Major changes include:

-   Upgrade to Spring Framework `7.0 GA` and Spring Boot `4.0 GA`
-   Migrate nullability annotations and checks to [jSpecify](https://jspecify.dev)
-   Improve project modularity to streamline dependencies
-   Improve command programming model and APIs

For the complete list of changes, please check the [release notes](https://github.com/spring-projects/spring-shell/releases/tag/v4.0.0-M2).

## [](#whats-next)What's next?

We plan to release a third milestone and a release candidate, and eventually Spring Shell `4.0` GA by the end of this year.

## [](#feedback)Feedback

Thanks to all contributors who helped make this release possible! As we continue our work on Spring Shell `4.0`, we look forward to your feedback on [GitHub Issues](https://github.com/spring-projects/spring-shell/issues) and [GitHub Discussions](https://github.com/spring-projects/spring-shell/discussions).
