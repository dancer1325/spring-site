# This Week in Spring - July 7th, 2015

_Engineering | Josh Long |  July 07, 2015 | 1 Comment_

Welcome to another installment of *This Week in Spring*! As usual, we've got a lot to cover so let's get to it! This week I'll be in Shanghai, China, talking to some customers (say hi on Twitter!) and then it's off to Shenzhen, China, where [I'll be giving a keynote at the ArchSummit conference](http://www.archsummit.com/).

-   Spring Integraton lead Gary Russell [has just announced Spring Integration 4.2 and 4.1.6 are now available](https://spring.io/blog/2015/07/07/spring-integration-4-2-milestone-2-is-available-and-4-1-6). The new release includes support for Apache Zookeeper, Java 8 `CompletableFuture`s, (S)FTP `-stream`s, and several performance improvements.
-   Spring Boot co-lead Phil Webb [has just announced Spring Boot 1.2.5](https://spring.io/blog/2015/07/02/spring-boot-1-2-5-released).
-   Spring framework 4.2 is *fast* approaching (in about a week!) and there are many new versions of many different (and sometimes new) projects supported. To find out more, check out [this detailed blog by Spring framework lead Juergen Hoeller](https://spring.io/blog/2015/07/01/hibernate-jackson-jetty-etc-support-in-spring-4-2)
-   Spring Data ninja Christoph Strobl has [just announced the Spring Data Fowler SR1 release](https://spring.io/blog/2015/07/01/spring-data-fowler-sr-1-released). This is a maintenance release packed with lots of new features so be sure to get the bits.
-   Spring Data lead Oliver Gierke has just announced [the final Spring Data Service Releases (Evans SR3 and Dijkstra SR6)](https://spring.io/blog/2015/07/01/final-spring-data-service-releases-evans-sr3-and-dijkstra-sr6-released).
-   Daniel Jagielski has put together a nice look at the pros/cons of using Spring Boot and DropWizard. It should be noted that we on [the Spring Boot team were definitely inspired by DropWizard](http://www.schibsted.pl/2015/07/spring-boot-and-dropwizard-in-microservices-development/)
-   I found an oldie-but-goodie: this is Agile luminary [Brian Marick's review of agile development practices from this 2008 blog](http://www.exampler.com/blog/2008/11/14/agile-development-practices-keynote-text/)
-   Want to catch up with all things Redis? [I quite like the Redislabs newsletter, whose latest issue I'm linking to](https://redislabs.com/redis-watch-archive/49)
-   Our pal Marten Deinum has put together a [nice look at some great Spring framework *gems*](https://mdeinum.wordpress.com/2015/07/01/spring-framework-hidden-gems/)
-   I love *Cloud Foundry After Dark* the livestream'd Google Hangout featuring various Cloud Foundry luminaries every Sunday. In this episode, [one of our heros - Andrew Clay Shafer](http://twitter.com/littleidea) - [shaves his head](https://www.youtube.com/watch?v=MBl18nUde0U)
-   I really dug [Michael Cote's interview on continuous delivery using Pivotal Cloud Foundry](http://blog.pivotal.io/pivotal-conversations/features/the-agility-frontier-continuous-delivery-and-pivotal-cloud-foundry) - check it out!
