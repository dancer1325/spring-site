# Spring Cloud Square 0.4.1 has been released

_Releases | Olga Maciaszek-Sharma |  March 04, 2022 | 0 Comments_

On behalf of the community, I am pleased to announce that the 0.4.1 release of [Spring Cloud Square](https://github.com/spring-projects-experimental/spring-cloud-square) is available today. The release can be found in [Maven Central](https://repo.maven.apache.org/maven2/org/springframework/cloud/spring-cloud-square-dependencies/0.4.1/). You can check out the 0.4.1 [release notes for more information](https://github.com/spring-projects-experimental/spring-cloud-square/releases/tag/v0.4.1).

This is primarily a bug fix release.

## [](#notable-changes-in-041)Notable changes in 0.4.1:

-   Support for configuring custom `WebClient.Builder` beans for non-load-balanced scenarios ([#44](https://github.com/spring-projects-experimental/spring-cloud-square/issues/44))

The documentation for this release can be found [here](https://spring-projects-experimental.github.io/spring-cloud-square/docs/current/reference/html/index.html).
