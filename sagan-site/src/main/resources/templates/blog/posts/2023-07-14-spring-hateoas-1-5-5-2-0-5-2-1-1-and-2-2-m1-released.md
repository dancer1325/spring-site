# Spring HATEOAS 1.5.5, 2.0.5, 2.1.1 and 2.2 M1 released

_Releases | Oliver Drotbohm |  July 14, 2023 | 0 Comments_

I'm happy to announce the releases of Spring HATEOAS [1.5.5](https://github.com/spring-projects/spring-hateoas/releases/tag/1.5.5), [2.0.5](https://github.com/spring-projects/spring-hateoas/releases/tag/2.0.5), [2.1.1](https://github.com/spring-projects/spring-hateoas/releases/tag/2.1.1) and [2.2 M1](https://github.com/spring-projects/spring-hateoas/releases/tag/2.2.0-M1). The releases primarily contain a few bug fixes, dependency upgrades and a fix for [CVE-2023-34036](https://spring.io/security/cve-2023-34036) potentially and only affecting applications using Spring HATEOAS with Spring WebFlux. Make sure to consult [the advisory](https://spring.io/security/cve-2023-34036) and update to the latest versions.

The releases will be also included in the upcoming releases of Spring Boot 2.7.14, 3.0.9, 3.1.2 and 3.2 M1.
