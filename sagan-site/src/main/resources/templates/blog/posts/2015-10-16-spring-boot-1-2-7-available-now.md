# Spring Boot 1.2.7 available now

_Releases | Stéphane Nicoll |  October 16, 2015 | 0 Comments_

Spring Boot 1.2.7 has been released and is available now from [repo.spring.io](http://repo.spring.io/release) and [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22org.springframework.boot%22).

This maintenance release includes a [number of fixes](https://github.com/spring-projects/spring-boot/issues?q=milestone%3A1.2.7) and most importantly updates the managed Spring Framework dependency to [version 4.1.8](https://spring.io/blog/2015/10/15/spring-framework-4-2-2-4-1-8-and-3-2-15-available-now) (which includes an important [security fix](http://pivotal.io/security/cve-2015-5211)). Spring Boot itself also includes [a security fix](https://github.com/spring-projects/spring-boot/issues/4015) that impacts applications using Undertow.

We recommend an immediate upgrade for all Spring Boot users.
