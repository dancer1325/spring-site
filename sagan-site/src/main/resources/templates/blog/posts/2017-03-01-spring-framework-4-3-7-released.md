# Spring Framework 4.3.7 released

_Releases | Stéphane Nicoll |  March 01, 2017 | 0 Comments_

It is my pleasure to announce that Spring Framework [4.3.7 has been released](https://jira.spring.io/jira/secure/ReleaseNote.jspa?projectId=10000&version=15827).

This maintenance release is the base for the upcoming Spring Boot 1.4.5 and 1.5.2 releases and includes over 20 fixes and selected improvements.
