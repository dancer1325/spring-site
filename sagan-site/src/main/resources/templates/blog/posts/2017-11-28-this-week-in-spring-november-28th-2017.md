# This Week in Spring - November 28th, 2017

_Engineering | Josh Long |  November 28, 2017 | 0 Comments_

Hi Spring fans and welcome to another installation of *This Week in Spring*! This week I'm in San Francisco and then off to Austin, TX [for the epic G3 Summit](https://g3summit.com/conference/austin/2017/11/home). If you're around don't hesitate to say hi. Then, it's straight back to San Francisco for next week's premier agile, ops, apps and cloud-centric event worldwide, [SpringOne Platform 2017](http://springoneplatform.io). Do *not* miss this!

-   In last week's [installment of Spring Tips](https://spring.io/blog/2017/11/22/spring-tips-bootiful-testing), I look at testing in Spring-based applications. Check it out!
-   Spring Framework [lead Juergen Hoeller delves deeper into our decision to end first-class JDK 6 support](https://spring.io/blog/2017/11/27/end-of-first-class-jdk-6-support) for Spring in this blog post.
-   Spring Framework lead and Spring co-founder Juergen Hoeller just [released Spring Framework 5.0.2 and 4.3.13](https://spring.io/blog/2017/11/27/spring-framework-5-0-2-and-4-3-13-available-now). The new releases are recommended refinements. Of particular note, our Castor XML support has been deprecated, and we're effectively deprecating support for JDK 6.
-   Spring Data lead Mark Paluch just announced [Spring Data Ingalls SR9 and Spring Data Kay SR2](https://spring.io/blog/2017/11/27/spring-data-ingalls-sr9-and-kay-sr2-released). The new release ships over 90 issues fixed and is a recommended upgrade for all users.
-   \[Spring Cloud co-founder Spencer Gibb [just announced Spring Cloud Edgware.RELEASE Available](_URL_) . The new release features updates to Spring Cloud Bus, Spring Cloud Sleuth and Zipkin, Spring Cloud Contract, Spring Cloud Security, and Spring Cloud Stre\]([https://spring.io/blog/2017/11/27/spring-cloud-edgware-release-available](https://spring.io/blog/2017/11/27/spring-cloud-edgware-release-available))
-   Spring Integration ninja Artem Bilan just [announced Spring Integration for AWS 1.1 GA](https://spring.io/blog/2017/11/27/spring-integration-for-aws-1-1-ga-available) is now available. The release includes great support for S3, Kinesis, and DynamoDB.
-   Dr. Mark Pollack just announced [Spring Cloud Skipper 1.0 M2](https://spring.io/blog/2017/11/21/spring-cloud-skipper-1-0-m2-released). From the post: "Skipper is a lightweight tool that allows you to discover Spring Boot applications and manage their lifecycle on multiple Cloud Platforms. You can use Skipper standalone or integrate it with Continuous Integration pipelines to help implement the practice of Continuous Deployment."
-   Spring Security OAuth ninja Joe Grandja has just released [Spring Security OAuth 2.2.1](https://spring.io/blog/2017/11/21/spring-security-oauth-2-2-1-released). This release is primarily a bug fix release
-   Alright, this post doesn't have all that much to do with Spring, *per se*, but it is newsworthy nonetheless. It's a really interesting read on [what motivates and animates the agile philosophy: to learn and improve](https://johnfergusonsmart.com/exponential-learning-the-key-to-effective-leanagile-practices/).
-   I really liked this post by Arne Vandamme on [tips and tricks for the Spring `CacheManager` abstraction](https://foreach.be/blog/spring-cache-annotations-some-tips-tricks) and - in particular - the annnotations that support it.
-   I liked this German-language post that [looks at the Accenture and Pivotal partnership](https://www.heise.de/amp/meldung/Accenture-und-Pivotal-gruenden-gemeinsame-IT-Beratungsgesellschaft-3899624.html).
-   This is a great look at just some of the great content coming [in the SpringOne Platform 2017 conference event](https://seroter.wordpress.com/2017/11/26/cant-figure-out-which-springone-platform-sessions-to-attend-ill-help-you-out/amp/) here in beautiful San Francisco in just a few days.
-   Community friend Eberhard Wolff looks at [building microservices with Spring Boot and Java](https://www.youtube.com/watch?v=sHNYcEOsxp4&app=desktop) in this interesting video. Check it out!
-   Michael Simons looks at integration testing [with Spring Boot, Gradle and Docker Compose](http://info.michael-simons.eu/2017/11/20/integration-testing-with-docker-compose-gradle-and-spring-boot/) in this article. Well worth a look, as always.
-   I really like this article that looks at the integration of JUnit 5 and AssertJ. Hat tip to my [friend Billy Korando for this link](http://blog.codeleak.pl/2017/11/junit-5-meets-assertj.html?spref=tw&m=1).
-   It's with great joy that we announce that Reactive Couchbase support has now been added [to the Spring Initializr](https://twitter.com/snicoll/status/930866250120646663).
-   Christophe Bornet [tweets](https://twitter.com/cbornet_/status/933738622674759680) that "Reactive-gRPC the @grpcio binding for reactive implementations with backpressure v0.7 is out with support for @RxJava 2 and @ProjectReactor. Check it out!" So, do.
-   It's a big honor to [see Pivotal in this list of the 51 best enterprise startups to bet your career on in 2018](https://amp.businessinsider.com/51-enterprise-startups-to-bet-your-career-on-in-2018-2017-11).
