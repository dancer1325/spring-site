# Spring Data Ingalls SR15 and Kay SR10 released

_Releases | Mark Paluch |  September 10, 2018 | 0 Comments_

On behalf of the Spring Data team, I'd like to announce the availability of the Ingalls SR15 and Kay SR10 maintenance releases. Kay SR10 ships on top of the just-released Spring Framework 5.0.9. Spring Boot 2.0.5 is going to pick up Kay SR10 for your convenience. Ingalls SR15 will be picked up by Spring Boot 1.5.16. Both releases ship with mostly bugfixes and a few dependency upgrades.

Ingalls SR15 ships with [40 tickets fixed](https://jira.spring.io/issues/?filter=16150) and Kay SR10 contains [75 issues fixed](https://jira.spring.io/issues/?filter=16149).

Also, make sure you register for this year's [SpringOne Platform](https://springoneplatform.io/) (Sept. 24th-27th, Washington, D.C.) to learn about the latest additions in Spring Data and even get a preview of the features planned for the next release train. The entire team will be on site and happy to answer questions and take feedback.

To round things off, here are the links to the changelogs, documentation, and artifacts:
