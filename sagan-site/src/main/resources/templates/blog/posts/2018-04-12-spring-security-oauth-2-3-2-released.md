# Spring Security OAuth 2.3.2 Released

_Releases | Joe Grandja |  April 12, 2018 | 0 Comments_

On behalf of the community, I’m pleased to announce the release of Spring Security OAuth 2.3.2.RELEASE.

This release resolves a runtime incompatibility issue with `RedisTokenStore` and [Spring Data Redis 2.0.x](https://projects.spring.io/spring-data-redis/). See [#1319](https://github.com/spring-projects/spring-security-oauth/pull/1319) and [#1335](https://github.com/spring-projects/spring-security-oauth/issues/1335). This release also includes a few minor enhancements and bug fixes.
