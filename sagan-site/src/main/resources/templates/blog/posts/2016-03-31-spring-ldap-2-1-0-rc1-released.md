# Spring LDAP 2.1.0.RC1 Released

_Releases | Rob Winch |  March 31, 2016 | 1 Comment_

I’m pleased to announce the release of Spring LDAP 2.1.0.RC1. The highlights of this release include:

-   [#380](https://github.com/spring-projects/spring-ldap/issues/380) - Support for Spring Data Hopper
-   [#384](https://github.com/spring-projects/spring-ldap/issues/384) - Early support for Spring IO Platform 2.1
-   [#351](https://github.com/spring-projects/spring-ldap/issues/351) - Support for commons-pool2
-   [#370](https://github.com/spring-projects/spring-ldap/issues/370) - Support property placeholders in XML Namespace
-   [#392](https://github.com/spring-projects/spring-ldap/pull/392) - Document Testing Support
-   Migrated from JIRA to [GitHub Issues](https://github.com/spring-projects/spring-ldap/issues)
-   Added [Gitter Chat](https://gitter.im/spring-projects/spring-ldap)

For complete details of 2.1 refer to the [changelog](https://github.com/spring-projects/spring-ldap/issues?q=milestone%3A2.1.0.RC1)
