# Spring HATEOAS 1.5.4, 2.0.3 and 2.1 M1 released

_Releases | Oliver Drotbohm |  March 23, 2023 | 0 Comments_

I am excited to announce the releases of versions 1.5.4, 2.0.3 and 2.1 M1 of Spring HATEOAS. The bug fix releases primarily ship dependency upgrades, including one to the latest Spring Framework 5 and 6 releases which include fixes for a CVE. The 2.1 milestone includes a couple of new features, such as:

-   Property metadata for forms now also considers JSR-03's `@Size` annotation – [#1920](https://github.com/spring-projects/spring-hateoas/issues/1920)
-   Introduction of `SlicedModel`, a simplified version of `PagedModel` to navigate slices but not calculate a total – [#1856](https://github.com/spring-projects/spring-hateoas/issues/1856)

For more information, please check out the full changelogs [on GitHub](https://github.com/spring-projects/spring-hateoas/releases).
