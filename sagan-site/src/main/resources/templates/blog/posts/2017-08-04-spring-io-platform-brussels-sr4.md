# Spring IO Platform Brussels-SR4

_Releases | Andy Wilkinson |  August 04, 2017 | 1 Comment_

I am pleased to announce that [Spring IO Platform](http://platform.spring.io/platform) Brussels-SR4 is now available from both [repo.spring.io](https://repo.spring.io/release) and Maven Central.

This maintenance release upgrades the versions of a number of the projects in the Platform:

-   Spring Batch 3.0.8
-   Spring Boot 1.5.6
-   Spring Data Ingalls SR6
-   Spring Framework 4.3.10
-   Spring Integration 4.3.11
-   Spring Retry 1.2.1

The versions of a number of third-party dependencies have also been updated.
