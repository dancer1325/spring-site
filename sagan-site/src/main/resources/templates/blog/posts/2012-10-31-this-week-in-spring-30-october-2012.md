# This Week in Spring - 30 October, 2012

_Engineering | Josh Long |  October 31, 2012 | 0 Comments_

Welcome to another installment of *This Week in Spring*!

1.  [Spring Data GemFire 1.2.1 and GemFire 7.0 have been released](http://www.springsource.org/node/3716)! The new Spring Data release includes complete Spring XML namespace support for configuration of all GemFire resources, eliminating the need for GemFire's native cache XML (cache XML is also supported), and Spring Data Repositories for GemFire. Also of note, the GemFire shell (gfsh) was built with [Spring Shell](http://www.springsource.org/spring-shell). For more on the GemFire 7.0 release, be sure to check out the post which in turn links to release notes and documentation.
