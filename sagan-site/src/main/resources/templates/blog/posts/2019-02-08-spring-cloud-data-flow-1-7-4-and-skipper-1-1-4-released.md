# Spring Cloud Data Flow 1.7.4 and Skipper 1.1.4 Released

_Releases | Mark Pollack |  February 08, 2019 | 0 Comments_

The [Spring Cloud Data Flow](https://cloud.spring.io/spring-cloud-dataflow/) team is pleased to announce the release of `1.7.4` of Data Flow for Local and Cloud Foundry. The Kubernetes release is at version `1.7.3`.

Follow the *Getting Started* guides for running on [Local](https://docs.spring.io/spring-cloud-dataflow/docs/1.7.4.RELEASE/reference/htmlsingle/#getting-started), [Cloud Foundry](http://docs.spring.io/spring-cloud-dataflow-server-cloudfoundry/docs/1.7.4.RELEASE/reference/htmlsingle/#getting-started), and [Kubernetes](http://docs.spring.io/spring-cloud-dataflow-server-kubernetes/docs/1.7.3.RELEASE/reference/htmlsingle/#kubernetes-getting-started).

Various bugs since the `1.7 GA` release have been fixed. You can see the issues for [1.7.1](https://github.com/spring-cloud/spring-cloud-dataflow/issues?q=is%3Aissue+milestone%3A1.7.1.RELEASE+is%3Aclosed), [1.7.2](https://github.com/spring-cloud/spring-cloud-dataflow/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.7.2.RELEASE+is%3Aclosed+), [1.7.3](https://github.com/spring-cloud/spring-cloud-dataflow/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.7.3.RELEASE+is%3Aclosed+), and [1.7.4](https://github.com/spring-cloud/spring-cloud-dataflow/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.7.4.RELEASE+is%3Aclosed+) for more details.

We have also released `1.1.4` release of [Spring Cloud Skipper](https://cloud.spring.io/spring-cloud-skipper/). The [getting started section](https://docs.spring.io/spring-cloud-skipper/docs/1.1.4.RELEASE/reference/htmlsingle/#getting-started) in the reference guide is the best place to start if you want to use Skipper separately from Data Flow.

Various bugs since the `1.1 GA` have been fixed. You can see the issues for [1.1.1](https://github.com/spring-cloud/spring-cloud-skipper/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.1.1.RELEASE+is%3Aclosed+), [1.1.2](https://github.com/spring-cloud/spring-cloud-skipper/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.1.2.RELEASE+is%3Aclosed+), [1.1.3](https://github.com/spring-cloud/spring-cloud-skipper/issues?utf8=%E2%9C%93&q=is%3Aissue+milestone%3A1.1.3.RELEASE+is%3Aclosed+), and [1.1.4](https://github.com/spring-cloud/spring-cloud-skipper/issues?q=is%3Aissue+milestone%3A1.1.4.RELEASE+is%3Aclosed) for more details.
