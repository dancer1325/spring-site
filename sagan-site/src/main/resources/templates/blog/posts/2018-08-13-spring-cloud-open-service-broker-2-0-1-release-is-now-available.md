# Spring Cloud Open Service Broker 2.0.1.RELEASE is now available

_Releases | Roy Clarkson |  August 13, 2018 | 0 Comments_

We’re pleased to announce that the 2.0.1 release of [Spring Cloud Open Service Broker](https://spring.io/projects/spring-cloud-open-service-broker) is now available. This release resolves a few [issues](https://github.com/spring-cloud/spring-cloud-open-service-broker/milestone/13?closed=1) that were raised since the 2.0.0 release. Thank you to the community for your interest and feedback!

Spring Cloud Open Service Broker is a framework for building Spring Boot applications that implement the Open Service Broker API. The [Open Service Broker API](https://www.openservicebrokerapi.org) project allows developers to deliver services to applications running within cloud native platforms such as Cloud Foundry, Kubernetes, and OpenShift.
