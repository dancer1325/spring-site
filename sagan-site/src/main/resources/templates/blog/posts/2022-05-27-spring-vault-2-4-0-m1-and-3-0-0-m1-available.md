# Spring Vault 2.4.0-M1 and 3.0.0-M1 available

_Releases | Mark Paluch |  May 27, 2022 | 0 Comments_

On behalf of the team and everyone who has contributed, I’m happy to announce that Spring Vault `2.4.0-M1` and `3.0.0-M1` milestones have been released and are now available from [repo.spring.io](https://repo.spring.io).

Notable new features include:

-   Support for PEM-encoded certificates and private keys including Elliptic Curve ("EC")
    
-   Support for Vault Repositories using versioned Key/Value secrets engines
    
-   Support for Vault-based RevisionRepository using versioned Key/Value secrets engines
    

Please see the [release notes for more details](https://github.com/spring-projects/spring-vault/wiki/Spring-Vault-2.4.0-M1-Release-Notes) and upgrade instructions.

Additionally, Spring Vault `3.0.0-M1` ships with:

-   Upgrade to Java 17 baseline, Spring Framework 6
    
-   Upgrade to AWS SDK v2
    

Please see the [release notes for more details](https://github.com/spring-projects/spring-vault/wiki/Spring-Vault-3.0.0-M1-Release-Notes) and upgrade instructions.

Thanks to all those who have contributed with issue reports and pull requests.
