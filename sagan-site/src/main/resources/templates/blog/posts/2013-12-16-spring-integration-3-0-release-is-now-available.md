# Spring Integration 3.0 Release is Now Available

_Releases | Gary Russell |  December 16, 2013 | 0 Comments_

We are pleased to announce that Spring Integration 3.0 is now available.

The features in the release were covered in the [recent announcement for the release candidate](https://spring.io/blog/2013/11/27/spring-integration-3-0-release-candidate-1-is-now-available); a few minor changes made since then can be seen in the [release notes](https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10121&version=14250).

[Reference](http://docs.spring.io/spring-integration/reference/html/) | [API](http://docs.spring.io/spring-integration/api/)

Please visit the [project page](http://projects.spring.io/spring-integration/) for more information and how to get this important release.
