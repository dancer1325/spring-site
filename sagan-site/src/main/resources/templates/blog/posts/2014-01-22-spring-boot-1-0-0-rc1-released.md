# Spring Boot 1.0.0.RC1 Released

_Releases | Phil Webb |  January 22, 2014 | 0 Comments_

We are pleased to announce the first release candidate of Spring Boot v1.0.0. This release contains a number of [bug fixes and enhancements](https://github.com/spring-projects/spring-boot/issues?milestone=2&state=closed) over the previous milestone.

Binaries have been published to the [Spring Maven Repository](https://github.com/spring-projects/spring-framework/wiki/Spring-repository-FAQ), please take a look at the [project page](http://projects.spring.io/spring-boot) for download instructions and Maven/Gradle co-ordinates.

Please [raise an issue](https://github.com/spring-projects/spring-boot/issues) if you find any problems, or want to suggest some future improvements.
