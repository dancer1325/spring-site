# Spring Data Redis 1.2.0 RC1 Released

_Releases | Christoph Strobl |  February 19, 2014 | 0 Comments_

We are pleased to announce the first release candidate of Spring Data Redis 1.2.0! We raised the minimum Spring version to 3.2.8. This release is tested against Redis 2.6 and current 2.8 as well as the latest version of Spring Framework 4.0.2. It introduces a set of bug fixes as well as some improvements like support Jackson 2 serialization, a transaction aware cache manager, support for the [TIME](http://redis.io/commands/time/) command and upgrades of the Redis drivers.

For a complete list of changes please see this [JIRA query](https://jira.springsource.org/issues/?filter=14427&jql=project%20%3D%20DATAREDIS%20AND%20resolution%20in%20\(Complete%2C%20Fixed\)%20AND%20fixVersion%20in%20\(%221.2.0.RC1%22\)%20ORDER%20BY%20updated%20DESC). The next step will be RC2 scheduled for March. We’re looking forward to your feedback on [Twitter](https://twitter.com/SpringData) or in the [issue tracker](https://jira.springsource.org/browse/DATAREDIS).
