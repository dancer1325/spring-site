# Spring Data Moore SR4 and Lovelace SR15 released

_Releases | Mark Paluch |  January 15, 2020 | 1 Comment_

On behalf of the community, I'd like to announce that that Spring Data `Moore SR4` and `Lovelace SR15` are available now.

`Moore SR4` ships with [59 tickets fixed](https://jira.spring.io/issues/?filter=16631) and `Lovelace SR15` ships with [34 tickets fixed](https://jira.spring.io/issues/?filter=16630). Both releases contain mostly bug fixes and dependency upgrades. `Moore SR4` is built on top of the recently released Spring Framework `5.2.3` and will be picked up by Spring Boot `2.2.3` for easier consumption. Similarly, `Lovelace SR15` uses Spring Framework `5.1.13` and will be included by Spring Boot `2.1.12` in the coming days.
