# This Week in Spring - January 31st, 2017

_Engineering | Josh Long |  January 31, 2017 | 0 Comments_

Welcome to another installment of *This Week in Spring*! It's been a very busy week and we've got a lot to cover so let's get to it!

-   first, the *big* news: [Spring Boot 1.5 has just been released!](http://spring.io/blog/2017/01/30/spring-boot-1-5-1-released) The new release improves the integration with Cloud Foundry, features Spring Data Ingalls support, LDAP support, a loggers endpoint, and Apache Kafka auto-configuration support. [Get started with Spring Boot now](http://start.spring.io)!
-   Spring Data guru March Paluch put together a nice look at [Spring Data Ingalls](http://spring.io/blog/2017/01/30/what-s-new-in-spring-data-release-ingalls). The new release is packed with lots of great features including a 20-70% performance improvement, event publication for aggregate roots, improved pagination and *so* much more! Read the post!
-   Spring Data lead Oliver Gierke just announced Spring Data Ingalls has gone GA! Hurray! [See the release post for more](http://spring.io/blog/2017/01/26/spring-data-release-train-ingalls-goes-ga)!
-   This is a cool SpringOne Platform talk on migrating Spring XD-based [applications to Spring Cloud Data Flow](http://spring.io/blog/2017/01/30/springone-platform-2016-replay-migrating-from-spring-xd-to-spring-data-cloud-flow)
-   Rajini Sivaram's SpringOne Platform talk on [Apache Kafka is a good primer](http://spring.io/blog/2017/01/30/springone-platform-2016-replay-reactive-kafka)
-   Nivesh Gopathi gave a nice talk on secure, [dynamic configuration with Spring Cloud Vault](http://spring.io/blog/2017/01/30/springone-platform-2016-replay-secure-dynamic-app-config-at-gaptech-with-spring-cloud-vault-and-consul)
-   Spring Boot guru Stephane Nicoll just [announced Spring Boot 1.4.4](http://spring.io/blog/2017/01/26/spring-boot-1-4-4-available-now) which includes a slew of updates and fixes.
-   Spring Data lead Oliver Gierke [just announced Spring Data Hopper SR7](http://spring.io/blog/2017/01/26/spring-data-release-train-hopper-sr7-released)
-   last week I put together a [Spring Tip on Spring Data Couchbase](http://spring.io/blog/2017/01/26/spring-tips-couchbase).
-   going to be at Devoxx US? [We on the Spring team will be too](http://spring.io/blog/2017/01/25/spring-team-at-devoxx-us-2017)! Join us!
-   Spring Integration lead [Gary Russell just announced Spring Integration 4.3.7](http://spring.io/blog/2017/01/25/spring-integration-4-3-7-is-available)
-   Spring Boot lead Stephane Nicoll just announced [Spring Framework 4.3.6, which is a packed maintenance release](http://spring.io/blog/2017/01/25/spring-framework-4-3-6-available-now)
-   Spring I/O Platform lead [Andy Wilkinson just announced 1.0 of the Dependency Management Plugin](http://spring.io/blog/2017/01/24/dependency-management-plugin-1-0-0-release)
-   Google Developer Advocate Ray Tsang just put together a nice post on [Spring Boot-based tracing with Google Stackdriver](https://medium.com/google-cloud/distributed-tracing-spring-boot-microservices-with-stackdriver-trace-7fe42c6de3f3#.hlaikdzdo)
-   speaking of distributed tracing, the Baeldung has a great [post on distributed tracing with Spring Cloud Sleuth](http://www.baeldung.com/spring-cloud-sleuth-single-application)
-   Pivotal's own Fred Melo has a great post [on messaging-patterns for event-driven microservices](https://content.pivotal.io/blog/messaging-patterns-for-event-driven-microservices)
-   the Stark and Wayne blog has a nice post that [demystifies the Cloud Foundry Diego scheduler](http://www.starkandwayne.com/blog/demystifying-cloud-foundrys-diego/)
-   Oliver Wolf put together a nice post describing some of the [key features of a cloud platform like Cloud Foundry](https://specify.io/systems/cloudfoundry/features-and-usecases)
-   this isn't strictly Spring related, but I thought it interesting: [Java 9 is the end of the road for some features](https://dzone.com/articles/jdk-9-is-the-end-of-the-road-for-some-features)
