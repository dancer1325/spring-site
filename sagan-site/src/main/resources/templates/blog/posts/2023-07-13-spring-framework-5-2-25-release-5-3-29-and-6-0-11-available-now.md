# Spring Framework 5.2.25.RELEASE, 5.3.29 and 6.0.11 available now

_Releases | Brian Clozel |  July 13, 2023 | 1 Comment_

On behalf of the team and everyone who has contributed, I am pleased to announce that Spring Framework `5.2.25.RELEASE`, `5.3.29` and `6.0.11` are available now.

Spring Framework `5.2.25.RELEASE` ships with [2 fixes improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.2.25.RELEASE). This is an out-of-cycle release that will not be shipped with a new Spring Boot version, [as Spring Boot 2.3.x is EOL](https://spring.io/projects/spring-boot#support).

Spring Framework `5.3.29` ships with [22 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v5.3.29), including [3 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A5.3.29+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 2.7.14, [to be released next week](https://calendar.spring.io/).

Spring Framework `6.0.11` ships with [71 fixes and documentation improvements](https://github.com/spring-projects/spring-framework/releases/tag/v6.0.11), including [7 fixes for regressions](https://github.com/spring-projects/spring-framework/issues?q=is%3Aclosed+milestone%3A6.0.11+label%3A%22type%3A+regression%22). This version will be shipped with Spring Boot 3.0.9 and 3.1.2, [to be released next week](https://calendar.spring.io/).

The next Spring Framework releases are scheduled for August 17th with the last 6.1.0 milestone. On September 14th we are expecting a new 6.0.x maintenance release and the first release candidate for 6.1.0.
