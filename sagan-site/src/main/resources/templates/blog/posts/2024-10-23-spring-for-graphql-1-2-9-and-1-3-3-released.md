# Spring for GraphQL 1.2.9 and 1.3.3 released

_Releases | Brian Clozel |  October 23, 2024 | 0 Comments_

I am pleased to announce that Spring for GraphQL 1.2.9 and 1.3.3 are now available on Maven Central.

1.2.9 closes [5 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.2.9). This version will ship with Spring Boot 3.2.11, [to be released this week](https://calendar.spring.io/).

1.3.3 closes [20 issues](https://github.com/spring-projects/spring-graphql/releases/tag/v1.3.3). This version will ship with Spring Boot 3.3.5, [to be released this week](https://calendar.spring.io/).
