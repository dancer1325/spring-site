# This Week in Spring  - Feb 5th, 2013

_Engineering | Josh Long |  February 06, 2013 | 0 Comments_

Welcome back to another installment of *This Week in Spring* \--

There's some great video content is available online this week, so be sure to check the content out. Also, you asked and we are delivering - we now maintain an single index page for all [SpringOne2GX recordings](https://springone2gx.com/conference/washington/2012/10/video_list) along with the link to the InfoQ page for their recordings of the event. On with the roundup!

1.  Rossen Stoyanchev has announced the [2.4M1 and 2.3.2 releases of Spring Web Flow](http://www.springsource.org/node/3801).
2.  Dream team Sam Brannen (Swiftmind) & Rossen Stoyanchev (SpringSource) join forces on Feb 21st, 2013 for a [Webinar: Testing Web Applications with Spring 3.2](http://www.springsource.org/node/3800)
3.  Dave Syer is taking his SpringOne2GX talk to an online audience on Feb 14th, 2013 [When and Why Would I Use OAuth2?](http://www.springsource.org/node/3799)
4.  Jeremy Grelle's talk from SpringOne2GX 2012 introducing practical patterns [for asynchronous, push-enabled applications is now available online](http://www.springsource.org/node/3803).
5.  Craig Walls' presentation from SpringOne2GX 2012 [Introducing Spring Social](http://www.springsource.org/node/3802) is now available on YouTube in HD.
6.  Craig Walls' Javascript - focused talk from SpringOne2GX 2012 [Client Side UI Smackdown](http://www.springsource.org/node/3802), is now available on YouTube in HD.
7.  Over on InfoQ China (where the content is in Chinese...), blogger Ding Xuefeng has done a marvelous job shining [a light on some of the various Spring sub-projects, including Spring Data, Spring Batch, Spring Integration](http://www.infoq.com/cn/news/2013/01/spring-subprojects). Definitely worth a read!
8.  The slides from Spring framework committer [Sam Brannen's talk on Spring Framework 3.2 are available, and make for a fascinating read](http://www.swiftmind.com/de/2013/01/30/spring-framework-3-2-whats-new-slides-from-swiss-jug/).
9.  This blog explains how to lookup and use a [JavaMail ` Session` as configured in Tomcat's JNDI from a Spring application](http://www.scarba05.co.uk/blog/2013/01/sending-email-using-smtp-authentication-with-tomcat-jndi-resource/).
10.  The *softtech* blog has a code-heavy post introducing how to [create a one-to-many relationship using Spring Data JPA](http://softtechgmr.blogspot.com/2013/02/spring-data-jpa-onetomany-example.html).
