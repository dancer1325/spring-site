# This Week in Spring - July 14th 2015

_Engineering | Josh Long |  July 14, 2015 | 1 Comment_

Welcome to another installment of *This Week in Spring!*

This week in I'm in Shanghai, China and Hangzhou, China and Shenzhen, China, talking to some of the world's largest websites (of the same scale of Netflix!) about their use of Spring Boot and Spring Cloud. (More on that later!)

Anyway, without further ado..

-   Paul Chapman put together a nice walkthrough Spring Cloud today. [Way to go Paul](https://spring.io/blog/2015/07/14/microservices-with-spring)!
-   Andy Wilkinson just [announced the Spring IO Platform BOM 1.1.3 has been released](https://spring.io/blog/2015/07/13/spring-io-platform-1-1-3-released)
-   [Spring Boot 1.3M2](https://spring.io/blog/2015/07/10/spring-boot-1-3-0-m2-available-now) - along with new support for JOOQ, `/flyway` and `/liquibase` Actuator endpoints, ANSI colored `banner.txt`, and Apache Artemis integration (among *many* other things) is now available! Check it out.
-   If you missed Spring rockstar and Spring Boot ninja Greg Turnquist's talk on debugging and maintaing Spring Boot applications (why would you miss it??), [then check it out](https://spring.io/blog/2015/07/09/webinar-replay-debug-and-maintain-your-spring-boot-app)
-   This is a nice [post on using Spring Boot with the H2 database console](https://dzone.com/articles/using-the-h2-database-console-in-spring-boot-with)
-   I only just noticed DZone has a refcard [on Cloud Foundry which seems like a worthy read](https://dzone.com/refcardz/cloud-foundry)
-   this is a great (Chinese-language) post on what Spring Boot is and how it works. I've asked a [friend](http://twitter.com/resouer) and he - a native Mandarin Chinese speaker - insists it's a good read. Enjoy!
-   check out this [jumpstart into Spring Boot](http://wanderbishty.blogspot.in/2015/07/spring-boot-using-boot-for-your.html)
-   our pal [@making](http://twitter.com/making) is at it again, this time with a nice look at [the upcoming Spring Boot 1.3's DevTools integration](https://blog.ik.am/#/entries/349)
-   the *Codes and Notes* blog has a nice post on [using AngularJS web apps for Spring-based REST services](http://www.codesandnotes.be/2015/07/12/angularjs-web-apps-for-spring-based-rest-services-security-the-server-side-part-1/)
