# This Week in Spring - September 6th, 2016

_Engineering | Josh Long |  September 06, 2016 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in Shanghai, China and Hangzhou, China where I spoke at the *ginormous* Huawei Connect conference in Shanghai and where I'll be working with Alibaba and Huawei for a week or so. I'll also be speaking at the Hangzhou Java User Group, too.

(Can you believe we're already in September?? Seriously blown away! 2017 is just around the corner!)

I'm also, technically, on vacation, so I'll keep this post to a minimum! :)

-   Spring REST Docs lead Andy Wilkinson [has just announced Spring REST Docs 1.1.2](https://spring.io/blog/2016/08/31/spring-rest-docs-1-1-2-release)
-   Spring Security ninja Joe Grandja just announced a [very important fix for Spring Security OAuth 2](https://spring.io/blog/2016/08/31/spring-security-oauth2-client-authentication-issue)
-   Spring ninja Stéphane Nicoll put together a *very* interesting article on how [to write a custom spring Boot test *slice*](https://spring.io/blog/2016/08/30/custom-test-slice-with-spring-boot-1-4)
-   our buddy Eugene is back at it again, with updates to his guide on [using Spring Cloud's Netflix Hystrix support](http://www.baeldung.com/spring-cloud-netflix-hystrix)
-   learn more about [Spring Cloud's Config Server](https://www.iancollington.com/centralised-configuration-using-spring-boot/) in this nice post from Ian Collington.
-   I love [this Forbes.com interview with Paul Maritz](http://www.forbes.com/sites/andrewcave/2016/08/31/paul-maritzs-vision-for-pivotal-and-the-battle-with-amazon-over-open-source-cloud-computing) (Pivotal's founding CEO, current-chairman, former top Microsoft executive, and former VMWare CEO on his dreams
-   I'm honored to be included on [a TechBeacon roundup of Java folks in the Twitterverse you might like to follow](http://techbeacon.com/java-leaders-you-should-follow-twitter). The list is pretty good. About half of those people are *definitely* worth a follow, so check it out!
-   I did a Chinese-language [interview with InfoQ on microservices and cloud-native applications](http://www.infoq.com/cn/news/2016/09/bug-fix-speed-matters-microservi)
-   this isn't a Pivotal project, per se, but I thought it interesting; Netflix's just announced a new tool [called NDBench Runner that helps to benchmark data stores](http://techblog.netflix.com/2016/09/netflix-data-benchmark-benchmarking.html)
-   This is interesting: a thin JavaScript library called Hybind [to bind to Spring Data REST / HATEOAS hypermedia resources](http://www.swisspush.org/web/2016/08/30/spring-data-rest-hybind)
-   The Baeldung blog has a nice post [on setting up Swagger with a Spring REST API](http://www.baeldung.com/swagger-2-documentation-for-spring-rest-api)
-   There are multiple ways to acheive CORS in a Spring MVC application. If you're using an older variant of Spring MVC, before Spring Framework 4.2, then this post on [creating a `javax.servlet.Filter` will work ideally](http://www.veli.io/enable-cors-spring-boot). Otherwise, check out [this Spring guide on CORS](https://spring.io/guides/gs/rest-service-cors/) and [this Spring announcement blog](https://spring.io/blog/2015/06/08/cors-support-in-spring-framework)
-   wow! this is an exhaustive look [at Spring Boot's Actuator](http://www.dineshonjava.com/2016/08/spring-boot-actuator-complete-guide.html#.V84zypN96EI) on the *Dinesh on Java* blog.
