# This Week in Spring  - June 2, 2015

_Engineering | Josh Long |  June 02, 2015 | 0 Comments_

Welcome to another installment of *This Week in Spring*! This week I'm in New York City in advance of QCon NYC next week talking to customers and working with the local Pivotal Labs team. If you're around [be sure to say hi](mailto:jlong@pivotal.io) and let's talk code.

So, as usual, we've got a lot to cover so let's get to it.

1.  Our pal Nicolas Frankel's put together an awesome post on [modularizing configuration for integration testing with Spring](http://blog.frankel.ch/spring-configuration-modularization-for-integration-testing)

-   Spring Cloud co-lead Spencer Gibb just announced [that Spring Cloud Consul 1.0.0.M1 is now available](https://spring.io/blog/2015/05/27/spring-cloud-consul-1-0-0-m1-available-now)
-   Don't miss Jamie O’Meara's upcoming webinar on [continuous integration with Jenkins, Spring and Cloud Foundry](https://spring.io/blog/2015/05/28/webinar-ci-on-the-cloud-with-jenkins-spring-and-cloud-foundry)
-   Check out Phil Dutson's [upcoming webinar on moving `.jsp` and JSTL pages to Thymeleaf](https://spring.io/blog/2015/05/28/webinar-jstl-to-thymeleaf-transitioning-your-template)
-   Spring Integration lead Gary Russell [just announced Spring Integration 4.2M1 is now available](https://spring.io/blog/2015/05/28/spring-integration-4-2-milestone-1-is-available)
-   check out the replay of Marten Deinum's *awesome* talk that looks [at how to modernize legacy applications with Spring](https://spring.io/blog/2015/05/28/webinar-replay-improving-your-legacy-application-with-spring)
-   Janne Valkealahti just announced that [Spring Statemachine 1.0.0.M2 has been released](https://spring.io/blog/2015/06/02/spring-statemachine-1-0-0-m2-released).
-   our pal Eugen Paraschiv did an interesting [survey on Spring and Java 8's latest adoption numbers](http://www.baeldung.com/java-8-spring-4-and-spring-boot-adoption)
-   our pal Rafal Borowiec did a great post on [using the amazing Groovy language to simplify Integration testing](http://www.javacodegeeks.com/2015/05/groovier-spring-boot-integration-testing.html)
-   Yoichi Kikuchi put together a Japanese-language look at [using Spring Boot and MyBatis](http://yo1000.hateblo.jp/entry/2015/05/27/120647)
-   Kazuhira put together a Japanese-language post on using [Hibernate, the Spring Cache abstraction, and RedHat's Infinispan](http://d.hatena.ne.jp/Kazuhira/20150530/1432996038) distributed data grid together
-   The JavaTechPoint blog has a *super* simple [look at using Spring Boot](http://www.javatechpoint.com/spring-mvc/hello-world-spring-mvc-4-with-spring-boot-framework/)
-   Christoph Deppisch put together a nice post on [testing Apache Camel solutions with Spring](https://www.voxxed.com/blog/2015/05/apache-camel-integration-testing-part-2/)
-   Arvind Rai put together a nice look [at using Spring Quartz 2 support with Java configuration](http://www.concretepage.com/spring-4/spring-4-quartz-2-scheduler-integration-annotation-example-using-javaconfig)
