# See how to build, test, secure, and add hypermedia with this new tutorial, "Building REST Services with Spring"

_Engineering | Greg L. Turnquist |  October 23, 2014 | 1 Comment_

Greetings Spring Community!

Today we have released a new tutorial written by Spring geek [Josh Long](http://spring.io/team/jlong), the man who travels the globe talking about Spring: [Building REST Services with Spring](http://spring.io/guides/tutorials/bookmarks)

In this sleek tutorial, you can read about the simplicity and power of building RESTful web services using Spring. You'll also see how to:

-   Test them with Spring's powerful MockMvc
-   Secure them with [Spring Security](http://projects.spring.io/spring-security)
-   Plugin error handling
-   Add [hypermedia controls](http://projects.spring.io/spring-hateoas)
-   Dabble with [OAuth](http://projects.spring.io/spring-security-oauth)
-   See it all using [Spring Boot](http://projects.spring.io/spring-boot)

It's a really good read with LOTS of code (you know, the important stuff). What about the existing REST tutorial? It was a bit dated on concepts we wanted to cover and state-of-the-art, so it has been retired in favor of this one.

Share and enjoy!
