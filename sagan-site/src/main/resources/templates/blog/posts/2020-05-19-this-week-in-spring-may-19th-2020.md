# This Week in Spring - May 19th, 2020

_Engineering | Josh Long |  May 19, 2020 | 0 Comments_

Hi, Spring fans! Welcome to another installment of *This Week in Spring*! What a wonderful, *bootiful* week it's been! This last week saw the long-awaited release of Spring Boot 2.3.0!

There are a *lot* of wonderful features in this release including, but not limited to, Java 14 support, Docker image generation, graceful shutdown, liveliness probes, upgrades to Spring Data Neumann, and so much more. Here are some resources on some of those features. And, also, [*check out the blog*](https://spring.io/blog/2020/05/15/spring-boot-2-3-0-available-now).

-   R2DBC support (see this Spring Tips installment on using [R2DBC](https://www.youtube.com/watch?v=fIMdlE5Hvzk)
-   Docker container image support. See this blog [on Docker image generation](https://spring.io/blog/2020/01/27/creating-docker-images-with-spring-boot-2-3-0-m1)
-   Java 14 support. See this Spring Tips installment [on Java 14](https://www.youtube.com/watch?v=mr-7kGy8Yao)
-   This is *also* the version to use if you want to consume the Graal Native Image builder feature. Here's a [Spring Tips video](https://www.youtube.com/watch?v=u1XJTI1PVLw&feature=emb_title) on that, too.
