# Spring AMQP 1.0 GA released

_Releases | Mark Fisher |  August 25, 2011 | 0 Comments_

We are pleased to announce that Spring AMQP 1.0 GA (for Java) has been released! You can find links to all of the resources (documentation, samples, source code, forum, issue tracker, etc) at the [Spring AMQP Home Page](http://www.springsource.org/spring-amqp). The artifacts are available in the [SpringSource Maven repository](http://shrub.appspot.com/maven.springframework.org/release/org/springframework/amqp/) as well as [Maven Central](http://repo1.maven.org/maven2/org/springframework/amqp).

This project has been extremely popular during its milestone and release-candidate phases, and we would like to thank those of you in the community who have provided valuable feedback and raised JIRA issues along the way. We are looking forward to watching the community grow even more now that we have a GA release.

Be sure to checkout the recently announced [RabbitMQ service on Cloud Foundry](http://blog.cloudfoundry.com/post/8713844574/rabbitmq-cloud-foundry-cloud-messaging-that-just-works) along with [this tutorial](http://support.cloudfoundry.com/entries/20322602-getting-started-with-the-rabbitmq-service-from-a-spring-application) and [this blog](http://blog.springsource.com/2011/08/16/chatting-in-the-cloud-part-1/) that show how to use Spring AMQP within your cloud applications.

For those of you using the [Spring Integration](http://www.springsource.org/spring-integration) AMQP adapters, we will be releasing the first milestone of Spring Integration 2.1 that includes those adapters within the next few days. We have added a brand new [sample](https://github.com/SpringSource/spring-integration-samples/tree/master/basic/amqp) demonstrating those adapters.

Cheers! the Spring AMQP team
